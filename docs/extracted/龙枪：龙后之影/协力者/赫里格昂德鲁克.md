# 赫里格昂德鲁克

**赫里格·昂德鲁克 Hrigg Roundrook**

赫里格是一个交游甚广的矮人，是脸上总是挂着爽朗笑容的汉子，是爱吃如命、追求珍馐的老饕，也是与大奸大恶斗争的勇士。他还有四位兄弟姐妹，每一位都被战神奇力·裘理斯选中而获得了神圣的力量。五兄妹从家乡卡约林出发，分别踏上了只属于自己的圣途，贯彻奇力·裘理斯的教义直面巨龙大军。赫里格的圣途指引他来到了卡拉曼，如他所愿地站在了对抗入侵的前线。

**赫里格·昂德鲁克 Hrigg Roundrook** *1级施法者 （治疗者）； 中型类人（矮人）， 守序善良*

护甲等级：16（半身板甲）

生命值：11（2d8+2）

速度：30尺

力量14（+2） 敏捷12（+1） 
体质12（+1） 

智力10（+0） 感知14（+2） 
魅力11（+0）

豁免：感知+4

技能：运动+4，历史+2，医药+4，察觉+4

感官：黑暗视觉60尺，被动察觉14

语言：通用语，矮人语

熟练加值：+2

**额外熟练 Bonus Proficiencies。** 赫里格拥有简易武器、军用武器以及轻、中甲的熟练项。 

**矮人体魄 Dwarven Resilience。** 赫里格在进行对抗或结束自身中毒状态的豁免检定时具有优势。

**施法 Spellcasting。** 赫里格的施法关键属性是感知（法术豁免 DC 
12，法术攻击命中+4）。他准备有以下的牧师法术： 

*随意：神导术 guidance，圣火术 sacred flame1环法术（2法术位）：疗伤术 cure wounds*
