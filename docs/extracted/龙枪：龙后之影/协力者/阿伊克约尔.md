# 阿伊克约尔

**阿伊克·约尔 Ayik Ur**

年轻的弓箭手阿伊克是一位技艺高超的幸存者。当他的家乡克尔被蜂拥而至的巨龙大军占领时，阿伊克的腿被一匹冲撞的战马所伤，后来他成了一位难民。在逃离时，他发誓要用自己的弓箭报仇雪恨，对抗巨龙大军。阿伊克桀骜不驯，自傲到了鲁莽的地步，不顾一切地挑战他的仇人，誓要将故乡从巨龙大军的手中夺回。

**阿伊克·约尔 Ayik Ur**

*1级武者 （攻击手）； 中型类人（人类）， 中立善良*

护甲等级：15（镶钉皮甲）

生命值：11（2d8+2）

速度：30尺

力量11（+0） 敏捷16（+3） 
体质12（+1） 

智力11（+0） 感知13（+1） 魅力11（+0）

豁免：敏捷+5

技能：运动+2， 
自然+2，察觉+5，隐匿+5，求生+3

感官：被动察觉15

语言：通用语

熟练加值：+2

**攻击手 Attacker。** 阿伊克所有的攻击检定+2（下文动作已包含加值）。

**额外熟练 Bonus Proficiencies。** 阿伊克拥有简易武器、军用武器、盾牌以及全部护甲的熟练项。

**动作 Actions**

** 短剑 Shortsword。**近战武器攻击：命中+7， 触及5尺，单一目标。命中：6（1d6+3）点穿刺伤害。

** 长弓 Longbow。**远程武器攻击：命中+7，射程150/600，单一目标。命中：7（1d8+3）点穿刺伤害。
