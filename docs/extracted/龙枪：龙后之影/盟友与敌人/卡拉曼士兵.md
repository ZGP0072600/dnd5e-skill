# 卡拉曼士兵

卡拉曼士兵 Kalaman Soldier

卡拉曼士兵是卡拉曼军队中的基层部队。他们被训练去进行防御性作战，在战斗中互相支援。

卡拉曼士兵 Kalaman Soldier 中型或小型类人，任意阵营护甲等级：18（链甲，盾牌）生命值：16 (3d8 + 3)速度：30尺力量13（+1）敏捷12（+1）体质12（+1）智力10（+0）感知11（+0）魅力10（+0）技能：运动+3、察觉+2感官：被动察觉12语言：通用语挑战等级：1/2（100 XP）熟练加值+2

动作 Actions 多重攻击 Multiattack。士兵进行两次长剑攻击。长剑 Longsword。近战武器攻击：命中+3，触及5尺，单一目标。命中：6 (1d8 + 2）或双手持用时的7（1d10+2）挥砍伤害。反应 Reactions 坚守阵线 Hold the Line。若士兵5尺内的一个盟友将要进行一次豁免检定，士兵可以鼓励友军，令其在该豁免上具有优势。
