# 骷髅骑士

骸骨骑士 Skeletal Knight 骸骨骑士是强大的亡灵，他们从死亡中醒来，为更强大的邪恶存在——通常是死亡骑士——服务。成为骸骨骑士的人过去通常品德高尚，然而却背叛了自己的誓言，在耻辱之中死去。他们无法安息，坚定不移地忠于所服务的恶徒。而如果骸骨骑士的主人被摧毁，骑士也将获得自由意志，寻求自己的终局。骸骨骑士在战斗中残酷无情，并且难以摧毁。那些让他们受到诅咒的罪业化为执念，将他们留在世间，持续战斗直到连头骨都化为碎片。他们的刀剑造成的伤口久治不愈，会逐渐夺去受伤者的生命力，就算接受了魔法治疗，也会留下发黑的伤痕。

骸骨骑士 Skeletal Knight 中型亡灵，普遍守序邪恶 护甲等级：18（板甲） 生命值：112（15d8+45）速度： 30尺

力量20（+5）敏捷10（+0）体质16（+3）智力13（+1）感知14（+2）魅力10（+0） 豁免：体质+6，感知+5

伤害免疫：毒素 状态免疫：力竭，中毒

感官：黑暗视觉60尺，被动察觉12

语言：理解生前的语言但不会说

挑战等级：7（2.900xp）熟练加值+3

不死坚韧 Undead Fortitude。当骸骨骑士因受非钝击非重击伤害而使其生命值减至0时，他必须进行一次 DC 为5＋所受伤害值的体质豁免，豁免成功则生命值降至1。

非凡本质 Unusual Nature。骸骨骑士不需要呼吸，饮食和睡眠。

动作 Actions

多重攻击 Multiattack。骸骨骑士进行共计三次的衰亡之刃或投掷飞斧攻击，随意组合。

衰亡之刃 Enervating Blade。近战武器攻击：命中+8，触及5尺，单一目标。命中：15（3d6+5）暗蚀伤害，若目标为一个生物，则他直到骸骨骑士的下一回合开始之前不能恢复生命值，

投掷飞斧 Throwing Axe。近战或远程武器攻击：命中+8，触及5尺或射程20/60尺，单一目标。命中：14（2d8+5）挥砍伤害。
