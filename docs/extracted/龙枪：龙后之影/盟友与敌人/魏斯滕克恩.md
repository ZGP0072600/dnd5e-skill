# 魏斯滕克恩

魏斯滕·克恩 Wersten Kern

魏斯腾·克恩是索思爵士的旗手与勇士。和许多其他索思的骑士一般，她也在大灾变中和索思爵士一起死去，并被诅咒成为亡灵。在战斗中，她挥舞着一杆飘扬着索思的黑玫瑰旗帜的恶毒长枪，而她的尖叫也可以令强大的武者心脏停跳。薇斯滕无比坚定地听从索思的命令，并会为其服务直到自己再度迎来毁灭。

魏斯滕·克恩 Wersten Kern 中型亡灵，守序邪恶护甲等级：18（板甲）生命值：178（21d8+84）速度： 30尺

力量21（+5）敏捷10（+0）体质18（+4）智力13（+1）感知14（+2）魅力16（+3） 豁免：体质+9，感知+7

伤害免疫：毒素 状态免疫：力竭，恐慌，中毒

感官：黑暗视觉60尺，被动察觉12

语言：通用语，炼狱语，索兰尼亚语

挑战等级：14（11.500xp）熟练加值+5

不死坚韧 Undead Fortitude。当魏斯腾因受非钝击非重击伤害而使其生命值减至0时，她必须进行一次 DC 为5＋所受伤害值的体质豁免，豁免成功则生命值降至1。

非凡本质 Unusual Nature。魏斯腾不需要呼吸，饮食和睡眠。

动作 Actions

多重攻击 Multiattack。魏斯滕进行三次旌旗攻击并使用震惧连祷（若可用）。

旌旗 Banner Pike。近战武器攻击：命中+10，触及5尺，单一目标。命中：10（1d10+5）穿刺伤害外加13（3d8）暗蚀伤害。若目标为一个类人生物，则他必须成功通过一次 DC 16的魅力豁免，否则将被诅咒。诅咒持续直到其被移除诅咒或相似效应移除。当被诅咒时，黑玫瑰的荆条从目标身上生出，令其在属性检定和攻击检定上具有劣势并使其速度减半。豁免成功的生物在接下来的24小时内免疫诅咒效应。

施法 Spellcasting。魏斯滕施展以下法术中的一道，不需要任何材料成分并使用魅力作为施法关键属性（法术豁免 DC 16）：每项1/日：解除魔法，石墙术

震惧连祷 Terrifying Litany（充能5-6）。魏斯滕诵念那些被索思及其盟友所杀之灵魂的名字，引导他们死前的惊惧。任何在她周围30尺的非亡灵必须进行一次 DC 16的感知豁免。豁免失败者受到22（4d10）心灵伤害并被魏斯滕恐慌一分钟。豁免成功伤害减半且不会被恐慌。被恐慌的生物可以在自己回合结束时重新进行豁免，成功则结束该效应。
