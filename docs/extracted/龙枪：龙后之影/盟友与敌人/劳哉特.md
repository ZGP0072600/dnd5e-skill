# 劳哉特

劳哉特 Lohezet

劳哉特是一名高等术法会的黑袍法师，也是一名痴迷于研究灭绝生物和陨落帝国的学者。他认为巨龙大军的存在是重现安塞隆昔日荣光的一种方式。在巨龙大军入侵克尔和坎德摩尔期间，劳哉特对伊斯塔遗迹的研究揭露了浮空城昂亚瑞的位置。现在他和红龙军团同行，执着于找到众名失落之城，并令其重回天空。

劳哉特在战斗中使用的战术来自已经灭绝的剧毒捕食者们。他的魔法剧毒甚至能逐渐耗竭最为强大的对手，为他创造撤离的机会，或者让敌人衰弱而死、化为褪色的尸骸。

劳哉特 Lohezet 中型类人（人类，法师），中立邪恶护甲等级：12（使用法师护甲时为15）生命值：137（25d8+25）

速度30尺

力量9（-1）敏捷14（+2）体质12（+1）智力20（+5）感知14（+2）魅力11（+0）

豁免：体质+5，感知+6

技能：奥秘+9，历史+9，医药+6

伤害免疫：毒素

状态免疫：中毒

感官：被动察觉12

语言：通用语，矮人语，精灵语，炼狱语

挑战等级：12（8,400XP）熟练加值+4

精通毒性 Toxic Mastery。劳哉特无视生物对毒素伤害的抗性。

动作 Actions

多重攻击 Multiattack。劳哉特发动三次凋灵爆攻击并使用瘴气（若可用）。他也可以将其中一次攻击替换为施法。

凋零爆 Withering Blast。近战或远程法术攻击：命中+9，触及5尺或射程60尺，单一目标。命中：18（2d12+5）暗蚀伤害。

瘴气 Miasma（充能4-6）。劳哉特在以他120尺内一点为中心20尺半径的球型空间中魔法性地召唤一道汹涌翻腾的紫色云雾。球形区域内为重度遮蔽，且当一个生物在该区域内开始其回合，或在一个回合中第一次进入该区域时，必须进行一次 DC 17的体质豁免。豁免失败者受到39（6d12）毒素伤害且陷入中毒直到其下一回合开始。豁免成功者只受到一半伤害且不会中毒。云雾将存在一分钟，或由劳哉特提前结束（无需动作），或在劳哉特再次使用此动作时结束。一道强风可以吹散该云雾。

施法 Spellcasting。劳哉特施展以下法术中的一道，不需要任何材料成分且使用智力作为施法关键属性（法术豁免 DC 17）：*随意：侦测魔法，光亮术，魔法伎俩*

*每项2/日：任意门，法师护甲*

*每项1/日：秘法眼，支配类人，力场墙*

反应 Reactions 剧毒反噬 Noxious Rebuke（3/每日）。当一个劳哉特60尺内的生物对他造成伤害时，劳哉特魔法性地使用一道邪秽的紫色毒雾进行反击。该生物必须进行一次 DC 17的体质豁免，豁免失败受到16（2d10+5）毒素伤害，豁免成功伤害减半。
