# 波扎克龙人

波扎克龙人 Bozak Draconian 波扎克龙人生自青铜龙蛋，能使用魔法在战斗中支援友军。他们的翅膀没有结实到足以令其飞行的程度，但也使他们能在坠落时滑翔。当波扎克龙人死亡时，他们的血肉会枯萎，然后骨头会爆炸，在所有方向上产生一场魔法性的碎屑之雨。

波扎克龙人 Bozak Draconian

中型怪兽（术士），通常守序邪恶

护甲等级：15（天生护甲）

生命值：40（9d8）

速度：30尺

力量14（+2）敏捷10（+0）体质11（+0）

智力11（+0）感知10（+0）魅力14（+2）

豁免：智力+2，感知+2，魅力+4

感官：黑暗视觉60尺，被动察觉10

语言：通用语，龙语

挑战等级： 2（450XP）熟练加值+2

临终苦难 Death 
Throes。当波扎克龙人的生命值降低为0时，他的鳞片和血肉迅速枯萎，骨头炸裂开来。每个波扎克龙人周围10尺内的生物都必须成功通过 DC 10的敏捷豁免否则受到9（2d8）力场伤害。

滑翔 Glide。当波扎克龙人坠落且并未处于失能状态时，他在计算坠落伤害时从中减去最多100尺高度。每坠落1尺，他能够最多水平移动2尺距离。

动作 Actions

多重攻击 Multiattack。波扎克龙人进行两次三叉戟或闪电迸发攻击。

三叉戟 Trident。近战或远程武器攻击：命中+4，触及5尺，或射程20/60尺，单一目标。命中：5（1d6+2）穿刺伤害，或以双手发动近战攻击时6（1d8+2）穿刺伤害。

闪电迸发 Lightening Discharge。远程法术攻击：命中+4，射程60尺，单一目标。命中：10（3d6）闪电伤害。

施法 Spellcasting。波扎克龙人施展以下法术中的一个，他无视材料成分并使用魅力作为其施法关键属性（法术豁免 DC 12）：

每项1/日：变巨术/缩小术，隐形术，臭云术，蛛网术
