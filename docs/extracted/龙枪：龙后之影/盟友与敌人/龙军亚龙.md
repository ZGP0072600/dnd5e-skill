# 龙军亚龙

龙军亚龙 Dragon Army Dragonnel 来自圣克仙周围火山的亚龙与红龙亲缘相近，有着相似的鳞片与外形。它们是残忍而自私的生物，由巨龙大军训练为那些可靠的军官的坐骑。虽然它们并没有红龙毁灭性的吐息武器，但这些亚龙也已习惯于火焰——它们致命的尖牙与利爪闪耀着炙热的余烬。
