# 高等亡骸龙

高等亡骸龙 Greater Death Dragon

一些亡骸龙设法保留了生前几乎全部的自我，尽管这自我已被他们现在的状态所扭曲。这些高等亡骸龙会有计划地实现自己的目的，还如同生前般贪婪地搜刮并严密地守护宝藏。在战斗中，他们是致命的强敌，他能引导灾变业火，这火焰缭绕于他，也为他们的吐息武器供能。他们享受用利齿咬住敌人，冲入高空，将其扔下令其坠亡的残酷乐趣。

高等亡骸龙 Greater Death Dragon

巨型亡灵，普遍混乱邪恶

护甲等级：16（天生护甲） 生命值：230（20d12+100）

速度：40尺，飞行80尺

力量23（+6）敏捷10（+0）体质20（+5）

智力11（+0）感知14（+2）魅力10（+0）

豁免：敏捷+5，感知+7

技能：察觉+7，隐匿+5

伤害抗性：穿刺

伤害免疫：暗蚀，毒素

状态免疫：力竭，中毒

感官：盲视60尺，黑暗视觉120尺，被动察觉17

语言：通用语，龙语

挑战等级：14（11，500xp）熟练加值+5

传奇抗性 Legendary Resistance（3/日）。高等亡骸龙豁免失败时，可以将其改为豁免成功。

非凡本质 Unusual Nature。高等亡骸龙不需要呼吸，饮食和睡眠。

动作 Actions

多重攻击 Multiattack。高等亡骸龙发动一次啃咬与两次爪击。

啃咬 Bite。近战武器攻击，命中+11，触及10尺，单一目标。命中：17（2d10+6）穿刺伤害外加4（1d8）暗蚀伤害。若目标是一个大型或更小的生物，则他被擒抱（逃脱 DC 19）。该生物在被擒抱期间陷入束缚，并且高等亡骸龙不能啃咬其他目标。

爪击 Claw。近战武器攻击，命中+11，触及5尺，单一目标。命中：10（1d8+6）挥砍伤害。

灾变吐息 Cataclysmic Breath（充能5-6）。高等亡骸龙呼出一道60尺锥形的幽魂般的紫色火炎之潮。区域内的所有生物必须进行一次 DC 18的敏捷豁免，豁免失败受到45（10d8）暗蚀伤害，豁免成功伤害减半。一个被该伤害降至0生命值的生物会死亡。此外，任何被该吐息伤害杀死的中型或更小的类人生物，以及在吐息区域内的此类生物的尸体都将变为受高等亡骸龙控制的丧尸 zombie。丧尸的先攻算作和高等亡骸龙相同，在高等亡骸龙的回合结束后立即行动。除非有其他命令，否则丧尸会尝试杀死它们遇到的任何非亡灵。

传奇动作 Legendary Actions 高等亡骸龙拥有3传奇动作，用来选择执行下列动作项。他每次只可选择执行一个传奇动作项，且必须在另一生物回合结束时执行。高等亡骸龙在其回合开始时恢复所有消耗的传奇动作数。

爪击 Claw。高等亡骸龙发动一次爪击攻击。

灾变疾驰 Cataclysmic Rush（消耗2动作）。高等亡骸龙携带着其擒抱的生物移动至多等于其飞行速度一半的距离且不会触发借机攻击。若其在移动中进入了一个中型或更小的生物的空间，该生物受到4（1d8）暗蚀伤害。一个生物在一个回合内只能受到一次该伤害。
