# 伊斯塔自动机械

伊斯塔自动机械 Istarian Drone 伊斯塔自动机械是由大理石和闪亮金属制成的古代构装生物，其上镶着的晶体闪烁着电流的火花。这些自动机械的设计目的是用来建造，正是它们建造了那些曾经象征着伊斯塔帝国荣耀的奇观。它们看起来就像一只矮胖的螳螂，有着四条昆虫型的步足和一对带有倒钩的镰刀状的手臂，用来搬运和放置建筑材料。自动机械还能制造一种用来硬化结晶砂浆的粘稠凝胶，这也可以用来束缚攻击者。

伊斯塔自动机械 Istarian Drone 中型构装，无阵营护甲等级：17（天生护甲）生命值：127（15d8+60）速度:30尺，攀爬30尺力量20（+5）敏捷10（+0）体质18（+4）智力4（-3）感知10（+0）魅力 4（-3）伤害免疫：闪电，毒素状态免疫：魅惑，力竭，恐慌，麻痹，中毒感官：盲视60尺（此范围外视为目盲），被动察觉10语言 理解其创造者说的语言但不会说挑战等级：6（2，300xp）熟练加值+3

蛛行 Spider Climb。自动机械可以攀爬在难以攀爬的表面上，包括倒过来在天花板上行走，且不需要进行属性检定。非凡本质 Unusual Nature。自动机械不需要呼吸，饮食和睡眠。动作 Actions 多重攻击 Multiattack。自动机械进行两次爪击攻击。爪击 Claw。近战武器攻击：命中+8，触及5尺，单一目标。命中：9（1d8+5）穿刺伤害外加4（1d8）闪电伤害。若目标是一个中型或更小的生物，则它被擒抱（逃脱 DC 15）。自动机械有两只爪子，每只能且只能擒抱一个目标。晶化喷射 Crystalline Spit（充能5~6）。自动机械喷出凝胶覆盖一处20尺长，5尺宽的线性区域。每个位于该区域内的生物都必须进行一次 DC 15的敏捷豁免。豁免失败者受到14（4d6）闪电伤害并被硬化为结晶的凝胶束缚。该生物被束缚直到结晶被摧毁。结晶的 AC 为15，拥有15点 HP，免疫毒素和心灵伤害并对雷鸣伤害具有易伤。豁免成功伤害减半并不会被束缚。
