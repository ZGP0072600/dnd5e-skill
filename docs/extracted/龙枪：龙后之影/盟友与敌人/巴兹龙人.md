# 巴兹龙人

巴兹龙人 Baaz Draconian 巴兹龙人是自黄铜龙蛋中诞生的狂热步卒。巴兹龙人是龙人这一族中个头最小的成员，他们脆弱的翅膀无力飞行但仍然能够控制坠落的势头。这些翅膀小到足够藏在斗篷或袍子下面，因此巴兹龙人有时也会作为巨龙大军的间谍行动。龙的出现会令巴兹龙人陷入极度的狂热，而当巴兹龙人死去时，他们的身体会变成石头，放出石化气体的云雾，将他们的尸骸以及附近的所有生物变成石头。

巴兹龙人 Baaz Draconian

中型怪兽，普遍守序邪恶

护甲等级：14（天生护甲）

生命值：22（4d8+4）

速度：30尺

力量13（+1）敏捷11（+0）体质13（+1）

智力8（-1）感知8（-1）魅力10（+0）

感官：黑暗视觉60尺，被动察觉9

语言：通用语，龙语

挑战等级： 1/2（100XP）熟练加值+2

可控坠落 Controlled 
Fall。当巴兹龙人坠落且并未处于失能状态时，他在计算坠落伤害时从中减去最多100尺高度。

临终苦难 Death 
Throes。当巴兹龙人的生命值降低为0时，他的身体变为石头并释放出石化气体。每个巴兹龙人周围5尺内的生物都必须成功通过 DC 11的体质豁免否则被束缚并开始变为石头。被束缚的生物必须在其下个回合结束时再次进行豁免检定。成功的情况下，该效应结束；否则该生物被石化1分钟。1分钟后，巴兹龙人的尸体碎作尘埃。

龙类忠诚 Draconic Devotion。若巴兹龙人能看到一个非敌对龙类，则他在攻击检定上有优势。

动作 Actions

多重攻击 Multiattack。巴兹龙人进行两次短剑攻击。

短剑 Short 
Sword。近战武器攻击：命中+3，触及5尺，单一目标。命中：4（1d6+1）穿刺伤害。
