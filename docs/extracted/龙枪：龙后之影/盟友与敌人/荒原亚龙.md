# 荒原亚龙

荒原亚龙 Wasteland Dragonnel 安赛隆的北方荒原是荒原亚龙的家园，这种龙类生物与赤铜龙亲缘相近。荒地亚龙身形灵巧迅捷，鳞色为暗铜色。这些贪玩的生物靠着飞离敌人的触及范围并喷吐强酸来保护自己的领地，某种意义上这与它们的赤铜龙表亲有些相似。

荒原亚龙 Wasteland Dragonnel 大型龙类 通常混乱善良护甲等级：13生命值：65（10d10+10）速度 30尺，飞行速度60尺力量16（+3） 敏捷16（+3） 体质12（+1） 智力 8（-1） 感知13（+1） 魅力10（+0）技能 察觉+3伤害抗性 强酸感官 盲视30尺，黑暗视觉120尺，被动察觉13语言 能听懂通用语和龙语但是不会说挑战等级： 3（700xp) 熟练加值+2飞掠 Flyby.荒原亚龙飞出敌人触及范围时不会触发借机攻击。动作 Actions 多重攻击 Multiattack。荒原亚龙发动两次撕裂攻击。撕裂 Rend。近战武器攻击：命中+5，触及5尺，单一目标。命中：10（2d6+3）的挥砍伤害。喷酸 Acid Spit。远程武器攻击：命中+5，射程60尺，单一目标。命中：20（5d6+3）强酸伤害。
