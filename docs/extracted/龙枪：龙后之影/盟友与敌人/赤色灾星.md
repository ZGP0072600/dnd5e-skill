# 赤色灾星

赤色灾星 Red Ruin

亚龙骑士空战王牌‘赤色灾星’以惊人的意志与反应力而声名远播，领导着红龙军团的空降部队。她真实的姓名无人知晓，导致盟友仅能用‘指挥官’或‘赤色灾星’（因击坠了众多空中目标而得名）来称呼她。她很少摘下那独特的亚龙形头盔，很多人因此猜测着她的真实身份。而她同样很少说话，并更偏向于用简短明晰的手势来在空战中指挥亚龙骑士们。

赤色灾星 Red Ruin 中型类人，守序邪恶护甲等级：20（板甲，盾牌）生命值：150（20d8+60）速度：30尺力量19（+4）敏捷12（+1）体质17（+3）智力13（+1）感知14（+2）魅力15（+2）豁免：力量+8，敏捷+5技能：运动+8，察觉+6伤害抗性：火焰感官：被动察觉16语言：通用语，龙语挑战等级：10（5.900xp）熟练加值+4

龙类忠诚 Draconic Devotion。若赤色灾星能看到一个非敌对龙类，则她在攻击检定上有优势。骑乘战斗大师 Mounted Combat Master。在骑乘时，赤色灾星可以使得针对她坐骑的攻击变成针对她自己。骑乘反射闪避 Mounted Evasion。当赤色灾星或她的坐骑在为敏捷豁免成功后只承受一半伤害的效应进行豁免时，他们在豁免成功时不受伤害，豁免失败时只受到一半伤害。动作 Actions 多重攻击 Multiattack。赤色灾星发动三次余烬骑枪攻击。余烬骑枪 Ember Lance。近战武器攻击：命中+8，触及10尺，单一目标。命中：10（1d12+4）穿刺伤害外加7（2d6）火焰伤害。若目标是一个中型或更小的生物，则其必须通过一次 DC 16的力量豁免，否则应击倒地。爆裂手弩 Explosive Hand Crossbow（充能5~6）。赤色灾星用爆裂手弩向120尺内她可见的一点发射一道弩矢。当弩矢抵达该点，或提前命中一个物件，它将在其周围20尺半径球型空间产生爆炸。任何位于该区域内的生物都必须进行一次 DC 15的敏捷豁免，豁免失败受到35（10d6）火焰伤害，豁免成功则只受到一半的伤害。
