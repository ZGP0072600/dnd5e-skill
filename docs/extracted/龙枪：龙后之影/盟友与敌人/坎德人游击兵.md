# 坎德人游击兵

坎德人游击兵 Kender Skirmisher

坎德人游击兵是无畏的战士，活用隐秘的行动与狡黠的头脑来保卫朋友和家园。他们擅长通过蓄意破坏至关重要的物品或嘲讽对手、使其做出鲁莽的判断来扰乱敌人。他们使用坎德人标志性的武器：胡帕克杖，这是一种融合了矛和投石索的长杖。

坎德人游击兵 Kender Skirmisher

小型类人，任意阵营

护甲等级：14（皮甲）生命值：14（4d6）

速度： 30尺

力量8（-1）敏捷16（+3）体质10（+0）

智力12（+1）感知8（-1）魅力14（+2）

技能：察觉+3，巧手+7，隐匿+5

状态免疫：恐慌

感官：被动察觉13

语言：通用语，坎德语

挑战等级： 1/4（50xp） 熟练加值+2

动作 Actions 胡帕克杖 Hoopak。近战或远程武器攻击：命中+5，触及5尺或射程40/160尺，单一目标。命中：6（1d6+3）穿刺伤害或在用胡帕克杖的投石索进行远程攻击时5（1d4+3）钝击伤害。

嘲讽 Taunt。坎德人对一个60尺内的可见生物发出一连串辱骂。如果目标可以听到坎德人的声音，则必须成功通过一次 DC 12的感知豁免，否则直到下一回合结束前在攻击检定上都具有劣势。

附赠动作 Bonus Actions

神出鬼没 Elusive。坎德人执行撤离或躲藏动作。
