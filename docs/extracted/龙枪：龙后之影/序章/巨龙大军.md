# 巨龙大军

**巨龙大军 The Dragon Armies**

龙后的军事力量被组织成了五个巨龙军团。每个军团里都有龙人，也有人类、地精、大地精和食人魔，还有一些追求财富或者希望寻回大灾变前的荣光的人。每个军团都有色彩龙和他们体型更小的亲族——比如四足亚龙 dragonnel 和飞龙 wyvern——的协助。这些长着翅膀的恐怖生物提供了在战场上千年未见的惊人优势。

龙枪:克莱恩的勇士 Dragonlance:Warriors of 
Krynn

龙后选出了五个最为狡诈也最为忠诚的追随者，将他们提拔为龙骑将 dragon 
highlord，每人统领一个巨龙军团。所有的龙骑将都需要服从塔克西丝的勇士杜尔凯特·艾瑞阿卡斯 Duulket 
Ariakas，但无论是在军团之内还是在军团之间，背叛都时常发生。龙后鼓励这样的冲突，相信这样的行为能让最无情、最有能力的领导者脱颖而出。

红龙军团是塔克西丝麾下建立最早，规模最大，实力最强的军团。领导它的是狂热的龙骑将猛敏那 Verminaard。但现在猛敏那暂时把自己手中的许多部队交给了他的助手龙骑督 Dragon 
Highmaster 坎萨尔迪·火眼 Kansaldi 
Fire-Eyes 来领导，让她带头发起一次对索兰尼亚的凶猛进攻，以此来检验她的能力。与此同时，猛敏那的部队跟随龙后的低语进入了阿班尼西亚的南部地区。

野心勃勃的龙骑将奇蒂拉·钨斯·马塔 Kitiara Uth 
Matar 领导着蓝龙军团。在龙后的军队中，蓝龙军团的实力位居第二，正准备也加入红龙军团策划的对索兰尼亚多管齐下的入侵。绿龙军团目前由萨拉可汗 Salah-Khan 指挥，他是克尔的领袖，将整个国家的游牧民都团结到了巨龙大军的统治下。塔卡 Takar 的半食人魔卢西恩 Lucien 指挥着黑龙军团，集中力量支援其他军团，并在他们征服土地之后维持占领。诺德玛和古德兰就处于黑龙军团的控制下。最后一个军团是规模较小的白龙军团，由龙骑将费尔萨斯 Feal-Thas 领导，他是一个出身于西瓦那斯提的精灵，也是高等术法会的黑袍法师。
