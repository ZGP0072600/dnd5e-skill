# 囤积者

囤积者 Hoardsperson
“金币的支出由我自行决定，所以最好对我好一点哦。”
——囤积者开玩笑似地说（绝非戏言)
正如其名，艾奎兹玄有限责任公司(注:直译为资产并购有限公司)的核心业务就是收购，而囤积者的工作则正是负责保管那些辛苦赚来的战利品。每个囤积者都扮演着——管理特许经营机构的资源，包括冒险中使用的所有物资和获得的财宝——这一至关重要的角色。囤积者尤其要确保的是，物资是由艾奎兹玄有限责任公司所批准的渠道进行征用的，并确保在任务结束时将合适的战利品发送到总部。 

作为一名囤积者，你的守信用和负责任是为他人所称道的。你以工于心计、戒奢以俭、未雨绸缪来支持你的特许经营机构。
 知名的囤积者：克里斯·卓尔布（“C”小队），费尼克斯·锻砧（“B”小队），双件干披风（译者注：斑猫人名，下文的游荡者。）。
基本职能：管理库存和物品；跟踪资源去向。 
职位熟练项：除了下面列出的熟练项外，当你在为商品或服务讨价还价、为装备和财宝的品质进行评估，或是分析所在地区的资源时，你可以将你的熟练加值加到与这些活动相关的属性检定上去。

斑猫人囤积者（游荡者）
“两件干披风永远在寻找“闪烁之物”，对财宝深深着迷的同时，她本人对“财宝”也有一个宽泛的定义。正如一句古老的谚语所说——“财宝是特许经营的心脏”，而斑猫人天天都向她的特许经营机构念叨这句话。她相信那些囤积的财宝——通过囤积者的活化战利品背包进出总部——是团队活动中最正确、最理性的判断。” 
（斑猫人可在《瓦罗的怪物指南》中找到。）

活化战利品背包是一种魔法生物，用于保护特许经营机构的资金和贵重物品。它的内部与一个位于深水城总部保险库的安全保险箱相连接，背包会定期将特许经营机构的财富转移至保险箱中。
以一个动作，你可以通过一次 DC 15的敏捷（巧手）检定，将任意金额的特许经营资金转移到你的背包中。

作为一名3级囤积者，你的活化战利品背包升级为一件珍稀品质的魔法物品，并能够再现李欧蒙秘藏箱 Leomund's Secret Chest 法术中的复制箱功能。你可以通过活化战利品背包打开秘藏箱来存取物品，甚至可以存放那些大小原本不适合放在你的背包中，但适合放在箱子里的物品。由于总部与你不会想知道的外层位面生物达成了你肯定不会想知道的交易，这个法术效应绝不会被中止。
各取所需 That 
Thing You Need
第3级起，你可以使用一个附赠动作，把手伸到你的活化战利品背包里，并进行一次 DC 15的敏捷（巧手）检定。如果成功，你可以从玩家手册第5章的冒险装备表中选择一件物品拿出。 该物品必须适合放入你的秘密箱，并且价值不超过15 gp。一旦你尝试这样从背包中取出五件物品，直到第二天黎明之前，你将无法再次从中取出物品。 
次元洞背包 Portable Hole Satchel
第4级起，你的活化战利品背包再次升级，变成了一件极珍稀品质的魔法物品。现在你的背包可以使用一个特殊的秘密箱，它内部有一个次元洞 Portable 
Hole，直径达6英尺，深度达10英尺。跟以前一样，你可以把任意尺寸合适的物件放进这个次元洞里，即使这些物品的大小本不能放进你的背包里。
