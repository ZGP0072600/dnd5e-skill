# 赌徒

**赌徒 Gambler**

制造大量黄金只需要一点点黄金。 
而在你根本不需要金子的时候，不论多少都是一点点。无论你是一个好赌徒还是一个坏赌徒都无关紧要，因为没有人能预测到幸运女神的奇想。有时你会成功，有时你会失败。但是赌博的问题就是，总有人愿意打赌。

**技能熟练项：** 欺瞒，洞悉

**工具熟练项：** 一种赌具

**语言：** 任选一门语言

**装备：** 一种赌具，一个幸运护身符，一套高档服装，以及一个装有15 
gp 的腰包

**特性：永远别告诉我赔率 Feature: Never Tell Me the Odds**

 
胜算和概率是你的谋生之道。在涉及运气游戏或计算最佳计划赔率的休整期活动中，你可以根据 DM 的判断，切实了解哪些选择可能是最佳选择，哪些机会又似乎好得不真实。

**建议特征 Suggested Characteristics**

有些人赌博是迫不得已。另一些人则是出于无聊。还有一些人则沉迷于命运转折带来的输赢的快感。对有些人来说，赌博与其说是一个机会问题，不如说是一个寻求一切优势以确保能有收益的问题。最好的赌徒可能会输掉一切，而最差的赌徒有时也会赢钱。无论如何，你总能通过眼神分辨出赌徒。幸运女神的眷顾与否总是纠缠着他们。
