# 发条龙

**发条龙 Clockwork Dragon**

这些精心制作的构装生物通常是对金属龙形状的模仿。其身上镀有黄铜、青铜、赤铜或仿制的金银，乍一看往往会被认为是精美的巨龙塑像。发条龙是麻烦的守卫和防御者，其高等的智能让它们可以被编入大量命令，并独立对潜在的威胁做出反应。

尽管大部分发条龙都具有造成火焰伤害的龙息武器，但取决于制作者的奇想，这些龙息武器也可能造成强酸、寒冷或闪电伤害。
