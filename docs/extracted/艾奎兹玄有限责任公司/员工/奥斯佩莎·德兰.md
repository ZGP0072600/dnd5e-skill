# 奥斯佩莎·德兰

**奥斯佩莎·德兰 Auspicia Dran**

奥斯佩莎．德兰是 欧敏·德兰最小的半精灵姊妹，当她小时候被一个叫做游荡地穴 
的神秘生物抓住时，她的生活被撕裂了。多年来，她一直在地穴里做着某种黑暗的梦，最终被哥哥 欧敏 的不懈努力和他精心挑选的 C 小队 
的英勇救了出来。现在，她安全地回到了这个世界，带着地穴创造的虚假吉兆，她专注于试图发现自己的身份，并努力克服她的苦难所带来的持续的复杂性。

当 奥斯佩莎从游荡地穴里离开时，她的心灵力量与日俱增。 
她忍不住要看穿别人的心思，但她发现戴上钻石耳环有助于抑制这种不知不觉的能力。这种魔力也让她以一种她无法理解的方式与地穴保持着联系，尽管它让她偶尔想起仍被困在地穴里的其他人，并对遥远的地方产生了意识。 
在战斗中，她表现出对事件发生前的本能知识，经常利用这种能力使战斗停顿， 鼓励谈判。 但她才刚刚开始充分挖掘自己的预测潜力。
