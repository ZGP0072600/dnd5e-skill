# 潘德拉贡·贝辛格

**潘德拉贡·贝辛格 Pendragon Beestinger**

在 芭玛·路提尔 离开团队后，有能力的法师潘德拉贡·贝辛格接任 B 小队制图师。 
事实上，他的第一个任务是协助团队抓捕这位四处游荡的吟游诗人。 潘德拉贡是 C 小队 
的罗西·贝辛格的养子，几乎在所有方面都是他母亲的陪护——包括他想要将一些家族传闻中的犯罪利益主流化的目标。 事实上， 
潘德拉贡曾试图杀死他的母亲，并在此过程中自杀。 尽管他在 罗西 的要求下复活了，但两人之间的关系仍然很冷淡。

由于贝辛格家族的黑暗谣言， 潘德拉贡被法师学院开除了。 现在他是一个自学成才的法师， 
擅长使用危险的咒语和他在扮演神秘考古学家时经常遇到的奇怪饰品。 协助冒险团队给他带来家庭的感觉， 让他留在了 B 小队 ，即使不幸的是，奥克．图斯崔 
扮演的是父亲的角色。
