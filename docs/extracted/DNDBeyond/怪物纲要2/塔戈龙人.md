# 塔戈龙人

塔戈龙人 Traag Draconian

在腐化龙蛋的实验中，巨龙大军创造出了塔戈龙人——巴兹龙人士兵的前身。和他们的龙人兄弟们不同，塔戈龙人不过是嗜血的杀戮机器。很少有指挥官能够在战斗中成功利用塔戈龙人鲁莽的暴力以取得优势；对大多数人来说，塔戈龙人对盟友和敌人同等危险。即使魔法也无法控制住他们对暴力的渴望。

塔戈龙人的残翼没有足够的力量支持它们的巨体在空中飞行，只能最小限度地减缓塔戈龙人的坠落速度。他们的金属鳞片上覆盖着一层病态的铜锈。在战斗中，这种龙人可以用邪恶的火焰覆盖自己的身躯，燃烧途经一切。当一只塔戈龙人死去时，它的身体会爆炸并化作一滩厚重而恶心的淤泥。

塔戈龙人 Traag Draconian

大型怪兽，通常混乱邪恶

护甲等级 14（天生护甲）

生命值 68（8d10+24）

速度 40尺

力量21（+5）敏捷10（+0）体质17（+3）智力6（-2）感知8（-1）魅力9（-1）

豁免 力量+8，体质+6

状态免疫 魅惑，恐慌

感官 黑暗视觉60尺，被动察觉9

语言 通用语，龙语

挑战等级 5（1,800XP）
熟练加值+3

血腥狂怒 Blood Frenzy。龙人对生命值没有全满的生物发动近战攻击时，其攻击检定具有优势。

可控坠落 Controlled 
Fall。当龙人坠落且并未处于失能状态时，它在计算坠落伤害时从中减去最多50尺高度。

临终苦难 Death 
Throes。当龙人的生命值降低为0时，它会化作一滩污泥，充满半径10尺的区域。每个区域内的生物都必须成功通过 DC 14的敏捷豁免否则在1分钟内被污泥覆盖。当被污泥覆盖时，生物的速度会减半。一个生物可以用其动作刮掉自己或一个5尺内生物身上的污泥。

动作 Actions

多重攻击 Multiattack。龙人进行两次爪击。

爪击 Claw。近战武器攻击：命中+8，触及10尺，单一目标。

命中：14（2d8+5）挥砍伤害。

烈焰蹂躏 Flaming Carnage（充能5-6）。龙人沉浸于自己的怒火，将烈焰披挂全身并移动最多等于自身速度的距离。在执行这种移动时，龙人可以进入其他生物的空间。当它在移动中进入其他生物占据的空间时，该生物必须进行 DC 14的敏捷豁免。失败的情况下，该生物受到22（4d10）火焰伤害并倒地。豁免成功的情况下，该生物仅受到一半伤害，并被推到距离龙人的路径最近的空间内。一个生物在一回合内只能被同一只龙人的烈焰蹂躏影响一次。

反应 Reaction

暴力还击 Violent 
Retaliation。如果一个龙人10尺内的生物以一次攻击命中它，龙人可以对该生物进行一次爪击。如果爪击命中，它额外造成4（1d8）挥砍伤害。
