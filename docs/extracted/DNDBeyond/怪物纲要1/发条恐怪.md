# 发条恐怪 Clockwork Horror

发条恐怪一种是毫无情感的昆虫型机器人——身覆精金或其他贵金属装甲的骇人机器。它们乘坐着被征服的魔法船漫游荒宇，收集用于建造更多同类所需的贵金属和水晶。发条恐怪除了自我复制之外便没有其他的任何目标，并会不惜一切代价创造新的发条恐怪。假使它拥有必需的材料，制造另一个同类所需的时间大约是十天。

发条恐怪主体直径约两尺，但算上腿部之后它的总直径可以到达四尺。它的额部嵌有一颗让发条恐怪可以视物的水晶。

发条恐怪通过咔哒声、嗡鸣声和其他类似的机械声音相互交流，这些声音完美地模仿了斯里克林语，这也让某些人推测世上第一台发条恐怪是一位螳螂人奇械师的创意。发条恐怪也可以从它的水晶眼睛中发射出光线，向其他能看到光的生物发出点和破折号的信息。除了发条恐怪之外，其他生物也可以学习这种闪烁的光线密码，即光电码（Ziklight）。

当其他生物妨碍了一只发条恐怪时，它会用自己那锋利的下颚和两个安装在前肢尖端的微型旋转电锯攻击那个生物。发条恐怪也通过嵌在其体内的闪电短杆释放闪电束。

当发条恐怪死亡时，创造它的魔法会在1分钟内快速地将它吞噬。它的金属身体和水晶眼睛将迅速分解，最后只剩下一小堆闪闪发光的尘埃。

## 特性 Traits

**停机 Shutdown。** 如果发条恐怪成为解除魔法的目标， 它必须成功通过一次难度等同于施法者法术豁免 DC 的体质豁免，否则昏迷1分钟或者直到受到任何伤害。

**非寻常天性 Unusual Nature。** 发条恐怪不需要呼吸、饮食或睡眠。

## 动作 Actions

**多重攻击 Multiattack。** 发条恐怪发动一次啃咬攻击和两次旋转电锯攻击或发动两次闪电震荡攻击。

**啃咬 Bite。** 近战武器攻击 : 命中+4， 触及 5 尺， 单一目标。 伤害 : 6 (1d8 + 2) 点穿刺伤害。

**旋转电锯 Rotating Saw。** 近战武器攻击 : 命中+4， 触及5 尺， 单一目标。 伤害 : 4 (1d4 + 2) 点挥砍伤害。

**闪电震荡 Lightning Jolt。** 远程法术攻击 : 命中+4， 射程120尺， 单一目标。 伤害 : 7 (1d10 + 2) 点闪电伤害。

**接入法驱魔舵 Spelljamming Helm Interface。** 发条恐怪附着在其5尺以内可见的一个法驱魔舵上，并瞬间与之同调。如果有其他生物已经与船舵同调，则当发条恐怪的同调开始时，那名生物的同调随之结束。尽管发条恐怪不是一名施法者，但它依然可以操控魔舵。发条恐怪可以通过一个附赠动作从魔舵上脱离并结束其与魔舵的同调。
