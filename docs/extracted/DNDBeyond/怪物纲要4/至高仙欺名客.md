# 至高仙欺名客

**至高仙欺名客 High Fae Impostor**

至高仙欺名客是顽皮狡诈的阴谋家，喜欢用魔法伪装自身以惹怒其他生物。它们骗人的理由多种多样，从无害的恶作剧到恶意的渗透。至高仙欺名客在盗取用另一生物的身份时，它会魔法性地访问该生物的部分心智——并得知该生物可能与一个普通熟人分享的任何信息。虽然信息并不全面，但却足以助至高仙欺名客十分有效地冒充目标。
至高仙欺名客 High Fae Impostor 中型妖精，任意阵营 AC 15HP 149（23d8+46）速度30尺，飞行30尺（悬浮）力量13（+1）敏捷21（+5）体质15（+2）智力18（+4）感知16（+3）魅力23（+6）豁免：敏捷+9，感知+7技能：欺瞒+14，表演+10，隐匿+13伤害抗性：非魔法攻击的钝击，穿刺和挥砍感官：黑暗视觉60尺，被动察觉13语言：通用语，木族语挑战等级11（7,200XP）
魔法抗性 Magic Resistance。至高仙为抵抗法术和其他魔法效应而作的豁免检定具有优势。动作 Actions 多重攻击 Multiattack。至高仙发动两次妖精之剑攻击并使用一次恼人恶戏。妖精之剑 Fae Blade。近战武器攻击：命中+9，触及5尺，单一目标。命中：18（3d8+5）力场伤害。施法 Spellcasting。至高仙施展以下一道法术，无需任何材料成分并使用魅力作为施法属性（法术豁免 DC 18）：随意：交友术，次级幻象每项1/日：注目术，涅斯图魔法灵光恼人恶戏 Vexing Prank。至高仙使出一道魔法诡计，指定一个位于其60尺内的它可见的生物。目标必须成功通过一次 DC 18的感知豁免。豁免失败则受到21（6d6）心灵伤害并陷入恐慌状态直至至高仙的下一回合开始。豁免成功则只收到一半伤害。附赠动作 Bonus Actions 掠夺形踪 Loot Likeness。至高仙魔法性地变形为一个它可见的中型或小型生物的复制体。变形期间至高仙保留其游戏数据（除体型），并知晓被模仿生物的基本信息，诸如背景和个性（足以令其合理冒充该生物）。变形持续直到至高仙被降至0生命值，它再次使用此附赠动作或它用一个附赠动作变回真实形态。
