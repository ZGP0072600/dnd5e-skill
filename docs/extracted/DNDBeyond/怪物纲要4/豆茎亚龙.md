# 豆茎亚龙

**豆茎亚龙 Beanstalk Wurm**

> *巨人们的珍贵宠物逃离暴风域后，只见村庄尽毁、新生极茎拔地而起*。

亚龙 Wurms 是一种极为庞巨的生物，形似无肢无翼，缺乏吐息武器和敏锐智力的龙类。在艾卓世界上，许多亚龙已经适应了在直通天际的巨大豆茎间生活。豆茎亚龙生活在巨叶间，依靠光合作用及进食肉质维生。居于上的巨人常常将豆茎亚龙作为第一道防线，以抵御那些敢于攀登巨大豆茎——也被称为极茎——并入侵它们的高大宫殿之人。豆茎亚龙会盘绕在一段极茎上，紧贴豆茎，闭上双眼后便如同巨藤的一部分——直到美味的入侵者试图攀上其身。
