# 树妖

**树妖 Treefolk**

> *"听好了，你现在就是个小破柴，不过，我会让你多长几圈年轮的。"*
>
> > > > > —Borogrove, Tuinvale 
> > > > >  treefolk

古老且睿智的树妖隐居在艾卓森林仙境的深处。虽然绝大部分树妖的年龄与至高仙相媲美，但它们很少去干涉那些仙灵们的活动。相反，它们在自己的仙境树林之中担任着守卫与牧羊人的职位，并为那些穿过森林的旅人传授智慧与祝福。

**植被伪装 Plant Camouflage。** 树妖在森林地形中进行的敏捷（隐匿）检定具有优势。

**树跃 Tree Stride。** 在树妖的每个自己的回合中有一次，树妖可以消耗10尺移动力，隐入自身周围5尺范围内的一棵活着的树木中，再从60尺范围内的第二颗它能看见的活着的树木中跃出，它将出现在第二颗树周围5尺范围内的一处未被占据的空间内。作为入口和出口的树都必须至少有能容纳树妖体型的大小。

**动作 Actions 多重攻击 Multiattack。** 树妖发动两次粉碎藤鞭攻击，或两次龙葵箭攻击，或两种攻击各一次。

**粉碎藤鞭 Crushing Vine。** 近战武器攻击：命中+10，触及10尺，单一目标。命中：25（3d12+6）点钝击伤害。如果目标是生物，它还将陷入 
擒抱 Grappled 状态。擒抱期间，生物还将同时拥有 束缚 Restrained 
状态。树妖可以以此同时擒抱至多6个生物。

**龙葵箭 Nightshade Bolt。** 远程法术攻击：命中+9，射程60尺，单一目标。命中：33（6d10）点毒素伤害。 （译注：龙葵，又译作“夜影”）**施法 Spellcasting。** 树妖施展一道下列法术，不需要材料成分且用感知作为施法属性（法术豁免 DC 为17）：

随意：侦测毒素与疾病 detect 
poison and disease，德鲁伊伎俩 druidcraft

每项2/日：变巨术/缩小术 enlarge/reduce, 植物交谈 speak 
with plants

1/日：问道自然 commune with nature（以一个动作）

**附赠动作 Bonus Actions**

** 橡木恩赐 Oaken Boon。**树妖引导森林的睿智与力量，为它周围60尺范围内除它以外的另一生物施加祝福。受此祝福期间，生物可以使用树妖的 树跃 Tree 
Stride 特性，并在每个自己回合开始时获得5（2d4）点临时生命值。这一祝福将持续1分钟，如果树妖陷入 失能 Incapacitated 
状态，或再次使用这一附赠动作时，这一祝福将提前结束。
