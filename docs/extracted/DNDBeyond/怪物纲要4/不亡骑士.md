# 不亡骑士

**不亡骑士 Deathless Rider**

> *“要去幻野？当心效忠暗影女王的亡灵骑兵。”*
>
> > > > > ——边墙镇旅店主斯凯兰

艾卓诸王庭中有传说讲到，不亡骑士为一位名为暗影女王的魔契师女王服务，她在名为迪尼翠的城堡中统治。目前，没人知晓这传说是源自亲身经历还是对不可名状的可怖事件之解释。而无论如何，不亡骑士都是真实存在的。它们乃是曾为人类的亡灵骑士，乘着亡灵坐骑穿越最为偏远的幻野。有时，它们也被称为巫妖骑士，或是极恶骑士。他们有时被称为巫妖骑士或堕落骑士。不亡骑士与艾卓骑士截然相反，它们没有任何美德。它们数量稀少——而更多的正隐伏于无人知晓的深林之中。但在幻野中与不亡骑士相遇的凡人骑士很难有好下场。不亡骑士力量超凡，无惧伤口且战法无情。不亡骑士通常骑乘骷髅战马 warhorse skeleton，而有些巫妖骑士则骑乘如同梦魇 nightmares 的马类精魂。
不亡骑士 Deathless Rider 中型亡灵，通常守序邪恶 AC 18（链甲，盾牌）HP 84（13d8+26）速度20尺力量18（+4）敏捷10（+0）体质14（+2）智力6（-2）感知6（-2）魅力5（-3） 豁免：力量+7，体质+5 伤害抗性：暗蚀 伤害免疫：毒素 状态免疫：力竭，中毒感官：黑暗视觉60尺，被动察觉11语言：理解生前的语言但不会说挑战等级5（1,800XP）熟练加值+3
骑战大师 Mounted Adept。当骑士正在骑乘中且未陷入失能时，骑士对比它的坐骑体型更小的未骑乘生物发动近战攻击时所进行的攻击检定具有优势，且骑士和它的坐骑在进行敏捷豁免时具有优势，且骑士可以强迫针对它坐骑的攻击转而针对它自己。 不死坚韧 Undead 
Fortitude。当骑士因受非光耀非重击伤害而使其生命值减至0时，它必须进行一次 DC 为5＋所受伤害值的体质豁免，豁免成功则生命值降至1。 动作 Actions 多重攻击 Multiattack。骑士发动两次战斧攻击。它可以将其中一次攻击替换为极恶凝视。 战斧 Axe。近战武器攻击：命中+7，触及5尺，单一目标。命中：13（2d8+4）挥砍伤害。 极恶凝视 Fell Glare。骑士凝视一个位于其30尺内的生物，该生物必须成功通过一次 DC 13的感知豁免否则陷入恐慌状态一分钟。被恐慌生物可以在每次其自己的回合结束时再次进行该豁免（但若能看到骑士则具有劣势），豁免成功则终止自己身上的该效应。
