# 前言

自从我的一位背信弃义的前仆从认为将我收集的关于多元宇宙冲突的笔记公之于众是明智之举之后，我就被要求透露更多我来之不易，精心保守的秘密。我得说，这是胡说八道。共享的秘密不再是秘密。我的东西属于我自己。当我认为你值得听取我的见解时，你将是我第一个知晓它的人。然而，我确实从你身上看到了能打开我的一部分档案的一些智慧。平衡永远处于多种试图颠覆它的力量的危险角逐之中。八叶中的一些人指出，分享本册中收集的知识是让平衡远离其棕熊的第一步。我的同事中的其他人则告诉我，一个知晓更多的多元宇宙是更加安全的多元宇宙。毕竟，为什么我们这些闷闷不乐的少数人要承担保护多元宇宙不被其自身所影响的负担呢？我还被告知，这是为了一件“好”的事业，就其重要性和道德制高点而言，似乎任何目的都能高过我自己的目的。然而，当这项事业的目标和成就向我展现时，我仍然忍不住感到了一丝慈善的冲动。因此，我从我的资料中提供了一些信息。请注意，你在这里看到的知识我所知的极少一部分。它涉及了许多人认为不过是个笑话的生物，只有偶尔才会构成危险。也许这其中包含的真相会打消这些无知的念头。任何有力量的生物都会破坏平衡，因此即使不被畏惧，它们也值得谨慎尊重。继续读下去吧，然后你会了解到，也许对这些邪恶之物的嘲弄只是反映了那些嘲笑它们的人的恐惧和敬畏。也许，如果这本书反响不错，以后我还会放出更多内容。
