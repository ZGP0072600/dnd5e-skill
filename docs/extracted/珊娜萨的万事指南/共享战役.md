# 共享战役

**共享战役 Shared Campaigns**

协定一个 D&D 的固定“课程表”一向是难题，使战役保持活跃与生机更是一种挑战。如果本次游戏进程的 DM 或来的 PL 不齐，这一次进程就得被推迟。而如果反复出现这一情况，也许战役的连续性将被破坏。

（译注：实在没找到什么好词，session 意在我们所说的“今天晚上开团”的团字，但团字又能被用于“开团公告”中，指代一场战役 campaign，亦或者“我们这个团”中，指代一群跑团的人 group，故目前先保持“进程”这个翻译了，希望大家看得懂）

简而言之：在这个充满各种干扰和意外的世界里，想让一场长期战役一直进行下去是极度困难的。而共享战役这一概念因此应运而生。

在一场共享战役中，团队内可以有复数个人担任 DM 的职责。一场共享战役更像是一集一集的单元剧而非连续剧，每一场游戏进程都像是一次完整的冒险。

世界上最大的共享战役由 **D&D 冒险者联盟 **（D&D 
Adventurers League）运营，并由 ** 威世智 ** 
 
 
 （Wizards of the Coast）监管。不过你也可以在你的校园 D&D 社团、一家桌游店、一座图书馆、或别的什么有 D&D 玩家和 D&D 的 DM 们的地方创建属于你们的共享战役。

共享战役更像是一种框架，它允许玩家把一个角色从一名 DM 的游戏中带到另一个共享战役中的另一名 DM 的游戏中去。它创造了一种几乎无法被破坏的预订游戏进程。潜在同行玩家的名单也许会非常大，不论出什么事都能让你这场游戏有最少所需的人数来进行，而如果大家都在同一时间想去进行游戏，也将有多个 DM 能够带下这群人。

为了保证这一秩序的成功，一场共享战役需要一名总管——一个承担起组织和维护小组责任的人。如果你想了解更多关于如何开展共享活动的信息，并想知道
