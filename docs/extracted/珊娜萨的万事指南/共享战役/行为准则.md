# 行为准则

**行为准则 Code of Conduct**

核心规则书中一而再，再而三的强调，D&D 的游戏进程中最重要就是让所有玩家都能获得到乐趣。为了保持这一目标，在一场共同战役中制定一份行为准则将会是个很好的注意。毕竟大家都不一定会与自己熟悉的人一同在同桌上游玩共享战役，因此制定基本的行为准则是很有帮助和必要的。

进一步来说，共享战役中的每个人都有责任让其他人都能度过愉快的时光。如果有人因其他人的行为的感到被冒犯、被轻视、或被欺凌，那么大家一起愉快玩耍的初衷就落空了。

共享战役的行为准则可以参考其他组织或地点用的类似文件。此外，因为 DM 和 PL 的特殊关系，可能还需要添加一点特殊的准则。作为起步点，请参考以下摘自 
**冒险者联盟 Adventurers League** 
 
 
 
 
 行为准则的材料。

在一场游戏进程中，参与者应当：

- 听从 DM 的指引，并尽可能不要与 DM 或其他玩家争论规则。
- 让别的玩家也能说上话，并允许每个玩家都能从 DM 那边得到反馈。
- 避免太多与冒险无关的讨论。
- 阻止他人使用社交媒体欺凌、羞辱、或恐吓其他参与者。
- 让 DM 或战役的管理员了解破坏性或攻击性行为，以便采取适当措施。
