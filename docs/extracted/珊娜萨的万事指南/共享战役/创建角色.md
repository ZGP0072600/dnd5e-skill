# 创建角色

**创建角色 Character Creation**

一个共享战役的共定规则的“创建角色”章节应当明确规定那些种族选项与职业选项是玩家可以选择的而哪些不能，以及玩家如何生成属性值，以及玩家可以选什么阵营。

**《玩家手册》+1 Player's Handbook plus One** 

你应该好好思考一下玩家可以用哪些出版物以创建角色。** 冒险者联盟 Adventurers League** 
 
 
 
 
 
 
 
 
 中明确了玩家只能使用《玩家手册》与另一官方官方 D&D 资源（像是一本书或是一份 PDF）来创建一个角色。这种限制可以确保玩家不需要拥有大量书籍来创建一个角色，并且使 DM 更容易了解战役中所有角色的运作方式。由于在共享战役中，DM 每周不会如普通战役一般去处理相同的角色，他们每天都会接触到各种各样的不同角色，这很容易导致 DM 无法自由处理混乱不堪的可能选项并追踪每一种可能的交互。因此，我们强烈建议任何共享战役都采用这一规则。

**属性值 Ability Scores** 

 
 
 
 关于生成属性值，我们推荐仅允许玩家选取：

*(a)* 
 
 
 最标准的属性值组——即 15、14、13、12、10、8——作为角色的属性值；

*(b)* 使用《玩家手册》第一章的“**变体：自定义属性值 Variant: Customizing Ability Scores** 
 
 
 ”规则来生成角色的属性值。

**起始装备 Starting Equipment** 

 
 
 
为简化并提高效率，规定所有角色必须选择职业与背景特定的那些初始装备会是个非常不错的选择。
