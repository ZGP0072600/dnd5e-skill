# 策士

**策士 Mastermind**

哦，拜托！如果要说这附近哪里有策士，那只能是在说我。而你大概是扯士或撤士之类...不，你确实是！

策士专注于对人们施加影响和知晓他们的秘密。许多间谍，朝臣和阴谋家会追随这个范型，在密谋中渡过他们全部的生命。言语像你的匕首和毒药一般作为你的武器，通过它们你得到你最喜欢的宝藏--人们的要害所在以及他们言不由衷的支持。

**瞒天之魂 Soul of Deceit**

从17级起，除非经过你的同意，否则你的思想无法被心灵感应或者其他能力阅读。你还能在这种情况下，使用魅力（欺瞒）检定对抗思想阅读者的感知（洞悉）检定，对抗成功时你可以让其阅读到虚假的思想。

此外，不管你在说什么，你总是可以选择让魔法判定你在说真话并显现出相应的迹象。你也不会因为魔法效应的影响而被强迫说出真话。
