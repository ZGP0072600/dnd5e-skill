# 风流剑客

**风流剑客 Swashbuckler**

（译注：又称剑客、游荡剑客、侠盗、浪客等。在模组、怪物、职业、设定、魔法物品等地方使用的名称都不同，它们指代也不一定是同一个东西。故这里只是采用了风流剑客作为标题，剩下这些同样是正确的翻译。）

风流剑客是什么玩意？你们的剑法风流倜傥？还是说你们剑指风流人士？嘿，你可别说这双关笑话没意思！

你全身心投入剑术的训练，一种兼顾了速度、优雅和魅力的剑术。对比其他被重甲包裹成粗蛮罐头的战士，你战斗的方式看起来更象在表演。扫荡者、决斗家和海盗是这类的典型。

风流剑客擅长一对一的决斗，也精通于无所顾虑地双持武器攻击对手，并在那以后飞快远离敌人。
