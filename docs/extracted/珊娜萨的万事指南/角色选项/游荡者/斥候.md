# 斥候

**斥候 Scout**

你擅长在远离城市和街道的地方潜行和生存，这让你在探险中能离开你的同伴带头侦查。这一泛型的斥候许多都以荒野为家，与野蛮人和游侠同行，也有许多斥候充当着战争地带的眼睛和耳朵。伏击兵，间谍，赏金猎人--这些只是斥候在世界范围内所扮演的少数角色。
