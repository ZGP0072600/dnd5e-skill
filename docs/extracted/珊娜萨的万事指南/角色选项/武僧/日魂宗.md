# 日魂宗

**日魂宗 Way of the Sun soul**

所以你们有这个叫“魂”的玩意，但你们看不见也摸不着他。然后如果你们死了这个看不见的玩意就会自动离开你们的身体，然后去到一个你们永远见不到的地方.....哦你们几个是不是觉得我疯了？

日魂宗的武僧学习如何引导出自己的生命能量并将它们转换为炽热的光芒。他们相信在每一个活着的生物的灵魂中都有这样不熄的光芒存在，并且教导他们以一种特殊的冥想方式去解锁。

**热能袭 Searing Arc Strike**

从6级起，你获得引导自己的气，将其转化为灼热能量波的技艺。

当你在你的回合中执行了攻击动作，你可以立即消耗2点气，以附赠动作施放法术 
燃烧之手 burning hands 。

你可以消耗额外的气来施展更高环版本的法术 燃烧之手 burning hands 
，每消耗一点额外的气都能使法术的环阶上升1环。你最多可以在这个法术上花费的气（2+额外的点数）等同你武僧等级的一半。

**焰阳爆 Searing Sunburst**

从11级起，你得到创造出一个光球，以此引发一场毁灭性爆炸的能力。

以一个动作，你可以魔法性地创造一个法球，并将其向150尺内你选择的任意一点投掷出去。法球在那里瞬间扩张成一个光芒四射的区域。以目标点为中心，半径20尺范围内的所有生物必须立即通过一次体质豁免，豁免失败将受到2d6的光耀伤害。若生物处于完全掩护下，且那掩体不透光，那么它不需要为此进行豁免。

你可以花费气以提高这枚光球的威力。每消耗一点气都能使这一球体的伤害上升2d6，你最多为此消耗3点气。

**太阳盾 Sun Shield**

从17级起，你的身躯被永久包覆在魔法性的耀眼灵光之中。

你发散出30尺的明亮光照和那之外30尺的微光光照。你可以以一个附赠动作将这一光照熄灭或将其恢复。

光照未被熄灭的情况下，如果有生物对你发动近战攻击并击中了你，你可以用你的反应对那个生物造成 
5+你的感知调整值 点的光耀伤害。
