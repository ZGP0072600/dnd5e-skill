# 迷惑学院

**魅心学院 College of Glamour**

幻术？太古雅了。在我撕碎你之前，请帮我做一只活灵活现的大金鱼幻象吧——就和我这么大！呃...... 好像有点太大了。那，再见！

魅心学院向那些在充满生机却处处致命的妖精荒野里学习技艺的吟游诗人们敞开了大门。在半羊人，雅灵、与其他妖精指导，这些吟游诗人学会了使用魔法来取悦并吸引他人。

此学院的吟游诗人通常会带来敬畏与恐惧。他们的表演近乎传奇。此学院的吟游诗人风情万种，只是一场表演便足以让绑架者乖乖离开，让愤怒的巨龙归于平静。同样的魔法让他们在操弄他人心智的同时，也能掌控野兽。恶劣的魅心学院诗人可以混入一个社群几周，用魔法让主人变成奴隶；英勇的魅心学院诗人反而利用这种力量打击压迫者，为被压迫的人们带来快乐。

 

**魅心学院特性**

| 吟游诗人等级 | 特性 |
|---|---|
| 3rd | 激励光环，醉人表演 |
| 6th | 统御光环 |
| 14th | 绝对统御 |

激励光环 Mantle of Inspiration

**醉人表演 Enthralling Performance**

从3级起，你能将诱惑力和妖精的魔法充入你的表演。

若你进行了至少1分钟的表演，你就可以借由歌唱、吟诗、或舞蹈的方式来让你的观众们感受到奇妙的力量。在这次表演的结束时，选择你周围60尺范围内一定数量的、完整观看并聆听了你的表演的类人生物，可以选择的数量至多等同于你的魅力调整值个（至少1个）。每个目标必须通过一次对抗你的法术豁免 DC 的感知豁免，若豁免失败则其将被你魅惑。当被你以此法魅惑时，目标视你为偶像，他会热情的和任何和它交谈的人谈论你，反对任何对抗你的人，它会避免暴力，除非他已经站在你这边加入了战斗。每个目标身上的效应都将持续1小时。当受影响生物受到任何伤害、你攻击了它、或它目睹你对任何它的盟友进行攻击或造成伤害时会提前结束。

若一个目标对抗此效应的豁免成功，该目标不会察觉到你曾试图魅惑他的行为。

一旦你使用了此特性，直到你完成一次短休或长休前你都不能再次使用它。

 

** 统御光环 Mantle of Majesty**

从6级起，你得到了用妖精魔法覆盖自己的能力，令他人化为服从你指令的傀儡。以一个附赠动作，你施展法术 
命令术 command 
，无需消耗法术位。且同时你获得无上的美貌，持续1分钟，如果你的专注结束（如同你专注于一个法术），它也会提前结束。在这段时间内，你在每个你的回合中都能再用一个附赠动作施展法术 
命令术 command ，且同样无需消耗法术位。

以此施展的 *命令术 Command* 中，任何被你 **魅惑 Charmed** 
的生物将自动豁免失败。

一旦你使用了此特性，直到你完成一次长休前你都不能再次使用它。

 

** 绝对统御 Unbreakable Majesty**

从14级起，你的外貌变得超凡脱俗，这让你看上去更加漂亮与骇人。

此外，以一个附赠动作，你展现出魔法性的完美仪度，持续1分钟，如果你陷入 
失能 Incapacitated，它也将提前结束。持续时间内，任何生物在每回合中首次试图对你进行攻击时都必须通过一次对抗你的法术豁免 DC 的魅力豁免。豁免失败时，该生物在这个回合内将无法攻击你，且必须要为本次攻击选定一个新的目标，或放弃攻击。豁免检定成功时，该生物能够正常攻击你，但它在下个你的回合中，对抗你的法术所做的任何豁免检定都将具有劣势。

一旦你使用了此特性，直到你完成一次短休或长休前你都不能再次使用它。
