# 狂热者道途

**狂热者道途 Path of the Zealot**

一些神灵鼓励的追随者们投身于残酷的战斗狂怒中。这些野蛮人是狂热的战士，他们引导自己的愤怒转化为强大的神赐力量。

在 D&D 世界中有许多神都会鼓励自己的追随者寻求狂热者道途。最著名的是费伦的 
坦帕斯 Tempus ，以及灰鹰的 海克斯托 Hextor 和 厄瑞斯努 Erythnul 
。一般来说，那些培养狂热者的神往往是偏向于战斗、破坏和暴力等神职。当然，它们不一定全是邪恶的，还是有少数例外是善良的。

**狂热者道途特性 **

| 野蛮人等级 | 特性 |
|---|---|
| 3rd | 神性之怒，神之勇者 |
| 6th | 专心炽志 |
| 10th | 狂热威仪 |
| 14th | 怒不畏死 |

**神性之怒 Divine Fury**

从你在3级选择此道途开始，你可以引导神性怒火，将其注入你的武器打击中。狂暴期间，你在每个自己回合中用武器攻击命中的第一个生物会受到等同于1d6+你野蛮人等级的一半 
的额外伤害。此伤害的类型为暗蚀或光耀；你在你得到此特性时选择它的类型。

 

** 神之勇者 Warrior of the Gods**

在3级时，你的灵魂被无尽的战斗所烙印。若一个法术只具有让你起死回生（而非令你变为亡灵）的效果，例如法术 死者复活 raise dead 
，则施法者在对你施展这一法术时不需要材料成分。

 

** 专心炽志 Fanatical Focus**

在6级时，驱使你愤怒的神力也能庇护你免遭干扰。狂暴期间，若你豁免检定失败，你可以重新投掷该豁免检定，但你必须接受重骰后的结果。你每次狂暴中只能使用此能力一次。

 

** 狂热威仪 Zealous Presence**

在10级时，你学会了引导神圣之力，向他人传授你的狂热。以一个附赠动作，你愤怒的怒吼并释放出一道蕴含神圣能量的战吼。选择你周围60尺范围内至多10个能听到你战吼声的其他生物，直到你的下个回合开始前，他们进行的所有攻击检定与豁免检定均具有优势。

一旦你使用了此特性，直到你完成了一次长休为止你都不能再次使用它。

 

** 怒不畏死 Rage beyond Death**

从14级开始，驱使你狂暴的神力允许你忽视致命的打击。

狂暴期间，具有0点生命值并不会使你陷入 
昏迷 Unconscious 
。不过你仍然需要进行死亡豁免，你在0生命值下受到伤害时依旧计算豁免失败次数。然而，若你因为未通过死亡豁免检定而即将死去时，只要你仍处于狂暴状态，你将拒绝死亡。而如果你的狂暴结束后你仍保持0点生命值，那么你将死亡。
