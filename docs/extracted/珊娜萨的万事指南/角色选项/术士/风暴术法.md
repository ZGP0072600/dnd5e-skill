# 风暴术法

**风暴术法 Storm Sorcery**

天气怎么样关你我什么事？反正这破天空哭就哭，叫就叫，离我这么远与我无关。

你内在的魔法源于气元素之力量。这些力量几乎全都源自于那场惊世骇俗的 大暴雨 Great 
Rain，而那狂风因其威力之强而仍被市井传颂；不过也许你的血脉是受到强大气元素生物的影响而成的，例如风巨灵。不论情况如何，风暴之魔力早已浸透你的灵魂。

风暴术士是极具价值的海员。其魔法足以驾驭其周边的风与气象。且他们的能力亦可用于对抗鲨华鱼人、海盗、以及其他的水上威胁。
