# 神圣之魂

**神圣之魂 Divine Soul**

为啥这些个天界玩意都长着鸟翅膀，而那些个炼狱玩意都长着蝙蝠翅膀？这看起来好不公平。这得有一个“蝙蝠天使”才对啊！（珊娜萨手绘了一个长着蝙蝠翅膀顶着光环的黑皮蛋）

有时，支持着术士的魔力火花来自他灵魂中闪耀着的一点神力。拥有这样一种神迹的灵魂，表明你内在的魔法可能来自于一个遥远而强大的家族与某种神圣的存在之间有所联系。也许你的神是一个天使，变成了一个凡人，奉神的名义来战斗。或者你的出生可能与一个古老的预言相符，昭示着你是神的仆人，或者是被选为承载神圣魔法的容器。

一个具有天生法力的神圣之魂术士，它们常常被一些教会视为威胁。作为一个指挥着神圣的力量的局外人，一个神圣之魂术士可以通过声称其与神的直接联系，来随意的破坏教会的秩序。

在某些文化中，只有那些具备神圣之魂的力量之人才能统御宗教势力。在这些地方，教会的权利永远由几条血脉支配着，代代相传。
