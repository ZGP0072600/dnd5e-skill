# 战争魔法

**战争魔法 War Magic**

战争魔法？它会让你变成一名战法师么？我听说这群人啊，他们都穿的一身紫。然后我看你穿的不紫 ... 
哦，那你失格了！换个职业去！

许多奥术学院专门培养用于战争的法师。战争之传承精妙地融合了塑能与防护的机理，而非专精于独一学派。它既授予施法者增进法术威力的技巧，同时也为法师提供了改良防御能力的手法。这个奥术传承的追随者们被称为战法师。他们将他们的魔法视为武器和盔甲，比任何一块钢铁都更为坚韧的材料。战法师在战斗中行动迅速，用他们的法术夺取对战局的战术控制。他们的法术迅猛非常，而他们的防守技巧挫败了对手的反击势头。战法师也善于导引其他施法者的魔法能量反过来攻击他们。

于恢弘的大战中，战法师时常与塑能师、防护师和其他类型的法师相共事。塑能师最常嘲笑战法师分心于攻守之间，而一位战法师的经典回应是：“死人可丢不了火球。”

**魔力潮涌 Power Surge**

从6级起，你可以将魔法能量存于体内并于之后增幅你危险异常的法术。以这种方式存储的能量。便被称为一股魔力潮涌。

你可以储存至多等同于你智力调整值（至少1）股的魔力潮涌。当你完成一次长休时，你的魔力潮涌数量将变回一股。当你成功用 
解除魔法 dispel magic 或 法术反制 counterspell 
终止或打断一个法术时，你可以从被你挫败的法术中窃得魔力，以此获得一股魔力潮涌。而当你在没有任何一股魔力潮涌的情况下完成一次短休时，你也能获得一股魔力潮涌。

每回合一次，当你用一个法师法术对一个生物或物件造成伤害时，你可以消耗一股魔力潮涌来对那个目标造成等同于你的法师等级半数的额外力场伤害。

 

** 耐久魔法 Durable Magic**

从10级起，你引导的魔法也能帮你阻绝伤害。在你专注于一个法术期间，你的 AC 与你所进行的任何豁免检定具有+2加值。

 

** 偏斜罩幕 Deflecting Shroud**

在14级时，你的 奥术偏斜 Arcane Deflection 
被灌注了致命的魔法。当你使用你的奥术偏斜特性时，你可以让魔法能量从你身上涌出。选择至多三个你周围60尺内的可见目标，它们将受到你的法师等级半数的力场伤害。
