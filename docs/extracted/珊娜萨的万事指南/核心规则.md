# 核心规则

**核心规则 The Core Rules**

斯尔嘉，你说啥？不，还不行，我的鱼儿朋友。他们才刚刚到呢，咱俩先聊完吧。我觉得开餐前先聊一会儿能帮助消化。

本书基于三本核心规则书的规则。游戏会频繁用到《玩家手册》第7~10章的规则，包括：“属性值应用”“冒险”“战斗”“施法”，以及其附录 A 中包括的隐形和倒地等重要状态。你不需要死记这些规则，不过最好能在需要它们时知道要到哪里去查询。

如果你是 DM，你还要知道如何在《城主指南》中查找相应的内容，特别是一些魔法物品的运作方式（见该书第七章）。关于如何阅读怪物资料板的指引可以参见《怪物图鉴》的简介
