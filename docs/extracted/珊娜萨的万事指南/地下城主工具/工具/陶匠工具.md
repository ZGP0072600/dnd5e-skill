# 陶匠工具

**陶匠工具 Potter's Tools**

陶匠工具用于制造各种各样的陶器，最典型的是陶罐和类似的器皿。

**组件 Components。** 陶匠工具包括穿刺针、骨架刀、刮刀、小刀，以及卡尺。

**历史 History。** 你的专业知识有助于你鉴定陶器，包括生产时间，以及它们可能的产地或产地文化。

**调查 Investigation、察觉 Perception。** 你在检查陶器时具有额外的洞察力，通过一些小细节来发现他人忽略的线索。

**复原 Reconstruction。** 通过检查陶器碎片，你可以推断出该物件原本、完整的形状及其可能的用途。
