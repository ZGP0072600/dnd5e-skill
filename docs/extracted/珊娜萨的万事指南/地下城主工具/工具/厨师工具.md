# 厨师工具

**厨师工具 Cook's Utensils**

冒险可是个辛苦活儿。如果能和一位厨师一起旅行，你的伙食可比整天啃饼干嚼果脯好多了。

**组件 Components。** 厨师工具包括一个金属锅、餐刀、餐叉、一柄搅拌勺，和一柄长柄勺。

**历史 History。** 你对厨艺的了解允许你品味从餐饮习惯中透露出来的社会模式。

**医药 Medicine。** 进行治疗时，你可以改善药物的口感，使其不那么难以下咽。

**生存 Survival。** 寻找食物时，你可以把他人认为不能吃的玩意儿处理成可食用的东西。

**准备饭菜 Prepare Meals。** 
作为短休的一部分，你可以准备一顿美味的饭菜，帮助同伴恢复体力。在短休时，你和至多五个你所选择的生物每消耗一枚生命骰就可以多恢复1点生命值，只要你有机会使用你的厨师工具并且食材充足。
