# 玻璃匠工具

**玻璃匠工具 Glassblower's Tools**

具有玻璃匠工具熟练项的人不仅知道如何将玻璃吹制成形，还懂得生产玻璃制品的专业知识。

**组件 Components。** 这些工具包括一支吹管、一个小搓板、一些模具，和几把工钳。此外你还需要一个热源才能吹制玻璃。

**奥秘 Arcana、历史 History。** 你的玻璃制造技术有助于检查玻璃制品，例如在藏宝库中发现的药剂瓶或玻璃物品。例如，你可以研究玻璃药瓶受到内容物的影响，以判断药水的效果。（药水可能留有残液、使玻璃变形或造成色斑。）

**调查 Investigation。** 当你调查一个拥有碎玻璃或玻璃制品等线索的区域时，你的知识能够提供帮助。

**发现弱点 Identify Weakness。** 通过1分钟的研究，你能找出玻璃制品的薄弱点。击打该物件的薄弱点能够造成双倍的伤害。
