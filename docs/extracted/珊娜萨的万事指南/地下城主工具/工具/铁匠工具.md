# 铁匠工具

**铁匠工具 Smith's Tools**

铁匠工具允许你加工金属，加热它来塑性变形、修复损坏，或者把钢锭加工成有用的物品。

**组件 Components。** 铁匠工具包括锤子、钳子、木炭、破布和磨刀石。

**奥秘 Arcana、历史 History。** 当你检查金属物体，比如武器时，你的专业知识会给你额外的洞察力。

**调查 Investigation。** 在调查护甲、武器或其他金属制品时，你可以发现他人可能会忽略的线索并作出推论。

**修复 Repair。** 当你能够使用工具并且具备一处足以使金属热塑变形的明火时，你每花费1个小时的工作时间，就能为一件损坏的金属制品恢复10点生命值。
