# 文书伪造工具

**文书伪造工具 Forgery Kit**

文书伪造工具通常用于伪造一份信用文书和模仿某人的印章或签名。

**组件 Components。** 文书伪造工具包括几种不同类型的墨水、各种羊皮纸和普通纸、几支鹅毛笔、印章和封蜡、金银叶饰，以及伪造蜡封的小型雕刻工具。

**奥秘 Arcana。** 文书伪造工具与奥秘技能搭配可以鉴定一件魔法物品的真伪。

**欺瞒 Deception。** 一份精心伪造的文书，比如贵族血统证明或管制区通行证，可以让你的谎言更有可信度。

**历史 History。** 文书伪造工具与你的历史知识相结合，可以让你更轻易地伪造历史文献或鉴定文献的真伪。

**调查 Investigation。** 当你检查一个物件时，文书伪造工具熟练项有助于你判断该物件是如何制成的，以及它的真伪。

**其他工具 Other Tools。** 
掌握其他工具的用法可以提高你所制赝品的可信度。例如，同时具有文书伪造工具和制图工具熟练项可以制造一份假地图。

**快速伪造 Quick Fake。** 
你在一次短休中可以伪造篇幅不超过1页的假文书。而一次长休中，你至多可以制造4页的文书。你进行一次附带文书伪造工具熟练加值的智力检定，以决定他人识破赝品的智力（调查）检定 DC 值。
