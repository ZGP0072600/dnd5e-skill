# 乐器

**乐器 Musical Instruments**

具有某种乐器熟练项，表明你熟悉演奏它的技巧。你也知道一些该乐器的经典曲目。

**历史 History。** 你的专长帮助你回忆与乐器有关的知识。

**表演 Performance。** 当你把乐器融入表演时，你的表演会更有张力。

**作曲 Compose a Tune。** 作为长休的一部分，你可以为你的乐器谱写一首新曲子和歌词。你可能会借此给一位贵族留下深刻印象，或是在市井中传唱某些故事。
