# 制毒工具

**制毒工具 Poisoner's Kit**

对于盗贼、刺客还有其他喜欢干些见不得人的勾当的人来说，制毒工具可谓是倍受青睐。它允许你利用各种材料制取毒素并使用它们。你的毒药学知识也有助于解毒。

**组件 Components。** 制毒工具包括玻璃瓶、研钵和杵、一些化学制剂，和一根玻璃搅拌棒。

**历史 History。** 你的毒药培训有助于你回想起有关于毒药的事件。

**调查 Investigation、察觉 Perception。** 你的毒药学知识教会了你如何小心地处理这些物质，当你检查有毒物品或搜索与毒药有关的事件线索时具有优势。

**医药 Medicine。** 当你医治中毒的患者时，你的知识会让你知道如何为病人提供最好的处理。

**自然 Nature、生存 Survival。** 你能分辨有毒的动植物。

**操作毒药 Handle Poison。** 你的熟练项允许你安全地操作或使用毒药，而不会令你自身暴露在毒素环境中。
