# 草药工具

**草药工具 Herbalism Kit**

草药工具熟练项允许你辨认植物并安全地提取有效成分。

**组件 Components。** 草药工具包括装药材的袋子、采集草药的剪钳和皮手套、研钵和药杵，以及几个玻璃瓶。

**奥秘 Arcana。** 你对药材的性味归经和功能主治的了解，使你在对植物进行魔法研究或试图鉴定药剂时具有额外的洞察力。

**调查 Investigation。** 当你调查一个长有植物的区域时，草药工具熟练项能帮助你发现那些其他人可能会忽视的细节或线索。

**医药 Medicine。** 你对草药医学的掌握有助于提高你使用药用植物治疗疾病和创伤的水平。

**自然 Nature、生存 Survival。** 当你在野外旅行时，你的草药技能有助于辨认植物和找到一般人不知道的食物来源。

**辨认植物 Identify Plants。** 仅仅通过外观和气味，你就能快速辨认出大部分植物。
