# 珠宝匠工具

**珠宝匠工具 Jeweler's Tools**

珠宝匠工具的培训包括加工珠宝的基本技术，以及鉴定宝石的专业知识。

**组件 Components。** 珠宝匠工具包括一把小锯、小锤、锉刀、钳子，和镊子。

**奥秘 Arcana。** 珠宝匠工具熟练项让你能够了解各种宝石的魔法用途。这一能力在你对宝石或镶嵌宝石的物品进行奥秘检定时很有用处。

**调查 Investigation。** 当你检查珠宝类物件时，珠宝匠工具熟练项有助于你发现可能存在的线索。

**鉴定宝石 Identify Gems。** 你能很快鉴定出宝石的种类和价值。
