# 修补工具

**修补工具 Tinker's Tools**

修补工具用于修复各种普通物件。虽然你不能用修补工具制造很多东西，但你可以缝补衣服、磨砺锋刃，或是修补好破掉的链甲。

**组件 Components。** 修补工具包括各种手工用品、针线、磨刀石、一些零碎的布料和皮料，以及一小壶胶。

**历史 History。** 你可以确定物件的年龄和来历，即使你只剩下几块原件的碎片。

**调查 Investigation。** 当你检查一个损坏的物件时，你能推测出损坏的原因及时间。

**修复 Repair。** 你每花费1个小时的工作时间，就能为一件损坏的物件恢复10点生命值。你必须具有与之对应的修补材料。对于金属制品，你还需要具备一处足以使金属热塑变形的明火。
