# 石匠工具

**石匠工具 Mason's Tools**

你可以使用石匠工具制造石质建筑——当然，用砖头砌成的墙和房屋也属于石质建筑。

**组件 Components。** 石匠工具包括泥刀、锤子、凿子、刷子、和角尺。

**历史 History。** 你的专业知识有助于你确定一座石质建筑的建造日期和用途，以及可能是谁建造了它。

**调查 Investigation。** 当你检查石质建筑中的区域时，你具有额外的洞察力。

**察觉 Perception。** 你可以在石墙或地板上发现不规则的地方，从而更容易找到陷阱门和秘密通道。

**拆迁 Demolition。** 你对砖石结构的知识可以让你发现墙壁的薄弱点。你的武器攻击会对这种建筑造成双倍伤害。
