# 炼金工具

**炼金工具 Alchemist's Supplies**

炼金工具可以用来制作有用的化合物，如强酸或炽火胶。

**组件 Components。** 炼金工具包括两只玻璃烧杯、一个铁架台、一根玻璃搅拌棒、一个小研钵和药杵，以及一袋常用炼金术材料，包括盐、铁粉、纯净水。

**奥秘 Arcana。** 炼金工具熟练项可以让你在进行鉴定药水或类似物质的奥秘检定中获得更多信息。

**调查 Investigation。** 当你调查一个区域的线索时，如果该区域曾使用过任何化合物或其他物质，炼金工具熟练项将使你对此具有额外的洞察力。

**道具制作 Alchemical Crafting。** 
你可以使用这套工具熟练制作炼金道具。角色可以花钱收集原材料，每磅材料花费50金币。DM 可以允许角色使用指定技能进行检定时具有优势。

作为长休的一部分，你可以使用炼金工具制作一份剂量的强酸、炽火胶、抗毒剂、油、香水或者香皂。从你所携带的原材料总价值中减去所制作物品价值的一半。
