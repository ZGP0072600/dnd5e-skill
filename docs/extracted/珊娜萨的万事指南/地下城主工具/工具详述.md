# 工具详述

**工具熟练项 Tool Proficiencies**

工具熟练项是用来突出角色背景和专业水平的有效途径。然而在游戏进行中，工具的使用有时与技能的使用重叠，而且应该在何种情况下如何使用它们也不够明确。本节提供了各种工具可以在游戏中使用的方法。

**工具与技能一起使用 Tools and Skills Together**

工具相比起技能而言，它更加具有专业性质。历史技能的熟练可以应用在过去的所有大事件之上，但文书伪造工具却只能用来制作伪造文件与少数别的物品。因此，有机会在这两种熟练项之间选择其一的角色为什么要去选择工具熟练项而不是技能熟练项呢？

**检定优势 Advantage**

如果一个工具的使用和一个技能的运用都适用于同一个检定，而一个角色同时具有该工具和该技能的熟练，那么可以考虑允许这个角色在检定时具有优势。这个简单的收益对于鼓励玩家获取工具熟练大有帮助。在下文的工具说明中，这种益处通常被描述为额外的洞察力（或类似的东西），这意味着增加了检定成功的机会。

**额外奖励 Added Benefit**

此外，考虑给有相关技能和相关工具熟练的人一个检定成功后的额外奖励。这种奖励可以是提供更详细的信息，或者也可以模拟不同类型的检定成功的效果。例如，一个具有石匠工具熟练的角色成功地通过感知（察觉）检定在石墙中找到一扇密门。这个角色不仅注意到密门的存在，而且你可以决定该角色的工具熟练项使其在一个智力（调查）检定中自动成功，知道这扇密门该如何打开。

工具详述 Tool 
Descriptions 以下各节详细介绍了《玩家手册》中所提供的工具，以及如何在战役中使用这些工具的建议。

**组件 Components。** 以下每项工具说明都会在开头说明这套用品或工具都由什么部件组成。具有工具熟练项的角色知道所有部件的用法。

**技能 Skills。** 当一个角色具有工具和技能熟练项时，每一种工具都可能在与某些技能配合使用时提供检定的优势。

作为 DM，你可以允许一个角色在使用指定技能进行检定时具有优势。以技能名称开头的段落描述了这些可选项。但要注意的是，这些好处只适用于具有对应工具熟练项的人，而不是仅仅携带了该工具的人。

联系到技能时，工具熟练项的意思其实是有些抽象的；具体来讲，它假定了具有相应工具熟练项的角色同时也了解相关行业或专业方面的事情，尽管这些事情不一定跟使用本工具相关。

此外，你可以考虑给角色进行技能检定时一些额外信息或好处。本文也给出了满足相关条件的例子和建议。

**特殊用途 Special Use。** 顾名思义，将具有熟练项的工具用于特殊用途往往会带来特殊的好处。

**难度等级示例 Sample DCs。** 每一节末尾的表格列出了该工具所适用的一些用途，以及必要属性检定的推荐 DC 值。
