# 培根绿矛卷

**培根绿矛卷 Greenspear Bundles in Bacon**

绿矛，也被人类称为芦笋或 
"麻雀草"，是一种在许多精灵的饮食中已经上升到主食地位的多年生花卉植物。当精灵们生吃、调味、烘烤或撒上香草清蒸绿矛时，他们的半精灵兄弟--从某些烹饪禁忌中解放自己--提出了另外的备菜技巧。为了尊重他们半人部分的味蕾，半精灵在其中加入盐渍腌制的猪肉，这种做法可能有些不敬，但大大提升了绿矛的风味。至于培根，当然是切得越厚越好！

> **4人份用料 **
>
> 2磅（约32根）中等粗细的芦笋条，去根
>
> 1汤匙特级初榨橄榄油
>
> 粗盐和现磨黑胡椒粉
>
> 8片培根
>
> 1.5汤匙切碎的新鲜细香葱作为装饰

将烤箱预热到425华氏度，将烤架放在烤箱的中间。

使用一个大碗将芦笋用橄榄油拌匀。在芦笋上稍微撒些盐和胡椒粉，并翻拌均匀。

将一片培根放在台面上，斜放四根芦笋在上面，然后用培根缠绕在芦笋上，尽可能多地覆盖芦笋，并确保培根两端重叠；轻轻按压以粘合两端。四根芦笋为一组，重复制作过程，完成剩余的七组。

使用大的有框烤盘，将绑好的培根卷接缝处朝下排列，周围留出大约1英寸的空间。烘烤至培根和芦笋熟透并呈褐色，约15分钟。

将烤好的培根卷转移到盘子里，撒上香葱碎上桌。

厨师笔记

在选择芦笋时，使用底部约3/4英寸厚的芦笋（大约和荧光笔一样厚）；如果太细（像铅笔一样）会烤过头。如果你喜欢更脆的培根，用标准培根代替厚切培根。
