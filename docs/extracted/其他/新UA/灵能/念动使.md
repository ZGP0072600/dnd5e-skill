# 念动使 Psykinetic

塑灵能之力，予创造与毁灭

念动使掌控着他们那可塑的灵能力量。他们将念动力扭曲成坚不可摧的屏障，挥击时则宛如攻城槌般强劲。

**3级：念动使法术 Psykinetic Spells**

当你到达念动使法术表中特定的灵能使 Psion 等级时，你就始终准备着表中对应的法术。

** 念动使法术 Psykinetic Spells**

| 灵能使等级 | 法术 |
|---|---|
| 3 | 护盾术 Shield，雷鸣波 Thunderwave 匕首之云 Cloud of Daggers，浮空术 Levitate |
| 5 | 缓慢术 Slow，念力碾压 Telekinetic Crush* |
| 7 | 欧提路克弹力法球 Otiluke's Resilient Sphere，塑石术 Stone Shape |
| 9 | 心灵遥控 Telekinesis，力场墙 Wall of Force |

**3级：念力技巧 Telekinetic Techniques**

 当你使用你的念力驱使时，你可以对目标施加以下效果之一。

- **提速 Boost。** 目标的速度增加10尺，直至你的下个回合开始。
- **迷惘 Disorient。** 目标无法进行借机攻击，直至其下个回合开始。
- **念力箭 Telekinetic Bolt。** 若目标豁免失败，其受到等同于灵能骰骰值的力场伤害。

**6级：强效攻击模式 Empowered Attack Mode**

 在你的攻击模式激活期间，你具有60尺飞行速度并可以悬浮，且当你施展一个灵能法术时，你可以将你的智力调整值加入到该法术的其中一次伤害掷骰中。

**6级：反弹力场 Rebounding Field**

当你因被一次攻击检定命中时而施展护盾术 Shield 
 ，并使该次攻击失手时，你可以消耗一个灵能骰将这股力返还给攻击者。攻击者进行一次敏捷豁免。掷两次灵能骰。若豁免失败，攻击者受到等于总值的力场伤害，而你获得等于总值的临时生命值，豁免成功则攻击者仅受半伤。

10级：强化念力碾压 Enhanced 
Telekinetic Crush

当你施展法术念力碾压 Telekinetic 
Crush 时，你可以消耗一个灵能骰来增强该法术，使其目标无论是否通过了该法术的豁免，其速度都会减半，直至你的下回合开始。

**14级：升阶心灵遥控 Heightened Telekinesis**

当你施展法术心灵遥控 Telekinesis 时，你可以消耗四个灵能骰来增强该法术，使其无需专注。若你如此做，该法术的持续时间变为1分钟，并且你能够以超巨型生物和物件为目标。
