# 灵能 The Psion

该游戏测试文档是破解奥秘系列文章的一部分，旨在为未来产品开发提供设计原型。本材料使用了《玩家手册2024》中的规则。

# 这里有什么？ What’s Inside?

**灵能使 Psion。** 自从奇械师以来的第一个全新职业，灵能使通过自身意志的力量编织魔法。

**子职业 Subclasses。** 四个为灵能使提供独特身份的子职业：**蜕变使 Metamorph**，** 裂空使 Psi Warper**，** 念动使 Psykinetic**，** 传心使 Telepath** 。

** 法术 Spells。**一系列新老法术，为灵能使提供了更多的法术选项。

** 狂野天赋 Wild Talents。** 本 UA 包含一系列专长用于提供角色使用灵能的途径。那些代表与灵能相联系的背景将在未来的书籍中出现，与此同时，贵族 Noble 和智者 Sage 背景也可以用狂野天赋专长来替代起源专长。同时，任何角色都可以在职业允许选择专长时选择以下专长之一。
