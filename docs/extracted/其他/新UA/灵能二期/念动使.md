# 念动使 Psykinetic

塑灵能之力，予创造与毁灭

念动使掌控着他们那可塑的灵能力量。他们将念动力扭曲成坚不可摧的屏障，挥击时则宛如攻城槌般强劲。

**设计笔记：念动使更新 Design Note：Psykinetic Updates**

以下是该子职自上次在《破解奥秘》（UA）中出现以来的主要更新：

- 强化念力 Stronger Telekinesis 是一个新增特性。
- 念力技巧 Telekinetic Techniques 现在允许该灵能使子职比其他灵能使使用更多次的念力驱使 Telekinetic Propel。
- 毁灭迷狂 Destructive Trance（原强效攻击模式 Empowered Attack Mode）不再使用灵能模式 Psionic Modes，并且你可以在不消耗灵能骰的情况下，将一个灵能骰加入到你的法术伤害中。
- 反弹力场 Rebounding Field 无论目标是否豁免成功，都会给予临时生命值。
- 强化念力碾压 Enhanced Telekinetic Crush 现在会将被消耗的灵能骰加入到增强后的法术伤害中。
- 升阶心灵遥控 Heightened Telekinesis 现在只需要灵能骰，不再消耗法术位。

**3级：念动使法术 Psykinetic Spells**

当你到达念动使法术表中特定的灵能使等级时，你就始终准备着表中对应的法术。

** 念动使法术 Psykinetic Spells**

| 灵能使等级 | 法术 |
|---|---|
| 3 | 护盾术 Shield，雷鸣波 Thunderwave 匕首之云 Cloud of Daggers，浮空术 Levitate |
| 5 | 缓慢术 Slow，念力碾压 Telekinetic Crush* |
| 7 | 欧提路克弹力法球 Otiluke's Resilient Sphere，塑石术 Stone Shape |

3级：强化念力 Stronger Telekinesis

当你施展法师之手时，其施法距离增加30尺，且该法师之手可以承载至多20磅重。

**3级：念力技巧 Telekinetic Techniques**

 当你使用你的念力驱使时， 你可以掷1d4并使用掷骰结果而非消耗一个灵能骰。

此外，当一个目标在对抗你念力驱使的豁免中失败时，你可以对目标施加以下效果之一。

- **提速 Boost。** 目标的速度增加10尺，直至你的下个回合开始。
- **迷惘 Disorient。** 目标无法进行借机攻击，直至其下个回合开始。
- **念力箭 Telekinetic Bolt。**目标受到等同于灵能骰结果的力场伤害。

**6级：毁灭迷狂 Destructive Trance**

在你的回合开始时，你可以消耗一个灵能骰进入毁灭状态。接下来10分钟内，你获得20尺飞行速度并可以悬浮。且当你施展一个消耗法术位的灵能法术时，你可以掷你的灵能骰，并将掷骰结果加入到该法术的一次伤害掷骰中。该掷骰不会额外消耗灵能骰。

**6级：反弹力场 Rebounding Field**

当你因被一次攻击检定命中时而施展护盾术 Shield 
，并使该次攻击失手时，你可以消耗一个灵能骰将这股力返还给攻击者。攻击者进行一次敏捷豁免。掷一次灵能骰。若豁免失败，攻击者受到等同于掷骰结果+你的智力调整值的力场伤害，豁免成功则攻击者仅受半伤。

10级：强化念力碾压 Enhanced 
Telekinetic Crush

当你施展法术念力碾压 Telekinetic 
Crush 
时，你可以消耗一个灵能骰来增强该法术，无论生物在对抗该法术的豁免成功与否，其速度都会减半，直至你的下回合开始。此外，你可以掷消耗的灵能骰，并将掷骰结果加入到该法术的一次伤害掷骰中。

**14级：升阶心灵遥控 Heightened Telekinesis**

你可以通过消耗四个灵能骰而非法术位来施展法术心灵遥控 Telekinesis。当你用此特性无需消耗法术位地施展心灵遥控 Telekinesis 时。你可以调整该法术，使其无需专注。若你如此做，本次施展的该法术持续时间变为1分钟，并且你能够以超巨型生物和物件为目标。
