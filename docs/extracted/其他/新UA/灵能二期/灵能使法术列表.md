# 灵能使法术列表 Warlock Spell List

**设计笔记：法表扩充 **

灵能使法术表现在包括以下法术：

- 一环：生命虹吸、护盾术、庇护术、雷鸣波
- 二环：灵质踪迹、自我之鞭
- 三环：泣墨深暗、举目皆敌
- 四环：生命反转场域
- 六环：精神监狱、灵思形态、灵能震爆
- 八环： 亚比达奇凋死术

本段给出了灵能使的法术列表。其中的法术依环阶及首字母顺序排列，并给出了每个法术的学派。特殊一栏中，C 代表该法术需要专注，R 代表该法术是一道仪式法术，而 M 则代表该法术具有特殊的材料成分。
