# 奥术子职 Arcane Subclasses

该游戏测试文档是破解奥秘系列文章的一部分，旨在为未来产品开发提供设计原型。本材料使用了《玩家手册2024》中的规则。

# 这里有什么？ What’s Inside?

这篇文档介绍了九个子职：包括牧师（奥秘领域）、战士（魔射手）、魔契师（咒剑宗主）、法师（咒法师）、法师（惑控师）、法师（死灵师）和法师（变化师）的改进子职；以及武僧（文身武者）和术士（先祖术法）的全新子职。

本材料为游玩测试内容 This is Playtest 
Material 本文旨在进行游玩测试与意见征集。此处选项均属实验性草案，尚未正式成为游戏官方内容。您的反馈将直接影响其是否被采纳为正式规则。

**如何游玩测试本次破解奥秘 How to Playtes This UA。** 
我们邀请您在游玩中体验本测试内容，您既可选择将其融入现有战役框架，也可组织单次或系列专项试玩活动。进行此类测试时，既可以自主设计原创冒险模组，也可以选用《来自无尽阶梯的委托集》等模组集中的短篇模组。
