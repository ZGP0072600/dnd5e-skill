# 牧师：奥秘领域 Arcana Domain

奥术伟力与至高信仰之结合

魔力充盈于整个多元宇宙之中，既能带来破坏，亦能进行创造。本领域的牧师们不会将魔法知识视作用来达成个人私心的力量，而是作为一件他们有责任传播并分享的礼物。

与奥秘领域有关联的神熟知魔法的秘密与潜力。这些神通常也与知识有关联，因为学习往往与奥术之力紧密相关，又或是与秘密和力量有关。

**设计笔记：奥秘领域更新 **

以下是该子职自《剑湾冒险者指南》（SCAG）中出现以来的主要更新：

- **奥秘领域法术 Arcane Domain Spells**进行了重置。
- **奥术传承 Arcane Initiate**现在为你提供奥秘技能的专精。
- **修饰魔法 Modify Macgic**是一条全新的特性，取代了**奥秘阻绝 Arcane Abjuration**。
- **驱法复原 Dispelling Recovery**（原**破法者 Spell Breaker**）现在使用解除魔法 Dispel Magic，并且在你移除某个生物身上的状态时也会触发。
- **奥术精通 Arcane Mastery** 现在能让你改变你所选择的法师法术。

**3级：奥秘领域法术 Arcane Domain Spells**

你与此神圣领域的链接使你始终准备了特定的法术。当你到达奥秘领域法术表中特定的牧师等级时，你就始终准备了表中对应的法术。

**3级：奥术传承 Arcane Initiate** 

你获得以下增益：

** 奥术学识 Arcane Knowledge。** 
如果你此前没有奥秘技能的熟练项，则你获得它。你还会获得奥秘技能的专精。

** 戏法 Cantrips。**你习得你所选择的两道法师戏法。无论何时每当你获得牧师等级时，你都可以将这些戏法中的一道替换为另一道法师戏法。

 

**3级：修饰魔法 Modify Magic**

你可以使用你的引导神力在你施展法术时修改它们。当你施展一道法术时，你可以消耗你的一次引导神力使用次数来以如下方式改变此法术（无需动作）：

** 健体法术 Fortifying Spell。** 
本法术的其中一个目标获得等同于2d8加你的牧师等级的临时生命值。

** 稳固法术 Tenacious Spell。** 
当你施展一道迫使生物进行豁免检定的法术时，选择一个你能看见的本法术的目标，投掷一个1d6并使得该生物在此次豁免中获得此减值。

 

**6级：驱法复原 Dispelling Recovery**

在你施展了一道消耗环位且为其他生物恢复了生命值或终止了某个状态的法术后，你可以立即以一个附赠动作不消耗法术位地为该生物施展法术解除魔法 Dispel 
Magic 
。

你可以使用此特性的次数等同于你的感知调整值（至少为1），并且在你完成一次长休后恢复全部的使用次数。

 

**17级：奥术精通 Arcane Mastery**

你习得四道法师法术，六、七、八、九环各一道。此后你总是准备着这些法术。每当你获得牧师等级时，你都可以将这些法术替换为同环位的另一道法师法术。
