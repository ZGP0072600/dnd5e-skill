# 野蛮人：精魂守卫道途 Path of the Spiritual Guardian

呼唤精魂，援护战斗

踏上精魂守卫道途的野蛮人会呼唤精魂——无论是自然的野兽精魂，已故的先人精魂，还是原生的元素精魂——寻求引导和保护。当这些野蛮人进入狂暴时，他们会与精魂接触并请求它们的援助。

**设计笔记：精魂守卫道途更新 **

以下是该子职（旧称先祖守卫道途 Path of the Ancestral Guardian）自《珊娜萨的万事指南》（XGE）中出现以来的主要更新：

- **精魂护卫 Spiritual Protectors**（旧称先祖护卫 Ancestral Protectors）已被重新设计，此特性现在将原本效果和新效果作为选项，在你处于狂暴状态下命中生物时选择并生效。
- **精魂之盾 Spirit Shield**现在随着你的狂暴伤害成长。
- **复仇精魂 Vengeful Spirits**（旧称仇魂先祖 Vengeful Ancestors）现在是一个允许你在攻击掷骰中掷出18或以上时进行一次额外攻击的新特性。

**3级：精魂护卫 Spiritual Protectors**

你的狂暴会召来精魂战士助你一臂之力。在你的狂暴激活期间，当你以武器或徒手打击命中一个生物时，它会成为精魂的目标，这会导致一个你从下列选项中选择的后果。

** 分神。**直到你的下个回合开始，目标在使用攻击检定对抗你或另一个拥有此特性的野蛮人以外的生物时具有劣势。

** 守护。**直到目标的下个回合结束，下一次它以攻击检定命中你以外的生物时，那个生物获得那次攻击造成伤害的抗性。

** 袭击。**目标受到1d6点额外伤害，伤害类型可以是强酸，寒冷，火焰，力场，闪电或雷鸣（由你选择）。

**6级：精魂之盾 Spirit Shield**

你的守护精魂可以为你保护的生物提供超自然的防护。在你的狂暴激活期间，当你30尺内的另一个可见生物受到伤害时，你可以用反应减少那次伤害。投掷等于你狂暴伤害加值数量的 d6，将它们相加，即是你减少的伤害。
