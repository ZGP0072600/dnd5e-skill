# 魔契师：巫王宗主 Sorcerer-King Patron

怪怖君王的暴政传令官

你的契约源自于某个怪诞的、残暴的强大力量，那力量类似半神，或是一位极度强大的施法者。凭借契约，你成为这位暴君在世间的传令官，传递祂的意志，并导引那足以扭曲心智的灵能之力。

**3级：巫王法术 Sorcerer-King Spells**

宗主赐予的魔法使你始终准备着特定的法术。当你到达巫王法术表中特定的魔契师等级时，你就始终准备着表中对应的法术。

** 灵能施法 Psionic Casting。** 当你施展来自巫王法术表中的法术时，你可以无需材料或言语成分，那些会被消耗或有具体价值的材料成分除外。

** 巫王法术 Sorcerer-King Spells**

| 魔契师等级 | 法术 |
|---|---|
| 3 | 命令术 Command，强令对决 Compelled Duel，激愤斩 Wrathful Smite 定身类人 Hold Person，心灵尖刺 Mind Spike |
| 5 | 恐惧术 Fear，短讯术 Sending |
| 7 | 强迫术 Compulsion，惊惧斩 Staggering Smite |
| 9 | 支配类人 Dominate Person，突触静止 Synaptic Static |

**3级：暴君传令官 Tyrant's Herald**

你获得以下增益：

** 威慑气场 Intimidating Presence。** 若你不具有威吓技能熟练项，你获得威吓技能熟练项。你还会在威吓上获得专精。

** 暴政之声 Voice of Tyranny。**以一个附赠动作，你可以无需消耗法术位施展法术命令术 Command 
。你可施展的次数等同于你的魅力调整值次（至少一次），在完成一次长休后，你重获所有的已消耗的次数。

 

**6级: 决断诏令 Decisive Edict**

你宗主的力量在你体内涌动，使你能鼓舞你的盟友并征服你的敌人。当你使用契约法术位施展法术时，你能在源自你的30尺光环区域内引发亵渎之力的爆发，对每个该光环区域内你可见的生物，选择以下效果之一：

** 统帅 Marshal。** 
直至其下回合结束前，目标生物在攻击检定上具有优势 
。

** 压迫 Oppress。** 
目标生物必须成功通过一次对抗你法术豁免 DC 的感知豁免，否则陷入恐慌 状态，直至其下个回合结束。

此特性一经使用，直至完成短休或长休你都无法再次使用。当你使用 ** 秘法回流 Magical Cunning** 时，你也会回复此特性的使用次数。

 

**10级：复仇叱喝 Vindictive Rebuke**

你学会了回击那些挑战你或你宗主权威的存在。当敌人以一次攻击检定命中你时，你可以使用反应强迫该敌人重骰本次 d20，且该敌人必须使用新的结果。若此反应导致本次攻击检定未命中，触发该效果的生物受到等同于你魔契师等级的心灵伤害。

你可以使用此特性的次数等同于你的魅力调整值次（至少一次），在完成一次长休后，你重获所有的已消耗的次数。

 

**14级：绝对暴政 Absolute Tyranny**

你以绝对的信念行使你宗主的暴君强权。每当你施展法术命令术 Command 时，你能选择施法距离内一名额外生物作为目标。此外，被你恐慌的生物自动失败于你施展的任何命令术 Command 豁免。
