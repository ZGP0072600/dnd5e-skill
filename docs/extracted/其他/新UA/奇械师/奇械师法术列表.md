# 奇械师法术列表 Artificer Spell List

本段给出了奇械师的法术列表。其中的法术依环阶及首字母顺序排列，并给出了每个法术的学派。特殊一栏中，C 代表该法术需要专注，R 代表该法术是一道仪式法术，而 M 则代表该法术具有特殊的材料成分。

**设计笔记：法术列表的更新 Spell List Updates** 奇械师的法术列表包括了以下新法术：

**戏法：** 四象法门 Elementalism，克敌先击 True Strike

**二环：** 复苏秘法 Arcane 
Vigor，龙息术 Dragon's Breath， 
 人工生命仆从 Homunculus Servant**五环：** 原力法阵 Circle of Power

该列表包括《**玩家手册 **》和这篇文章的法术。如果你拥有《** 塔莎的万事坩埚 **》，你的奇械师也可以继续施展那本书的该职业的法术。
