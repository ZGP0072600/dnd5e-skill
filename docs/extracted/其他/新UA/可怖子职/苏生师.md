# 再起伴兵 Reanimated Companion

*重组尸骸，复生亡者*

苏生师为了进行他们的可怖试验而违背自然法则。这些残忍的奇械师们将来自不同团体的仆从缝合在一起。

**3级：苏生师法术 Reanimator Spells**

 在达到下方苏生师法术列表中列出特定奇械师等级时，你将始终准备着等级对应的法术。

| 奇械师等级 | 法术 |
|---|---|
| 3 | 虚假生命 False Life，维生术 Spare the Dying 巫术箭 Witch Bolt |
| 5 | 目盲/耳聋术 Blindness/Deafness，强化属性 Enhance Ability |
| 9 | 活化死尸 Animate Dead，闪电箭 Lightning Bolt |
| 13 | 枯萎术 Blight，防死结界 Death Ward |
| 17 | 防活物护罩 Antilife Shell，死者复活 Raise Dead |

**3级：电颤复律 Jolt to Life** 当你施展维生术 Spare the 
Dying 时，你可以调整该法术使其向目标发射一阵电流，令其复律*。目标恢复1生命值，且位于源自其10尺光环区域内的每名生物必须进行一次对抗你法术豁免 DC 的敏捷豁免，豁免失败则受到等于1d4加你奇械师等级一半（向下取整）的闪电伤害，豁免成功则受到一半伤害。

你能够以此法调整法术的次数等于你的智力调整值。当你完成一次长休时，你重获所有已消耗的使用次数。

译注：此处复律原文为“reviving”，但按照维生术原有目标（生命值为0生物），可能仅是风味描述而非复活，此处暂以复律代之。

**3级：再起伴兵 Reanimated Companion** 你能够以一动作，使用修补工具或你具有熟练的其他种类的奇械工具，通过亡灵法术与科学的力量创造一名 **再起伴兵 Reanimated Companion**（数据卡见下）。伴兵显现于你5尺的未占据空间中。伴兵的外观由你决定，但你的选择不会改变其游戏数据。

伴兵对你和你的盟友态度友善并服从于你。伴兵存在持续至你完成长休，或在你执行魔法动作令其解散时提前结束。此时伴兵无害地坍塌为一堆内脏。若你死亡，伴兵立即降至0生命值并死亡（触发其自爆特质）。

一旦你创造了一名伴兵，你无法再次创造伴兵直至你完成长休或你消耗一枚法术位来创造伴兵。你同时只能拥有一名伴兵，伴兵存在期间你无法创造伴兵。

** 战斗中的伴兵 The Copanion in Combat。**在战斗中，伴兵在你的回合内行动。其可以移动并自行执行反应，但除非你以附赠动作命令其执行动作，否则其唯一可用的动作是回避动作。若你陷入失能，伴兵自行行动且不再限制于回避动作。

## 特质 Traits

**自爆 Death Burst。** 伴兵在死亡时爆炸。*敏捷豁免检定*：DC 等于你的法术豁免 DC，源自伴兵的10尺光环区域内的每名生物。 *失败*：2d6暗蚀伤害。

**闪电吸收 Lightning Absorption。** 每当伴兵将受闪电伤害时，其恢复等于闪电伤害数值的生命值。

## 动作 Actions

**恐怖横扫 Dreadful Swipe。** *近战攻击检定*：加值等于你的法术攻击加值，触及5尺。*命中*：1d4+2加你的智力调整值的暗蚀伤害，且目标无法执行借机攻击直至其下个回合开始。
