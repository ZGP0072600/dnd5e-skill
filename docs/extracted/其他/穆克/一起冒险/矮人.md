# 矮人

**矮人 Dwarf**

虽然没有矮人居住在真正的住在阴林，但他们有时会在旅途中经过

或探索森林，寻找梦寐以求的材料和魔法石。 
他们为自己的技能感到自豪，并能用在大自然中找到的物品制作出精美的工艺品。

该地区的矮人天生对地精很谨慎。他们会想方设法避免与地精接触，而且很少在阴林长时间逗留。

**矮人 Dwarf**

*中型类人（矮人），守序善良*

护甲等级：13

生命值：30

速度：25尺

力量16（+1） 
*敏捷13（+2） 体质14（+2）

智力10（+0） 感知11（+0） 魅力10（+0）

挑战等级：1/4

**矮人体魄 Dwarven Resilience。** 矮人抵抗中毒进行的豁免检定具有优势，且它们具有毒素伤害抗性。

**动作 Actions**

** 手斧 Handaxe。**近战武器攻击：命中+3*，触及5尺，单一目标。命中：4（1d6+1）点挥砍伤害。

（译者注：该怪物力量调整值错误，应为16（+3），动作的手斧使用了错误点的力量调整值计算命中，应为+5。）
