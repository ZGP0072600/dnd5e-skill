# 水晶洞穴人鱼

**水晶洞穴人鱼 Crystal Cave Merfolk**

水晶洞穴人鱼拥有长而流动的鳍和银色的鳞片。 他们的声音悦耳动听，旋律优美。 
在洞穴水晶的振动下，它们的声音更加悦耳动听。

他们喜欢任何银制的东西，并愿意用他们在洞穴中发现的珍珠和它们在水下狩猎时发现的小块黄金交换。 
他们还有魔法药水，可以帮助水面上的居民在水下呼吸。

水晶洞穴人鱼 Crystal Cave 
Merfolk*中型类人（人鱼），绝对中立*

护甲等级：11

生命值：11

速度：10尺，游泳40尺

力量10（+0） 
敏捷13（+1） 体质12（+1）

智力11（+0） 感知11（+0） 魅力12（+1）

挑战等级：1/8（25 XP）

**水陆两栖 Amphibious。** 人鱼可以在空气和水中呼吸。

**塞壬之歌 Siren Song。** 动听的歌声吸引着300尺内的每一个类人生物或巨人。 目标必须在通过一次 DC 
11的感知豁免，否则就会被魅惑，直到歌声结束。

**动作 Actions**

** 矛 Spear。**近战或远程攻击：命中+2，触及5尺，单一目标，伤害：3（1d6）的穿刺伤害。
