# 布恩图猴

**布恩图猴 Boontu Monkey**

布恩图猴是一种胆小、鬼鬼祟祟的动物。 它们贪婪，总是在寻找食物。 虽然贪婪，但它们从来不吃它们生活的布恩图树 
boontu tree 上神圣的金色坚果。

有人说，它们曾经是贪婪的盗贼，后来从一个强大的树精的圣树上偷走了坚果而被抓住变成了猴子。

**布恩图猴 Boontu Monkey**

*微型野兽，无阵营*

护甲等级：12

生命值：3

速度：20尺，攀爬20尺

力量8（-1） 
敏捷14（+2） 体质11（+0）

智力5（-3） 感知12（+1） 
魅力6（-2）

挑战等级：0（10 XP）

**集群战术 Pack Tactics。** 若布恩图猴至少有一个未处于失能的盟友在目标生物5尺内时，它对该生物发动的攻击检定有优势。

**动作 Actions**

** 啃咬 Bite。**近战武器攻击：命中+1，触及5尺，单一目标。命中：1（1d4-1）点穿刺伤害。
