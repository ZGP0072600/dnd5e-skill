# 紫虫

**紫虫，伟大的喀戎姆 Great Kroom, Purple Worm**

伟大的喀戎姆是一种紫色蠕虫，每隔二十年就会来到阴林 Dankwood，以生长在森林下的各种东西为食。

伟大的喀戎姆是一种可怕而危险的怪兽，它可以轻易摧毁哥布镇 Gob-Town，但我们有办法阻止它：

- 你需要找到一根痒痒棒。这根魔法棍子可以用来挠紫虫的痒痒，让它变得温顺。然后就可以把它引离阴林。
- 你需要让它吃一种叫做 Pun derpuff 的稀有蘑菇。这种蘑菇生长在阴林深处，有一种强大的化学物质可以让紫虫沉睡。

**紫虫，伟大的喀戎姆 Great Kroom, Purple Worm**

*超巨型怪兽，无阵营*

护甲等级：18

生命值：247

速度：50尺，掘穴30尺

力量10（+0） 
敏捷13（+1） 体质12（+1）

智力11（+0） 感知11（+0） 魅力13（+1）

挑战等级：15（13,000 XP）

**掘道者 Tunneler。** 紫虫能够以其一半的掘穴速度通过坚硬的岩层，并在其身后留下直径10尺的地道。

**动作 Actions**

** 多重攻击 Multiattack。**紫虫发动两次攻击。

** 啃咬 Bite。**近战武器攻击：命中+14*，触及10尺，单一目标。命中：22（3d8+9）点穿刺伤害。目标必须成功通过一次 DC 
19的敏捷豁免，否则被紫虫吞掉！*

**尾刺 Tail Stinger。** 近战武器攻击：命中+14，触及10尺，单一生物。命中：19（3d6+9）点穿刺伤害。且目标必须进行一次 DC 
19的体质豁免，豁免失败则受42（12d6）点毒素伤害。

（译者注：该怪物错误数据较多，除力量10（+0）和命中+14外（推断为复制紫虫忘了改），该紫虫的吞噬没有写逃出的办法。）
