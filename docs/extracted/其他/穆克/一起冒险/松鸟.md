# 松鸟

**松鸟 Birdsquirrel**

松鸟是一只勇敢的小动物，它全心全意地爱着穆克。有人说松鸟是法师 杜恩格瓦 Doonwaggle 
创造的，它是阴林 Dankwood（也许是整个世界）唯一的一只松鸟，但谁知道呢？

**松鸟 Birdsquirrel**

*微型野兽，无阵营*

护甲等级：13

生命值：10

速度：20尺，攀爬20尺，飞行30尺

力量6（-2） 
敏捷14（+2） 体质13（+1）

智力4（-3） 感知12（+1） 魅力6（-2） 

挑战等级：1/4（50XP）

**集群战术 Pack Tactics。** 若松鸟至少有一个未处于失能的盟友在目标生物5尺内时，它对该生物发动的攻击检定有优势。

**动作 Actions**

** 多重攻击 Multiattack。**松鸟进行两次攻击：一次啃咬和一次爪击。

** 啃咬 Bite。**近战武器攻击：命中+4，触及5尺，单一目标。命中：5（1d6+2）点穿刺伤害。

** 爪击 Claws。**近战武器攻击：命中+4，触及5尺，单一目标。命中：4（1d4+2）点挥砍伤害。
