# 雪人幼崽

**雪怪幼崽 Yeti Tyke**

这个雪怪家族通常不会冒险去阴林，它们的家在北方冰天雪地的雪山上，那里空气凉爽，它们蓬松的白色绒毛是完美的伪装 
。

一只与家人失散的雪怪幼崽害怕、饥饿，正在寻找一张友好的面孔。 它响亮悲伤的嚎叫声能够找到它目前藏身之处。

要想把受惊的雪怪幼崽哄出来，你可以尝试 
以下方法：

- 雪怪的嗅觉非常灵敏，一块臭烘烘的小吃可能会起到作用！
- 你们可能语言不通，但一些雪怪的嚎叫声和雪怪的手势一定会表明你没有恶意。 （确保你要展示一下怎么做！）。
- 这只雪怪幼崽想念它的家人，有没有办法把自己伪装成失散多年的雪怪亲戚？

**雪怪幼崽 Yeti Tyke**

*小型怪兽，混乱淘气鬼 silly*

护甲等级：11（天生护甲）

生命值：9（2d6+2）

速度：20尺，攀爬20尺

力量10（+0） 
敏捷11（+0） 体质12（+1）

智力6（-2） 感知8（-1） 
魅力5（-3）

技能：察觉+3，隐匿+3

伤害免疫：寒冷

感官：黑暗视觉60尺，被动察觉9

语言：理解雪怪语但不会说

挑战等级：1/8（25 
XP）

**敏锐嗅觉 Keen Smell。** 雪怪依赖嗅觉进行的感知（察觉）检定具有优势。

**雪地伪装 Snow Camouflage。** 雪怪在雪地地形中进?躲藏时所作的敏捷（隐匿）检定具有优势。

**动作 Actions**

** 爪击 Claw。**近战武器攻击：命中+2，触及5尺，单一目标。伤害：2（1d4）点挥砍伤害，外加2（1d4）点寒冷伤害。
