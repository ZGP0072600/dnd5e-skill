# 泥鬼婆

**泥鬼婆 Sludge Hag**

在看到泥鬼婆之前，你早就能闻到她的气味。 

只要有泥鬼婆在，就会有一股超级臭的味道，就像臭鸡蛋混合着老花露水的超级臭味。 

泥鬼婆还会在她们栖息的地方留下油乎乎、黑糊糊的痕迹，靠近的小动物身上会沾上油乎乎的粘液 ，需要其他生物的帮助才能洗干净。 

有传言说，泥鬼婆曾经是纯洁的妖精，因为虐待环境而被黏糊糊的淤泥包裹于其中。 另一个传言则说，泥鬼婆的皮肤非常瘙痒 
，而黏糊糊的东西能让她们感到放松。

**泥鬼婆 Sludge Hag**

*中型妖精，中立泥泥 oozy*

护甲等级：17（天生护甲）

生命值：82（11d8+33）

速度：30尺

力量18（+4） 
敏捷12（+1） 体质16（+3）

智力13（+1） 感知14（+2） 
魅力14（+2）

技能：奥秘+3，欺瞒+4，察觉+4，隐匿+3

感官：黑暗视觉60尺，被动察觉14

语言：通用语，龙语，木族语

挑战等级：3（700 XP）

**无定形 Amorphous。** 泥鬼婆能够轻松穿过最窄1寸宽的狭窄空间而不用挤进去 
。

**水陆两栖 Amphibious。** 泥鬼婆可以在空气和水中呼吸。

**天生施法 Innate Spellcasting。** 泥鬼婆天生施法的关键属性为魅力（法术豁免 DC 
12）。它天生可施展以下法术且不需要任何相应的材料成分：

*随意：酸液飞溅 acid splash，舞光术 dancing lights，次级幻象 minor illusion*

**动作 Actions**

** 泥形 False Appearance。**泥鬼婆坍塌为一摊油乎乎的污泥，静止不动时与其他油乎乎的污泥并无区别。

** 给泥一巴掌 Sluge Slap。**近战武器攻击：命中+6，触及5尺，单一目标，伤害：8（1d8+4）点挥砍伤害，外加4（1d8）点强酸伤害。
