# 职业

卡拉德许

发明与魔法 Invention and Magic

发明创造是卡拉德许生活和文化的基石，无论是在执政院铸造厂中为改善人民生活而默默无闻地奉献的公职人员，还是只要在吉拉波街头现身就能引得人们驻足围观的明星逸匠，发明家都是地位崇高的风流人物。也正是因此，发明家和工匠团体堪称卡拉德许社会中的重要组成部分。社群将兴趣爱好相似的人才团结到一起，为他们提供社交活动和分享思想的平台。

能够真正施放咒语而无需借助乙太动力装置的魔法师在卡拉德许则颇为少见。在执政院的法令下，魔法师通常被视为危险且不安定的因素，以至于执政院专门设立了一支特殊的队伍，负责保护卡拉德许免遭魔法师的破坏。这支由监察长巴罗领导的队伍致力于对魔法师进行搜捕和研究，识破其阴谋并绳之以法不论他们是否真正犯下了罪行。

玩家角色的职业 Player Character Classes

由于真正能够施放咒语的能力在卡拉德许颇为少见，你也许希望在你的个人战役中限制可供选择的职业。与此同时，你也可以自由调整或忽略这些限制，从而符合你的玩家和你个人对卡拉德许的想象。

野蛮人 Barbarians 可能出没于偏远之地，像是皮默或勒图之外的地区。这类角色在吉拉波通常显得格格不入但这将是角色成长的良机。妖精1族野蛮人（大多属于摒弃城市生活和乙太技术的蒂厄赫一族）也许是一个极为合适的选择。“狂战士道途”比起“图腾武士道途”则更符合卡拉德许的世界设定。

吟游诗人 Bards 在卡拉德许大多被视为一类特别的魔法师，类似于烈焰术士。因此，他们会受到执政院的迫害。

牧师 Clerics 在卡拉德许并不存在。宗教在卡拉德许人民的生活中并不是一股强大的力量，源于诸神之力的魔法在此并不为人所知。

德鲁伊 Druids 是运用大通联之力制造魔法效应的妖精。非妖精种族无法成为德鲁伊，在此地也不存在“月亮结社”。

战士 Fighters 在卡拉德许颇为常见，但奥法骑士在此是个陌生的概念。战士可以是执政院的执法人员，也可以是乱匠游勇，还可能是普通（且健壮）的市民。

武僧 Monks 在卡拉德许不为人所熟知，除非你想在该时空的遥远边区创建一个隐修会组织。

圣武士 Paladins 也与武僧因同样的原因而无人知晓。也许一名战士会因献身践行某种誓约而获得魔法的力量，但这一角色大多会被看做是魔法师。

游侠 Rangers 类似德鲁伊，在卡拉德许须为妖精种族。

游荡者 Rogues 在此处颇为常见，然而诡术师则会因被视为魔法师而遭到迫害。考虑到他们出于各种原因大都站在律法的对立面上，大多数游荡者都是乱匠。

术士 Sorcerers 是在卡拉德许最常见的一类魔法师，屡遭执政院的追捕。说他们“常见”，是说很多人都对“烈焰术士 pyromancer”有粗浅的认识，一见即知；然而他们同时也罕见到人见人畏的地步。这里的术士不会拥有龙族血脉，但这里有狂野魔法师和风暴术士。类似茜卓·纳拉的烈焰术士则可能拥有独特的术法起源。

魔契师 Warlocks 即便存在也并不广为人知。一名魔契师也许是沉迷“黑暗图纲”用于制作恶魔的图纸并最终与这些邪魔订立契约的发明家。与异界宗主订立契约在此地更为罕见，但由于卡拉德许邻近充满乙太的黑暗虚空，这也许足以致使某种类似奥扎奇的异界意识得以投射到这个世界之中，从而与魔契师形成契约关系。此外，大通联自身也可以被拟人化地视为拥有妖精宗主特性的存在。

法师 Wizards 在卡拉德许极为罕见，然而维多肯乙太科学家也许充分认识乙太的流动，从而能够加以引导转化为法术效果。
