# 附录

附录 Appendix:The Colors of Magic

万智的颜色

MTGCN 翻译组

- **地**：不同类别的地貌出产不同颜色的法术力。举例而言，山脉出产红色法术力，而平原出产白色法术力。
- **处世哲学**：每种颜色都与一系列原则、理念和价值观念相关。白色与保护和秩序相关，绿色则关联着生机和自然。
- **魔法效应**：不同颜色的魔法用于为不同种类的咒语提供能量。火焰、闪电以及加速魔法有赖于红色法术力，而水流、冰雪和心智类魔法使用蓝色法术力。
- **种族和生物**：作为例外，人类对各色魔法都没有特别的天资或亲缘。而其他种族，甚至包括非智慧生物都与颜色有着很强的联系。例如妖精通常与绿色相关。他们运用自然魔法，重视平衡、稳定和相互依赖。多头龙也与绿色相关，尽管它们不会施放魔法，也没有什么价值观可言尽管它们自身会被认为是这些价值观的体现。所以如果你想召唤多头龙，你需要施放绿色的咒语。
- **形上本质**：魔法甚至是强烈的感情有时会改变人们或者生物的本质，改变他们相关联的颜色。邪恶的诅咒也许会将一个人从绿色的阵营变为黑色（或者黑绿双色）。痛失所爱让人寻求复仇，从而为其颜色增添红色无论是暂时的还是永远。
- **性格品质**：不同的颜色与不同的性格品质相关。随性而有创造性，精力旺盛的人自然会被红色法术力吸引；同时也许正是因为运用了红色法术力激发出了此人的这些特质。最经常的情况，这两种效应是并存共行。
