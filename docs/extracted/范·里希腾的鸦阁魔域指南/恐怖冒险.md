# 第四章：恐怖冒险Horror Adventures

创造你自己的恐怖冒险其实和创作其他的 D&D 冒险没有什么不同，但有一项例外：你创作的目的是，以最有趣的方式让你的玩家们体验到恐惧。恐怖冒险得益于毛骨悚然的气氛，而这种气氛由对恐怖情节的描述、戏剧化呈现、鼓励恐怖的游戏元素组成。不过要想勾出玩家焦虑或期待的情绪，那就需要深思熟虑。本章节提供了一些工具和技巧，它们可以确保你的游戏即恐怖又安全，以适合你的特定群体。一个以恐怖为核心的规则工具箱也为你可能创造的恐怖冒险提供了一些选择。在本章的最后，一个恐怖冒险会将这些方法都运用起来，并引导角色们迈出他们踏入恐惧领域的第一步。
