# 领域特征 Featured Domains

下面的章节将带你探索一些恐惧诸领域中最为恶名远扬的领域。这些领域每个都是自成一体，可以成为你冒险开展的舞台。对这些著名领域的描述都遵从以下格式：

概览 Overview.对每个领域的简短叙述，包含黑暗领主的名字、启发领域创作的恐怖题材（详见第二章），领域的标志性特征，以及与该领域相关联的迷雾护符。

显要特征 Noteworthy 
Features. 记述了一些该领域的居民和曾经到过此地的旅行者们所知道的关于该领域的细节。

聚落与地点 Settlements and Sites.概述了本领域中一些最为声名狼藉的地点。这些地点一般都在地图上标示出来了。但每张地图上还有许多未被说明的地点，等待着在你的冒险中被探索。

黑暗领主 Darklord.这个部分会对该领域的黑暗领主进行描述，包含对他们堕入邪恶的根源的详细记述。

冒险 Adventures.该部分描述了天然适合该领域的一些冒险。

领域焦点 Domain Focus.在大多数领域的介绍中，最后一部分都强调了某些独特的故事元素，还提供了专属于该领域的工具，帮助你围绕该领域的黑暗领主和其国度中的恐怖之处创作冒险。
