# 鲁道夫·范·里希藤

## 鲁道夫范里希滕 Rudolph van Richten

身为一名学者和怪物猎手，鲁道夫范里希滕曾经到过数十个领域，对各种怪异存在的报告进行了调查，并将成果整理成一系列指南书出版。其中最有名的是《范里希滕的吸血鬼指南》。

鲁道夫曾经生活美满，他和妻子英格丽德、儿子伊拉斯谟住在达肯领域的瑞瓦利斯，拥有一幢温馨的家宅。当年的鲁道夫刚刚成为医生且脾气急躁，与拉达纳维奇家族起了冲突，这个家族的人用维斯塔尼人的传统做掩护，暗地里进行劫掠的勾当。

当范里希滕医生拒绝治疗该家族里某个病入膏肓的成员，团伙首领伊雷娜拉达纳维奇 Irena 
Radanavich 便命令她的手下绑架了鲁道夫的儿子，随即将这名年轻人卖给了吸血鬼梅图斯男爵。

鲁道夫追上了拉达纳维奇一伙，粉碎了他们的犯罪活动并将伊雷娜绳之以法，但伊雷娜留下了诅咒：“你的生活将一直与怪物为伍，并看着你所爱的人死在怪物的爪下。”接下来的一个星期诅咒就见效了。在鲁道夫追上并且杀死梅图斯男爵之前，这个吸血鬼就杀害了英格丽德和伊拉斯谟。

在那之后的几十年里，范里希滕持续追猎着怪物，并向其他与黑暗做斗争的人们传授所需的知识。尽管结交到了许多忠心耿耿的同伴，但他一直惧怕自己身上的诅咒，与同伴们保持着距离。不在旅途中的时候，鲁道夫靠经营药草店过活。他的店铺位于莫登特领域的莫登特郡。

鲁道夫范里希滕角色特点：

理想 Ideal 
“邪恶必须被惩罚。”

牵绊 Bond “为了守护我挚爱之人，我必须让他们远离我的敌人并不为他们所知。”

缺点 Flaw “我被诅咒了。因此，我永远无法得到安宁。”

与鲁道夫范里希滕一起冒险

范里希滕会欣然教导那些决心要与暗夜中的生物战斗的角色们。来自《怪物手册》中的祭司 Priest 数据卡和来自《施特拉德的诅咒》里的瑞克塔维欧 Rictavio 数据卡均可以用作他的数据卡。若是要将范里希滕引入某个冒险，可以考虑以下情节：

很久以前，范里希滕杀死了一个以恐怖手段统治着某个聚居地的反派，这个反派具有超自然的能力。现在，邪恶卷土重来。冒险队伍必须找到范里希滕医生，只有他知道如何击败这个生物最好这次是一劳永逸地击败。

范里希医生滕的某个敌人抓住了他，盗用他的名义与冒险者们联系，将那些想要知道医生的秘密的人们诱入致命的陷阱。
