# 威瑟梅-狐林孪生姐妹

## 威瑟梅-狐林孪生姐妹 Weathermay-Foxgrove

莫登特郡市长爱丽丝威瑟梅 Alice 
Weathermay 的双胞胎女儿珍妮弗 Gennifer 和劳莉 Laurie 是一对形影不离的捣蛋鬼。就在她们十六岁生日之前，她们的叔叔、富有声望的怪物猎手乔治威瑟梅 George 
Weathermay 带着未婚妻娜塔莉亚沃里什科娃 Natalia 
Vhorishkova 回到了莫登特。双胞胎发现娜塔莉亚正在操纵她们亲爱的叔叔，最终，她们揭发出了真相：娜塔莉亚其实是个狼人，正在利用乔治来追猎莫登特另外一位著名的居民：鲁道夫范里希滕。双胞胎姐妹从狼人手里拯救了范里希滕和她们的叔叔之后，这两位长辈都鼓励她俩发展精于探查的天性，训练她们成怪物猎手。

从那之后，劳莉和珍妮弗成为了不屈不挠的冒险者。劳莉像她叔叔一样习得了与不死者战斗的武艺。她总是带着伙伴琼 Joan 和提朗 Tirran 他叔叔猎狐犬的后代，也配着他叔叔的魔法剑：轻丝 Gossamer。

而珍妮弗与范里希滕密切合作，学习了来自许许多多领域的传统以及医药知识。她尤其钻研了兽化症，便于姐妹俩追捕仍然逍遥法外的娜塔莉亚沃里什科娃。珍妮弗暗中害怕自己已经感染了兽化症，尽管没有任何迹象显示出她已经受到诅咒，她依然服用自己调制的药物来防治。

珍妮弗的角色特点

理想 Ideal："任何事情都有解决的办法，我只需要足够聪明，把它找出来。"

牵绊 Bond："我的家族已经承受了太多，现在轮到我接过使命了。"

缺点 Flaw："我永远也没法像我的双胞胎姐妹那样强而有力，所以我必须要变得更加聪明。"

劳莉的角色特点

理想 Ideal："任何事情都有解决的办法，我必须能够及时抓住机会。"

牵绊 Bond："行动，而非空谈，使我的家族持续受到尊重。"

缺点 Flaw："我永远没法像我的双胞胎姐妹那样聪明，所以我必须要变得更加强而有力。"

与双胞胎姐妹一同冒险

珍妮弗和劳莉是技艺纯熟、准备充分的冒险者，分别使用德鲁伊 druid 和老兵 veteran 的数据卡。若是要将双胞胎姐妹引入某个冒险，可以考虑以下情节：

角色们遇到了一件应付不来的事情，正好，威瑟梅-狐林双胞胎姐妹在追踪相同的事件，角色们得到了她们的帮助。事后，姐妹俩鼓励小队参加一场前往某个恶名远扬的地方的训练远征。

珍妮弗失踪了。劳莉害怕是娜塔莉亚沃里什科娃绑架了她，于是寻求援助。实际上，是珍妮弗害怕自己会兽化症发作而独自行动，这导致她陷入了更大的危机中。
