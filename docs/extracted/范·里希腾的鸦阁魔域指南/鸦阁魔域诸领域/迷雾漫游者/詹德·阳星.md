# 詹德·阳星

## 詹德阳星 Jander Sunstar

高等精灵詹德阳星生于名叫托瑞尔的世界，他曾是一个冒险者，而后被悲剧地转化成了一个吸血鬼。在击败他的尊主后，阳星漫游于远方，奋力抵抗着他的吸血冲动并与一个叫安娜的冒险者相爱，而安娜据说来自另外一个世界。当詹德揭示了自己的真实身份后，安娜的同伴们帮助她逃离了詹德。被愤怒所笼罩的詹德杀死了所有他认为需要对他与安娜之分离负责的人。而这场屠杀的后果，便是迷雾之笼罩于詹德，将他传送到了巴洛维亚。在那里，詹德遇见了施特拉德冯扎洛维奇，并在鸦阁城堡中生活了一段时间。当那更古老的吸血鬼试图把詹德变成他的奴仆时，詹德尝试着去杀死这个伯爵。他失败了，并向迷雾之中逃去。

在之后的几世纪中，詹德以一个神秘冒险者的身份四处游荡，寻找着吸血鬼化的解药。他已找到了许多种疗法，但无一能对他起效。在近几十年里，他尝试拯救一名冒险者的举动导致了萨弗拉 Savra 阳星，一位詹德视其为己出的半血裔之出生。但是萨弗拉却憎恨着他，并发誓要奉献一生以猎杀他与其他吸血鬼。

他与他女儿的不和推动着詹德寻求那些更激进方法来解除自己的吸血鬼化。在一次协助莫登特领域之中一位神秘的炼金术士的实验里，阳星成为了第一个测试一台神秘置魂装置 Apparatus 原型机之人。但是机器发生了故障;机器没有消除他的吸血鬼化，而是制造出了大量他的分身，这些分身分散在各个位面之中。尽管已有不知道多少阳星的分身已经迎来了他们半悲剧的结局，但所有分身仍都相信他们才是真正的詹德阳星。至少还有一个詹德还被困于鸦阁魔域之中，永远地寻求着他永不应得的平静安息。

詹德阳星的角色特点

理想。“吸血鬼化是必须要被铲除的诅咒尤其对我而言。”

牵绊。“我将承受高尚的苦难，我会一直忍受着诅咒，这样我就能为历史之中最严重的灾难画上句号。”

缺点。“我的领悟与痛苦所带来的比那些更年轻、幼稚生命的领悟、痛苦的沉重的多。”

与詹德阳星一起冒险 Adventures with Jander Sunstar

詹德阳星是一个吸血鬼 vampire，也是鸦阁最首要的吸血鬼化例子。他帮助玩家们猎杀吸血鬼并寻找盟友让他免受其他同类的侵扰。在把詹德阳星加入到一场冒险中时，请考虑以下点子。

詹德需要援助以摧毁另一个吸血鬼:他的分身之一。

詹德让玩家们帮助他解救一个遭遇了强大敌人的怪物猎人。在恶者的巢穴之中，玩家们发现那猎人是詹德的女儿萨弗拉，而她从未请求，也从未要求过援助。
