# 面对黑暗领主 Facing Darklords

本章节中的每个黑暗领主都拥有自己的数据卡，这些数据卡来自《怪物手册》或者本书第五章，你可以直接取用，也可以把它修改一下，以更适用于你的冒险。虽然把黑暗领主变成一个难以抵御的强力威胁非常有诱惑力，但这样也可能会使玩家错过其丰富而多面的反派形象。黑暗领主往往不是领域内肉体最强大的生物，但作为黑暗领主的本质会使得他们非常难以被彻底击败。要战胜一位黑暗领主，角色们应该专注于秘密破坏黑暗领主阴谋，直击黑暗领主的软肋(对该话题更详细的论述在第二章的 
“领域的崩溃” 
部分），采取何种方式在何时与黑暗领主对决，应该由玩家们自己选择。归根结底，恐惧并不是由挑战等级或者玩家等级决定，而是由你在冒险中创造的悬念所决定。
