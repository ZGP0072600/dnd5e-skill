# 附录A：通灵板 Spirit Board

被称为“羽卫”（详见第三章）的灵媒们所使用的通灵板是降神中的焦点，请神的人们尝试通过它与别的世界联系。通灵板的使用者们一起把手放在小乩板上，依靠聚集在通灵板周围的人们的集体意志，与不可思议的力量交流。

与这块通灵板配套的小乩板 planchette 和通灵板的具体使用方法请翻阅第四章。将通灵板和小乩板影印下来，黏贴在木板或者硬纸板上，就可以为你的恐怖冒险做出一个神秘的道具了。

**插图：范·里希滕的鸦阁魔域指南：通灵板和小乩板**

本页的通灵板和小乩板可以在你氛围十足的冒险中充当小道具。

把它们打印出来，贴在硬质背板上。玩家们围坐在通灵板前，每个人都把手放在小乩板上，等待精魂（或者 DM）引导小乩板指向字母或神秘的符号，传达信息。

更多关于通灵板和小乩板的介绍请阅读《范·里希滕的鸦阁魔域指南》。
