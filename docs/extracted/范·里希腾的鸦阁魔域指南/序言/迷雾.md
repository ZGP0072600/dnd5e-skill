# 迷雾

## 迷雾 The Mist

在整个鸦阁魔域，迷雾都是黑暗力量意志的化身。它们分隔、包围、孤立每个领域，形成雾气重重的边界，将黑暗领主囚禁在他们的领土之内。迷雾也会从恐惧诸领域里溜出去，带走毫不知情的旅行者，或者将恐怖施加在毫无戒备的村民身上。最为臭名昭著的是，迷雾会偷偷爬进其他的世界，带走无辜的人、做坏事的人或者整片土地，把他们拖回鸦阁魔域，成为又一批新的囚犯。

恐惧诸领域的居民们知道迷雾的存在，也知道许多和它有关的凶险的故事，但他们也接受了迷雾是故乡环境里天然存在的一部分。许多在领域之间穿行的人们甚至把他们的世界叫做“迷雾之地”。任何超自然现象的发生、莫名其妙的失踪或者恶毒力量的出现都可以归咎于迷雾。任何与迷雾打交道的人或者来自迷雾之外的人都会受到怀疑。若是有人想要离开一个地方到另一个地方，又或者，想要逃离自己所在的国度，迷雾都会阻碍他们。

除了上述事实之外，你也可以运用迷雾来为你所叙述的恐怖故事划出边界。在第三章小节“鸦阁魔域的基本特性”中，详述了在你的恐怖冒险中如何使用迷雾收束玩家注意力，将不必要的东西隐藏起来，只引导角色们接触特定的要素。
