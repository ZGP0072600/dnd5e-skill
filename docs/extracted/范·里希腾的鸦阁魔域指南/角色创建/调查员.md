# 调查员

## 调查员 Investigator

你不屈不挠地寻找真相。也许你的动机是对法律的信仰和一种普世正义感，或者也许一条法律让你失望了，而你想拨乱反正。你可能目睹过一些不寻常或可怕的东西，现在你必须更多地了解这个隐藏的真相。或者你可能是个雇佣侦探，为高薪客户解开谜团。不论你卷入的神秘事件是本地罪案还是跨越国度的阴谋，你都会受到自身需要的驱使，去搜寻哪怕最难分辨的一丝线索，揭露隐藏在阴影中的秘密。

技能熟练项：从洞悉, 调查，或察觉中选择两项

工具熟练项：易容工具，盗贼工具

装备： 一个放大镜，过去案例中的证据(从后文中的恐怖饰品列表中选择或随机投掷一个)，一套普通服装，以及10GP

神秘之路 Path 
To Mystery

你的第一个案子影响了你感兴趣的神秘事件类型。为什么这个案子对你有如此深刻的影响，对你而言如此重要，或者给你带来了如此重大的创伤？除了你之外，它影响了谁？你如何又为何被牵扯进来？这个案子解决了吗？它是如何让你走上调查其他神秘事件的道路的？从第一个案子表中选择或投掷细节，以揭开你作为调查员职业生涯起始的谜团。

| 5 | 你被错误地指控并被判有罪。 你设法逃脱了，并图帮助其他人避免你遭受的经历，哪怕你现在仍然受到法律追捕。 |

| 8 | 你在一家调查当地执法部门无法解决的犯罪的事务所找到了一份工作。 你经常想哪个对你更重要，真相还是你的薪水。 |

特性：官方调查 Official 
Inquir

你在接触他人和进入场所以获取你所需要的信息方面饶有经验。通过三寸不烂之舌、毅力和官方文件的结合，你可以访问一个与你正在调查的犯罪相关的地方或个人。那些没有参与你的调查的人会替你传递请求或者避免阻碍你。此外，当地执法部门对你有着坚定的看法，认为你是一个讨厌鬼，或者是一个自己人。
