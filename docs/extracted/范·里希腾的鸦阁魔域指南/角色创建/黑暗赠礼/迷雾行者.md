# 迷雾行者

### 迷雾行者 Mist Walker

迷雾缠绕着那些踏入恐惧领域的人们，但是你知道如何从它的魔掌中逃脱。你可以在迷雾中成功地辨识方向，需要足够的时间和一点点运气，但是这份自由伴有代价；如果你在一片区域停留太久，迷雾将会吸走你的生命力。

掷骰或从迷雾穿梭表中选择一项，以此决定你是如何发展出这份黑暗赠礼的。另外，你将会获得与其相应的特性。

迷雾穿梭表 Misty 
Travels

d4 由来 Origin

1. 你得知了一个关于迷雾本质的秘密。从此以后，你可以操纵它们，但是你脚下的土地会与你作对。

2. 你为了逃离一个正在追猎你的人或事物，来到迷雾中。你不能休息太长时间，因为你知道你仍被追赶着。

3. 你属于一个组织，一个家族，或是一个巡回团体，他们拥有着穿梭于迷雾中的经验。他们也教会了你要怎么做。

4. 迷雾将你从你的家中窃出，并将关于你从何处来的记忆切断。从那之后，你的使命便是找到回去的路。

迷雾穿行者 Mist 
Traveler.当你进入迷雾中，意图抵达一片特定的领域时，你将会像是拥有一个与那个领域相符的迷雾护身符一般。当你使用这个特性时，你必须知道你所选择的目的地领域的名字，但是你并不需要曾抵达过那片土地。这个特性不会允许你绕过因为一位暗黑领主的意愿，而封闭的领域边境。

已被毒害的根源 Poisoned 
Roots.当你完成一次长休，你附近半径10尺的世界将会如同虹吸管一般，最终会吸干你的生命力。你可以安全地在一片区域停留与你体质调整值相同数值的周数（最少一周）。之后，每次你在那片区域完成一次长休时，你必须成功通过一次 DC 15的体质豁免检定，否则获得一级当你在那片区域内，不可被移除的力竭。
