# 领域的崩溃

## 领域的崩溃 The Domain's Downfall

一旦你确定了领域的黑暗领主，就要提前设想好你冒险的高潮部分。

不太可能只靠战斗来击败一位黑暗领主，那么，他的弱点是什么？在一两句话之内，说清楚这个弱点：它在领域中位置、黑暗领主如何隐瞒这个阿基里斯之踵（致命的弱点），以及冒险者们如何把它找出来，击败黑暗领主。

接下来，假设这个黑暗领主已经被击败，思考一下接下来会发生什么，以及这种情况如何与玩家角色相关联，考虑如下问题：

黑暗领主可以永久死亡吗？

在什么样的情况下他们会归来？

他们的领域怎么样了？

会有一个新的黑暗领主崛起吗？

这个领域会永久消亡吗？

冒险者们可以逃离这个领域吗？

冒险者们是返回家园还是去往另外一个领域？
