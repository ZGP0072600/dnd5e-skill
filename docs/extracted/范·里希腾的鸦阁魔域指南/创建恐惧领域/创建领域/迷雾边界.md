# 迷雾边界

## 迷雾边界 Misty Borders

用几句话描述一下鸦阁魔域的迷雾是如何在你的领域中运作的。这可能与第三章开头所详述的大致相同，或者它们也可能反映出黑暗领主的本性。请考虑以下问题:

- 
迷雾中出现了什么形状、声音和气味？

- 
迷雾是否以某种可预测的方式行事？

- 
该领域的人给迷雾赋予了什么故事？他们是否为它们赋予了个性？

- 
除了该领域的边界，迷雾还出现在哪里？

- 
黑暗领主是如何利用迷雾来关闭他们领地的边界的（详见第三章）？
