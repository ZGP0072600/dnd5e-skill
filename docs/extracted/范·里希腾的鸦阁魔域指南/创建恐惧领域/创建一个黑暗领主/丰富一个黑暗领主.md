# 丰富一个黑暗领主

## 丰富一个黑暗领主 Developing A Darklord

黑暗领主之所以是反派，出自他们的选择而非天生。当你完善你对黑暗领主的想法时，确定黑暗领主所做的事，谁受到了伤害，黑暗势力如何鼓励他们，以及黑暗领主付出的代价。描述他们为什么选择实施这些邪恶行为，包括他们的理想、缺陷和纽带。

在创作你的黑暗领主的背景故事时要考虑这些问题：

- 
黑暗领主选择实施的第一个堕落行为是什么，他们的理想是如何鼓励他们走上这条道路的？

- 
黑暗领主是否因其邪恶而受到奖励或庆祝？这种反应是否鼓励了更大的犯罪？这些奖赏是赢得的或正当的吗？

- 
黑暗领主是否重复或升级了他们的邪恶行为，以获得他们自私的愿望？

- 
未来的黑暗领主是否意识到他们失去了赎罪的希望，却选择了其他令人发指的行为以保持他们的缺陷？

- 
什么邪恶行为如此残暴，以至于未来的黑暗领主的朋友和家人抛弃了他们？黑暗领主认为这是有理由的还是不合理的判断？他们是如何反应的？

- 
这些行为是在什么时候以及如何引起黑暗力量的注意的？

- 
黑暗力量是如何利用这些行为来为黑暗领主打造完美的监狱领域的？
