# 本质败坏

## 本质败坏 Wicked Personality

详述黑暗领主的个性，给他们一个独特的身份，这将有助于在你的冒险中为他们树立动机。要做到这一点，要赋予他们理想、羁绊和缺陷，就像你可能赋予任何 NPC 的那样，详见《城主指南》中关于创建非玩家角色的章节。

理想 Ideal.用一句话来描述黑暗领主所珍视的、支配其更大行动的理想。

羁绊 Bond.用一句话来描述黑暗领主所渴望的人、地方或物体。避免仅仅是关于权力的纽带；权力本身并不是目的权力是获得纽带的一种手段。

缺陷 Flaws.用一句话来描述黑暗领主的个性是如何导致他们违背自己的最佳利益的，特别是在他们寻求赢得他们的羁绊和欲望的时候。缺点通常是一种负面的情绪或破坏性的行为比如恐惧、仇恨、不安全感、嫉妒、混乱、迷恋、自私、羞耻促使一种非理性的习惯导致黑暗领主或其他人受到伤害。或者，你可以从 "致命缺陷 "表中选择或随机确定一个缺陷。
