# 黑暗奇幻

## 黑暗奇幻 Dark Fantasy

> 
 黑暗奇幻既是一种幻想题材，又是一种恐怖题材。一般来说，任何既有超自然元素又有恐怖主题的故事都可能被认为是黑暗奇幻，无论是充满恐怖元素的奇幻故事，还是以超自然恐怖为特征的现实世界。黑暗奇幻是指在幻想世界中，冷酷的主题、虚无主义的情节或恐怖的元素构成了一个奇幻故事。邪恶支配着黑暗奇幻的设定，堕落是司空见惯的，生命毫无价值。

你的幻想有多黑暗由你自己决定，但要记住英雄的角色，确保黑暗奇幻领域中有光明的地方。如果一个领域没有希望，需要抵抗和有意义的情节也就会变得很少。

此外，在创造黑暗奇幻领域时，你需要考虑以下题材:

善不常胜。拥有强大权力和无可匹敌的计划的坏蛋可能是一种常态。

善恶之间的界限是模糊的。选择一个至少不是最坏的决定。

腐败和猜疑在组织和个人中盛行。

魔法和魔法物品可能比较稀有，或者需要讨价还价或牺牲才能使用。

反英雄十分常见被世界的邪恶所玷污的角色或者是那些不愿被视作英雄的人。

黑暗奇幻怪物 Dark Fantasy Monsters

黑暗奇幻折磨 Dark Fantasy Torments

黑暗奇幻设定 Dark Fantasy Settings
