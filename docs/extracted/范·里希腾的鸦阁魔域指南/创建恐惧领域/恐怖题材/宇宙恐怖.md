# 宇宙恐怖

## 宇宙恐怖 Cosmic Horror

宇宙恐惧是围绕着个人渺小感的恐惧。这种题材基于异星实体的想法，这些想法如此庞大，如此真切，超越我们的理解，我们无法理解他们最简单的动机，看到它们就会迷失在它们的巨大之中，而这也就证明了我们过去、现在、将来都绝无可能对整个宇宙产生影响。

这一题材讲述了异星力量如何改变我们，扭曲我们对自我的期望和理解，贬低我们的思想，并将我们与使我们作为人类的本质分离。有时它是我们招致的后果。其他时候，它突然就发生了，只是一个环境的意外，我们只能希望能够活下来。

不管你怎么理解它，这类游戏都包含了控制力的丧失，自我的缺失，以及在一个冷漠的宇宙中所产生的渺小感。

此外，在创造你的宇宙恐怖领域时，考虑以下题材:

没有善与恶，没有秩序与混乱。

模糊不清。宇宙恐怖是难以名状的，要通过想象而不是细节呈现。

最好的情况是，宇宙恐怖让角色觉得自己逐渐脱离了熟悉的现实。

宇宙恐怖关于由人类无法理解的动机所驱动的不可言喻的力量。

邪教、禁书和怪诞的符号构成了宇宙恐怖的基石。

这种题材的题材有着将被边缘化的人口污名化为可怕而臭名昭著的精神病患的历史传统。你应当注意并避免这种特征*。

译注：*HP 洛夫克拉夫特歧视少数人种，喜欢把偏远地区的少数人种描写成畸形儿，精神病患或邪教徒，这种描写也影响了后世的克苏鲁神话的衍生作品。

宇宙恐怖怪物 Cosmic 
Horror Monsters

在宇宙恐怖冒险中表现出色的怪物会迫使他们不情愿的受害者产生改变。宇宙恐怖集中于未知的实体和生物，他们看到敌人的思想，并使用发现的思想对付敌人本身。任何怪物都可以轻易地被修改以适应这题材的冒险，它们也许是被幕后的智慧生物或巨大的宇宙力量所控制。

宇宙恐怖反派 Cosmic 
Horror Monsters

当宇宙的恐怖反派还是凡人时，他们作为可怜的生物承受了难以想象的恐怖，渴望着以一种他们无法恰当表达的方式得到解脱。除此之外的是可怕的生物，恐怖的衍生物或那些自认为自己也是如此的人。此外，以下这些征兆也是真正的宇宙恐怖：无法理解的存在，像神一样的恐怖，以及无法沟通的力量的化身，以及那些存在本身就令角色们憎恨的事物。

宇宙恐怖情节 Cosmic 
Horror Plots

在宇宙恐怖领域的冒险是暗淡，绝望的事情，其中能期盼的最好的事情就是相对完整的生存。 没有阻止究极邪恶的方法，但玩家可以努力暂时将其封闭起来。
