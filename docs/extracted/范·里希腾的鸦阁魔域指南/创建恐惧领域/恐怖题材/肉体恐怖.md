# 肉体恐怖

## 肉体恐怖 Body Horror

肉体恐怖作为一种题材审视了一种普遍的恐惧:我们对自身剖析的失败。我们很少思考皮肤下面发生了什么。我们只知道器官的运作是和谐的：心脏跳动，肺部呼吸，胃肠系统为我们提供营养。但我们却不会去思考细节。比如说，寄生虫是否会在我们的大脑中产下包囊，或者说我们可能正处于肝硬化的某个阶段，又或者说有一颗我们希望永远不会看到的肿瘤可能会在身体深处爆发。

当创造涉及肉体恐怖的冒险时，要利用这种对警惕的缺失。关注肉体的不可预测性。我们都知道，表面上的健康可能只是一种错觉，我们不是神圣的动物，我们可能正在孵化生命力旺盛的，饥饿的或恶毒的寄生虫。

此外，在创造你的肉体恐怖领域时，可以参考以下题材:

物质上的变化会很常见，并且可能影响的不仅仅是肉体，物体、建筑和自然世界可能会以可怕的方式获得人格。

角色们遭受异常或过度生长的折磨，无论是器官、不协调的附肢还是异常的身体。

尽管血腥经常成为肉体恐怖故事的中心，但它往往只是对于突变、分歧、死亡、身体伤害、疾病或其他恐惧的副产物。

肉体恐怖不需要太混乱。交换身体，变成石头，甚至某人获取完美形态形态都可能是这题材游戏的特征。

这种题材的故事一直以来都把残疾描绘成可怕的东西。你应该注意并避免使用这些特征。

肉体恐怖怪物 Body 
Horror Monsters

任何怪物都可能是肉体恐怖情节的焦点。先了解玩家的期望，这样你就可以曲解他们的期望。一个儒雅随和的精灵的真身是由一条智慧蠕虫扮演的躯壳，这和任何魔法的恐怖一样可怕。

各种怪物可以不用修改就可以使用。这包括所有形式的寄生野兽，异怪，亡灵，和其他具有异常消化系统或诡异繁殖循环的生物。
