# 哥特恐怖

## 哥特恐怖 Gothic Horror

哥特恐怖是自内心而不是外在的恐惧。它以一种尖锐的方式粉碎了人性的假象，它为我们举起一面镜子，说:看我们真实的样子，然后再看我们伪装的假象。在文明的面具下，是堕落。在薄薄的社会虚饰下，是邪恶。在所有复杂的外表下，我们不都是猎食者或猎物吗?哥特恐怖粉碎了我们自欺欺人的谎言，并表明了我们不是某个遥远的存在，而是，并且将永远是自己的不幸的缔造者。对完美的追求让我们发现自己的不完美。我们对神性的追求使我们相信我们自己就是神。这些都是在哥特恐怖故事中作祟的主题。

当你在创造哥特恐怖领域时可以考虑以下题材:

哥特式故事包含强烈的，甚至夸张的情感。爱情、竞争和改变人生的事件在这种冒险中如同家常便饭。

氛围和恐惧感是实现哥特感的关键。你的故事选址可以是一座破旧的公寓，毁坏的大教堂，或其他不祥的地方。

哥特式的英雄通常品德高尚，充满激情，或勇敢无畏。为冒险寻找测试角色们的信仰和道德的方法。

哥特式的反派无可救药的邪恶，但这一点不应该立刻揭露。在揭露反派们可怕的秘密之前先用一些细节暗示它们。

牺牲在哥特小说中非常突出。给予角色们痛心的选择。

哥特恐怖怪物 Gothic

哥特恐怖反派 Gothic 
Horror Villains

哥特恐怖故事中的反派在他们的真实本性被揭露之前都是狡猾或谦逊的。任何一个有着黑暗秘密的有趣人物都可以成为一个哥特恐怖反派，而残忍、恶德或崇善之敌的惊人本性被慢慢揭露的哥特式反派是最成功的。

哥特恐怖的折磨 Gothic
