# 诺斯费拉图

## 诺斯费拉图 Nosferatu

诺斯费拉图是邪恶的不死猎手，它们空有吸血鬼无尽的饥渴，却缺乏其风度翩翩。对它们而言，存在不过是一连串永恒不变，寒冷而绝望的夜晚，其中不时点缀着象征短暂温暖和清醒的血色斑点。这些欢愉转瞬即逝，因为它们的血瘾永远也无法平息。

诺斯费拉图以一切流淌着鲜血的生物为食。成堆的老鼠残尸和沦为屠宰场的马厩便是它们捕食最典型的初期迹象。当太阳打断它们的狩猎时，诺斯费拉图会撤退到它们偏爱的废墟、下水道或者洞穴中，毫不在意巢穴的舒适。它们不会隐居在墓穴里，而是寻找肮脏或难以接近的裂隙这些任何活物都会避开的地方作为自己的居所。

在进食后的几分钟，诺斯费拉图会恢复清醒的神志，能够思考下一顿饭意外的东西。在这转瞬即逝的时间里，诺斯费拉图能够回忆起它们是谁，它们知晓骄傲，知晓自己的目的，知晓阴影之外的世界。它们甚至可能会暂时被说服与其他生物交谈。然而，那些与诺斯费拉图交谈的人通常都会发现它们是自私、口是心非的生物，它们的记忆已经消散，而对生命基本的尊重也早已消失殆尽。

再生 Regeneration。诺斯费拉图在其自己的回合开始时恢复10点生命值，前提是它至少有1点生命值且并未处于日光中。如果其受到光耀伤害，则诺斯费拉图的该特质在其下一回合开始时无法生效。

蛛行 Spider Climb。诺斯费拉图无需进行属性检定就能攀爬于困难表面，包括倒挂在天花板上。

高度日光过敏 Sunlight 
Sensitivity。诺斯费拉图在日光照射下开始回合时会受到20光耀伤害。当处于日光中时，它的攻击检定和一切属性检定具有劣势。

非凡本质 Unusual 
Nature。诺斯费拉图不需要空气。

动作 Actions

多重攻击 Multiattack。诺斯费拉图进行两次爪击攻击，紧接着进行一次啃咬攻击。如果两次爪击都命中的是同一生物，则此次啃咬攻击具有优势。

爪击 Claw。近战武器攻击：命中+8，触及5尺，单一目标。

命中：9（1d8+5）挥砍伤害。

啃咬 Bite。近战武器攻击：命中+8，触及5尺，单一生物。

命中：9（1d8+5）穿刺伤害，外加7（2d6）暗蚀伤害。如果目标失去过生命值，改为受到额外11（2d10）暗蚀伤害。

目标的生命值上限降低受到的暗蚀伤害的数值，诺斯费拉图则回复等量的生命值。减损效应持续至目标完成一次长休。如果该攻击将目标的生命值上限降至0，则目标死亡。以该方式死亡并被葬入土中的类人生物，将在1d10日后复活为一只诺斯费拉图。

呕血 Blood Disgorge（充能5~6）。诺斯费拉图喷出血液，覆盖15尺锥状区域。区域内的每个生物都必须进行 DC 16的体质豁免。失败的情况下，该生物受到18（4d8）暗蚀伤害，并且1分钟无法恢复生命值。成功的情况下，该生物受到一半伤害并不受额外效果影响。
