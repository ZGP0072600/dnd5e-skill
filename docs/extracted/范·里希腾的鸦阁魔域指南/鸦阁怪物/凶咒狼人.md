# 凶咒狼人

## 凶咒狼人 Loup Garou

凶咒狼人拥有一种比普通狼人更强的兽化诅咒。除了比他们的狼人亲戚更致命之外，凶咒狼人还积极地传播狼人诅咒。那些被兽化症折磨的人只有在凶咒狼人死后才能摆脱他们的诅咒。

兽化症 Loup 
Garou Lycanthropy

一个类人生物如果屈服于凶咒狼人的兽化诅咒，就会变成一个狼人。这种形式的兽化症在施加诅咒的凶咒狼人活着的时候是无法消除的。关于兽化症的详情参见《怪物图鉴》。

一旦凶咒狼人被杀，在满月之夜对任何其创造的受诅咒折磨的狼人施展移除诅咒法术，会迫使目标做出 DC 17的体质豁免。如果成功，诅咒就会被移除，目标就会恢复正常形态，并获得3级力竭。如果失败，诅咒仍然存在，并且目标在1个月内为移除这个诅咒所做的任何豁免都会自动失败。

动作 Actions

多重攻击 Multiattack.凶咒狼人发动两次攻击:两次长剑攻击(类人形态)或者发动一次啃咬和一次爪击(恐狼形态或者半人形态).

啃咬 Bite(仅限恐狼形态或者半人形态).近战武器攻击:命中+9,触及5尺,一个目标。命中:13(2d8+4)穿刺伤害外加14(4d6)暗蚀伤害.如果目标是一个类人生物，则其必须成功通过一次 DC 17的体质豁免，否则将遭受凶咒狼人的兽化诅咒。

爪击 Claws(仅限恐狼形态或者半人形态).近战武器攻击:命中+9,触及5尺,一个目标。命中:11(2d6+4)挥砍伤害。如果目标是一个生物，则其必须成功通过一次 DC 17的力量豁免，否则应击倒地。

长剑 Longsword (仅限类人形态).近战武器攻击:命中+9,触及5尺,一个目标。命中:13 (2d8+4)挥砍伤害或者以双手发动时造成15(2d10+4)挥砍伤害。

Shape. 凶咒狼人变成大型的半人半狼的形态或者大型的恐狼形态，或者变回原形，类人生物形态。除了体型和速度以外，它的任何游戏数据在所有形态下都是相同的。任何凶咒狼人穿着或持用的装备不会随它变形。凶咒狼人死后变回原形。

啃咬 Bite (消耗3动作).凶咒狼人变形为半人形态或者恐狼形态，然后发动一次啃咬攻击.
