# 捕体植物

## 捕体植物 Bodytaker plant

无论是来自星辰还是从隐藏的深渊发芽，这种被称为捕体植物的恶毒植被，不论出现在何处，都试图成为占主宰地位的生命形式。这些入侵性有机体通过吞噬个体，并用被称为荚苗的复制品取代他们来颠覆整个社会。捕体植物认为自身是完美的有机体，并试图统治他们所生长的土地。在他们看来，世界在他们的控制下将会更健康更高效。任何持不同观点的人要么缺乏远见，要么只适合当肥料。

捕体植物的根很深，在土壤中散布着几乎看不见的细丝。如果这些纤维中的任何一些在植物被破坏后存活了下来，则捕体植物在几个月后会再生。盐化或毒化植物生长的土壤会破坏这些细丝，阻止他重新出现。

藤鞭 Vine Lash.近战武器攻击：命中+7，触及20尺，一个目标。命中：11（2d6+4）点挥砍伤害，如果目标是一个生物则它被擒抱（逃脱 DC 15），目标被束缚，直到擒抱结束。植物有四条藤蔓，每条能擒抱一个目标。

诱捕荚.近战武器攻击：命中+7，触及5尺，一个被植物擒抱的中型或更小的生物。命中：22（4d8+4）点强酸伤害，目标被拉入植物所占的空间并被包裹进荚内，擒抱结束。当被包裹时目标被束缚，它对于荚外的攻击和效应视为处于全身掩护。被包裹的目标还必须立即成功通过一个 DC 16的体质豁免否则被植物的消化酶震慑，直到它被从荚内移除或植物死亡。被包裹的生物不需要空气，每在荚中停留1小时就获得1级力竭。如果目标在被包裹时死亡，它会立即作为一个活的荚苗出现在荚中，着装或携带着原生物的所有装备。

作为一个动作项，通过一个成功的 DC 15的力量检定，一个捕体植物周围5尺在荚外的生物可以打开荚并把目标拉出来。如果植物死亡，目标则不再受束缚，并可消耗10尺移动从荚中逃离，陷入倒地。植物有一个荚，每次可以包裹一个生物。
