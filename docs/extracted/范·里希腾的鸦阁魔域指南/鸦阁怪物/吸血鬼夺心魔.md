# 吸血鬼夺心魔

## 吸血鬼夺心魔 Vampiric Mind Flayer

当血痕领 Bluespur（见第三章）的夺心魔无法解决它们主人的痛苦时，它们退化中的主脑为了防止痴呆和死亡，会改用更激进的手段。这种手段的结果就是这些从被吸血鬼感染的蝌蚪中生出的野蛮残暴生物，吸血鬼夺心魔。这些经过特化但有缺陷的恐怖生物只为了一个目的：从富有智慧的头脑中汲取脑液。完成以后，它们会回到血痕领的主脑处，在主脑的池中溶解，并在带有激素的盐水里释放它们盗取的精华。这些怪诞的镇痛剂可以减缓主脑的退化，但远远不可能治愈。

吸血鬼夺心魔在身体和心灵上都不稳定。这些恐怖的生物不会让其他人妨碍它们达成它们存在的目的。虽然它们拥有夺心魔的心灵感应能力，但它们的大脑并不是用来施展这些能力的。相反，它们会以本能的心灵状态来轰击附近的生物。虽然这些饥渴的生物看上去很可怕，但它们并不会比认为它们可憎的其他夺心魔让人更加不安。

Climb。夺心魔可以攀爬在难以攀爬的表面上，包括倒吊在天花板上，且不需要进行属性检定。

日照敏感 Sunlight 
Sensitivity。在日光照射下，夺心魔的攻击检定具有劣势，其依靠视力的感知（察觉）检定也具有劣势。

非凡本质 Unusual Nature。夺心魔不需要呼吸、进食和睡眠。

动作 Actions

多重攻击 Multiattack。夺心魔发动两次爪击或是一次爪击和一次触须攻击。

爪击 Claw。近战武器攻击：命中+7，触及5尺，一个目标。命中：8（1d8+4）点挥砍伤害加10（3d6）暗蚀伤害。

触须 Tencacles。近战武器攻击：命中+7，触及5尺，一个目标。命中：7（1d6+4）点穿刺伤害，若目标是一个生物则被擒抱（逃脱 DC 15）

汲取智慧 Drink Sapience。夺心魔以一个被它擒抱的生物作为目标。目标必须通过一次 DC 15的感知豁免，否则受到14（4d6）点心灵伤害并获得一级力竭。夺心魔恢复等同于此次造成心灵伤害数值的生命值。因受到此次心灵伤害导致生命值降为0的生物会死亡。
