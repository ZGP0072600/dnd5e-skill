# 乌尔米斯特审判官

## 乌尔米斯特审判官 UImist Inquisitors

“潜伏的邪恶无处不在。我们将用心灵发掘它，探测它的深处，并消灭它。”以这些话为准则，在精神力量方面天赋异禀的祭司乌尔梅德 Ulmed 建立了乌尔米斯特审判庭，这是一个灵能审判官的法庭，旨在发现隐藏在人们灵魂中的邪恶。

在施特拉德冯扎洛维奇伯爵成为第一个吸血鬼之前，施特拉德与乌尔梅德一同席卷了许多地方。他们的使命很明确:摧毁那些腐化了世界的邪恶势力，并确保这些力量永远不会再出现。施特拉德、乌尔梅德和他们的同伴们猎杀邪魔、亡灵、异怪以及其他超自然的威胁，他们是奥西巴斯祭司等邪教组织的死敌。当施特拉德堕入黑暗时，乌尔梅德对他朋友的转变感到心碎，并改变了审判庭的使命。它将不仅专注于猎杀怪物，还将猎杀能腐化人的邪恶种子。

乌尔梅德和他的朋友科西玛 Cosima、安塞尔 Ansel 和崔斯提安 Tristian 把审判庭组织分为三个分会，每个分会专门研究一种灵能力量 psionic 
power。柯西玛分会利用了心灵之火 Mind 
Fire 这名字代表了在每个人心中燃烧的思想之火。他们利用这种能力来解读思想，重塑记忆，并控制胆敢违抗的人。安塞尔分会的审判官们让自己接受严酷的禁欲主义，致力于使用灵能来赋予自己的身体力量。他们成功了，并成为了审判庭的军事武器，他们的象征是一把剑。最后，崔斯提安分会致力于利用智慧，通过心灵感应的力量来改变环境，分会的成员成为了审判庭的学者，他们的象征是一个本典籍。

如今，审判庭统治着巴洛维亚原址以北的一个巨大的城市马里坦 Malitain，审判庭派遣其成员穿越多元宇宙，试图阻止邪恶的邪教、异界的恐怖怪物和凡人的恶意。审判官对这项工作的热情使他们给许多地方带来了恐怖，那里的人们担心过分狂热的审判官可能会是一个和他们原本要追捕的邪魔一样可怕的怪物。

动作 Actions

多重攻击 Multiattack.审判官发动两次银长剑攻击或是使用两次心灵之火。

银长剑 Silver Longsword.近战武器攻击：命中+7，触及5尺，一个目标。命中：8（1d8+4）点挥砍伤害，或者以双手使用时造成9（1d10+4）挥砍伤害，附带18（4d8）力场伤害

心灵之火 Ming Fire.审判官以一个120尺之内可以看见的生物为目标。目标必须进行一次 DC 15的智力豁免，豁免失败则承受17（3d8+4）心灵伤害并被震慑，直到审判官的下一个回合开始。

天生施法（灵能）Innate 
Spellcasting（Psionics）.审判官可以施展以下法术，不需要任何成分，施法属性为魅力（法术豁免 DC 15）：

随意：秘法眼 arcane eye、安定心神 calm emotions、侦测魔法 detect magic、侦测思想 detect thoughts、解除魔法 dispel magic、短讯术 sending、暗示术 suggestion

每项1/日：群体暗示术 mass suggestion, 篡改记忆 modify memory

审判官命令 Inquisitor's Command（充能5~6）.审判官所选择的，在其60英尺范围内且能看到的每一个生物都必须进行一次 DC 
15感知豁免，豁免失败则在审判官的下一个回合开始前将被魅惑。在受魅惑目标的回合中，审判官可以用心灵感应控制目标的移动、动作或同时进行。当以这种方式控制时，目标只能采取攻击(由审判官选择目标)或疾走动作。
