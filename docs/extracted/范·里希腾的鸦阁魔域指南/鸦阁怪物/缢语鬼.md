# 缢语鬼

## 缢语鬼 Gallows Speaker

缢语鬼源于集体死亡之所或是生物频发厄运之地。暮去朝来，那痛苦缠身并阴魂不散的魂灵结合成深谙林林总总死因的实体。这一纠缠在一起的灵魂被它们共同的苦楚所折磨，当他们侵扰活物之际在喋喋不休地埋怨着支离破碎的执念。由于通晓无数死亡，缢语鬼可以预测灾祸，预见袭向他们的厄运并用数以万计惨不忍睹的死亡景象来压垮它们的敌人。

缢语鬼很少言语连贯或与活物交流，取而代之的是它们沉溺在死亡的记忆中不可自拔。这些死灵喃喃自语，诉说着最后的咒怨，遗憾和悔恨。那些徘徊在缢语鬼周围聆听它们的人可以深入了解到它千万死亡中的任一一种。

缢语鬼 Gallows Speaker

中型亡灵

AC：12

HP：85（19d8）

速度：0，飞行40尺（悬浮）

力量8（-1），敏捷14（+2），体质10（0）

智力10（0），感知12（+1），魅力18（+4）

豁免检定：感知+4

技能：察觉+7

伤害抗性：非魔法的挥砍，穿刺和钝击

伤害免疫：暗蚀，毒素

状态免疫：魅惑，力竭，恐慌，受擒，麻痹，石化，中毒，倒地，束缚

感官：真实视觉60尺，被动察觉 17

语言：缢语鬼生前知道的任何语言

挑战等级：6（2300 
XP）熟练值+3

预知感官 Divination Senses.缢语鬼在物质位面可以看到60尺内的以太界，反之亦然。

虚体移动 Incorporeal Movement.缢语鬼可以穿过其他的生物和物体，它们在被穿过时视为困难地形，如果它回合结束时处于物体内，则受到5（1d10）力场伤害并视为困难地形。

非凡本质 Unusual 
Nature.缢语鬼不需要空气，食物，饮水和睡眠。

动作 Actions

凶兆触碰 Foretelling Touch.近战法术攻击：命中+7，触及5尺，一个生物，伤害：15（2d10+4）心灵伤害，并且目标必须骰一个 D4，在缢语鬼下回合开始之前，目标的下一次攻击或者豁免检定减去所骰出的数值。

悲恸的回声 Suffering Echoes.缢语鬼选择一个在它30尺以内可见的生物。该目标必须进行一次 DC 15的感知豁免。如果失败，目标受到19（3d12）心灵伤害并且目标迸发出痛苦的回忆涌向被缢语鬼选中离目标30尺内的另外三个生物，每个生物受到13（3d8）心灵伤害。
