# 尸变僵尸

## 僵尸 Jiangshi

当一个灵魂被困在它的尸体里时，痛苦可以使它的尸体复活，僵尸就是这样诞生的。这些渴望复仇的死者潜伏接近他们的后代和他们生前所熟知的社区，为那造成了他们被诅咒的复活的怠慢疏忽的葬礼而散播恐惧、带来报应。众所周知，死后僵直折磨着僵尸的四肢，使他们僵硬地伸直双臂，迈着僵硬的步子行走。这一点以及他们的飞行能力，使得许多人称他们跳跃吸血鬼。

僵尸在白天潜伏于他们的坟墓和隐藏的废墟中，以此躲避活人的注意。他们在晚上出现以吸取其他生物的生命，这些生命能量维持着它们异常的存在，赋予它们更大的力量。被僵尸杀死的类人生物会变成渴望生命的活尸，如果它们吃了活人的生命能量，它们自己也可能变成僵尸。

僵尸弱点 jiangshi 
Weaknesses。僵尸具有如下的缺陷:

恐惧倒影。如果僵尸可以看到自己的倒影，如果可以的话，它会立即使用其反应，尽可能地朝远离倒影的方向移动。

圣徽敏感。在僵尸装备或触碰一个圣徽时，它对抗驱散亡灵效应的豁免检定自动失败。

非凡本质 Unusual 
Nature。僵尸不需要呼吸

动作 Actions

多重攻击 Multiattack。僵尸发动三次猛击并使用一次吸收精气。

猛击 
Slam。近战武器攻击:命中+8，触及5尺,一个目标。命中:13点（2d8+4）钝击伤害。

吸收精气 Consume 
Energy。僵尸从30尺内一个它能看到的生物身上吸取精气。目标进行一次 DC 16的体质豁免，失败则承受18点（4d8）暗蚀伤害，成功则伤害减半。僵尸恢复相当于造成的暗蚀伤害数值的生命值。以此种方式恢复生命值后，在接下来七天里僵尸获得如下好处：它的步行速度提升至40尺，并且获得相当于步行速度的飞行速度并可以悬浮。

一个被此暗蚀伤害杀死的类人生物会在僵尸的回合结束时复活为一个尸妖（详见《怪物图鉴》）。尸妖使用僵尸的先攻，并且在僵尸行动后立即行动。如果这个尸妖用吸取生命杀死了一个类人生物，那么这个尸妖在五天后会变成一个僵尸。

变形 Change 
Shape。僵尸变形成一个中型或小型的野兽、类人或亡灵，或变回其真实形态。它的数据，除了体型以外，在所有形态下都相同。它穿戴或携带的任何装备都会被新形态吸收或携带（由僵尸选择）。它死亡时会恢复真实形态。
