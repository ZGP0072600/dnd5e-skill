# 无情杀手

## 无情杀手 Relentless Killer

无情杀手是可恨的痴迷复仇的生物，会与邪魔或其他邪恶实体达成协议。他们的协议会把他们变成恶毒的屠夫，他们的存在只是为了放纵他们无尽的嗜血欲。尽管有些无情杀手最开始也是个无辜者，但他们的任何过去的外表都会在血腥和愤怒的浪潮中被冲走。这些杀手的恐怖行径很快就成为了传说，使那些无辜者们产生了超越地域和岁月的恐惧。

创建无情杀手 Creating 
A Relentless Killer

无情杀手的诞生和疯狂行凶的原因有许多。在创建一个无情杀手的时候，考虑是何种境况导致了它们的转变和发展成某种类型。依照无情杀手起源表和无情杀手类型表掷骰，或在表中自行选择，以创建你独一无二的恐怖屠杀者。

无情杀手起源表

| d6 | 起源 |

| 2 | 它的家人与邪恶的力量做了交易，将它转化为复仇的武器。 |
| 3 | 它出生之前就被一个邪魔打上了标记，岁月流逝，它邪恶的天性逐渐显露出来。 |
| 4 | 一个邪恶的地方或者魔法遗物渗透了它的身体，将它转变为一个怪物。 |
| 5 | 无情杀手用自己的身体容纳了一个邪恶的实体。 |
| 6 | 无情杀手看起来像是普通人，并不知道自己在暴怒时会变为无情杀手。 |

无情主宰 Relentless Juggernaut

无情主宰是渴望屠杀的巨大暴君。他们的存在使他们周围的世界都变得扭曲，允许他们创造可以屠杀猎物的武器。锋利的铁栅栏、碾碎的石笋和玻璃刀刃都很方便地出现，以协助主宰的暴行。每个主宰都把某一地区视为其的领地，并摧毁所有侵入者。
