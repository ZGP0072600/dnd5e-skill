# 无骨者

## 无骨者 Boneless

并非所有生灵的尸体都会从坟墓里蹒跚而出。无骨者是没有骨骼的亡灵遗骸。大多数是从那些遭受过临终酷刑的人身体中产生的，比如蓄意剥皮或碾压。不死的怨念渗入遗骸，他们的外皮在追求复仇或邪恶主人的恶念下翻滚和滑动。这些鬼祟的亡灵从缝隙和门缝下滑过，试图再次附上活物的骨架。将自身包裹在受害者身上，用全身的力量将他们拧死。

无骨者产生的形式多种多样。虽然特定生物的活体皮肤是最常见的，但邪恶的施法者可能从失败实验的残屑、死灵的泥浆、一堆丢弃的头发、屠宰场、停尸间里的混合物中创造出这些恐怖之物。这些起源并不影响无骨者的数据，但赋予它独特的外形。无论是出于意外还是邪恶的天才，一些反派用一具尸体创造出了两个不同的亡灵。无骨者可能会附在其他亡灵的骨架上，比如骷髅 skeletons 或丧尸 zombie。一个无骨者从他那独立的亡灵骨架上剥离自身的景象是许多经验丰富的怪物猎手们经常做的噩梦。

速度30

力量16（+3）敏捷14（+2）体质15（+2）

智力1（-5）感知10（+0）魅力1（-5）

技能 隐匿+4

伤害抗性钝击；毒素

状态免疫魅惑、力竭、恐慌

感官黑暗视觉60尺，被动察觉10

语言理解生前掌握的语言但不会说

挑战等级1（200XP）熟练加值+2

压缩 Compression.无骨者可以通过任何至少1寸宽的狭缝而无需跻身。他也能进入一个微型生物能够进入的空间。

非凡本质 Unusual Nature.无骨者不需要空气、饮食和睡眠。

动作 Actions

多重攻击 Multiattack.无骨者发动两次猛击攻击。如果两次攻击都命中同一个大型或更小的生物，该生物被擒抱（逃脱 DC 13），且无骨者可以使用碾压拥抱。

猛击 Slam.近战武器攻击：命中+5，触及5尺，一个目标。命中：5（1d4+3）点钝击伤害。

碾压拥抱 Crushing Embrace.无骨者用他的身体包裹住一个被他擒抱的大型或更小型生物。当被无骨者贴上身时，目标处于目盲并且无法呼吸。目标必须在无骨者的每个回合开始时通过一个成功的 DC 13的力量豁免否则受到5（1d4+3）点钝击伤害。如果有什么东西移动了目标无骨者也会跟着它移动。无骨者可以花费5尺移动力来将自己脱离。一个生物，包括目标可以用一个动作尝试分离无骨者，并迫使其移动到最近的未被占据的空间，这样做需要一个成功的 DC 13的力量检定。当无骨者死亡，他脱离任何他贴住的生物。
