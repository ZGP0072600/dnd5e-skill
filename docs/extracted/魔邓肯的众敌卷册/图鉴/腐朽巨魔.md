# 腐朽巨魔

腐朽巨魔 Rot Troll

当一个巨魔在再生时被注入一股暗蚀能量时，会发展为一种与暗蚀能量的共生关系，巨魔的身体会萎缩，它的肉体会以一种很快的形式的从身上剥离出来。最终一个腐朽巨魔会变得无法再生，尽管它还是会像正常一样愈合。这些生物会不时在它们腐烂的躯体中流出暗蚀能量。仅仅站在腐朽巨魔的旁边也会暴露在它那致命的辐射之中。

**腐朽巨魔 Rot Troll**

*大型巨人，混乱邪恶*AC 16（天生护甲）

HP 138(12d10+72)

速度 30尺

力量18（+4）敏捷13（+1）体质22（+6）

智力5（-3）感知8（-1）魅力4（-3）

技能：察觉+3

伤害免疫：暗蚀

感官：黑暗视觉60尺，被动察觉13

语言：巨人语

挑战等级：9（5000XP）

**腐臭变质 Rancid Degeneration。** 巨魔的每个回合结束时，它周围5尺的每一个生物收到11（2d10）点暗蚀伤害，除非巨魔从它上个回合结束的时间内曾受到强酸或火焰伤害。

动作

**多重攻击 Multiattack。** 巨魔发动三次攻击：一次啃咬和两次爪击

**啃咬 Bite。** 近战武器攻击：命中+8，触及5 尺，单一目标。命中：7 
（1d6+4）的穿刺伤害，外加16（3d10）的暗蚀伤害。

**爪击 Claw。** 近战武器攻击：命中+8，触及5 尺，单一目标。命中：11 
（2d6+4）的挥砍伤害，外加5（d10）的暗蚀伤害。
