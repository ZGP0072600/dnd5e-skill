# 吉斯洋基人 Githyanki

吉斯洋基吉什精锐 Githyanki 
Gish 敏锐的头脑和灵能天赋使得部分吉斯洋基人掌握了魔法。吉什精锐通过训练将他们的魔法能力和剑术技巧结合起来，成为了战场上的致命敌人。他们的特殊能力还使得他们尤为适合暗杀、突袭和间谍活动。
