# 安祖魔 Amnizu

安祖魔带领地狱军团进行战斗，并负责指挥守卫通往九狱之通路。安祖魔傲慢，恃强凌弱且冷酷无情，但它们也是非常聪明的战术家并具备着不渝的忠诚——这些品质正是地狱的大魔鬼们所看重的。

**冥河守卫 Guarding the River Styx.** 
一些安祖魔执行的关键任务是在流经迪斯和斯提吉亚冥河枯萎沿岸的高塔和堡垒守卫冥河。以劣魔的形式到达的灵魂没有个性或记忆；他们只是被魔鬼们驱使去作恶。巡逻的安祖魔将九狱的法则灌入新来者的可怜的大脑中，并将劣魔编成军团。
