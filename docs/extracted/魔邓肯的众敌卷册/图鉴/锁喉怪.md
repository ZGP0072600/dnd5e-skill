# 锁喉怪 Choker

别看锁喉怪长了一副瘦小的身躯，那触手般的四肢表明它其实是一种非常危险的地下掠食者。

锁喉怪的骨骼多为软骨。这种灵活的骨骼结构使它们能够轻松钻进巢穴岩壁上狭窄的裂缝和裂口。它们悄无声息潜伏在这些地方，等待着猎物的出现。

**阴险狡诈 Sly Trappers。** 锁喉怪常常将其最新的猎物摆放在埋伏点外边以作诱饵。它会时不时地撕下一两块肉充饥。以此同时，尸体还能将那些好奇的类人生物引诱到它触手可及的地方，比如探险家、卓尔、矮人，或是锁喉怪最喜爱的猎物，地精。

当猎物出现时，锁喉怪那海星状的触手将从躲藏处突然伸出，扼住受害者的喉咙，并将这个倒霉蛋束缚在岩壁上，然后弄死。它的手臂很长，因此它能深藏在裂缝之中，大多数普通的武器都鞭长莫及。

**独行猎手 Lone Hunters。** 锁喉怪倾向于独自狩猎，而不是协同合作，但是只要你发现了一只，更多的同伴可能就在附近。它们通过怪异的嚎叫在深邃的岩洞中交流，而我们却很难在一个典型的复杂洞穴中以此辨别或定位它们的所在。
