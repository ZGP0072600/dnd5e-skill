# 剑下鬼（剑邪灵） Sword Wraith

当一个痴迷于荣耀的战士在战斗中死去，而没有得到它拼命追求的荣誉时，它那勇猛而饥渴的灵魂可能会化作剑下鬼在战场上游荡。

徘徊灵魂。最有可能遭遇剑下鬼的地方是古代伏击之地，士兵被魔法击倒而无可反击之战场，和被敌人无情围杀之所。

**荣誉高于一切。** 如果有人怀疑他们的英勇，剑下鬼就会暴跳如雷。相反地，他们很容易被赞扬所安抚。对他们来说，没有什么比听到一首赞颂他们荣誉的抒情歌谣更让他们高兴的了。位于古战场附近的城镇每年举行一次纪念活动，以保持那里的剑下鬼平静。

**亡灵本质。** 剑下鬼不需要空气、饮食或睡眠。
