# 布雷祖魔 Bulezau

有着病态动物愤怒的外表，布雷祖魔体现了自然的暴力。在深渊的各处，布雷祖魔潜伏在深谷和高耸的峭壁上，它们中的多数在恶魔领主的军队中找到了一席之地。

**杀戮欲望：** 布雷祖魔崇尚暴力，它们渴望杀戮和死亡，这些特性使得它们成为众多恶魔领主随行军队的主要成员。当没有被更强大的恶魔统御时，这些布雷祖魔就会聚集在一起互相，相互争夺，相互摔跤和争斗，直到有一个更好的目标出现或是有更强大的恶魔把它们变成随从。

**可憎外表：** 布雷祖魔看似被令人讨厌的疾病困扰着：这个怪物有着粗糙的眼睛，驱虫在裂开的疮上扭动着，并且无论它们走到哪里，它们都带着一股腐败肉的恶臭。
