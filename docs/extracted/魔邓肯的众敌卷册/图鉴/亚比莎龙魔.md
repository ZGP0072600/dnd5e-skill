# 亚比莎龙魔

**亚比莎龙魔 Abishai**

每个亚比莎龙魔都曾经是一个在生前以某种方式赢得提亚马特青睐的凡人，并且，作为奖励，在他们死后灵魂被转化为可怖的魔鬼在九狱之中为她服务。

**灾厄使者 Emissaries of Doom。** 提亚马特将亚比莎龙魔们作为使者，派它们来代表她对九狱之中和多元宇宙的关注。其中一些亚比莎龙魔被派遣了简单的任务，比如向邪教徒们传递一则信息或者安排崇拜者们执行一些敏感的任务。

其他一些则具有更重大的使命，比如领导大型集团，暗杀目标，以及在军队中服务。在任何情况下，亚比莎龙魔们都狂热地效忠于提亚马特，并随时准备着在必要时献出自己的生命。

**九狱的局外人 Outsiders in Hell。** 亚比莎龙魔们处于九狱通常的阶级体系之外，有它们自己的指挥链并最终向提亚马特（以及阿斯蒙蒂尔斯，当黑暗领主选择使用它们的时候）答复。其他大魔鬼们能够命令亚比莎龙魔们为他们工作，但他们很少会这样作，因为难以辨别一个亚比莎龙魔究竟是在执行提亚马特还是阿斯蒙蒂尔斯的命令。撤销提亚马特发出的指令伴随有一定的隐藏风险，但干扰阿斯蒙蒂尔斯的计划则必然会招致灭顶之灾。
