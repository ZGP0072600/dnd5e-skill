# 冰鲵 Frost Salamander

**火蜥蜴，但是冰霜版 Salamander, Frost。** 冰鲵是坐落于气之位面与风之位面交界处的冰之位面，也被称作霜落之地的原住民。冰鲵尤其喜欢狩猎温血的生物。他们有时会碰巧通过位面间的通道迁徙至物质位面的高寒地带。

**热量吞噬者 Devourers of Heat。** 冰鲵的对一切热量的攻击性食欲驱使它们攻击任何遇到的聚落。它们也许会把其他掠食者会避开的炉中火焰或一处营火错当做成是一顿美味的大餐。火矮人们有时会冒险进入霜落之地，使用巨大的火堆设下陷阱引诱冰鲵并杀掉它们，收集其牙齿和皮革用作打造武器与护甲。

**虚假庇护所 False Refuge。** 尽管冰鲵能够在不牢靠的固体物质中挖掘，它们更偏好在冰里挖洞。它们在碎冰中打滚，以背部摩擦冰屑。这一习惯使它们能在从巢穴壁上挖掘新的冰块时，创造出越发巨大的冰窟网道。

在巢穴中居住一段时间的冰鲵能开辟出足够的空间，让一支小部队在里面扎营。在有经验的旅行者中会把这样的洞穴当做是一个受欢迎的庇护所，尽管它们并非如此。冰鲵会贪婪地吞食任何足够大胆地尝试在它们的巢穴睡觉猎物。

在少见的情况下，霜巨人会捕捉并训练这些生物，使用它们挖掘冰洞来创造前哨和堡垒。
