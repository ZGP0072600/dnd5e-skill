# 人类

**人类 ** 人类在拉尼卡诸多种族中仅占少数，他们支配着一些公会，在其他公会中则并不突出。与其他世界中一样，人类是革新者、成功者和先驱者，他们的勇气和 
适应力使他们能成就伟大的事业——当然也有惨重的失败。

尽管欧佐夫集团鬼影议会源自人类，但十公会的元祖中没有一个是人类。同样的，目前没有一个公会会长是人类。不过人类在许多公会中担任副官，顾问或策士。他们的野心和干劲驱使着他们上进，但是远古龙，斯芬克斯和恶魔等生物的纯粹力量使人类远离了权力的顶峰。

拉尼卡的人类与其他世界的人类在身体上并没有多大区别。同样的，他们的倾向也多种多样：他们是唯一一个在所有公会中都存在的种族。

**人类的名字 **

一种特殊的人类语言战胜了所有其他语言，成为拉尼卡的通用语，它的音素和传统塑造了拉尼卡的大多数人类的名字。人类有时会借用其他种族的名字，但是他们有丰富的传统名字可供借鉴。

姓氏的使用似乎是一种特殊的人类习俗。姓氏是从父母双方传给子女的，有时孩子使用父母双方的姓氏。例如，米歇尔·萨沃德·祖尼奇（Micyl 
Savod Zunich）是范·祖尼奇（Fann Zunich）和婕德·瓦德·萨沃德（Jarad Vod 
Savod）的儿子。当其他种族使用姓氏时，他们通常来源于与人类家庭的某种联系（不一定是血缘关系）。
