# 创作冒险

第四章：创作冒险本章为地下城主们提供了相关工具，以便创造以拉尼卡公会为特色的冒险。各公会间的紧张关系，永恒的敌意与纷争，为创作一段冒险旅途提供了坚实的基础。本章末尾的范例冒险模组《追捕克伦可》KRENKO’S WAY ，可以用来进行一场拉尼卡的战役。本章的表格和建议进一步阐述了《城主指南》（Dungeon Masters Guide）第三章的内容。表格中的怪物和非玩家角色 NPC 可以在《怪物手册》（Monster Manual）或本书的第六章中找到。如果一个生物的名称以粗体标示，则该生物的数据表格你可以在怪物手册中找到。如果其数据表格出现在其他地方，文中会另行告知。以公会为基础的冒险每个公会都会在这个章节中得到一个小章节来帮助你设计由公会引发的冒险。在公会章节中，会提供四个创作冒险的基本起点：公会的地点、公会的反派、角色的目标和冒险引子。公会的地点每个公会章节中的条目都包含一个与该公会相关的标志性位置的地图。这些地点中的任何一处都可以作为一场冒险的场所。每个公会地点的描述中都包含了一个表格，上面列出了基于公会冒险活动的可能目标。你可以从你所选地点的描述表中选择一个冒险目标，或者随机地骰一个。如果这些目标都不能激发你的灵感，那就选择一个目标，或者从公会地点目标表中随机选择一个。公会地点目标表 d8 冒险目标1 阻止此地的一场不可思议的大灾难2 查找该地点或附近的奇怪事件的来源3 内部发生灾难时逃离此地（并协助他人逃走）4 平息内部或周围的骚乱5 杀死或驱赶一个已经把该地点变为巢穴的怪物6 坚守此地抵御外界的攻击者7 抢占该地点的控制权8 渗透该地区并报告其内部发生的事情 关联的地点你可以创建一个关联了两个地点的冒险，地点关联表提供了关于如何引导玩家从一个冒险地点到另一个冒险地点的概括性建议。地点关联表 d6 冒险关联1 将人、物品或在第一地点收集的信息安全地转送到第二地点。2 从第二个地点内的第一个地点提取某物以扣押第二个地点的某人。3 在第二个地点做同样的事情，就像在第一个地点做的一样。4 发现在第一个地点所遭遇阻碍的源头，它位于第二个地点。5 跟随逃跑的敌人或追踪可疑人物，从第一个地点来到第二个地点。6 通过调查第二个地点，获得了第一个地点的信息。公会的反派玩家角色可以隶属于任何公会，他们的对手当然也可以。本章中的行会条目提供了与每个行会相关，形形色色的反派以及恶毒阴谋的例子。每个公会中的反派都有一些特定的目标和计划。一些常见的邪恶计划出现在公会反派表上。你也可以使用《城主指南》第四章中反派的计划与反派的手段表来充实反派的活动，本章中公会部分中的相关表格，也描述了该公会的反派们如何制定他们的计划。公会反派表 d6 反派活动1 一个反派企图诋毁或诋毁一个有权势的人。2 一个反派在自己的行会里争取权力。3 一个反派已经在他们的公会中获得了权力，需要被取代。4 反派的密探渗透一个或多个公会5 反派的密探诱离另一个公会的成员6 反派的密探正在破坏另一公会的运作角色的目标冒险的动机可能来自玩家角色所处的公会，而非来自外部公会。本章中的每个公会条目都提供了公会可能派其成员执行的一些任务。这些任务也可以作为支线任务，要求角色在完成涉及全团队的主线任务时顺带完成。公会角色目标 d8 冒险目标1 寻找一件对公会十分重要的物品2 从公会成员中获取信息3 在灾难中营救或保护一个公会成员4 把一个公会成员从囚禁中解救出来5 找到失踪的公会成员6 杀死或俘虏公会的敌人7 帮助一个因触犯法律而遇到麻烦的公会成员8 招募某人加入公会冒险引子每个公会的部分都包含了一个冒险引子列表，这些引子可以引发冒险，但不一定涉及特定的地点或者反派。它们提供了可能导致冒险事件的通常描述。选择一个公会以下是为你的冒险选择公会的主要方法：1
