# 思想编码

### 思想编码｜Encode Thoughts

*惑控 戏法***施法时间：**1 动作

** 施法距离：**自身

** 法术成分：**S

** 持续时间：**至多8小时

 将一根手指放在你的头上，然后将一段记忆、想法或信息从你的心灵中拖出来转变为一段有形的由耀眼能量组成的链。这名为“思想链”的东西将一直存在直到持续时间结束或者到你再次施展这个法术为止。思想链在距离你5英尺范围的无人占据的空间内将表现为一个可以像丝带一样被你持握或携带的，微小的、极轻的、半固态的物体。除此之外，思想链都是静止的。如果你在施展这个法术时正在专注于一个允许你读取或操纵他人思想的法术或能力时（例如侦测思想 Detect Thoughts 或篡改记忆 Modify Memory）。你可以将你读取到的思想和记忆转化为思想链，而不是转化你自己的。

在你持握着一个思维链的时候施放这个法术将使你立刻接收到思维链里所包含的任何记忆、想法或信息。（向思想链施展侦测思想 Detect 
Thoughts 也具有同样效果）
