# 公会生物

公会生物这一部分的内容是讲述在怪物手册中的那些与拉尼卡公会相关的那些生物。每个公会的条目中都包含了一个关于怪物手册中怪物的简短讨论，可以在公会的成员中找到，它包括一个表格，其中的怪物按照挑战等级排序显示。大部分的生物你都可以再本书或者怪物手册中找到，但是其中有一些生物则在瓦罗怪物指南，或者魔邓肯的众敌卷册（光头书）中提到。

 拉铎司 GGR 析米克联合生物析米克联合的科学家们对所有生命形式如何适应其周遭环境变化的方式都很感兴趣，但他们对这里最大的生物有着格外浓厚的兴趣。在这样一个城市建筑覆盖的世界里，大多数这些生物在城市上空翱翔，或在地下海洋深处游弋，只有通过析米克联合的索诺特们才能够进入底栖生物析米克科学家使用术语“底栖（benthid）”来指代那些生活在深海的巨兽。这些生物无论是在大小和形态上都有很大的差异，所以它们中的代表性生物用统计数据表明有巨章鱼、蛇颈龙、虎鲸、巨鲨、龙龟或克拉肯海怪。人鱼拉尼卡的人鱼是析米克联合索独有的生物。他们在不到一个世纪前从位面深埋的海洋中出现，并立即在公会中担任领导职务。他们声称与拉尼卡的原始海洋有着十分深厚的联系，这些海洋几乎没有受到拉尼卡文明的影响。他们在怪物手册中使用人鱼的统计数据，除了他们说通用语和人鱼语，还有他们拥有双腿从而给他们带来30英尺的行走速度。挑战等级 生物 来源0

 克拉肯 MM 瑟雷尼亚盟会生物瑟雷尼亚盟会利用其与自然的密切联系，招募各种生物参与他们的事业。这些盟会的支持者在自然秩序中比起生物更像人类一搬与成员一起占据了应有的位置。元素当瑟雷尼亚盟会陷入纷争之时，元素生物会挺身而出担任公会的防御。这些元素大部分是由树木的根和枝干构成的，使它们相当于被唤醒的灌木或被唤醒的树木。大地元素将建筑物的材料与它们的生存形式结合起来，当作装甲和武器抗击敌人。植物生物许多赛雷尼亚联盟的建筑、寺庙和小径都受到树人或食肉植物的保护，包括枯萎怪和蔓生怪。有些生物的手臂像藤蔓一样，诱捕猎物，然后把它丢进如同瓶子一般充满酸液的身体里。另一些则被粘胶覆盖，粘胶会固定住并慢慢地消化那些倒霉的受害者。瑟雷尼亚生物挑战等级 生物 来源0

葛加理群落生物戴卡林精灵（卓尔）、致命的美杜莎和如同昆虫般的刻洛人是葛加理群落中最重要的三种力量，它们争相控制大量数量较小（或至少不那么野心勃勃）的生物。故军（Erstwhile）当刻洛人的死亡牧师马泽雷克发现了一个古老的陵墓时，葛加理群落的力量平衡发生了重大变化。在城市的深处，在已经建立起来的文明层次之下。千百年来，马泽雷克发现了一个隐藏的地下室网络，称为“乌美里列”，一个巨大的结构，主导一整个城市街区。里面有数百具保存完好的尸体，其中充满了潜在的亡灵力量被马泽雷克激活，使尸体复活归来成为了混乱的生命。这个新的不死族被称为故军（相当于怪物手册中的尸妖），在他们还活着的年代，故军是拥有巨大财富和权力的贵族精灵。他们死后仍然穿着华丽的服饰，尽管它已经发霉腐朽了，但是他们的存在给地下城的葛加理群落带来了一种高等文化的氛围。真菌（Fungus）葛加理群落所在的地区到处都是由来自公会腐烂农场的真菌和碎屑组成的生物。尖叫菌在葛加理的隧道中充当着哨兵的角色，紫色真菌从闯入者的尸体中发芽成长。在地下城充满孢子的环境中，没有固定形态的死亡和腐烂的物质渐渐变化成一头头蔓生怪。一些葛加理群落生物是真菌和生物的共生融合体。葛加理巨魔居住在一个巨大的下水道中，他们的身体被像鬃毛一样的真菌装饰着，为他们已经足够强壮的身躯起到了额外的作用。帮助他们隐藏起来并促进它们的再生。昆虫和蜘蛛类生物甲虫、蜈蚣、蜘蛛和无数其他昆虫和蜘蛛类生物寄生居住在拥挤的葛加理群落中。这些通常与腐烂和重生有关的生物，被公会格外珍视。巨大的甲虫可以充当负重的野兽、同伴、坐骑和守卫者。在这些生物中最大的是死桥巨人甲虫（Deadbridge 
goliath beetle）（相当于怪物手册中的三角龙）。食人鱼甲虫（piranha 
beetles）（典型的昆虫集群，来自《怪物手册》），它们在松软的土地上挖洞，聚集成一个集群，覆盖它们的受害者，并从他们身上剥下新鲜的血肉。线球蜘蛛（skein 
spider）（使用怪物手册中巨型蜘蛛的数据）是葛加理回收工作的重要组成部分：真菌沿着蛛丝网快速生长，扩展他们的范围直到到吞没建筑，加速了葛加理群落的吞并过程。丧尸（zombie）葛加理群落鼓励公会成员使用和探索死灵法术法。真菌覆盖了为公会服务的大多数亡灵的身体，其中大多数是真菌苦工（相当于怪物手册中的丧尸 zombie），他们是身体被真菌感染的无意识的仆人。葛加理生物挑战等级 生物 来源0
