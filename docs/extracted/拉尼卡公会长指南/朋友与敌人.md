# 朋友与敌人

第六章：朋友与敌人

拉尼卡是许多奇特生命的家，其中包括许多在其他龙与地下城世界中不为人所知的生物。

这一章提供了许多的这些生物的属性表格，然后以一些在拉尼卡公会中的非玩家角色作为结尾

在本章中有一些生物属性表会有一个特殊的因素：公会法术列表，如果生物属性表显示这个生物是一个公会中的正式施法者，比如伊捷施法者，这个生物的法术列表包含一个或多个在伊捷联盟公会法术列表中的法术。关于这一部分可以参考第二章的公会列表
