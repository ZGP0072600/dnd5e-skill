# 刻洛 Kraul

刻洛是葛加理群落中一股正在崛起的力量，他们长期在地下城的边缘徘徊，如今却在葛加理公会大厅渐渐聒噪可闻了起来。这种有六足似昆虫的生物头脑冷静但是匮乏想象力，很难理解隐喻或抓住细微差别。刻洛士兵刻洛社会由各个分工明确的角色沿世袭制度组成，大量的成年刻洛的构成了军队的各层级，上至于指挥官和精英部队，下至最底层的步兵团。刻洛死亡僧侣死亡僧侣在刻洛社会中身居高位。他们指挥着刻洛仪式上的嗡嗡圣歌。那不可理解的敲击和嗡嗡声能招徕具有严重后果的死灵魔法，和似乎用来加强他们的死亡现世。他们从击败敌人身上汲取能量并将其分发给追随者们，以此保证结社的延续。当前刻洛的领导者是一个叫马滋克 Mazirek 的死亡僧侣。边栏：有翼刻洛一些刻洛士兵拥有40尺的飞行速度，这得益于他们拥有轻盈的翅膀。这些翅膀给了他们在刻洛兵团中更高的地位。有翼刻洛常在公会中担任斥候或者奇袭部队的职务。
