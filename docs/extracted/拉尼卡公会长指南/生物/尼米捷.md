# 尼米捷

尼米捷 Niv-Mizzet

尼米捷建立了伊捷联盟并执掌整个公会。这位远古巨龙集出类拔萃的渊博智慧与惊世骇俗的强大力量于一身，这正与其傲慢虚荣的个性相适。他在自己位于伊捷公会大厅顶层的私人实验室里运筹帷幄，为不计其数的下属安排各式研究试验，整合众多看似毫无关联的项目，由此逐步迈向某个神秘莫测的终极目标。

在拉尼卡，足智多谋的尼米捷几乎无可置疑地位于智慧巅峰，更是最为强大的施法者之一。他像他的同族一样贪得无厌，可是他渴求的珍宝是科学与魔法的知识。他的野心对于其他公会长似乎是个威胁，但一想到他那惊人的魔力，加上龙类的火焰吐息与利剑牙齿，就几乎没人想直截了当地和他对垒了。

施法：

尼米捷是一名20级的施法者，它的施法能力基于它的智力（法术豁免 DC 26，攻击+18）。尼米捷已知下列法术，

戏法（任意）：火焰箭、魔法伎俩、冷冻射线、电爪

1环（4法术位）：侦测魔法、魔法飞弹、护盾术、雷鸣波、隐形仆役

2环（3法术位）：朦胧术、变巨术/缩小术、炽焰法球、灼热射线

3环（3法术位）：法术反制、火球术、定身类人、闪电束、缓慢术

4环（3法术位）：困惑术、任意门、鬼斧神工

5环（2法术位）：元素咒唤术、变形术、火墙术、力场墙（原文变形术及火墙术为五环..）
