# 葛加理祭司

葛加理祭司 Golgari 
shamans

葛加理祭司是葛加理群落的精神领袖。他们教导关于自然循环的公会理念，使用他们的死灵魔法来展示生命如何从死亡中诞生。葛加理祭司在他们的脸上涂装，使他们在他们的脸颊和下巴中显示出拥有着额外的眼睛。他们有时也会使用具有魔力的情绪标记涂料 moodmark 
paint（在第5章中描述的）来让他们可以通过这些标记进行交流。他们身着装饰有甲虫鞘翅、蛛网或层架真菌的衣物。
