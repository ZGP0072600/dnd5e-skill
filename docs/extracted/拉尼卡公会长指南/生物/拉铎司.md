#

拉铎司 Rakdos 拉铎司，这个用其名字命名其宗派的恶魔完美诠释了享乐主义。他自身也是一个完美的艺人，单纯其外表就是恐怖的表演艺术。一个身高30尺的怪物，展开巨大的翅膀，头戴火焰的皇冠，挥舞着燃烧的镰刀，聚光灯也随他而来。他的每次登场都令人惊叹不已。有时，在他伟大的登场后，拉铎司会蹲下来见证那些崇拜他的人的表演。拉铎司的评价是这些人唯一在乎的事，但拉铎司是一个苛刻的观众，他已经看过几千年的马戏表演了，对那些不尽全力的表演者毫无耐心。已经不止一个平淡无趣的表演在拉铎司那火红的镰刀下迅速而壮观地结束了。多年的审美疲劳让拉铎司很少参加宗派的表演。他经常一连数月或数年地窝在宗派会堂下面的巢穴里，但他的追随者们清楚他随时可能现身去观看最新的表演。

引用

拉铎司的特质理想：“每个人都应该追求他们最强烈的渴望。”牵绊：“如果你要讨我欢心，就给我看一场我前所未见的表演。”缺点：“欢乐是什么？”
