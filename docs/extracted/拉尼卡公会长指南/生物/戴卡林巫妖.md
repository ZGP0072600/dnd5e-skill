# 戴卡林巫妖

戴卡林巫妖 Devkarin Lich(https://s1.ax1x.com/2018/12/23/F6PZt0.png)沉浸在葛加理魔法中的戴卡林精灵之中的强大施法者可以超越死亡成为巫妖。对他们来说，生与死不仅仅是在一个不可避免的循环中相互追逐，在这个循环交替的过程中，巫妖们发现了巨大的力量，它以敬畏、嫉妒和恐惧支配着其他葛加理人。与他们所指挥的步履蹒跚的丧尸不同，巫妖保留了他们的记忆、个性，尤其是野心。他们也保持着精灵们的优雅和身形，但它们的本体却一直处于缓慢衰退的状态。各种各样的真菌在他们腐烂的肉间生长，把他们的躯干连在一起。亡灵本质 Undead Archives。巫妖不需要呼吸空气，也不需要饮食和睡眠。

引用

斯托雷 Storrev 斯托雷是故军（Erstwhile）的巫妖领袖。她擅长周旋于政坛，人们担心她有能力将死去的怪物从普通的甲虫变成强大的亚龙，或者不死的惊惧兽。这些生物伴随随她上战场。
