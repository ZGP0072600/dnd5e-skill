# 吸血鬼

**吸血鬼**

吸血鬼是依靠活人鲜血生存不老不死的夜行生物。它们是用世故和感性的表象掩饰其无尽饥渴的凶猛掠食者。那些啜饮金色圣杯鲜血的吸血鬼和那些用尖牙撕开受害者喉咙的吸血鬼一样贪婪，它们只是隐藏得更好。

拉尼卡的吸血鬼与怪物手册中的吸血鬼不同。他们缺乏其他吸血鬼所夸耀的特征和能力，但也缺乏阻碍这些吸血鬼的弱点。他们的共同之处在于对维持亡灵生存的血液有着不可抑制的渴望。

**饮血吸血鬼**

大量饮血者出没于拉尼卡的小巷和下水道里，捕食那些愚蠢到足以离开相对安全的人群的人。

欧佐夫吸血鬼

吸血鬼在欧佐夫集团发家，在那里他们可以通过血的形式从债务人那里收取什一税和报酬。他们的不死本性让他们与寡头精魂一样享受永生，但是他们仍然能够体验所有肉体上的快乐。相比于欧佐夫的精魂，他们还保留了自己普遍的残忍个性。

鲜血联系

吸食一个生物的血液会产生一种移情的联系，让饮血吸血鬼可以对受害者施加一些魔法的影响。

动作

多重攻击。

吸血鬼发动三次攻击，其中啃咬攻击仅能发动一次。

啃咬 Bite。

近战武器攻击：命中+7，触及5尺。单一自愿生物，或单一被吸血鬼擒抱生物，以及失能生物、被束缚生物。命中：7（1d6+4）的穿刺伤害，外加7（2d6）的暗蚀伤害。若该目标为类人生物，则必须通过 DC 15的魅力豁免，否则将被吸血鬼魅惑1分钟。以此方式被魅惑的目标会迷恋上这只吸血鬼。

目标的生命值上限承受等同该暗蚀伤害数值的减损，吸血鬼则回复等量的生命值。减损效应持续至目标完成一次长休。如果该攻击将目标的生命值上限降至0，则目标死亡。

**饮灵吸血鬼**

当吸血鬼加入底密尔会堂，他们可以学会通过受害者的血液将精神能量和记忆一同吸走的能力。此外他们还研习了底密尔心灵法师所擅长的魔法，这强大组合能给予了他们进行间谍和渗透活动的理想能力。

札戴克的后裔

底密尔会堂的创始人札戴克，是第一个所谓的饮灵吸血鬼。他的秘密只传给他公会的其他成员，任何离开底密尔会堂的饮灵吸血鬼都将成为公会的敌人，除此之外的唯一禁令是禁止饮灵吸血鬼同类相食。

小组领袖

拜其独特的天赋所赐。饮灵者们常常被任命为的底密尔隐秘小组的领导。但他们很少信任自己的密探，反而经常跟踪他们的小组成员以确保这些成员按照命令执行任务。最多疑的吸血鬼甚至可能从他们自己的下属那里吸取思想来侦测任何背叛的念头。
