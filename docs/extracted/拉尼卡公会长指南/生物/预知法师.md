# 预知法师

预知法师 Precognitive 
Mage

预知法师是一种罕见的俄佐立施法者，他们拥有能够获取瞥视未来的能力。他们通常被用来预测通缉犯的行动，从而帮助捕获他们。

引用

边栏：芝拉 Geetra 芝拉是一位预知法师，她最初把她的天赋作为小偷在拉尼卡的街头使用。在还是青少年的时候，她被逮捕然后赦免，以换取同意利用她的才能为俄佐立参议院服务。几十年来，她几乎完全匿名工作，她的天赋不受重视。但随着参议院对预知法师的才能越来越感兴趣，芝拉已经获得了一个更加杰出的职位，她热情地接受了这一点。她担心她的公会对拉尼卡其他人的影响，但她心里相信她正在帮助人民。混乱只能破坏，她的愿景是希望有一种工具能确保社会能够安全地生活在各种形式的未来中。
