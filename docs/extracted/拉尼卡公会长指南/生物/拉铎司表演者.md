# 拉铎司表演者

**拉铎司表演者 Rakdos Performer**

通过为许多不同天赋的人提供各自的位置，拉铎司宗派表演艺术家的数量不断增加，包括耍刀客，食火者和钢丝舞者。演员们将拉铎司名言带到街上：摆脱一切，从社会道德与期望的束缚获得自由，放纵你的欲望。
