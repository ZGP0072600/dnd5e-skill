# 卓塔妮

卓塔妮 Trostani

瑟雷尼亚公会长是由代表身体，意志和灵魂的三个树灵的结合体。每个树灵的身体都从一个中央树干中延伸出来，所以虽然他们拥有独立的思想，但他们只有一个名字——卓塔妮，和单一的生命力。通常卓塔妮用一个声音传达世界之魂 Worldsoul 的意志，但她保留了三个不同的个性，体现了瑟雷尼亚理想的三个部分：秩序，生命和和谐。在拉尼卡日益紧张的情况下，这三个人格最近在进行秘密会谈中，讨论如何最好地度过如此困难时期的问题上意见不一。

卓塔妮将大部分时间都花在瑟雷尼亚市政厅维图加基 Vitu-Ghazi 中高耸的树上。在那里，她与太瑟雷尼亚 Mat'Selesnya 以及领导拉尼卡中各个瑟雷尼亚社群的树灵进行交流。

卓塔妮的特质 Trostani’s
