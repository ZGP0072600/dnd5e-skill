# 炎鬃天使

波洛斯天使 Boros 
Angels————————————————波洛斯教团的天使们认为他们自己是造物主的意志化身。他们是英勇的战士，献身于正义，保护弱者不受邪恶和压迫。他们是指挥官，顾问，战略家和士兵。他们在战场上的出现激励着教团普通士兵们的正直和热情。大多数波洛斯的士兵认为天使是善良、智慧和仁慈的典范。现实情况是，天使们也无法幸免于抵御诱惑，而必要的政治阴谋可能让他们中的精英妥协。战领 
Warleaders 最有智慧、最有远见的天使负责制定和实施波洛斯教团的军事战略。由于它们的不朽本质，它们的计划可能跨越几个世纪，而且众所周知，他们接受几十年的损失，作为多年后取得更大胜利的合理代价。这些战争领袖们的数据使用 
星天神使 Planetar 或 炽天神侍 Solar（如怪物图鉴所描述）。但他们的外表与其他波洛斯天使相似。战力天使 
Battleforce Angel 战力天使是闪耀着光芒的主导者，他们飞入战斗，沐浴在波洛斯热情的光芒中。他们自空中率领着一队队凡人士兵，或者飞在空骑兵的前面。他们不会逃避战争的血腥、痛苦和混乱；相反，他们沉浸在不断变化的战争浪潮中。他们只有涉足战斗，他们才能履行战领者的指挥责任，使他们的战术适应地面上不断变化的局势。

炎鬃天使是专注于单兵作战的神圣斗士和战争典范。他们是强大的战士，在任何冲突中都能寻找最强大的敌人，信任弱小的士兵能处理弱小的敌人。

和其他许多波洛斯的天使一样，炎鬃天使通常有着红色的头发。在激烈的战斗中，炎鬃的头发会燃烧，变成一团火焰鬃毛，披在肩膀和背上。
