# 变异充能怪

**变异充能怪 Fluxcharger**

伊捷法师力求创造更易控制的怪奇，在这一过程中，他们曾试图约束闪电、火焰与烟雾元素，融合入魔法合金米捷铸成的架构。实验结果确有成功之处:至少变异充能怪 Fluxcharger 不会像某些怪奇那样易爆，只是具有更高智能且更为固执罢了。

变异充能怪的米捷形体很容易让人想起天使。不过它那类人的面庞缺少人性的意味，严肃刻板的表情与目不转睛的注视只令人不寒而栗。

**变异充能怪 Fluxcharger**

大型元素生物，混乱中立

--

AC:16（天生护甲）

HP:60（8d10+16）

速度:0尺，飞行60尺

--

力量:15（+2） 
敏捷:18（+4） 体质:15（+2） 智力:6（-2） 感知:10（+0） 魅力:7（-2）

--

伤害抗性:雷鸣，非魔法攻击的钝击、穿刺、挥砍

伤害免疫:闪电，毒素

状态免疫:力竭，受擒，麻痹，石化，中毒，倒地，束缚，昏迷

感官:黑暗视觉60尺，被动察觉10

语言:龙语

挑战等级:7（2900XP）

--

强化闪电 Amplify Lightning.

当一个或更多变异充能怪位于某一造成闪电伤害的法术施法范围内，该法术额外造成9（2d8）闪电伤害。

--

**动作**

多重攻击 Multiattack.

变异充能怪发动两次猛击，或两次弧光闪电。

猛击 Slam.

近战武器攻击:命中+7，触及10尺，单一目标。伤害:13（2d8+4）钝击伤害和10（3d6）火焰伤害。

弧光闪电 Arc Lightning.

远程法术攻击:命中+7，施法距离30尺，单一目标。伤害:16（3d10）闪电伤害，之后变异充能怪指定目标30尺内一可见生物，闪电跳跃至该生物身上。该生物必须进行一次 DC 为15的敏捷豁免，豁免失败则受到13（3d8）闪电伤害。命中或失手:变异充能怪受到5（1d10）力场伤害。
