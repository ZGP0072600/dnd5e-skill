# 尼米斯独眼巨人

尼米斯独眼巨人 
Nivix Cyclops

像那些*《怪物图鉴》*中描述的独眼巨人，主要是在古鲁部族中发现的。他们是自然的力量，即使他们偶尔被招进波洛斯教团，他们也永远不会被真正驯服。

独眼巨人作为工作间的保护者、个人护卫者，或重体力劳动者而替伊捷联盟效力，它们穿着米捷装甲以减少实验室事故所造成的伤害，他们戴着活塞式护腕以提升举重和锤击的强度，它们戴上伸缩式头盔，以尽量减少单眼视力的缺点。它们有时被称作单眼巨人或尼米斯独眼巨人，尼米斯一名源自伊捷公会大厅的名称。
