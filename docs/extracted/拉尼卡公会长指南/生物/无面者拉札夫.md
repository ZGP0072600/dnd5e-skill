# 拉札夫 Lazav

无面者拉札夫 Lazav 
the Faceless

拉札夫是唯一能够担任底密尔公会会长的生物。他是个具有通过整个底密尔网络的密探洞悉一切的天赋的变形生物 Shapechanger。他根据的自己和计划的需要而变换多种多样的伪装。他可能伪装成一个拉尼卡街道上的老妪去偷听集市上的对话，或者伪装成俄佐立参议院的维多肯轻骑兵回避检查站，又或者变成一个锡街的商人去欺骗一个路过的显贵。他真正的形态可能是变形怪 doppelganger 或者其他什么生物，但是从没有人见过。
