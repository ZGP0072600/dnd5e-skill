# 地底城美杜莎 Undercity Medusa

地底城美杜莎 Undercity 
Medusa

拉尼卡的美杜莎通常称为蛇发妖 gorgons，这骇人的生物种族外表近似于人类女性，不过没有头发与双手。取而代之的，是一片翻滚扭曲、蜿蜒似蛇的黑缆绳，一对铺着鳞片的可怖利爪。

来自美杜莎发光双眼的凝视能石化活生生的组织，这种转化十分迅速，受害者会变作一尊石像，通常动作都定格在极度恐惧或暴跳如雷的情态当中，而美杜莎的阴森收藏中又多了一件绝妙藏品。石化需要美杜莎集中意志施展，因此一位态度友善或是受到突袭的拉尼卡蛇发妖的凝视并不会造成伤害。

尽管美杜莎十分稀少，她们却在葛加理凭借以死亡凝视攻击的能力夺得了与其数量不成比例的权力。公会下属的势力较小的组织大多由美杜莎掌权，遍布整个拉尼卡，而现在，至少有一位美杜莎被认为正企图执掌整个公会。

当然，也有的蛇发妖没那么野心勃勃，只愿追寻着地底城无尽的阴影前行，有如饥渴的捕食者。

## 动作 Actions

**多重攻击 Multiattack.** 地底城美杜莎发动两次爪击，她可以在攻击前或攻击后发动她的石化凝视

**爪击 Claw.** 近战武器攻击:命中+7，触及5尺，单一目标。伤害:13（2d8+4）挥砍伤害。

**石化凝视 Petrifying Gaze.** 美杜莎凝视60尺内一生物，该生物与美杜莎需能互相看见对方。目标必须进行一次 DC 为14的体质豁免，如果豁免骰值低出5或更多，则该生物立即被石化。否则豁免失败的生物被束缚并开始逐渐变为石头。受束缚生物必须在自己的下一个回合再次进行该豁免，豁免成功则终止该效应，豁免失败则被石化，直到由高等复原术 greater restoration 或类似魔法解除效应。
