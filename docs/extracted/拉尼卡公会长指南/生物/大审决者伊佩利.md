# 大审决者伊佩利

大审决者伊佩利 Supreme 
Judge Isperia

伊佩利是俄佐立参议院的现任公会长。作为一个斯芬克斯，她冷漠而孤傲的。然而，她被迫放弃她的隐居以应对拉尼卡犯罪和混乱的增加。伊佩利致力于她的公会的信念，即法律是抵御混乱的最终保障，而正是她沉着的手段则指导俄佐立度过这些不稳定的时期。作为公会长，伊斯佩里亚担任大审决者 supreme 
judge，利用她对拉尼卡复杂的法律制度有着广博的知识来担任这一职位。如果遭遇变成暴力，伊佩利尽量避免使用致命的手段，尽可能制服不法行为，这样法律制度才能伸张正义。伊佩利的特质 Isperia’s 
Traits 理想：“法律为生命的每个领域提供方向，它是我唯一需要的指南针。”牵绊：“我不能袖手旁观，看着混乱蔓延整个城市。没有俄佐立，拉尼卡就会崩溃。在没有秩序的情况下，我们必须建立它。缺点：“如果拉尼卡不需要我，我倾向过着孤独而沉思的生活。”伊佩利 Isperia 超巨型怪兽，守序中立————————————————AC
