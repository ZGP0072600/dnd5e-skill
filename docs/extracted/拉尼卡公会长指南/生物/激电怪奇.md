| | 值 |

- **抗性**：寒冷，闪电，非魔法攻击的钝击，穿刺，挥砍

## 特性 Traits

--

**自爆 Death Burst.** 激电怪奇死亡时，将爆炸成混杂的闪电与寒冰。其周围10尺范围内的每个生物必须进行一次 DC 为13的敏捷豁免，豁免失败则受到7（2d6）闪电伤害，豁免成功则伤害减半。

--

## 动作 Actions

**猛击 Slam.** 近战武器攻击:命中+4，触及5尺，单一目标。伤害:5（1d6+2）钝击伤害和5（2d4）闪电伤害。若目标为生物，该生物必须进行一次 DC 为13的体质豁免，豁免失败则在激电怪奇的下一回合开始前，该生物不能执行任何反应。
