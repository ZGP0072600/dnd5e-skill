# 唤角师

唤角师 Horncaller

名为唤角师的是一种特别的祭师，他们利用他们的魔法召唤野兽与瑟雷尼亚一起战斗。在局势较为平缓的时候，他们会照料瑟雷尼亚飞地及园林中的动物。

Tandris 作为一个精灵祭司，艾玛拉谭吉全身心地致力于在各个公会之间寻求和平，尽管卓塔妮在自我冲突中犹豫不决。作为杰斯贝连 Jace 
Beleren 的朋友，她理解他永远无法在拉尼卡定居，所以她相信这个世界需要一个十会盟的替代品。艾玛拉在她的社群里扮演着积极的角色，除了十会盟之外，她还有很多非瑟雷尼亚的朋友。她是公会之间最积极的合作倡导者之一，她依靠与其他公会成员的个人关系来帮助这一努力。
