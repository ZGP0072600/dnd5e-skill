# 古式德鲁伊

古式德鲁伊 Druid 
of the Old Ways

古式德鲁伊是古代古鲁传统的守护者，这种传统致力于原始凶猛的动物之神，像野猪毁灭之神哈格 Ilharg 
the Raze-Boar 和伏击者之神卡莎斯 Kashath 
the Stalker。末日毁灭 The 
End-Raze。古式德鲁伊认为，文明将永远不会被零散的掠夺和小规模战争所根除。他们坚持即将到来的末日——末日毁灭 the 
End-Raze 的想法，那时哈格 Ilharg 的蹄子将把拉尼卡高耸天际线以下的每一块砖头和石头都踩成碎片。世界将回归自然原始状态，在这种状态下，凭蛮力说话和靠野蛮行事的无序法则将再度盛行。

引用

边栏：古式信徒妮恰 Nikya 
of the Old Ways 筑塔族的领导者妮恰是一位强大的人马同时也是一名古式德鲁伊。事实上，她认为自己是真正的古鲁德鲁伊的最后一个。
虽然成长为一名战士，但妮恰很快就发现了她的魔法天赋。
她可以从拉尼卡古老的地壳中召唤出巨大的束缚藤蔓。她不怎么喜欢咕噜力莫，她相信另一个忠于古式的领导者，将在末日毁灭 the 
End-Raze 开始之前出现在古鲁中。
