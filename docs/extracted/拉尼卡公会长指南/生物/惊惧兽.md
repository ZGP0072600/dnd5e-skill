# 飞行惊惧兽 Flying Horror

惊惧兽 Horrors

从地底城的深渊到最黑暗的夜空，可怖的邪恶在拉尼卡黑暗的角落中伏行着、翱翔着、疾驰着。这些生物统称为惊惧兽——一种潜伏在黑暗中，将拉尼卡人内心最深处的恐惧具象化的多种类的生物。所有的这些邪恶生物都有着朦胧的理性和不可思议的诡计。

目前强迫惊惧兽为其服务的已知的公会至少有三个。在底密尔会堂，惊惧兽能在市民中散步恐惧与绝望并能承担对那些被严密保护的目标实施暗杀与绑架的工作。在拉铎司宗派，惊惧兽被作为又一种会登台亮相的惊险又刺激的生物武器。在葛加理群落，惊惧兽清扫着下水道并保卫着公会的领地。

底密尔会堂会使用所有三种常见的惊惧兽，拉铎司宗派更青睐幽影惊惧兽，而葛加理则使用掠行惊惧兽。

惊惧兽和疯狂 Horrors 
and Madness

拉尼卡的惊惧兽是具象化的恐怖与疯狂。你可以使用地下城领主手册中的疯狂规则来表现这种联系。

每当一个角色在对抗飞行惊惧兽的*恐怖尖啸 Frightening Screech*，幽影惊惧兽的爪击或者掠行惊惧兽的*癫狂威仪 Maddening Presence*的豁免中失败，记录下这个结果。在遭遇结束后，每一个至少豁免失败一次的角色需要通过一个 DC 13的感知豁免。豁免成功则无事发生。豁免失败则该角色根据地下城领主手册获得一种疯狂，并根据角色在遭遇中豁免失败的次数决定疯狂的严重程度，详见疯狂程度表。

自定义一只惊惧兽 Customizing 
A Horror

惊惧兽通常拥有一些常见的身体形态，但是其特征却五花八门。某一只的惊惧兽形态可能融合了一些恶魔、昆虫或者爬行动物的元素。要自定义一只惊惧兽，首先选择一个基底生物（飞行惊惧兽，幽影惊惧兽或者掠行惊惧兽），然后在 **首要特征表 Primary Features table** 和 **次要特征表 Secondary Features table** 里分别投一个结果（或者在每个表格里选择一个）。将这些特性加在基底生物上。

## 特性 Traits

——————————————

**恐惧狂怒 Fear Frenzy。** 惊惧兽对陷入恐慌的生物发动攻击时，其攻击检定具有优势。

**日照敏感 Sunlight Sensitivity。** 身处阳光下时，惊惧兽进行的攻击检定，以及依赖视觉进行的感知（察觉）检定具有劣势。

## 动作 Actions

——————————————

**爪击 Claw。** 近战武器攻击：命中+7，触及5尺，单一目标。命中：8（1d6+5）的挥砍伤害，外加14（4d6）的心灵伤害。

**恐怖尖啸 Frightening Screech（充能5~6）。** 惊惧兽发出一声恐怖的尖叫。其身边30 尺范围内每位可以听见其叫声的生物，必须进行一次 DC 13 的感知豁免，豁免失败则陷入恐慌，恐慌效果持续1分钟。陷入恐慌的生物可以在每个其自己回合结束时再次进行该豁免，并在豁免成功时结束恐慌效应。豁免成功或身上的该效应结束的生物，将在接下来的24 小时内免疫惊惧兽的恐怖尖啸。
