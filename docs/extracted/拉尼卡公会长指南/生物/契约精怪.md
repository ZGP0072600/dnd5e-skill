# 契约精怪

**契约精怪****Indentured spirit** 那些因背负着欧佐夫集团债务而死去的人不会得到缓刑。相反，他们的精神服务于集团，直到他们履行了他们的义务。有时这意味着他们作为契约精怪将存活数年甚至数千年。契约精怪是一种披着幽灵般黑色长袍、戴着遮掩住他的任何面目的兜帽的无形灵魂。它的胸膛和双臂上挂着锁链，标志着它永远被奴役的身份。**欧佐夫精怪****Orzhov Spirits** 对于欧佐夫集团的成员来说，死后的生命可以是一份礼物，也可以是永远的奴役。分离灵魂和身体的过程通常是由欧佐夫集团政党之中最古老、最受尊敬的家族的首领自愿经历的。这使得他们可以在永恒的余下时间享受颓废的战利品。认为他们可以度过余生享受腐朽的赃物。这些精怪以鬼魂 ghost 的身份开始他们的不死生涯，使用怪物图鉴中的鬼魂 ghost 属性面板。然而，随着时间的推移，他们倾向于抛弃个性上的细微差别，并讽刺他们活着时代的自我，最后成为怪物图鉴中描述的幽灵 specter。**契约精怪****Indentured spirit** 中型不死界生物，任意阵营————————————————护甲等级
