# 语言

**语言 Language**

各类语言在拉尼卡市井间争鸣，每种声调都配以百花齐放的方言和地域性变体。通用语言普及是公会发挥效用的要素，不过其余语言在各家各户与部落聚会中活跃依旧。

标准语言 Standard 
Languages

>

拉尼卡城中也不乏与众不同或不为人知的语言。偶尔，德鲁伊语 Druidic 的秘密交流会在不同公会间暗中进行（比如瑟雷尼亚和葛加理）。盗贼黑话 Thieves' 
Cant 则在街头团伙间盛行，有时底密尔或是葛加理的游荡者也会略一使用。
