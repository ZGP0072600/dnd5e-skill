# 底密尔会堂

**底密尔会堂 House Dimir**

一只吸血鬼自黑暗中显现，如微风般悄然而至。他的双眼如猫一样反射着光芒，尽管在这寒冷的地下，他依然裸露着自己的腰腹。他毫不费力地从会堂的高处飘落下来，降落在他们两人之间的地面上。

 
 
 
——道戈·拜耳， 《再访拉尼卡：解秘人》

底密尔会堂是拉尼卡阴暗的秘密：在一个令人尊敬的信使和记者组织的表象背后隐藏着的一个间谍和刺客的结社，而其存在从不会被一般市民所怀疑。“秘密”既是底密尔会堂最棒的武器，也是它最佳的防御，因此公会大多数的工作都会对其他成员保密。底密尔密探的隐秘行动不会留下任何痕迹，他们会扭曲那些窥见他们犯罪行为的目击者的记忆，甚至抹消他们自己的精神来移除有关已完成任务的证据。

底密尔会堂的最初公会会长是一个名为札戴克的吸血鬼，他的组织同意作为信使、情报中介和图书馆管理员为拉尼卡服务。但是札戴克却利用公会那收集情报的专业技术构建了一个庞大的间谍网络，而在不久之后，公会不为人知的行动数量就远远超过了公布于众的行动数量。最后，整个公会都消失在了暗影中，而大多数拉尼卡人也开始怀疑它的存在。作为一个不朽的存在，直到札戴克在万年庆的动乱中被逮捕并处死为止，他已经统治了底密尔公会长达一万年。

**会堂内部**

底密尔会堂具有一个仅让那些最重要的公会成员才知道的秘密区域。公会会长“拉札夫”和他的直接联络人指导和操纵着公会的秘密行动。底密尔会堂的成员只会得到指令，但最终对发布指令的人和原因却一概不知。信息以思想链灌注（参见这章节之后的“法术：思想编码”）或心灵感应信使的方式被传递给密探。

你与你公会的其他成员之间被刻意安排的不会有太多交互。你可能从未与你真正的公会有过面对面地接触，取而代之的是通过编码和发送秘密信息来接收任务和发送报告。

**底密尔的目标**

底密尔会堂那有关秘密和虚假信息的一切，即使是它自己的成员也需要谨慎对待。底密尔密探所被允许知晓的仅限于一小点儿的后巷接头和只言片语的文件。一个密探只会知道另外一个密探的代号，他们仅能在特定时间地点的特定场所才能见面并传递信息。每一个有点见解并自称为专家的人都会有自己的一套关于公会意图的理论，而所有对底密尔动机和追求的猜测都彼此矛盾，最终挫败了任何试图了解事情真相的尝试。底密尔公布与众的那一面仍然让人难以理解，而有些人则将其解释为证明底密尔公会那对拉尼卡的平民们来说无比可怖的真实计划的最好证据。

对于底密尔会堂，知识就是力量。公会饥渴地学习着它所不知道的一切，尤其是它那些敌手的弱点，它会掌握并最终利用这些弱点。而另一方面，会堂会严密地保护住它们自身的弱点，不会让他的敌人有翻盘的希望。底密尔潜伏在暗影之中，有条不紊地收集它们所需要的知识，改变拉尼卡从而让自身更有优势。

底密尔会堂的发展历程让它依赖于一个由精心搜集的情报织就的网络去达成目的。其他公会不可预测的行为可能会动摇这张网络。当波洛斯被突然爆发的狂热鼓舞而实施迅捷有力的行动时，底密尔会猝不及防。瑟雷尼亚的行为可以从整体上进行预测，但是盟会的成员是如此地众多以至于底密尔很难对他们个体的行为进行追踪。而其他公会，伊捷和古鲁最让底密尔担忧，他们毫无章法地作出决定，结合它们无惧危险的态度，能够从根本上瓦解底密尔那些试图遏制他们的策略。

**加入底密尔会堂**

你在底密尔会堂的职业生涯开始于众多用谎言来保持公会文雅表象的店铺老板、图书管理员、信使和商贩之间。一方面，你不会暴露公会任何的秘密活动，而另一方面，你会有助于败坏那些相信底密尔在计划操纵拉尼卡城的人的名声，把他们渲染成妄想阴谋论者。但在某一时刻你的天赋将得到认可，你会被邀请进入到公会更深一层的秘密之中。你将会被重点培训隐匿、间谍和渗透技术。作为一个底密尔间谍，你多样的才华能在工作中被尽情施展。

你的公会角色将从事监视、盗窃、破坏、渗透和其他多种间谍活动。你只在口袋或者睡袋里工作，对于其他密探或者公会领袖的身份一无所知。你可能是一个盗贼、刺客或者诡术师范型的游荡者，一个暗影宗的武僧，甚至是一个诡术领域的牧师。

如果你是一个施法者（也许是一个专精于预言、惑控或幻术学派的法师），你所具有的技艺将允许你加入拉尼卡最令人畏惧的施法者阶层——底密尔会堂的心灵法师。你的基本工作和角色同其他间谍一样，但是你能够学会将思想和记忆从一个人的心灵中拖拽出来，利用某个目标自己的思想对抗他的集体，制造妄想来直接攻击敌人的精神，或者将敌人的心灵清洗一空。

**背景：底密尔特务**

你是一个间谍。你收集秘密与虚假信息并进行交易。你潜伏在阴影里，渗透进其他公会里，并盗取最宝贵的秘密，无论这些秘密是写在上锁的日记里还是藏在某人的脑海里。甚至你可能毫不关心你所执行的任务背后的任何原因。有时，一个任务的唯一目的就是隐藏在城市的另一个地区进行的另一次袭击背后的动机，又或者只是为了散布恐慌。

作为你在底密尔会堂的隐秘工作的一部分，你可能会伪装成另一个公会的一员。你能够选择你的第二个公会或者投骰来随机决定。你的第二个公会的身份会决定你的一部分初始装备，同时也是你大多数人际交往的来源。

**关系**

作为一个暗中活动的底密尔会堂密探，你与你公会的关系非常有限。而你与你伪装的虚假身份所在的第二公会的关系通常更加广泛。

在底密尔关系表掷一次骰子来决定一个你在底密尔里有关联的盟友，再在你的第二公会的关系表里掷两次骰子，其中一个是你的盟友，另一个则是你的对头。

> 底密尔人际关系

**如何扮演**

作为一个底密尔的冒险者，你是公会隐藏在虚假的公众形象背后的，由间谍、窃贼、刺客和心灵法师组成的庞大网络中的一员。在表面上，底密尔会堂是一个从事情报交易和新闻传播的由信使、调查员、媒体记者、档案管理员构成的网络。但你和你的同伴却在交易着各种秘密。你经常会在众目睽睽之下使用简单的标志、符文、信号偷偷摸摸与其他底密尔密探进行交流。

和任何优秀的间谍一样，你具有多种身份：你真正的身份——底密尔会堂的密探；一个无属者的身份；和一个其他公会的成员的身份。你可能已经从底密尔会堂获得了一个与你的第二个公会有关的任务，被指派到那个公会里当间谍，收集有关某人的信息，或者从那个公会的阶层中招募另一位间谍。又或许那个公会能够成为你真正任务的起始点。举个例子，你或许被命令去渗透俄佐立，试图接近俄佐立监狱里关押着的一位臭名昭著的囚犯。

底密尔小队

底密尔会堂可能会派遣一支冒险者小队执行有关盗窃和诡计的人物。这样的一支小队的大多数成员是游荡者和/或武僧，一名心灵法师（法师）负责提供魔法支援，药水能够帮助这支队伍弥补缺乏治疗能力的缺陷。

**等级与声望**

随着你在底密尔会堂中获得声望，你将会获得越来越重要的任务作为奖励。你的进步不会伴随着通常的阶级变化，但是声望提高到一定程度时会改变你在公会中的地位。或许根据底密尔会堂的处世之道，更高的地位能够带来更多有关拉尼卡十公会内部运作方式的知识，这才是最重要的。

**敌人与盟友**

对于一个由隐匿和秘密造就的公会而言，与任何其他公会保持一个积极的关系是很难的。所有公会都被底密尔严密监控，他们当前和未来的威胁程度以及对底密尔计划的有用程度都会被评估。每个小组都会自己培养与其他公会的关系，但是他们最终都要遵从底密尔那神秘领袖的总体战略的调控。

你的任务可能通过要求你们渗透进某个公会去洞察他们的计划、从他们的公会大厅盗取机密信息或者颠复他们的领袖等方式，使你与其他任何公会发生冲突。例如：你可能被委派去盗取一次俄佐立审讯的记录、一个葛加理间谍的记忆，或者一本欧佐夫账本的内容。

只要他们的目标不与你的冲突，你就可以与其他公会的成员或公开或私下地展开合作。有时候刺杀你的一位敌人的最简单的方式就是将法律的力量放到他的踪迹上。所以，有时候与俄佐立和波洛斯联手对大家都有帮助。
