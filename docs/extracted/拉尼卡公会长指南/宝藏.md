# 宝藏

第五章 宝藏 Chapter5: Treasures

拉尼卡愿以其丰富宝藏嘉奖那些技巧精熟、英勇无畏的冒险者的功绩:暗巷中一再转手的硬币、驮兽担负的成箱财物，甚至博物馆与艺术廊中展览的绝妙艺术品。污秽的非法积蓄深匿于欧佐夫不为人知的宝库，俄佐立参议院的工资开据满篇官腔套话，底密尔会堂则以不计其数的财宝屡屡行贿，更有蕴藏惊人力量的魔法物品诱惑着来自各地的盗贼与冒险者。

宝藏中的硬币 Coins in 
Treasure

如果你想将收获钱币与某一公会建立联系，可以在城主指南第七章宝藏表的基础上改变货币类型，详细信息参见本书介绍部分的“流通货币:齐布与齐诺 Currency:Zibs 
and Zinos”一栏。

铜币 Copper 
pieces

多以标准俄佐立式齐布 standard 
Azorius zibs 或欧佐夫式救济币 Orzhov 
alms-coins 形式出现在宝藏中。你可能会从穷困潦倒者或是对他们强取豪夺的惯犯身上搜取一小包 pouch 救济币，或在用于存放这类货币的欧佐夫锁箱中找到整整一木箱 chest，堆积在这里的救济币原本将发放给下层人士。

银币 Silver 
coins

中由俄佐立参议院铸造的币种面值25cp，因而出现时数量一般少于宝藏表 treasure 
tables 中给出的、适用于标准银币 Silver 
pieces 的数据。拉尼卡大街小巷各色人等都可能装着一包这样常见的硬币，吝啬鬼的钱袋与银行保险库则贮藏着大量银币 silver 
coins，当然，刚从中偷窃的精明小偷也可能敛财颇多。

银金币 Electrum 
pieces

以俄佐立式50齐布币 Azorius 
50-zib 
coins 的样式在宝藏中星罗棋布。在拉尼卡，这类硬币比大多 D&D 世界中都平常得多，与25齐布银币的用途大同小异，因此 DM 完全可以将宝藏中的银币 silver 
pieces 等值换作银金币 electrum pieces。

金币 Gold 
coins

则可等价改为1齐诺币1-zino 
coins，分为俄佐立式和波洛斯式。教团常用这些波洛斯式铸币作为士兵的每周报酬。宝藏中也许还会出现波洛斯式5齐诺币 Boros 5-zino 
coins，它们多被教团用于支付军官的薪水或是进行公会间交易。

白金币 Platinum 
pieces

则是欧佐夫式10齐诺币 Orzhov 
10-zino 
coins。这些宝藏或许原本归在某位欧佐夫大主教名下，也可能刚从欧佐夫下辖银行账户取出，来自一名重权在握或富甲一方的人物，背后估计还暗藏着罪恶勾当。重量级的宝藏中还可能存放着欧佐夫式100齐诺币 Orzhov 
100-zino coins，极有可能来自权力巅峰的欧佐夫传人，而这类人可不会就此罢手。其数量则为宝藏表 treasure 
table 中白金币数量的十分之一。
