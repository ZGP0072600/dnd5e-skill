# 蛇之家

**蛇之家 Cair Ophidian**

尤瑞艾莉的家隐藏在外域一个与世隔绝的地方，蛇之家。曾经有一个水巨灵贵族家族在这里建造了一座环绕瀑布的富丽堂皇的山庄，但它早已被创造者抛弃。在阿斯忒瑞亚与尤瑞艾莉一起冒险的过程中，她们偶然发现了这些僻静的废墟，并开始把其作为一个安全屋。许多年过去，这两位重建了这处山庄的部分地区，将其从一个临时的藏身处变成一处家园，她们将其命名为蛇之家。

现在，尤瑞艾莉照料着她的花园，也会偶尔招待迷路的旅行者。在不探索多元宇宙的时候，阿斯忒瑞亚也可以在蛇之家被找到，在开始另一场冒险之前，她总是会拜访她最亲爱的朋友并在这里好好修养一下。
