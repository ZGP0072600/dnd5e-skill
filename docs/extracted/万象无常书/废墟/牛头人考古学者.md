# 牛头人考古学者

牛头人考古学者 Minotaur Archaeologist

*中型类人，任意阵营*

护甲等级：11（天生护甲）

生命值：18（4d8）

速度：30尺

力量12（+1） 敏捷10（+0） 体质11（+0）

智力13（+1） 感知12（+1） 魅力10（+0）

技能：奥秘+3，历史+5，察觉+5，宗教+3

感官：被动察觉15

语言：通用语

挑战等级：1/4（50 XP） 熟练加值+2

**迷宫追忆 Labyrinthin Recall。** 牛头人可以清晰的回忆起自己曾经走过的任何路径。

动作 Actions

**角撞 Gore。** 近战武器攻击：命中+3，触及5尺，单一目标。伤害：4（1d6+1）点穿刺伤害。如果牛头人在直线向着目标移动了至少10尺后立刻用角撞命中目标，目标将额外受到3（1d6）点穿刺伤害。若目标是生物且其体型不超过大型，该生物还必须通过一次 DC 11的力量豁免，豁免失败将陷入 倒地 Prone 状态或是被牛头人推离10尺。
