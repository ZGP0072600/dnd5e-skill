# 宿命之英雄

宿命之英雄 Heroes of Destiny

贤者之宿命表格中提供了一些受 万象无常牌 Deck of Many Things 所启发的编故事的点子可以让你在创建你的人物卡时使用。此表中的条目主要面向施法者而作，不过它们可以用于丰富任何角色的背景故事。当使用这个表格的时候，你可以直接选择你需要的条目，也可以通过从卡组中抽取卡片的方式。

**贤者之宿命 Sage Destinies**

| 卡牌 | 英雄契机 |
|---|---|
| 平衡 | 你曾是献身于防止魔法物品落入滥用权力者手中的事业中的一员 |
| 彗星 | 一位预言家曾预言你命中注定会拥有强大的力量。你是选择追寻这种力量还是抗拒它？ |
| 城堡 | 你梦见一个被囚禁的跨位面实体 |
| 尤瑞艾莉 | 你希望学习强大的魔法以保护亲人免受伤害** |
| 命运 | 你的魔法研究由操纵现实的欲望所驱动 |
| 火焰 | 你的魔法能力来源于你和邪魔的一场打赌......虽然你不确定自己到底是赢了还是输了 |
| 愚者 | 你用你的神秘力量去欺骗毫无戒心的人 |
| 宝石 | 你随身携带着一件从一位强大的大法师那里偷来的看似普通的物品，但它会让你身边发生一些莫名其妙的事情 |
| 小丑 | 你继承了野生法师的悠久传统，即使你自己的法力与众不同 |
| 钥匙 | 你的法术书外观上像是一副卡牌 |
| 骑士 | 你那名门武士家族会根据你所拥有武术天赋（或否）来对你进行评判 |
| 月亮 | 你的一位亲人受到了诅咒，而你决心要打破这个诅咒 |
| 谜题 | 你为追寻神秘的秘密和失落的传说而活 |
| 浪客 | 你所信任的人实际上是怪物伪装而成，比如龙或者巫婆 |
| 废墟 | 一位占卜师曾警告你命中注定会造成巨大的破坏，而你一直尝试避免其一语成谶 |
| 贤者 | 你用占卜牌来决定每天准备哪些法术 |
| 头骨 | 你寻找一种能逃避死亡的方法 |
| 星辰 | 你的吟唱和魔法能力在表现上如同在玩卡牌 |
| 太阳 | 曾有一名 炽天城塞 Solar Bastion （见第十章）的特工曾经救过你的命 |
| 利爪 | 你想要夺回一件失窃的魔法物品 |
| 王座 | 你从宫廷魔法师学徒做起，开始你的魔法学习。 |
| 虚空 | 你瞥见了另一个世界的存在，但没人信你。你努力去证明他们是错的。 |
