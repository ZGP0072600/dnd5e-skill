# 星迹

**星迹 Zodiac Phenomena**

万象星天的星座可以施加或有利，或有害，或临时，或永久的强力魔法效果。一次星象或许可以在横跨一整代人的时间里席卷整个星球，又或是仅仅每个月在某个村庄发生一次。DM 可以自由决定星迹的发生时间与地点，以及它们对受影响地区的效应。

星迹表格列出了和万象星天的星座产生短暂邂逅时的效果。你可以直接使用在游戏当时最突出的星座的效果，也可以在表格中随机选择。以下是可能产生一次星迹的例子。

- 一次季节变更，伴随着不同的星座在夜空中凸显
- 一颗或更多行星在给定的星座内排列
- 夜幕降临于无月之夜
- 角色进行了一次长休
- 角色从一台*跃迁望远镜 observatory’s telescope*的目镜向外看
- 云层散开，露出其中的星星
- 一个生物施展了一道预言系法术

**星迹表格 **

| D12 | 效果 |
|---|---|
| 1 | 颅骨座。亡灵涌现出全新的活力，所有区域内的亡灵在每天黄昏获得13(3d8)点临时生命值 |
| 2 | 天秤座。每个生物都能直观地看见区域内其他生物的免疫，抗性，易伤属性 |
| 3 | 慧贤座。感情在这个区域内被压制，但心灵却变得更加开放。每个区域内的生物在为了改善区域内其他生物的态度进行的魅力检定中获得优势 |
| 4 | 王权座。守法者坚定了决心，而反叛者则更有勇气追逐他们的事业。区域内的所有生物免疫 恐慌 Frightened 状态 |
| 5 | 宝钥座。一个生物可以通过触碰的方式打开一扇门，就算门上的锁是魔法性的也一样。 |
| 6 | 热焰座。这里的情绪变得更加容易激动，无论什么时候，当区域内的一个生物尝试改变其他人对他的态度时，一次失败的魅力检定都会使后者的态度变为敌对。 |
| 7 | 英骑座。骑士的虚影在天空中盘旋。当角色在这个区域的室外进入战斗时，一个或更多的骑士从空中降落并给予角色帮助，前提是角色中没有一个是邪恶的。在此情况下，每轮先攻轮到20时（同时则靠后），这些骑士对每个角色的敌人造成9(2d8)点力场伤害。当战斗结束时，这些骑士会鞠躬，随后消失 |
| 8 | 尤瑞艾莉座。区域内的自然植被披上了荆棘的外衣。每一个在自己回合内进入这些植被造成的困难地形或在其中结束回合的生物都需要通过一次 DC12的体质豁免，否则受到2d6毒素伤害。如果这个伤害会令他的生命值降为0，这个生物会维持1点生命并陷入石化状态，直到被一道高等复原术 Greater Restoration 或类似的法术解除。 |
| 9 | 盲盗座。在这个区域内，潜行变得更加容易。区域内的生物在进行隐匿（敏捷）检定时具有优势。 |
| 10 | 宝石座。在一个晴朗的午夜，星星貌似增加了十倍，随后细小的银色陨石如雨水般落下。总共价值100GP 的碎银不造成任何损害地落下，并散落在这个区域 |
| 11 | 利爪座。一些魔法在这个区域内被压制。普通魔法物品，戏法，以及一环法术无法在这个区域内生效。 |
| 12 | 弄臣座。不幸降临在这些区域的人身上。无论什么时候，当一个类人生物在一次技能检定或攻击检定中骰出1(在 d20检定中),他会魔法性地突然开始无法抑制的大笑，陷入失能状态直至他的下一个回合结束。 |
