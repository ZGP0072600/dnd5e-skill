# 天文台示例

**天文台示例 Example Observatory**

图13.1展示了一个中等大小的天文台，其中有2d4名学者，研究着星象，并为参观者提供信息帮助。如果需要的话，使用 **魔法师 Mage** 作为他们的数据。

尽管资源匮乏，但在这个偏远天文台工作的学者们并不想要冒险者们为他们的服务付费。他们会更喜欢让冒险者们帮忙，比如清理当地的怪物巢穴，或者从临近的山顶收集一种珍惜的魔法元素。作为回报，学者们允许角色将天文台当做他们的临时据点，并使用内部如下所示的设施.

天文台的正门通往建筑的一楼。一个十英尺宽的钟面悬挂在天花板上，被缓慢运转的齿轮包围。细长的金属链从齿轮向下，穿过地板上的小孔，连接在地下水车上。由于天文台位于一个没有水源的山顶上，学者们转而用一个*无尽水瓶 Decanter of Endless Water* 
 
装满一个为水车供水的高位水池。当水车旋转时，连接在水车上的链子带动上面的齿轮旋转，从而控制时钟运作。

这个钟不仅可以告知当前时间，也可以告知日出日落的时间，月相，十二宫的位置。天文台会向参观者收取参观以及提供学术解读的费用。细水长流，这是天文台主要的经济来源。

毗邻的会议室可以容纳天文台的学者，有一架梯子从时钟室通往图书馆。

> **边栏：遥远奇观 Distant Wonders**
>
> 魔法望远镜可以看见普通望远镜看不见的天体，比如遥远的世界到即将到来的彗星。炽天之壁的学者，或者彗星先驱，这些天文学家可以用魔法望远镜定位隐藏的奇观，比如分别在第10,12,17章中有介绍的炽天之壁，彗星先驱以及球牢城堡。

**图书馆 Library**

天文台第二层的图书馆藏有关于外星奥秘的书籍，还有星图，年鉴，荒宇地图参考资料。不寻常的外星物品，比如陨石和从外星获取的植物，也被一并保存在这里，并进行着研究。

**星图室 Planetarium**

这个没有窗户的房间的拱顶被永久地施加了魔法，以不分黑夜地显示头上的恒星与行星。四张舒适的沙发可以让游客（或者寻求休息的学者）安稳地观看星星。常驻的学者可以使用相邻房间的水晶球以模拟一年中任何时候的夜空。

**传送法阵 Teleportation Circle**

使用*传送术 Teleport*或*传送法阵 Teleportation Circle* 
的施法者可以使用在这个永久传送门上的符文定位。

**望远镜 Telescope**

放置在此处的沉重机械是一台无价的 *跃迁望远镜 Telescopic Transporter* 
 
（将在本章稍后描述）。天文台的天文学家们会无比自豪的、滔滔不绝的和游客们介绍他们的望远镜是如何设计、打造、组装的故事——以及这耗费了他们多大的资金。这些故事的最后往往总是以请求游客们捐献一些微博的资金，来帮助天文台渡过长久以来的财政困难，为结尾。
