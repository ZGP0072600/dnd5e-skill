# 凯瑟拉苦风

**凯瑟拉·苦风 Kathra Bitterwind**

人类

**秃鹫人 Werevulture** 凯瑟拉·苦风（秃鹫人数据详见第二十一章）是奥古斯都老大和德卢尔老大最信任的副手。与大多数成员不同的是，凯瑟拉在加入公会时已经是一名兽化人，她通过证明自己对公会首领坚定不移的忠诚，很快就爬到了高层

由于首领们大部分时间都在月之潜猎者的公会会馆里与强大的客户进行策划和谈判，大多数公会成员与凯瑟拉相处的时间远远多于与德洛尔或奥古斯都相处的时间。她训练成员，领导高优先级的任务，总是希望她的同伙能像她一样提供专业精准的服务。她脾气暴躁，对新成员没有耐心，但在关键时刻她很少失去冷静。
