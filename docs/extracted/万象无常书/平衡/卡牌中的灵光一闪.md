# 卡牌中的灵光一闪

卡牌中的灵光一闪 Card Sparks

你可以使用卡牌上的牌面元素来快速获取灵感。以下是三种寻找灵感的方法：

**快速创造非玩家角色 Quick Nonplayer Characters**

当你需要创造一个非玩家角色时，你可以抽一张牌，然后从中寻找灵感来描绘这个 NPC 的特点。例如，如果卡牌描绘着某个人物，这个 NPC 可能很像那个人。如果卡牌具有某种徽记，也许这个 NPC 的衣着或首饰就带有这种徽记。如果卡牌具有某种占卜上的含义，这一含义也许对应着这个 NPC 的名字、个性或目标。

**灵感牌组 Idea Decks**

你可以尝试构建多个自定义牌组，将他们放在手边随时供你汲取灵感。只要你的卡牌够用，你就可以试着构建牌堆来随机抽取特定角色、生物、事件、法术、宝藏或任何你需要的东西。

**填充场景 Filler Scenes**

你可以利用卡牌填充游戏的叙事。举例而言，你可以在一次团的开始让每名玩家抽一张牌，然后你或玩家（或者你们一起）可以把抽到的卡牌当作头脑风暴的素材，用于设定这次团开始时的场景或解释两次冒险之间的休整期里发生过什么事件。除此以外，你还能在带团的过程中抽牌来协助你创作故事的节拍点。

**激励牌池 Inspiration Hand**

你可以使用*万象无常牌*实体牌组（或使用二十二张对应扑克牌的牌组）为玩家们提供另一种使用激励的方法。

抽取等于玩家数量的卡牌，将它们正面朝上地放在桌面上，确保每个人都能看到它们。当一个玩家获得一次激励时，他\她必须挑走一张卡牌，并在之后合适的时机按 **激励卡牌表 ** 中展示的效果使用卡牌。之后，这名玩家抽一张新牌加入公共牌池，然后把用过的牌洗入牌堆。

**激励卡牌表 Inspiration Cards**

| 卡牌 | 效果 |
|---|---|
| 平衡（或黑桃2） | 当你对生物造成伤害时，你立刻花费1个生命骰治疗自己。投掷你的生命骰，然后你回复 投掷结果+你的体质调整值 生命值（至少1点）。 |
| 彗星（或方片2） | 作为一个附赠动作，你将一个生物指定为仇敌。这个生物对你造成的伤害具有易伤，对其他任何伤害具有抗性，并且当该生物攻击除你以外的生物时，其攻击检定具有劣势。这一效果在任何除你以外的生物对目标生物造成伤害时终止。 |
| 城堡（或黑桃 A） | 作为一个动作，你被传送到一个空旷的半位面，直到你的下个回合结束。直到你返回之前，你无法影响原位面的任何东西，也不会被它们影响。当你返回时，你回到之前所在的地方，或者当那里被占据时改为最近的未被占据的空间。 |
| 尤瑞艾莉（或黑桃 Q） | 当一个生物进行豁免检定时，这个生物在那次检定中具有劣势。 |
| 命运（或红桃 A） | 重投你正在进行的一次 D20检定。你必须使用重投的结果。 |
| 火焰（或梅花 Q） | 当你对生物造成伤害时，那个生物额外受到1d6点火焰伤害。 |
| 愚者（或大王） | 作为一个反应，当你受到伤害时，你使对你造成伤害的生物陷入倒地 Prone 状态。 |
| 宝石（或红桃2） | 在你的回合中，你执行一个额外的动作。 |
| 小丑（或小王） | 使用附赠动作执行撤离 Disengage 动作。 |
| 钥匙（或红桃 Q） | 作为一个附赠动作，你与你着装和携带的所有物品一同传送到最远30尺外的一处你能看到的空间。 |
| 骑士（或红桃 J） | 作为一个反应，你迫使你能看见的一个生物所进行的攻击检定具有劣势。 |
| 月亮（或方片 Q） | 作为一个附赠动作，回复一个已消耗的1环法术位或每日能使用特定次数的能力的一次使用次数。 |
| 谜题（或梅花2） | 在一次豁免检定中获得优势。 |
| 浪客（或黑桃 J） | 在一次攻击检定中获得优势。 |
| 废墟（或黑桃 K） | 作为一个附赠动作，你迫使你能看见的一个生物在其所进行的属性检定中具有劣势，直到该生物的下个回合结束。 |
| 贤者（或方片 A） | 使用附赠动作执行协助 Help 或搜索 Search 动作。 |
| 头骨（或梅花 J） | 作为一个动作，你以死亡的罩幕环绕自身。直到你下个回合开始前，所有在你身边5尺内结束回合的生物受到1d6点暗蚀伤害。（译注：理论上也包括你自己） |
| 星辰（或方片 J） | 在一次技能检定中获得优势。 |
| 太阳（或方片 K） | 作为一个反应，当一个生物进行的近战攻击命中你时，那个生物获得目盲 Blind 状态，直到你下个回合结束。 |
| 利爪（或梅花 A） | 作为一个附赠动作，将一个你5尺内的生物的速度降低至0，直到你的下个回合开始。 |
| 王座（或红桃 K） | 作为一个反应，当你受到伤害时，你使对你造成伤害的生物陷入恐慌 Frightened 状态，直到该生物的下个回合结束。你是其恐慌的源头。 |
| 虚空（或梅花 K） | 作为一个反应，当你受到伤害时，你获得对那种伤害的抗性，直到你下个回合开始。 |
