# 废墟蚀灭蛛

**废墟蚀灭蛛 Ruin Spider**

万象无常牌的强大魔法能对环境产生难以预测的神秘影响，特别是这组无常牌长期停留在某一区域时。

废墟蚀灭蛛便是被无常牌的影响所转化的巨蜘蛛，这种危险的怪物所产生的酸蚀毒液会解离所接触到的一切。熟悉万象无常牌的人很容易辨认废墟蚀灭蛛——蛛腹外壳上的花纹与无常牌的卡背插画一模一样。

废墟蚀灭蛛 Ruin Spider

*大型怪兽，无阵营*

AC 16（天生护甲）

HP 105（14d10+28）

速度40尺，攀爬40尺

力量16（+3）敏捷18（+4）体质14（+2）

智力2（-4）感知13（+1）魅力4（-3）

技能：隐匿+10

伤害免疫：强酸

感官：黑暗视觉60尺，被动察觉11

语言：—

挑战等级5（1,800XP）熟练加值+3

**毁蚀酸液 Ruinous Acid。** 任何击中蚀灭蛛的非魔法武器都会被腐蚀。造成伤害后，该武器所作的伤害掷骰将永久承受-1减值且累计叠加。如果减值总和达到-5，则该武器被摧毁。由金属制成的非魔法弹药在击中蚀灭蛛并造成伤害后将随即被摧毁。 

**蛛行 Spider Climb。** 蚀灭蛛可以攀爬在难以攀爬的表面上，包括倒吊在天花板上，且不需要进行属性检定。

**蛛网感知 Web Sense。** 与蛛网接触时，蚀灭蛛可以知晓其他与该蛛网接触生物的准确位置。

**蛛网行者 Web Walker。** 蚀灭蛛在蛛网上移动时无视蛛网造成的移动限制。

动作 Actions

**多重攻击 Multiattack。** 蚀灭蛛发动两次毁蚀啃咬攻击。它可以将其中一次攻击替换为使用蛛网。

**毁蚀啃咬 Ruinous Bite。** 近战武器攻击：命中+7，触及5尺，单一目标。伤害：8（1d8+4）穿刺伤害外加9（2d8）强酸伤害。若目标为一个着装了非魔法护甲的生物，则其护甲提供的 AC 还将永久遭受-1减值且累计叠加。护甲提供 AC 被降至10时，该护甲将会被摧毁。

**蛛网 Web（充能5~6）。** 远程武器攻击：命中+7，射程30/60尺，单一生物。伤害：目标受蛛网束缚。以一个动作，被束缚者可以进行一次 DC 13的力量检定，检定通过则挣脱蛛网。蛛网可以被攻击和被破坏（AC10，5点 HP，火焰易伤，免疫钝击、毒素和心灵伤害）。
