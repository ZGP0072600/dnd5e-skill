# 利爪裂法兽

**利爪裂法兽 Talon Beast**

我真希望利爪裂法兽能用它们吸收的魔法变出点毛来，或者至少变出点衣服！但很可惜，它们只能是这种令人不安的裸体形象！

利爪裂法兽是一类身无毛发的四足生物，有着秃鹫模样的脑袋和禽爪。它们能感知魔法，吸收法术和魔法物品的力量，并以此为食。只靠魔法不足以维持其生存，故而它们也会捕食其他生物。利爪裂法兽于荒野中栖居，但偶尔也会因寻找魔法而徘徊在聚居地，古老废墟，地下墓穴甚至是幽暗地域。

许多学者认为，利爪裂法兽，利爪星座（见第十三章）以及万象无常牌中的利爪 Talons 牌之间存在着某种联系，但并不清楚其中哪一个才是最先出现的。利爪牌可能是因能摧毁魔法物品而被以这种怪兽的名字命名，但反过来利爪裂法兽的名字也可能正是取自利爪牌。

利爪裂法兽 Talon Beast

*大型怪兽，无阵营*

AC 14（天生护甲）

HP 114（12d10+48）

速度40尺

力量22（+6）敏捷14（+2）体质19（+4）

智力5（-3）感知12（+1）魅力5（-3）

技能：察觉+4

状态免疫：恐慌

感官：黑暗视觉60尺，被动察觉14

语言：—

挑战等级7（2,900XP）熟练加值+3

**魔法抗性 Magic Resistance。** 利爪裂法兽为抵抗法术和其他魔法效应而作的豁免检定具有优势。

**魔法感知 Sense Magic。** 利爪裂法兽能侦测并定位其120尺内的魔法。*（译注：原文只有一个 magic，大概只能自行解读）*

动作 Actions

**多重攻击 Multiattack。** 利爪裂法兽发动两次禽爪攻击和一次喙啄攻击。

**喙啄 Beak。** 近战武器攻击：命中+9，触及5尺，单一目标。伤害：17（2d10+6）穿刺伤害，且若目标身上有任意法术的效应或是拥有任意魔法物品，则其必须进行一次 DC 15的魅力豁免。豁免失败则其身上随机一项法术的效应结束。若目标身上没有法术效应，则随机一件其拥有的魔法物品的魔法属性被压制1分钟。若这个魔法物品是药水或卷轴，则改为该物品失去其魔法属性。

**禽爪 Talon。** 近战武器攻击：命中+9，触及10尺，单一目标。伤害：15（2d8+6）挥砍伤害，且目标陷入擒抱状态（逃脱 DC 17）。被擒抱期间目标还会陷入束缚状态，且利爪裂法兽不能对其他目标使用禽爪攻击。
