# 秃鹫人

秃鹫人 
Werevulture

学者们认为，秃鹫人诅咒与鹫魔 Vrock 有关。鹫魔是一种形似秃鹫的恶魔，以其贪婪与残暴而闻名。据学者们所言，最初，一群鹫魔穿过位面间的裂隙，降临到物质位面，在世间留下了瘟疫与苦难。许多被这些恶魔伤害的人幸存了下来，并极度感激其神灵为其带来的好运。然而，就在下一个月圆之夜，它们却被转化为了最初的秃鹫人：渴求死者尸肉的丑陋鸟人。自那以后，这种诅咒便在世间传播开来。

类人形态下的秃鹫人外形十分普通。然而在满月之时，或是其他其屈服于诅咒的时候，它们便会转变。肮脏的翅膀痛苦地在其背后胀开，头发从脑袋上凋落，只留下一副长着鸟喙且布满皱纹的面容。秃鹫人像普通秃鹫一样以腐肉为食，但它们还有着如鹫魔般的贪婪天性。它们在高空跟踪旅行者，而当漂亮的小玩意吸引其注意时，它们便会迅速降下，杀死旅行者并拿走受害者的宝藏。

秃鹫人 Werevulture

*中型邪魔，通常混乱邪恶*

AC 12

HP 75（10d8+30）

速度 类人形态30尺；秃鹫形态10尺，飞行60尺；半人形态30尺，飞行60尺

力量16（+3）敏捷14（+2）体质17（+3）

智力11（+0）感知13（+1）魅力8（-1）

技能：察觉+5

感官：被动察觉15

语言：通用语（不能在秃鹫形态说话）

挑战等级4（1,100XP）熟练加值+2

**再生 Regeneration。** 秃鹫人在它的回合开始时回复10点生命值。如果秃鹫人受到光耀伤害，则该特质在它的下一回合里无法生效。秃鹫人只会在以0生命值开始其回合且无法再生时死亡。

动作 Actions

**多重攻击 Multiattack。** 秃鹫人发动两次禽爪攻击，或是一次禽爪攻击和一次喙啄攻击。

**喙啄 Beak（仅秃鹫与半人形态）。** 近战武器攻击：命中+5，触及5尺，单一目标。伤害：10（2d6+3）穿刺伤害。若目标为类人生物，则其必须成功通过一次 DC 13的体质豁免，否则被诅咒，直至被法术移除诅咒 Remove Curse 或类似效应指定为目标。若被诅咒的目标降至0生命值，它会变为一个由 DM 控制的秃鹫人 Werevulture 并恢复10点生命值。

**禽爪 Talon。** 近战武器攻击：命中+5，触及5尺，单一目标。伤害：13（4d4+3）挥砍伤害。

**长弓 Longbow（仅类人与半人形态）。** 远程武器攻击：命中+4，射程150/600尺，单一目标。伤害：11（2d8+2）穿刺伤害。

附赠动作 Bonus Actions

**变形 Change Shape。** 秃鹫人变身成一种半秃鹫半类人的半人形态，或秃鹫形态，或者以同样的动作变回其类人生物的真实形态。其资料除速度外各个形态均相同。其着装及携带的装备不会随之转化。死亡时它将恢复其真实形态。
