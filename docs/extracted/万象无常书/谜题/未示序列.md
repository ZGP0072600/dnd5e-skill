# 未示序列

未示序列 Unseen Order

*难度：中等*

(译注：该谜题名称的 Unseen 旨在“未公开”，指代手牌、盖卡、或是卡组等“对手无法确认的卡牌”，而非该词常意)

这个猜谜游戏需要有着不得不解谜的重要原因的角色们来推测出正确的四卡序列。他们必须使用一面游戏板与6组卡牌来进行猜测。

给予玩家们一份未示序列传单1与传单2，然后响亮的阅读以下的框内文本：

“猜猜手牌，赢下游戏。

 摆下猜测，我将提示：

 卡对位对？一球变红！

 卡对位错？那就变蓝。

 十次机会，不可作弊，祝你好运。

 若是能在，六次以内，猜对谜题...

 那还能有，额外大奖！呜—呼—！”

谜题特性 Puzzle Features

桌上的这些牌是非魔法版本的 万象无常牌 Deck of Many Things ，如果角色完全无法理解这个谜语在说什么，鬼魂就会更详细的讲述出来：

**目标 Objective。** 角色必须猜出鬼魂手中的牌是哪些，以及那些牌的顺序。

**规则 Rules。** 要进行它们的首次猜测，角色必须将游戏版上堆着的卡中的4张，放置到第一列的空槽里，鬼魂手里的牌也许是4张不同的，也许有2张牌相同的，甚至3张或全部都是相同的。如果在此行放入了一张正确的卡，且这卡在4张卡中的顺序位置与鬼魂手中的牌相同，那么右侧就会有一颗小球会亮红。如果放入了一张正确的卡，都这卡的顺序位置不对，那么右侧就会有一颗小球会亮蓝。依照以上的规则，角色可以一行一行猜下去，直到猜出答案为止。

**赢得游戏 Winning the Game。** 这里一共有10行空槽，意味着玩家只有10次机会来猜鬼魂手中的卡牌。如果他们用不超过10次行就猜出了顺序，他们将赢得一份由你抉择的宝藏。如果角色只用了不超过6次就猜出了谜题的解，鬼魂将告知角色们一份更大更好的宝藏，就在他们冒险中的某处地点上。

**输掉游戏 Losing the Game。** 如果玩家用完了10次猜测机会，未能猜出答案却还想猜测。鬼魂会重新挑选4张牌，游戏重新开始。

**其他材料 Other Materials。** 这一谜题你并不需要实体的卡牌就能使用。你只需要用一些纸、一根红笔、以及一根蓝笔来模拟它。在一张纸上写下谜题的，并将纸放在玩家看不见的地方。

 给予玩家一份未示序列传单1或一张画有表格的纸张。然后告知他们将猜测的卡牌符号自上至下的画入每一行中，卡牌符号在未示序列传单2上有写出。偷偷对比纸张上的答案与玩家猜测的顺序，并在他们的猜测右边的圈圈中，用红笔涂满一个圈来表示卡牌正确位置也正确，用蓝笔涂满一个圈来表示卡牌正确但位置不正确。

提示检定 Hint Checks

任何角色都可以进行以下属性检定来获取谜题的提示：

 · 魅力（说服）DC 10。如果角色们礼貌的向鬼魂询问，鬼魂会举例说明这一谜题是如何运作的。向玩家展示图4.4，并告知它们图上所示的解与实际谜题的解不同。

自定义谜题 Cuztomizing the Puzzle

只要保证基本规则不变，谜题就可以改为各种不一样的形态，像是用国际象棋的棋子、会是不同形状的代币之类的物品代替卡牌来猜测顺序。

 · 增加难度 Increasing the Difficulty。你可以增加鬼魂手中的卡牌数量或是增加可选牌的数量，以此增加谜题的难度。每个变量的增加都会导致游戏变得更为困难，你可以考虑在增加难度的同时增加角色可以进行的猜测次数。
