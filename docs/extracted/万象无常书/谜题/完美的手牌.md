# 完美的手牌

完美的手牌 The Perfect Hand

*难度：简单*

(译注：该谜题名称的 Hand 意为“手牌”或是“手中持有的牌”，而 Prefect Hand 就是传说中的“完美的手牌”——不，不是游戏王那个，肯定不是那个梗)

这套谜题可被用于放置在隐藏宝藏或是秘密通道的入口前。它非常适合用于地牢、闹鬼的老房子、以及废弃的赌场中。

“仍在寻找完美的手牌？找到4张遗失的卡并把它们带给我。阴险狡诈的 浪客 Rogue 正准备背刺贤者。神神颠颠的 贤者 Sage 叨叨着小丑的卑劣，完全不想身处那个小丑的左右。愚蠢至极的 小丑 Jester 正做着倒立杂技，眺望着远处的骑士。忠心耿耿的 骑士 Knight 誓死守卫在贤者身旁。而在那小丑身后，那藐视一切的 命运 Fates 三女神如蝙蝠般倒悬于天空之下。”

谜题特性 Puzzle Features

这尊雕塑会在任何人任何时间靠近时重复上述那段信息。

桌子上的这张卡，是 万象无常牌 Deck of Many Things 的 浪客 Rogue 卡的非魔法版本。而谜题所需的另外4张卡——贤者 Sage、小丑 Jester、骑士 Knight、命运 Fates——分别散落在这周围的不同区域内。

雕塑双手的插槽中可以正正好好放上数张卡牌。而每张卡牌都可以以正位、或是逆位（上下颠倒）的放入插槽中。

解 Solution

角色们必须找齐丢失的那4张卡片并返回这里。利用雕塑提供的信息，这5张卡必须以某种特定的顺序与特定的摆放方式（正位还是逆位）放在雕塑手中。图4.1展示了谜题的解。

如果雕塑手中的牌，其顺序或摆放不正确，雕像会发出空洞的砰砰声来提醒手牌错误。而一旦牌的顺序与摆放均正确，这尊雕塑变回向旁边挪开，并向角色们展露出她背后的密门。

提示检定 Hint Checks

任何角色都可以进行以下属性检定来获取谜题的提示：

 · **智力（调查）DC 10。** 这些卡片上的图案似乎是有朝向的，也许放到雕像手里时，其中的某些得上下颠倒反过来放才行。

 · **感知（洞悉）DC 10。** 这尊雕塑的信息好像在陈述一套卡牌的摆放顺序，包括谁该摆在谁边上。

 · **感知（察觉）DC 10。** 这尊雕塑的手，似乎正好可以拿上一手开扇的5张卡牌。

自定义谜题 Cuztomizing the Puzzle

因某些重要原因，角色必须去猜测某些物品的摆放顺序。这些物品并不一定需要是卡牌——他们也可以单纯是一些雕塑或是墙上的挂画。只要角色们拿到了足够的信息而不需要猜测，这个谜题就能被修改为任何你想要的样式。

 · **丢失卡牌 Missing Pieces。** 你可以选择让某些卡片不在附近，需要去某处寻找。举例来说，为完成这一谜题，角色也许需要去探秘一张藏宝图，或是通过一座迷宫才能找到这些卡。你还可以让这些卡牌分散在各种各样的地方来增加难度。具体来说，你可以让这些卡存在于地牢中的不同雕像或是羊皮纸碎片堆中。

 另一种增加此谜题难度的方式就是增加卡牌的获取难度。也许角色还得去讨伐某个怪物、或是完成与某项技能有关的试炼才能拿到这些卡牌。

 · **起手抽满 Starting with a Full Hand。** 为让这个谜题更简单更便捷，你可以让5张卡都出现在雕塑旁的石桌上，这样角色们就可以立刻开始尝试这些卡牌的顺序而不需要再四下寻找这些卡牌了。
