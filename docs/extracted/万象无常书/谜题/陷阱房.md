# 陷阱房

陷阱房 Trap Rooms

陷阱，往往如谜语与迷宫般谜惑，却远比这些致命！这一章节包括了一些陷阱房：各种装满了危险道具的，测试冒险者智慧与能力的地城房间。一间陷阱房包括了一系列专门连锁或组合运作的陷阱，每个陷阱也许对角色而言构不成什么大碍，但当这些元素组合在一起，它们将形成一套独一无二且令人难忘的挑战。

不过陷阱房往往不会出现在什么废弃的庄园或是失落的庙宇中；只有在地下城中才会出现这样的房间，用以来守护那些真正珍贵的宝藏，或是惩戒贪心的冒险者。

以下介绍的每个陷阱房都包含了其难度适配的角色等级范围，以及陷阱房的各种特性，以及一张呈现该房间设计的插图。你可以用《城主指南》的“陷阱”指南来调整这些陷阱的危害度。
