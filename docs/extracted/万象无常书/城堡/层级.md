# 层级

层级 Layers

球牢城堡的中空部分分为几个嵌套在一起的层级，如图17.1所示。由金属隧道和齿轮组成，只有球牢中专门编程过的拘禁机器才能在其复杂网络中导航，将这些不同的层级分开来。

**外层表面 Outer Surface**

球牢的外层表面十分光滑，由数以百计的罕见太空金属板制成。当宇宙碎屑或攻击者破坏表面时，修复无人机会在数个小时内更换损坏的金属板。在球体的赤道附近有一个50尺宽的圆形舱口，这里是表面唯一的入口。无人机进出球牢时，舱门会自动开启和关闭。

**镀金迷宫 Gilded Labyrinth**

黄铜金属铸造的三维迷宫将表面和内部层级割开。这个所谓的“镀金”迷宫延缓了囚犯脱逃。在被球牢中的拘禁无人机抓回之前，很少有逃离的囚犯能在镀金迷宫中待上很久。这些无人机了解迷宫中的每一寸。

此外，像大脑的皱褶一样，镀金迷宫无数的反射表面和边缘放大了球牢核心辐射出的无形能量。大多数生物无法察觉这种效应，但收集能量对球牢持续运作至关重要。

**能量核心 Power Core**

球牢城堡的核心是一个巨大的灼热球体，由底层核心区域33的位面传送门创造。整个球牢的依靠这个火球产生的能量运作。
