# 前言

第九章：骑士 
Knight

从*万象无常牌 Deck of Many Things*中抽出 **骑士 Knight** 牌的角色会因此得到一名忠诚战士的协助。这一章包含了为 DM 们准备的各类建议，帮助他们将这位强大的 NPC 融入到战役中正在进行的故事里，当然，还包括了一些如何防止这位强大盟友过度抢占风头导致玩家体验下降的通用建议。这一章同样提供给了一个新的生物，作为抽出这张卡时可用的召唤对象：一名属性与抽出骑士卡者息息相关的、由卡牌构成的构装生物。

除 DM 建议之外，这一章还包含了一些供那些毕生钻研格斗的角色们可用的材料：一组魔法物品，与一些受到*万象无常牌 Deck of Many Things*
 
 启发的背景元素。
