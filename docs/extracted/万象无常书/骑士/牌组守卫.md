# 牌组守卫

牌组守卫 Deck Defender

牌组守卫是以数张大型卡牌折叠而成的构装生物。你可以在玩家抽到 骑士 Knight 卡时，让卡牌召唤出一只牌组守卫来作为骑士卡提供的那位友好盟友。

自 **万象无常牌 Deck of Many Things** 的多重魔法所造的，这个牌组守卫会听从抽出骑士卡的角色之命令。它能变化为多种形态：召唤时，它会以手持剑盾的、偏防守性的守护者形态。而召唤出牌组守卫的角色也可以根据需要，将这名守卫折成更具进攻性的狂战士形态，或是更加注重灵活性的特技师形态。

无关乎卡牌守卫的形态，它的类卡牌本质从不会发生变化。若一个牌组守卫被击败，它也将化为一滩飞散的非魔法卡牌。

**牌组守卫 Deck Defender**
*中型构装，无阵营*

护甲等级：16（天生护甲）

生命值：5+你等级的五倍（牌组守卫拥有等同于你等级数量枚的 d8生命骰）

速度：30尺

力量16（+3） 敏捷16（+3） 体质14（+2）

智力3（-4） 感知10（+0） 魅力1（-5）

伤害免疫：毒素

状态免疫：目盲，魅惑，耳聋，力竭，恐慌，麻痹，石化，中毒

感官：盲视60尺（此范围外不可视），被动察觉10

语言：理解你的语言但不能说

挑战等级：——

熟练加值（PB）：等同于你的熟练加值

**友之骑士 Allied Knight。** 牌组守卫的一部分数据基于抽取那张骑士卡的角色决定。牌组守卫面板中的所有“你”一词均指代抽出那张卡的角色。

**万变折纸 Folded Versatility。** 在完成一次长休后，你可以重塑卡组守卫的外形，将其转化为他的特技师形态、狂战士形态、亦或是防卫者形态。

**易碎 Fragile。** 如果牌组守卫的生命值被降至0。它将化为一团无害的非魔法卡牌并消散。且无法被复活或重构。

动作 Actions

**多重攻击 Multiattack。** 牌组守卫发动 你熟练加值一半（向下取整） 次的攻击。

**纸裁刀 Paper Cut。** *近战武器攻击*：命中+PB+3，触及5尺，单一目标。*命中*：7（1d8+3）点挥砍伤害。

**鲁莽强袭 Reckless Strike（仅限狂战士形态）。** *近战武器攻击*：命中+PB+3（且具有优势），触及5尺，单一目标。*命中*：10（2d6+3）点挥砍伤害。且命中后直到牌组守卫的下个回合开始前所有向牌组守卫发动的攻击都将具有优势。

附赠动作 Bonus Actions

**灵巧步伐 Swift Step（仅限特技师形态）。** 牌组守卫执行 疾走 Dash 或 撤离 Disengage 动作。

反应 Reactions

**守护 Protection（仅限防卫者形态）。** 当一个牌组守卫可以看见的生物，对牌组守卫周围5尺范围内的其他生物进行攻击时，牌组守卫能迫使此次攻击承受劣势。

（译者注：与钢铁守卫的需要敌人在守卫身边不同，这个是需要被守护者在牌组守卫身边）
