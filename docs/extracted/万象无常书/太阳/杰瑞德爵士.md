# 杰瑞德爵士

**杰瑞德爵士 Sir Jared**

杰瑞德是一个流浪骑士，他和炽天之壁赞助的队伍都拥有良好的联系。作为一名前雇佣兵，杰瑞德认为希拉里恩帮助他改变了他的生活方式，将他的技艺投入于做善事而不是为钱财，但他认为自己不配成为这个高尚组织的一员，在离开炽天之壁那么长的一段时间之后，他变得孤独了。

动作 Actions

**多重攻击 Multiattack。** 杰瑞德发动三次长剑攻击

**长剑 Longsword。** 近战武器攻击：+7命中，触及5尺，单一目标。命中：8（1d8+4）挥砍伤害加4（1d8）光耀伤害。杰瑞德的攻击检定骰出19或20时都视为重击。

反应 Reactions

**保护盟友 Protect Ally**。当某个杰瑞德能看见的生物攻击杰瑞德身边5尺内除杰瑞德之外的目标时 
，他可以用反应迫使该次攻击的攻击检定具有劣势。
