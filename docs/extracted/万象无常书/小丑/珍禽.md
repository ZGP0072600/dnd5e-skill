# 珍禽

**珍禽 Rare Bird**

**树精 drya**d“好天气”展示了她摊位上的鸟儿。她这儿有猫头鹰、山雀、猎鹰和蜂鸟等等，由于在妖精荒野度过了一段时间，它们都有着奇特的色彩，歌唱着不同寻常的旋律。一些鸟儿坐在笼子里，而另一些则在露天的栖木上跳来跳去，或绕着好天气飞来飞去。

**价钱 Price**

 如果角色成功向一只鸟儿表达了自己的心迹，好天气就会将它交给角色。想要特定鸟儿的角色可以靠近这只鸟并进行 DC 15的感知（驯兽）或魅力（表演）检定。一次成功的检定中，他们赢得了那只鸟儿的喜爱，而好天气会将它送给对应的角色。她只会给每个角色一只鸟，但如果角色没能对鸟儿表白成功，他们可以对不同的鸟儿尝试，直到成功为止。

** 给雨果牵线 Matchmaking with Hugo。**好天气一直暗恋着雨果·裂角，但由于担心雨果不会同样喜欢自己，她没有表达自己的感受。根据雨果的要求来将他引见给好天气的角色可以成功通过 DC 12的感知（洞悉）检定发现她和雨果一样暗恋对方，因此让她去拜访雨果十分容易。很多方法都可能奏效，包括表达他希望见她的兴趣，偷走她的什么东西放到雨果的摊位边，或者为雨果和好天气见面编造个好借口。
