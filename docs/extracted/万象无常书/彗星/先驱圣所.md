# 先驱圣所

先驱圣所 Sanctums of the Heralds

彗星先驱的大多数会面总是同一届的门徒聚在私人住宅里解读卡牌占卜，寻找着大灾变的先兆。不过，入会仪式却是在圣所——一些位置偏远的据点——举行的，如地图12.1所示。圣所是一座神殿、会面场所和供教宗和作为侍从的门徒所住的居所。圣所通常距离大城市有几英里远，这允许秘会举行隐秘的聚会，并让成员能在一两个小时内于城市与圣所间往返。

*地图12.1 邪教据点 Cult Hideout*

圣所地点 Sanctum Locations

地图12.1中所示的建筑是一座建在一处天然洞穴的开口前的大宅，由一位富裕的渴求者捐赠给彗星先驱。宅子的一楼包括一间奢华的客厅、一间厨房和宽敞的餐厅，以及一楼的一间卧室套房。二楼（未绘于地图上）则有着更多的卧室和一间书房。

**前厅 Antechamber。** 在通往卧室套房的走廊尽头，一扇装饰华丽的玻璃彩窗描绘了吞噬万物之星，如同万象无常牌中的彗星 Comet 牌一般。在宅子的西北角，有一个供渴求者们在入会仪式正式开始前聚集的前厅。奥秘图画装饰着房间内的墙壁和支柱，而万象无常牌中的每一张卡的卡图都装饰在洞穴入口所在的西墙上。

**洞穴 Cave。** 这个蜿蜒的洞穴大部分保持着自然本态，但秘教成员拓宽了狭窄的通道，并用砖石支撑了不稳的洞壁。钟乳石悬挂在天花板上，不过大多数石笋都已清理干净，以便通行。少量天然石柱仍然留存着。

这里有着入会仪式中需要用到的两个天然水池：新晋门徒会在较大的水池中沐浴，然后在较小的水池中饮水，而据传水池中的冷水和苦水将使它们更容易接受在最后一个房间等待着它们的景象。

洞穴隧道蜿蜒而下，穿过洞穴的小路稳步下降；最后一个房间的地面比宅子的地面低大约25英尺。路边大约每隔8英尺便有金属圆盘嵌入洞壁上。这些圆盘上有许多事物牌的标志，当门徒向着吞噬万物之星的最终启示靠近时，这些图纹便会作为秘教信条的象征性提醒。

**具现秘阁 Manifestation Chamber。** 最后的房间禁止任何人进入。在入会仪式的高潮时刻，教宗使用吞噬万物之星 All-Consuming Star 动作项（见彗星教宗 Hierophant of the Comet
的数据卡），令这个饕餮实体占据房间的绝大部分空间。此时，任何敢于进入房间者都有被吞噬殆尽的风险。
