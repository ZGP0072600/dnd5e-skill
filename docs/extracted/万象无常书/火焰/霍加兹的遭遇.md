# 霍加兹的遭遇

遭遇霍加兹 Encounters with 
Hulgaz

对热衷于角色扮演的玩家而言，霍加兹是一个很好的反派。如果你的玩家更喜欢敌人交谈而非交战，便可考虑在战役中使用霍加兹。

你可以运行以下几种将霍加兹介绍给玩家的遭遇。这些遭遇需在几次团里逐步展开——霍加兹首先测试角色们的德行，之后试图诱惑他们，直至最后的对峙。在每次相遇的结尾，霍加兹都会给予角色一个或更多的炼狱祝福。大多数关于霍加兹的遭遇都着重于扮演，角色的等级在此并非关键，但与诱惑者的最终决战应当在角色们抵达11级或12级时展开（取决于队伍里有多少角色）。

骑士诈术 Deceptive Knight

当角色们旅行穿过一处偏远地带时，它们遇到了一位被三名强盗包围的骑士。骑士似乎已经负伤。事实上，骑士和强盗是四个以魔法易容的魅魔 Succubus，上演这出戏码为的是替霍加兹考验角色们的善心。骑士会大声呼救，看起来显然是遭到了强盗的袭击。

魅魔们与角色的互动取决于角色们此时的反应：

**出手相助。** 若角色们介入，骑士会感谢它们的勇气并提出要报答它们。

**袖手旁观。** 若角色们什么都不做，强盗会大笑着向作为“共犯”的玩家们提供从战败的骑士身上搞来的礼物作为临别赠礼。

**识破真相。** 若角色们看穿了诡计，魅魔们会变回其真实形态，并用奖励以赞许角色们的

判断力。

无论如何，角色中的一员都会得到从两种炼狱祝福中选择一个的机会。魅魔会暗示这份礼物来自其女主人霍加兹，但不会为此透露更多。无论角色是否接受祝福，魅魔们都会在交流结束之后离去。

危急时刻 Time of Need

在一场极为艰难的战斗中，霍加兹在团队眼前现身。她传送至最艰难的角色面前，并在战斗中随性提供帮助，而她所要求的回报只是让角色接受她的炼狱祝福。若角色们拒绝她的提议，霍加兹会失望地“啧”一声（原文为 tsks），召唤两个魅魔攻击队伍，并传送离开。

歧路对决 Crossroads Confrontation

霍加兹的最终登场会在距上次会面很久之后。这场遭遇会发生在一个合适且十分戏剧性地点，诸如一个废弃的乡间路口，而角色们在此之前已经确信自己摆脱了霍加兹。

霍加兹提供最后一次交易。她为每个角色都制作了一些代表她的炼狱祝福的小饰品。霍加兹把这些礼物赠给角色们以示尊重。为了强调她对角色们的敬意，每个饰品都是量身定制的，符合角色的志向与目标。

与往常一样，这些祝福既带来恩赐也带来了诅咒。但这一次，霍加兹不会接受角色们的拒绝。如果角色拒绝接受炼狱祝福，霍加兹会遗憾地摇头，并说道，如果她无法说服角色们接受九狱的邀请，她就不得不亲手把角色们送进地狱。随后，霍加兹不顾一切地疯狂进攻。
