# 奥诺茨的巢穴

奥诺茨的巢穴 Aurnozci's 
Lair

奥诺茨的巢穴是无底深渊中的一个位层，叙勒戈（详情见下文）。

当在巢穴中遭遇时，奥诺茨的挑战等级提升至23（50,000XP）。

巢穴动作 Lair 
Actions

在先攻值20（相同则靠后）时，奥诺茨可以使用以下一项巢穴动作；它不能在两轮内使用同一项巢穴动作：

**魔茧缚身 Cocoon。** 奥诺茨指定一个位于其60尺内的它可见的生物，并令目标陷入虫茧的束缚之中。目标必须成功通过一次 DC 23的力量豁免否则陷入束缚状态1分钟。以此法被束缚的生物会在每个它的回合开始时受到9（2d8）钝击伤害。一个生物可以用动作来尝试撕开虫茧，进行一次 DC 23的力量（运动）检定，成功则结束此效应。

**灾祸虫群 Insect Swarm。** 奥诺茨指定300尺内它可见的一点，创造一群蜇人的毒蜂，覆盖一处以该点为中心半径30尺的球形区域。毒蜂覆盖的区域会沿拐角扩散。毒蜂区域持续10分钟或直至奥诺茨再次使用此巢穴动作。毒蜂区域成为困难地形，且处于重度遮蔽。当毒蜂出现时，任何位于区域内的生物都必须进行一次 DC 20的体质豁免，豁免失败者受到16（3d10）穿刺伤害外加16（3d10）毒素伤害，豁免成功则伤害减半。在一个回合中第一次进入该区域或在该区域内开始回合的生物也必须进行此豁免。

区域效应 Regional 
Effects

叙勒戈是一处极为闷热的国度，具有以下区域效应：

**沸腾水体 Boiling Water。** 叙勒戈中的水体永远处于沸腾之中，甚至暴露在空气中的瓶装液体也会迅速冒烟。浸入水体之中不再会提供对火焰伤害的抗性，而在水中开始回合的生物会受到7（2d6）火焰伤害。此外，喝下液体（比如一瓶药水 Potion）的生物还会受到3（1d6）火焰伤害。

**高速蜕变 Fast Metamorphosis。** 叙勒戈中的 **蛋 Egg**，** 幼虫魔 Larva**和 ** 蛹 Pupae**都会以比通常快10倍的速率发育或蜕变（不会被营养或生长条件限制）。

如果奥诺茨死去，这些效应将会在 1d10日内消失。
