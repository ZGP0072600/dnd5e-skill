# 玛拉希克斯的诅咒

玛拉希克斯的诅咒 Curse of Malaxxix

同调了玛拉希克斯的诅咒物品的生物会被玛拉希克斯的诅咒影响。这些诅咒令他可以用特殊的方式与被诅咒的生物互动：

> **探知术 Scrying。** 玛拉希克斯可以对被诅咒的生物施展法术探知术 Scrying 且无需任何材料成分与法术位。使用此能力时玛拉希克斯不必与被观测的生物位于同一位面，且该生物自动失败于该法术的豁免。
>
> **传送 Teleport。** 以一个动作，玛拉希克斯可以与其携带或着装的装备一同传送至位于被诅咒的生物60尺内的一处未被占据的空间，无论被诅咒者在多元宇宙中的何处。
>
> 玛拉希克斯的诅咒持续直至被一道移除诅咒 Remove Curse，一道高等复原术 Greater Restoration，或其他类似的魔法解除。
