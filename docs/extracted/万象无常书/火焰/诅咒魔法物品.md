# 诅咒魔法物品

诅咒魔法物品 Cursed Magic Items

此处是两个玛拉希克斯制作并置入物质位面的魔法物品，用以坑骗那些毫无戒心的凡人。除了列出的效果，这些物品中的一个或所有这些物品都可能蕴含了玛拉希克斯的诅咒（见上文）。

爆裂目镜 Blasted Goggles

*奇物，非普通（需同调）*

这副工匠护目镜有3发充能。以一个动作，你可以消耗一发充能并从目镜中向一个位于你120尺内生物射出一道烈焰光线。目标必须成功通过一次 DC 15的敏捷豁免否则受到3d6火焰伤害。目镜在每日黎明恢复1d3发已消耗的充能。

**诅咒 Cursed。** 目镜被诅咒了，而与目镜同调会令你也被诅咒。你无法摘下目镜，直到同调结束或者你被法术移除诅咒 Remove Cures 或类似魔法指定为目标。

每当你使用目镜的烈焰光线且目标用于此豁免的 d20骰出了20时，目镜都会令你被强烈闪光刺目。你陷入目盲状态，持续24小时。

尖啸腿甲 Shrieking Greaves

*奇物，珍稀（需同调）*

这对漆黑的腿甲上装饰着无数恐惧中尖叫的可怖颅骨。

这对腿甲有3发充能。着装这对腿甲期间，你可以用一个附赠动作，消耗1发充能以令你的步行速度增加30尺且在敏捷豁免上具有优势。这些效应持续1分钟。腿甲在每日黎明恢复1d3发已消耗的充能

**诅咒 Cursed。** 腿甲被诅咒了，而与腿甲同调会令你也被诅咒。你无法卸下腿甲直到同调结束或者你被法术移除诅咒 Remove Cures 或类似魔法指定为目标。

你在为抵抗恐慌状态而进行的豁免具有劣势。每当你在陷入恐慌期间开始回合时，腿甲都会释放一道刺耳的尖啸。你和每个位于你10尺内的生物都必须进行一次 DC 15的体质豁免，豁免失败者受到9（2d8）雷鸣伤害，豁免成功则伤害减半。
