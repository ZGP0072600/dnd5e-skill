# 诸神与凡人的国度

**神界与凡界 Realms of Gods and Mortals**

*“世界上已知最遥远的地方在哪里？”艾紫培问道。“在东方，那狮族的土地更远处有另一片森林。”达克索斯回答她，“它比奈西安森林还要辽阔，没人知道它延伸至何方。”“那么西方的海对岸有什么？”艾紫培问道。“那是克罗芬斯之树生长之所，世界在那瀑布处断绝。”他解释道，“海洋自那边界向下，坠入虚空。”“我的家乡就在你说的那片无垠森林之外。”她说*。

*——Jenna Helland, Godsend*

塞洛斯的住民所了解的世界包括三大领域：凡界、天界、以及冥界。它们是藏在各自的多元宇宙空间里三个不同的实体世界，因神祇的力量而与宇宙的其他部分隔绝。这一章节将依次描述每一个领域，不过重点放被诸神密谋交织着的、众生所在的凡界。
