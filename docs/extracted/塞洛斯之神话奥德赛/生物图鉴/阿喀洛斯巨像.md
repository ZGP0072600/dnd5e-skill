# 阿喀洛斯巨像

一个青铜与铁铸的巨大魔像俯瞰着通往阿喀洛斯城邦的道路。虽然很少有人召唤它去保卫城邦，但它高耸的外形足以让民众安心。在真正绝望的时候，普罗烽斯的牧师用他们的魔法召唤巨像复活，于是大地隆隆作响，从它的双基座上走下来，面对威胁。构装本质。阿喀洛斯巨像不需要空气、食物、水以及睡眠。
