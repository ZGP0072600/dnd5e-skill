# 祸蹑行者

祸蹑行者由那些被切断命运纽带的人的灵魂所化。数个世纪间，这些亵渎宇宙秩序的存在逐渐蜕变为身形佝偻、肢体瘦长的可怖怪物。作为天性嗜虐的生物，祸蹑行者一心寻求将自身重新编织回命运网络的办法。为达成此愿，它们会深入其他生物的躯体内部探寻答案，通过残忍的脏器占卜来追寻自己被宇宙遗弃的宿命。当未能找到答案时，它们那非自然的嘶吼会令现实震颤，瓦解魔法的同时还摧残心灵。
