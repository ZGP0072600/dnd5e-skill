# 统领

统领 Archons 被成为统领的神秘征服者层统治着广阔的帝国。这些全副武装的军阀将自己视为无情正义的拥护者，并以铁腕统治。但他们的统治最终结束了。随着统领们的君王倒台，他们分散到了世界的边缘。他们的领地发展至今。尽管统领的时代已经过去了很久，许多人仍想知道，幸存的少数统领是否有朝一日可能会重建自己的帝国，或者他们是否真的已经听任自己在世界上的地位越发次要。星落统领 Archon
