# 绒毛狮

绒毛狮的体型是普通狮子的两倍，拥有华丽的银色或金色鬓毛，它们在蕴含天界力量的场所巡游并守护此地。尽管这些巨型狮子与天界之间的具体关联尚不明确，但诸多神话记载，这类致命的掠食者会追踪凡人并散播恐惧，直至被勇敢的猎手所击败。因此，战胜一头绒毛狮，被广泛视作踏上成为真正英雄之路的第一步。

敏锐嗅觉 Keen Smell。

绒毛狮进行依赖嗅觉的感知（察觉）检定时具有优势。

猛扑 Pounce。如果狮子向目标直线移动至少20 尺，并在同一回合对其发动爪击攻击且命中该目标，则目标必须进行一次 DC 14 
的力量豁免，豁免失败者将应击倒地。如果目标陷入倒地，则狮子可以用一个附赠动作对目标发动一次啃咬攻击。

助跑跳跃 Running Leap。狮子在10 尺助跑后可以跳25 尺远，且视为移动的一部分。

法术回旋 Spell Turning

狮子对任何仅以它为目标的法术(而不是一个区域)的豁免具有优势。如果狮子豁免成功，且法术等级为4或以下，则该法术对狮子没有效果，而法术目标锁定为施法者自身。

传奇动作

绒毛狮拥有2 
传奇动作，用于选择执行下列动作选项。每次执行传奇动作时，每个选项只可以选用一次，且只能在另一生物的回合结束时执行。绒毛狮在它的回合开始时恢复其所有已消耗的传奇动作数。

爪击 Claw：狮子进行一次爪击攻击。

吼叫 Roar（消耗2动作）：狮子发出魔法吼叫。在狮子周围60尺内的每一个能听到吼叫的生物，必须成功一次 DC 
12的感知豁免检定，或者陷入对狮子的恐慌，直到狮子的下一轮结束。
