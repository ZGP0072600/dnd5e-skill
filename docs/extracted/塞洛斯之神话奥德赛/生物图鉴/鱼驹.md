# 鱼驹

鱼驹 Hippocamp 鱼驹是海中的骏马，在无数传说中都是梭螺鱼人和其他海洋英雄的向导和坐骑。虽然这些好奇但谨慎的水生马自然地成群旅行，许多在梭螺鱼人社会中占有重要地位。由于被认为是塔萨的仆人，鱼驹体通常是不可预测的，但当他们被神的意志所驾驭时，他们巨大的速度和力量可以证明令人敬畏。鱼驹 Hippocamp 大型怪兽
