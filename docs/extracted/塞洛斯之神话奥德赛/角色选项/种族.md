# 种族

种族 
Races

译者的话：本人能力有限，译文并非能完全体现英文原本的含义与韵味。一切种族名以 D&D5e 民间通用译名为准，括号内为该种族在万智牌的译名。除狮族与半羊人这两支新的种族，其余的种族翻译参考已有翻译但并非完全照搬，部分种族特质及其译名参考怪物图鉴1.3V 和万智牌官方翻译，并按个人喜好加以调整。 如有任何意见请等待《Mythic 
Odysseys of Theros》翻译组的高质量翻译版。

各种各样的民族居住在塞洛斯的土地上。除了人类，《玩家手册》中提到的其他种族在塞洛斯是未知的，除非他们来自其他世界

本章提供了塞洛斯以下常见的可使用种族，以及除了人类之外所有种族的特质：

人类：塞洛斯的人类与其他时间中的人类相似：适应性强，雄心勃勃，而且非常多样化。

人马（人马）：人马利用狡猾与马的力量来更近一步地实现苟那与斐力兹族群的目标

狮族：狮族是宛如高傲雄狮般的猎人，他们中的许多成员都活在对神明的蔑视之中。

牛头人（牛头怪）：牛头人因嗜血而臭名昭著，他们之中即使是最不诉诸战争的避战者也有这样的名声。

半羊人（羊蹄人）：羊蹄人跟随自身的激情与追寻狂野的欢宴与全新的冒险。

梭螺鱼人（梭螺鱼人）：梭螺鱼人是一支神秘的，居住在海洋之中有着无尽探索欲望的种族。

（括号内的译名为万智牌译名）
