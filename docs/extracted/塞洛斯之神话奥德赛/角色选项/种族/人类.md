# 人类

人类 Human

在塞洛斯众多的凡人种族中，人类体现出的凡人意志从荒野中开辟出了一块文明之地。尽管有其他较小的居住地存在，但大多数人类都与塞洛斯的三大城邦: 迈勒提斯, 阿喀洛斯或瑟特萨中的一个有着紧密的联系。

姓氏在人类中并不常见，同名的人必须想办法把自己与他人区分开来。有些人把他们的出生地或居住地附在他们自己的名字上。例如，一个来自迈勒提斯的名为布莉戈涅的商人，将被称为迈勒提斯的布莉戈涅。另一些人则把他们的名字与自身职业联系起来，比如博学的黎司托和猎人阿拉斯塔。

然而那些最具威望的名字是通过行动或性格换得的。这些名字通常是英雄们的典型标记，例如翼马的希帕提娅，末日之锤的卡洛格罗斯，或勇威依安忒。
