# 阿喀洛斯

**阿喀洛斯 Akros**

*“唯有胜利长存”*

——阿喀洛斯人 穆托

高墙环绕的城邦阿喀洛斯桀骜的坐落在一座陡峭悬崖的顶部。险峻的群山围绕着它，充当它的要塞和塞洛斯其他地方之剑的盾牌。鲜少有人胆敢进犯它久负盛名的要塞，克洛丰，而且从未有过可以触及它城墙的攻击。对塞洛斯的居民来说，阿喀洛斯人拥有着接近传说的地位：一个以为了战争而完善肉体精神为中心的文化里产生的令人畏惧的战士。他们的军队在扩张阿喀洛斯的边境，攫取新的土地和财富时鲜有败绩。
