# 迈勒提斯历

**迈勒提斯历 Meletian Calendar**

迈勒提斯的天文学家和哲学家建立一个适用于阿喀洛斯和瑟特萨的历法。它将一年分为十二个二九天或三十天的月份，每个月都随着新一轮的月相开始。每三年，年末就要加上一个三十天的闰月来保证历法的准确性。

一年的开始于冬天的结束，所以新一年开始于春天。每个月都是一个特定神祗的圣月，每个月都在迈勒提斯举行一个大型庆祝活动后命名。瑟特萨和阿喀洛斯也使用这个历法，但是他们并不举行相同的仪式，比如：五月（迈勒提斯的萨拉米比月）在阿喀洛斯被称为伊洛安格月，在伊洛安斯竞技会之后，每年都会持续举行一个月。
