# 卡塔冲山脉

**卡塔冲山脉 The Katachthon Mountains**

民间传说俄拉尼阿德山脉被诸神祝福，卡塔冲为黑暗古老魔法统治。卡塔冲比俄拉尼阿德范围更小，更险峻，更多火山活动。雪崩、暴风定期发生，小型恶劣天气更无法预测。云层挡住了星象，向导因此迷失方向，探险者在小圈子里兜兜转转，而最可怕的是，巨龙统治着这片地方，这些喷火生物据说满是普罗峰斯打造的火焰。

居住在卡塔冲山脚的人不会承认任何有人宝藏或传说生物在那里被发现过，只会重复警告旅者：如果你必须穿过山脉，那就沿着路走。独眼峡和柏树巨门并不安全，但是总比自己尝试开拓路径强。
