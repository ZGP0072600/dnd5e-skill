# 瑟特萨

**瑟特萨 Setessa**

*“这座城市在我曾是孤儿，圄于锁链时拯救了我。现在轮到我来保护它了。”*

——卡莉雅斯，欧菲斯塔指挥官

瑟特萨是一座被卡拉美特拉钟爱的城市，它的建筑线条完美融入森林，难以分清内外界限。民众在茂密森林，梯田和瑟特萨驯化的动物间和谐生活，用隆重节日庆祝四季循环。

瑟特萨在塞洛斯诸城邦中也独一无二，城中鲜少有成年居民为男性。女性组成了人口的大部分，担任了几乎所有领袖角色和多数工作。男性少之又少，在城邦中基本是边缘角色。儿童可以自由的在城邦中穿游，他们是最重要的，事实伤瑟特萨人从全塞洛斯收养被遗弃的儿童。
