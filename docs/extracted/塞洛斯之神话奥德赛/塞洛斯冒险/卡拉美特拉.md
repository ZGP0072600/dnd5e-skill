# 卡拉美特拉

**卡拉美特拉 Karametra**

所有生命——植物与动物——都需要繁衍才能继续生存 ，这使卡拉美特拉的长存变为了维持一个宜居的凡世的重要环节。

**卡拉美特拉的神选 Karametra's Champions**

卡特美特拉的审讯建立社区，发展文明， 并以他们神祗的名义无论走到那里都会赐予馈赠。卡拉美特拉寻求供养所有人，终结饥荒和绝望，确保每个人都被家庭与社区支持。她并非苛求的神祗，但是她的神选勤奋的通过努力推进她的目标来获取她的宠爱。

卡拉美特拉的使命表格给出了集中该神神选可能将自己卷入的几场冒险。

**卡特美特拉战役 A Karametra Campaign**

初看来，一场战役中玩家角色成为卡拉美特拉的神选的战役似乎是反直觉的，因为卡拉美特拉的目标看起来是促进冒险生活的对立面。但是她的理念可能会吸引一些冒险家，比如流亡者，对他们来说在一个安全，繁荣的社区里安顿下来的主义有着非常诱人的前景。

卡拉米特拉将她的宠爱慷慨地给予那些保护已建立社区的人，但她最重视的神选是那些想要建立新定居点，为努力奋斗或处在生死关头的社区提供援助的人。她最爱的是那些体现了她价值观的追随者，把社区放在自己之前，抛下平静的生活，寻找他人加入这个队伍。当他们努力建立了一个精挑细选的与他们一起旅行追求同样目标的家庭时，她对他们的骄傲便越来越多。

在战役开始的时候，角色们可能会发现一个处于绝境的社区，并在他们恢复它的安全和繁荣的状态是获得了卡拉美特拉的祝福。也有可能他们发现自己的家被摧毁了，但却得到了卡拉美特拉的托梦，她将社区的难民交给他们照顾。

对卡拉美特拉的神选来说，最大的危险游走在培育馈赠和保护弱者与变成贪心与家长统治的对抗的斗争之间。卡拉米特拉看重谦逊，并经常给那些成长的过度膨胀的人一场尖刻的教训

卡拉美特拉的侍僧为她在瑟特萨的盛宴收集荒野的馈赠。

**卡拉美特拉的反派 Karametra's Villains**

卡特美特拉的追随者可以体现文明最好和最坏的一面。正如社会可以保护弱者一样它也能加强不公并给予它合法权利。太多的物质财富和安全感会让人们更不倾向于同情那些拥有更少的人，甚至会让他们相信财富是美德的标志。忽视这些真相会把卡拉美特拉的追随者引向邪恶的道路。

在最糟糕的情况下，卡特美特拉信仰可以被扭曲成强制服从。它会能根据自然和人民所能提供的东西来评估他们。它可以在照顾孤儿的同时对使他们成为孤儿的环境不采取任何手段去改善。它可以把某人对社区的保护变成仇外。它可以将富人视为道德典范而将穷人视为道德失败者败坏者。那些为他人提供服务的人可能会相信这给了他们控制被服务者的权利。当卡特美特拉的追随者不再小心平衡馈赠的时候，他们可能会与卡特美特拉的反派表格里呈现的敌人结盟。

**卡拉美特拉作为战役反派 Karametra as Campaign Villain** 自然内在的野性亦是卡拉美特拉本质的一部分。一场将卡拉美特拉作为反派刻画的战役可能探索她体内沸腾的凶残，野性面相被凡人将她刻画为一名温柔，母亲般的丰收之源的信仰之力所抑制。她不再要求她的信徒血祭是因为他们不再相信这种祈祷室必需的——而不是她决定停止。在万神殿中，她是一个多数诸神未能完全领会但却感到恐怖的真相的化身：尽管他们可以很大程度的影响凡人的生活，但是凡人信仰的集体力量可以改变神祗的真正本性。

作为战役反派，卡拉美特拉可能寻求回归她的血腥起源，派遣她的神选去说服社区他们必须再次进行血祭来取悦她。卡拉美特拉可能向城邦派出怪物来让他们回想起愤怒的她有多危险，甚至造成杀戮。战役可能结束于英雄们直面卡拉美特拉或寻找别的什么方法来安抚她。

**乔木瞭望塔 Arbor Watchtower**

这些地方是卡拉美特拉的圣地，从她在瑟特萨中心的大圣殿到荒野中的哨站之间遍布，结合了精细的建筑与自然的生长。神宠塔楼和空中花园将翠绿的自然的世界抬高朝向天界以表达对她的尊敬。在奈西安森林，瑟特萨人建造了具有军事防御和崇拜双重作用的瞭望塔

**瞭望塔地图 Watchtower Map**

所示的瞭望塔是一座大约六角形的石塔，雕刻助其与自然景观融为一体。古树生长在建筑内部和周围，使树枝和藤蔓从每一层开放的阳台上萌芽。

第一层包括的主入口，护柱，和献给四季的壁龛祭坛。第一层大部分被花园和果树占据。螺旋式楼梯通到所有更高层。

二楼有一个封闭区作为仓库，食品补给，和厨房。宽阔的阳台以树木、藤蔓和结满食物的植物组成的花园为特色。

三楼看起来和二楼很像，但有守卫的营房和食堂。阳台上的一些空间放置了射箭标靶，但大部分仍然是花园。

第四层与前两层类似，有祭司房间和一个图书馆。这里的阳台也有花园，可以一些更稀有或更奇特的植物为特色。

顶层露天，是卡拉米特拉的主要圣所。这里有一座神像和一座祭坛。几棵古树组成了一个树盖，而它们的树干被雕刻成伟大英雄的样子。
