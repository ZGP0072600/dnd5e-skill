# 雅睿欧司

**雅睿欧司 Athreos**

冥河向导横跨生死分界，维护宇宙均衡且不在乎凡人和凡人的感情。雅睿欧司的神选追求推进他的目标，但除了他们冷酷的保护者很少有人会对此给出更多的赞同。

**雅睿欧司的神选 Athreos's Champions**

雅睿欧司期待他的神选维护生者与死者之间的平衡，就像他所做的一样。除此之外，他的仆人 维持他们文化中独特的丧葬习俗并保存逝者的记忆。在凡人世界之外，冥河向导阻止任何扰乱生死循环的人，这些举动偶尔会给他与其他神明之间带来冲突——特别是厄睿柏斯和赫利欧德，偶尔也会是斐纳克和芳瑞卡。

雅睿欧司的使命表格建议该神祗的神选可能令自己卷入的冒险。

**雅睿欧司战役 An Athreos Campaign**

在一个战役的早期，雅睿欧司的神选可能面对危险或失控的亡灵，例如食尸鬼或幽魂，或是将神圣埋葬处当做居所的怪物。随着故事推进，转世者和幻灵逐渐成为主要敌人，根据情况神选可能被迫阻止那些寻找爱人或抱憾而去者的灵魂的人。在某一时间，雅睿欧司的神选可能去对付自大而狡诈的厄睿柏斯，赫利欧德和斐纳克，或是恶魔的狡计，亦或转世者陵寝。

冥河向导的凡人仆从会发现他们的感情经历着考验——特别是他们的道德感和同情心。任何凡人都有可能去试图复活一名死于战争中的统治者或让因死亡而分离的矮人摆脱苦苦。雅睿欧司不关心这种事，他希望自己的追随者去无情的保护生死边境。如果一名他的仆人衰弱了，雅睿欧司将很快收回他的赐福。

环世冥河的海岸被雅睿欧司和他的随从监视。

**雅睿欧司的反派 Athreos's Villains**

雅睿欧司的神选很容易放弃与其他凡人的联系。在无穷的死者与悲痛中工作足以消蚀一个人的同情心让任何灵魂变得冷酷。结果，雅睿欧司的追随者在他们将死者的需求至于生者之上时成了冷酷的敌人。他们的任务也使他们准备好与转世者和他们的盟友战斗。某些人寻求更高效的让必死之人死亡，例如阻碍医治或破坏一艘注定前往一场绝望战斗的战船。雅睿欧司并不赞同残忍的做法但也不特变关心，只要死者付钱渡河就好。

雅睿欧司的反派表格提出了该神祗的追随者中各种可能出现的敌人。

**雅睿欧司作为战役反派 Athreos as Campaign Villain**

雅睿欧司视他的责任为必须之事，但并不珍视。如果有某种更有效的办法维护生死循环，或创造某种停滞状态，他可能回去追寻它。此外，有意或无意，雅睿欧司可能会让一些靠不住处在去接手他的苦工。不管什么情况，任何冥河向导的角色拜年话都会在生者和塞洛斯的不朽者之间产生剧烈的风波。卷入雅睿欧司的诡计中的冒险者其存在的基础则可能会被波及。

修正生死循环可能需要其他神祗的援助。如果雅睿欧司真的抛弃了他的职责，它可能需要别人承担他的工作。候选者包括泰坦和诸神，但一名无私的凡人也有可能牺牲他们自己来成为新的雅睿欧司。

**冥界渡口 Underworld Crossing**

塔提克斯河岸两侧遍布作为凡界与冥界之间渡口的雅睿欧司神殿废墟。死者的灵魂在河堤上那些荒凉的地点等待冥河向导献身让他们通向对岸冥界。

荒废神殿建筑和他们的庇护神一样神秘。没有牧师居住或保养这些地方，但是它们被狱犬和冥卫保护着。当生者进入这些神殿，建筑物会在魔法陷阱的作用下活化并试图让活物远离亡者领域和让亡者回归冥土。

**冥河渡口冒险 Underworld Crossing Adventures**

冥河渡口冒险通常涉及角色们进入和逃离冥界，或是帮助或组织某人某物这么做的故事。交付完钱币的灵魂在渡过塔提克斯河到达冥界时通常会抛弃他们的其他财物——他们的财宝在冥界毫无用处。被丢掉的钱币在冥界河岸堆成的小山可能足以诱惑凡人的宝藏猎手，就算风险是被雅睿欧司的仆人永恒的追杀。

也有可能，是灵魂而非财宝诱使冒险者前去塔提克斯河的岸边。一个不能支付雅睿欧司的酬劳的死者灵魂可能在河边徘徊。或许冒险者前去搜寻这样的一个迷失灵魂帮助它前去冥界，也许如果一个灵魂分享了有价值的信息之后他们会愿意为渡河付钱。在别的情况下，冒险者也可能帮助死者到达他们的最终安息之地。

**冥河渡口地图 Underworld Crossing Map**

所示冥河渡口由一座荒凉，破碎的神殿位于塔提克斯河两岸的两半组成的。神殿的每一班都面向河水并以码头为终点。凡域一侧的河底被金币覆盖，这一班的神殿包括一个有着破碎地板和下沉祭坛的巨大的圣所，冥卫和狱犬守卫着那里。围绕着这个空旷的大厅的是通向码头的昏暗走廊，覆盖着当凡域生物存在时便会激活的可以爆炸并炸毁天花板的陷阱徽记。

神殿冥界的一半由变形的墓室大厅组成，这里没有存放尸体，尸骨或其他残骸——只有尘土和守卫。这些大厅通向飘着阴沉丧乐——虽然看不见有乐器演奏——的第二圣所。据说活着的生物听到这歌曲便会陷入绝望的深渊。一尊巨大的雅睿欧司雕像看管着这个房间，而守卫也在这里居住。
