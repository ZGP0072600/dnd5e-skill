# 倪勒娅

**倪勒娅 Nylea**

狂野而不可预料，倪勒娅能给生活在自己领域里的英雄同时带来祝福与困苦。

**倪勒娅的神选 Nylea's Champions**

倪勒娅是一名诸神中对凡域更为主动的神祗，但她也不能同时出现在任何地方。她的英雄便是她在塞洛斯的眼睛，耳朵和见识，在任何她不能亲自到来的时间和地点保护她的利益。

倪勒娅的使命表格给出了几种该神神选可能将他们自己卷入的冒险。

> | 3 | 在一片偏远的由怪物守卫的树林里种植黄金橡树。 |

**倪勒娅战役 A Nylea Campaign**

玩家角色作为倪勒娅神选的战役会意伟大的守卫自然的英雄壮举为特色。倪勒娅可能会指示角色们去守护奈西安森林的一角，让他们和偷猎者，炫耀的猎人，非自然的憎恶，和想要把森林为凡人利益而改造的普通人对抗。倪勒娅不在乎角色们对付的是单纯的农民还是超过角色处理能力的怪物。他们可能需要一些灵感，召唤倪勒娅的仆人——比如多头蛇——来击退更强大的威胁。

在战役开始时，英雄们可能帮助瑟特萨防御来自来自奈西安森林的威胁的攻击，那里是他们的主要关注点。虽然防御城邦只是他们的次要目标，但是仍然被誉为英雄，他们继续享受倪勒娅的宠爱同时对抗更强的威胁。

或许他们将会与城邦的领导人结盟，亦或瑟特萨发现它与倪勒娅的荒野理念互有抵牾。倪勒娅的决定简单却无法改变，而她的神选发现如果他们疏忽自己对森林的职责那么倪勒娅的宠爱将变成怒火。她可能简单的将英雄从奈西安森林放逐，或是在角色们寻求安抚她或者找到另一名神祗保护他们时变成一名反派。

倪勒娅的神圣猞猁保护着荣耀奈西安森林的人并猎杀不这么做的人。

**倪勒娅的反派 Nylea's Villains**

从倪勒娅的角度看来，她的愿望是良好的，但是她不愿意祝福那些压制荒野动物和植物的类人以至于有时她成了个威胁性的人物。但是居住在永久性定居点的居民往往不理解也不尊重倪勒娅的领域，哪怕出于最好的意愿也会与她发生冲突。

倪勒娅的追随者可能痴迷于清除文明的理念，这让他们变成了毁灭代行者。在较小层面上来说，他们可能把倪勒娅对狩猎的支持扭曲成了对错待他们的人实行仇杀的借口。当自然的平衡被破坏的时候，野生动物也有可能成为威胁，而当她自己的生物和追随者转向邪恶时倪勒娅可能亲自排出神选去干涉。

**倪勒娅作为战役反派 Nylea as Campaign Villain**

作为反派，倪勒娅最可能被对她领域的侵犯驱动，不论是被凡人还是其他神祗。她可能从奈西安森林中驱逐所有智慧生物开始报复，或是因为少数市民而对整个城邦带去惩罚。从此，她可能最后会在一个逐渐加剧的对抗全塞洛斯的凡人的冲突中卷入多位神灵并威胁到城邦概念的存在。作为其他神祗神选的玩家角色可能发现他们被不满他们保护神的猎手打上了死亡标记。

**森林祭坛 Forest Shrine**

森林祭坛是为了荣耀倪勒娅而建在塞洛斯的森林里。这些地方只有很少几个部分使用石头和金属，照料祭坛的德鲁伊和宁芙使用魔法指引植物生长，制造出美丽的自然祭坛，雕塑，和圣所。但是这些奇妙的造物并没有主导祭坛的布局，因为倪勒娅和她的信徒更喜欢让自然花卉不受约束的在她的圣地开放。

负责祭坛的德鲁伊有时会因为人物离开一段时间，但是他们从不会让这些地方没有守卫。野兽，植物，和宁芙总是在警戒着入侵者。

**森林祭坛冒险 Forest Shrine Adventures**

森林祭坛地处偏远，那里的居民很难让人发现。单纯的寻找不能过得到进入祭坛就可以成为一场冒险，因为德鲁伊和宁芙谨慎地从外来者手里守卫着这个地方并准备好了打到对森林来说非必要的人。

被接纳进入森林的角色会发现祭坛是个守卫严密，平静祥和的地方，是个对抗敌人和天气的绝佳避难所。神圣的动物和狩猎武器被保存在祭坛里。德鲁伊可能愿意给那些获得了他们信任的人分享知识和森林秘密。森林祭坛冒险表格提供了再这样一个地方可能发生的冒险。

**森林祭坛地图 Forest Shrine Map**

所示森林祭坛清楚的被沙枣树林包围。从北向南的溪流在中央形成一个池塘。葡萄藤从地面拱起，每一个都创造了一个私密谈话的隐蔽区域。几个灌木被塑成倪勒娅和森林掠食动物的形象，比如熊和狼，分布在祭坛各处。如果祭坛保护者呼唤他们，这些植物能活化成为首位。

一颗古老的，弯曲的柠檬树在猎手宿舍旁生长。弓，枪，和其他狩猎武器在树枝上的蜂箱旁悬挂。树南边是一座围着用于坐下的卵石的火坑。傍边，原木桌子和饲养动物的木架前的一颗茂密的鲜花盛开的灌木被塑造成祭坛的样子。

几个狩猎伪装隐藏在祭坛周围的森林里，从这些伪装的岗哨里弓箭手们监视着猎物和威胁。
