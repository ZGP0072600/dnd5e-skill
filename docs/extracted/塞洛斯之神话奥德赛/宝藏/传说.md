# 传说

**魔异盒 Pyxis of Pandemonium**

*奇物，传说*

一个生物可以触摸这个华丽的木盒1分钟，以此获得完成一次短休的好处。且该生物还将获得法术 *祝福术 bless* 
 
的效应，持续直至该生物完成下一次短休或长休为止。受益于此的生物在完成下一次长休前无法再次受益于这些好处。

若盒子被打开，投掷魔异盒表，某场灾祸将会发生。由盒子施展的任何法术豁免 DC 均为17。在盒子被打开的一分钟后它将消失，并在24小时后重新出现并被封印在同一位面的某处随机位置上。

**诅咒 Curse。** 任何因容器获得短休好处的生物都将听到从盒中传来的、阴森的心灵感应低语。该生物必须通过一次 DC 17的感知豁免检定。若豁免失败，该生物将会被盒子魅惑1小时。以此被魅惑的生物会尽其所能的去打开盒子。豁免成功后，该生物接下来的24小时内免疫盒子的耳语效应。
