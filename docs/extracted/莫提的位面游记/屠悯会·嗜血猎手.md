# 屠悯会·嗜血猎手

**屠悯会·嗜血猎手 Mercykiller Bloodhound**

嗜血猎手们身着汽配铠甲，手上绑着尖刺臂铠与伸出的金属钢爪。嗜血猎手会追猎那些曾在印记城犯下严重罪孽的人们，而他们会为了追逐那些罪犯，横跨数个位面。

屠悯会·嗜血猎手 Mercykiller 
Bloodhound

*小型或中型类人，任意阵营*

护甲等级：18（板甲）

生命值：104（16d8+48）

速度：40尺

力量17（+3）敏捷12（+1）体质16（+3）

智力12（+1）感知15（+2）魅力8（-1）

技能：察觉+8，求生+8

感官：被动察觉18

语言：通用语，外加任意一门语言

挑战等级：7（2,900XP）熟练加值+3

**侦测传送门 Portal Sense。** 嗜血猎手可以感知到自身周围30尺范围内的传送门存在（包括未激活的传送门），且他能本能性的知晓这些传送门各自的目的地为何处。

动作 Actions

**多重攻击 Multiattack。** 嗜血猎手发动三次铠爪击攻击。

**铠爪击 Clawed Gauntlet。** 近战武器攻击：命中+6，触及5尺，单一目标。命中：7（1d8+3）点挥砍伤害，外加10（3d6）点暗蚀伤害。

附赠动作 Bonus Actions

**寻猎印记 Marked for Pursuit（3/日）。** 嗜血猎手尝试在其周围30尺范围内的一个可见生物身上打上印记。该生物必须通过一次 DC 13的魅力豁免，豁免失败其将被诅咒，持续24小时。若生物生命值不为满，则此次豁免将具有劣势。以此法诅咒期间，只要嗜血猎手与该生物同处一个存在位面中，嗜血猎手可以感知到自己与目标之间的方向与距离；如果它们处于不同位面之中，嗜血猎手将会知晓目标所处的那个位面是哪个。
