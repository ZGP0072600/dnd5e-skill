# 诡脸飞首

诡脸飞首 Vargouille 
Reflection

飞首是一种会飞的邪魔，它的外形酷似一颗类人生物无身体的头颅，外加一对翅膀。虽然绝大多数飞首都会在各位面之中四处游荡，寻找并诅咒类人生物以创造更多的飞首，但有一种被称为诡脸飞首 Vargouille 
Reflection 的飞首特殊变种仅生活在 
底层印记城 Undersigil。当这些诡脸飞首发现类人生物时，它就会变成那个生物的惊悚无身之首模样，以此让那个生物感到惊悚与恐惧。

诡脸飞首 Vargouille 
Reflection

*微型邪魔，混乱邪恶*

护甲等级：12

生命值：22（5d4+10）

速度：5尺，飞行40尺

力量6（-2） 
 敏捷15（+2） 体质14（+2）

智力6（-2） 感知10（+0） 
 魅力2（-4）

伤害抗性：寒冷、火焰、闪电、心灵

伤害免疫：毒素

状态免疫：中毒

感官：黑暗视觉60尺，被动察觉10

语言：理解深渊语、地狱语、以及该生物成为飞首前知晓的语言，但都不会说

挑战等级：1（200 
XP） 熟练加值+2

**魔法抗性 Magic Resistance。** 飞首对抗法术或其他魔法效应所作的豁免具有优势。

动作 Actions

**啃咬 Bite。** 近战武器攻击：命中+4，触及5尺，单一目标。命中：5（1d6+2）点穿刺伤害，外加10（3d6）点心灵伤害。

**深渊诅咒 Abyssal Curse。** 飞首以自身周围5尺范围内的一个处于 失能 Incapacitated 状态的生物为目标。目标必须通过一次 DC 12的魅力豁免，豁免失败将被诅咒。诅咒期间，诅咒对象的魅力属性无法被增加，且只要诅咒对象不处于日光或是法术 
昼明术 daylight 
的效应范围内，他的头部的外貌就会越来越趋于邪魔化，导致每个小时它的魅力属性都将减少1点。当诅咒目标的魅力属性变为2时，目标将死亡，而它的头将脱离身体，成为一只新的诡脸飞首。在对象被诅咒彻底转化之前，对其施展一次法术 移除诅咒 remove curse、高等复原术 greater 
restoration、或是类似的法术便可以终止这一诅咒，并恢复其魅力属性。

**汝之诡脸 Horrific Reflection（充能5~6）。** 飞首将头拟态成飞首周围120尺范围内它能看见的一个类人生物的模样。那个生物必须通过一次 DC 12的感知豁免，豁免失败将受到10（3d6）点心灵伤害并陷入 恐慌 Frightened 状态，持续1小时。若飞首的专注中断（如同专注于一个法术），恐慌效应也将提前结束。如果目标通过了这次豁免，或是它身上的汝之诡脸效应结束，这一目标就将在接下来的1小时内，免疫于任何诡脸飞首的汝之诡脸动作项。
