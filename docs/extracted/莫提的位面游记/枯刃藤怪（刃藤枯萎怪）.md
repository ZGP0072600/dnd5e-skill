# 枯刃藤怪（刃藤枯萎怪）

枯刃藤怪 Razorvine 
Blight

>

穿梭于印记城与下层位面的旅行者们都得小心避开那些“刃藤”——刃藤，顾名思义，是一种有着长满尖刺的茎与锋利如刀刃的叶片的匍匐植物。尽管一般情况下，刃藤往往只是环境中让你困扰或不想接近的存在，但一旦它们接触并吸收不幸被划伤的路人流出的血液，它们就会转化为一种恐怖的存在......

枯刃藤怪 Razorvine 
Blight

。

枯刃藤怪常常埋伏在人迹罕至的小路上，俯卧在地上，静止不动，如同一株普通的刃藤。而那些粗心大意的旅行者路过这种不起眼的“路边杂草”时，枯刃藤怪就会展露出它们扭曲的、如人形的形态，向这些不幸的上门猎物发起攻击，用它们那锋利的刃形藤蔓来满足自己的嗜血欲望。

虽然普遍而言，枯刃藤怪是一种危险至极的生物，但在印记城，它们有时会模仿着城中居民的行为举止——当然，不论好的还是坏的都模仿。像这样的枯萎怪至少有一名......被称为“

补丁 Patch

”的，在城中散步自己复制体以组成间谍网络的枯萎怪，而它的目的是什么依旧是个谜。

*中型植物，普遍中立邪恶*

护甲等级：12

生命值：27（5d8+5）

速度：30尺，攀爬30尺

力量10（+0） 
敏捷15（+2） 体质13（+1）

智力5（-3） 感知10（+0） 
魅力3（-4）

技能：隐匿+4

状态免疫：目盲，耳聋

感官：盲视感官60尺（此范围外不可视），被动察觉10

语言：理解通用语但不会说

挑战等级：1（200XP） 
熟练加值+2

**拟形 False Appearance。** 只要枯萎怪保持静止不动，它进行的先攻检定将具有优势。此外，如果一个生物未曾观察到过枯萎怪移动或行动，那个生物就必须通过一次 DC 18的智力（调查）检定才能辨别出这只枯萎怪是会动的。

**蛛行 Spider Climb。** 枯萎怪可以攀爬在难以攀爬的表面上，包括倒吊在天花板上，且不需要进行属性检定。

动作 Actions

**多重攻击 Multiattack。** 枯萎怪发动两次爪击。

**爪击 Claw。** 近战武器攻击：命中+4（若生物已损失任意数量生命值，则具有优势），触及10尺，单一目标。命中：5（1d6+2）点挥砍伤害。

**饮命之藤 Life-Draining Vines（充能6）。** 如蛇形的藤蔓从枯萎怪的体中突出。每个处于枯萎怪周围10尺范围内的生物都必须通过一次 DC 12的敏捷豁免，豁免失败时将受到9（2d8）点挥砍伤害。豁免成功则伤害减半。如果本次豁免有至少一个生物豁免失败于此豁免，则枯萎怪还将恢复9点生命值。
