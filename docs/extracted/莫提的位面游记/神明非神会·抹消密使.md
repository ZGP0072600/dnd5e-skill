# 神明非神会·抹消密使

神明非神会·抹消密使 Athar 
Null

神明非神会的所有人都否认神明的正确性，他们认定所谓的神只是一群强大的施法者。而为了战胜这些“伪神”，他们训练出了被冠以“空无 NULL”之名的，可以抵消魔法使用者力量的“抹消密使”。抹消密使们藏于阴影之中，承担着神明非神会的刺客与间谍之职。

神明非神会·抹消密使 Athar 
Null

*小型或中型类人，任意阵营*

护甲等级：14（皮甲）

生命值：84（13d8+26）

速度：30尺

力量12（+1）敏捷16（+3）体质14（+2）

智力15（+2）感知14（+2）魅力10（+0）

豁免：敏捷+6，感知+5

技能：调查+8，察觉+5，隐匿+6

感官：被动察觉15

语言：通用语外加任意两门语言

挑战等级：5（1,800XP）熟练加值+3

**虚无闪避 Avoidance。** 如果抹消密使受到一次豁免成功则只受半数伤害的效应影响时，其豁免成功则不受任何伤害，而豁免失败时则只受一半伤害。

动作 Actions

**多重攻击 Multiattack。** 抹消密使发动两次力场匕首攻击。

**力场匕首 Force Dagger。** 近战或远程武器攻击：命中+6，触及5尺，或射程20/60尺，单一目标。命中：5（1d4+3）点穿刺伤害，外加13（3d8）点力场伤害。不论是否命中：攻击后，这把匕首都会魔法性的返回抹消密使的手势。

附赠动作 Bonus 
Actions

**周旋者的灵机一动 Defier's Whim。** 抹消密使执行 **疾走 Dash**、** 回避 Disengage**、或 ** 使用物件 Use a Object** 动作。

反应 Reactions

** 抹消法术 Nullify Spell（3/日）。**抹消密使念叨一段消除性的魔法咒语，以此打断它可以看见的正在施展法术的生物。如果那一法术的环阶为3环或以下，该法术失败且不会产生任何效果。如果那个法术为4环或以上环阶的法术，则抹消密使必须通过一次智力检定（DC 10+该法术的环阶）。检定成功时，该法术失败且不会产生任何效果。
