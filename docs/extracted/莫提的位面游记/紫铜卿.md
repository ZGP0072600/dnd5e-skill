# 紫铜卿

紫铜卿 
Cuprilach Rilmani

紫铜卿在整个多元宇宙之中作为密探与刺客渗透那些带有力量的位置。它们在黄金卿的命令下无情、坚决地发动外科手术式的打击。

紫铜卿们有着细线般的铜制框架，其躯干漂浮于腰部之上，两者之间由一个浮空的光亮球体所分隔。紫铜卿们灵敏机巧，配备着为数众多的潜伏魔法，它们会做任何必要之事以完成其任务。

动作 Actions

**多重攻击 Multiattack**。紫铜卿发动三次锋芒剑刃或飞矢攻击。

** 锋芒剑刃 Burnished Blade.**。近战武器攻击：命中+9，触及5尺，单一目标。命中：8（1d6+5）点穿刺伤害外加13（2d12）点心灵伤害。

** 飞矢 Bolt**。远程武器攻击：命中+9，射程20/60尺，单一目标。命中：7（1d4+5）点穿刺伤害外加13（2d12）点心灵伤害。

随意：侦测魔法，易容术

每项1/日：云雾术、变身术、沉默术

附赠动作 Bonus 
Actions

** 刺客灵敏 Assassin's Agility**。紫铜卿发动疾走或撤离动作，或其发动一次锋芒剑刃。
