# 妖魔(格莱斯魔)

**妖魔 Demodands**

妖魔，又称为格莱斯魔 gehreleths，它们是来自塔尔塔洛斯深狱之卡瑟利的邪魔。亘古以先，它们因早已被遗忘的罪孽而被投入到监狱位面之中，这些痛苦、卑劣的生物已经将自己视作这座位面的狱卒。妖魔们满怀恶意地守护着那极其稀少的，能够通向卡瑟利之外的传送门，并无情地折磨着任何被困于此位面的生物。

成功离开卡瑟利的妖魔们都知道它们注定会被拖回去；一个在卡瑟利之外死亡的妖魔会在2d20天内以极其痛苦的进程缓慢重组。即便那些在其他位面幸存下来的妖魔也会发现自己最终被一些位面束缚所捆回去。
