# 日光萤 Sunflies

奇特，扑闪着翅膀嗡嗡作响的日光萤是外层位面的居民。这些生物的旅行范围广泛，并成为其居住区域的健康风向标：但日光萤出现痛苦和挣扎的迹象时，它们所栖息的区域亦是如此。日光萤能通过毒刺想其他生物注入毒素，其效果与它们所处的位面相关联，因为位面的魔法会改变他们毒素的成分。许多位面居民都对日光萤抱有强烈的感情，将其视为恶心的害虫或可爱的宠物。在印记城，日光萤往往受到其主人的溺爱和保护。
