# 鹏羽天

鹏羽天 Avoral 
Guardinal

鹏羽天们形如有着鹰躯的双足生物，它们的臂膀生羽，其金色双目敏锐无比。鹏羽天们热爱漫游，这些翔于高空的天族比他们的同族更经常地离开极乐境，在外域中随着风的轨迹自在翱翔。鹏羽天们都是独立的斥候、散兵，它们也皆为灵巧的空战好手，能够迅捷地从空中向敌人俯冲而下，并用其锐利的鹰爪将敌人撕裂。

**鹏羽天 Avoral Guardinal**

*中型天族，普遍中立善良*护甲等级：16（天生护甲）

生命值：172（23d8+69）

速度：30尺，飞行50尺

力量16（+3）敏捷19（+4）体质17（+3）

智力16（+3）感知16（+3）魅力18（+4）

豁免：敏捷+8，魅力+8

技能：察觉+11，宗教+7

伤害抗性：光耀

状态免疫：恐慌

感官：黑暗视觉120尺，被动察觉21

语言：天界语，通用语

挑战等级：9（5,000 XP） 熟练加值：+4

**飞掠 Flyby。** 鹏羽天飞行离开敌人的触及范围时不会引发借机攻击。

**俯冲攻击 **。如果鹏羽天正在飞行中，且朝一个中型或更小的生物直线俯冲了至少30尺距离，并在目标生物5尺内结束移动，那个生物便必须通过 DC 15力量豁免，否则就会受到14（4d6）点穿刺伤害并陷入倒地状态。

** 动作 Actions**

** 多重攻击 **。鹏羽天发动两次鹰爪攻击。它能将其中一次攻击替换为施法。

** 鹰爪 Talon**。近战武器攻击：命中+8，触及5尺，单一目标，命中：11（2d6+4）点挥砍伤害外加13（2d12）点光耀伤害。
