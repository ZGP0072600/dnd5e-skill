# 莫提的位面游记

前言：多元宇宙级生物展览 
Multiversa

“莫提。一名自称‘著名’的位面旅行者，常常假扮成一个智颅来” 
 
 Partial mimir response on the subject "Morte"

“喔！嘿嘿嘿！停一停停一停！哎，我可不知道谁塞给你了个有瑕疵的智颅，头儿，但没事，接下来我来告诉你什么是什么。“ 
 
 莫提

这些智颅 Mimir 
，常常被那些位面旅行者视为珍宝。它们利用智颅了解事实，躲开死在那些“手”啊、“爪”啊、“牙”啊、“触手”啊、还有“长鼻子”啊、还有“产卵管”啊、还有数不胜数的其他多元宇宙捕食者（或者别的什么暴躁居民）的各种各样稀奇古怪肢体之下的命运。莫提，呃，其实并不是一个智颅。那颗漂浮在空中的头颅，总是自说自话直言不讳，还声称自己有着各样的专业知识。莫提是印记城众多特色居民之一，也是一名停下脚步的位面探索者。只要给足了金币或是欠他个人情，莫提很快就会供出他对各种位面的居民的各种建议、各种见解、以及各种评论。这本书包含了自莫提的故事与随言中提取出来的有用数据。然而，这些位面是辽阔且时刻变化的，且莫提亲身经历过的那些也不一定与实际情况相符。请读者注意甄别。

| 使用资料卡 Using a Stat Block 这本书是一份与《怪物图鉴》类似的怪物合计，且与那本书使用类似的格式。如果你并不熟悉怪物资料卡的格式，请在查看本书前提前阅读《怪物图鉴》前言的相关内容。那本书中解释了资料卡上的各种属于并提供了怪物部分特性的专有规则而那些信息此处不会再重复。在下方提供了按挑战等级排序的生物资料卡表格，这些生物中的大多数可以在怪物图鉴 A-Z 章节找到，但那些标有星号(*)的生物则录入在"派系特工"的部分不常见的攻击与魔法 Unusual Attacks and Magic 有些生物具有能造成不常见类型的伤害，或有能以一种非典型的方式施法的能力。这些例外都是这张资料卡上的特殊特性，用于呈现这些生物是如何使用他们的武器或是施展它们的法术的；这些例外对其他使用这些武器或是法术的生物没有任何的效果。遇上魔法物品先决条件 Meeting Magic Item Prerequisites 如果资料卡在生物的名字下方标注了某一职业的名字，那么那个生物在处理魔法物品先决条件时，就可以被视为相应职业的一员。 |  |
