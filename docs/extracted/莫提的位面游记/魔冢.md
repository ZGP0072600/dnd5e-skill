# 魔冢

**魔冢 Modrons**

魔冢是一种部分由机械组成的生物，它们遵从着严格的阶层秩序，并被铸造于机械境位面。每一个魔冢都尽责地遵守来自阶层直接高于它的魔冢的命令，并作为阶层直接低于它的魔冢的上级。如此，它们将命令从最高秩序典范处传达最低阶的一元冢之中。大部分魔冢都是低阶的单元魔冢......一元冢，*二元冢*，*三元冢*，*四元冢*，*五元冢*，而 **高阶魔冢 hierarchy modron** 则作为领导者，维持着机械境与该位面之外的秩序。若需要更多关于魔冢的信息，请见“*怪物图鉴*”。
