# 亚空天族

亚空天族 
Archons

亚空天族是居住在 天界山 Mount Celestia 
上，七层天堂的公民。它们自秩序与仁爱的力量中诞生，从邪魔入侵中守卫家园，从邪恶力量下保护弱者。亚空天族还是专业的交涉者，能够说能够说出多元宇宙中的所有语言。当被卷进战斗时，它们倾向于制服敌人。但当面对邪魔时，亚空天族就会化身为愤怒的战士，展现出天界山的正义之怒，将邪恶之徒彻底击溃。

 
每个亚空天族的形态，都与其在天界等级体系中的地位相对应。在战斗中遭遇敌人时，亚空天族会散发出上层位面的全部怒火，在为盟友提供支援的同时，也能给予敌人以震慑
