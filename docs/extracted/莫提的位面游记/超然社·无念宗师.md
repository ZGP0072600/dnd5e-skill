# 超然社·无念宗师

**超然社·无念宗师 Transcendent Order Conduit**

超然社的成员们练就了无需思考便可采取行动的技艺，以及消除动作之中思想的功法。无念宗师参照着多元宇宙的节奏行动，无念宗师敏锐的反应能力如同预知能力一般，能在敌人行动前仅凭直觉击倒它们。

超然社·无念宗师 Transcendent Order 
Conduit

*小型或中型类人，任意阵营*

护甲等级：18（无甲防御）

生命值：97（15d8+30）

速度：50尺

力量10（+0）敏捷19（+4）体质14（+2）

智力10（+0）感知18（+4）魅力12（+1）

豁免：感知+7，魅力+4

技能：特技+7，表演+4，察觉+7

感官：被动察觉17

语言：通用语，外加任意一门语言

挑战等级：8（3,900XP）熟练加值+3

**直觉反应 Instinctive Reflexes。** 无念宗师进行的先攻检定具有优势，此外，它进行的任何攻击检定都无法具有劣势。

**无甲防御 Unarmored Defense。** 只要未穿着护甲也未持用盾牌，无念宗师就可以将它的感知调整值加入到 AC 中。

动作 Actions

**多重攻击 Multiattack。** 无念宗师发动三次徒手打击攻击。

**徒手打击 Unarmed Strike。** 近战武器攻击：命中+7，触及5尺，单一目标。命中：11（2d6+4）点钝击伤害。如果目标是生物，无念宗师还可以附加以下额外效应之一（每项每回合只能使用至多一次）：

 · **失能 Incapacitate。** 目标必须通过一次 DC 15的体质豁免，豁免失败将陷入 **失能 Incapacitated** 状态，持续直至无念宗师的下一回合结束。

 · ** 推击 Push。**目标向着远离无念宗师的方向被水平推离至多10尺。

反应 Reactions

无念宗师每轮可以执行至多3个反应，不过每回合仅能执行一次。

** 招架攻击 Deflect Attack。**若无念宗师即将被一次攻击检定命中，无念宗师招架该次攻击以减轻伤害。无念宗师因本次攻击而受到的伤害将减少1d10点。

** 我可不在那儿 Don't Be There。**当无念宗师受迫进行一次豁免时，它可以移动至多等同于其速度一半的距离（不会引发借机攻击）。如果新的位置移出了该效应的范围，或是处于一处该效应无法指定的位置，无念宗师将彻底避免被该效应影响。
