# 灰烬先驱·驻世余烬

**灰烬先驱·驻世余烬 Heralds of Dust Remnant**

驻世余烬是归尘者中那些受仪式洗礼飞升为亡灵......但失败的人们。联络员与间谍，这些特工以某种介于生命与死亡分界线上的方式存在着。

灰烬先驱·驻世余烬 Heralds of Dust 
Remnant

*小型或中型类人，任意阵营*

护甲等级：12（有法师护甲时15）

生命值：45（7d8+14）

速度：30尺

力量8（-1）敏捷14（+2）体质15（+2）

智力17（+3）感知14（+2）魅力11（+0）

技能：奥秘+5，察觉+4，隐匿+6

伤害抗性：暗蚀

感官：被动察觉14

语言：通用语，外加任意三门语言

挑战等级：4（1,100XP）熟练加值+2

动作 Actions

**多重攻击 Multiattack。** 驻世余烬发动两次黯流蚀涌攻击。

**黯流蚀涌 Necrotic Surge。** 近战或远程法术攻击：命中+5，触及5尺，或射程120尺，单一目标。命中：14（2d10+3）点暗蚀伤害。

**施法 Spellcasting。** 驻世余烬施展以下法术之一，无需材料成分且使用智力作为其施法属性（法术豁免 DC 13）：

 随意：法师护甲 Mage Armor，法师之手 Mage 
Hand，魔法伎俩 Prestidigitation 每项1/日：灾祸术 Bane，任意门 Dimension 
Door，蛛网术 Web

附赠动作 Bonus 
Actions

**虚相 Phase（2/日）。** 驻世余烬的身体变得如同虚体魂灵一般，它获得钝击、穿刺、与挥砍伤害的抗性。为受益于此能力，驻世余烬需要为其保持专注（如同专注于一个法术一般）。
