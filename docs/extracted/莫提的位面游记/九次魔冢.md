# 九次魔冢

九次魔冢 Nonaton 
Modron（注：该生物为高阶魔冢，切勿误解成基础魔冢“九元冢”——虽然并不存在这样的基础魔冢）

每当有魔冢反抗奇点的意志时，九次冢就会负责将这些浪人单位重置回到序列之中。这些审讯者还负责审问那些被逮捕的机械境过路客。九次冢有着蠕虫般的身躯，九条机械臂固定在其身躯上。

公理之念 Axiomatic 
Mind。九次冢不会被迫去做违反其天性或其原有指令的行为。战斗预备 Combat 
Ready。九次冢在先攻掷骰上具有优势。解体 Disintegration。九次冢死亡时，它的身体会瓦解为尘埃，留下它的武器和其他携带的东西。动作 Actions 多重攻击 Multiattack。九次冢进行三次手臂攻击，而后使用诚实之柱或施法。手臂 Arm。近战武器攻击，命中+8，触及10尺，单一目标。命中：13（2d8+4）点穿刺伤害。若目标为中型和更小的生物，则其陷入受擒状态（逃脱 DC 14）。在对这个生物的擒抱结束前，十次冢不能使用这条触手攻击另一个目标。九次冢有九条手臂，每一条都能擒抱一个生物。诚实之柱 Pillar of 
Truth。九次冢选择自己60尺内能看见的一处地面。一道60尺高、20尺半径的魔法力场柱会从那一点升起。每一个在目标区域内的生物必须通过 DC 15敏捷豁免。若豁免失败，生物就会受到21（6d6）点力场伤害，且生物必须恢复到其原始形态（如果受影响生物在别的形态下），且其直到下个自己的回合结束前无法再变形成别的形态。施法。九次冢施展下列法术之一，且无需任何材料成分，智力是它的施法关键属性（法术豁免 DC 15）：随意：侦测魔法，解除魔法，修复术（一动作）每项1/日：位面转移（仅自身）、防护善恶

你想学魔冢语？好好，我会这门语言。第一课“哔哔”这句话的意思是：你好，老伙计，我没有敌意。真的，我发誓！
