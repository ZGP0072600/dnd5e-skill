# 十次魔冢

十次魔冢 Decaton Modron（注：该生物为高阶魔冢，切勿误解成基础魔冢“十元冢”——虽然并不存在这样的基础魔冢）

 
作为高阶魔冢中的最低阶成员，十次冢们管理着单元魔冢的身体状况。十次冢由两条坚实的腿支撑着它们的球状身体，十条机械触手从它们圆形框架的每个方向中延伸而出。它们的头部拥有十只随时准备监视周围单元魔冢们之行动的眼睛。在紧急状况下，一只十次魔冢能够从其十只眼睛中释放出一道闪电能量，并将这道能量加速同步到致死级别。
十次魔冢 
Decaton Modron 大型构装，普遍守序中立护甲等级：16（天生护甲）生命值：144（17d10+51）速度：30尺，飞行30尺（悬浮）力量18（+4）敏捷12（+1）体质16（+3）智力15（+2）感知15（+2）魅力11（+0）豁免：智力+5，感知+5技能：察觉+8状态免疫：魅惑，恐慌感官：真实视觉120尺，被动察觉18语言：魔冢语，心灵感应120尺挑战等级 ：8（3,900 
XP）熟练加值：+3
公理之念 Axiomatic 
Mind。十次冢不会被迫去做违反其天性或其原有指令的行为。战斗预备 Combat 
Ready。十次冢在先攻掷骰上具有优势。解体 Disintegration。十次冢死亡时，它的身体会瓦解为尘埃，留下它的武器和其他携带的东西。动作 Actions 多重攻击 Multiattack。十次冢进行三次触手攻击。触手 Tentacle。近战武器攻击，命中+7，触及10尺，单一目标。命中：15（2d10+4）点穿刺伤害。若目标为中型和更小的生物，则其陷入受擒状态（逃脱 DC 14）。在对这个生物的擒抱结束前，十次冢不能使用这条触手攻击另一个目标。十次冢有十条触手，每一条都能擒抱一个生物。闪电光束 Lightning 
Ray（充能6）。十次冢从其眼部释放出一道闪电束网，十次冢30尺内的所有生物必须通过 DC 13敏捷豁免，否则受到38（7d10）点闪电伤害，豁免成功则伤害减半。施法。十次冢施展下列法术之一，且无需任何材料成分，智力是它的施法关键属性（法术豁免 DC 13）：随意：侦测魔法，解除魔法，修复术（一动作）每项1/日：位面转移（仅自身）、防护善恶

*我以前认识一些魔冢。它们大部分都是易上当无公害类型的。但要当心那些高阶魔冢。它们不仅开不起玩笑，而且还会指责你破坏了多元宇宙的守序事业，然后会试图把你抓起来，研究你的骨头在变成粉之后有多重*。

*莫提*
