# 无望党·虚无医师

**无望党·虚无医师 Bleak Cabal Void Soother**

虚无医师会去照料那些遭受诅咒或是受各位面负面影响的生物们。这些战医们会行走于危险位面与冲突频发的地带，并为那里的人们提供治疗与救助。

无望党·虚无医师 Bleak Cabal Void 
Soother

*小型或中型类人，任意阵营*

护甲等级：16（胸甲）

生命值：55（10d8+10）

速度：30尺

力量16（+3）敏捷14（+2）体质12（+1）

智力12（+1）感知15（+2）魅力10（+0）

豁免：体质+3，感知+4

技能：医药+4

感官：被动察觉12

语言：通用语，外加任意一门语言

 
挑战等级：3（700XP）熟练加值+2

动作 Actions

**多重攻击 Multiattack。** 虚无医师发动两次硬头锤或虚空之矢攻击

**硬头锤 Mace。** 近战武器攻击：命中+5，触及5尺，单一目标。命中：6（1d6+3）钝击伤害外加3（1d6）力场伤害。

**虚空之矢 Void Bolt。** 远程法术攻击：命中+4，射程90尺，单一目标。命中：9（2d8）力场伤害

**施法 Spellcasting。** 虚无医师施展以下一道法术，使用感知作为施法属性（法术豁免 DC 12）：

 随意：神导术 Guidance，光亮术 Light 每项1/日：安定心神 Calm 
Emotions，次等复原术 Lesser Restoration，移除诅咒 Remove Curse，防护能量 Protection from 
Energy

附赠动作 Bonus 
Actions

抚慰真言 Soothing 
Word（3/日）。虚无医师念出一道悲悯的真言，治愈一个位于其60尺内它可见的生物。目标恢复4（1d4+2）生命值。
