# 超然社·直感拳师

**超然社·直感拳师 Transcendent Order Instinct**

武艺之精锐，直感拳师习得了无需思考便能战斗的技艺。它们能够随意接下敌人接二连三的攻击。

超然社·直感拳师 Transcendent 
Order 
Instinct

*小型或中型类人，任意阵营*

护甲等级：16（无甲防御）

生命值：49（9d8+9）

速度：40尺

力量10（+0）敏捷16（+3）体质12（+1）

智力10（+0）感知16（+3）魅力12（+1）

豁免：感知+5，魅力+3

技能：特技+5，表演+3，察觉+5

感官：被动察觉15

语言：通用语，外加任意一门语言

挑战等级：3（700XP）熟练加值+2

**无甲防御 Unarmored Defense。** 只要未穿着护甲也未持用盾牌，直感拳师就可以将它的感知调整值加入到 AC 中。

动作 Actions

**多重攻击 Multiattack。** 直感拳师发动三次徒手打击攻击。

**徒手打击 Unarmed Strike。** 近战武器攻击：命中+5，触及5尺，单一目标。命中：7（1d8+3）点钝击伤害。

反应 Reactions

直感拳师每轮可以执行至多3个反应，不过每回合仅能执行一次。

**招架攻击 Deflect Attack。** 若直感拳师即将被一次攻击检定命中，直感拳师招架该次攻击以减轻伤害。直感拳师因本次攻击而受到的伤害将减少1d6点。
