# 星界冒险者指南 Astral Adventurer's Guide

# 简介：浩瀚的冒险之海 Introduction: Vast Oceans of Adventure

在 D&D 
游戏中，冒险可以在多元宇宙的任何角落展开不仅在物质位面的地牢和荒野中，而且在其他存在位面，包括天体导航员所说的荒宇。当你站在一个物质位面世界抬头仰望夜空时，你所看到的是荒宇，以及，比这更遥远的星界海。探索这些领域及其周围的世界是魔法船战役的内容。

## 术语 Terminology

星光位面，顾名思义，就是星辰位面。更准确地说，它是星辰和通往天堂的门户所在的地方一个无限广阔的天上虚空，环绕着物质位面的所有世界。

每个 
D&D 
世界无论是圆形的、扁平的还是其他形状都存在于一个被称为荒宇的无空气的虚空之中。一个世界可能是孤立的，也可能有邻居：一个或多个恒星、世界、卫星、小行星、彗星或其他天体。这个由各个天体和行星体所组成的星区被称为荒宇系。

在荒宇中，物质位面和星光位面重叠。 
荒宇中的生物和物体正常老化，并且同时存在于这两个位面。如果你要离开你的家乡世界并继续向外直到你接近你所处的荒宇系的边缘，你会开始看到微弱的银色烟雾。通过穿行这片雾霭，你穿越荒宇进入了又被称为“银色虚空”的星界海。你越深入星界海，雾霾就会变得越是浓厚和明亮，但总是可以看见透过它闪耀的星星。荒宇和星界海共同构成了星光位面。

星界海如同荒宇一般也是一片虚空；但是，它不是真空的。在这里，你可以正常呼吸，永远存在，永不衰老，也永远不需要饮食。尽管许多星界航行者明智地乘坐装备精良的舰船旅行因为此处是许多可怕生物的家园但你也可以仅靠你的思想力量推动自己穿越星界海。在这里，旅行者可能会发现已死之神那石化的躯体和作为通往其他存在位面门户的漩涡彩池。 
（有关彩池的更多信息，请参阅《城主指南》。）

就像桨和强风可以支持地表世界的海上航行一样，被称为法驱魔舵的魔法物品被用来推动和引导舰船通过荒宇和星海。法驱魔舵通常以华丽座椅的形式呈现，而舰船上的舵手就坐在上面。只有施法者能与法驱魔舵同调。而一艘魔法船的舵手则被称为法驱者。

## 使用本书 Using This Book

《星界冒险者指南》的大部分内容都是为与玩家共享而设计的。将本书的章节视为在围绕物质位面每个世界的未知虚空中创建角色和运行冒险的入门书吧：

第 
1 章提供了新的玩家可选项，包括两个背景和六个种族选项。

第 2 章给出了玩家和 DM 
的法驱规则以及各种魔法船的统计数据和插图。本章还包括一些在星际航行者中流行的法术和魔法物品。

第 3 
章描述了布拉之岩，一座建在小行星上的城市，你可以将其用作停靠港或战役中心。你可以将它放置在荒宇或星界海的任何地方。

| 1 | 冒险从布拉之岩开始，一个自动侏儒（参见《布布的星界怪兽展》）需要角色的帮助才能找到自己在城市中失踪的创造者。 |
| 2 | 其中一个角色继承了一艘配备了法驱魔舵的舰船和一本老船长的日志，该日志暗示了要探索的地方、要寻找的宝藏和要避免的危险。 |
| 3 | 在角色们家乡世界的荒宇中，他们在一艘坠毁的蝎舰（见第 2 章）的残骸中发现了一个法驱魔舵，并与想要掠夺残骸的地精发生冲突。 |
| 4 | 一位富有的赞助人赠送给了角色们一艘舰船和一个法驱魔舵，并期望他们完成一项危险的任务作为回报。 |
| 5 | 一艘魔法船降落在角色们居住的村庄的郊区。船长正在寻找新兵来参加荒宇中的一场战争。 |
| 6 | 村民们请求角色们调查山上的一个奇怪物体，结果发现是一艘坠毁的夜蛛（见第 2 章），上面还有幸存者：一个纽吉怪海盗和一个纽吉怪幼体集群（两者都见《布布的星界怪兽展》）。 |
| 7 | 一位哈多兹鼯猴人探险家（见《布布的星界怪兽展》）需要角色们的帮助才能杀死一头不断攻击她的飞鱼舰（见第 2 章）的白色鲸驮骊（见《布布的星界怪兽展》）。哈多兹鼯猴人无法判断这头鲸驮骊是怀有敌意还是爱意。也许它把这艘舰船误认为是一个对手或配偶！ |
| 8 | 角色们是一艘在魔舵发生故障后坠毁在一个陌生世界的魔法船的乘客。角色们必须抵挡外星怪物，直到他们获救。 |
| 9 | 角色们受雇于一家贸易公司，以保护运往某个遥远世界的贵重货物。角色们成为运送珍贵货物的鱿鱼舰（见第 2 章）上的船员。 |
| 10 | 冒险从布拉之岩开始，一个或多个角色与一个流浆体头目结了梁子（见《布布的星界怪兽展》）。他们要么逃离布拉之岩，要么与它的手下抗衡。 |
