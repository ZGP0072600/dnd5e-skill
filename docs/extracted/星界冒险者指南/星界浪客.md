# 星界浪客

星界冒险者指南

角色选项

背景

## 星界浪客 Astral Drifter

早在你有记忆之前，你就在星界海之中遨游。在星界海里，你切身体会到了银色虚空的奥妙：你停止老化并不再感受到饥渴。在流浪欲的驱使下，你像风中的尘埃一般，在星界海中四处飘荡。自从你来星界海至今到底度过了多少年月，已经无法计数。

在你的旅途中，你于死去神灵的巨大石化残骸上扎营，惊险地逃脱了横扫星界海的心灵之风，同时还得躲避该位面最危险的居民们。 

**技能熟练 Skill Proficiencies：** 
 洞悉，宗教 

** 语言 Languages：** 两项你选择的语言（推荐天界语或吉斯语） 

** 装备 Equipment：**一套旅行服装，一本日记，一只墨水笔，一瓶墨水，还有一个装着 10gp 的小包

**长寿 Longevity**

你的实际年龄要比你的外表年长 20d6 岁，因为你在停止老化的星界海中逗留了这么长的时

 间。

**特性：神性之遇 Feature: Divine Contact**

你获得来自玩家手册中的魔法学徒专长，且在选择该专长中的职业时，你必须选择“牧师”一项。 
在星界海，你与一位流浪的神道路相会。这次相遇简短且友善，但仍给你留下了长久的影响。这位神觉得适合同你分享一个秘密或一点晦涩的宇宙知识。与你的 DM 
合作，确定这些知识的细节及其对战役的影响。 

投掷神性之遇表来决定你遇见了哪一位神，或和你的 DM 合作选取一个更恰当的选项。
