## 暴君舰 Tyrant Ship

眼魔使用他们的解离射线从石头中雕刻出暴君舰，并使用这些舰船在星界漫游，寻找要征服的世界和要摧毁的敌对眼魔。没有两艘暴君舰长得一模一样，但她们都有一个或多个让人想起其创造者的特征，比如球根般的形状和眼柄。 

暴君舰指挥甲板上的球形房间视同一个只有眼魔才能同调的法驱魔舵。 
任何占据这个房间的眼魔都可以与其进行同调。 将暴君舰的生命值降至 0 不仅会摧毁她本身，还会摧毁舰上的法驱魔舵和眼柄光炮。 

暴君舰不能漂浮在水面上，但可以安全地降落在地面上。
