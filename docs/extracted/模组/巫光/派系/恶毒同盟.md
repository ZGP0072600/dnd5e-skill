# 恶毒同盟

恶毒同盟 League of Malevolence

恶毒同盟是一群出于一个目的——增长力量而联合起来的反派组织。它的创始人，术士凯雷克希望他的盟友们为共同的利益而合作，但他也同样鼓励他们施展自己的邪恶计划。

这里描述了包括凯雷克在内同盟的五位成员，尽管其中一位成员（丝绮拉）最近已经与其他人分道扬镳。

凯雷克 Kelek

凯雷克是一个贪婪、自恋的反社会恶棍，他沉迷于制造混乱，但心底却是一个胆小鬼。他实际非常聪明，这让他更添了一分危险。他最希望得到的威力法杖 staff of power 握在他最为憎恨的敌人林格伦的手中。

阵营。混乱邪恶

人格特质。“所有听到我大名的人都该吓得瑟瑟发抖！”

理想。“仁慈不过是软弱的另一种写法。”

牵绊。“我必须拿到林格伦的法杖！有了它，恶毒同盟将会不可阻挡！”

缺陷。“我不相信任何人，也不值得任何人信任。”

战术要点 Combat Note

凯雷克使用奥术防御特性来尽可能增强自己微薄的护甲等级，而他的挑战等级则反映了他尽量频繁地使用烈焰爆破能力。就算一位或以上的盟友会被爆炸波及，那也没什么所谓。

随意：光亮术 light，法师之手 mage hand，魔法伎俩 prestigitation

丝绮拉 Skylla

丝绮拉——有时候也使用“查梅”的化名——和芭芭·雅嘎签订了魔契师契约。正是从这位强大的至高妖精那里，她获得了施展魔法的能力。丝绮拉恶毒而奸诈，她可能会背叛自己的盟友，也可能会助他们一臂之力。她声称自己与芭芭·雅嘎的契约让她比任何人都更适合担此重任，说服了凯雷克让她担任了恶毒同盟和沙漏集会之间的主要联络人。

恩德琳·月冢预见到了丝绮拉的盟友们有朝一日会背叛她，并与魔契师分享了这一消息。丝绮拉对此的反应是和同盟保持距离，而现在她为恩德琳工作。丝绮拉对重新加入同盟的想法持开放态度，但无论如何都不是在凯雷克掌权期间。

阵营。混乱邪恶

人格特质。“傻瓜才会把后背暴露给我。”

理想。“我渴求之物正是力量。一个人永远不可能一直占有太多力量。”

牵绊。“芭芭·雅嘎是我的宗主，而她教会了我自力更生。不过我感觉自己有必要和她的邪恶后代，沙漏集会的鬼婆们保持良好关系。”

缺陷。“我没有一个称得上是朋友的人。”

Note

战爵 Warduke

作为一位无情的雇佣杀手，战爵的服务可以很容易地买到。这位邪恶的剑士时不时会作为凯雷克的同谋和冒险伙伴为其服务。他并不特别聪明，因此他会将阴谋和策划的工作留给其他人。

战爵的恐怖头盔之下是什么模样？没有人知道，他从不摘下头盔，绝不向其他人露出自己的模样，但头盔下面的其实是一张冷酷、伤痕累累的角斗士的面容。

阵营。混乱邪恶

人格特质。“我用我剑上的烈焰和刀锋来说话，在它们面前，道理一文不值。”

理想。“我希望为我的工作拿到丰厚的报酬。”

牵绊。“恶毒同盟是个方便的生意——直到有更好的人出现。”

缺陷。“我宁愿死也不会投降。”

扎拉克 Zarak

扎拉克是一位毫无荣誉和良知可言的刺客。对于一个有着兽人血统的家伙而言，他生的矮小异常。如果不是一口腐烂的獠牙和灰白的肤色，他很容易会被认作是一个丑陋的、没有胡须的矮人。他唯利是图，如果盟友们挡在他和梦寐以求的财富之间，他会毫不犹豫地背刺他们。

阵营。混乱邪恶

人格特质。“我杀人，不打诳语（I mince people，not word）。”

理想。“我要的就是钱。”

牵绊。“一旦我同意要杀掉什么玩意儿，我会追着它直到工作结束。”

缺陷。“朋友只是还没和我为敌的家伙。”

战术要点 Combat Note

当对付复数敌人时，扎拉克喜欢在一定距离外投掷匕首作战。但当瞄准一个落单的家伙，尤其是他被雇佣来杀死的目标时，他更喜欢贴近距离并使用绞杀。如果失败不可避免，他会喝下自己的隐形药水 potion of invisibility 并逃之夭夭。

扎伽什 Zargash

扎伽什崇拜不死魔君奥喀斯。奥喀斯向他承诺，在终其一生虔诚侍奉之后，他会将扎伽什转化为一只吸血鬼。尽管作为一位奥喀斯活人祭司十分愚蠢，但扎伽什的心智已经扭曲，再无法被救赎。他的爱好包括屠杀活人和唤醒死尸。

阵营。混乱邪恶

人格特质。“我不害怕死亡，因为当我死的时候，奥喀斯会将我变为一个吸血鬼。”

理想。“熵不仅仅是存在的克星。它乃多元宇宙之末日，而我的工作便是加快它的发展。”

牵绊。“礼赞奥喀斯！愿 k 予我神力，以 k 之名，我将散播死亡，带来腐朽，唤醒不死！”

缺陷。“我把活人看做他们注定要成为的尸体，并以同样的态度对待他们。”

战术要点 Combat Note

战争是扎伽什的领域。在长袍下面，他穿着一件生锈的链甲衫。他喜欢以沉默术的区域包裹住敌方的施法者来阻碍他们施法。如果失败看起来不可避免，他会对自己施展气化形态 gaseous form，然后逃走，抛下队友自生自灭。
