# 炸脖龙

### 炸脖龙 Jabberwock

炸脖龙是一种孤僻、急躁的掠食者，他们萦绕于原始森林与古老遗迹之中。对炸脖龙具体细节的描述很难被得到，因为那些与其遭遇而又幸存的少数幸运儿们对它只留下了一点困惑的印象，而非对它整体的感知。一连串连在一起的描述将炸脖龙讲成了一种类龙型的强健生物，它用它的两只后腿走路时与它用四条腿走路时一样轻松。而它的双眼则能散发出炼狱般的激光。

当一只炸脖龙选定它的目标后，它会专注于攻击这个目标直到他被杀死（并被它吞噬），或炸脖龙被杀死，或直到目标通过传送魔法等其他方式逃跑。

如果一个炸脖龙被杀死，那么另外一只炸脖龙会在3d8年后再度出现，它会显现于老炸脖龙死亡地点的一千英里内。至今为止，没有任何一只未发育成熟的炸脖龙被目击到，而它们看取来也不会衰老。
