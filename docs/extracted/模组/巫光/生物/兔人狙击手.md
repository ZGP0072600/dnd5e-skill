# 兔人狙击手

兔人狙击手 Harengon 
Sniper

兔人狙击手们为一组兔人强盗提供火力支援，他们或躲在绝壁之上，或躲在树顶之上与掩体后方，准备在安全距离上解决掉目标。

狙击手们将他们的弩使地出神入化，当他们在用他们的弩发动远程攻击时，其目标无法从全身掩护以下的掩护中得到好处。另外，他们可以攻击最远达武器最大射程内的目标而不用承受一般情况下长射程需要承担的减值。

在发动攻击后，这兔人可以跳出对方的视野来让反击变得困难——前提是对方真的做得到的话。
