# 兔人强盗

兔人强盗 Harengon 
Brigand

与兔人强盗的遭遇一般在他们埋伏于此处的田间野道与大路旁发生，兔人们可以在这里轻松地发现、突袭他们的受害者。兔人强盗们乐于向旅者索要买路财并把他们弄得乱七八糟。他们的暴行在对上那些看起来像是迷路了的人或着急赶路的人时特别有敌意。

对付兔人强盗的最好方法是在数量上多过他们或是狠狠揍他们一顿。当局势不利时，他们会飞快溜走。
