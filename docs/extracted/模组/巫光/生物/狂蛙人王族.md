# 狂蛙人王族

啵灵蛙王族 Bullywug Royal

最大只或最聪明的啵灵蛙最终总是负责领导它的族群。一位王族经常骑乘或伴随着一只巨蟾蜍（见怪物图鉴以获得巨蟾蜍的数据）。

啵灵蛙王族 Bullywug Royal 中型类人 任意阵营护甲等级 15（兽皮甲，盾牌）生命值 52 (8d8 + 16)速度 30尺 游泳40尺力量16（+3）敏捷12（+1）体质14（+2）智力10（+0）感知11（+0）魅力14（+2）豁免：力量+5，敏捷+3技能：运动+5，威吓+4，隐匿+3感官：被动察觉10语言：啵灵蛙语，通用语挑战等级：3（700XP）

两栖 Amphibious。啵灵蛙可以在空气和水中呼吸。

蟾蜍骑士 Frog Rider。啵灵蛙王族在骑着蟾蜍坐骑进行近战攻击时具有优势。

蛙类交流 Speak with Frogs and Toads。啵灵蛙可以用啵灵蛙语同任何青蛙或蟾蜍进行内容简单的交流。

立定起跳 standing leap。无论是否助跑，啵灵蛙最远可以跳20尺，最高10尺。

动作 Actions

多重攻击 Mutiattack。啵灵蛙王族发动两次矛攻击。

矛 Spear。近战或远程武器攻击：命中+5，触及10尺或射程20/60尺，单一目标。

命中：6（1d6+3）的穿刺伤害，或使用双手发动 近战攻击时7（1d8+3）的穿刺伤害。 如果目标是中型或者更小的生物，则目标必须进行 一次 DC 13的力量豁免，豁免失败则应击倒地。

附赠动作 Bonus Actions。

呱之宣言 Croaked Decree（充能5-6）。王族发出响亮的宣言。在啵灵蛙王族60尺范围内，每一个能听到判决的盟友在它们下个回合的第一次攻击中。
