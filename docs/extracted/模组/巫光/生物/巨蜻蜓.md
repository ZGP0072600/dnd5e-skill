# 巨蜻蜓

### 巨蜻蜓 Giant Dragonfly

一只巨蜻蜓有着20尺长的翼展长度。它的双翅在悬浮于空、飞驰过空时振振作响，它会寻找更小的昆虫作为食物。

振动 Drone。当它振翅时，蜻蜓会发出一阵能够在120尺内被听见的响声。
