# 兔人

### 兔人 Harengons

兔人们是妖精荒原的本土兔子种族，不过他们经常迁居到主物质位面。他们热爱靠双脚旅行且鲜少在一个地方待上太久。

并非所有兔人都像此处所列的这样气量小。每一个兔人都有自己走过漫漫生命的独特道路，他们的性情在一部分上由他们所处的环境塑造。一个兔人可能会旅行到远方，在一路上结朋交友，醉心于开阔的野地与自由之感，并觅得内心的宁静。还有一些兔人可能会成为怀揣炽热梦想的坚强冒险者。欲了解更多作为玩家种族的兔人的信息，请查看本冒险的介绍部分。
