# 巨蜗牛

### 巨蜗牛 Giant Snail

这些巨大的软体动物会在被威胁时撤退到它们的坚硬甲壳中。它们特别容易因为碰到盐而受伤。

巨蜗牛 Giant 
Snail

大型野兽 无阵营

护甲等级 11（天生护甲）

生命值 22（4d10）

速度10尺，攀爬速度10尺

力量15（+2）敏捷3（-4）体质11（+0）智力3（-4）感知10（+0）魅力3（-4）

感官：黑暗视觉60尺，被动察觉10

语言：-

挑战等级：1/4（50XP） 
 熟练加值：+2

盐分透析 Salt 
Osmosis。当巨蜗牛于其回合开始时正与一磅以上的盐接触，那么它就会受到1d4点暗蚀伤害。当它没有在壳里面时，用一个动作将一磅盐撒到巨蜗牛身上会立刻对它造成1d4点暗蚀伤害，且它还会在下个回合开始时再受到1d4点暗蚀伤害（在盐分消失后）。

动作 Actions

猛击 Slam。近战武器攻击，命中+4，触及5尺，单一目标，命中：5（1d6+2）点钝击伤害

甲壳防御 Shell 
Defence。蜗牛缩进甲壳之中，直到它从壳中出来为止，它的 AC+4。它可以在它的回合中以一个附赠动作离开甲壳。
