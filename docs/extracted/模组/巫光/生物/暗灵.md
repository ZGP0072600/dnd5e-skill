# 暗灵

### 暗灵 Darklings

在古老的传说中，一个喜乐妖精对仲夏女王犯下了背叛的罪行。盛怒之下，女王诅咒了这个叛徒家族的每一个成员。如今他的真名已从历史中被抹去，但故事中叫他 Dubh 
Catha（通称“暗鸦”）。其他妖精提到他们的子孙时会用 dubh 
sith——“暗灵”来代指。大多数暗灵都偏好居住在隐蔽的洞穴或者其他种族的城镇角落里。在这些不见光之处，他们悄悄地充当着盗贼和刺客的角色。

暗灵 Darkling

仲夏女王的诅咒导致暗灵的身体吸收光线，这会对这些生物造成类似快速老化的效果。因此，当暗灵有暴露在阳光下的风险时，他们会用衣服密实地包裹住身体的每一部分。暗灵死亡时，在其一生中吸收的光能会猛烈爆开，把它连同所携带的大部分物品都烧为灰烬。

焚身耀 Death 
Flash。当暗灵死亡的瞬间，超乎寻常的光芒从他的身体和所带物品中爆发出来，覆盖他周围半径10尺，将它的身躯和物品中除了金属或魔法物品以外的全部燃烧成灰烬。任何位于区域内的生物，如果能够看到闪光的话，则必须通过 DC 10的体质豁免，否则目盲直到他的下一个回合结束。

光照敏感 Light 
Sensitivity。当位于明亮光照下时，暗灵在攻击骰，以及依靠视觉的感知（察觉）上有劣势。

动作 Action

匕首 Dagger。近战或远程武器攻击：命中+5，触及5尺或射程20/60尺，单一目标。

命中：5（1d4 + 3）外加7（2d6）暗蚀伤害。
