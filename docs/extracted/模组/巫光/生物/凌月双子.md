# 凌月双子

### 凌月双子 Selenelion Twins

月辉 Gleam 与日耀 Glister 是一对高等精灵双子，她们最近在巫光嘉年华 Witchlight Carnival 
上表演着展现其惊人平衡感、优雅的表演。她们在一起被称为凌月双子。（带食月落 Selenelion 是一种天文现象，它是一场能够同时看到太阳与被侵蚀的月亮的月食，而其发生时间恰好是在日落已先或是日出之后，因为此时这两个天体恰好在地平线之上的两处相对的天空。）

月辉与日耀都出生在这么一场天文现象中，而他们曾分别被月光（授予月辉）与阳光（授予日耀）祝福以魔法能力。这对双子能够轻松地以他们穿戴的面具被分辨出来，月辉戴着一面新月面具，而日耀戴着一面太阳面具。

这对双子被科瑞隆所祝福，因而她们能够在一场长休结束时改变自己的性别，尽管她们很少在双子中的另一位不这么做时单独改变自己的。下列数值栏会使用“她”或者“她的”来作为双子的一般称呼。

这对双子彼此喜爱，正如所有的一对兄弟姐妹一般。月辉是她们之中较为内向的那个，而日耀则是双子之中的外向者。她们两人的行为举止皆友善和蔼。附录 D 包含了对月辉与日耀的额外扮演笔记。

妖精血统 Fey 
Ancestry。凌月双子——月辉与日耀在进行对抗魅惑的豁免时具有优势，并且不会因魔法效应而陷入睡眠。

再生 Regeneration。只要凌月双子二人均存活且在彼此的60尺内，其中的任何一人在其回合开始时便恢复5点生命值。双子之一只会在其回合开始时有0点生命值且不再再生时死亡。

双子羁绊 Twin 
Bond。当双子均存活且位于同一位面时，她们彼此都能感知到对方的心情。

动作 Actions

匕首 Dagger。近战或远程武器攻击，命中+6，触及5尺或射程20/60尺，单一目标，造成6（1d4+6）点穿刺伤害。

月光射线 Moon 
Ray（仅月辉，3/天）。远程法术攻击，命中+5，射程60尺，单一生物。

命中：12（2d8+3）点光耀伤害，目标必须通过 DC 13感知豁免，否则就会被变成一只蝙蝠并持续一分钟，如同被法术变形术 polymorph 影响了一般。目标可以在其每回合结束时重复豁免，并在豁免成功时结束效应。

阳光射线 Sun 
Ray（仅日耀，3/天）。远程法术攻击，命中+5，射程60尺，单一生物。

命中：12（2d8+3）点光耀伤害，目标必须通过 DC 13感知豁免，否则就会陷入目盲一分钟。目标可以在其每回合结束时重复豁免，并在豁免成功时结束效应。

双生之眼 Twin 
Sight（短休或长休后充能）。以其心中之眼，双子之一可以最多看到另一位双子的视觉一分钟，这时她们必须位于同一位面且均存活。维持这个效果需要专注（如同专注一个法术般）。
