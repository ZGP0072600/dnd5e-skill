# 活化玩偶

### 活化玩偶 Living Doll

直到一个活化玩偶动起来前，它都很容易会被与一个非活化玩偶或是与一个毛绒玩具搞混。被困在活化玩偶中的自私精魂们怂恿它身边的人去糟糕地对待他人。玩偶以捉弄被嫉妒折磨者、沮丧者为乐，它会把他们的嫉妒、沮丧逼成绝望或偏执。由于害怕自身被摧毁，这玩偶鲜少直接伤害他人。当被迫自卫时，它会用自己疯狂扭动的关节削弱对方，它还会以引导敌人心中最糟糕之梦魇的方式攻击他们的心灵。

拟形 false 
appearance。当玩偶在战斗开始之时保持一动不动，那么它在先攻检定上具有优势。而且，若一个生物没有注意到玩偶的移动或行动，那么那个生物必须通过 DC 18智力（调查）检定才能分辨出这个玩偶是个活化玩偶

再生 Regeneration。玩偶在它的每回合开始时恢复5点生命值。若它受到了火焰或心灵伤害，那么它在下个回合开始时不会再生。只有在它以0生命值开始回合且不会再生时，玩偶才会被摧毁。

非凡本质 Unusual 
Nature。玩偶不需要食物、饮水或睡眠。

动作 Actions

贪婪爪攫 Grabby 
Hands。近战武器攻击，命中+2，触及5尺，单一生物。

命中：目标则被擒抱（逃脱 DC 6）且在其每回合开始时受到11（2d10）点心灵伤害，直到擒抱结束才停止。玩偶一次只能擒抱一个生物

附赠动作 Bonus Actions

咯咯癫言 Cackle（充能4-6）。玩偶向它能看见的30尺内的一到两个生物发出一阵咯咯笑声。每一个能够听见其笑声的目标必须通过 DC 11感知豁免，智力4及以下的生物自动通过豁免。若失败，那么目标则受到5（2d4）点心灵伤害且会在一分钟内因无法抑制的大笑而陷入失能。在目标的每回合结束时，他能够重复豁免，成功则结束效应。一个成功通过该豁免的生物在接下来24小时内对该效应免疫。
