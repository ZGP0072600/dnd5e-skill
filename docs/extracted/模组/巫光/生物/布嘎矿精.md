# 布嘎矿精

### 布嘎矿精 Brigganock

布嘎矿精是一种居住在矿洞中的生物，他们体小似鼠且总是手忙脚乱。每当有一个凡人用在他的生日蛋糕上吹灭蜡烛、在一口许愿井中抛下硬币的方式许下一个非魔法的愿望之时，那丝愿望的回声便会储存在妖精荒原地下深处的一块石头之中。布嘎矿精们寻找着这些寄宿了祈愿的石头，它们扔掉那些寄宿了恶意企图的石头并收集那些保有善愿的石头。矿精们将这些石头雕琢为宝石，它们用宝石装点自己的建筑与艺术品，又或者用来与其他妖精交换食物与它们喜爱的物品。一个比矿精们更加强大的妖精可以使用愿望石来探知许下愿望的人，用愿望石来授予他人超自然赠礼，或者用它来提供制造一件活化物件所需要的情感。当然，愿望石也可以用来做其他事情。

布嘎矿精的灵魂居住在它的身体之外，其显现为漂浮在矿精身侧且散发着苍白光芒的一茎光球，它能够帮助矿精在黑暗之中看清事物。

布嘎矿精拥有一些能够帮助他们生存下来的次级施法能力。它们同样拥有操控时间的能力，这能力允许它们在数秒之内完成原先需要一个小时才能完成的任务，尽管有的布嘎矿精在这些爆发性的暂时加速之间必须要休息上一会。

布嘎矿精们喜好啮咬像奶酪、莴苣之类的软食品。对于它们这个体型的生物来说，矿精们吃的东西量算是相当大，而且它们从来没闲下来吃上一顿好的大餐过。

妖精血统 Fey 
Ancestry。布嘎矿精进行对抗魅惑的豁免时具有优势，并且不会因魔法效应而陷入睡眠。

魂之明光 Soul 
Light。布嘎矿精被一个无实体、无敌的光球伴随着，这个光球中容纳了它的灵魂。布嘎矿精不能关掉光球的灯光或是控制它的明亮度。魂之明光会散发出10尺明亮光照与10尺微光光照。若一个布嘎矿精死亡，那么魂之明光便会熄灭。

掘道者 Tunneler。当使用一把镐子或类似的物件时，一个布嘎矿精能够以5尺的掘穴速度通过坚硬的岩层，并在其身后留下直径6寸的地道。

动作 Actions

镐击 Pickaxe。近战武器攻击，命中+4，触及5尺，单一目标。命中：4（1d4+2）点穿刺伤害。

施法 Spellcasting。布嘎矿精施展下列法术之一，它的法术不需要材料成分且魅力为其施法关键属性（法术 DC 11）

随意：次级幻象 minor illusion，维生术 spare the dying

每项1/天：化兽为友 animal friendship，妖火 faerie fire，融身入石 meld into stone，沉默术 silence

时间失速 Time 
Lapse（短休或长休后充能）。布嘎矿精能够加速其自身周围的时间流动，这使它能够在几秒钟内完成原先最多需要1小时的工作。这件工作不能影响除了布嘎矿精以外的任何生物，或任何被其他生物穿着、携带的物件，且其活动范围必须再一个10尺的立方体内。举个例子，布嘎矿精可以使用这个动作项来快速雕刻出一个南瓜，来烹煮、食用它的晚餐，来挪走一堆石头，或是在一段绳子上打上一打的结。

附赠动作 Bonus Actions

移动魂光 Move 
Soul Light。布嘎矿精能够往任意它能看见的未被占据空间移动它的魂之明光30最多尺距离。在当前回合结束时，魂之明光会回到布嘎矿精身上。
