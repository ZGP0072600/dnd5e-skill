# 巫光雇工

巫光雇工 Witchlight Hand 在你的孩童或青年时期时，你不知不觉的就加入了巫光嘉年华，从那时起就没有离开过，你在嘉年华里谋取了一份职位，执行幕后工作，保证嘉年华的正常运转。作为一名工人，你工作和享乐都贯彻极致。嘉年华带着你游历了许多奇妙的世界，以八年为周期，嘉年华会回到你家乡世界一次，但是你对这些世界几乎没有任何了解，因为你把所有的时间都花在了嘉年华里。你和其他的雇工有着不错的了解，但是对于嘉年华的雇主——巫老板和光老板——知之甚少，仅管你在嘉年华里工作了那么多年。
也许你加入嘉年华是为了逃离家乡惨淡的生活，又或是拜访新世界的愿望促使你加入，还可能是你想成为嘉年华里明星人物。现在你长大了，嘉年华的生活也不如当初那么吸引你。日常的工作开始变得乏味，嘉年华的循环性质让旅途变得枯燥。嘉年华再也不能满足你的好奇心。在嘉年华的大门外，可能会有更伟大的冒险在等候着你。

**技能熟练 **：表演，巧手
** 工具熟练 **：易容工具或一种乐器
** 语言 **：自选一种语言
** 装备 **：一套易容工具或一件由你选择的乐器，一套卡牌，一套嘉年华制服或戏服，一件饰品（通过投掷妖精荒野饰品表决定），和一个装着8gp 的小包。

** 嘉年华同伴 CARNIVAL COMPANION**
这么多年来，你和另一位嘉年华班底结下了友谊。投掷一枚 d8并参照嘉年华同伴表格决定谁或什么成为了你的好朋友，你也可以自己选择喜欢的选项。和你的 DM 一同讨论充实这段友谊。这位同伴会当你在嘉年华里时陪伴在你左右，但是它不会自愿离开嘉年华。
DM 能使用第一章的巫光雇工的数据卡来代表雇工，表演者，以及受训的动物嘉年华同伴的数据。其他的同伴的数据卡在怪物图鉴当中。

**特性：嘉年华班底 CARNIVAL FIXTURE**
巫光嘉年华为你提供免费的简朴住宿和食物。此外，你能漫步在嘉年华中，参观各种能人奇景而不需要花费，只要你不打扰表演者或是造成其他的麻烦。
