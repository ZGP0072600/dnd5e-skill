# 破碎波涛祭司

**破碎波涛祭司 Crushing Wave Priest**

破碎波涛祭司是狂热崇拜者的核心。他们聚集在秘密的阴谋集团中，举行黑暗的仪式来颂扬；奥莉德拉是水的破坏性的化身。他们相信用水来杀死敌人，比如通过仪式溺死，可以赢得奥莉德拉的青睐。
