# 修拉_坎德拉

**修拉·坎德拉 Shoalar Quanderil**

修拉·坎德拉是一个拥有快乐的生活方式和充满幽默感的水元素裔，他看起来似乎是最不可能成为一个被期待去搞破坏的邪教成员之一。然而，修拉是一艘骚扰着剑湾的海盗船的船长，也是一名高等破碎波涛的信徒。他把奥莉德拉的力量看作是使自己尽可能富有的一种手段。
