# 融石者

**融石者 Stonemelder**

融石者是从欧雷莫克那里获得力量的施法者，他们使用土元素魔法，将自己的身体包裹在岩石形成的甲壳中。他们是邪教的中坚拥护者，只对邪教领袖负责。甚至黑土祭司也不能指使融石者该怎么做。

每个融石者都携带着一种被称为黑土杖的武器。这类杖就像普通的狼牙棒，但在融石者的手中，它是欧雷莫克发泄愤怒的途径。
