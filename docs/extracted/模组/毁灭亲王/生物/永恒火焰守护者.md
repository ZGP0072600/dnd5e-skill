# 永恒火焰守护者

**永恒火焰守护者 Eternal Flame Guardian**

永恒火焰守护者是一群被火的力量所吸引的笨蛋，他们渴望向邪教的敌人们展示燃烧的感觉。他们点燃燃烧的东西是为了纪念元素之火，只是为了娱乐，但他们并不总是提前考虑附近可能会着火的东西。

**火焰武器 Flaming Weapon（在一次长休或短休后恢复）。** 以一个附赠动作，守护者可以用火焰缠绕一把挥舞中的近战武器。守护者不会受到这场大火的伤害，直到守护者的下个回合结束。当被火焰缠绕时，该武器在命中时会造成额外的3（1d6）火焰伤害。
