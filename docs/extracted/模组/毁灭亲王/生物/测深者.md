# 测深者

**测深者 Fathomer**

测深者是与邪恶的水元素公主奥莉德拉达成协议的信徒。除了他们的施法能力，测深者还能把他们的身体变成水，变成蛇一样的形状。在水蛇形态中，测深者可以获得许多元素生物所拥有的抵抗能力，以及用它们的水族身体抓住和粉碎敌人的能力。

测深者经常充当水元素教的间谍、渗透者和刺客，因为他们可以从锁着的门缝底下溜出去，或者轻松地穿过铁栏和类似的障碍。

**变形生物 Shapechanger（2次/天）。** 测深者可以使用它的一个动作变形为一只由水组成的中体型的蛇。或者变回到它的真实形态。 
在改变过程中，任何时候穿着或携带的物品都被归入蛇形状，直到物质体返回其真实形态才能进入。 
4小时后，测深者恢复其真实形态，除非它再次使用这种特性。如果该测深者被打昏或者死了，它也会恢复到它的真实形态。

在蛇的形态中，测深者可以获得40英尺的游泳速度，水下呼吸能，对毒素伤害的免疫，以及对火伤害的抗性，以及对非魔法武器的钝击、穿刺和挥砍伤害的抗性。它对以下的状态也有免疫力：力竭、擒抱、麻痹、中毒、束缚、倒地、昏迷。蛇形态可以进入敌对生物的空间并停留在那里。此外，若是流水可以通过的空间，该蛇也可以这样做而不受到挤压。

**奥莉德拉的盔甲 Olhydra’s Armor（仅类人形态）。** 测深者可以随意施放法师护甲 mage armor，不需要法术位与法术成分。
