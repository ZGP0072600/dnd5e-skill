# 犁风者

**犁风者 Windharrow**

犁风者是一个月精灵血统的半精灵投机主义者和强盗，原名哈拉尔德 灰矛 Harald 
Grayspear。当阿莉希 Aerisi 还和她的家人在一起的时候，哈拉尔德就被她的魅力所折服，并讨取她的欢心。当阿莉希离家出走成为大气先知时，她带上了犁风者。

在阿莉希 Aerisi 的坚持下，犁风者招募了一批新皈依大气教派的长笛乐手，称他们为空织匠 Windwyrds。大多数人根本没有音乐天赋，他们的音乐常常是刺耳刺又不和谐的。在所有的大气教徒之中，Windwyrds 是最不狂热，最珍惜自己的命的。 
Aerisi 一时兴起处理这些吟游诗人，取而代之的是其他提升者。 
那些掌握了一些乐器技巧的艺人生存时间最长，但是失败的代价在艺人之中创造了一个高度竞争的环境。

只要犁风者害怕阿莉希的力量，他就会忠于她。 
如果他的生命受到威胁，或者有更丰厚的报酬出现在眼前，风犁者便会头也不会地就背叛了大气教派。
