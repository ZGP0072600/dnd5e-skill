# 黑土守卫

**黑土守卫 Black Earth Guard**

黑土守卫是崇拜土地的勇士。他们是残忍、全副武装的暴徒，效忠于欧雷莫克。这种献身精神赋予了他们一种不可思议的能力，使他们能够坚守阵地，抵御猛烈的攻击。这也为他们赢得了佩戴由元素石制成的特殊板甲的权利，这种板甲在守卫被杀时就会失效。

这些守卫狂热地效忠于黑土祭司和邪教领袖。如果上级命令他们这样做，他们会战斗到死。那些更有潜力的人会成为穴居鲨鱼 burrowsharks。
