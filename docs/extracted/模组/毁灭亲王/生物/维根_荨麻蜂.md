# 维根_荨麻蜂

**维根·荨麻蜂 Wiggan Nettlebee**

维根·荨麻蜂是荨麻蜂家族的族长，这是一个富裕的半身人家庭，他们饲养家畜，耕种土地。他表面上是一个谦虚的农场主，禁止在他的农场上铺张浪费。维根认为自己已经找到了一个信奉主宰自然和利用其黑暗面的教派，于是加入了黑土教派。但维根并不知道教派计划的全貌。

 
维根不喜欢也不信任外来人。他认为外来人都是坏人，一有机会外来人就会窃取他的财务。他把黑土教派视为一个强大而必要的盟友，来保管他的财务以及维持他的地位。维根是一个贪婪的守财奴，一想到要从他囤积的宝库中拿出哪怕是几枚多出来的硬币，他也会感到明显的不安。
