# 巴斯蒂安_萨曼德

**巴斯蒂安 萨曼德 Bastian Thermandar**

就像许多对火的崇拜，巴斯蒂安燃烧着内心的火，但他的火是野心，而不是希望看到世界燃烧。巴斯蒂安正计划取代凡尼弗，为自己争取火种源。

作为一名奥术的实践者，巴斯蒂安从凡尼弗本人那里学到了很多火焰魔法，他在战斗中依赖于他的法术，他是一种“快速燃烧”的类型，试图在战斗的早期提供最大的影响。如果他知道一场战斗即将来临，但不能先发制人，巴斯蒂安就会变得更加谨慎，在他向战斗中投掷魔法之前，先用火墙来保护自己。
