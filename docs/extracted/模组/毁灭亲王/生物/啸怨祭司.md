# 啸怨祭司

**啸怨祭司 Howling Hatred Priest**

最狂热的气之崇拜者，祭司推动他们的身体到物理的极端，以减少他们的密度和质量，成为轻如树叶。憔悴、蜡黄的脸，凹陷的腹部，凸出的眼睛，紧绷的皮肤，每一根骨头都清晰可见，这些活的骨架仅仅依靠空气和对邪教教义的狂热奉献支撑着自己。风为他们服务。

哀嚎之怨的祭司把自己看作是延西冰活着的延伸，他们帮助新晋者摆脱旧生活的枷锁，拥抱一个“更高的目标”。一旦新成员被净化，祭司们就会决定他们是否值得作为提升者进行教化，或者更适合作为祭品为哀嚎之怨邪教服务。祭司们也坚持他们的先知 Aerisi 
Kalinoth 的教义，严厉惩罚任何偏离正道的人。

虽然没有人敢站出来反对，但大多数其他的大气教徒都憎恨这些祭司。那些挑战祭司的人往往会成为净化的新尝试的对象。
