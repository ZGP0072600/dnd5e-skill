# 叛逆的格鲁明克

**叛逆的格鲁明克 Grumink the Renegade**

格鲁明克是个变坏的矮人。他是一个混乱邪恶的矮人 **刺客 **（使用怪物图鉴的刺客数据），有着矮人的种族特征。嗜血，冷酷，唯利是图，格鲁明克是一个卑鄙的歹徒，他不择手段地得到他想要的东西。大多数小偷懒得与全副武装的对手较量，宁愿避免不必要的打斗。格鲁明克以建立冷血杀手的名声为乐，当他杀人的时候，他会不遗余力地去杀人，这样做可能会增加他的名声或者引起法律的注意。他可能会说这不是私人恩怨，但的确如此。

格鲁明克目前在一位名叫奥布拉图(Obratu)的恶鬼 oni 手下工作，奥布拉图正在寻找贝西尔默失而复得的夏宫的金库。这名恶鬼相信那座宫殿就在温柔誓言的神殿附近，在一个秘密的矮人隐居在跳舞的山谷里。它雇佣了格鲁明克和他的杀手小队来占领神殿并着手挖掘（见第六章“跳舞的溪谷”）。

**暗杀 Assassinate。** 在开始战斗的首个回合，刺客对任何未开始其回合的生物发动攻击时，其攻击检定具有优势。刺客击中被突袭生物的所有攻击均被视为重击。

**反射闪避 Evasion。** 某个敏捷豁免通过后伤害减半的效应指定刺客为目标时，其豁免成功则不受任何伤害，而豁免失败时则只受一半伤害。
