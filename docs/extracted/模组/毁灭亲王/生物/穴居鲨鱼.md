# 穴居鲨鱼

**穴居鲨鱼 Burrowshark**

崇拜着土教徒战士的穴居鲨鱼是训练凶猛鲨蜥坐骑的好手。当它们驾驭着强大的坐骑，撕碎敌人的时候，穴居鲨鱼会跳到地上毫不留情地杀死敌人。

穴居鲨鱼很像黑土守卫，因为它们都有不可思议的步伐及特殊的盔甲。对于穴居鲨鱼来说，欧雷莫克的一个额外赠礼，是在穴居鲨鱼和一个鲨蜥之间建立一种神奇的联系，允许乘骑兵使用它当坐骑钻洞并与它的坐骑的共享感知。

**黑土之缚 Bond of the Black Earth。** 穴居鲨鱼神奇地绑定了一个训练有素的鲨蜥作为它的坐骑。当穴居鲨鱼乘骑在它的鲨蜥上时，它会分享鲨蜥的感官，并且可以在挖洞的同时保持骑行。连结的鲨蜥遵守穴居鲨鱼的命令。如果它的坐骑死了，穴居鲨鱼可以训练一个新的鲨蜥作为它的坐骑，这个过程需要一个月。
