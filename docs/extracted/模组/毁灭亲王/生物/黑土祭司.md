# 黑土祭司

**黑土祭司 Black Earth Priest**

黑土的祭司们是欧雷莫克邪教的推动者和组织者。他们是真正的信徒，认为欧雷莫克是一种神圣的力量，他们已经发展出一种扭曲的教条，来解释为什么地球元素的邪恶注定要改造世界，统治一切。黑土祭司组成了一个由领袖组成的阴谋集团，他们的狂热和魔法力量是邪教的支柱。拥有不同寻常权力的个人在邪教中拥有更高的地位，但是祭司们为邪教领袖提供了他或她对其他欧雷莫克追随者的权威。
