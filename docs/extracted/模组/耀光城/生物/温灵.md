# 温灵

**温灵 Wynling**

顽皮而淘气的温灵保卫着高山和峡谷，抵御入侵者。温灵很少直接进行威胁，它们更喜欢通过盗窃和恶作剧来骚扰入侵者，以威慑他们。许多旅行者从温灵保护的土地上回来，他们讲述着装备消失的故事，好奇的眼睛从阴影中凝视着。

温灵通常生活在它们保护的山上，但它们经常在欢快的音乐和甜食的诱惑下冒险进入附近的定居点。远离它们开放的，野生的家园，并引起各种神秘的事故。

**动作 Actions**

** 猛击 slam。**近战武器攻击：命中+7，触及5尺，单一目标。命中：7（1d4+5）钝击伤害

** 山之斗篷 Cloak of the Mountain（充能4~6）。**温灵魔法性地化作隐形，包括它穿戴或携带的任何装备，持续1分钟或直到它发动攻击为止。

** 反应 Reactions**

** 骗术飞行 Trickster’s Flight。**当一个温灵可见的生物攻击温灵但未命中时，温灵可以立刻移动30尺。这种移动不会引起借机攻击。
