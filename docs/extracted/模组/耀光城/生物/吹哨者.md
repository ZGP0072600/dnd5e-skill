# 吹哨者

吹哨者 Whistler

从遥远国度的空气中发出刺耳的声音。它们很难被看到，因为它们不局限于空间中的一个点，在一种永久的物理不确定性状态下变得模糊。一只死去的吹哨者看起来像一只灰色的、没有特征的人形两足动物，四肢长，手指细。那些被吹哨者缠住的人无法拒绝它无声的七音符旋律，这是一种侵入并折磨心灵的超凡旋律。遇到吹哨者的生物很少逃脱； 
那些这样做的人永远被跟踪者可怕的曲调所困扰。

** 多重攻击 Multiattack。**吹哨者进行三次心灵冲刷。

：15（2d10+4）心灵伤害。

** 超凡旋律 Otherworldly Melody（充能5~6）。**吹哨者在至多两个其心灵感应范围内、可见的生物脑中吹起超凡的哨声。每个目标均须进行一次 DC 16的感知豁免，失败则受33（6d10）的心灵伤害，并在接下来的一分钟内对吹哨者陷入恐慌。该被恐慌生物可以在其自己回合结束时重复进行该豁免，豁免成功则终止该生物身上的恐慌效应。
