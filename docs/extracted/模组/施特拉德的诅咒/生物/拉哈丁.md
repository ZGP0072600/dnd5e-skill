# 拉哈丁 Rahadin

黄昏精灵拉哈丁是鸦阁城堡的总管，他已经忠心耿耿地服侍了施特拉德家族将近 500 
年之久。他是施特拉德永远的仆人、长久而可靠的战友，同时也是一名夺走上千人性命的铁血战士。 

**流亡 Exile。** 拉哈丁因为拒绝向一名在他看起来软弱且堕落的黄昏精灵王子跪拜而被判处流放之刑。因此，当黄昏精灵拒绝臣服于巴罗夫王之时，拉哈丁倒戈相向，协助巴罗夫王征服了他的同族。黄昏精灵的皇室血脉被彻底清剿，而其他的黄昏精灵们则被像兔子一样肆意屠杀捕猎。少数幸存者要么降伏于巴罗夫王，要么就被迫在维斯塔尼人中苟延残喘。巴罗夫王对拉哈丁十分满意，赐予这名黄昏精灵荣誉王室成员的身份。

**宫廷总管 Chamberlain。** 在巴罗夫王去世后，拉哈丁继续作为施特拉德的一名将军为他效力。当战争最终结束，施特拉德把自己的注意力转移到了建造鸦阁城堡上后，拉哈丁负责挑选那些巫师与匠人，并确保把他们带到巴洛维亚参与建设。数年之后，施特拉德任命拉哈丁为他城堡的总管。拉哈丁乐意完成施特拉德的所有要求，通过经常鞭笞那些没有达到他严格标准的城堡佣人，他将恐惧深深植入他们的内心。 

当一名叫做帕特里娜·维里科芙纳的黄昏精灵敲响城堡的大门时，他就知道她会勾搭上斯塔德，然而他对女精灵的动机抱有怀疑。帕特里娜以不朽的秘密——施特拉德最渴望的东西——来引诱施特拉德。她告诉了施特拉德一个被称为琥珀神殿的知识禁库，而获得永生的秘密就被藏在那里。在施特拉德忙于探索琥珀神殿的期间，拉哈丁就全权代管了他的事务，同时也开始寻找一个能够将施特拉德的注意力从帕特里娜身上引走的女人，但他没能找到。不过，当施特拉德的兄弟谢尔盖找到塔季扬娜的时候，总管的目标达成了。 

塔季扬娜正是施特拉德喜欢的类型——美丽精致且举止文雅。当施特拉德回到鸦阁时，她立刻牢牢吸引住了他的目光。

拉哈丁很高兴地通知帕特里娜，鸦阁城堡不再欢迎她的到来。

即使在塔季扬娜去世、施特拉德转化为吸血鬼之后，他对施特拉德的耿耿忠心依旧未有分毫动摇。他继续履行着自己主人的命令。施特拉德急于忘掉塔季扬娜，便引诱了更多的女性来到城堡，从中又挑选了一些，在她们的生命被榨干、被变成初生吸血鬼之前，娶她们做自己的新娘。拉哈丁会确保那些女性在鸦阁城堡期间拥有大量的奢华珠宝与华贵服饰，并且享有优渥舒适的生活环境。 

**刽子手 Executioner。** 当获悉塔季扬娜的死讯以及施特拉德受到诅咒的消息之时，帕特里娜·维里科芙纳和她的族人们正生活在维斯塔尼人中。青春永驻的帕特里娜返回了鸦阁魔域希望能获得施特拉德的爱。然而这一次显而易见，她是渴求施特拉德的力量而来，而施特拉德也绝不会爱上她。拉哈丁认定帕特里娜必遭受此前到来的其他女性一样的命运。但事实证明他猜错了。帕特里娜的族人为了阻止施特拉德娶她做新娘，在此之前以石刑将帕特里娜活活砸死。 

施特拉德因为黄昏精灵们从他手中夺走了帕特里娜而勃然大怒。在找寻到帕特里娜的尸骸并将之埋葬在鸦阁城堡的地下墓穴后，他派遣拉哈丁前去惩罚那些黄昏精灵。拉哈丁屠戮了所有女性黄昏精灵阻止他们繁衍后代，并且将亲手促成石刑的帕特里娜的兄弟卡西米尔的耳朵生生割下。

** 牵绊 Bond。**“我是巴罗夫·冯·扎洛维奇的儿子，我会服侍他的儿子——我的主人与手足兄弟——直到永远。” 

** 缺点 Flaw。**“我已经杀戮了数以千计的生命。我还将继续屠杀数以千计的生命来保护冯·扎洛维奇家的传世的荣耀。”
