# 捣蛋皮德尔二世 Pidlwick II

在多芙尼雅·迪里西亚女公爵 Duchess Dorfniya Dilisnya 
的丈夫在一场战斗中殒命之后，她便着眼于成为施特拉德·冯·扎洛维奇伯爵的新娘，然而她没能博得他的爱。虽然如此，她依旧频繁地拜访城堡，当然身边还跟着她形影不离的小丑——讨人喜欢的捣蛋皮德尔。这个小矮人就如照入同鸦阁城堡的一缕阳光，尽管他没能逗乐施特拉德，但是却用他的把戏和笑话让塔季扬娜和谢尔盖喜笑颜开。这也是为什么施特拉德并不拒绝捣蛋皮德尔和女公爵拜访的原因。 

渴望取悦施特拉德同时也作为礼节性的回报，女公爵委派传奇玩具工匠弗里茨·冯·魏尔格制造了一个模仿捣蛋皮尔德的发条人偶，作为送给施特拉德一家的礼物。尽管女公爵用心良苦，但是这个人偶完全没有捣蛋皮德尔的能力，它没能带给哪怕一个人以欢笑。甚至在捣蛋皮德尔自己耗费数月调教这个人偶后，它也无法开口说话，行动只是更显笨拙而非有趣。

 某个严酷的冬天把女公爵——和她的小丑与小丑人偶一起——困在鸦阁城堡数月。不久之后，女公爵因病辞世，此后塔季扬娜就邀请捣蛋皮德尔留在鸦阁城堡。 

**捣蛋皮德尔一个足矣 One Pidlwick Too Many。** 冯·魏尔格不是一个平凡的玩具工匠，他会在自己的每一个造物中添加自己的一小部分。换句话说，他的每个造物都带有一点其制造者的疯狂。捣蛋皮德尔二世知道只要捣蛋皮德尔留在鸦阁，它的存在就毫无意义，因此他将捣蛋皮德尔从一条高耸的楼梯上推下，谋杀了他。而其他任何人都以为这仅仅是一起意外。在之后的日子里，捣蛋皮德尔二世一直试图取代捣蛋皮尔德的价

值，但仅仅是看见他就会让塔季扬娜心烦意乱。它再也没被要求去表演过。最终，它像一个被抛弃的玩具一般让人锁了起来。 

**邪恶的玩具 Evil Toy。** 捣蛋皮德尔二世被保存在一间客房旁的小衣柜里。偶尔在有人待在那时，人偶会在午夜偷偷溜出衣橱，用一个枕头蒙在客人的头上使其窒息，然后再撤回衣柜。城堡中的里的佣人们从来未曾想到这个人偶可能是罪魁祸首，而是猜测客人们是在梦中死亡。 

但施特拉德并未被欺骗，他很快就意识到这个发条人偶已经杀人成性。然而，他并没有将这个人偶销毁，相反的，他将这个小丑留了下来，时不时地用来“处理”一些烦人的客人。 

在塔季扬娜和谢尔盖死后，这个城堡几乎已经被废弃，自然不在会再有供捣蛋皮尔德二世“娱乐”的客人了。所以，人偶离开了小衣柜，开始寻找新的藏身之所。它畏惧着施特拉德，而渴望追随那些给予它梦寐以求的关注的人。 

捣蛋皮德尔二世算得上是个超大玩偶——作为一个 4 
尺高的机械构装物，它体内塞满齿轮、弹簧和其他的机械部件，还巧妙地装配在一起使它能够拥有栩栩如生的外观。它的皮肤是用皮革缝制成的，被紧紧地蒙在有关节的木头框上。捣蛋皮德尔二世将炉灰擦在自己的眼睛和嘴巴周围，使得它看起来有一双三角形的眼睛和杰克南瓜灯一般的锯齿状微笑。
