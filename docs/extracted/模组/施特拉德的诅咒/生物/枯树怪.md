# 枯树怪 Tree Blight

枯萎怪 
Blights（详细描述见《怪物图鉴》）是邪恶的、可动的植物类生物，而它们之中的枯树怪是一个特别庞大的种类。

它看上去就像一株枯木或者一个有着海绵状木质血肉、荆棘触手和橡胶树根的 
30 尺高的巨大树人。它以血液为树汁浸透了全身，因此不易着火。 

**凶残的肉食生物 Vicious Carnivore。** 这种树以温血动物为食，在由此产生的屠杀中它们能够获得极大地快感。它会用自己的树枝击打猎物并且能够用自己的根须碾碎它们。它还可以张开自己那张裂开的、长满利齿的血盆大口咬向任何被自己根须抓住的生物。枯树怪的根须可以被切断，尽管这样并不会造成伤害。 

**枯萎憎恨 Blight Animosity。** 一个枯树怪通常会和其他不同种类的枯萎怪一同战斗，但是它痛恨其他枯树怪，以至于只要有机会就会互相攻击。另外，枯树怪也痛恨树人，并且这种恨意是相互的。
