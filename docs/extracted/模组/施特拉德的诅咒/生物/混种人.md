# 混种人 Mongrelfolk

混种人并非是自然种族，他们或者他们的先祖都是在经历了耸人听闻的魔法实验之后才转变成现在的形态。而在转变完成后，他们仅仅能够残存一丝原先样貌的痕迹。他们的人形躯体融合了各种野兽的特征。例如，一个混种人可能有着矮人的身体、混杂着猫和蜥蜴特征的头部而一只手臂的顶端是螃蟹的钳子、一条腿的末端则是一只偶蹄。而其他的混种人也可能有着牛的皮肤和尖角、蜘蛛的复眼、青蛙一样的腿和一条有鳞蜥蜴的尾巴。每一个混种人都是人和数种动物形态的疯狂混杂体，这也导致了他们的行动缓慢而笨拙。 

**模仿声音 Sound Mimicry。** 混种人有畸形的嘴巴和声带。他们说通用语时语句支离破碎，同时夹杂着各种动物的叫声和胡言乱语。然而，他们可以有效模仿他们听见过的动物或人类声音。混血儿不足以使用这些声音作隐蔽的交流方式，但他们可以用这些声音来引诱敌人进入陷阱或分散他们的注意力。 

**边缘弃儿 Outcasts。** 混种人鲜少能够在其他类人社会中获得欢迎，通常在那里他们只会受到虐待、奴役或者干脆无视。

他们基本上生活在文明的边缘，居住在废墟、废弃的建筑物或者其他人形种族曾经居住或者建造的场地。当他们不在自己家园之时，他们往往显得胆怯懦弱；而在他们自己的巢穴中，则会变得无比凶猛残忍。 

**伪装专家 Camouflage Experts。** 混种人通常会将他们的畸形外貌掩盖在斗篷与连帽之下。在这样的打扮下，他们有时会被当作粗壮的人类或者瘦小的矮人。他们热衷于伪装。他们会将树叶、树枝附在自己的斗篷上，用棕色的颜料覆盖自己的皮肤并会专门编制用以隐藏的草网。当在野外狩猎或者站岗守卫自己的巢穴时，他们都会采取这种伪装。因此一个伪装中的混种人在用以躲藏的隐匿检定中获得优势，直到他已经被看见。 

**恐怖后裔 Horrific Offspring。** 混种人有可能被一发高等复原术 greater restoration 
还原成其最初的样子，但该手段在混种人后代身上却不起作用。只有被魔法转变而来的混种人可以用魔法恢复至最初的样子。但生下来就是混种人的人们是彻彻底底的混种人，身上并没有可供解除的魔法或者效应。混种人能够和其他的类人生物交配繁衍，但生下来的孩子几乎都是混种人。（大概只有百分之一的子嗣生出来像他们父母中非混种人的那一方。）
