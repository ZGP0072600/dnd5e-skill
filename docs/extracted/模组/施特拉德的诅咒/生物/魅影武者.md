# 魅影武者 Phantom Warrior

魅影武者是那些死于战场上或为捍卫誓言而死的战士或骑士的残留幽魂。除了显得半透明之外，它们看起来就如同依旧活着一样。 

**不负使命 Task Driven。** 尽管经常被与幽灵混为一谈，但是幽灵武士并不会被对未尽之事的渴求所束缚，它可以在任何时候选择结束自己的不死存在。它的灵魂完全是心甘情愿地滞留在世上，要么是处于对前主人的忠诚，要么就是它必须完成某项任务来满足自己的荣誉感与责任感。比如，一个誓死保卫堡垒的护卫可能会在死后化为魅影武者归来继续守卫此地，直到有新的守卫来到接替它的职责，或者堡垒被彻底摧毁，才会永久消失。另外，魅影武者的复苏时间通常会是其本体死亡的24 
小时后。 

**记忆淡薄 Faded Memories。** 魅影武者会保留它生前的阵营和性格，并且它可以记住自己是如何死亡的。但是，它仅仅能模糊地回忆起死前短暂时间内的记忆，至于更久远的记忆则会完全遗忘。一名魅影武者通常可以记住他生前 
1d10+10 天内发生的事，而任何在此之前的记忆将会化作无法看透的迷雾。

**力场加身 Forceful Presence。** 尽管它们如今是无形的，但是魅影武者可以利用周围的能量将来犯的攻击偏转并以强大的力量予以痛击。在魅影武者的幽魂护甲、盾牌以及武器的外围包裹有一层无形的能量，因此他们会坚硬如钢同时也不会影响武士们如同灵体般穿透其他固态物体。 

**亡灵本质 Undead Nature。** 魅影武者不需要空气、食物、饮水或者睡眠。
