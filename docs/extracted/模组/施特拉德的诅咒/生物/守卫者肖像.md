# 守卫者肖像 Guardian Portrait

守卫者肖像看起来像是一件精心制作的精美艺术品，通常会以写实的风格描绘一个重要人物。这一幅画和它的画框是以强大的魔力结合在一起并不可分割的。 

**活体画像 Living Image。** 画作中人物的眼睛被赋予了黑暗视觉，而且看上去似乎会跟着他们面前的生物移动。 

**天生法术 Innate Spells。** 当守卫者肖像攻击时，画中的人会动起来并且就如同活人一样（尽管只是二维的）。守卫者肖像无法进行有效的挥砍攻击，但是它拥有一张能够施放的天生法术表，当它进行施法时，画中人物会做出该法术所需的所有姿势成分和言语成分。
