# 巴洛维亚巫师 Barovian Witch

这群被称为巴洛维亚巫师的疯狂男女们与施特拉德和鸦阁魔域中蕴含的黑暗力量签订了契约以换取魔法和长生。他们倾向于生活在阴影之中并且能在黑暗中视物。当他们在外游走时，他们会使用 
*变身术 alter self* 将自己变成不易引起怀疑的外形，并且他们会用咒语使自己长出锋利、森长的利爪作攻击武器。 

**施特拉德的兄弟姐妹 Brothers and Sisters of Strahd。** 巴洛维亚巫师行事毫无顾忌。他们为了换取力量可以和任何人交易；反之亦然，他们可以为了力量背叛任何人。唯一使他们畏惧的只有施特拉德，他的任何想法都是对他们的命令。巴洛维亚巫师们有时会把自己称作施特拉德的手足，尽管他们从来不在施特拉德面前这么说。 

**松鼠病与爱猫狂 Pack Rats with Cats。** 巴洛维亚巫师都是是收藏狂。他们会认为他们发现的任何东西——例如一片小碎骨、一只死掉的啮齿类动物、一把尘土或一些其他根本没有价值的物品、材料——都可能是价值连城，或者可以被用作某个法术的媒介、某个仪式的对象亦或是某种毒药的配方。 

巴洛维亚巫师们能够利用 *寻获魔宠 find familiar* 
法术为自己召唤亲密的魔宠。他们尤其喜欢猫，尽管蛇和蟾蜍也是常见的选择。这些动物会蛰伏在巫师藏身之处的杂乱环境中，很少会远离它们邪恶的主人。
