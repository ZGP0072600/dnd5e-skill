# 活化战斗扫帚 Broom of Animated Attack

被施以战斗活化魔法的扫帚很容易被和飞行扫帚搞混。但是事实上，活化战斗扫帚会攻击任何试图抓取或骑行它的人。

**飞行扫帚 Flying Broom。** 有一些活化战斗扫帚能够让它们的创造者骑行它们，在这种情况下它们会表现得就像典型的飞行扫帚。不过，活化战斗扫帚的承载量只有飞行扫帚的一半。（见《城主指南》第 
7 章“宝藏”）
