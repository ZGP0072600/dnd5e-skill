# 艾丝美拉达·德·阿维尼尔 Ezmerelda d'Avenir

艾丝美拉达·德·阿维尼尔，一位维斯塔尼人，是鲁道夫·范·里希腾的门生——尽管事实上她与这吸血鬼猎人的第一次相遇一点也不令人愉快。 

**悲剧的见证人 Witness to Tragedy。** 当艾丝美拉达还是一个小女孩时，她的家人绑架了鲁道夫·范·里希腾十几岁的儿子伊拉斯谟，并把他交到了一个吸血鬼的魔掌中。即使在多年后的今天，艾丝美拉达仍然能听见伊拉斯谟的恳求他们放过自己的声音。这成为了她童年时的噩梦。 

在儿子被绑架后，范·里希腾很快追踪到了艾丝美拉达的家人，但却没能赶在维斯塔尼人卖掉男孩之前。尽管范·里希腾本来能伤害他们，但他并没有这么做，而只是质问艾丝美拉达的父母他失踪儿子的下落。得到满意回答的范·里希腾带着信息离开时饶了他们的命。艾丝美拉达见证了范·里希腾仁慈的行为并被其深深打动。 

**范·里希腾的悲剧故事 Van Richten’s Tragic Tale。** 十五岁时，艾丝美拉达仍然困扰于她的家人曾经对范·里希腾做过的事，于是便离家出走了。两年后，经历了许多可怕的冒险的她追寻到了范·里希腾。范·里希腾认为她是一个维斯塔尼人刺客，将一柄剑抵在她的咽喉上威胁要让她见见血。艾丝美拉达使他相信她是真心想帮助他找到失踪的儿子，于是范·里希腾讲出了故事里最悲惨的那个部分——他找到了自己的儿子，但伊拉斯谟已经被转化为一个初生吸血鬼。当伊拉斯谟恳求他的父亲给予他救赎的时候，范·里希腾用结束儿子的存在的方式完成了他的心愿。 

**告别 Farewel。** 艾丝美拉达留在范·里希腾身边待了两年，帮助他追踪并杀死了许多暗夜里的生物。但范·里希腾没有办法让自己完全信任一个维斯塔尼人，他对她隐瞒了许多事情。

两个吸血鬼猎人使彼此神经紧张，他们的争论变得更加频繁了。

最后，艾丝美拉达建议在他们间还保有一些友谊时拆伙，范·里希腾同意了。

**艾丝美拉达的秘密 Ezmerelda’s Secret。** 自从告别了范·里希腾，艾丝美拉达已经积累了一笔可观的个人财富，其中的一

些她用来购买了一辆马车，用以携带她猎杀吸血鬼的装备。在一次不太成功的冒险中，一头狼人咬掉了她膝盖以下的右腿，尽管没有患上狼化症，艾丝美拉达还是养了好几个月的伤。她委托一位工匠大师手工制作了一个包含小腿和脚部的义肢。几经尝试后，那名工匠做出了一个恢复了她移动力的义肢。从那以后她就很好地适应了这条假肢，并小心地将其隐藏起来。

**对大吸血鬼的猎杀 The Great Vampire Hunt。** 和一个维斯塔尼大篷车队结伴同行时，艾丝美拉达听到传闻说鲁道夫·范·里希腾已经前往巴洛维亚，去杀死那一切吸血鬼中最强大的那名。她认为他可能需要帮助，就旅行了数月到达了施特拉德的领土。她驾着她的马车去了瓦拉吉镇，并且在那里打听到附近有一座旧塔。这座塔听起来很像范·里希腾会用来做基地的那种地方。艾丝美拉达到达旧塔的时候，她发现了一些范·里希腾的随身物品，但是吸血鬼猎人本人却不在那儿。虽然急切地想要知道导师身在何处，但艾丝美拉达也渴望得到导师的信任和尊重。为此，她已经通读了范·里希腾的研究成果并学习了施特拉德和鸦阁城堡的相关知识，打算独自一人迅速除掉吸血鬼。
