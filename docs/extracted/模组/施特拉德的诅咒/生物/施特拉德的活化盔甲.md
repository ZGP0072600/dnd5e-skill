# 。施特拉德的活化盔甲 Strahd's Animated Armor

这件铠甲原先是施特拉德尚且作为一个生者之时在战场中所穿，如今它作为一具无头活化板甲获得了生命。铠甲整体被漆成紫红色并以金色天使图案装饰其上。 

**邪恶之物 Thing of Evil。** 
施特拉德将自身存在的一小部分注入了这句可自主活动的铠甲中，使它具备了大多数活化物件所没有的恶意。除此之外，为了使活化盔甲成为一个更好的城堡守护者，施特拉德还重新强化了它并添加上了许多永久的法术效果。 

铠甲能够理解通用语但是只会服从它主人的命令。
