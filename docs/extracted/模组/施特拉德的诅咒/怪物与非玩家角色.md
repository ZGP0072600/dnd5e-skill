# 怪物与非玩家角色

**附录 D：怪物与非玩家角色 Monsters and Npcs**

巴洛维亚是一片遍布吸血鬼、幽魂和狼人的土地。此外，探索这片邪恶国度的冒险者们还会遭遇其他在暗夜中作祟的存在，其中也包括施特拉德的邪恶造物们。本章将对出现在此次冒险中的新怪物做出详细介绍，此外，还介绍了几个吸血鬼的盟友和敌人，这些人的命运已注定和冒险者们有所纠葛。当然！施特拉德本人的信息也会在下文中列出。
