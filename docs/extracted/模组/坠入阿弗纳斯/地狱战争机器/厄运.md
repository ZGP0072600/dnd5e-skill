# 厄运

厄运 Mishaps

如果地狱战争机器处于运动中是发生了一下情况之一，请根据“厄运表”骰出随即情况：

l 地狱战争机器一次受到了超过厄运阀值的伤害。

l 地狱战争机器在属性检定中失败（或是驾驶员使用战车的能力值在属性检定中失败），且投掷结果比结果少5或更多。

如果此厄运有维修 DC，则可以通过维修来避免此次厄运（见下方“维修”一栏）。

| D20 | 厄运 | 维修 DC |
|---|---|---|
| **厄运 Mishaps** |  |  |
| 1 | 引擎起火。引擎处起火并蔓延到了车上。直到厄运结束前，所有在车内部的生物都要在其回合开始时，受到10（3D6）点火焰伤害。 | 15（敏捷） |
| 2-4 | 转向锁死。战车只能直线前进。直到厄运结束前，它的所有敏捷检定和敏捷豁免自动失败。 | 15（力量） |
| 5-7 | 熔炉破裂。厄运结束前，车辆的速度减少30尺。 | 15（力量） |
| 8-10 | 武器故障。厄运结束前，战车的一个武器无法使用（由 DM 决定）。如果此车辆没有正在使用的武器，则无事发生。 | 20（力量） |
| 11-13 | 蔽目烟雾。直到厄运结束前，驾驶室处于重度遮蔽。所有在驾驶室的生物都会因烟雾而目盲。 | 15（敏捷） |

结束厄运 End a Mishap

生物可以用一个动作，根据厄运的效果进行一次属性检定（见厄运表），如果战车在移动中，则此检定取劣势。如果生物拥有用来维修的工具的熟练项，则在检定中加入熟练加值。检定成功，则厄运结束。没有维修 DC 的厄运无法被维修。
