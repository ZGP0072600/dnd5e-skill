# 车辆损坏与维修

车辆损坏 Vehicular Exhaustion

九狱严酷的环境会使得地狱战争机器的停止正常运作，并最终完全损毁。这种损耗可以用力竭状态来体现，如《玩家手册》附录 A 中所写，但做出如下变化：

l 当一辆地狱战争机器达到6级力竭，它的 HP 降为0，且车辆完全损坏。

l 解除战车力竭状态的唯一手段是维修。

维修 Repairs

当地狱战争机器受损，遭到厄运，或是得到1级或更高的力竭时，人物可以尝试维修战车。进行维修的生物必须满足下列条件：

l 维修过程中，生物不能驾驶车辆或是使用武器。

l 损坏区域必须在生物的触及范围内。

l 生物必须持有相应的工具（如铁匠工具或修补工具）。

在开始维修前，生物必须决定维修的目的是结束厄运，解除1级力竭，或是恢复战车的 HP。三种选项的详述见下文。

移除力竭 Remove Exhaustion

如果地狱战争机器有1级或更多等级的力竭，那么生物可以尝试用1个小时来减少战车的力竭等级。战车必须处于静止状态，且生物必须携带有维修必要的配件。经过1小时的工作后，维修的生物需要通过一个 DC 15的智力检定，如果生物拥有所用工具的熟练项，则加上熟练加值。如果检定成功，则战车的力竭等级降低1级。如果失败，则力竭等级不变，但可以用同一配件再次尝试维修。

回复生命 Restore Hit Points

如果地狱战争机器受到了伤害，但剩余生命值大于1，那么生物可以用一个小时或更多的时间，通过修补车体及更换零件的方式回复车辆的 HP。战车必须处于静止状态，且生物必须携带有维修必要的配件。经过1小时的工作后，维修的生物需要通过一个 DC 15的敏捷检定，如果生物拥有所用工具的熟练项，则加上熟练加值。如果成功，则战车回复2D4+2点 HP。如果失败，则 HP 不变，但可以用同一配件再次尝试维修。
