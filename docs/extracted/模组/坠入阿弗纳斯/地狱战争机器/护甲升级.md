# 护甲升级

护甲升级 Armor Upgrades

一辆地狱战争机器可以获得一次护甲升级。升级的例子如下。

卡尼亚装甲 Canian Armor

战车所用的地狱铁的材料是从卡尼亚，九狱中最寒冷的一层开采的。装备卡尼亚装甲的地狱战争机器的 AC=22+其敏捷调整值。当战车处于静止状态时，对战车的攻击取优势。除了原有的伤害免疫外，这种战车也对免疫寒冷伤害。同时，战车也会免疫极寒效果。

镀金死亡装甲 Gilded Death Armor

这种战车的表面，镀了一层从大魔鬼马曼处偷窃或交易来的黄金。在战车的 HP 降为0，被完全摧毁之前无法收回此黄金。装备有镀金的死亡装甲的战车对钝击，穿刺和挥砍伤害有抗性。

灵魂尖刺装甲 Soul Spike Armor

战车的装甲上覆盖着雕刻着渎神符号的尖刺。附在这些尖刺上的发光的幽灵在痛苦中哀嚎着，并伸出手来。

如果一个非魔鬼生物死在战车的30尺内，它的灵魂（如果有的话）会被剥离它的身体，加入这些尖刺中的灵魂。只要生物的灵魂被困，此生物就无法以任何形式复活。可以通过摧毁战车，对困住灵魂的尖刺施放移除诅咒 remove curse 或相似的法术，或者摧毁尖刺的方式来让被捆的灵魂重获自由。每根尖刺都被视作小型物体，拥有19AC 和15HP，拥有对除光耀伤害以外的所有伤害抗性。
