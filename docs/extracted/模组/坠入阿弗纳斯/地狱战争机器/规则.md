# 规则

规则 Rules

地狱战争机器的使用规则如下。

魔法要素 Magical 
Elements

地狱战争机器的引擎，熔炉以及武器都是具有魔法，因此在反魔场中无法运作。当战车的引擎或熔炉遭遇此种效果时，战车会停止运作，并且在引擎和熔炉都起来反魔场前都无法重新启动。

借机攻击 Opportunity Attacks

地狱战争机器会像其他生物一样，正常触发借机攻击。当地狱战争机器触发借机攻击时，攻击者可以选择战车本身，或是任何正在骑乘战车或在战车内部的生物，只要他们不处于全隐蔽并且位于攻击者的攻击范围内。

操作台 Action 
Stations

生物可以使用其所在的操作台的一个动作。在一个操作台动作被使用后，知道该生物的下一个回合开始前，它都不能被再次使用。每个操作台只能同时存在一个生物。

不处于操作台的生物可以坐在乘客席或是攀附在战车的外部。此状态下的生物可以如常进行动作。

驾驶室 Helm

地狱战争机器的驾驶室包括一张椅子，一个方向盘，杠杆，踏板，以及其他操纵装置。驾驶室内需要一位驾驶员来驾驶。没有驾驶员的地狱战争机器会自动在敏捷豁免中失败。

拥有陆地载具熟练的驾驶员，可以在战车进行属性检定和豁免检定时，在战车属性的基础上加入自己的熟练加值。

驾驶 Drive。当地狱战争机器的引擎正在运行时，驾驶员可以用一个动作给战车加速或者进行紧急制动。当战车处于移动中时，驾驶员可以沿任何路线驾驶它。

当驾驶员处于失能状态，或离开驾驶室，或不做任何动作来改变战车的移动路线和速度，那么战车会按照驾驶员上一回合所设定的速度和方向前进，直到撞上足够令它停止的障碍物为止。

附赠动作 Bonus 
Actions。驾驶员可以用一个附赠动作做出如下行为之一：

l 启动或停止地狱战争机器的引擎。

l 在引擎已经启动的前提下，让战车使用疾走或撤离动作。

l 向熔炉内放入一枚灵魂币或是一瓶恶魔秽液（见下方的“灵魂燃料”和“恶魔秽液加速”）

武器操作台 Weapon 
Stations

地狱战争机器的每个武器都会占据一个操作台。

战车上的武器可以被替换成不同种类（见“武器替换”），前提是操作两种武器所需要的成员要求相同。

灵魂燃料 Soul 
Fuel

每辆地狱战争机器的引擎核心都是一个用灵魂币作为燃料的熔炉。战车的驾驶室内由个窄槽，可以从此处放入灵魂币。放入此槽的灵魂币会进入熔炉，熔炉的大小比车辆的大小要小两级。

地狱战争机器的熔炉会立即消耗掉灵魂币，并同时一次性消耗掉灵魂币剩余的充能，在此过程中，灵魂币会被摧毁。原本困在灵魂币内的灵魂会被困在熔炉内，根据灵魂币被消耗时剩余的充能来给引擎提供相应时间的能量：1次充能，24小时；2次充能，48小时；3次充能，72小时。如果在给引擎的充能时间结束后，灵魂依旧被困在熔炉内，那么它会被摧毁。此情形下，即使是神圣干预也无法救回被摧毁的灵魂。

无论其大小，熔炉都可以容纳任意数量的灵魂，它们痛苦的悲鸣声可以在60尺外被听见。每个被投入熔炉的新灵魂都会给战车提供燃料，让它们运行更长的时间。

恶魔秽液提速 Demon 
Ichor Boost

往战车的熔炉倒入一瓶恶魔秽液可以为战车提供1分钟额30尺移速加成。在此状态下，每当驾驶员的回合开始时，投一个 D20。如果结果为1，那么战车会遭受厄运表（P220）上，熔炉破裂的情况。
