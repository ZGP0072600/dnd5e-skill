# 地狱战争机器

附录 B：地狱战争机器 Infernal War 
Machines

地狱战争机器是在九狱打造而成的，以被诅咒的灵魂作为燃料的车辆。小号的战车通常交给掠夺小队或是斥候使用。而那些更大，更危险的战争机器则是能击溃整群的恶魔，或者向地方阵线的后方投送突击部队。

这些战车用地狱铁制成，装备有尖刺，刀剑，锁链以及攻城武器。地狱战争机器的引擎会发出低沉的轰鸣声，通常还伴随着痛苦的尖叫声——来自于它熔炉中燃烧着的灵魂。

此附录收录了各类地狱战争机器的数据，包括它们的驾驶和维修规则，以及阿弗纳斯中追逐战的处理方法。
