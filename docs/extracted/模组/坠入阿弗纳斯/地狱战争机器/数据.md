# 数据

数据 Stat 
Blocks

附录中收录了四中地狱战争机器的数据。地狱战争机器的数据处理方式和生物的处理方式相同，但要加入以下额外规则。

乘员数量和载货量 Creature 
and Cargo Capacity

乘员数量决定了地狱战争机器能容纳多少生物。可以通过挤着坐，或是抓在战车的外边来容下更多生物。

载货量决定了地狱战争机器能装下多少货物。

防御等级 Armor 
Class

地狱战争机器使用地狱铁制作而成的，它的防御等级=19+敏捷调整值。当战车处于静止状态时，对它发动的攻击有优势。

属性值 Ability 
Scores

地狱战争机器有六项属性值以及对应的调整值。它的力量值由其大小和重量决定。敏捷反映了战车的操纵性能。战车的体质值反映了它的耐用性以及结构的牢固程度。地狱战争机器的智力，感知和魅力通常为0。

生命值 Hit 
Points

地狱战争机器的生命值可以通过维修来恢复（见“维修”节）。当一辆战车的生命值降为0时，它将停止运作并且无法修复，所有被困于其熔炉中的灵魂都会被释放（见“灵魂燃料”节）。

伤害阀值 Damage 
Threshold

地狱战争机器的体积和护甲让它们能无视一些轻微的伤害。拥有伤害阀值的战车免疫所有小于其伤害阀值的伤害；当其受到的伤害值大于或等于其伤害阀值时，它会照常受到伤害。所有小于其伤害阀值的伤害不会降低战车的生命值。

厄运阀值 Mishap 
Threshold

当地狱战争机器一次性受到大于或等于其厄运阀值的伤害时，它必须根据厄运表投掷一次厄运。

操作台和操作人员 Actions 
Stations and Crew

地狱战争机器本身不能执行动作。它必须依靠占据操作台的人员用他们的动作来操作战车的各种功能。所有在地狱战争机器数据中，操作台一栏出现的功能都需要一个动作来使用。
