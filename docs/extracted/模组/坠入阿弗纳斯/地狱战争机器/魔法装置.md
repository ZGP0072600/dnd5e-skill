# 魔法装置

魔法装置 Magical Gadgets

魔鬼们喜欢给地狱战争机器加上点邪魔风格的装置来给敌人造成困扰。这些装置通常从驾驶室启动

暗蚀烟幕 Necrotic Smoke Screen

战车的驾驶员可以用一个附赠动作启动此魔法装置，让战车从车头或车尾释放出覆盖30尺立方区域的不透明烟雾。烟雾是静止的，存在时间为1分钟。烟雾内的区域处于重度遮蔽。所有生物在第一次进入烟雾或是处于烟雾的第一个回合时，会受到21（6D6）点暗蚀伤害。一阵强风可以驱散烟雾。此装置须经过24小时充能才能再次使用。

传送装置 Teleporter

战车驾驶员可以用一个附赠动作启动此魔法装置，让战车传送到驾驶员可看到的一处未被占据的空间，最远能达到300尺。所有和战车有解除的生物和物件都会被一起传送过去。此装置须经过24小时充能才能再次使用。

阿弗纳斯的追逐战 Chases on Avernus

使用城主手册第八章的规则以及此处的阿弗纳斯追逐战事件表格，你可以构建拥有阿弗纳斯特色危险的追逐战。以地狱战争机器的体质调整来决定它在一次追逐战中能使用几次疾走。

当事件迫使战车进行一次属性检定或是驾驶员使用战车的属性来进行属性检定时，若骰出的结果比要求数值低5或更多，则触发一次厄运（见“厄运”）。
