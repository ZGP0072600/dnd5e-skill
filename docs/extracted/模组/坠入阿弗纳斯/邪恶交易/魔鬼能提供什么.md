# 魔鬼能提供什么

魔鬼能提供什么 What Devils 
Offer

魔鬼所能提供的极限取决于它在地狱等级制度中的地位。地狱等级表按权力的升序列出了魔鬼，从低级的劣魔到强大的大公和女大公。（为了完整起见，本表中包含了追猎魔，尽管追猎魔不会出现在本冒险或怪物图鉴中。你可以在《魔邓肯的众敌卷册》一书中找到更多关于它们的信息。）

魔鬼喜欢用宝藏和魔法物品诱惑凡人。魔鬼赐予的魔法物品通常都有着魔鬼的特征。它们可能是用地狱铁制成的，带有炼狱语的文字，发出微弱的痛苦的哀号，硫磺的味道，或者有一些其他细思恐极的线索（看看它有什么小习性？参考《城主指南》第7章中的表格）。

魔鬼也可能同意为凡人服务一段时间，充当顾问和保镖。

低阶魔鬼 Lesser Devils

低阶魔鬼可以提供以下任意一项或全部：

l 价值500gp 的艺术品、硬币或宝石

l 一件非普通的魔法物品

l 不超过九年的忠诚服务

高阶魔鬼 Greater Devils

高阶魔鬼可以提供以下任意一项或全部：

l 价值5000gp 的艺术品、硬币或宝石

l 一件非普通或珍稀的魔法物品

l 不超过九天的忠诚服务

大魔鬼 Archdevils

大魔鬼可以提供以下任意一项或全部：

l 价值50000gp 的艺术品、硬币、宝石或不动产

l 一件非普通、珍稀或极珍稀的魔法物品

l 一个无法通过其他方式获得的有价值的信息

l 完成一项大魔鬼或它的一个仆从可以在接下来的九天内可以完成的任务

l 一个形式为护咒 Charm 的超自然赠礼（见《城主指南》第七章中的“超自然赠礼”和第215页中的“大魔鬼的护咒”）

l 一个祈愿术 wish 能得到的好处（无需承担除了模仿其他法术外的任何目的所需承担的负担）
