# 圣飞象

圣飞象 HOLLYPHANT

圣飞象是一种来自上层位面的温和而强壮的生物。善良阵营的神和天使用他们作为信使和助手。圣飞象珍视友谊和真心实意。

一只圣飞象看起来像一只小型的大象，有着闪烁的金毛，翅膀迅速地拍打着，不仅能让它飞向高空，还能让它飞得极快。虽然它宽容体贴，但如果一个圣飞象目睹了一项恶行必将严惩做恶之人。它珍珠般的象牙并不可怕，但它能用象鼻发出号角声，使生物耳聋，或将作恶者吞噬在正能量的闪烁之中。圣飞象也有强大的天生施法，用以痛击邪恶并保护它的朋友。

坚不可摧光环。圣飞象周围围绕着一个10英尺半径球形的隐形光环并只要圣飞象活着就一直存在。任何来自光环外的本身环阶5环或更低的法术都不能影响光环内的生物或物体，即使该法术是升环施法。这样的法术可以以光环内的生物和物体为目标，但对它们没有效果。同样，光环内的区域也不会受到光环外的法术影响。圣飞象可以使用一个动作来抑制这个特性，直到它的专注结束（就像专注于一个法术一样）。
