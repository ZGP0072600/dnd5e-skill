# 克鲁科克托伊克

克鲁科克托伊克 CROKEK'TOECK

克鲁科克托伊克是耶诺古的恶魔宠物。宛如一只有着鬣狗的牙齿、皮毛、耳朵和笑声的巨大弹涂鱼一般，这只特别的恶魔能在它的食道里装下更小的恶魔，并在耶诺古想要的地方将它们吐出来。由于免疫于冥河粉碎心智的效应影响，克鲁科克托伊克常常用于运送小型恶魔（以及有时耶诺古自己）穿过冥河可怕的水域。
