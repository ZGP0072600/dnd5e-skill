# 海盗大副 Pirate First Mate

这些毫不废话的水手是海盗船长值得信赖的盟友。前士兵和训练有素的雇佣兵常常担任海盗船上的大副。在《盐沼的不祥秘闻》中，血腥比约恩是海鬼号上的大副。

多重攻击 Multiattack。大副可以进行两次长剑攻击。
