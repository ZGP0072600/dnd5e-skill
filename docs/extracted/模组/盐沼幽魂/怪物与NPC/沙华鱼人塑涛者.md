# 鲨华塑涛者 Sahuagin Wave Shaper

为了换取瑟寇拉变异魔法，这些鲨华牺牲了自己的躯体，变得佝偻驼背。塑涛者们为鲨华鱼人的军团添加了元素的魔法（如同在最后的宿敌 The 
Final Enemy 中所展现的那样），并喜欢制造破坏性的涡流。

血腥狂怒 Blood Frenzy。塑涛者对生命值没有全满的生物发动近战攻击时，其攻击检定具有优势。

有限两栖 Limited Amphibiousness。塑涛者可以在空气中与水里呼吸，但至少每4小时要回到水里，以避免窒息。

鲨心感应 Shark Telepathy。塑涛者可以用魔法进行有限的心灵感应，以命令其周边120 尺内的鲨鱼。

天生施法 Innate Spellcasting。塑涛者天生施法属性为智力（法术豁免 DC 14，法术攻击命中+6）。塑涛者天生可施展如下法术，且无需任何相应的材料成分：

随意：传讯术 Message

1次/日：通晓语言 comprehend languages

动作 Action

多重攻击 Multiattack。塑涛者进行两次攻击，一次啃咬和一次爪击。

啃咬 Bite。

近战武器攻击：命中+4，触及5尺，单一目标。

伤害：10（2d8+1）穿刺伤害，外加13（3d8）寒冷伤害。

爪击 Claws.

近战武器攻击：命中+4，触及5尺，单一目标。

伤害：10（2d8+1）挥砍伤害，外加13（3d8）寒冷伤害。

涡流 Whirlpool (1次/日).

塑涛者以一个至少50平方尺和25尺深的水体为目标，并在该区域的中心形成一个漩涡。漩涡的底部为一个5尺宽的漩涡，在顶部可达50尺宽，25尺高，该效应持续1分钟或直到塑涛者失能为止。任何在水中的生物或物体，在距离漩涡的25尺内，都会被向内拉近10尺。一名生物可以进行一次 DC 14力量（运动）的检定，成功则游离漩涡。

当一个生物第一次进入涡旋或在涡旋内部开始其回合时，它必须进行一次 DC 
14的力量豁免。在一次失败的豁免中，该生物受到9 
（2d8）钝击伤害，并被困在漩涡之中，直到其回合结束。若豁免成功，则该生物受到一半的伤害，并且不会被卷入漩涡中。被漩涡困住的生物可以用它的动作试图游出漩涡，但是它在力量（运动）的检定上具有劣势。

当一个物体第一次进入漩涡时，它会受到9 (2d8)的钝击伤害。该物体在漩涡内的每一轮都会受到该种伤害。
