# 多利先生

多利先生 MR.Dory 作为四位管理斯戴亚斯的议员之一，多利先生将他被诅咒的本质藏在朴素的外表之下。与创造出鱼魔怪的方式相同，他罕见的，液体过敏的“皮肤症状”实际上也来自于底栖魔鱼的折磨，虽然多利先生的症状并不那么严重，而且他设法保留了自己的自由意志。

多利先生凝视一个他能看到的30英尺内的生物。目标必须进行 DC 15体质豁免。若豁免失败将受到27 （5d10）点暗蚀伤害和27 （5d10）点毒素伤害，并在1分钟内同时获得暗蚀伤害和毒素伤害的易伤。若豁免成功则受上述一半的伤害，并不会获得易伤。
