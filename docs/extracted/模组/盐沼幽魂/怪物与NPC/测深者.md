# 测深者 Fathomer

测深者是与邪恶的水元素公主奥莉德拉达成协议的信徒。除了他们的施法能力，测深者还能把他们的身体变成水体的蛇一样的形态。在水蛇形态中，测深者可以获得许多元素生物所拥有的抗性，以及用它们的液态身体抓住和粉碎敌人的能力。测深者 Fathomer 中型类人（人类），中立邪恶
