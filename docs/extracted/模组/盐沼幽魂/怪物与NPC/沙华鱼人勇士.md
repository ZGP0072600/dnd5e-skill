# 鲨华勇士 Sahuagin Champion

那些以英勇行为证明自己的鲨华战士们被授予头衔和地位。鲨华勇士们经常在鲨华鱼人的部队中担任副官，就像他们在最后的宿敌 The 
Final Enemy 中所展示的那样。

鲨华勇士 Sahuagin Champion

中型类人（鲨华鱼人），守序邪恶

血腥狂怒 Blood Frenzy。勇士对生命值没有全满的生物发动近战攻击时，其攻击检定具有优势。

有限两栖 Limited 
Amphibiousness。勇士可以在空气中与水里呼吸，但至少每4小时要回到水里，以避免窒息。

鲨心感应 Shark Telepathy。勇士可以用魔法进行有限的心灵感应，以命令其周边120 尺内的鲨鱼。
