# 杀手藤 Assassin Vine

杀手藤是一种可以移动的植物，它通过抓住并压碎猎物将猎物的尸体放到自己根部附近来作为肥料。除非需要捕捉猎物不然它通常都保持一动不动。一颗成熟的杀手藤由一根长约20英尺的主藤蔓组成。从主藤上每隔6英尺生长着5英尺长的小藤蔓。在夏末，次藤上会结出一串串类似野生葡萄的小果实。这种果实很坚硬，有一种爽口而且苦涩的味道。

杀手藤有一种地下变种生长在温泉，火山口或者其他热源附近。一颗生长在地下的杀手藤通常能产生足够的食物残渣来支持蘑菇和其他真菌的繁殖，这些真菌会在杀手藤周围生长并帮助隐藏它。
