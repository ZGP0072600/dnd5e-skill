# 鲨华鱼人幼体集群 Sahuagin Hatchling Swarm

这些鲨华鱼人幼体集群在最后的宿敌 The 
Final Enemy 的水中翻滚，对它们遇到的任何生物而言都是危险的。其他鲨华鱼人躲避该集群，而集群个体成员则互相吞食，直到只有最强壮的幼苗存活下来，长大成熟。

血腥狂怒 Blood Frenzy。鲨华鱼人对生命值没有全满的生物发动近战攻击时，其攻击检定具有优势。

沸腾 Seething。一旦它进入战斗，如果它于该回合没有发动攻击，则在回合结束时对自己造成10点挥砍伤害。此伤害忽略抗性，并且不能将集群的生命值降至0。

集群 Swarm。集群可以进驻另一生物身处的空间位置，反之亦然。而且集群可以通过任何足够一只微型生物通过的通道。集群不能恢复生命值也不能获得临时生命值。

水下呼吸 Water Breathing。该集群只能在水下呼吸。

动作 Action

————————————————

啃咬 Bite。

近战武器攻击：命中+6，触及0尺，单一身处集群内的生物。

伤害：14（4d6）穿刺伤害，或者集群生命值等于或低于一半时7（2d6）的穿刺伤害。
