# 海盗船长 Pirate Captain

海盗船长的一生都在海上摸爬滚打，靠鲜血和勇气赢得并维持指挥权。船长令人印象深刻的剑术和尖刻的言语会激起船员们的信心与恐惧。在《盐沼的不祥秘闻》中一伙走私犯的首领，西格尔德“蛇眼”就是一位海盗船长。
