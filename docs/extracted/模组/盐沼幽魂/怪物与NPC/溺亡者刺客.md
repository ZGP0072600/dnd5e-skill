# 溺亡者刺客 Drowned Assassin

溺亡者刺客在可怕的沉默中进行跟踪并实施杀戮。在塔默劳特的命运 Tammeraut's 
Fate 的不死族的攻击中，它从一个阴影潜行到另一个阴影，并以布满盾壳的手弩进行精确而致命的瞄准。这个生物的脸隐藏在一个朴素的浮木面具之后，一旦暴露，会引发人类心中的恐惧。

水底行者 Bottom 
Treader。溺亡者刺客不会游泳，它会沉到任何水体的底部。它在水底的移动和攻击不受惩罚，且免疫在水深100英尺底下的影响。

共有链接 Bound 
Together。溺亡者刺客与他周围1英里的其他溺亡者共享思维，并且可以不受限制地将他的思想和所观察到的事物立即传递给他们。

不死坚韧 Undead Fortitude。当溺亡者因受非光耀或重击伤害而使其生命值减至0时，它必须进行一次 DC 为5+所受伤害值的体质豁免，豁免成功则其生命值降至1。
