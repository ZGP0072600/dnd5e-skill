# 蜥蜴人副头领 Lizardfolk Subchielf

蜥蜴人副头领（见《灰水河危机》）是塞缪安亚的虔诚祭司，用与牧师类似的方式崇拜着其神灵。它挥舞着由巨大鳄鱼牙齿制成、塞缪安亚祝福的匕首，其代表着副头领在战斗与虔诚两方面的造诣。
