# 溺亡者之刃 Drowned Blade

在塔默劳特的命运 Tammeraut's 
Fate 中，这名身披铠甲的亡灵战士袭击了隐修院，并以惊人的狡猾与之战斗。它那稀疏的胡须上紧紧攀附着海星，它那肿胀的灰色眼睛里透出它那邪恶的愤怒。

水底行者 Bottom 
Treader。溺亡者之刃不会游泳，它会沉到任何水体的底部。它在水底的移动和攻击不受惩罚，且免疫在水深100英尺底下的影响。

共有链接 Bound 
Together。溺亡者之刃与他周围1英里的其他溺亡者之刃共享思维，并且可以不受限制地将他的思想和所观察到的事物立即传递给他们。

不死坚韧 Undead Fortitude。当溺亡者因受非光耀或重击伤害而使其生命值减至0时，它必须进行一次 DC 为5+所受伤害值的体质豁免，豁免成功则其生命值降至1。

多重攻击 Multiattack。溺亡者之刃发动两次生锈的长剑攻击。

Longsword 近战武器攻击：+5命中，触及5尺，一个目标。伤害：7（1d8+3） 
挥砍伤害。目标须成功于一次 DC 12的体质豁免，否则陷入蓝腐病（见边栏“Bluerot”）。
