# 吸血玉雕 Vampiric Jade Statue

一尊大型的、雕刻精美的吸血玉雕被黑暗魔法赋予了生命，守卫着圣堂岛上的隧道。它的石质獠牙吸取血液，并以此诅咒受害者。

传奇抗性 Legendary Resistance（3/日）。雕像豁免失败时，它能将其改为豁免成功。

动作 Action————————————————

多重攻击 Multiattack。塑像发动三次攻击：一次使用其啃咬，两次使用其爪击。

啃咬 Bite 近战武器攻击：命中+6，触及5尺，单一目标。命中：10（2d6+3） 
穿刺伤害。如果目标是一个生物，该生物被雕像所诅咒。诅咒持续十分钟。当生物被诅咒时，雕像对其进行的攻击具有优势。

爪击 Claws.近战武器攻击：命中+6，触及5尺，单一目标。命中：10（2d6+3） 挥砍伤害。

传奇动作 LEGENDARY 
ACTIONS——————————

雕像拥有3 传奇动作，用于选择执行下列动作选项。每次执行传奇动作时，每个选项只可以选用一次，且只能在另一生物的回合结束时执行。雕像在其回合开始时恢复其所有已消耗的传奇动作数。

啃咬 Bite.雕像进行一次啃咬攻击。

血腥收割 Blood Reaper.

所有目前处于雕像20尺内并被雕像诅咒的生物受到5点暗蚀伤害。

移动 Move.雕像移动至多等于其自身速度的距离，且不会因此引发借机攻击。
