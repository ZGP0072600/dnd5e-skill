# 裂潮祭司 Rip Tide Priest

裂潮祭司们聚集在其秘密结社里，进行着赞颂水元素之邪恶亲王奥莉德拉 Olhydra 的黑暗仪式。他们相信通过用水干掉敌人（譬如进行把人淹死的祭祀）可以得到奥莉德拉的青睐。
