# 海盗甲板法师 Pirate Deck Wizard

这些散发着盐味的魔法实践者兼具博学和迷信。海盗船的船员们通常会在他们的甲板法师从一处跳到另一处，并释放他们海盐风味的法术对付袭击者时远远地避开。《盐沼的不祥秘闻》中的海鬼号是一位甲板法师庞克塔的家。
