# 桑巴莱特 Sanbalet

在 The Sinister Secret of Saltmarsh 中，Sanbalet 是一个半陆上走私集团的领导者。他是一个醉心于幻象和精神控制的狡猾的自恋狂。他看护着藏匿在鬼屋下的洞穴里的违禁品。

桑巴莱特 Sanbalet

护甲等级11

速度30尺

施法 Spellcasting。桑巴莱特是一个3级施法者。其施法能力基于智力（法术豁免 DC 13，法术攻击命中+5）。他准备了以下法师法术：

戏法（任意）：舞光术 dancing light，次级幻影 minor illusion，冷冻射线 ray of frost

1环（4法术位）：魅惑类人 charm person，七彩喷射 color spray，魔法飞弹 magic missile，无声幻影 silent image

2环（2法术位）：魔嘴术 magic mouth，灼热射线 scorching ray

动作 Actions

匕首 dagger。近战或远程武器攻击：命中+3,触及5尺或射程20/60尺，单一目标。伤害：3（1d4+1）穿刺伤害。
