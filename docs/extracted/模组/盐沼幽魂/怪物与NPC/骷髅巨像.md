# 骸骨巨像 Skeletal Juggernaut

这个由骷髅组成，过于巨大的两足骸骨巨像，被目睹到蹒跚地穿过圣堂岛的沙滩，每走一步就会失去一部分骨头。在分解成单独的亡灵之前，这些骸骨守卫只能维持很短的时间。

## 动作 Actions.

**多重攻击 Multiattack。** 巨像发动两次爪击。

**爪击 Claws。**
*近战武器攻击*：+6命中，触及10尺，单一目标。*命中*：12（2d8+3）挥砍伤害。

**骸崩 Avalanche of Bones（充能5~6）。**

 
巨像坍塌成一大堆骸骨然后快速重塑。每一个在10尺内的生物都必须进行一个 DC 14的敏捷豁免检定，失败则受到18（4d8）钝击伤害，成功则只受到一半伤害。在豁免检定中失败的生物同时也应击倒地。
