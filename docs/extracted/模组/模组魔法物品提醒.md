# 关于模组相关资源

已出版的纸质模组请看这里

线上模组请看这里

目前没有收录模组具体内容的打算，仅收录一些玩家或 dm 可能用得上的速查，如有帮助收录的兴趣请联系。

**关于模组魔法物品：**

这些魔法物品都是出自于各个模组的，极大可能与你目前所在团的世设冲突，所在在使用前请向你的 DM 确定这些魔法物品是否可以出现在当前的世设中。 

很多模组翻译未完工，所以资源不全，请直接去找翻译组成员催（迫）稿（害）。

**关于模组怪物：**

此处只收录各个模组相对于已出资源而言“多出来”或是“不一样”的怪物，不会重复收录已在 MM 等书籍中出现的怪物。如有需要，请自行寻找或使用搜索功能。
