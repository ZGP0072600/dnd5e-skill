# 考古学家

一位考古学家会学习失落已久的衰败文化，通过这些文化的遗留之物——他们的骨头、废墟、幸存下来的文献以及坟墓。这种人不断精进考古学知识，周游世界边缘的角落，在残破城市与失落地下城中寻根溯源而且不断挖掘，同时搜寻那些可能会告诉他们关于君主与高阶祭司、战争与大灾难故事的神器。技能熟练项：历史，生存工具熟练项：制图工具或领航工具语言：任选一门语言装备：一个木匣，里面装有一座废墟或地下城的地图，一盏牛眼提灯，一把矿工镐，一套旅行服装，一把铲子，一座双人帐篷，一件在挖掘点发现的饰品，和一个装有25gp 的腰包。尘土挖掘者

在成为一个冒险者之前，你消耗了你的大部分年轻时光在尘土中摸爬滚打，从教会地下和废墟中偷窃那些疑似有价值的文物。尽管你已经出售掉一小部分你所发现之物，并且赚够了足够的钱来买更好的冒险装备，但你仍然保留着一件对你来说极具感情价值的物品。根据“重要物品”表格来掷骰决定你拥有什么，或是从中自选一件。重要物品 D8 物品1 10尺木杆2 撬棍3 帽子4 附盖提灯5 圆形坠饰6 铲子7 大锤8 鞭特性：历史学识

当你进入一座废墟或地下城时，你能够正确地识别出建造它的目的，并确定它的建造者是矮人、精灵、人类、蛇人还是其他已知种族。另外，你能够确定那些历史久于一个世纪的艺术品的价值。

建议特征

很少有考古学家能抵抗住未探索的废墟或地下城的诱惑，对于那些属于传说的起源地或者据称藏有法师、领主、王室的宝藏和遗物的地方来说尤其如此。有些考古学家掠夺财富或名望，还有一些想要实现他们的愿望，例如让过去的事物重现天日或是阻止世上最珍贵的宝物落入错误的人手中。无论考古学家的动机是什么，他们都结合了历史学家的博学性和寻宝恶棍的独立性。D8 特点1 好的谜团或难题令我心动。2 我有收藏癖，从不丢掉任何东西。3 对我来说，名望比钱财更重要。4 从死人身上偷东西时，我的内心毫无波动。5 与待在文明中心相比，我更喜欢待在积满尘土的旧坟墓里。6 陷阱不会令我紧张，那些触发陷阱的傻子才令我紧张。7 我也许会失败，但绝不会放弃。8 你也许认为我是个学者，但我忠爱酣畅淋漓的战斗。我的双拳为此而生。D6 理想1 保存。神器都应该存放于博物馆(善良)2 贪婪。我不会毫无意义地浪费我的生命。我期待着某种报酬。(任意)3 死愿。没什么比从死神的手边惊险逃出更令人兴奋的了。(混乱)4 高尚。死者和他们的物品应当被加以尊敬。(守序)5 不朽。我的计划是发现永生之秘。我的所有探索都只是该计划的一部分。(任意)6 危险。每一个从坟墓里带出的大发现都伴随着大危险。这两者是总是如影随形。(任意)

D6 牵绊1 
从我孩提时代起，我就听说过失落之城的故事。我下定决心要找到它，了解它藏有的秘密，并在史书里留下我的位置。2 我想找到我的导师，他在之前的一次探险中消失了。3 我有一个好对手。我俩之中只有一个是最棒的，而我会证明那个人是我。4 我不会出售任何具有历史意义或已成孤品的艺术品或其他宝物。5 我暗恋着赞助我进行考古发掘的人。6 我希望为一座图书馆、博物馆或大学带来声望。D6 缺点1 我暗自害怕着一些常见的野生动物——而在我的工作场所，他们遍地都是。2 在尚未搜索房间是否有密门前，我不会离开任何房间。3 当我并未探索地下城或废墟时，我会焦躁不安。4 我没有时间能分给朋友或家庭。我的每一步都在考虑与准备着下一次探险。5 当需要选择往左走还是往右走时，我总是选择走左边。6 只有彻底黑暗时，我才能睡着。
