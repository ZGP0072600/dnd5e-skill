# 三花株Tri-flower Frond

一棵完全成熟的三花株高6到7英尺。它有着三朵明亮的喇叭状花朵，每朵都和人类脑袋一样大，并且颜色各异：大红色、鲜橙色及亮黄色。每朵花都能用不同的可怕方式伤害其他生物。

三花株 Tri-flower Frond

中型植物，无阵营

AC：10

HP：11（2d8+2）

速度：5尺

力量1（-5） 敏捷10（+0） 
 体质12（+1） 智力1（-5） 感知10（+0） 
 魅力1（-5）

状态免疫：目盲，耳聋，力竭，倒地

感官：盲视30尺，被动察觉10

语言：—

挑战等级：1/2（100 XP）

动作

多重攻击。三花株先使用它的橙花，再使用黄花，最后使用红花。

橙花。三花株选择5尺内它能看见的一个生物为目标。目标必须成功通过一次 DC 
11的体质豁免，豁免失败则中毒一小时。以此方式陷入中毒的生物同时陷入昏迷状态。在每分钟结束时，该生物可以重复该豁免，豁免成功则结束该效应。

黄花。三花株选择5尺内它能看见的一个生物为目标。目标必须成功通过一次 DC 
11的敏捷豁免，豁免失败则被消化液包裹，在其每回合开始时受到5点强酸伤害。在该生物身上每浇一品脱或一瓶水，强酸伤害便减少1点。

红花。近战武器攻击：命中+2，触及5尺，单一生物。命中：2（1d4）点穿刺伤害，目标被擒抱（挣脱 DC 
11）。直至擒抱终止前，目标在其每回合开始时受到5（2d4）点毒素伤害。红花只能同时擒抱一个目标。位于三花株的触及范围内的其他生物可以使用他的动作来结束目标的受擒状态。
