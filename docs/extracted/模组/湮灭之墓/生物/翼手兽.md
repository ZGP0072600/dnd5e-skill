# 翼手兽 Pterafolk

翼手兽(读作 TAIR-eh-folk)是一种高10英尺，长着翅膀的爬行类两足生物，原产于楚尔特半岛。这种恶毒的隐世者认为大部分其他生物都是它们的敌人。

在楚尔特文化中，翼手兽经常被称为“骇人一族”，因为它们习惯从天而降，抓住任何它们能抓住的东西，然后把宝物和食物带回它们的栖息地。

翼手兽的翼展长20英尺，能诞下幼鸟。古老的仪式曾赋予翼手兽以翼龙和无翅蜥蜴族的形态现身的能力，但这些仪式早已被遗忘。

挑战等级：1（200 XP）

恐怖俯冲。如果翼手兽飞行并向一名目标直线俯冲至少30英尺，并在同一回合内对目标发动近战武器攻击并命中，则目标被恐慌，直到其下回合结束。

动作

多重攻击：翼手兽发动三次攻击：一次啃咬和两次爪击。或者，它用标枪进行两次近战攻击。

啃咬：近战武器攻击：命中+4，触及5尺，单一目标。命中：7（2d4+2）点穿刺伤害。

爪击：近战武器攻击：命中+4，触及5尺，单一目标。命中：5（1d6+2）点挥砍伤害。

标枪：近战或远程武器攻击：命中+4，触及5尺或射程30/120尺，单一目标。命中：9（2d6+2）点穿刺伤害。
