# 暴龙僵尸

暴龙丧尸 TYRANNOSAURUS 
ZOMBIE

巨型亡灵，无阵营

AC：11（天生护甲）

HP：136（13d12+52）

速度：40尺

力量25（+7） 敏捷6（-2） 体质19（+4） 
 智力1（-5） 感知3（-4） 魅力5（-3）

伤害免疫：毒素

状态免疫：中毒

感官：黑暗视觉60尺，被动察觉8

语言：—

挑战等级：8（3900 XP）

口吐丧尸。作为一个附赠动作项，暴龙丧尸可以吐出一个普通丧尸，它出现在10尺内一处未占据空间。吐出的丧尸按照它自己的先攻顺序行动。丧尸被吐出后，骰一个 D6，骰出1时，暴龙丧尸耗尽所有丧尸并失去该特质。如果暴龙丧尸死时仍拥有该特质，那么在其下回合开始时，1d4只普通丧尸从它的尸体中爆发出来，这些丧尸按照它们自己的先攻顺序行动。

不死坚韧。当丧尸由于受到非光耀非重击伤害而使其生命值降低至0时，它必须进行一次 DC 为5+所受伤害值的体质豁免，豁免成功则生命值改为降低至1。

动作

多重攻击。丧尸发动两次攻击：一次用啃咬，一次尾击。两次攻击必须指定不同目标。

啃咬。近战武器攻击：命中+10，触及10尺，单一目标。命中：33（4d12+7）点穿刺伤害。如果目标为中型或更小的生物，则目标被擒抱（挣脱 DC 
17）。被擒抱期间，该目标陷入束缚状态，且暴龙丧尸无法啃咬其他目标或口吐丧尸。

尾击。近战武器攻击：命中+10，触及10尺，单一目标。命中：20（3d8+7）点钝击伤害。
