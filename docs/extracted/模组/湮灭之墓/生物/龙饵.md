# 龙饵 Dragonbait

龙饵是一名善良斗士，也是一名类蜴人——一个起源于遥远世界并且长寿的种族。很少有类蜴人在被遗忘的国度中居住，并且人们相信类蜴人族群并不存在于这个世界的任何一个角落。

类蜴人的语言混合着咔嚓声与口哨声，并高出人类的听觉范围，所以龙饵倾向于依靠气味来传达自己的情感。为了被大部分用语言沟通的生物所理解，他必须“呼出”一种气味。龙饵已知的气味包括：硫磺味（困惑）、玫瑰香（悲伤）、柠檬香（欢愉或满意）、烤面包味（愤怒）、紫罗兰香（危险或恐惧）、金银花香（温柔或关心）、焚香味（奉献或虔诚）、焦油香（胜利或庆祝）、火腿香（紧张或担心）。

龙饵身高4尺10寸，重150磅，皮肤干燥而褶皱。他手持一把神圣复仇者 holy avenger 长剑，并装备着一面绘有蓝红白三色涂漆的盾牌。尽管他与圣武士有着相同的特性，但龙饵并非他们中的一员。他使用一种名为圣感的能力，能探测出60尺内任何生物的阵营。

意外之友。龙饵在南扎路港遇到了阿图斯·辛贝尔，两人很快知交莫逆。当阿图斯诉说出他不断寻找、只为与妻子艾丽珊达重逢之事时，龙饵用安静的仪态与不同寻常的交流方法安抚了他。龙饵只是微笑着点了点头，就同意帮助阿图斯。作为楚尔特的常客，这名类蜴人十分了解丛林中的危险。

扮演龙饵。龙饵从未去过奥姆城，他并不知道这座城市在哪里。如果玩家们扮演的角色说服了阿图斯·辛贝尔帮助他们摧毁易魂装置，则龙饵也会一同协助。

龙饵的特征

理想：龙饵思考着探索世界，见证更多世界奇迹、并在路上帮助他人等事宜，他散发出柠檬香味。（翻译：他有一颗探险者般的心。）

牵绊：龙饵帮助一位朋友进行他的探索之旅，他散发出焚香的味道。（翻译：他会为了朋友赴汤蹈火。）

缺点：龙饵前往其他人害怕步入之所在，他“呼出“一股火腿般的味道。（翻译：他从不担心风险，只会一往无前。）

龙饵

中型类人（类蜴人），守序善良

AC：17 （胸甲，盾牌）

HP：120 (16d8+48)

速度：30尺

力量15（+2） 敏捷13（+1） 
 体质17（+3） 智力14（+2） 感知16（+3） 
 魅力18（+4）

豁免加值：感知豁免+6，魅力豁免+7

技能加值：运动+5 医药+6

感官：被动察觉13

语言：理解通用语但不会说

挑战等级：5 （1800 XP）

神佑：龙饵免疫疾病。

魔法抗性灵光：当龙饵握持着神圣复仇者 holy avenger 时，其周围散发出半径10尺的灵光。当灵光起效时，龙饵和灵光范围内其他友方生物为了抵抗法术及其他魔法效应而进行的豁免检定具优势。

动作

多重攻击：龙饵发动两次近战武器攻击。

神圣复仇者（+3长剑）。近战武器攻击：命中+8，触及5尺，单一目标。伤害：单手持用时造成9（1d8+5）点挥砍伤害；双手握持时造成10（1d10+5）点挥砍伤害。如果目标是邪魔或亡灵，则它额外受到11（2d10）点光耀伤害。

阵营感知。龙饵探测出60尺内一名他可见的生物的阵营，前提是该生物的阵营未被预言系魔法（如受到法术或其他魔法效果影响）所隐藏。
