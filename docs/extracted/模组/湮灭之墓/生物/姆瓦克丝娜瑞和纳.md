# 姆瓦克丝娜瑞和纳MWAXANARé AND NA

十七岁还是有些幼稚的年纪，但姆瓦克丝娜瑞公主（简称为娜尔）已经成为了奥姆王室尚存活的后裔中的最为年长者，同时也是他们已陷落王国的自然继承人。她和她六岁的弟弟纳是基尔·萨巴隐修地中的鸟羽人们的客人。鸟羽人们想要保护这对年轻的王室姐弟，直至奥姆城中的邪恶被消灭殆尽。

姆瓦克丝娜瑞为了自己的利益有些过于固执。她渴望取得她的王权，并会急切地利用冒险者来达成她的目的。她的声调显得有些高高在上，而对于那些顶撞她的举动，她的态度就不会那么友善了。

在与外层位面的古老存在——艾夸的气之公爵达成协议后，公主成为了一名新生的魔契师。她对于肉体劳作的憎恶让她在法师之手和隐形仆役的应用上颇有心得。通过她的契约，她也获得了心灵感应的能力。

年轻的王子纳是一个 AC 10、HP 
3的非战斗人员。他的大部分时光都在鸟羽人这里度过，因此他将自己也认为是他们中的一员。他的穿着如同鸟羽人一般，而且更倾向于说气族语而非通用语。

姆瓦克丝娜瑞的特征

理想。“有朝一日，我将君临楚尔特。”

牵绊。“我已与艾夸的气之公爵达成协议，将使用祂们的力量来保护我的弟弟和那些守护我们的鸟羽人们。”

缺点。“我是一名公主，我应当享有与之对应的尊重与地位。我不是听令者，而是号令者。”

纳的特征

理想。“我想成为一名鸟羽人，我想学习如何像鸟儿那样飞翔！”

牵绊。“我深爱我的姊姊，我深爱鸟羽人们。她们就是我的全部。”

缺点。“我对这世界一无所知。”
