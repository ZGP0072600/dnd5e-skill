# 魔王鹤 EBLIS

在人们众口相传的故事中，魔王鹤是因盗窃与绑架受到诸神降罪的奸邪之人所转世而生的物种。这些既聪明又邪恶的鹤类高达8英尺，与大多数鸟类一样是卵生的，同时它们使用足与喙来操作其他物件。

魔王鹤总是试图忽悠别的生物帮它们做事，它们的目的通常都是自私自利的。它们的天性就是摧毁其他来犯的生物，并以诱骗类人生物进入死亡陷阱为乐，有时它们也会这样“报答”帮他们做事的生物。它们用草丛和芦苇搭建巢穴，并把宝藏藏于其中。
