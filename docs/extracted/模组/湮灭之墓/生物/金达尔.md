# 金达尔 ZINDAR

这位半金龙管理着南扎路港的码头并记录船舶舱单。作为一个能力出众的术士，金达尔的工作获得了贸易亲王们丰厚的报酬。他也是叶特凯社的一名关键成员，并是一个重要的信息来源，知道许多与这座城市有关的信息。金达尔对冒险者们“情有独钟”，但他深知楚尔特的危险，因此他明白，大多数踏上丛林探险之旅的人永远不会回来。

金达尔在他日常工作中广泛使用法术，施放传讯术 Message 向码头工人传递信息，侦测思想 Detect Thoughts 为了读取船长的想法，敲击术 Knock 为了开启货柜进行检查，鹰眼术 Clairvoyance 为了窥视船舱，支配野兽 Dominate Beast 安抚紧张的动物，诸如此类。

金达尔角色特征

理想。“我为我的工作感到自豪，我喜欢确保我参与的所有商业交易都童叟无欺。”

牵绊。“南扎路港是我的家，我对那些会给这座城市及其居民带来危害的人无甚好感。”

缺点。“与其生气，不如报复。”

龙翼 。作为一个附赠动作，金达尔可以在其回合内从背后长出一对龙翼。他获得30尺飞行速度，直到他用一个附赠动作解除它们。

动作

长棍。近战武器攻击：命中+5，触及5尺，单一目标。命中：5（1d6+2）点钝击伤害；双手持用时造成6（1d8+2）点钝击伤害。

吐息武器（充能5-6）。金达尔使用以下选项之一：

火焰吐息 。金达尔呼出火焰并覆盖15尺锥状区域。该区域内的每个生物都必须进行一次 DC 
15的敏捷豁免，豁免失败则受到22（4d10）点火焰伤害，豁免成功则伤害减半。

弱化吐息。金达尔呼出毒气并覆盖15尺锥状区域。该区域内的每个生物都必须进行一次 DC 
15的力量豁免，豁免失败则其在进行所有基于力量的攻击检定、基于力量的属性检定和力量豁免时具劣势，持续1分钟。该生物可以在其每回合结束时重复该豁免，豁免成功则终止其身上的该效应。
