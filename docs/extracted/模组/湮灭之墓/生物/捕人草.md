# 捕人草 MANTRAP

捕人草是一种巨大的食肉植物，其外形如同被放大的捕蝇草。为了捕捉猎物，捕人草会释放出一种花粉，利用其魔法效应引诱猎物。尽管它会吞噬触及范围内的所有温血生物，但它还是偏好类人生物。

捕人草 MANTRAP

大型植物，无阵营

AC：12

HP：45（7d10+7）

速度：5尺

力量15（+2）
 敏捷14（+2）
 体质12（+1） 
 智力1（-5）
 感知10（+0） 魅力2（-4）

状态免疫：目盲，耳聋，力竭，倒地

感官：震颤感知30尺，被动察觉10

语言：——

挑战等级：1（200 XP）

诱人花粉（1次/日）。当捕人草发现附近有任何生物时，它可以用一个反应释放出一片花粉，覆盖半径30尺区域。区域内的所有野兽和类人生物都必须成功进行一次 DC 
11的感知豁免，豁免失败则其在自己回合内将使用所有移动力尽可能地靠近捕人草。受影响目标可以在其每回合结束时重复该豁免，豁免成功则结束效果。

拟形。当捕人草静止不动时，它与普通的热带植物无异。

动作

吞没 Engulf。近战武器攻击：命中+4，触及5尺，单一中型或更小的生物。命中：目标被困在捕人草的叶颌之中。因此被困的目标陷入目盲和束缚状态，其对来自捕人草体外的所有攻击和其他效果视为具有完全掩护。目标在其每回合开始受到14（4d6）点强酸伤害。如果捕人草死亡，则其内部生物不再被束缚。捕人草一次只能吞没一个生物。
