# 拉兹·纳斯 Ras Nsi

“拉兹”是类似于“公爵”的一个贵族头衔。拉兹·纳斯曾是一名楚尔特人圣武士、乌塔欧的选民、麦兹罗城的宣誓守护者。他背叛了自己的誓言，被放逐出了城市。拉兹·纳斯变成了一位复仇心切的军阀，带着一支亡灵军队试图征服麦兹罗城，但却被击败了。比起寻求救赎，他更想寻求复仇，于是被剥夺了神赐力量。那群亡灵大军脱离了他的控制，分散在楚尔特大陆的森林中。

撤退到奥姆城后，纳斯仍紧抓着自己的贵族头衔不放。他与住在奥姆城内的蛇人结盟，通过一个可怕仪式，变成了一只强大的恶咒蛇人——保有自己的人类头颅和双手，同时获得了一副蛇形下半身。在乌塔欧离开这个世界后，拉兹·纳斯和他新的蛇人追随者们开始寻找将暗夜巨蛇丹达尔带到现世的方法。阿瑟瑞克承诺帮助他们，但要求拉兹·纳斯答应保护九神之墓。事实上，阿瑟瑞克并未打算遵守这一交易，而拉兹·纳斯也同样开始怀疑了。

异国收藏。拉兹·纳斯拥有大量来自遥远地区的服饰与装备，其中很多都是从已死的探险者身上获取的。这些收藏包括一件精良的科米尔斗篷、一个桑比亚酒壶、一件珠宝装饰的安姆紧身上衣以及一个与上衣配套的腰包。他还携带着一柄焰舌 长剑。

缓慢死亡。拉兹·纳斯已经历过多次起死回生，因此，他正承受易魂装置所引发的死亡诅咒。他的生命值上限被降低了，麻风病似的伤痕布满了他的肌肤。讽刺的是，拉兹·纳斯并未意识到阿瑟瑞克就是诅咒的成因，他和蛇人都不知道九神之墓里到底有什么，只是通过墓名得知里面埋葬着死去的神。如果角色们让拉兹·纳斯明白他们的努力是在帮助他，那么他或许会被说服，不再阻拦角色们。

拉兹·纳斯角色特征

理想。“我想毁灭这个世界，然后统治下一个。”

牵绊。“楚尔特的蛇人由我统治，他们会帮助我建立一个帝国。”

缺点。“为了拯救我自己，我可以做任何事，背叛任何人。”
