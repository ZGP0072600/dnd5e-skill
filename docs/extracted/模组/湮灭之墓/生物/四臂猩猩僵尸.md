# 四臂猩猩僵尸

四臂猩猩丧尸 GIRALLON ZOMBIE

大型亡灵，混乱邪恶

AC：11（天生护甲）

HP：59（7d10+21）

速度：30尺，攀爬30尺

力量18（+4） 敏捷12（+1） 体质16（+3） 
 智力3（-4） 感知7（-2） 魅力5（-3）

伤害免疫：毒素

状态免疫：中毒

感官：黑暗视觉60尺，被动察觉8

语言：——

挑战等级：3（700 XP）

好斗。作为一个附赠动作项，丧尸可以向一个它能看见的敌对生物移动至多相当于其速度的距离。

不死坚韧。当丧尸由于受到非光耀非重击伤害而使其生命值降低至0时，它必须进行一次 DC 为5+所受伤害值的体质豁免，豁免成功则生命值改为降低至1。

动作

多重攻击。丧尸发动五次攻击：一次啃咬，四次爪击。

啃咬。近战武器攻击：命中+6，触及5尺，单一目标。命中：7（1d6+4）的穿刺伤害。

爪击。近战武器攻击：命中+6，触及5尺，单一目标。命中：6（1d4+4）的挥砍伤害。
