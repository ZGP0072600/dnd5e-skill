# 荆棘兽 Thorny

荆棘兽是一种四脚菌类生物，居住在孢子人之中，作为它们的看门狗或者坐骑。

## 特性 Traits

**植物伪装。** 在任何被足够多的植物生命遮蔽的地形上，荆棘兽进行的敏捷(隐匿)检定具优势。

**再生。** 荆棘兽在自己回合开始时恢复5生命值。如果它受到寒冷、火焰或暗蚀伤害，则该特质在荆棘兽下回合开始时不会发挥作用。只有当它以0生命值开始其回合并且不能再生时，荆棘兽才会死亡。

**荆棘之躯。** 在它的回合开始时，荆棘兽对任何正擒抱着它的生物造成2（1d4）穿刺伤害。

## 动作 Actions

**啃咬。** 近战武器攻击，命中+3，触及5尺，单一目标。伤害：8（2d6+1）挥砍伤害。
