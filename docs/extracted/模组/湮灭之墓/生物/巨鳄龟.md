# 巨鳄龟 GIANT SNAPPING TURTLE

巨鳄龟的直径可达12尺。尽管看上去缓慢迟钝，但它们实际上可以做到夸张的加速，且对接近它们的体型更小的生物发起侵略性的攻击。巨龟的一口可以咬断一个人类的躯干，而它们对食物也不怎么挑剔。

巨鳄龟

大型野兽，无阵营

AC：17（天生护甲），倒地时为12

HP：75（10d10+20）

速度：30尺，游泳40尺

力量19（+4） 敏捷10（+0） 
 体质14（+2） 智力2（-4） 感知12（+1） 
 魅力5（-3）

感官：黑暗视觉60尺，被动察觉11

语言：—

挑战等级：3（700 XP）

水陆两栖。巨鳄龟可以在空气和水中呼吸。

稳定姿态。当某种效应使巨鳄龟陷入倒地时，它可以进行一个 DC 
10的体质豁免，豁免成功则避免倒地。倒地的巨龟处于四脚朝天状态。要想站起来，它必须在自己回合内进行一次 DC 
10的敏捷检定，检定成功则耗尽当前回合的所有移动力脱离倒地状态，检定失败则依然倒地。

动作

啃咬。近战武器攻击：命中+6，触及5尺，单一目标。命中：18（4d6+4）点挥砍伤害。
