# 幻色蛇 JACULI

幻色蛇通常是一条十五尺长的蛇，能够改变自身的颜色和纹理以伪装自身。当缠绕在石柱或树干上时，它会将自身的外表变得如同石头或木纹般，并以极强的力量和极高的精准度将自己像标枪一般射出去，向它的猎物发起袭击。

幻色蛇通常会在气候较潮湿的地方出现，且它们偏好雨林和阴冷的地下城。幻色蛇所蜕之皮是制造跳跑之靴和隐身斗篷的上佳材料。

幻色蛇

大型野兽，无阵营

AC：14（天生护甲）

HP：16（3d10）

速度：30尺，攀爬 20尺

力量15（+2） 敏捷14（+2） 
 体质11（+0）
 智力2（-4） 感知8（-1） 
 魅力3（-4）

技能加值：运动+4，察觉+1，隐匿+4

感官：盲视30尺，被动察觉11

语言：——

挑战等级：1/2（100XP）

伪装。幻色蛇为了躲藏而进行的敏捷（隐匿）检定具优势。

敏锐嗅觉。幻色蛇基于嗅觉的感知（察觉）检定具优势。

动作

啃咬。近战武器攻击：命中+4，触及5尺，单一目标。伤害：9（2d6+2）点穿刺伤害。

弹射。幻色蛇以一条直线轨迹射出最远30尺，并对触及范围内的一个目标进行一次啃咬攻击。如果幻色蛇射出至少10尺，则本次攻击具优势。如果攻击命中，本次啃咬额外造成7（2d6）点穿刺伤害。
