# 丧尸 ZOMBIES

以下是几个新的丧尸变体。有关丧尸的更多信息，见怪物图鉴。

甲龙丧尸 Ankylosaurus 
Zombie，这种生物很缓慢，对周围环境几乎浑然不觉。

四臂猩猩丧尸 Girallon 
Zombie，四臂猩猩是长着白色皮毛的四臂猩猩猴，四臂猩猩丧尸比他们的非不死同类缓慢，但同样凶猛。赋予他们力量的死灵能量，点燃了他们凶残的心和对生肉的渴望。

暴龙丧尸 Tyrannosaurus 
Zombie，暴龙丧尸的食道里满是较小的丧尸，它可以把它们吐出来。这些丧尸不在暴龙丧尸的控制之下。
