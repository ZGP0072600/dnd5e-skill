# 六岐蛇豹 KAMADAN

六岐蛇豹是一种猫科的掠食者，其形如猎豹，但是从它的肩处还长有六条蛇。尽管与移位兽有较少的相似之处，但实际上两种生物并无关系（但有些学者不这么认为）。

六岐蛇豹通常独行或与配偶一同狩猎。它们可以喷出大量催眠气体，这一行为在它们近战时十分常见。如果六岐蛇豹在其突袭范围内同时存在清醒和昏迷的敌人，它会尝试在处决昏迷敌人之前先尝试杀死尚且清醒的敌人。

挑战等级：4（1100 XP）

敏锐嗅觉。六岐蛇豹的基于嗅觉进行的感知（察觉）检定具优势。

猛扑。如果六岐蛇豹朝目标生物直线移动至少20英尺，并在同一回合内对其发动一次爪击攻击并命中，则目标必须进行一次 DC 
13的力量豁免，豁免失败则应击倒地。如果目标倒地，则六岐蛇豹可以用一个附赠动作对该目标执行两次额外攻击——一次啃咬，一次蛇吻。

动作

多重攻击。六岐蛇豹进行两次攻击：一次啃咬或爪击，一次蛇吻。

啃咬。近战武器攻击：命中+5，触及5尺，单一目标。伤害：6（1d6+3）点穿刺伤害。

爪击。近战武器攻击：命中+5，触及5尺，单一目标。伤害：6（1d6+3）点挥砍伤害。

蛇吻。近战武器攻击：命中+5，触及5尺，单一生物。伤害：6（1d6+3）点穿刺伤害，目标必须进行一次 DC 
12的体质豁免，豁免失败则受到21（6d6）点毒素伤害，豁免成功则伤害减半。

沉眠吐息（短休或长休后充能）。六岐蛇豹呼出一道呈30尺锥形的催眠气体。锥形区域内的所有生物必须进行 DC 
12的体质豁免，豁免失败则昏迷10分钟。当昏迷生物受到伤害时或被他人用一个动作唤醒时，昏迷状态结束。
