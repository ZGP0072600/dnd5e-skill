# 白矮人 Albino Dwarves

楚尔特的白矮人们因火山运动逃离出了他们的地下家园，其中那些没有在南扎路港寻求庇护的白矮人选择了在丛林中谋生。他们用恐龙皮制作护甲；用恐龙骨、燧石和木头打造武器；并用骨头、羽毛、兽牙和石珠打造出华丽的首饰。白矮人们并未忘记如何熔炼金属，但他们很少有条件做这种事。
