# 藏妲拉 Xandala

半精灵术士藏妲拉为了寻找一件由阿图斯·辛贝尔持有的神器“冬之戒”而来到楚尔特大陆。藏妲拉知道这件神器使阿图斯获得了超过一个世纪的生命，她渴求它的力量。如果她碰到冒险者们，藏妲拉会假称自己是阿图斯的女儿，并尝试说服他们帮她找到阿图斯。她的计划是对阿图斯使用支配类人，让他交出冬之戒，随后使用飞行术 Fly 逃跑。

藏妲拉的术法起源来自于某条巨龙的血脉，她的某些位置的皮肤覆盖着薄薄的、富有光泽的鳞片。她有一条叫做“夏天”的伪龙伙伴，夏天认为藏妲拉关于冬之戒的计划十分危险，但是它已经放弃去说服藏妲拉了。在善良阵营角色的努力下，夏天可能转而对付藏妲拉。

藏妲拉的特征：

理想：我的血管里流淌着巨龙的血脉。我注定会成就伟大。

牵绊：冬之戒会给我带来永生与力量。

缺点：我周围的人全是蠢货。
