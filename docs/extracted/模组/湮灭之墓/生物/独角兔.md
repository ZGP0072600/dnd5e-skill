# 独角兔 Almiraj

独角兔是一种体型较大而胆子很小的兔子，一尺长的螺旋状独角从它的前额伸出，神似独角兽的角。如果被迫要发动攻击，它会尝试用它的尖角刺向敌人。

在很久以前，独角兔被来自遥远的扎哈拉地区的商人带到了楚尔特。擅长于躲避捕食者的它们在整个热带半岛繁衍生息。它们居住在土洞里，并且可以被捕获和驯化。在 DM 的允许下，法术获得魔宠 find familiar 可以召唤一只独角兔。

小型野兽，无阵营

 体质10（+0） 智力2（-4） 感知14（+2）

敏锐感官。独角兔依靠视觉和听觉进行的感知（察觉）检定具优势。
