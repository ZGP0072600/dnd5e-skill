# 尼希卢尔

尼希卢尔 NIHILOOR

灵吸怪尼希卢尔为珊娜萨工作，它整天都在制作噬脑怪，将它们放进深水城的下水道里。每只噬脑怪都会攻击它们最先遇到的类人生物，将受害者当作傀儡，侦察城市并向尼希卢尔提供信息。有时候，噬脑怪会奉命寻找某个特定目标，例如城市守卫或城市哨兵的成员。有时候，它们也会奉命抓捕特定的深水城市民，将其带到头骨港作为奴隶或换取赎金。

尼希卢尔拥有异界思维，认为自己和珊娜萨是平等的。但它没兴趣和眼魔竞争，而是更喜欢在幕后行动，实现自己的黑暗计划。
