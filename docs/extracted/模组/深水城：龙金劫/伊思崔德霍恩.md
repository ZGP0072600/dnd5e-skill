# 伊思崔德霍恩

伊思崔德·霍恩 ISTRID HORN

伊思崔德被视为黑暗情报网深水城分部的交易和货币大师。这名盾矮人在码头区一座守卫森严的仓库里经营着非法借贷生意，向那些亟需用钱的人提供贷款。她的利息水平和她的竞争者们（包括从事银行业的贵族家族，例如卡萨兰特家族和伊灵星家族）相当，但无法偿还贷款的罚息十分惊人。

伊思崔德崇拜矮人的财富与幸运之神维加丁。她喜欢让其他人对她负债，并雇用恶棍和讨债者帮她收债。如果这还不够，伊思崔德还会叫她曾经的冒险者同伴来帮忙。
