# 莱拉银手

莱拉·银手 LAERAL SILVERHAND

阿娜马努·莱拉·银手生于风帽之年（谷地历765年），在女神密斯特拉的七个女儿中排行第五。七姐妹全都力量强大，青春不老，对奥术魔法情有独钟。

很久以前，莱拉统治着一个名为斯多南特的国度，被称为北地的女巫之王。后来，她遇到了凯尔本·奥罗桑并与之结婚，后者当上了黑杖，深水城的首席大法师。凯尔本死后，莱拉从公共事务中隐退，又在奥法瘟疫和大裂变后再度复出，但密斯特拉的死亡、重生和离开削弱了她的力量。

莱拉的魔法已经不如曾经那么强大了，但她竭尽全力隐瞒这一事实。只有她最信任的朋友和顾问伊尔明斯特才知道她变弱了多少。尽管力量有所衰退，莱拉依然是一名能力高强，头脑清醒的法师，拥有大量可供使用的魔法。

几年前，达高特·奈沃兰波被剥夺了深水城公开领主一职。在蒙面领主的请求下，莱拉勉为其难地填补了这一空缺。一开始，贵族和公会首领们的要求令她应接不暇，但她已经很好地适应了她的新角色。她不常使用魔法，而是依赖于值得信任的顾问和副手。如果时间允许，她也喜欢乔装打扮离开深水宫，梳理一下思绪，或是去看看她的老朋友（和敌人）。

莱拉和现任黑杖瓦婕拉·莎法尔关系不佳。首先，莱拉比瓦婕拉年长得多，明智得多，也强大得多。在她眼中，瓦婕拉就像是个缺乏安全感的孩子。除此之外，瓦婕拉持有黑杖，其中困着凯尔本·奥罗桑和历任黑杖的灵魂。莱拉想要这根法杖，因为里面有她丈夫留下的一切。可以想见，这两名法师尽可能避开彼此。

情况危急之时，莱拉可以命令瓦婕拉动用灰色武装。在此之前，灰色武装不得在深水城中执行命令，但莱拉的密探告诉她，瓦婕拉一直在偷偷动用这支精英小队的成员执行一些未经授权的任务。莱拉不愿在这件事上和瓦婕拉对峙，她认为自己是在测试瓦婕拉的忠诚，以此来解释她的不作为。
