# 卓尔火枪手

卓尔火枪手 DROW GUNSLINGER

枪械在北地并不常见，但达耶特佣兵团的一些成员却配有兰檀手铳、子弹和袋装的烟雾火药。这些卓尔火枪手都是用枪专家，就像最好的弓箭手善长用弓一样擅长用枪。

菲尔莱科特·拉菲恩 FEL'REKT 
LAFEEN

菲尔莱科特是一名生为女性的男性卓尔。不满于卓尔社会对待男性的态度，他请求加入达耶特佣兵团。贾拉索几乎是立即就对菲尔莱科特产生了好感，从那之后，年轻的卓尔就成了贾拉索最忠诚的副官之一。因为急于证明自己，菲尔莱科特总是第一时间自告奋勇，活力四射地投入到战斗之中。

菲尔莱科特和克莱比格·马斯齐利尔是亲密的朋友，他们合作共事，用敌人打趣逗乐。菲尔莱科特缺乏大多数卓尔身上那种常见的残忍，除非别无选择，他不会杀死对手。

游戏数据。菲尔莱科特·拉菲恩是一名中立善良的卓尔火枪手。除了武器之外，他还带着四袋烟雾火药（参见附录 A）和一只装着二十枚手铳子弹的小袋子。

克莱比格·马斯齐利尔 KREBBYG 
MASQ'IL'YR

克莱比格的家族在很久之前就已经被摧毁了，这让他和他在幽暗地域的生物再无联系。他是个年轻而冲动的卓尔。

克莱比格和菲尔莱科特·拉菲恩密切合作，尽管阵营不同，两人合作得却很好。克莱比格喜欢跟随菲尔莱科特的领导，让他负责大部分思考和交谈。

游戏数据。克莱比格·马斯齐利尔是一名混乱中立的卓尔火枪手。除了武器之外，他还带着四袋烟雾火药（参见附录 A）和一只装着二十枚手铳子弹的小袋子。

索卢恩·泽布林达斯 SOLUUN 
XIBRINDAS

索卢恩是个嗜血且狂热的恶棍，对达耶特佣兵团——特别是对贾拉索——怀有强烈的忠诚。他的弟弟纳尔成功混进了珊娜萨公会。索卢恩将他视为一个弱者，试图通过修习奥术的方式加以补救。他对弟弟本人和他的能力都没什么信心。

索卢恩对地表精灵和半精灵恨之入骨，他从小就被教导要抓住一切机会消灭他们。不为达耶特佣兵团工作时，索卢恩就趁着夜色在深水城漆黑的大街小巷中游荡，寻找落单的精灵或半精灵，将之杀害。他尽可能隐瞒自己的夜间行动，但贾拉索、菲尔莱科特和克莱比格都知道他在干些什么。

游戏数据。索卢恩·泽布林达斯是一名中立邪恶的卓尔火枪手，但有如下变化：

? 索卢恩使用弯刀而非短剑（造成挥砍而非穿刺伤害）。

? 他穿着一双卓尔制的精灵之靴。

? 除了武器之外，他还带着四袋烟雾火药（参见附录 A）和一只装着二十枚手铳子弹的小袋子。
