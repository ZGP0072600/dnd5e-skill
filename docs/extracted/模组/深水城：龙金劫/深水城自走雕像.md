# 深水城自走雕像

深水城自走雕像 WALKING 
STATUES OF WATERDEEP

八座巨大的雕像散落在深水城各处，一旦危机降临，它们就会保卫城市。因为破坏性太强，这些自走雕像只用来抵御军队和看似不可战胜的敌人。

每尊雕像都有自己的名字和独特的外表（参见第196页的“自走雕像”），但其数据并无分别。名为持剑少女的雕像损坏太过严重，已经无法激活，只有黑杖（参见附录 A）的持有者才能激活另外七尊雕像。

地标。多年以来，深水城市民在雕像四周和几尊雕像顶上大兴土木，以为它们只是地标而已。在未被激活的状态下，雕像十分安全——但只要自走雕像被激活，附着在它们身上的建筑就会被立即摧毁。

构装本质。自走雕像不需要空气、饮食或睡眠。
