# 纳尔泽布林达斯

纳尔·泽布林达斯 NA 
R'L XIBRIN DAS

珊娜萨的顾问是一个精神紧绷、诡计多端的男性卓尔，名叫纳尔·泽布林达斯。纳尔的家族在很久之前就被摧毁了，他和哥哥索卢恩双双活了下来，加入了达耶特佣兵团。一年前，纳尔接到了一个艰巨的任务——潜入珊娜萨公会并尽可能接近眼魔。他不仅成功完成了任务，还在赢得珊娜萨信任的过程中说服眼魔除掉了其他顾问。然而，疑神疑鬼的眼魔还是开始怀疑卓尔的忠诚，纳尔越来越为自己的未来感到担忧。如果被迫在自己和达耶特佣兵团之间做出选择，他会选择前者，背叛他的卓尔盟友以求活命。

珊娜萨知道纳尔不太对劲。最近，它给他派了一个触须怪保镖，并命令它在发现纳尔有任何不忠迹象时立即干掉他。

游戏数据

纳尔·泽布林达斯是一名卓尔法师 drow mage。每当需要联系他的兄弟时，他就会准备并使用短讯术 sending。除了其他装备，纳尔还有一只小瓶，里面盛有三剂伤眼药，一种接触性毒药。接触毒药的生物必须成功通过 DC 14的体质豁免检定，否则中毒1小时，并在中毒期间陷入目盲状态。次等复原术或类似魔法可以结束该效果。
