# 黑毒蛇

黑毒蛇 THE 
BLACK VIPER

黑毒蛇是一个臭名昭著的夜贼、扒手、强盗和刺客，在经历了漫长而邪恶的职业生涯后，于一个世纪前死去。最近，爱丝维蕾·罗兹纳，一名傲慢的年轻贵族，使用黑毒蛇的名号展开了她不可告人的犯罪生涯。和许多其他贵族一样，爱丝维蕾也在谈论黑毒蛇的战绩，让这名恶徒显得益发神秘。

罗兹纳家族曾因奴隶贸易——严重违反了深水城的法律——而获罪，在一百多年前被驱逐出城，但之后又回来了。无法取得他们自认为应当得到的尊重令罗兹纳家族愤恨不已。他们依然在安姆和其他地区贩奴，但所有贵族家族都有一些见不得光的秘密。只是因为一些家族争吵传入外人之耳，一个世纪以来，人们一提起罗兹纳的大名就会面露冷笑。为了证明他们的错误并重新赢得市民的尊重，罗兹纳家族一直在不断努力，为此，他们将爱丝维蕾培养成了一名最为得体的贵族。她参加礼仪课和舞蹈课，言行举止无不符合她应有的身份。

年轻时，爱丝维蕾公然反抗，这导致她晚上被锁在屋里，白天被守卫们严密监视。因此，她自己学会了如何撬锁，如何从腰带上和腰包里偷钥匙，如何翻墙并溜过守卫——白天扮演完美的女儿，晚上则享受自由。为了不让见到她的人认出她的身份，爱丝维蕾戴着兜帽和面具，脱下精致的服饰，换上实用的装束——父母永远不会同意她穿的那种。

爱丝维蕾学会了一件事：接受黑毒蛇的传说。当她拉开斗篷，露出她拼凑的衣服，大部分人就会扔下钱包，夺路而逃。通常情况下，她要做的只是拔出匕首而已。当然，某些被她盯上的贵族不愿轻易屈服，因此爱丝维蕾也学会了战斗的技巧。她知道自己正在玩一场危险的游戏，但令她满意的是，至少有一名罗兹纳的家族成员赢得了深水城贵族的尊敬。
