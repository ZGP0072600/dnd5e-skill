# 索文双须

索文·双须 THORVIN 
TWINBEARD

索文是珊娜萨的首席工程师和陷阱制造师。他也是竖琴手同盟的收费线人，尽可能令其获悉珊娜萨的阴谋。索文假装维护设备，趁机在头骨港等地和竖琴手密探偷偷见面。

索文带着一副铁框眼镜，拿着一把又大又沉的扳手，可以当作短棒使用。他还有一套石匠工具、一套铁匠工具和一套盗贼工具，而且拥有这三种工具的熟练项。

游戏数据

索文是一名盾矮人平民 commoner，但有如下变化：

索文属于守序中立阵营。

他拥有如下种族特质：他的基础步行速度为25尺。他在对抗毒素的豁免检定上具有优势，并对毒素伤害拥有抗性。他拥有60尺黑暗视觉。他使用通用语和矮人语。

他的智力值为16（+3）。
