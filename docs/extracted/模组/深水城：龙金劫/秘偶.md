# 秘偶

秘偶 NIMBLEWRIGHT

秘偶是一种用来担任保镖或刺客的魔法构装。它主要由轻质木材制成并由魔法驱动，穿上衣服后可以冒充类人生物。有些秘偶衣着朴素，另外一些则要华丽得多。秘偶没有感情，脸上永远挂着创作者赋予它的表情。

决斗者。秘偶移动时有如舞者，战斗时有如剑客。它靠躲闪和格挡避免伤害，同时灵巧地刺向敌人。

构装本质。秘偶无需空气、饮食和睡眠。使用修复术可以修复它所受到的伤害，但生命值降为0的秘偶将被永远摧毁。
