# 阿莫戈

阿莫戈 AHMAERGO

珊娜萨的总管阿莫戈对牛头人十分迷恋。尽管看起来彬彬有礼，这名盾矮人就像最坏的魔鬼一样阴险狡诈，腐败堕落。然而，他对他的眼魔主人忠心耿耿。

除了眼魔本人，阿莫戈是珊娜萨公会中最有影响力的成员。
