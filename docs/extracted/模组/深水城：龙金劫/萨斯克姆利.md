# 萨斯克姆利

萨斯·克姆利 SAETH 
CROMLEY

萨斯·克姆利是一名退休的城市哨兵尉官，是个惹人喜好的家伙，头脑犀利，喜欢冷嘲热讽。有时，在巴尼布斯·阵风的要求下，他会暂时放下他的退休生活，协助法师调查城市里的特殊案件。克姆利帮助巴尼布斯向平民百姓问话，他十分擅长从他们口中取得信息。克姆利曾经严格遵守哨兵的规章和着装准则，但在已经正式退休的如今，他在这两方面都有些松懈了。
