# 法典

法典

根据罪行的性质，罪行所侵犯的客体，以及已经定罪的既往犯罪记录，对罪行的刑罚可能包括以下一项或多项内容：

死刑

驱逐出城（为期数年或数个夏天）

鞭刑（特定数量）

劳役（为期数天、数月或数年，取决于罪行的严重程度）

在深水堡地牢接受监禁（为期数天或数月，取决于罪行的严重程度）

罚金（向城市支付；未能支付罚金将导致监禁及/或劳役）

损害赔偿金（向受害人或其亲属支付；未能支付损害赔偿金将导致监禁及/或劳役）

禁令（禁止从事某种活动；违反禁令将导致监禁、劳役及/或罚金）

I． 危害领主、官员及贵族罪

| 袭击或拘禁领主罪： | 死刑 |
|---|---|
| 袭击或拘禁官员或贵族罪： | 鞭刑、十日以下监禁，并处500gp 以下罚金 |
| 敲诈官员罪： | 鞭刑，并处10年以下驱逐出城 |
| 谋杀领主、官员或贵族罪： | 死刑 |
| 贿赂或试图贿赂官员罪： | 20年以下驱逐出城，并处金额最高等同于行贿金额双倍的罚金 |
| 未经许可而使用魔法影响领主罪： | 一年以下监禁，并处1,000gp 以下罚金或损害赔偿金 |
| 未经许可而使用魔法影响官员罪： | 1,000gp 以下罚金或损害赔偿金，并处禁令 |

II． 危害城市罪

| 纵火罪： | 死刑或1年以下劳役，并处金额等同于修复费用外加2,000gp 的罚金及/或损害赔偿金 |
|---|---|
| 无正当理由挥舞武器罪： | 十日以下监禁及/或10gp 以下罚金 |
| 间谍罪： | 死刑或永久驱逐出城 |
| 倒卖盗赃罪： | 金额等于盗赃价值的罚金，并处禁令 |
| 伪造公文罪： | 鞭刑，并处10个夏天以下驱逐出城 |
| 妨害司法罪： | 200gp 以下罚金，并处十日以下劳役 |
| 乱丢垃圾罪： | 2gp 以下罚金，并处禁令 |
| 水井投毒罪： | 死刑 |
| 盗窃罪： | 鞭刑，后处十日以下监禁、1年以下劳役或金额等同于盗赃价值的罚金 |
| 叛国罪： | 死刑 |
| 破坏公共财物罪： | 十日以下监禁，并处金额等同于修复费用外加100gp 的罚金及/或损害赔偿金 |
| 未经许可使用魔法影响官员罪： | 1,000gp 以下罚金或损害赔偿金，并处禁令 |

III． 危害神祇罪

| 袭击牧师或世俗信徒罪： | 十日以下监禁，并处500gp 以下损害赔偿金 |
|---|---|
| 神殿内不当行为罪： | 5gp 以下罚金，并处禁令 |
| 公开亵渎神祇或公开亵渎教堂罪： | 禁令 |
| 盗窃神殿物品或供品罪： | 十日以下监禁，并处金额最高等同于盗赃价值双倍的罚金 |
| 盗墓罪： | 十日以下监禁，并处金额等同于修复费用外加500gp 的损害赔偿金 |

IV． 危害市民罪

| 袭击市民罪： | 十日以下监禁、鞭刑，并处1,000gp 以下损害赔偿金 |
|---|---|
| 敲诈或威胁市民罪： | 500gp 以下罚金或损害赔偿金，并处禁令 |
| 入室盗窃罪： | 3个月以下监禁，并处金额等同于盗赃价值外加500gp 的损害赔偿金 |
| 破坏财产或牲畜罪： | 金额等同于修复或重建费用外加500gp 的损害赔偿金 |
| 扰乱治安罪： | 25gp 以下罚金，并处禁令 |
| 无正当理由谋杀市民罪： | 死刑或10年以下劳役，并处1,000gp 以下损害赔偿金（向受害人家属支付） |
| 有正当理由谋杀市民罪： | 5年以下驱逐出城、3年以下劳役或1,000gp 以下损害赔偿金（向受害人家属支付） |
| 抢劫罪： | 1个月以下劳役，并处等同于盗赃价值外加500gp 的损害赔偿金 |
| 奴役他人罪： | 鞭刑，并处10年以下劳役 |
| 未经许可使用魔法影响市民罪： | 1,000gp 以下罚金或损害赔偿金，并处禁令 |
