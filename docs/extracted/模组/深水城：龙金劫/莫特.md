# 莫特

米尔特 MIRT

米尔特曾被叫作无情者米尔特，或是老狼。在他曾经四处留情的冒险者生涯中，米尔特名利双收。如今，年迈而睿智的米尔特是一位蒙面领主，一名竖琴手，也是莱拉·银手的贴身顾问。岁月并未压垮他，尽管肉体已经变得软弱，他看上去却依然强壮而充满活力，而且头脑精明。在魔法的作用下，米尔特活了好几百年，是所有蒙面领主中最不屑隐瞒身份的一个。

尽管腰围惊人，如果情况所需，米尔特还是能用不错的速度灵活移动。不仅如此，他从未荒废他的战斗技巧。他的妻子阿斯帕在几年前去世了，他杂乱的豪宅也曾见证过美好的岁月。米尔特白天忙于政务，晚上则纵情酒色。

宝藏

米尔特拥有各种各样的魔法物品，但他只随身携带其中几样。需要时，他也可以用其他魔法物品武装自己。

除了其他魔法装备之外，米尔特还有一身领主套装（参见附录 A）。他只在使用官方身份和其他蒙面领主会面时穿上套装。
