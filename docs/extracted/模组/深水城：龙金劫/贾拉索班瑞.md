# 贾拉索班瑞

贾拉索·班瑞 JARLAXLE BAENRE

贾拉索是个张扬华丽、虚张声势、特立独行的卓尔。他领导着一个名为达耶特佣兵团的卓尔流民组织，由失去了公民权利的男性卓尔组成，其中大部分都来自惨遭毁灭或名誉扫地的卓尔家族。靠着与生俱来的精明头脑、幽默感、高超剑技和大量有用的魔法物品，贾拉索渗透了路斯坎，为这片不法之地带来了某种秩序，并宣布自己才是路斯坎的秘密领主。

贾拉索喜欢布下一张错综复杂的阴谋之网，让他的敌人不知所措——他最近的计划是让路斯坎加入领主联盟，从而获得合法地位。这座城市的糟糕名声挫败了他之前的所有努力。领主联盟的现任首领也对路斯坎的加入表示反对。一些人甚至直接宣称这座城市永远也不会得到领主联盟的欢迎。尽管如此，贾拉索希望能游说深水城的公开领主莱拉·银手支持路斯坎的加入——即便这意味领主联盟将失去其他成员。将路斯坎的命运和深水城绑在一起，将大大增强贾拉索在剑湾一带的政治和经济力量。

贾拉索以伊路斯克人类扎多兹·佐德的身份来到了深水城。“佐德船长”是海少女的狂欢的拥有者，这是一支在剑湾沿岸巡回表演的嘉年华舰队，由三艘船组成：美人号、浪子号和狂徒号（三个词都是用来形容贾拉索的）。他大部分时间都在他的私人船只美人号上。另外两艘船则用来运载嘉年华成员以及游行车队。

贾拉索和兰檀（一个南方的岛国）达成了联盟，并为达耶特佣兵团的副官装备了依靠魔法烟雾火药发射子弹的枪械。他还收购了一艘名为红十月号的兰檀潜水艇。这艘水下设备接在美人号下方，以免被人发现。贾拉索计划在阴谋暴露时使用这艘潜水艇逃离深水城。

贾拉索首先忠于自己，其次则忠于达耶特佣兵团。
