# 塔什琳雅菲拉

塔什琳·雅菲拉 TASHLYN YAFEERA

塔什琳（数据面板见下页）是深水城散塔林会的武器和佣兵大师。在这个职位上，她为黑暗情报网的佣兵提供护甲、武器和训练。

塔什琳为驻扎在南门的矮人法官沃荣达·标石担任保镖，以此掩饰自己的身份。她喜欢这名矮人，也赢得了他的信任，并因此当上了城市守卫的校官。在这个位置上，她监视着通过城门的人流，确保她在黑暗情报网的盟友可以自由进出。

塔什琳出生于一个富余的卡林珊家族，有一种不受约束的优越感。她暴躁易怒，不愿在战斗面前退缩。她尊重任何能在近战中胜过她的人。
