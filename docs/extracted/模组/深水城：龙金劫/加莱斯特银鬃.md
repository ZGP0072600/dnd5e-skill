# 加莱斯特银鬃

加莱斯特·银鬃 JALESTER 
SILVERMANE

加莱斯特是一名二十五岁上下的热血青年，来自遥远的科米尔。在那儿，他为一个名为钢影的佣兵团工作，以此寻求刺激。几年前，加莱斯特和佣兵团的另外几名成员一同离开谷底来到深水城，其中，法雷尔·暗刃成为了他的挚友和爱人。

法师伊尔明斯特和这两个年轻男性都交了朋友，并将他们介绍给了莱拉·银手，后者让他们担任副手和间谍。加莱斯特和法雷尔帮助公开领主揭发了一场推翻政府的阴谋，法雷尔却在将罪犯绳之以法的过程中惨遭杀害。后来，加莱斯特留在了深水城，成为莱拉的外勤特工，为深水城和领主联盟效力。自从法雷尔死后，他再也没有谈过恋爱，但一直渴望着爱情。

宝藏

加莱斯特持有一枚哨兵徽章（参见附录 A）。如果徽章丢失或被从他身上取走，它会回到莱拉·银手的手中。
