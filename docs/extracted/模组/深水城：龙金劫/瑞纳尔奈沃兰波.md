# 瑞纳尔奈沃兰波

瑞纳尔·奈沃兰波 RENAER 
NEVEREMBER

瑞纳尔是前任深水城公开领主和现任无冬城领主的儿子，两人关系疏远。父子彼此憎恨，特别是当瑞纳尔不得不处理父亲留下的烂摊子时，他可以说是不开心极了。两人也有其相似之处：玉树临风的外表、对酒精的热爱和高超的交际手腕。然而瑞纳尔缺少父亲的好斗性情、坏脾气和糟糕的判断力。

瑞纳尔靠着母亲留给他的一大笔遗产生活。他即将步入中年，于是放弃冒险，在某种程度上安顿了下来。作为一名竖琴手，他花费大量时间保护深水城市民，以免他们被像他父亲那样的人夺走财富和权利。他拥有奈沃兰波的家族宅邸，一栋位于滨海区的四层大宅。然而，瑞纳尔尽可能不在那儿多待，因为它一直受到忠于他父亲的密探的监视。他公开邀请所有朋友随意使用这座大宅，而他自己则将大部分空闲时间花在酒馆和娱乐城里。

一些人认为瑞纳尔和他父亲的疏远关系只是刻意制造的假象，所有身负奈沃兰波之名的人都是深水城的敌人。面对这种指责，瑞纳尔只能摇摇头，继续过着自己的日子。他有许多强大的朋友照看着他的背后。
