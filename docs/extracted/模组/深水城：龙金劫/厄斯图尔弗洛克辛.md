# 厄斯图尔弗洛克辛

厄斯图尔·弗洛克辛 URSTUL 
FLOXIN

厄斯图尔·弗洛克辛效力于曼松，是格劳洪德家族的散塔林小队中等级最高的成员。厄斯图尔是一个掩饰了本性的暴徒 thug，有着蛇一般的魅力和性情，但格劳洪德家族之所以容忍他，是因为他的间谍能向他们提供从全城各处搜集而来的有用信息。厄斯图尔听命于曼松的拟像，它通过一个传送法阵往来于格劳洪德家族和克拉特双子塔之间。

厄斯图尔是一个四十多岁的男人，高大而壮硕。他在格劳洪德家族横冲直撞，仿佛这地方属于他一样。他有一群黑色的飞蛇 flying snake，用来向他遍布全城的手下送信。
