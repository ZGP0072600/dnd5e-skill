# 杜尔南

杜尔南 DURNAN

杜尔南是哈欠门酒馆的主人和老板。他看起来像个风华不再的中年人，实际上却有着敏锐的头脑，也能在情况所需时挥剑作战。他不喜欢谈论自己的过去，也不会透露他作为冒险者时的故事。

即便杜尔南还有任何依然在世的家庭成员，他也从未提起过他们。如果一个字够用，他很少会说两个字。他有一种阴郁的幽默感，对那些冒险进入山下区的人不抱有丝毫同情。

他将他的魔法巨剑冷酷之竿放在吧台下方触手可及的地方。如果他愿意，他可以用它将桌子劈成两半。他还有一副双发弩（一种射程较短的重弩，能向同一目标射出两发弩箭）。尽管如此，如果他参与战斗，他更喜欢挥舞拳头或是投掷酒杯。

杜尔南很少远离哈欠门，如有需要，他会让雇员替他跑腿。如果他心情不错，他或许会温柔游说那些他认为无法幸存的“无名小卒”别去山下区冒险。他也会把冒险者介绍给那些可能向他们提供帮助或有用信息的酒馆常客。
