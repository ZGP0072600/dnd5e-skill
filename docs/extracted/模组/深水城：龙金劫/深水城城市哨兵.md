# 深水城城市哨兵

城市哨兵 CITY WATCH

城市哨兵是深水城的警察部队，负责维护治安，逮捕罪犯。城市哨兵巡逻队通常由四至十二人组成。准备面对棘手时间时，巡逻队也可能获得一名祭司 priest（从当地神殿暂借）或一位魔法师 mage（来自法师与安民法师会）作为援军。

城市哨兵的警衔

城市哨兵的成员被称为警官。他们的警衔由低到高分别是：

警员

警长（武长）

尉官（民长）

校官（高级民长；哨所首领）

将官（城区民长；每个城区一人）

哨兵指挥官

城市哨兵还有一位高级军备官，负责物资供养并听命于哨兵指挥官。哨兵指挥官则听命于公开领主莱拉银手。

游戏数据

城市哨兵的大部分成员都是老兵 veteran。一些警衔最高的成员是骑士 knight。执勤时，所有城市哨兵在都头戴头盔，手提短棒。
