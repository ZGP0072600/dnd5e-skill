# 哈巴兹

哈巴兹 HRABBAZ

哈巴兹是一名肌肉发达，上颚开裂的半兽人，为格劳洪德领主夫妇担任保镖。他举止有礼，衣着得体，但这令人放松的外表下却藏着一刻凶残的心。他对雅拉·格劳洪德夫人十分尊重，却不怎么喜欢她喜怒无常的丈夫。奥隆德丢面子并不会让他伤心。
