# 维克托罗卡萨兰特

维克托罗·卡萨兰特 VICTORO CASSALANTER

卡萨兰特家族的主人是一名极其英俊的半精灵，喜欢金钱和权力。通过与阿斯摩蒂斯达成交易，他和他的妻子二者兼得——在这场交易里，他们卖出了三个孩子的灵魂。

维克托罗是一名阿斯摩蒂斯祭司 priest，只有他的妻子和他最亲密的朋友才知道他对九狱之主的信仰。大多数深水城市民将他视为一个成功的银行家，慈善家和洛山达信徒。他用一部分利润为贫苦之人提供食物和住所。但在慷慨的表象之下，维克托罗是一只自私自利的野兽。

维克托罗的长子奥斯瓦多永远失去了灵魂，无可挽回。为了减轻他的负罪感，维克托罗制定了一个计划，以求赎回两个年幼的双胞胎，特兰佐和艾泽瑞娜的灵魂。依照契约的规定，他们的灵魂会在九岁生日时被夺走，而那天正在迅速逼近。然而，正如契约所说，维克托罗可以通过提供“将近一百万枚金币和将近一百个不幸的灵魂”来免除这项义务。

维克托罗受过良好的教育，温文尔雅，不易发怒，而且得到了健康长寿、免疫疾病的祝福。他穿着最时尚的服饰，行走时拿着一根顶端镶有红宝石的手杖——尽管他根本用不着它。这根手杖拥有支配权杖 rod 
of rulership 的魔法特性。
