# 胡拉姆

胡拉姆 HLAM

这位可敬的人类武僧住在深水山侧半山腰的一座洞穴里。胡拉姆是崇拜提尔的小型修道组织公正之手教团的大宗师。想要成为学徒的人会定期前去拜访他，学习圣拳宗的技艺——一种结合了牧师魔法和武僧训练的技艺。返回城市时，他们往往一头雾水，遍体鳞伤，而且再也不想去拜访他了。

遇到重大危急时，人们可能会寻求胡拉姆的帮助。有时他给出金玉良言，有时他离开洞穴，用拳头纠正错误。他可以在故事中任何时点出现，为角色提供帮助。如果角色们需要指引或训练，也可以主动去他的洞穴里拜访他。臂铠教团将他视为忠实的盟友。

胡拉姆对疾病免疫，而且无需饮食。尽管也会衰老，他不会因为年迈而变得虚弱。
