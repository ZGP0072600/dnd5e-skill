# 麦伦战龙

麦伦·战龙 MELOON WARDRAGON

麦伦正值盛年，是一名英俊而强大的战士。他信仰女神泰摩拉，喜欢酣畅淋漓的战斗。他的朋友们——其中也包括瑞纳·奈沃兰波和瓦婕拉·莎法尔——将他形容为一个诚实、乐观而且极为幸运的人。直到最近，他都是灰色武装的成员，直接听从黑杖的命令。在最近几个月里，麦伦将大部分时间都花在哈欠门酒馆。

三个月前，因为无聊，麦伦陪着一群初出茅庐的冒险者进入山下区冒险。在那儿，他的好运用完了。在地城里休息时，冒险小队遇到了珊娜萨放出的怪物，包括几只噬脑怪。一只噬脑怪成功吃掉了麦伦的大脑，并取代了它的位置，将泰摩拉的战士变成了一个傀儡。麦伦干掉了他毫无戒心的同伴，作为珊娜萨公会的密探回到了深水城。

珊娜萨雇用的夺心魔尼希卢尔培育了麦伦脑壳里的噬脑怪。它知道麦伦知道的所有事，而麦伦的一举一动和他进入山下区之前并无分别。他常常呆在哈欠门酒馆，试图结交冒险者，并在合适的时候向他们伸出援手。噬脑怪的首要目标是让冒险者们远离山下区，去完成那些能帮助珊娜萨达成目的的任务。这些任务往往需要消灭珊娜萨的敌人，因此麦伦十分愿意和那些上当的冒险者们并肩作战。
