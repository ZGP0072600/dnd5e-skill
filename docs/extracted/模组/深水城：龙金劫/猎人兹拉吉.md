# 猎人兹拉吉

猎人兹拉吉 ZIRAJ THE HUNTER

兹拉吉是一名半兽人猎人，用一把加大号长弓射出与配套的大号箭矢。他是暗黑情报网的暗杀大师。如果兹拉吉打算杀人，那是因为他的朋友（达维尔、伊思崔德、斯齐莫或塔什琳）要求他这么做。角色们可能会成为兹拉吉的目标，兹拉吉也可能帮他们消灭某个共同的敌人。他是那种强大而沉默的人。

城市哨兵收到报告称，有个人影在深水城的屋顶出没。那是一个魁梧的影子，在它的栖身之地虎视眈眈，用长长的黑箭降下死亡，悄然离去。他从哪儿来——是否有家——始终是谜，就像没人知道他下次会在哪儿出现一样。

宝藏。兹拉吉身穿+2皮甲，拿着一把超大号的长弓。这把独特的武器只能由中型或更大体型，且力量不低于18的生物使用。长弓射出超大号的弓箭，造成2d6+使用者力量调整值的穿刺伤害。其射程和普通长弓相同。
