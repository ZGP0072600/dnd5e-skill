# 福卢恩布拉格玛

福卢恩·布拉格玛 FLOON 
BLAGMAAR

福卢恩是一个三十岁出头的深水城本地人，相貌英俊却不太聪明。他曾经在娱乐城担任护卫，但现在没在工作。几个月前，他遇到了一名行为不端的已婚贵族，对方用大笔的贿赂让他保持沉默，这就是他如今的经济来源。

福卢恩的朋友遍布全城，他将大部分时间花在饮酒作乐上。他靠外表过活，对人生毫无规划，也没有工作谋生的兴趣。
