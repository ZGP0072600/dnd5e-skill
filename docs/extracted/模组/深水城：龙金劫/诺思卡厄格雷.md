# 诺思卡厄格雷

诺思卡·厄格雷 NOSKA 
UR'GRAY

诺思卡是珊娜萨公会的执行者，十分残忍。山下区的一只绿色软泥怪溶解了他的左手和左侧小臂，取而代之的是一把接在残肢上的重弩。

游戏数据

诺思卡·厄格雷是一名盾矮人暴徒 thug，但有如下变化：

诺思卡属于中立邪恶阵营。

他拥有如下种族特质：他的基础步行速度为25尺。他在对抗毒素的豁免检定上具有优势，并对毒素伤害拥有抗性。他拥有60尺黑暗视觉。他使用通用语和矮人语。

因为残疾，他在攀爬的力量检定上具有劣势。
