# 阿玛利亚卡萨兰特

阿玛利亚·卡萨兰特 AMMALIA 
CASSALANTER

卡萨兰特家族爱慕虚荣的女主人接受过奥术训练。和她的丈夫维克托罗一样，她也是阿斯摩蒂斯的信徒。在他们年轻时，阿玛利亚和维克托罗与大魔鬼签署了一份契约，用自家孩子们的灵魂换取权力、健康和长寿。他们的长子奥斯瓦多被立即夺走了灵魂，变成了一只链魔。年幼的双胞胎特兰佐和艾泽瑞娜也会在年满九岁时失去灵魂。

根据协议中的某项条款，卡萨兰特家族可以支付代价找到出路。但这么做不仅需要大量金币，还将献祭许多不幸的人。维克托罗负责追寻达高特·奈沃兰波失落的龙金秘宝，与此同时，阿玛利亚则负责筹备一场用来庆祝建城节的毒宴。

阿玛利亚举止有礼、博览群书、游历甚广而且异常精明。她以擅长讨价还价而著名。她喜欢研究鳞翅类昆虫学，庄园里有一座极为华美的蝴蝶花园。她让她年幼的孩子们在她的监督下在花园里玩耍。
