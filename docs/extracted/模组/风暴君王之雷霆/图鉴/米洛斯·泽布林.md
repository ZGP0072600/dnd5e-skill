# 米洛斯·泽布林

米洛斯·泽布林

MIROS XELBRIN

中型类人生物（达玛拉人类），中立善良

护甲等级 10

生命值 22（4d8 + 4）【原文如此，实际应为26（4d8+8）】

速度 30尺

STR　　　DEX　　　CON　　　INT　　　WIS　　　CHA

16（+3） 10（+0） 15（+2） 11（+0） 12（+1） 14（+2）

技能 胁迫+4，察觉+3

感官 被动察觉13

语言 通用语

动作

熊抱。近战武器攻击：命中+5，触及5尺，一个生物。伤害：5（1d4+3）点钝击伤害，目标被擒抱（逃脱 DC 13）并且在接下来米洛斯的每个回合开始时受到5（1d4+3）点钝击伤害，直到擒抱中止。米洛斯在擒抱一个生物时无法做出攻击。。

短棒。近战武器攻击：命中+5，触及5尺，一个目标。伤害：5（1d4+3）点 钝击伤害。

重弩。远程武器攻击：命中+2，射程100/400尺，一个目标。伤害： 5（1d10）点穿刺伤害。米洛斯带着十支弩矢。

扮演信息

旅店老板米洛斯是一位退休的嘉年华奇景，绰号“雪怪”，这是因为他酒桶造型的身体，以及覆盖着他手臂、胸部、后背、头部的浓密白色毛发。当金色原野遭受损害时，他的生意也同样会变差，所以大会大步流星地前去保护这座院子。

理念：“就如翡翠闲庭一样，我相信文明和旷野需要学会共存。”

牵绊：“你可以尽管取笑我，但是不要说我旅馆和员工们的坏话！”

缺陷：“当某件事情让我沮丧苦恼的时候，我会倾向于陷入狂怒。”
