# 沙尔乌斯·马托里奥

沙尔乌斯·马托里奥

SHALVUS 
MARTHOLIO

中型类人生物（图拉米人类），绝对中立

护甲等级 13（皮甲）

生命值 27（6d8）

速度 30尺

STR　　　DEX　　　CON　　　INT　　　WIS　　　CHA

10（+0） 15（+2） 10（+0） 12（+1） 14（+2） 14（+2）

技能 欺瞒+4，洞察+4，调查+3，察觉+4，巧手+4，隐匿+4

感官 被动察觉12

语言 通用语，精灵语

偷袭（1/回合）。当沙尔乌斯以一次武器攻击命中一个目标，且他在攻击骰上有优势的时候，或者当目标位于一个并未失能的沙尔乌斯的盟友的5尺内且沙尔乌斯在攻击骰上没有劣势时，沙尔乌斯对该目标造成额外7（2d6）点伤害。

动作

长棍。近战武器攻击：命中+2，触及5尺，一个目标。伤害：3（1d6）点钝击伤害，或双手使用时4（1d8）点钝击伤害。

手弩。远程武器攻击：命中+4，射程30/120 尺，一个目标。伤害：5（1d6+2）点穿刺伤害。沙尔乌斯带着十支弩矢。

扮演信息

驳船匠旅馆的纳拉斯科?塞隆德交给了牧羊人沙尔乌斯一个重要的任务：找出让金色原野处于暗黑情报网控制下的最好办法。沙尔乌斯相信，如果成功的话将确保他在散塔林会的等级中获得迅速的提升。

理念：“我会尽全力向散塔林会证明我自己。”

牵绊：“我喜欢动物，并且会非常保护它们。”

缺陷：“我无法抗拒为了满足自己的野心而冒险。”
