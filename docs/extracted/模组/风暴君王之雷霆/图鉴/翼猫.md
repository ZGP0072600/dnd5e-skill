# 翼猫 Tressym

翼猫是一种顽皮的有翼猫类，像家猫一样大，翼展3尺。

翼猫被认为是在家猫身上进行魔法实验的产物，它们具有智能，并且已知会与类人生物，尤其是游侠们和法师们，建立牢固的友谊。翼猫与它们的同类相处的很好，但它们很少居住在一起或一块狩猎。它们平和地忽略蝙蝠、妖精龙以及类似的东西，但仇恨蚊蝠和邪恶的飞行怪兽，比如刺尾狮。它们也喜欢戏弄狗。

## 翼猫 Tressym 微型怪兽，混乱中立

护甲等级：12

生命值：5（2d4）

速度：40尺，攀爬30尺，飞行40尺。

力量3（-4）
敏捷15（+2）
体质10（+0）智力11（+0）
感知12（+1）
魅力12（+1）

技能：察觉+5，隐匿+4

伤害免疫：毒素

状态免疫：中毒

感官：黑暗视觉60尺，被动察觉15

语言：理解通用语但不会说。

挑战等级：0（10 XP）

侦测隐形 Detect Invisibility。在翼猫60尺内，魔法的隐形在翼猫眼中无法隐藏任何东西。

敏锐嗅觉 Keen Smell。翼猫依赖嗅觉进行的感知（察觉）检定具有优势。

毒性感知 Poison Sense。翼猫能通过味觉、触觉或嗅觉检测物质是否有毒。

### 动作

爪击 Claws。近战武器攻击：命中+0，触及5尺，一个目标。伤害：1点挥砍伤害。翼猫以小型啮齿动物、鸟类和昆虫为食，像普通的猫一样追踪并扑击猎物，但有着飞行带来的额外优势。然而，翼猫不会攻击雏鸟或掠夺鸟蛋。翼猫会与其他同类交配，但它们不会相伴终生。一只翼猫也能与普通的猫交配，然而它们的后代中每十只仅有一只是头翼猫，其他的则是普通的猫。翼猫有很好的记忆力，特别是在它遇到危险的时候。例如，一头看到人类使用闪电束魔杖 wand of lightning 
bolts 的翼猫会在它的余生都记得“人类握持的木棒”的危险。一头幸运健康的翼猫能活到20岁。若 DM 许可，施放法术获得魔宠 find familiar 的人可选择召唤一只翼猫来代替一只普通的猫。
