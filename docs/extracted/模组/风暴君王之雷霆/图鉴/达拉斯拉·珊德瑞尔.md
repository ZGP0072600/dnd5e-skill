# 达拉斯拉·珊德瑞尔

达拉斯拉·珊德瑞尔

DARATHRA SHENDREL

中型类人（琼达斯人类），守序善良

护甲等级 14（胸甲）

生命值 52（8d8 + 16）

速度 30尺

STR　　　DEX　　　CON　　　INT　　　WIS　　　CHA

16（+3） 11（+0） 14（+2） 11（+0） 11（+0） 15（+2）

技能 历史+2，威吓+4，调查+2，察觉+2，游说+4

感官 被动察觉12

语言 通用语

勇气。达拉斯拉在进行对抗恐慌的豁免检定时具有优势。

动作

多重攻击。达拉斯拉进行两次近战攻击。

巨剑。近战武器攻击：命中+5，触及5尺，一个目标。伤害：10（2d6+3）点挥砍伤害。

重弩。远程武器攻击：命中+2，射程100/400尺，一个目标。伤害：5（1d10）点穿刺伤害。达拉斯拉带着二十支弩矢。

扮演信息

作为三猪镇的护民官及竖琴手同盟的一位秘密特工，达拉斯拉立誓要保护这个镇子。她非常认真地履行自己的职责。除了自身的装备外，她还有一匹没有装甲的战马 warhorse，遏制者 Buster。

理念：“好人理应获得一切成功得利的机会，并且免受暴政压迫。”

牵绊：“我会舍弃生命来保护三猪镇和它的镇民们。”

缺陷：“我绝不让步。敢来逼迫我，我就逼迫回去。”
