# 贝尔朵拉

贝尔朵拉

BELDORA

中型类人生物（伊路斯坎人类），混乱善良

护甲等级 12

生命值 18（4d8）

速度 30尺。

STR　　　DEX　　　CON　　　INT　　　WIS　　　CHA

10（+0） 14（+2） 10（+0） 16（+3） 12（+1） 16（+3）

技能 欺瞒+5，洞悉+3，调查+5，察觉+3，游说+5

感官 被动察觉13

语言 通用语，龙语，矮人语，半身人语

动作

短剑。近战武器攻击：命中+4，触及5尺，一个目标。伤害：5（1d6+2）点穿刺伤害。

手弩。远程武器攻击：命中+4，射程30/120尺，一个目标。伤害：5（1d6+2）穿刺伤害。贝尔朵拉带着十支弩矢。

反应

躲闪掩护。对一次将会命中贝尔朵拉的远程攻击，她的 AC 增加2。若要如此做，贝尔朵拉必须能看到攻击者，并且不能被擒抱或束缚。

扮演信息

贝尔朵拉是竖琴者联盟的一员，用她的智慧和诡计求生。她看上去是一个无家可归的流浪者，但其实是个回避物质财富的生存者。

理念：“我们都应当为了帮助他人而奋斗。”

牵绊：“我会为了保护弱者而冒生命危险。”

缺陷：“我喜欢欺骗他人。让生活更有趣一点，不好么？”
