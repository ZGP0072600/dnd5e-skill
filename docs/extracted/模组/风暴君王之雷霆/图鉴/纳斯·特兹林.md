# 纳斯·特兹林

纳斯·特兹林

NARTH TEZRIN

中型类人生物（泰瑟尔人类），混乱善良

护甲等级 12

生命值 18（4d8）

速度 30 尺

STR　　　DEX　　　CON　　　INT　　　WIS　　　CHA

10（+0） 15（+2） 10（+0） 12（+1） 14（+2） 16（+3）

技能 洞察+4，调查+3，察觉+6，游说+5

感官 被动察觉16

语言 通用语，矮人语

灵巧动作。纳斯在自己的回合里可以用一个附赠动作执行疾走，撤离，或者躲藏动作。

动作

短剑。近战武器攻击：命中+4，触及5尺，一个目标。伤害：5（1d6+2）点穿刺伤害。

手弩。远程武器攻击：命中+4，射程30/120尺，一个目标。伤害：5（1d6+2）点穿刺伤害。纳斯带着二十支弩矢。

扮演信息

纳斯向冒险者出售装备，而他自己也具有冒险精神。狮盾商会给了他的酬劳很好，但是他渴望为自己取得声望。然而，他与他的伙伴阿莱斯特拉一起经营生意，他知道如果自己逃跑了并且不回来的话，阿莱斯特拉绝不会原谅他的。

理念：“风险越大，回报越多。”

牵绊：“我爱慕我的同事阿莱斯特拉，我想做些什么来打动她。”

缺陷：“我愿意冒生命危险去成为一个传奇。”
