# 紫虫幼虫 Purple Wormling

一条紫虫幼虫是一只不超过六周大的紫虫幼崽。它的柔软有弹性的身体有9英尺长，重1,500磅。它的嘴和肌肉还没有强壮到足以让幼虫穿过岩层。它尾刺中的毒素和它体内的酸性物质还相对较弱。尽管如此，幼虫是一个贪婪的进食机器，攻击它能用嘴包下的几乎任何东西。一个幼虫的食道可以容纳4个小型生物。

### 动作

多重攻击 Multiattack。幼虫进行两次攻击：一次啃咬和一次尾刺。

啃咬 Bite。近战武器攻击：命中+5，触及5尺，一个目标。伤害：7（1d8+3）的穿刺伤害。如果目标是小型或更小体型的生物。则目标必须成功通过 DC 
13的敏捷豁免，否则被幼虫吞下。被吞下的生物目盲且被束缚，对紫虫幼虫体外的攻击或其他效果视为处于全身掩护，并且在紫虫幼虫的每个回合开始时承受3（1d6）点强酸伤害。 如果幼虫在一个回合里承受了10点或更多来自其体内生物对其造成的伤害，则该幼虫在该回合结束时必须成功通过 DC 21的体质豁免，否则将吐出所有被其吞咽的生物，并使这些生物倒在幼虫周围10尺范围内的某处。如果该幼虫死亡，则被吞下的生物将不再被束缚，且可以消耗5尺移动力从尸体中逃出，以倒地状态脱离。

尾刺 Tail Stinger。近战武器攻击：命中+5，触及10尺，一个生物。伤害：5（1d4+3）点穿刺伤害，且目标必须进行一次 DC 13的体质豁免，豁免失败则受到10（3d6）点毒素伤害，豁免成功则伤害减半。
