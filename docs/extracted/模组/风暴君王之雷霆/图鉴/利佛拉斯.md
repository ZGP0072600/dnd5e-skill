# 利佛拉斯

利佛拉斯

LIFFERLAS

巨型植物，无阵营

护甲等级 13（天生护甲）

生命值 59（7d12 + 14）

速度 20 尺

STR　　　DEX　　　CON　　　INT　　　WIS　　　CHA

19（+4） 6（-2） 15（+2） 10（+0） 10（+0） 7（-2）

伤害易伤 火焰

伤害抗性 钝击，穿刺

感官 被动察觉10

语言 通用语

虚假外形。当利佛拉斯保持静止时，难以将其与普通的树木区分开来。

动作

多重攻击。利佛拉斯进行两次猛击攻击。

猛击。近战武器攻击：命中+6，触及10尺，一个目标。伤害：14（3d6+4）点钝击伤害。

扮演信息

翡翠闲庭的德鲁伊们用法术启蒙了利佛拉斯。金色原野是他的家，这里的人们是他的朋友。孩子们喜欢将他们的名字和姓名首字母刻在他身上，喜欢挂在他的树枝上，而他也对这些事感到快乐。

理念：“我的存在就是为了保护金色原野的人们和庄稼。”

牵绊：“小孩子太棒了，我会尽一切努力来让他们感到开心且安全。”

缺陷：“我记不清人名，经常把他们弄混。”
