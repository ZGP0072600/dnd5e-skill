# 梓梁

梓梁

ZI LIANG

中型类人生物（受国人类），混乱善良

护甲等级 15

生命值 22（5d8）

速度 40尺

STR　　　DEX　　　CON　　　INT　　　WIS　　　CHA

12（+1） 15（+2） 11（+0） 14（+2） 16（+3） 11（+0）

技能 特技+4，运动+3，察觉+5，隐匿+4

感官 被动察觉15

语言 通用语，精灵语，地精语

无甲防御 。 当梓没有着甲拿盾时，她的 AC 增加上感知调整值。

动作

多重攻击。梓进行两次近战攻击。

长棍。近战武器攻击：命中+3，触及5尺，一个目标。伤害：4（1d6+1）点钝击伤害，或双手使用时5（1d8+1）点钝击伤害。

投石索。远程武器攻击：命中+4，射程30/120 尺，一个目标。伤害：4（1d4+2）点钝击伤害。梓带着二十颗投石索弹丸。

扮演信息

梓梁是大地之母裳缇亚的虔诚信徒。她对金色原野的守卫没有太多信心，所以在工作之外的时间里，她会在神殿到农场的这一路上进行巡逻。

理念：“如果我们诚心地照料我们的菜园和田地，裳缇亚将会对我们微笑。”

牵绊：“金色原野是北地的粮仓。人们依赖着它的安全与繁荣，我也会尽我最大的努力去保护它。”

缺陷：“我不相信权威，我只做我的心认为正确的事情。”
