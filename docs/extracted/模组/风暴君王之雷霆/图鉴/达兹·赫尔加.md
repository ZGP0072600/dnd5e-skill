# 达兹·赫尔加

达兹·赫尔加

DARZ HELGAR

中型类人生物（伊路斯坎人类），绝对中立

护甲等级 12

生命值 27（5d8 + 5）

速度 30尺

STR　　　DEX　　　CON　　　INT　　　WIS　　　CHA

15（+2） 15（+2） 12（+1） 10（+0） 11（+0） 11（+0）

技能 威吓+2，巧手+4，隐匿+4

感官 被动察觉10

语言 通用语

偷袭（1/回合）。当达兹以一次武器攻击命中一个目标，且他在攻击骰上有优势的时候，或者当目标位于一个并未失能的达兹的盟友的5尺内且达兹在攻击骰上没有劣势时，达兹对该目标造成额外7（2d6）点伤害。

动作

短剑。近战武器攻击：命中+4，触及5尺，一个目标。伤害：5（1d6+2）点穿刺伤害。

投石索。远程武器攻击：命中+4，射程30/120 尺，一个目标。伤害：4（1d4+2）点钝击伤害。达兹带着二十颗投石索弹丸。

扮演信息

在达兹年轻的时候，他是深水城的珊娜萨盗贼公会的成员。在因为罪行而经历了十年的牢狱之苦后，他切断了与深水城的一切关系，前往北方成为了一名营地管理员。

理念：“你可以逃离你的过去，但是却无法躲过它。”

牵绊：“我在三猪镇获得了新生，这次我绝不会逃走了。”

缺陷：“我从不后悔。为了活下去我什么都做。”
