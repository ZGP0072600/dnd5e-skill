# 娜可塞纳·德莱斯卡拉

娜可塞纳·德莱斯卡拉

NAXENE DRATHKALA

中型类人生物（图拉米人类），中立善良

护甲等级 10（带法师护甲13）

生命值 27（6d8）

速度 30尺

STR　　　DEX　　　CON　　　INT　　　WIS　　　CHA

8（-1） 11（+0） 11（+0） 17（+3） 12（+1） 11（+0）

技能 奥秘+5，历史+5

感官 被动察觉11

语言 通用语，龙语，矮人语，精灵语

施法。 娜可塞纳是一位6级施法者。她的施法属性是智力（法术豁免 DC 13；法术攻击命中+5）。她准备有以下法师法术：

戏法（随意）：火焰箭 fire 
bolt，光亮术 light，法师之手 mage hand，魔法伎俩 prestidigitation

一环（4法术位）：法师护甲 mage 
armor，魔法飞弹 magic 
missile，护盾术 shield

二环（3法术位）：迷踪步 misty 
step，暗示术 suggestion

三环（3法术位）：飞行术 fly，闪电束 lightning 
bolt

动作

法杖。近战武器攻击：命中+1，触及5尺，一个目标。伤害：2（1d6-1）点钝击伤害，或双手使用时3（1d8-1）点钝击伤害。

扮演信息

金色原野的作物是深水城生存所必须的，这也是为什么安民法师会派遣娜可塞纳前来，以确保神殿农场拥有足够的防护力量。一开始她认为这是某种处罚，但是现在她非常珍重这里的和平与宁静。

理念：“没有什么问题是魔法不能解决的。”

牵绊：“我极其尊重深水城的莱拉·银手女士，她和领主联盟给那些无法之地带来了必要的秩序。”

缺陷：“我这么聪明，绝不会在任何事上犯错。”
