# 死拥水母

死拥水母 Death 
Embrance

死拥水母是一种在幽暗的溟渊中漂流的不祥恶兆。在它的躯体下方是一团散发着魔法的丝带般的卷须，边上则围绕着六条60尺长的带刺触须。死亡拥抱用这些触手擒抱并石化它的猎物。

尽管死拥水母有着不详的名字，但它并不是一种具有极端侵略性的生物。它更喜欢等待时机，并用触手捕捉的生物作为肉盾来抵挡攻击。

石化卷须 Petrifying 
Tendrils。任何在死拥水母的空间内开始回合的生物必须进行一次 DC 16的体质豁免，豁免失败则该生物被束缚。被束缚的生物必须在其下一回合结束时重复该豁免，豁免失败则被石化，豁免成功则束缚结束。石化状态将一直持续直到其被一道高等复原术 greater restoration 或类似的魔法结束。

卷缠 Reel。死拥水母可以在其回合开始时将每个被其擒抱的生物向它拉近至多20尺（不需要任何动作）

水中呼吸 Water 
Breathing。死拥水母只能在水下呼吸。

动作 Actions

多重攻击 Multiattack。死拥水母发动两次触须攻击

触须 Tentacle。近战武器攻击：命中+10，触及60尺，单一目标。命中：13（2d6+6）穿刺伤害外加11（2d10）心灵伤害。如果目标为大型或更小，它也会被擒抱（逃脱 DC 16）。死拥水母有六条触须，每一条都能擒抱一个目标。

反应 Reactions

肉盾 Body 
Shield。当死拥水母被一次攻击命中后，一个被其擒抱的生物（由死拥水母选择）代替它承受该伤害
