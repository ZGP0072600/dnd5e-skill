# 腐化巨鲨

腐化巨鲨 Corrupted Giant Shark

除了它恶毒的撕咬外，这种巨鲨还能投射出一种精神灵光，对周围生物造成痛苦的精神损伤。腐化巨鲨身上的晶体令其能够再生，只有光耀能量或毁灭性的打击才能压制这种非自然的防御。

心灵漩涡 Phychic 
Maelstrom。任何在巨鲨15尺内开始其回合的生物必须成功通过一次 DC 17的感知豁免否则受到11（2d10）心灵伤害。

再生 Regeneration。巨鲨在它的回合开始时回复10点生命值。如果巨鲨受到光耀伤害或一次重击，则该特质在它的下一回合开始时无法生效。巨鲨只在以0生命值开始其自己回合且无法再生时才会死亡。

水中呼吸 Water 
Breathing。巨鲨只能在水下呼吸。

动作 Actions

啃咬 Bite。近战武器攻击：命中+10（若目标生命值没有全满则本攻击检定具有优势），触及5尺，单一目标。命中：22（3d10+6）的穿刺伤害。若目标为一个生物，则其必须成功通过 DC 17的魅力豁免否则获得一级力竭。
