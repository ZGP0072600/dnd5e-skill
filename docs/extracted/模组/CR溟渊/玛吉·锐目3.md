# 玛吉·锐目3

玛吉·锐目 Maggie Keeneyes（3阶段）

大型巨人（食人魔），守序中立

护甲等级 15（板甲）

生命值 142 
(15d10+60)

速度 30尺，游泳30尺（月蚀护甲）

力量20 
(+4) 
敏捷14 (+2) 体质18 (+4) 智力14 (+2) 感知14 (+2)* 魅力8 (-1)

豁免力量+8，体质+7

伤害抗性 心灵（由月蚀护甲提供）

状态免疫 恐慌，魅惑

技能运动+6，洞悉+5，察觉+5，游说+5

感官黑暗视觉60尺，被动察觉 15

语言 通用语、巨人语

挑战等级 8 
(3,900 XP) 熟练加值+3

推撞 Push。每回合一次，当玛吉以一次攻击击中一个生物后，玛吉可以令其进行 DC 15的力量豁免，失败的情况下，目标被推水平离玛吉10尺并应击倒地。

战术准备 Tactical 
Readiness。只要玛吉没有处于失能状态，她和她周围30尺范围内的友方在投掷先攻时有优势。

动作 Actions

多重攻击 Multiattack。玛吉进行两次大号巨锤或飞锤攻击。

重型巨斧 HeavyGreataxe。近战武器攻击：命中+8，触及10尺，单一目标。命中：16（2d12+5）钝击伤害。

飞锤 HammerToss。远程武器攻击：命中+8，射程20/60尺。命中：15（4d4+4）钝击伤害。

附赠动作 Bonus Actions

振奋 Rally（1/天）。玛吉以一个她能看见的60尺内的生物为目标，以语言振奋其力量。目标获得30临时生命值。

*玛吉的感知不知为何从2阶段16变成了14
