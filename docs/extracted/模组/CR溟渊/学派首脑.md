# 学派首脑

学派首脑 Scholarly Mastermind

学派首脑是学术研究和组织的可敬领袖，他们负责计划和执行那些设计往日秘密的工作。一些学派首脑是研究院或大学的校长，而另一些则独自在暗中行动，执行一些可能和组织其他人意见相违的工作。
