# 蛇口蛮蟹

蛇口蛮蟹 Scuttling 
Serpenmaw

蛇口蛮蟹看起来像一只有着光滑黑色外壳的巨大螃蟹。它潜伏在裂缝和深海峡谷。一旦发现了猎物，它就会飞奔而来，并展现出它最具骇人的特征：一条覆有甲壳的蛇形口器。这种从壳中伸出扭曲副肢会把猎物撕成碎片。

虽然蛇口蛮蟹栖息在水中，但它们不会游泳。它们成群结队地沿着海床移动，蜂拥而上地撕碎那些更为巨大的生物。

两栖 Amphibiousness。蛇口蛮蟹可以在空气和水中呼吸。

集群战术 Pack 
Tactics。蛇口蛮蟹至少有一个未处于失能的盟友在目标生物5尺内时，它对该生物发动的攻击检定有优势。

动作 Actions

多重攻击 Multiattack。蛇口蛮蟹发动一次啃咬攻击和两次爪击攻击。

啃咬 Bite。近战武器攻击：命中+5，触及15尺，单一目标。命中：10（2d6+3）穿刺伤害，或当该攻击检定有优势时17（4d6+3）穿刺伤害。

爪击 Claw。近战武器攻击：命中+5，触及5尺，单一目标。命中：7（1d8+3）挥砍伤害。
