# 加萨利亚德·埃尔迪斯2

加萨利亚德·埃尔迪斯 Galsariad 
Ardyth（2阶段）

中型类人（卓尔、精灵、法师），绝对中立

护甲等级 12（法师护甲15）

生命值 67 
(15d8)

力量8 
(-1)

智力18 (+4) 感知15 (+2) 魅力11 (+1)

技能奥秘+7，调查+7，自然+7

妖精血统 Fey Ancestry。加萨利亚德在对抗魅惑的豁免检定中具有优势，并且魔法无法让他陷入睡眠。

动作 Actions

汲取潜能 Drain Potential。近战或远程法术攻击：命中+7，触及5尺或射程60尺，单一生物。命中：22(4d10)暗蚀伤害，且加萨利亚德获得5临时生命。

重力波 Gravity Wave（充能5-6）。加萨利亚德制造出一股魔法的重力波动，充满100尺内一处20尺半径球形区域。每个区域内的生物都必须进行 DC 15的力量豁免。失败的情况下，该生物受到17（5d6）力场伤害并倒地和被向球形中心拉入20尺。成功的情况下，生物受到一半伤害并且不会被移动或倒地。

施法 Spellcasting。加萨利亚德施展以下一个法术，以智力作为施法关键属性（法术豁免 DC 15）：

随意：法师护甲 magearmor，法师之手 magehand

每项1/天：大步奔行 longstrider，探知术 scrying（以一个动作），缓慢术 slow

附赠动作 Bonus Actions

扭转重力 Distort Gravity（1/天）。加萨利亚德以自身或60尺内他能看见的自愿生物为目标，以魔法扭转其周围的重力。任何位于目标5尺内的生物都会受到7（2d6）力场伤害。此外，目标能够以一个反应来上浮最高20尺，而不会触发借机攻击。该效应会在加萨利亚德下个回合开始结束，而目标会保持悬浮状态轻柔地回落20尺。
