# 哀恸鱼

哀恸鱼 Sorrowfish

哀恸鱼生活在水下的深沟里，水面上生物很少遇到哀恸鱼。有人说，哀恸鱼的名字来源于它们那令食客心生沮丧的味道。而当哀恸鱼群聚在一起时，其他生物才真正感到悲伤。潜伏在深海中的恶毒存在影响了这些鱼，把鱼群变成了邪恶的大群。

一条哀恸鱼有着食人鱼 quipper 的数据（见怪物图鉴），但它是一个异怪并免疫心灵伤害。

哀恸鱼集群 Swarm 
of Sorrowfish

哀恸鱼集群因饥饿而攻击。尽管他们不会恐惧，他们仍经常潜伏在死拥水母 death embrace 的石化卷须中，用这些卷须来防御其他捕食者。
