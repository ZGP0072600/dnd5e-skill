# 秘会入门者

秘会入门者 Occult Initiate

秘会入门者是最近才加入的组织成员，也是外人最常遇到的成员。

秘会入门者 Occult 
Initiate

中型或小型类人（术士），任意阵营

护甲等级 12（使用法师护甲后15）

(12d8 + 12)

(+0) 敏捷14 (+2) 体质12 (+1) 智力14 (+2) 感知10 (+0) 魅力16 (+3)

(2d6) 心灵伤害.

奥术冲击 Arcane 
Blast (充能 5~6)。入门者创造一个距离他120尺内可见的任意一点为中心的20尺球形范围内的魔法力量的爆炸。在此效应范围内的生物需要进行一次 DC 13的敏捷豁免。豁免失败的情况下，此生物受到10 
(3d6) 力场伤害，并被推离此区域中心10尺。豁免成功的情况下，受到一半伤害，且不会被推离。

施法 Spellcasting。入门者施展以下一个法术，无需施法材料，并使用魅力为施法主属性（法术豁免 DC 13）：

随意：光亮术 Light，法师之手 Mage Hand

每项1/天：侦测魔法 Detect Magic，浮空术 Levitate
