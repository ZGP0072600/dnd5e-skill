# 学派干员

学派干员 Scholarly Agent

学派干员是学术组织的低等级成员。他们有几手帮助他们避开危险的手段，也学习了帮助他们破解过去的秘密和鉴别魔法遗物的魔法。

学派干员 Scholarly Agent

中型或小型类人（法师），任意阵营

护甲等级 11（使用法师护甲后14）

生命值 22 
(4d8 + 4)

速度 30尺

力量10 
(+0) 敏捷12 (+1) 体质12 (+1) 智力16 (+3) 感知12 (+1) 魅力10 (+0)

动作 Actions

奥术电击 Arcane

3)闪电伤害。如果目标是一名生物，直到该生物下一回合开始时无法使用反应动作。
