# 噬光鮟鱇

噬光 cKLight 
Devourer

噬光 cK 因其从周围环境吸收明亮光线——无论其是自然光还是魔法光的能力而得名字。吸光之后，它会开始发光，而当 cK 将偷来的光芒转化为炽热的能量来撕裂猎物时，它那脊背上的尖刺和巨口中无数的尖牙会变得如同熊熊燃烧般明亮。

而在没有发光时，噬光 cK 潜伏在黑暗中，在海洋深处搜寻它的下一顿美餐。

Absoption。若噬光 cK 在其回合内经过一处处于明亮光照的空间或受到光耀伤害，它将在开始持续发光1小时，期间辐射出一道10尺半径的微光区域。

光能释放 Radiant 
Discharge。如果噬光 cK 正在发光（见光线吸收特性），它可以释放出体内储存的光能，覆盖以自身为中心的20尺半径球型空间并结束发光。每个位于该球形空间内的生物必须进行一次 DC 15的体质豁免，豁免失败则受到24（7d6）光耀伤害，豁免成功伤害减半。
