# 血鳍蛇鳗

血鳍蛇鳗 Slithering Bloodfin

生着着可怖的猩红色脊骨和覆盖体表的光滑鱼鳞的血鳍蛇鳗在溟渊那无光的海水中一闪而过。而当它游近后，那布满尖牙利齿的巨口便会出现在眼前，随后猎物便会被它狠狠咬住并囫囵吞下。

血鳍蛇鳗看起来就像一条有着鲨鱼般头部的巨大鳗鱼。它魔法性地从被吞下的猎物身上吸食生命能量，并用其修复自己受到的伤害。

当血鳍蛇鳗死亡时，它的身体会爆炸为一团毒性物质，持续威胁着周边生物直到被水流驱散。

死亡爆发 Death 
Burst。当血鳍蛇鳗死亡时，它爆炸成一团毒性云雾，任何在爆炸的血鳍蛇鳗10尺内的生物和被其吞咽的生物必须成功通过一次 DC 15的体质豁免否则受到10（3d6）毒素伤害。位于血鳍蛇鳗体内的生物在其爆炸时于血鳍先前占据的空间出现并倒地，且不再目盲和束缚

水中呼吸 Water 
Breathing。血鳍只能在水下呼吸。

动作 Actions

多重攻击 Multiattack。血鳍发动一次啃咬攻击和一次尾击攻击

啃咬 Bite。近战武器攻击：命中+9（若目标生命值没有全满则本攻击检定具有优势），触及5尺，单一目标。命中：14（2d8+5）穿刺伤害，且如果目标为一个生物，它也会被擒抱（逃脱 DC 15）。直到擒抱解除前目标被束缚，且血鳍不能啃咬另一目标。

尾击 Tail。近战武器攻击：命中+9（若目标生命值没有全满则本攻击检定具有优势），触及10尺，单一目标。命中：19（4d6+5）的钝击伤害。

附赠动作 Bonuse 
Actions

吞咽 Swallow。近战武器攻击：命中+9，触及5尺，单一被血鳍擒抱的中型或小型生物。命中：血鳍吞下目标。被吞下的目标不再被擒抱但陷入目盲且被束缚，其对于血鳍体外的攻击或效应视为处于全身掩护，其每次回合开始时受14（4d6）暗蚀伤害，且血鳍回复等同造成的暗蚀伤害的生命值。血鳍一次只能吞下一个生物。

如果血鳍蛇鳗在同一回合累计受到30点来自其体内的伤害，则其在该回合结束时必须成功通过一次 DC 15的体质豁免否则将吐出被其吞咽的生物，被吐出的生物出现在血鳍蛇鳗周围5尺内，并且不再陷入目盲和束缚状态。
