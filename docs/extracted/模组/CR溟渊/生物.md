# 生物

附录 A: 
生物 Creatures

这一附录提供了许多出现在这次冒险中的生物的资料卡和描述。你可以将它们用在你自己的冒险中——无论这些故事是发生在艾桑椎亚还是 D&D 多元宇宙中其他地方。

这些生物按照首字母顺序排列。

使用资料卡

如果你不熟悉怪物资料卡的格式，在更进一步前先阅读怪物手册的介绍部分。那本书解释了资料卡的术语并制定了多数怪物特性的规则（本书没有复述的信息）。

特殊攻击与法术

一些生物拥有造成特殊伤害类型的武器和使用非典型方式运作的施法能力。这种特例是一份资料卡的特殊组成部分，表现了该生物如何使用武器或施展法术；这种特例对其他生物的武器和法术不生效。

符合魔法物品的先决条件

如果一份资料卡中，在生物名称下方包含了职业的名称，该生物被视为该职业的一员，以符合魔法物品的先决条件。

冒险对手 Adventuring 
Rivals

下文详细介绍了以下几位非玩家角色、他们的性格和他们的动机。

艾优·贾贝是一名水元素裔，她是一位天生的领袖，她的兽人父母在海中强烈的风暴中诞下了她。

德莫特·沃尔德是一名地精祭司，他崇拜拉克桑，一位光明与重生之神。

加萨利亚德.埃迪斯是一名来自克灵王朝的卓尔法师研究并实践深奥的秘迹魔法。

埃尔文·荒行是一位青年男性，他的前世是一只克灵王朝的熊地精。在拉克桑的力量的作用下，他以一名人类游牧民的身份重生。

玛吉·锐目是一名食人魔战士，她才智过人，战术头脑与蛮力相得益彰。

对手阶段 Rival Tiers

就像玩家角色会在冒险的过程中升级并获得新的装备一样，对手们的能力也会提升。每一名对手均有3张资料卡。

对手的战斗台词 Rivals’ 
Battle Chatter

战斗台词表列出了对手有可能在战斗中对同伴或玩家角色说出的话。这些台词符合他们冒险早期的大致形象，随着他们的成长与改变，你可以编出新的台词。
