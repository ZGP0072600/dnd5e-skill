# 宗门院主

宗门院主 Monastic 
High Curator

院主们收集知识并传播真理。他们处理难解的谜团并监视其他权力机构的成员们以操纵他们或打破他们的平衡，并为被压迫者提供逃离困境的机会。院主们知晓他们组织最深处的秘密以及低级代理人的行动。他们的专业知识使他们能够操纵能量来保护自身并扰乱敌方能力的运作。

动作 Actions

多重攻击 Multiattack。院主进行3次力场打击并使用烧脑（如果可用）。

力场打击 Force 
Strike。近战或远程法术攻击：命中+10，触及5尺或射程10尺，单一目标。命中：17 (2d10+6)力场伤害。

烧脑 Brain 
Burn（充能4-6）。院主以自身周围30尺范围内最多两个可见生物为目标。每个目标均需进行一次 DC 18的体质豁免。豁免失败的情况下，目标受到28 (8d6) 
心灵伤害且直到他的下一回合开始时不能执行反应动作。豁免成功的情况下，目标受到一半伤害且不受其他效应影响。

随意：侦测魔法 detect 
magic
