# 秘会银舌

秘会银舌 Occult Silvertongue

秘会银舌是极富魅力的秘会领袖们。他们接触的异界神秘令他们陷入寝食难安、偏执多疑的境地。他们开始看见并不存在之物，并臆想他人谋划着要将自己取而代之。

Mind。当一名生物试图读取银舌的想法或对银舌造成心灵伤害时，该生物需进行一次 DC 16的智力豁免，否则直到他的下一回合开始将处于震慑 stunned 状态。

每项1/天：任意门 dimension door

附赠动作 Bonus Actions

元素侍从 Elemental 
Servitor（1/天）。银舌使用魔法召唤出一个土元素 earth elemental，出现在一处距银舌自身60尺范围内可见的未被占据的空间中。土元素使用银舌的先攻并在银舌的回合结束后立即行动。土元素服从银舌的命令。土元素于1分钟后或银舌死亡后消失。
