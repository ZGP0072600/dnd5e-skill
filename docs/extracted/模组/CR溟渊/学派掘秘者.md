# 学派掘秘者

学派掘秘者 Scholarly Excavator

实地考古研究常常是一项危险的工作，尤其是涉及古代魔法和怪物丛生的废墟时。学派掘秘者受过专业训练，懂得如何对付这些危险、保护自己的盟友和清理前路的障碍。
