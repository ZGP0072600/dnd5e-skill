# 艾优·贾贝

艾优·贾贝 Ayo Jabe（1阶段）

中型类人（游侠、水元素裔），混乱善良

护甲等级 14（镶嵌皮甲）

生命值 65 
(10d8+20)

速度 30尺，游泳30尺

力量14 
(+2) 敏捷14 (+2) 体质14 (+2) 智力10 (+0) 感知12 (+1) 魅力12 (+1)

豁免力量+4，敏捷+4

技能运动+4，察觉+3，生存+3

伤害抗性 强酸

感官 被动察觉 13

语言 通用语、地精语、兽人语

挑战等级 3 
(700 XP) 熟练加值+2

水陆两栖 Amphibious。艾优能在空气中和水下呼吸。

动作 Actions

多重攻击 Multiattack。艾优使用鱼叉或长弓进行两次攻击。

鱼叉 Harpoon。近战武器攻击：命中+4，触及5尺，单一目标。命中：11（2d8+2）穿刺伤害。

长弓 Longbow。远程武器攻击：命中+4，射程150/600尺，单一目标。命中：11（2d8+2）穿
刺伤害。

每项1/天：化兽为友 animal friendship，神莓术 goodberry
