# 宗门渗透者

宗门渗透者 Monastic 
Infiltrator

渗透者们担任间谍、蓄意破坏者和其他秘密行动中执行者的角色。他们研习魔法的技艺来强化他们战斗中的能力，并帮助他们更好的完成自己的任务。
