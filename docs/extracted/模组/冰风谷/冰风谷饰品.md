# 冰风谷饰品

**附录 A：冰风谷饰品 Icewind Dale Trinkets**

**d100 饰品 **

01-04 
一个大张着嘴巴的海狮的木制小雕塑，被漆成了红黑色。

05-08 一对贝壳雕刻而成的袖口，上面刻着一位渔夫坐在船上的图案。

09-12 
一支小小的铁钥匙，磨损了的蓝金线系在上面。

13-16 一小本有关北地人神话的解说书，有几页丢失了。

17-20 
一枚受损的贝壳浮雕，刻画着一只人鱼。

21-24 一块从墓穴石冢里得到的石头，一个小小的矮人符文刻在上面。

25-28 
一块被撕裂的布料，上面印着你认不出来的图案。

29-32 一把有着贝壳把手的乌鲁刀。

33-36 
一个罐子，装着甜而粘稠的无法辨认的物质。

37-40 一枚涂着雪花的易碎的玻璃球，顶端装着一个带着小钩子的金属环。

41-44 
一本少了几页的冒险日志，一朵被压扁的花被夹在里面作为书签。

45-48 一个用鲸骨雕刻成的猫头鹰雕塑。

49-52 
一个带着旧木气息的缝纫盒，三卷蓝色丝线放在里面。

53-56 一支贝壳柄的钢笔，笔身上有黑色符文的图案。

57-60 
一枚用虫珀制成的胸针。

61-64 一个贝壳质胡椒粉瓶，蚀刻着字母 W。

65-68 一个老旧的木柄冰凿，浸染着洗不掉的血迹。

69-72 
一个布织洋娃娃，有着愤怒的表情。

73-76 一套由海贝做成的风铃。

77-80 一个漂亮的银罐头，打开时会发出腐烂的鱼的气味。

81-84 
一个由渔线，金丝和雪地枭熊的羽毛制成的捕梦网，染着血迹。

85-88 一尊用永不融化的寒冰雕刻成的北极熊小塑像。

89-92 
一颗无需摇动的雪景球。

93-96 一小片海玻璃，形状像独角兽的角。

97-00 
深蓝色围巾，佩戴者所处的海拔越高，围巾颜色越浅。
