# 硬头鳟

**硬头鳟 Knucklehead Trout**

美味而顽强的硬头鳟不能轻易被网捕捉。而且，用一根线把这么强壮的鱼钓上来是一项重大的任务。粗心的渔民一旦被拖入冰冷的水中，很快就会丧命，尤其是当他们被厚重的毛皮和斗篷压得很重的时候。

一只雄性硬头鳟的体重可以超过70磅。雌性的体型往往较小，重约50磅。它们都因其象牙状的骨骼而备受赞誉。

**水下呼吸 Water Breathing。** 硬头鳟只能在水下呼吸。

**动作 Actions:啃咬 bite。** 近战武器攻击，命中+4，触及5尺，单一目标。命中：4(1d4+2)穿刺伤害。

**尾击 Tail。** 近战武器攻击，命中+4，触及5尺，单一目标。命中：4(1d4+2)钝击伤害。
