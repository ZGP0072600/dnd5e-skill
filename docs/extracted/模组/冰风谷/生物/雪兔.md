# 雪兔

**雪兔 Hare**

北地雪兔是一种温和的食草动物，它们的身影遍布全冰风谷。它们比起其他野兔来耳朵更短，且它们适应寒冷天气。

**雪兔 Hare** *微型野兽 无阵营*

AC 13

HP 
1(1d4-1)

速度20尺，掘地5尺

力量1(-5) 敏捷17(+3) 体质9(-1)

智力2(-4) 感知11(+0) 
魅力4(-3)

技能:察觉+2，隐匿+5

感官:被动察觉12

语言:--
