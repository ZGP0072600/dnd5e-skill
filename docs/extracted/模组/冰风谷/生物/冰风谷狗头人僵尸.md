# 冰风谷狗头人僵尸

**冰风谷狗头人丧尸 Icewind Kobold zombie**

*小型亡灵，中立邪恶*

护甲等级：9（破损的皮甲）

生命值：19（3d69），

速度20尺

力量8（-1） 
敏捷6（-2） 
体质16（+3）

智力3（-4） 
感知6（-2） 
魅力3（-4）

豁免：感知+0

伤害免疫：毒素

状态免疫：中毒

感官：黑暗视觉60尺，被动察觉8

语言：理解生前掌握的语言，但不会说

挑战等级：1/8（25 
XP）

**不死坚韧 Undead Fortitude。** 当丧尸因受非光耀非重击伤害而使其生命值减至0时，它必须进行一次 DC 为5＋所受伤害值的体质豁免，豁免成功则生命值降至1。

亡灵本质：狗头人丧尸无需要空气，饮食以及睡眠。

**动作 **

** 标枪 javelin。**远程武器攻击：命中+0，触及5尺或射程30/120尺，单一目标。伤害：2（1d6-1）的钝击伤害。
