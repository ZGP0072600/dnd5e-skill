# 雪人淘气鬼

**雪怪 Yeti**

雪怪和巨恶雪怪已经在怪物图鉴中有详细的描述了，这个条目为人们很少遇到的雪怪类型提供游戏信息。成年雪怪非常注重保护它们的幼崽，使它们不被外界所知。雪怪淘气鬼一般是年龄处于1到7岁的幼年雪怪。

雪怪淘气鬼同他们的父母一样反复无常和卑鄙，他们的恐惧感只因他们的体型而减弱。它们站立时平均身高3英尺，喜欢欺负比它们身高矮或更小的生物。但它们很容易被更强大的生物吓倒。一只被吓坏的雪怪淘气鬼至少在一段时间内可以被控制，但是把一只雪怪幼崽养大成为一只野蛮的食肉动物是非常困难的（虽然不是不可能的）。
