# 青铜龙茹娜拉

**青铜龙 Bronze Dragons**

许多青铜龙都是友好的沿海居民，喜欢观看忙碌的船只与水手们。

**茹娜拉 Runara。** 尽管她的伪装和对和平的献身，巨龙休息处的领导人其实是一条成年青铜龙——可怕的敌人或可靠的盟友。她的能力比下属更加强大。

**水陆两栖 Amphibious。** 茹娜拉可以在空气和水中呼吸。

**传奇抗性 LegendaryResistance（3/日）。** 茹娜拉豁免失败时，可以将其改为豁免成功。

**动作 Actions 多重攻击 Multiattack。** 茹娜拉发动一次啃咬和两次爪击。

**啃咬 Bite。** 近战武器攻击：命中+12，触及 
10 尺，单一目标。命中：18（2d10+7）点穿刺伤害。

**爪击 Claw。** 近战武器攻击：命中+12，触及 5 
尺，单一目标。命中：14（2d6+7）点挥砍伤害。

**吐息武器 Breath Weapon（充能5~6）。** 茹娜拉可以使用下列吐息武器之一：

**闪电吐息 Lightning Breath。** 茹娜拉呼出闪电并覆盖 90 尺长、5 尺宽的线状区域。该区域内的每个生物都必须进行一次 DC 19 的敏捷豁免，豁免失败者将受到 
66（12d10）点闪电伤害，豁免成功则伤害减半。

**斥力吐息 Repulsion Breath。** 茹娜拉呼出斥力能量并覆盖 
30尺的锥状区域。该区域内的每个生物都必须进行一次 DC 19的力量豁免检定。豁免失败则被向龙所在的方向推后 60 
尺。变形 ChangeShape。青铜龙可以用魔法变形为类人生物或野兽，并保持她的游戏数据（除了体型）。这种变形持续到茹娜拉的生命值降为 0 
或她用一个附赠动作解除。
