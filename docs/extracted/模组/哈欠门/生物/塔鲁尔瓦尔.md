# 塔鲁尔·瓦尔 Tarul Var

在早年为红袍法师进行的任务失败后，巫妖塔鲁尔·瓦尔隐居于《死于赛尔》的末日穹顶，以躲避萨扎斯坦 Szass 
Tam。闯入他居所的人将被巫妖及其手下的恐惧战士收押，但如果能将瓦尔逼至绝境，他则会设法逃离地下城。

**亡灵本质 Undead Nature。** 巫妖不需要空气、饮食和睡眠。
