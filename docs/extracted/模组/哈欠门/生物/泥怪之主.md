# 泥怪之主 Ooze Master

《死于赛尔》的末日穹顶中有处名为不朽洞穴 The Immortal Caverns 
的地方，这里是许多奇异生物的家园，而其中最为奇特的当属泥怪之主。它一开始以一种粘稠的红色软泥状物质构成的柱状体形象出现。而实际上这个生物藏在柱子里面，一旦打扰到它，它就会带着邪恶的意图苏醒。

**亡灵本质 Undead Nature。** 泥怪之主不需要呼吸、饮食和睡眠。
