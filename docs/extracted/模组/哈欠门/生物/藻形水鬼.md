# 藻形水鬼 Kelpie

《白羽山》的藻形水鬼是一种有意识的水生植物，外观像一团潮湿的海藻。它能把身体塑造成各种各样的形状，通常会选择变成一个美貌的类人生物，以吸引人们进入深水。
