# 恐惧战士 Dread Warrior

《死于赛尔》中的红袍法师制作了许多种亡灵爪牙，其中一只便是被称为恐惧战士的特殊奴仆。在举行一个秘密仪式之后，红袍法师可以更为得心应手地控制恐惧战士。通过精神链接，红袍法师能够代入恐惧战士的感官，并直接操控它。

**亡灵本质 Undead Nature。** 恐惧战士不需要空气、饮食和睡眠。
