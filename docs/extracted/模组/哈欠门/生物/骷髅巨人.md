# 骷髅巨人 Giant Skeleton

在《恐怖墓穴》中，有时候宝藏就大摇大摆地任人索取。而有时候，大发横财的结果就是遭遇一具巨人的骸骨。其危险性不仅仅在于它的体型和力量，更是因为它所拥有的强大的死灵之力。

**亡灵本质 Undead Nature。** 骷髅不需要空气、饮食和睡眠。
