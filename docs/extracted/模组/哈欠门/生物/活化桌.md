# 活化桌 Animated Table

在《愤怒熔炉》中出场的部分堡垒守卫是矮人法师阿伦迪尔 Arundil 当年创造的活化物件 animated 
objects。这其中包括了活化桌。

**构装本质 Constructed Nature。** 活化桌不需要空气、饮食和睡眠。
