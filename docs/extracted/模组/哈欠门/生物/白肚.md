# 白肚 White Maw

《死于赛尔》的红袍法师在位于末日穹顶的实验室中进行了大量泥怪实验。被命名为白肚的生物是他们的早期成果之一，这是一只体型庞大，具有智能和沟通能力的灰软泥。只不过这些天赋已经被其完全扭曲的心智所掩盖。

泥怪本质 
Ooze Nature。白肚不需要睡眠。
