# 塞壬 Siren

塞壬被囚禁在恐怖墓穴中一件雾气弥漫的房间里，以显示阿瑟瑞拉克 Acererak 
的冷酷无情。这只神秘的妖精渴望摆脱牢狱之苦，但半巫妖施加在她身上的魔法却令其无法向来访者袒露释放她的办法。如果有人能成功解救她，将赢得她终身不渝的友谊。
