# 大丧尸 Greater Zombie

许多勇敢探索恐怖墓穴的勇者认为，在某个犄角旮旯里碰上一只特大号的骷髅已经够离谱了。但事实上，比那更离谱的是一只无比强壮的丧尸，在类人生物尸体上施展魔法创造出来的这个怪物远比常见的丧尸更为坚韧。

**亡灵本质 Undead Nature。** 丧尸不需要空气、饮食和睡眠。
