# 巨冰蟾 Giant Ice Toad

在《挑战巨人》中冰风谷 Glacial Rift 
的一个洞穴里，一群巨冰蟾坚决地守卫着它们的领地。这种生物自发地散出寒气来折磨不速之客，而不幸被其吞咽下肚的家伙也会被胃液冻伤。
