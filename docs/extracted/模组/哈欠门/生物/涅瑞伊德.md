# 涅瑞伊德 Nereid

云雾之国的失落神殿中最美的场景是一位美人坐在散发着淡淡光彩的水池边低吟浅唱。这是一个涅瑞伊德，一种可以依照需求改变周边环境的水中妖精。

无论男女，他们都拥有一副超凡脱俗的美貌。
