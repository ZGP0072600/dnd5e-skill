# 超自然赠礼

**不可能之眼 Eyes of the Impossible**

*超自然赠礼（护咒）*

你因拥有不该知晓的知识而留下了疤痕。你的眼睛变黑，只有瞳孔变成白色。这种护咒使你在与邪恶生物互动时进行的魅力（游说）检定上具有优势。此外，你可以使用一个动作施展法术 
*摄心目光 eyebite*。你施放此法术时使用的施法属性为智力、感知或魅力（选择此护咒时决定）。一旦你施放了此法术，直到次日黎明之前你无法再次施放此法术。这个护咒会在13天后消失，你的外貌恢复正常。

**龙之时光 Time of the Dragon**

*超自然赠礼（护咒）*

当你握持这片鳞片时，你可以以一个动作召唤雷尼斯努普拉，一条 **成年时光龙 adult time dragon**（见《莫提的位面游记》）。这只龙会出现在你能看到的60尺内的一处未被占据的空间处，并在10分钟内帮助你。之后，时光龙消失，鳞片也会破碎。
