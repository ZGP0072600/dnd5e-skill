# 发条观察者

**发条观察者 Clockwork Observer**

发条观察者为其创造者充当空中探子。它看起来像一个机械的、柚子大小的球体，悬浮在轻声嗡鸣的螺旋桨叶片下。球体中嵌入了敏锐的水晶眼，使观察者能够同时看到多个方向。当它察觉到一些令人不安的东西时，观察者会发出一声在300英尺范围内都能听到的尖叫。观察者还可以通过心灵感应将它所发现的东西转达给它的创造者，只要两者相距1英里以内。

*微型构装，无阵营*

**飞掠 Flyby。** 观察者飞出敌人的触及范围时不会引起借机攻击。

**心灵联结 Telepathic Bond。** 只要观察者与其创造者之间的距离不超过1里，他就可以魔法性的将它所看到的东西传达给它的创造者，且两者之间可以以心灵感应的方式进行交流。

**非凡本质 Unusual Nature。** 观察者不需要吃、喝、睡眠、以及呼吸。

**动作 Actions**

** 尖叫 Shriek。**观察者发出机械性的尖叫声，直到其下一回合开始或生命值降至0位置。这声尖叫声可以在300尺范围内听到。
