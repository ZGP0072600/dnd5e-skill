# 破碎孤石

**破碎孤石 Shard Solitaire**

*奇物，传说（需同调）*

这颗宝石中含有一个不稳定的异次元裂缝。它的切面上有条纹般的彩虹色纹路，似乎能够自行移动。如破碎孤石类型表所示，已知存在五种这样的魔法宝石，每一种都是不同的类型的宝石。

**裂隙穿梭 Rift Step。** 当穿戴或手持这颗宝石时，以一个附赠动作，你可以将你自己和穿戴与携带的一切传送到30尺内你可以看到的一点。

当你使用这项属性时，你可以选择使用宝石内部的异次元裂隙的力量来增加最多30尺传送距离，但如果你传送到超过30尺距离外时，你必须成功通过 DC 16的体质豁免否则在传送后立即受到3d10力场伤害。

**施法 Spellcasting。** 这颗宝石具有6充能，并在每天黎明时恢复1d6已消耗的充能。以一个动作，你可以消耗对应数量的充能施展宝石对应的法术，而无须任何材料成分（豁免 DC 16）。
