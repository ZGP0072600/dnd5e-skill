# 蝠人

蝠化症是一种罕见的类人生物兽化症状，它将类人生物转化为半人半蝠的怪物和巨大的蝙蝠。（更多关于兽化症的信息请参阅怪物手册）绝大多数蝠化症发生在地精族群中。 

**黑暗居民。** 蝠人因其饮血维生的食性而被地精群体所排斥。他们偏好在黑暗的区域（例如阁楼和洞穴中）筑巢，并在夜间狩猎。 

**饮血者。** 蝠人每晚需要摄入至少1品脱的鲜血，否则就会变得虚弱并获得 1级无法通过长休恢复的力竭。每一品脱鲜血将会恢复 
1级因此方式累计的力竭等级。 

**蝠化症。** 当一个玩家角色感染蝠化症，该角色敏捷若然未达到17 
则提升至17。其啃咬攻击的攻击检定及伤害投骰基于其力量或敏捷属性中较高值。
