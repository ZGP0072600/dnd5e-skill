# 金属魔蝎

金属魔蝎是由海拉斯特的学徒特洛布里安创造的构装生物。它们像巨型蝎子一样移动和攻击，用两只大钳子抓住猎物，并用金属毒刺尾巴发出致命的闪电。 

特洛布里安的戒指。特洛布里安创造了一系列魔法戒指来控制金属魔蝎。戒指是黑色金属材质，上有毒刺形状的装饰。金属魔蝎无法攻击这些戒指的佩戴者，且必须服从 
100尺内戒指佩戴者的口头命令，即便佩戴戒指的生物没有语言能力，戒指上的魔法也能使金属魔蝎理解佩戴者的旨意。如果一个金属魔蝎接收到两个戒指佩戴者向矛盾的指令，它会立即停止工作并失能1小时。特洛布里安有一枚主控戒指，可以在不使金属魔蝎失效的情况下覆盖其他使用者的指令。特洛布里安创造的戒指只在地脉迷城中生效。 

**构装本质。** 金属魔蝎不需要呼吸、食物、水和睡眠。
