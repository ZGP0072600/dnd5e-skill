# 魔威鲁

魔威鲁曾是一位杰出的人类战士，他长期担任疯法师海拉斯特的保镖。当他向海拉斯特要求学习魔法后，他便步步陷入疯狂之中。魔威鲁掌握了能将自己变成半蝎半人怪兽的魔法，并以“畸形者魔威鲁”为世人所知。

当他卸任海拉斯特的保镖之职后，他来到了地脉迷城中日后被称为“魔威鲁护手”的地城中，驱逐和杀死了此地原本的卓尔精灵居民。现如今，魔威鲁以杀戮冒险家和其他入侵者为乐。漫长的孤独岁月，以及海拉斯特的影响，使他完全发疯。
