# 海拉斯特·黑袍

海拉斯特，地脉迷城的疯法师，是在深水城的大地下城中所发现的大多数布下的陷阱和恐怖幕后的疯狂存在。地脉迷城就是他的家，一个娱乐的长廊，其他人在里面表演以取悦他。这个疯法师比任何人都清楚地脉迷城中那些多变的通路，因为他是控制这些变化的人。

他更喜欢让别人注意不到他，偷偷摸摸地四处走动，或者透过被闪烁的光点包围的像瞪大的眼睛一样的探测装置窥视。海拉斯特的能力远远超过了世上大多数法师。

他在魔法门方面的学识使他能够周游各地从事魔法研究。他花了大量的时间来制造魔法门，穿梭于它们之间，并在上面施放古老法印。海拉斯特的大门连接着地脉迷城的不同层，这样他就可以把新的怪物带到地城去补充那些死亡或逃跑的怪物。即使一群冒险者试图获得地脉迷城中的一小部分回廊的决定性控制权，哈拉斯特也会不断改变地下城的难度来挫败他们。 

海拉斯特的真实形态是一个高大而又憔悴的男性，但他使用魔法让自己展现许多其他的面貌和形态。但不管他出现时的样貌如何，这个疯狂的法师总是不停地傻笑和呢喃。然而，与他的表现相悖的是，海拉斯特对他周围的一切生物的行为和预谋都很警觉。他绝不愿意在没有先施放法师护甲的情况下进入战斗。
