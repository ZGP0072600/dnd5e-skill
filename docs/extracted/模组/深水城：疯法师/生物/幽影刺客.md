# 幽影刺客

幽影刺客形同亡灵：幽影（见 MM 描述）。他们手持由影子构成的短剑。幽影刺客的存在即是为了杀死活物。 

**亡灵本质。** 幽影刺客不需要呼吸、食物、水和睡眠。
