# 熔岩之子

熔岩之子的外形完全相同，他们有着健硕的肌体与童稚的，永远咧着嘴笑的面容。他们大多诞生于“创造之泉”（“岩浆位面”的别称），一处夹在土元素位面与火元素位面之间的侧元素位面。熔岩之子在火山岩洞中组成族群，他们服侍那些因对土地与火焰的恐惧与崇拜而诞生的远古存在。 

尽管他们天性并不暴力，但熔岩之子会激烈地捍卫他们自封的领土。许多入侵者被熔岩孩子的微笑所欺骗，以为自己会受到欢迎，但事实上熔岩之子正打算将他们的四肢扯下。熔岩儿童通常在其巢穴附近活动，但在食物变得稀缺时也会主动狩猎。 

熔岩降生。第一只熔岩之子由大地与火焰精魂融合而成。成年后，熔岩之子可以自行繁殖。一个成年的熔岩儿童在其生命周期（大约五十年）内会产下几个卵。

这些卵在熔浆池中孵化直至孵化。粉嫩的、带着笑脸的新生儿的成熟速度与人类相同。熔岩之子会保护自己的后代，直到其成长到足以自卫并独立狩猎。熔岩之子吃肉、骨头、纸、植物以及几乎所有其他非金属和非石制的东西。其中熟肉是他们的最爱。
