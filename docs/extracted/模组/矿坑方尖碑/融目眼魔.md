# 融目眼魔

融目眼魔 Oculorb 许多融目眼魔都诞生自真眼魔的梦境。遥远国度的诡异力量能渗入眼魔的梦中，并导致融目眼魔的诞生。融目眼魔就像一团由大量大小、形态皆不相同的眼球融合而成的黏糊团块，即使是“梦出”它们的那个眼魔也会被这可怖之物吓跑。融目眼魔是一个负面情绪——其创造者的愤怒，忧心和痴狂——的聚合体。融目眼魔能像真眼魔一样从眼中射出能量射线，但却缺乏真眼魔视线中精细的魔法本质。作为替代，融目眼魔能释放出强大的负面情绪能量波，用恼怒与绝望驱离敌人。
