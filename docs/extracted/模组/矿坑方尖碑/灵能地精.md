# 灵能地精

灵能地精 Psionic Goblins 灵能地精起源众多。一些是源自遥远国度泄露出的能量带来的天生影响，另一些则是依靠灵能力量改造自身，或是与其他异怪缔约，以为其充当突击部队作为提供改造的代价。无论如何，最终的产物都是相同的：有着并非自然且几乎不受控的心灵异能的地精。灵能地精很难掌控其身心中涌动的心灵能量。那些学会如何安全掌控这种心灵异能的地精在战斗中异常强大。灵能地精经常靠心灵感应来增强军事技艺，由灵能地精武者组成的潜伏小队依靠心灵感应互相交流，这使其成为了绝佳的渗透者与伏击者。

地精灵斗士 Goblin Psi Brawler 地精灵斗士用其心灵异能来增强躯体力量。它们的打击注入了心灵能量，而在被激怒时，地精灵斗士也能释放出足以把敌人击倒在地的念力冲击。

地精控灵师 Goblin Psi Commander 地精控灵师是为数不多能完全掌控其心灵力量的灵能地精。彻底参透了其灵能力量的地精控灵师挥舞着完全由心灵能量凝成的利刃。它们能投射出念力组成的屏障，同时还能靠撕裂心灵的迸发来击溃敌人。
