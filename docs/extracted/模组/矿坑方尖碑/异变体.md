# 异变体

异变体 Mutates 暴露在遥远国度的奇异能量之下的生物有着发生异变的风险。遥远国度的力量会覆写被异变者的存在本质，令其经历彻底的转变。任何生物都能成为异变体，这会使其身体结构展现出与遥远国度相关的特征。一些生物成为异变体是因长时间暴露于遥远国度的力量之下，或是受到从那个诡异位面汲取而来的魔法的影响。有些生物刻意改变自己，以迎合他们所追求的堕落力量。而在遥远国度的影响衰退之后，化为异变体的生物往往会以难以理解的方式变回先前的形态。尽管许多邪恶势力比如曾经的夺心魔帝国进行了大量相关研究，但强迫生物进行转变的实验总是不可理解地失败。数个世纪以来，夺心魔一直在尝试制造一支由异变体组成的军队，以此横扫幽暗地域。然而，他们的努力总是被证明是无效的，最终夺心魔放弃了这项计划。有几种异变体较为常见。本段中带有资料卡的是在前文的冒险中会出现的类型，而其他类型的异变体也是存在的。一些生物更容易成为遥远国度异变体，例如以下几页中所展示的蛰伏伪怪和食腐兽。异变体通常会获得灵能，生出翅膀，被异界的黏液覆盖，或者表现出其他令人不安的特征。具体的异变体生物将在下面的段落中进一步描述。

蛰伏伪怪异变体 Cloaker 
Mutate 遥远国度的有害能量令蛰伏伪怪与其上一餐的腐尸融合，形成了一个丑陋的，如同在表演拙劣木偶剧的异怪。蛰伏伪怪异变体能靠一团由幻象分身构成的瘴气来遮掩行动，还能靠融在身下的尸体来抵御伤害并攻击猎物

类人异变体 Humanoid 
Mutate 被遥远国度影响的类人生物可能会化为一个噩梦般的存在。大量膜翼在异变体的身后肆意生出。它的躯体会覆上粘液，质感变得如蝾螈般顺滑。一些聪慧的类人异变体保留了它们暴露在遥远国度影响之前的人格。另一些则被异界力量所引诱，依靠这种转变成为恶毒的先知和预言家。此外，还有一些异变体找到了变回原先形态的魔法手段。

食腐兽异变体 Otyugh 
Mutate 食腐兽异变体从暴露在遥远国度能量中的成堆腐肉和污物中诞生。它的肢体上生长着漆黑的几丁质甲壳，与其幽灵般的半透明血肉形成鲜明的对比。同时，半透明的血肉也为想观察食腐兽今天吃了什么的人提供了这样做的诡异途径。
