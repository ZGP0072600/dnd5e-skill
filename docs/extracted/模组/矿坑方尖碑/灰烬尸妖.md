# 灰烬尸妖

灰烬尸妖 Ashenwight 惨死于受遥远国度所腐化之地的类人生物可能会复生为一只灰烬尸妖。这些骇人不死者皮肤枯槁，眼瞳中时常闪烁着异界力量。

凶残灰烬尸妖 Feral Ashenwight 凶残灰烬尸妖保留了一部分生前形象的残片。然而其心中已不再有着过去的记忆与思想，其唯一的念头便是毁灭任何遇到的活物。

灵能灰烬尸妖 Psionic Ashenwight 当灰烬尸妖在被注入了畸变力量的晶簇脉络附近复生，或是独自徘徊在充盈着堕落魔法之地时，灰烬尸妖便有机会觉醒灵能力量。灵能灰烬尸妖比其他同类更具智慧，尽管其心智乃是后来形成而非继承自生前原主。 一些灵能尸妖致力于拼凑自己生前的痕迹，而另一些则致力于让其他灰烬尸妖也得到这份灵能力量。
