# 邪魔钻机

邪魔钻机 Fiendish 
Auger 邪魔钻机会在一个恶毒的灵魄进入一台掘进钻机时产生，其巨大无比的螺旋钻头上闪着地狱火的亮光。依靠其内邪恶灵魄狂热的侵略性助燃，邪魔钻机可以飞快地开凿岩石。然而想利用邪魔钻机的人必须时刻紧盯这台机器。要是没有准确的指令，邪魔钻机会不加区分地钻开活物和大地。
