# 髓脑体

髓脑体 Encephalons 当一个夺心魔社群危险地将卵产在靠近遥远国度裂隙处之时，卵囊就有可能异变为一种被称为髓脑聚簇体的事物——一种饥饿的，遍布粘液的，具有灵能力量的，形似大脑的团簇，并能产下名为髓脑芽生体的怖人之物。

髓脑聚簇体 Encephalon Cluster 髓脑聚簇体不会去理会主脑或是其夺心魔亲属。这种奇诡的生物在黑暗荒废的洞穴中蔓行，产出扭曲的后代来繁殖。因此，许多夺心魔将髓脑聚簇体视为潜藏的病毒。如不加以控制，一个髓脑聚簇体很快就能形成一道疯狂进食的威胁，甚至足以摧毁整个夺心魔社群。

髓脑芽生体 Encephalon Gemmule 由于遥远国度的影响，髓脑聚簇体不再产出夺心魔蝌蚪。作为替代，当髓脑聚簇体的卵成熟之时，它便会从中脱离，形成一个髓脑芽生体。初生之时的髓脑芽生体看起来像是一个生着细腿的，形似走形的金字塔般的软块。若是任其自行生长，不需要多少日子它就能生长成一个髓脑聚簇体，产出它自己的髓脑芽生体。
