# 畸变狂热者

畸变狂热者 Aberrant 
Zealot 畸变狂热者是为了力量而勾结遥远国度的邪教徒。畸变狂热者的身躯充盈着诡异的能量，其凶残的伟力借自那些居于堕落位面的怖人存在。
