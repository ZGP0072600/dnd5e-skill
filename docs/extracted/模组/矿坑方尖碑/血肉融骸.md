# 血肉融骸

血肉融骸 Flesh 
Meld 血肉融骸是由尖牙，鲜血还有尸块融合而成的巨大嵌合体，因贪食饥饿而颤抖着。其众多的秽恶附肢渗着鲜血，末端的遍布利齿的巨口会咬住猎物，将其吞入血肉融骸的核心。致死的恶臭环绕着血肉融骸。被血肉融骸吞噬殆尽的生物，其心智会融入血肉融骸的意志之中。以些更坚韧的心智也许能抵御同化，但却不可避免，最终屈服于不谐的思想片段，随之杀戮，吞噬。

Creature。血肉融骸指定一个其5尺内的大型或更小且被其擒抱的生物。目标必须成功通过一次 DC 15的敏捷豁免否则被血肉融骸吞下。血肉融骸一次能吞下一个生物。被吞下的目标不再被擒抱但陷入目盲且被束缚，其对于血肉融骸体外的攻击或效应视为处于全身掩护，每次该血肉融骸回合开始时，被吞下的生物受到10（3d6）暗蚀伤害。若该伤害将被吞下的生物降至0生命值，则该生物死亡，且尸体会被血肉融骸吞噬殆尽。若血肉融骸在同一回合累计受到30点来自一个被吞下的生物的伤害，则其在该回合结束时必须成功通过一次 DC 15的体质豁免否则将吐出被其吞咽的生物，并使其摔下倒地于血肉融骸周边5尺范围内。若血肉融骸死亡，则被吞下的生物将不再受束缚，且可以消耗10尺移动力逃出，然后在尸体旁现身并陷入倒地状态。
