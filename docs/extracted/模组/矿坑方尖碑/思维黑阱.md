# 思维黑阱

思维黑阱 Intellect 
Snare 当自遥远国度而来的诡异恐怖撕碎一个心灵时，一些失控的思维残片会聚合在一起，化为一个思维黑阱。思维黑阱如同一个长着触须的扭曲球体，周身回荡着被其吞噬的心灵发出的嘈杂声响。思维黑阱也是活跃在夺心魔所肆虐之处的清理者，它们会吞噬不幸的生还者。思维黑陷会用一条触须缠住猎物，并从中抽取思维残片为食，留下一个破碎的心灵。
