# 夺心魔

夺心魔 Mind Flayer 夺心魔，又称灵吸怪，在多元宇宙中以的类人生物大脑为食。其显著特征包括紫色调的皮肤，章鱼般的头部和扭曲伸展的触须。

夺心魔洞见者 Mind Flayer Clairvoyant 一部分夺心魔追求着重构其失落帝国，为了寻求解答，它们回归了其家园位面，遥远国度。 夺心魔洞见者便是长久以来一直观察这国度内无光的深空，并从中得到了异常收获的存在。夺心魔洞见者不再服从主脑，而是倾听遥远国度所传来的低语与奇想。比起直接进食大脑，夺心魔洞见师更倾向于召唤触须撕裂现实的结构并扭曲敌人的心灵。
