# 枯星怪

**枯星怪（星体枯萎怪） Astral Blight**

*中型植物，无阵营*

护甲等级：12（天生护甲）

生命值：26（4d8+8）

速度：10尺

力量 16（+3） 
敏捷 8（-1） 体质 14（+2）

智力 6（-2） 感知 10（+0） 
魅力 3（-4）

伤害抗性：寒冷，光耀

感官：盲视60尺（范围外不可视物），被动察觉10

语音：——

挑战等级： 1（200 
XP） 熟练加值：+2

**照明 Illumination。** 只要星体枯萎怪拥有至少1点生命值，其散发出10尺半径的微光光照。

**非凡本质 Unsual Nature。** 枯萎怪无需空气和睡眠。

**动作 Actions**

** 多重攻击 Multiattack。**枯萎怪进行两次吸热藤蔓攻击。

** 吸热藤蔓 Heat-Draing Vine。**近战武器攻击：命中+5，触及10尺，单一目标。命中：6（1d6+3）点光耀伤害。此外，如果目标是生物，且其体型不超过大型，则其被 
擒抱 Grappled（逃脱 DC 
3）。擒抱期间，每当该生物回合开始时它将受到3（1d6）点寒冷伤害。枯萎怪有两条藤蔓，每一条可以擒抱一个生物。
