# 网站介绍

各种网站科普

本页面旨在为读者普及一些 D&D 常用网站，帮助读者更便捷地入门。

DND 官方网站

网址：https://www.dndbeyond.com

简称 dnd beyond，或 dndb，是现在仍然在使用的龙与地下城官方网站，你可以在这里获取最新的 UA（测试内容），并且购买官方正版的模组、扩展与周边，但请注意国内用户无法邮寄实体版。

5E tools

网址：https://5e.tools

简称5et，是国外最大最详细的 dnd5e 资源汇集之处，你可以在这里查看各种扩展原文等，也有一些比较实用的功能，比如 CR 计算器，城主帷幕等。但请注意需求特殊网络环境，不然会较为卡顿。除英文原站外有一些汉化组制作了中文版5e tools，此处不一一列举网址。

**龙与地下城中文维基 **

基于灰机的 wiki 系统制作的以 DND5为主的维基百科，目前尚在建设中，欢迎大家参与进行维护补充。

网址：https://dnd.huijiwiki.com/wiki/%E9%A6%96%E9%A1%B5

Foundry VTT

网址：https://foundryvtt.com

简称 FVTT，是目前国内外社区生态最完善的跑团软件之一，为枭熊（见下文）的上位替代版本，功能完善。只需要 DM 拥有即可开团参团。但请注意本体需50美元（每年两次打折则为40美元），以及需要（云）服务器支持，且备团工作量极大，学习成本极高，不建议新手直接入手。官网注册与下载 mod 需求特殊网络环境。

Roll 20

网址：https://roll20.net

简称 r20，与上文 Foundry VTT 类似，会员以及买一些东西需要花钱，且刚需全员特殊网络环境。

Owlbear Rodeo

网址：无

简称枭熊1，为 Foundry VTT 的下位替代版本，无需安装软件，无需付费。但官方目前已将枭熊1关闭，转向枭熊2。国内有几个枭熊1镜像站仍然可以使用，此处不一一进行列举。

Owlbear Rodeo

网址：https://www.owlbear.app

简称枭熊2，为 Foundry 
VTT 的下位替代版本，无需安装软件，会员需付费，不付费的话有上传文件的大小限制。需求特殊网络环境，不然会较为卡顿。

纯美苹果园

网址：http://45.79.87.129 
或 https://www.goddessfantasy.net

简称果园，是国内 TRPG 爱好者最大的集中性站点之一，历史悠久。你可以在这里找到各种规则译文，并发帖讨论。目前暂时关闭了游客访问，需要注册账号，有极大可能无法正常进入或注册。目前搜索功能无法使用，需等待新果园。

DND 贴吧

网址：https://tieba.baidu.com/f?kw=dnd

这个大概不需要介绍，此处仅放网址。总之是百度贴吧的 DND 吧。

Forgotten Realms Wiki

网址：https://forgottenrealms.fandom.com/wiki/Main_Page

外网被遗忘的国度 Wiki，无中文，但是可以查找各种资料。需注意有些页面内容多版本相互混杂，有些页面则完全不可信。
