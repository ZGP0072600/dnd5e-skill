# 骰娘指令

**骰娘指令及使用相关**

此处以 
DicePP by 梨子 & 调零 Ver 1.4.2β 
版本进行有关骰娘指令的部分介绍，不同骰子或骰系的指令不尽相同，具体请查阅各骰系的用户手册。

！重要！希望骰娘离开群聊时请使用指令，不要直接踢出（直接解散群聊没问题），也不要随便禁言骰娘，更不要随便将骰娘拉入非跑团群！

下列指令中，英文句号与中文句号均可，+号与-号均可，为方便查看，空格键将使用*标注

开启骰娘：。bot*on

关闭骰娘：。bot*off

让骰娘离开本群：。dismiss 或。bot*bye

查看先攻：。init

清除先攻：。init*clr

结束当前的回合：。ed

直接设置先攻：。ri 数字*名字

删除单一先攻：。init*del*名字

骰值先攻：。ri+加值*投骰原因

最简单的投掷骰子：。rd20

扩展：。r 数量 d 面数+数量 d 面数+加值*投骰原因

投掷多枚骰子并分别显示结果：。r 数量#d 面数+数量 d 面数+加值*投骰原因

优势劣势：直接将中文“优势”“劣势”加在面数的后边，见下文的例子

让骰娘记住你在本群内的名字：。nn*名字

roll 属性（也就是俗称的“天命几”或“dnd 几”）：。dnd 次数

以下是一些例子：

.dnd5 ————让骰娘为你生成5次六大属性值

.ri22 提亚马特 ————将提亚马特的先攻设置为22

.r3#d20+1d4+5————投掷三枚 d20骰子，每个骰子分别加上1d4和5的加值

.r2#d20优势+5————投掷两次 d20骰子，每次投掷都取优势，且每次都加上5的加值

.init del 提亚马特 ————在先攻表中删除提亚马特的名字
