# 新祈唤 New Invocations

**野性转化 Feral Transformation**

*先决：等级7+*

以一个动作，你可以变形为恐狼、巨蜘蛛或巨章鱼，遵循与法术变形术 polymorph 相同的规则，但你保留你的智力，感知和魅力值。你同样保留你所有的豁免熟练，并获得这些生物的豁免熟练；此外，你还会保留该野兽形态在物理上能执行的技能熟练项。如果该生物拥有与你相同的熟练项，且数据卡中的加值高于你，则你使用该生物的加值。你保留说话的能力，并且可以施展仅具有言语成分的法术。

你可以保持该形态1小时，或直到该形态的生命值降至0时，你恢复为正常形态。若你因生命值降至0而恢复原形，则该伤害的溢出伤害量将传递至你的正常形态。

一旦你使用了此祈唤，直至完成一次长休你都无法再次使用。

**巫咒撕裂者 Hexshredder**

*先决：刃之魔契*

你的手指成长为野兽般的利爪，你可以将其用于发动徒手打击，造成1d6点挥砍伤害。你在使用此类徒手打击进行攻击检定和伤害掷骰时，可以使用你的魅力调整值代替力量调整值。

除了你所选的近战武器之外，你的徒手打击也被视为你的契约武器，并且它们也被视为轻型武器，可用于双持武器战斗。

**原初唤者 Primal Summoner** 

*先决：链之魔契*

你可以召唤出更偏向怪物的魔宠。当你施展法术寻获魔宠 Find 
Familiar 时，你可以为你的魔宠选择通常的形态选项，或者下列特殊形态之一：**闪现犬 Blink Dog**，** 亡命恶犬 Death Dog**，** 豹 Panther**或 ** 座狼 Worg** 
 
。

此外，你可以在你的回合消耗一个附赠动作来命令你的魔宠对你选择的敌人使用攻击动作。

**猎人秘典 Hunter's Grimoire**

*先决：书之魔契*

你的影之书中显现出了新的一页，专用于你的猎物。当你从某个生物处得到了实物证据（如一滴血、一缕毛发、一根羽毛、一颗牙齿或爪子等）时，你可以将其献给你的影之书以进行一场持续10分钟的特殊仪式。仪式会消耗献上的证据，而其来源生物的名字将以红色墨水铭刻于这一页上。在完成仪式后，只要你携带着影之书，你针对该生物进行的所有智力检定将获得优势。

此外，你可以花费1分钟来审阅该页面，以了解以下有关该生物的实际情况中的一项：

- 该生物的生物类型与阵营。
- 该生物所使用的任何名字或别名。
- 该生物当前与你之间距离与方向（前提是它与你处于同一位面）。
- 该生物的任意弱点或缺陷。
- 该生物当前是否受伤。
- 该生物是否处于某种状态、诅咒或类似效应影响下，以及该效应的性质。
- 该页面一次只能记录一个生物的名字，每当你完成该仪式时，你可以选择是否替换当前占用页面空间的生物的名字。
