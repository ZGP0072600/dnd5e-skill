# 新法术

### 植物修复｜Mend Plants

*变形 戏法（德鲁伊、术士、法师）*

**施法时间：** 动作

**施法距离：** 触碰

**法术成分：**V、S、M（一根小树枝和一圈麻绳）

** 持续时间：**立即

你利用原初魔法，为附近的植物注入活力，在施法距离内产生以下效应：

- 你立即修复你触碰的活着的植物上的一处断层或撕裂处，如撕裂的叶子、烧焦的树枝、树皮的疤痕或断成两片的植物碎片。只要损伤区域在任意方向的长度均不超过1尺，你就能修复其，不会留下任何被破坏过的痕迹
- 你使落叶的植物立即长出叶子，使种下的种子立即发芽，或让死去的种荚可以重新发芽。
- 你令5尺立方区域内的活着的植物枝叶繁茂，例如长满草或叶子。该区域的植物将变得茂密并足以隐藏不超过中型的生物，或是过度生长并成为困难地形（由你施展时选择）。
- 你能够触碰一株活着的植物或是一名植物生物令其获得1d4临时生命值。
