# 灰烬种 Ashborn

灰烬种是一种类似 **小魔鬼 Imp**，原生于切尔纳波斯的种族，通常由该行省的狱火与硫磺火山的硫酸灰烬凝聚而成。这一种族被地狱的邪魔力量所浸染，他们凭借狡黠、大胆以及勤奋，在该行省的荒原与恶地中生存下来，甚至繁荣兴盛。据说灰烬种被命运本身所青睐，常能在最凶险的时刻依靠运气扭转乾坤。

他们身材矮小，体格或瘦弱或健壮，但丝毫不会减损他们的危险程度，因为每个灰烬种都生有一条如蝎子般的毒尾，时刻准备着发动攻击。他们头生犄角，肤色呈现出深浅不同的红灰色调，显露出其魔鬼起源，灰烬种的寿命通常为100岁。

灰烬种特质 Ashborn Traits

**生物类型：** 邪魔

**体型：** 小型（约2-3尺高）

**速度：**30尺

作为灰烬种，你拥有以下特殊特质：

**灰烬遗赠 Ashen Legacy。** 你习得戏法次级幻象 Minor Illusion。此外，从3级起，你总是准备着法术魅惑类人 Charm Person；从5级起，你始终准备着法术隐形术 Invisibility。你可以在不消耗法术位的情况下施展这些法术各一次。你可以在完成一次长休后再次以这种方式施展这些法术。你也可以使用合适的法术位施展这些法术。

当你通过该特质施展这些法术时，你的施法属性为智力、感知或魅力（选择此种族时决定）。

**黑暗视觉 Darkvision。** 你拥有60尺黑暗视觉。

**邪魔幸运 Fiendish Fortune。** 当你被一次非重击的攻击检定命中时，你可以使用反应魔法性地使该次攻击对你失手，并且一个位于你周边5尺范围内你选择的生物会受到等于你熟练加值的力场伤害。使用此特质后，直至你完成一次短休或长休为止都不能再次使用。

**蝎刺 Scorpion Sting。** 你拥有一根可用于发动徒手打击的尾刺，使用你的力量或敏捷（由你选择）进行攻击检定和伤害掷骰。命中时，你造成1d4穿刺伤害，并且你可以造成额外1d6毒素伤害。你可造成此额外伤害的次数等于你的熟练加值。你在完成短休时恢复一次已消耗的使用次数，完成长休时恢复所有已消耗的使用次数。

当你的角色等级达到10级时，额外伤害增加至2d6。
