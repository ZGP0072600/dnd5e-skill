# 诡木种 Gnarlborn

| 诡木种（发音为 NAR-ull-born）是一种类似树木，原生于艾登格罗姆的种族，常从那片扭曲阴暗森林的土壤中生长而出。他们的家园萦绕着早已逝去的妖精灵魂，这些灵魂寄居于被称为长老木的、带有面容的巨木中。他们与这些存在交流，从中汲取原初的智慧。他们形似扭曲的树木，树皮上镂刻出发光的面容，其独特的枝条、苔藓和地衣图案，使得他们的形貌呈现出千姿百态。大多数诡木种的寿命都超过500年，但据说与长老木联系最紧密的那些个体可活到数千年。 诡木种特质 Gnarlborn Traits 生物类型：植物体型：中型（约5-8尺高）速度：30尺 作为诡木种，你拥有以下特殊特质：根深蒂固 Deep Roots。 你为避免非自愿移动和倒地状态而进行的属性检定和豁免检定具有优势。长老木低语 Elderwood Whispers。 你可以汲取长老木精魂的记忆。当你完成一次长休时，你获得一项你选择的技能或工具的熟练，或者掌握一门你选择的语言，持续至你完成下一次长休。缠枝 Grasping Branches。 你可以使用附赠动作魔法性地释放黑暗森林的束缚力量。选择你周边30尺范围内的一个大型或更小的生物。该生物必须进行一次力量豁免（DC=8+你的熟练加值+你的体质调整值）。豁免失败则目标会因被显现的枝条缠绕而陷入束缚状态，持续1分钟。豁免成功则目标的速度减半，直至你的下一回合结束。被束缚的生物可以在其自身每个回合结束时重复该豁免，豁免成功则终止自身效应。该状态终止时，枝条消失。使用此特质后，直至你完成一次短休或长休为止都不能再次使用。根须感知 Root Sense。 你可以使用附赠动作来获得60尺震颤感知，持续10分钟。使用此特质后，直至你完成一次短休或长休为止都不能再次使用。参天之姿 Towering Size。 你为终止自身受擒状态所进行的属性检定具有优势。此外，计算自己的载重时，你视为大一级的体型。 |  |
