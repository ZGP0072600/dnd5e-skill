# 深潜种 Deepborn

深潜种是一种类似鱼类，原生于奥尔马什的种族，常自该行省漆黑之海的深渊中浮现 —— 那里是名为 **饥饿者 Hungering Ones** 的超乎想象之恐怖存在的栖息之地。他们长期浸淫于源自深海、降自星外的诡怖气息之中，通过接纳并驾驭这些异象与宇宙力量，来控制自身的偏执状态。

虽然他们的解刨结构主要类似于鱼类，但也可能带有多种海洋生物的特征，包括头足类与甲壳类生物。他们通常能活150年左右，但某些神秘教派的成员已知能活到600年。

深潜种特质 Deepborn 
Traits**生物类型：** 异怪

**体型：** 中型（约5-7尺高）

**速度：**30尺，等于你速度的游泳速度

作为深潜种，你拥有以下特殊特质：

**水陆两栖 Amphibious。** 你可以在空气和水中呼吸。

**黑暗视觉 Darkvision。** 你拥有60尺黑暗视觉。

**诡怖诳语 Eldritch Gibbering。** 
得益于你与饥饿者之间的诡怖联系，你会说深潜语。你可以使用魔法动作来用该语言发出怪诞乱语。源自你的30尺光环区域内所有你选择的生物都必须进行一次感知豁免（DC=8+你的熟练加值+你的智力、感知或魅力调整值；选择此种族时决定），豁免失败则掷1d6并查阅下表。该效应持续至你的下一回合开始。一旦任何目标为对抗该特质而进行的豁免检定失败，则直至你完成一次短休或长休为止都不能再次使用该特质。

|---|---|
| 1-2 | 目标的攻击检定具有劣势。 |
| 3-4 | 目标的速度变为0。 |
| 5-6 | 目标不能执行反应。 |
