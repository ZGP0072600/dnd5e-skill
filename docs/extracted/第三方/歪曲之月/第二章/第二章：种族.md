# 第二章：种族Species

德鲁斯肯瓦尔德是诸多异界种族的家园，他们皆由最终归于这片月影与迷雾之地的逝者精魂转生而来。这些灵魂大多是搭乘幽灯特快列车而至，但也有其他陌行客引导任性灵魂安全穿过迷雾，使之在歪曲之月下重获新生。本章所介绍的各个种族皆是诞生于德鲁斯肯瓦尔德的十三个行省之中。虽然每个种族在其起源行省最为常见，但这些种族的许多成员都已经迁徙漂泊到其他行省，能在德鲁斯肯瓦尔德的各个角落找到他们的身影。
