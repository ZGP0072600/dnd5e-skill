# 通用专长 General Feats

这些专长从属于通用分类。

拒绝死亡 Death Defier

*通用专长（先决：等级4+）* 
 
当死神的召唤来到你面前，你找到了一种方法拒绝它。你获得以下增益：

- **属性值提升 Ability Score Increase。** 你的体质提升1，至多提升至20。
- **紧握生命 Cling to Life。** 你在死亡豁免中加入你的体质调整值（至少 +1）。
- **坚毅活力 Enduring Vitality。** 如果某个效应会直接杀死你，且不触发死亡豁免，或在未造成伤害的情况下直接将你的生命值降至0，你的生命值改为降至等同于你等级的数值。你必须完成一次长休才能再次使用此增益。

迅捷巫术 Swift Witchcraft 

*通用专长（先决：等级4+，施法或契约魔法特性）*

 
你已在巫术仪式中精进技艺，能够在需要时迅速施展魔法。你获得以下增益：

- **属性值提升 Ability Score Increase。** 你的智力、感知或魅力提升1，至多提升至20。
- **高速咏唱 Swift Invocation。** 你可以使用动作来施展一个施法时间为1分钟的法术。使用此增益后，直至你完成一次长休为止都不能再次使用。
