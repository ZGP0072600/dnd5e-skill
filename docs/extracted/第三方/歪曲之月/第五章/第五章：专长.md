# 第五章：专长 Feats

本章节介绍了一些与第四章中民俗恐怖背景对应的专长，当然只要得到 DM 的许可，任何角色都可以选择这些专长。

**变体：通用化的起源专长 Generalized Origin Feats** 

 
起源专长被设计成在1级时选取，因此它们整体上略弱于通用专长，但这也会使得它们在高等级时不再那么有吸引力。

在 DM 的酌情裁定下，如果角色在4级或更高等级时才选择一个起源专长，则可以为该专长额外增加以下增益：

** 属性值提升 Ability Score Increase。** 
你选择的一项属性提升1，至多提升至20。
