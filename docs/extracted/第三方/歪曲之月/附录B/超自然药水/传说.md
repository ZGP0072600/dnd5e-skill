# 传说

##### 赤红饥渴饮剂 Draught of the Red Hunger

药水，传说

饮下这瓶药水后，你会对鲜血产生贪婪的渴望，并获得法术吸血鬼之触 Vampiric 
Touch （六环版本，无需专注）的效应，持续1分钟。你可以在后续回合内使用附赠动作而非魔法动作重复进行此攻击。

这种液体呈微微发光的红色，比天然血液更加深邃和浓烈。

强效 Heightened Potency。持续时间内，若你未身处阳光下，你可以使用魔法动作获得法术气化形体 Gaseous 
Form （无需专注）的效应。你可以使用魔法动作恢复正常形态。
