# 诅咒奇珍 Cursed Curios

某些魔法物品带有一种诅咒，会渗入肉体、心智或灵魂，并且同调时间越长，诅咒侵蚀越深。在德鲁斯肯瓦尔德，力量永远伴随着代价和危险，然而一旦你掌握了力量，你还有勇气放弃它吗？

**缠魂诅咒 Lingering Curses。** 
 
结束与这些物品的同调会移除其诅咒，但诅咒异宝的力量不会轻易放开对使用者的控制。如果你在28日内再次与该物品同调，则其带来的诅咒会恢复至你结束同调之前的程度。
