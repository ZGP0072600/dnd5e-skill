# 寓言遗珍 Fabled Heirlooms

在德鲁斯肯瓦尔德的幽幕领域中，古老的缚魂魔法扭曲流转，为那些蒙受命运眷顾的传奇存在所吸引。这种灵性能量缠绕于这些存在所持有的遗物之中，使这些宝物承载其主人的本质与力量。一旦与塑造它们的神话人物分离，这些寓言遗珍便能被新的持有者继承，并赐予他们种种神秘而骇人的能力——这些能力将随着主人灵魂不断强大而逐步觉醒。

每件寓言遗珍都是一件能够跟随其同调者的成长而增强的魔法物品。其稀有度会随着角色等级的提升而升高，如遗珍稀有度表所示。每件遗珍的描述中列出了它始终具备的词条，以及随着稀有度提升逐步解锁的额外词条。遗珍在任何时刻都具备其当前稀有度及更低稀有度所对应的所有词条。
