# 柳艺结社 Circle of Wicker

编织祭献与守护的草偶

柳艺结社的的德鲁伊们深谙古老技艺，他们编织树枝人偶和塑像，将魔法束缚进这些粗糙的形体中。这些制品通常形似动物、符文图案、自然符号或人类。护佑符咒和针对入侵者的惩戒誓言，是这些德鲁伊的看家本领。你能将大地魔法编织进你的柳条造物中，将其化作赐予友军的恩惠和降临敌人的灾祸。

3级：巫咒塑像 Bewitched 
Effigy

你的原始魔法能以柳条塑像的形式显化并以其为核心向外辐射。你可以使用附赠动作消耗一次荒野形态特性的使用次数，在你周边30尺范围内一处你可见的未占据空间内创造一个柳条塑像。

塑像是一个小型物件（AC15；HP 等于你德鲁伊等级的五倍；具有火焰伤害易伤；免疫毒素和心灵伤害）。塑像在1分钟后、当其生命值减少至0时或当你再次使用该特性时消失。它在源自自身的30尺光环区域内发出微光光照和以下灵光之一（由你在创造塑像时选择）。

**祭焰 Sacrifice。** 当灵光内的一个生物受到伤害时，你可以使用反应对该生物造成1d8+你感知调整值的火焰伤害。

**慰灵 Sooth。** 当你创造塑像时，以及在后续回合内使用附赠动作，使灵光内一个你可见的生物恢复1d6生命值。你能够使用该塑像来恢复生命值的次数等于你施法属性调整值的两倍。

**结界 Ward。** 若身处灵光内，你和你的盟友的 AC 获得+1加值。

3级：柳编符咒 Wicker 
Token

你可以编织带有自身魔法的柳编符咒。当你完成一次长休时，你可以创造数量等于你熟练加值的符咒。符咒是一个微型物件（AC5；HP1；免疫毒素和心灵伤害）。符咒会在你完成下一次长休时消失。

当你和携带你符咒的生物位于同一存在位面时，你可以对它施展施法距离为触碰的法术。

6级：精进柳艺 Wicker 
Rising 你通过柳条塑像引导的能量变得更为强大，从而使其赋予的增益得到强化。

**祭焰 Sacrifice。** 火焰伤害增加至2d8+你的感知调整值。

**慰灵 Sooth。** 塑像提供的治疗增加至1d8，并且当你通过人偶指定一个生物时，你可以消耗5点塑像的生命值，移除目标身上的以下状态之一：魅惑、恐慌、麻痹、中毒、震慑。

**结界 Ward。** 选择一种除力场外的伤害类型。位于灵光内时，你和你的盟友对所选类型伤害具有抗性。

10级：缚魂娃娃 Soulbound 
Poppet

你可以使用魔法动作消耗一次荒野形态 Wild 
Shape 特性的使用次数，选择你周边60尺范围内一个你可见的生物，创造出一个与目标外观相似的柳条娃娃。娃娃是一个微型物件（AC10；HP10；具有火焰伤害易伤；免疫毒素和心灵伤害）。娃娃会在1分钟后或当其生命值减少至0时消失。娃娃存在期间，目标对携带娃娃的生物进行的攻击检定具有劣势，当一个携带娃娃的生物在一回合中首次对目标造成伤害时，目标会额外受到2d10暗蚀伤害。

14级：极境柳艺 Wicker Zenith

你的柳条塑像已达到其力量的极致之境。在你的回合开始时，你可以将人偶的灵光更改为另一种。此外，所有灵光都会按如下方式变得更强。

**祭焰 Sacrifice。** 
火焰伤害增加至3d8+你的感知调整值。

** 慰灵 Sooth。** 
当你或你的盟友在灵光内开始自己回合时，其魅惑或恐慌状态（由目标选择）会终止。

** 结界 Ward。** 
选择一项属性。位于灵光内时，你和你的盟友以所选属性进行的豁免检定具有优势。
