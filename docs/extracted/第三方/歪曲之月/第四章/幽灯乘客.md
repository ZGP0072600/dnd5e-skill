# 幽灯乘客 Ghostlight Passenger

曾几何时，甚至早在你最近一次前往德鲁斯肯瓦尔德之前，你便曾搭乘过那辆死亡列车——幽灯特快列车 Ghostlight 
Express。那段旅程的记忆可能如梦似幻、模糊不清，也可能格外清晰，甚至令人心悸。你对这辆幽灵列车的某些运作方式有了一些了解，也与列车长和同行的旅伴建立起了某种深厚的联系，这段经历至今仍留存在你心中。有时，你甚至觉得与死者相处比与生者相处更为自在。

专长： 幽灯灵媒（见第五章）

技能熟练：洞悉，游说

工具熟练：铁匠工具

装备： 
选择 A 或 B：(A)铁匠工具、牛眼提灯、燃油（3扁瓶）、高档服装、5GP；或(B)50GP。

## 创建幽灯乘客 Building a Ghostlight Passenger

在德鲁斯肯瓦尔德，每个人都至少搭乘过一次幽灯特快，但像你这样能与它建立起深厚且持久联系的人，少之又少。你第一次搭乘这趟列车时，也许正徘徊在生死之间，最终幸得救回；又或者，你的确死去，却在之后奇迹般地复生。

那些与精魂或亡者之地相契合的角色，例如牧师、圣武士，或是曾与死亡擦肩而过之人，最常展现出这份深层的联结。
