# 暗夜猎手 Night Stalker

曾经，你只是猎物，在黑夜中被怪物无声追踪。如今，你化身为猎人，那些曾令你恐惧的存在，将亲身领悟何为真正的惊惧。你变得更致命、更贪婪、更冷酷，远胜当初那个追捕你的怪物。如今，你自己便是那怪物——那种曾想以你或你的亲近之人果腹的可怖生灵。你存活了下来，而那意图将你捕杀的猎手，或许反倒要落得毁灭的结局。

专长： 猎兽猎手（见第五章）

技能熟练：隐匿，以及如下列表中你选择的另一项：奥秘、历史、宗教、自然

工具熟练：皮匠工具

装备：选择 A 或 B：(A)皮匠工具、书（一种生物类型）、附盖提灯、捕猎陷阱、燃油（3扁瓶）、旅行服装，8GP；或(B)50GP。

## 创建暗夜猎手 Building a Night Stalker

想一想，你是如何投身于这场对抗暗夜生物的狩猎之中的。是因为失去了至亲，还是从某次袭击中死里逃生？或许你自幼便在一个世代传承的猎手家族中长大。游侠、游荡者和野蛮人都是令人畏惧的暗夜猎手，任何对这些致命生物充满对抗热情的角色亦是如此。
