# 猩红求道者 Crimson Aspirant

你曾深入研究驱动一切生命的力量——即便是那些模仿生命的造物，如构装与亡灵。这些研究大多都专注于血液的本质，以及其维系生命、传递生命精粹、将灵魂锚定于肉身的力量。你或许拜在某位心机深沉的导师门下，或加入了某个专研生死之道、乃至操纵生死的秘教团体。又或许，你从一部详尽的论著中习得技艺，并大胆将其付诸实践，精通那些禁忌之术。

专长： 猩红祭仪师（见第五章）

技能熟练：奥秘，医药

工具熟练：草药工具

装备：选择 A 或 B：(A)草药工具、匕首、高档服装、医疗包、小瓶×3、10GP；或(B)50GP。

## 创建猩红求道者 Building a Crimson Aspirant

那些注重学识与研究，以及具备疗愈他人伤痛与疾病之力的角色，往往很容易成为猩红求道者。而那些体魄强健者可能借由血之教义，激发自己更深的生命潜能；同样地，体弱多病者也可能寄希望于此，以期扭转命运。

**建议的故事线索 Suggested Story Threads。** 猩红求道者故事线索表列举了可能成为你角色故事的开端，并贯穿整个战役的事件与人物。

猩红求道者故事线索 Crimson Aspirant Story Threads

| D6 | 线索 Threads |

| 1 | 有时，当你流血时，便会听见血中低语，似乎蕴含深意，却总是含糊难辨。 |

| 3 | 一种通过血液传播的瘟疫无药可解，你必须追查它的起源。 |
