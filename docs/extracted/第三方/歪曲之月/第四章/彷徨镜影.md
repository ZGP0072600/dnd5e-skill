# 彷徨镜影 Reflected Wanderer

你曾经是完整的，然而如今你的部分自我被剥离，像你一样在这个世界上漂泊游荡。你的分身带走了你的倒影与阴影，以与你一模一样的幻像姿态游走于世。你明了这另一个自我的存在——你能够感知到它——但它所求为何，以及缘由何在，你却一无所知。偶尔，你会遭遇某位仿佛早就认识你的人——或怀恨在心、或亲近友善——但你却从未与其见过面。不禁揣测：他们是否与那个“另一个你”有过交集？

专长： 无影之形（见第五章）

技能熟练：欺瞒，调查

工具熟练：易容工具

装备：选择 A 或 B：(A)易容工具、镜子、旅行服装、16GP；或(B)50GP。

## 创建彷徨镜影 Building a Reflected Wanderer

任何角色都可以是彷徨镜影。你的存在是如何分裂的？你对自身的现状了解多少？你真的就是你以为的那个“你”吗，还是其实只是那个“真正的你”所遗弃的影子？

**建议的故事线索 Suggested Story Threads。** 
彷徨镜影故事线索表列举了可能成为你角色故事的开端，并贯穿整个战役的事件与人物。
 

彷徨镜影故事线索 Reflected Wanderer Story Threads

| D6 | 线索 Threads |
|---|---|
| 1 | 一段亲密且长期的关系突然恶化，仿佛你曾做错了什么事。 |
| 2 | 你的另一个自我会在游历时给你留下讯息和标记，有时显得亲昵，有时又充满怨恨。 |
| 3 | 动物在你身边表现出不安，易惊易躁、心绪难平。 |
| 4 | 你被通缉，罪名多是一些你从未犯下的小罪，但目击者都指认了你。随着时间推移，罪行的严重程度还在持续增加。 |
| 5 | 某个与你有嫌隙之人开始以尊重与善意对待你。 |
| 6 | 镜子有时会在你触碰时碎裂。 |

**彷徨镜影饰品 Reflected Wanderer Trinkets。** 当你创建角色时，你可以在彷徨镜影饰品表，而非常规的起始饰品表中掷一次。 

彷徨镜影饰品 Reflected Wanderer Trinkets

| D6 | 饰品 Trinkets |
|---|---|

| 2 | 一封来自某位权贵的邀请函——你从未见过此人。 |

| 4 | 一封来自你自己，短小而带有嘲讽意味的信，你却不记得曾写过。 |
| 5 | 一枚你从未购买过、却被送到你手中的胸针。 |
| 6 | 一支写出的颜色与它本身颜色不同的粉笔。 |
