# 岔路赌徒 Crossroads Gambler

你曾在各式各样的博弈游戏中闯出一番名堂，然而某天，你的游戏越过了那条界线，演变成了远比寻常更加凶险的境况。你或许曾与邪魔对弈，以灵魂为赌注；或是与古老森林的妖精在谜题中斗智斗勇；抑或是与传说中的岔路先生 Mister Crossroads 掷骰一赌。

专长： 命运赌徒（见第五章）

技能熟练：欺瞒，洞悉

工具熟练：选择一种赌具

装备： 选择 A 或 B：(A)赌具（与上述相同）、高档服装、铁蒺藜、优质红酒（瓶装）、23GP；或(B)50GP。

## 创建岔路赌徒 Building a Crossroads Gambler

靠机敏和智谋立足的角色，如游荡者和吟游诗人，往往会被岔路赌徒的呼唤所吸引。但实际上，任何人都有可能身陷其中，赌上的筹码往往远超他们的想象——甚至超出所能支付的范畴。不慎召唤错误存在的施法者，也可能走上赌徒的道路。而那些在绝望中找上神秘人物、以掷骰和博弈换取力量或救赎之人，他们的命运便可能就此与岔路紧密相连。

**建议的故事线索 Suggested Story Threads。** 岔路赌徒故事线索表列举了可能成为你角色故事的开端，并贯穿整个战役的事件与人物。

岔路赌徒故事线索 Crossroads Gambler Story Threads

| D6 | 线索 Threads |
|---|---|
| 1 | 你曾以为是朋友的人，却欺骗你达成了那笔交易。 |
| 2 | 你发现自己若太久不下注，运气就会开始变差。 |
| 3 | 唯有在赌局的对抗中，你才能真切感受到自己“活着”。 |
| 4 | 你输掉了赌注，如今正逃离着那必定前来索取的存在。 |
| 5 | 你亲近的某人下了赌注并输了。你为了寻找他们而达成了自己的交易。 |
| 6 | 你已经赌了太久，身心俱疲，只想找到一个“退出游戏”的办法。 |
