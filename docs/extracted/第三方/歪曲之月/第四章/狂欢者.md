# 狂欢者 Reveler

你曾与那位愚人之王那奇异而令人不安的狂欢队伍一同巡游。无论你如今是否仍被他们接纳，又或是早已离团独行，你与这支队伍相处的时光都向你揭示了其部分本质——那位愚者之王侍奉着某个强大的存在，祂珍视幽默与欢愉，往往以牺牲强者为代价。但他绝非等闲弄臣。在那戏谑与欢愉的假面之下，隐藏着一个索求献祭的疯狂生物。

专长： 狂欢愚者（见第五章）

技能熟练：表演，隐匿

语言：选择一项（推荐深渊语）

工具熟练：选择一种乐器

装备：选择 A 或 B：(A)乐器（与上述相同）、滚珠、戏服（带笑脸面具的快乐角色）、捕猎陷阱、镣铐、三根火把（用于杂耍）、旅行服装、10GP；或(B)50GP。

## 创建狂欢者 Building a Reveler

狂欢者令人既迷醉又不安的真相在于，他们可能是任何人。隐匿于他们的面具和戏服之下，你的朋友、家人、仆役，乃至领主，都可能在人群中围绕缎带柱跳舞，趁着狂欢退场之时，引诱他人随之而去。在不同的伪装下，狂欢者会展现出怎样不同的行为模式?

**建议的故事线索 Suggested Story Threads。** 狂欢者故事线索表列举了可能成为你角色故事的开端，并贯穿整个战役的事件与人物。

狂欢者故事线索 Reveler Story Threads

| D6 | 线索 Threads |
|---|---|
| 1 | 愚人狂欢结束后，镇上陆续张贴出描绘你狂欢装扮的海报。 |
| 2 | 一旦你得知狂欢背后的真相——那些表面上加入狂欢而消失的人，实际上是被献祭了——你便在第一时间设法脱身。 |
| 3 | 最近，在狂欢之后，城镇里失踪的人比往常更多。 |
| 4 | 乌鸦、猫头鹰和其他鸟类似乎在你面前咯咯地笑着。 |
| 5 | 你开始怀疑，另一位狂欢者在他们的弄臣装扮之下，隐藏着险恶的动机。 |
| 6 | 你离开了狂欢队伍，因为你瞥见了隐藏在愚人之王那欢快表象之下的邪恶。 |
