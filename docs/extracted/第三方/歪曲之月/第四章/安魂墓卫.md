# 安魂墓卫 Rest Warden

在一处比生者之地更靠近亡者的所在，那些负责安葬逝者，并守护其长眠之人，既备受尊崇又令人心生畏惧。你曾从事过掘墓人、殡葬师和敛尸官的行当。很多时候，你是唯一一个为逝者献上善意悼词的人。你对亡灵的侵扰再熟悉不过，也绝不容许它们干涉你所看护之人的安息。

专长： 守墓人（见第五章）

技能熟练：宗教，求生

工具熟练：石匠工具

装备：选择 A 或 B：(A)石匠工具、附盖提灯、燃油（3扁瓶）、铲子、肥皂、旅行服装（有厚手套与深色兜帽）、10GP；或(B)50GP。

## 创建安魂墓卫 Building a Rest Warden

安魂墓卫往往要么阴郁沉闷，要么与他们所从事的职业形成鲜明反差，洋溢着对生命的热忱与活力。只有能看透生死之间那道微妙界线，或是拥有能于月夜下挥锄掘墓、在骸骨之园照料亡者长眠的大毅力者，才会应召成为安魂墓卫。

**建议的故事线索 Suggested Story Threads。** 安魂墓卫故事线索表列举了可能成为你角色故事的开端，并贯穿整个战役的事件与人物。
