# 实验体 Experiment

你曾经历一次剧烈且永久的身体改造，甚至可能变得如怪物般异形。科学、炼金与魔法的亵渎结合将你改造一新，这或许是为了治愈某种不治之症，又或许为了测试你生理机能的极限。这一过程某种意义上算是成功了，但它也在你身上留下了的痕迹。你形体的不可磨灭之扭曲，对那些无法理解自己所见之物的人来说，可能会成为其恐惧的根源。

专长： 异变者（见第五章）

技能熟练：威吓，医药

工具熟练：炼金工具

装备：选择 A 或 B：(A)炼金工具、镣铐、香水（用于掩盖化学残味）、旅行服装（带兜帽斗篷）、5GP；或(B)50GP。

## 创建实验体 Building a Experiment

实验体可以出身于任何职业或阶层。你是被绑架并被迫接受改造，还是主动选择接受这场骇人的手术？这场实验是出于纯粹的研究探索？为了逃避更可怕的命运？还是某人妄图扮演“造物主”所为？无论原因如何，这个过程绝不可能是令人愉快的；它总会留下伤痕——既有肉体可见的，也有深植灵魂的烙印。
