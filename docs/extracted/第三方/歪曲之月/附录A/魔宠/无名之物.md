# 无名之物 Nameless Thing

*来自遥远之地的未知生命体（陨石碎片）*

**栖息地**：任意　　**宝藏**：无

无名之物是被束缚在物质容器——通常时小型动物、物件或怪物碎片——中的异星心智。当异世影响侵入时，这些生物就能潜入此界，但它们通常是被人有意邀请到此的。

## 特质 Traits

**飞掠 Flyby。** 无名之物飞行离开敌人的触及范围时不会引发借机攻击。

## 动作 Actions

**心灵打击 Psychic Strike。** *近战攻击检定*：+5，触及5尺。*命中*：9（1d12 + 3）心灵伤害，且若目标为生物，其陷入恐慌状态，持续至无名之物的下个回合开始。

**隐形术 Invisibility。** 无形之物对自身施展隐形术 Invisibility，无需法术成分并使用智力作为施法属性。

施展该法术时，无名之物能令法术效应延展至一名其正触碰的生物。如果二者任一打破其身上的该法术或者无名之物停止触碰另一生物，法术提前结束。一旦无名之物延展该法术后，直至其完成长休才能再次如此做。。
