# 巫偶 Witchwoven

*线与树枝制成的助手（包裹在柳条笼中的红宝石）*

**栖息地**：森林，山地，城市　　**宝藏**：无

巫偶是用木棍、麻绳和其他简易材料制成的、被注入生命的娃娃或人偶。有些巫偶是在充满古老自然魔法的地方自然诞生的，但大多数巫偶是由熟悉传统的施法者创造的。

## 动作 Actions

**诅咒创痕 Cursed Scratch。** *近战攻击检定*：+5，触及5尺。*命中*：6（1d6 + 3）穿刺伤害，外加3（1d6）暗蚀伤害，且若目标为生物，其攻击检定具有劣势，持续至巫偶的下个回合开始。

## 反应 Reactions

**保护联结 Protective Bond。** *触发*：60尺内一名巫偶可见的生物受到伤害。*响应*：该生物对触发伤害具有抗性，且巫偶受到10（3d6）力场伤害。
