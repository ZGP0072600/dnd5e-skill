# 盗羊兽 Goatstealer

*饥肠辘辘的家畜掠夺者（熏香与腐肉）*

**栖息地**：荒漠，森林，草原　　**宝藏**：无

浑身无毛、散发恶臭的盗羊兽是凶猛的捕食者和食腐动物，因捕食牲畜的习惯而得名。它们用邪恶的针状牙齿抓住猎物，释放致命的毒液，吞噬比其瘦削身躯大得多的生物。在成群出现时，盗羊兽更加危险，传说成群结队的盗羊兽会发出腐肉与泥炭的臭气。

## 特质 Traits

**集群战术 Pack Tactics。** 若盗羊兽的攻击目标生物5尺内存在有至少一名盗羊兽未失能的盟友，则盗羊兽对该生物进行的攻击检定具有优势。

**恶臭 Stench。** *体质豁免检定*：DC 12，在源自盗羊兽的5尺光环区域内开始其回合的任意生物。*失败*：目标陷入中毒状态，直至其下个回合开始。*成功*：目标在1小时内免疫所有盗羊兽的恶臭。

## 动作 Actions

**啃咬 Bite。** *近战攻击检定*：+4，触及5尺。*命中*：5（1d6 + 2）穿刺伤害，外加3（1d6）毒素伤害。若目标生物的体型不超过中型，则其陷入受擒状态（逃脱 DC 
12）。每当盗羊兽的回合开始时，目标受到3（1d6）毒素伤害，持续至擒抱结束。
