# 魔宠 Familiars

本节中介绍的生物是可以与他人订立契约、成为魔宠的特殊存在——不仅是施法者，任何人都能如此。这种契约会创造一种紧密的魔法联系，因此这些存在不会轻易同意成为魔宠。它们要求受到尊重，但作为回报，它们会提供忠实的陪伴和服务。如果魔宠受到虐待或冤枉，它们会终止契约，并可能成为前同伴的眼中钉。

如何与这类生物订立契约并非常识，你可以将如何发现订立契约的方法作为一则隐秘处理，正如附录 A 的的怪物秘辛部分所讨论的那样。

**魔宠契约 Familiar Bond。** 当一名生物使用价值至少100gp 的材料（在每个魔宠的副标题中指定）花费10分钟进行仪式时，目标与该生物之间将形成魔法契约，使其成为该生物的魔宠。该契约在该生物死亡或该魔宠决定终止该契约时结束。契约持续期间，该生物与魔宠获得以下好处：

- **契约友伴 Bonded Sympathy。**魔宠可以选择让契约生物免受自身创造的区域效应的影响，例如伤害性球状区域效应或**盗羊兽 Goatstealer**的特质恶臭产生的光环区域效应。此外，如果魔宠具有其他能够建立联系的特性（例如**魔鬼手套 Devil’s Glove**的附赠动作共生），这份联系视作魔宠契约的一部分。
- **传导魔法 Channel Magic**。当契约生物施展施法距离为触碰的法术时，其魔宠可以帮其传递这道触碰法术。此时其魔宠必须距其100尺内，且其魔宠必须在其施展这道法术时执行一个反应将之传递。
- **心灵连接 Telepathic Connection。**契约生物的魔宠距其100尺内期间，该生物可以与其魔宠进行心灵交流。此外，契约生物还可以用一个附赠动作将自己的视觉和听觉切换到魔宠的感官，获得魔宠具有的特殊感官的好处，持续至其下一回合开始。
- **复活 Revival。**如果魔宠在契约存在期间死亡，契约生物可以使用魔宠契约材料并消耗1小时来进行契约仪式的变种仪式（学习该变种仪式不需要进一步的研究），复活魔宠并恢复其全部生命值。
