# 未来的你宗主 Future You Patron

以未来之助力操纵时间

你的宗主正是你自己——来自数十年后的未来。或许，未来的你自己找到了一件强大无比的神器，使其得以与过去建立联系，并且现在必须引导你去找到它；又或许，未来的你曾在久远的过去经由未来的自己教导秘法，如今你必须延续这一个循环——总有一天你也要教导你自己。未来的你自己已经遗忘了一些事情的细节，对于某些“现在还不能知道”的事情更是绝口不提，尽管如此，他/她仍能给与你深刻的见解与引导。你并不完全清楚他/她对你的未来（以及他/她的过去）有何打算，但有一点是确定的——他/她需要你活下去。

**未来的你的怪癖 Future You Quirks**

在你与未来的你自己沟通之后，奇异的影响残留在了你的身上。你获得以下特质之一：

| d6 | 个性特质 |
|---|---|
| 1 | 你经常用错时态，或者以复数形式称呼自己。 |
| 2 | 你有时会在对方自我介绍之前，就直接喊出他们的名字。 |
| 3 | 即便身处绝境，你依然异常冷静。 |
| 4 | 在特定光线下，你看起来比实际年龄年长许多。 |
| 5 | 见到某些人还活着，会让你瞬间泪流满面。 |
| 6 | 你的未来穿搭品味与当今的潮流格格不入。 |

**扩展法术表 Expanded Spell List**

宗主赐予的魔法使你始终准备着特定的法术。当你到达未来的你法术表中特定的魔契师等级时，你就始终准备着表中对应的法术。新法术以星号*标记。

**3级：事情是这样发生的 It Happened Like This**

你与未来自己的交流使得你对未来的发展略知一二。每当你完成一次短休或长休时，DM 会暗骰一个 d20和一个 d4，并记录下 d20的结果。除非 d4的结果掷出4，否则 DM 会如实告诉你所记录的 d20点数；如果 d4的结果是4，则 DM 谎报这个点数。

你可以将你自己或你可见的一个生物进行的任意 D20检定，替换为这个被记录的 d20掷骰（无需动作）。你必须在掷骰前选择是否替换，并且这个数字只能使用一次。

如果 DM 之前对你说了谎，只有在你替换检定之后才会告知你真实的记录结果。

**6级：我可不想再多几道伤疤 I Could Do With Fewer Scars** 未来的你自己经常会提醒你注意某些特定的攻击。当一个你可见的生物用一次攻击检定命中你时，你可以使用一个反应动作，使你的 AC 在抵挡这次攻击时获得+10加值，从而有可能使该攻击落空。

你可以使用此特性两次。你可以在完成一次短休后恢复1次已消耗的使用次数，完成一次长休后恢复所有已消耗的使用次数。

**10级：小心埋伏 Expect an Ambush**

你在进行先攻检定时具有优势，并且在战斗的第一轮获得对所有伤害的抗性。

**14级：祖父悖论 Grandfather Paradox**

你可以用一个动作挑衅一个你60尺范围内能看见且能听见你的生物，使其引发一次悖论。该生物必须进行一次智力豁免，对抗你的法术豁免 DC。豁免失败时，该生物会受到10d6心灵伤害，并因被困于相互冲突的时间线之间而陷入震慑状态，持续1分钟。豁免成功则目标仅受半伤。被震慑的生物可以在其每回合结束时重复该豁免，豁免成功则结束该效应。

此特性一经使用，直至完成长休你都无法再次使用。
