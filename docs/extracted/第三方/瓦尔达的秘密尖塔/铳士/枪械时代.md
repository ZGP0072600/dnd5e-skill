# 枪械时代 Firearm Eras

枪械在历史中经历了翻天覆地的变化，并将持续演变至遥远的未来。因此，除了分类为简易与军用武器外，以下枪械还被划分出不同的时代，即这些枪械可能出现的时间段。某些枪械可能在多个时代出现，尤其在某些故事需要使用特殊武器的情形，但大多数枪械更适合与其科技水平相符的战役设定。

战役所处的武器时代由 DM 决定。

## 文艺复兴枪械 Renaissance Firearms

文艺复兴时代的枪械，例如手铳和火铳，是武器首次从加农炮的形式发展成为便携式的手持枪械，亦是所有现代枪械的鼻祖。这个时代的枪械主要使用火铳球弹和散装的黑火药，因此换弹速度较慢。这些武器在许多奇幻设定中与弓箭、剑斧并存，尤其在一些海盗出没的设定更是如此。

## 工业时代枪械 Industrial Age Firearms

工业时代的枪械，例如左轮枪和双管霰弹枪，诞生于机械与流水线的进步，赋予这些枪械更多可互换的零件和基于弹药筒的子弹。这些枪械虽然缺乏现代流水线的一致性，但却奠定了此后几未改变的设计基石：经典的六弹容左轮经久不衰又威力十足。工业时代枪械的鼎盛时期在狂野的西部背景，但其简单可靠的构造意味着即使是现代依旧有它们的一席之地。

## 现代枪械 Modern Firearms

现代枪械意味着自动化、弹夹和小口径的高速子弹。对这些枪械而言，其样式就决定了功能：武器被设计了特定的用途，例如狙击步枪用于远距离狙击；泵动式霰弹枪用于近距离缠斗。此外，一些在西部时期广受欢迎的武器（比如双管霰弹枪），在现代仍大放光彩。
