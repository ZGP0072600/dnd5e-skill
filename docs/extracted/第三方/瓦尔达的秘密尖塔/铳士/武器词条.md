# 武器词条 Weapon Properties

以下是枪械表中词条列中各词条的定义，此处仅列出本书新增或有改动的词条。

**枪械 Firearm**

 除非另有说明，否则你无法将你的属性调整值添加到武器的伤害中。枪械的弹药在使用后即摧毁。

**后坐 Recoil**

在你使用此武器发动攻击后，直至当前回合结束，你都无法发动超出武器常规射程的远程攻击。

**换弹 Reload** 此武器在必须换弹前可以进行一定次数的攻击。如果你熟练于此武器，则为其换弹为一个动作或附赠动作；否则，换弹为一个动作。
