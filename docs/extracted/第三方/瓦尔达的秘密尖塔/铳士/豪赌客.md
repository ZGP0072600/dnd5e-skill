# 豪赌客 High Roller

生死为筹

命运变幻莫测——除非你是一名豪赌客。这些铳士是牌技大师、掷骰高手，他们将对风险的热爱与枪战的天赋完美结合。豪赌客会将自己的运气梭哈直至其消耗殆尽，然后他们将押上更多。既然能来一发大的，那何必满足于一时的胜利？

**3级：扑克脸 Poker Face** 你获得所有赌具的熟练，并自选以下一项技能获得熟练：欺瞒、洞悉或察觉。

**3级：欺天骰【战技】Liar's Dice** 

当你使用远程武器进行伤害掷骰时，你能够以附赠动作消耗一枚风险骰，并声明进行隐秘掷骰。你秘密地进行掷骰并宣言任何你期望的总值。然后 DM 可以选择揭穿你的虚张声势，此时你需要公布你的掷骰结果。

根据你是否撒谎，将产生不同的结果。

**DM 揭穿你且你撒了谎。** 你造成的伤害减半。

**DM 揭穿你但你未撒谎。** 你造成的伤害翻倍。

**DM 未揭穿你。** 使用你宣言你造成的伤害，即使你的掷骰结果并非如此。

**6级：暴虎冯河 Risky Business**

每回合一次，当你对敌人进行一次不具有劣势的攻击检定时，你可以选择获得劣势。若如此做，你重获一枚风险骰。

**10级：行险狂徒 Risk Taker**

你使用战技牛仔精神和千钧一发时，可以不消耗风险骰。若如此做，你投掷 d6以替代风险骰。

**14级：所有，或一无所有 Double or Nothing**

当你使用远程武器造成重击命中时，你可以赌一个更高的结果。掷一枚 d20，若结果为10或更高，则你伤害掷骰时投掷所有的伤害骰四次而非两次，并将结果相加。若结果为9或更低，则此次重击被视为普通命中。
