# 密间客 Secret Agent

顺密间之性，行暗杀之举

情报就是力量。击败敌人的最佳方法是窃取他们的情报，并替换为虚假情报。为此，你接受了隐蔽作战的训练以掌握多种技能，好辅助你令人生畏的枪法。

**3级：特工训练 Operative Training**

你的隐蔽训练赋予你以下增益：

** 隐蔽射击 Concealed Shot。**你习得戏法隐蔽射击 Concealed 
Shot 
 。你此戏法的施法属性为智力、感知或魅力（在选取该子职业时选择）。

** 特工工具 Operative Tools。** 
你获得一套易容工具和一套盗贼工具，并获得其熟练。

** 技能熟练 Skill Proficiencies。** 你自以下技能中选择两项获得熟练：欺瞒、调查、游说、巧手或隐匿。

**3级：抛下凶弹【战技】Parting Shot**

当你在你的回合内执行疾走、撤离或回避动作时，你可以消耗一枚风险骰以附赠动作使用武器发动一次远程攻击。命中时，将风险骰加入伤害掷骰中。

**6级：战场记忆 Fieldcraft** 

你的野战经验赋予你以下增益。

** 快速变装 Quick Change。** 
以一个附赠动作，你可以使用易容工具制作一套戏服并穿戴之。

** 巧舌如簧 Slick Talker。** 
每当你进行魅力（欺瞒）或魅力（游说）检定时，你可以将 D20投出的9及以下的结果视为10。

**10级：走为上计 Exit Strategy** 当你受到伤害时，你能够以反应避免进一步的损害。你可以立即移动至多10尺，且直至你的下个回合开始，你处于隐形状态。

此特性一经使用，直至你完成短休或长休都无法再次使用。

**14级：杀戮许可 License to Kill**

每当你使用远程武器造成伤害时，你可以消耗一或两枚风险骰加入伤害掷骰中。若你的风险骰掷出最高点，你可以无消耗地再次掷骰并将其加入伤害中，若仍掷出最高点，则再次掷骰，以此类推。你可以加入伤害中的风险骰数最多等于你的熟练加值。
