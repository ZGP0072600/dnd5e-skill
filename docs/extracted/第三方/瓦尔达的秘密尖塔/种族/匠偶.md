# 匠偶 Geppettin

| 匠偶 Geppettin 匠偶对孩子们来说充满乐趣，它就像用木头、布匹或瓷器制成的活着的玩具。虽然很容易被误认为是人偶，但是匠偶并非被某位看不见的人偶师操纵的玩偶，而是活化的、有知觉的独立生物。 作为一个种族，匠偶称得上奇异。虽然罕见，但它们数量众多且彼此之间具有足够多的共同点，因此算得上一个真正的种族而非仅仅是怪异的魔法事故。虽然匠偶可能有专门的创造者——就像一些魔像一样——但不少是自己活过来的。 通常而言，匠偶比半身人还矮一些。它们的体型特征因类型而异，但几乎都与类人生物相似。尽管有些匠偶是由脆弱的材料制成，但知觉赋予它们奇异的耐寒能力。它们不会饥饿，不知疲倦。大多数匠偶从事娱乐相关的工作，但少数匠偶在暗杀和间谍领域取得了巨大的成功。 素瓷 Bisque 素瓷匠偶是一种瓷娃娃，采用精湛的工艺且通常着装华丽的服饰。它们被设计得看起来尽可能逼真，但它们毫无生气的面容令其通常被认为是最恐怖的匠偶。 枢木 Marionettes 枢木匠偶主要由木头制成，以颜料绘制面孔并雕刻出衣物。由于它们的构造，它们的关节非常灵活且常常以出色的舞者闻名。最大号的枢木匠偶又被称作人体模型，看上去和人类一样高且有着无表情的空白脸孔。 褴布 Raggedy 褴布匠偶，又被称作毛绒匠偶，是任何填充娃娃或软质玩偶形态的匠偶。它们通常看上去是人型的，但也有看着像动物、怪物或是其他任何拟人生物的褴布匠偶。 匠偶特质 Geppettin Traits 生物类型 Creature Type：构装 体型 Size：小型（2-3尺高）或中型（约6尺高，仅限枢木） 速度 Speed：30尺 作为匠偶，你具有以下特殊的种族特质： |  |
