# 魔法物品 Magic Items

一名人类手中把玩着一根瓷白色的魔杖，随后从她的背部召唤出一对闪烁的天马之翼，并一跃升入空中。

一名肌肉发达的提夫林用一把冒着烟的锤子猛击地面，一边咧嘴大笑，一边看着他的武器分裂开来并喷发出一阵熔岩。

一名警惕的半身人小心翼翼地触碰着一幅画的表面，在瞥了一眼身后之后，便毫无痕迹地消失在其油滑的表面之中。

魔法物品拥有为故事增添神秘色彩的力量，同时也为玩家了提供角色成长的机会。无穷无尽的变化使魔法物品总能在挑战性的冒险后成为珍贵的奖励。

本章节为您提供了504种新的物品，可供您用于你的5版战役设定中。下文将假定您已熟知魔法物品的规则，包括其稀有度、类别、诅咒以及与其同调或通过其施法的需求和条件。
