# 天鹏旅人 Rocborne

天鹏旅人是骑乘始祖鹏鸟遨游天空，旅遍世界的游牧者。这些旅行者们坚信“气”与“风”绝不仅仅只是一种能量，认定这是由数不胜数的精魂在流动与牵引的共同影响下产生的力量。如同灵媒师倾听亡者之语，天鹏旅人可以聆听风之语，以此，他们不仅能获悉更多有关天气与前行道路的情报，还能更加了解自身。许多天鹏旅人一生都只跟随他们自己的文化聚落生活，但有一些旅人——那些为人所知的天鹏旅人——选择离开了自己的聚落，踏上了属于自己的旅程。有的时候，你的旅行轨迹可能未来注定会与某位随风而行的强大精魂有所交汇，而你可以感受到、倾听到你内心的那阵指引你前进的风。

通过呼唤风之精魂的协助，这一范形的游侠可以在一定程度上操控周遭的风与天气。这些技能是由鹏鸟们传授给首位天鹏旅人，再经由代代天鹏旅人长老（不论是死是活）的细心教导延续下来的技能。你可能会在受天鹏旅人精魂们的指引下，在那些轻语、托梦、冥想中，习得或拥有这些技能。

**天鹏旅人魔法 Rocborne Magic**

自3级起，当你在此职业上达到特定等级时你将习得一项额外的法术，如天鹏旅人法术列表所示的那般。该法术对你而言视为游侠法术，但不会占用你可已知的游侠法术数量。

| 游侠等级 | 法术 |
|---|---|
| 3级 | 羽落术 Feather Fall |
| 5级 | 造风术 Gust of Wind |
| 9级 | 闪电束 Lightning Bolt |
| 13级 | 冰风暴 Ice Storm |
| 17级 | 通晓传奇 Legend Lore |

**指引之风 Guiding Wind** 自3级起，当你发动攻击时，你可以轻声低语，让风指引你攻击的方向。你可以在每个你的回合之中，从以下两道增益中选择一次并获得它：

- 当你的武器攻击命中时，目标将受到额外1d4点与武器同类型的伤害。当你在此职业上达到11级时，此额外伤害增加至1d10点。
- 当你的武器攻击失手时，你可以改变它的轨迹去攻击另一个敌人，新的目标必须位于此武器的攻击范围内，且必须位于原先敌人周围5尺内。此时，你对新的目标的攻击检定直接采用你原先这次攻击检定的结果，外加上你的感知调整值（至少+0）点加值。当你在此职业上达到11级后，新的目标可以是原目标周围15尺内的目标（但其仍需位于武器攻击范围内）。

**知识之语 Whispers of Knowledge**
同样在3级起，你可以询问随风飘摇的精魂，以寻得你需要的知识。你可以用10分钟时间与行过的精魂交谈，让它们为你提供它的所见所闻：选择一项你还未具备其熟练的技能或工具，你获得其熟练。在下一次你完成短休或你完成长休后，这项熟练会消失。

你可以使用此特性的次数等同于你的感知调整值次（至少可以用一次）。你完成一次长休时可以重获全部已消耗的使用。

**行云之风 Windswept**

拜7级所赐，风将为你开辟前路，引领你前行。你的步行速度+5尺，并且可以无视强风为你带来的所有移动惩罚（movement 
penalty） ，且若非你自愿，你将无法被强风推动 
。

此外，你具有寒冷伤害的抗性，并且能够适应高原环境（high 
altitudes） 
（包括海拔20,000尺的地方）。

**天鹏之翼 Soar**

在11级后，在你助跑起跳后，你可以立刻以附赠动作展开双翼，一对如鹏鸟般的灵体翅膀显现在你的身后并持续存在至你的下个回合结束，或至你重新落回地上为止。在双翼存在期间，你拥有等同于你步行速度的飞行速度。

**飓风之护 Hurricane**

在15级时，风与你之间的联系变得无比紧密，风中的精魂甚至能化为一场魔法的狂风来守护你。当你受到伤害时，你可以用你的反应为你自己提供该次伤害的抗性（如果这次伤害是坠落伤害，那么效果将改为免受此次伤害）。此时，你可以选择15尺内任意数量的生物，每个所选生物都必须进行一次力量豁免（DC 使用你的游侠法术 DC）。豁免失败的生物将受到等同于你游侠等级点的钝击伤害并受击倒地。豁免成功时，生物将受到豁免失败时伤害的一半，且不会被击倒。

使用此特性时，你还能让风之精魂立刻带你飞到30尺内的一处可见的未被占据的空间中（不会引发借机攻击）。

此特性一经使用，直至完成短休或长休你都无法再次使用。
