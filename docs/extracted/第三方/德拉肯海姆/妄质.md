# 妄质

妄质 Delerium
妄质是陨石坠落遗留下来的一种魔力矿物。它表现为一种带有锋利边缘、可反射第八虹色光的半透明晶体。这些怪异的晶块会用刺耳的音调低声呢喃，当夜幕降临，或它们被魔法影响时，会闪烁出耀眼的光芒。淤积在城市里的这些晶体在德拉肯海姆随处可见，通常与石制街道和建筑融为一体的它们看起来就像晶状的苔藓。
通常一块妄质晶片约有手指大小。妄质水晶则能有拳头大小或稍大些，而妄质晶簇可以大到和一个南瓜大小相近。巨型晶簇团甚至可以长到比一个人还要高。

妄质特性 Delerium 
Properties
所有妄质样品无论大小，都具有以下特性：
·免疫暗蚀，毒素和心灵伤害，以及非魔法武器的钝击，穿刺和挥砍伤害。
·对强酸，寒冷，火焰和闪电伤害，以及魔法武器的穿刺，挥砍伤害具有抗性。
·对魔法武器造成的钝击伤害具有易伤。
·除非被放置在反魔法场中，否则当妄质的生命值归零时，它将会瓦解并碎裂成一片无用的灰烬。然而，被摧毁的妄质晶体会随机释放一种奥术异常（见下文）。

妄质危害 Delerium 
Hazards
当一个类人生物在没有防护措施（如着装手套、使用钳子）的情况下在回合内第一次触碰妄质，或以接触妄质的状态结束他们的回合时，他们必须通过一次 DC 10的体质豁免，失败则受到1d6点暗蚀伤害并获得一层污染等级（见附录 C）。
开采妄质 Delerium 
Harvesting
 在德拉肯海姆发现的妄质通常都已经与地面或石制建筑融为一体，并且它们可以被铲子，镐，凿子等手持工具小心地开采。请参考表格来决定角色们需要花费多长时间才能开采一块妄质。缺少合适工具的情况下，开采妄质花费的时间是通常的十倍。如果缺少重型设备或强力的魔法，巨型晶簇团无法被开采。
妄质的诸多用途 Uses 
for Delerium
妄质具有巨大的奥术潜力。除了可以用来交易，它们还具有以下用途：
施法成分 Spell 
Component。当一名角色施展术士，魔契师或法师法术列表的任何法术时，妄质可以被用作奥术法器或替代任何的法术耗材。当施展一个法术耗材有耗价的法术时，作为代替，施法者可以使用等价值的妄质来施展此法术。
魔法物品 Magic 
Items。妄质是制作所有种类魔法物品的绝佳材料。本书中描述了几件新的魔法物品，除此之外任何核心规则所包含的魔法物品都可以用妄质来制作。晶片、晶块、水晶、晶簇大小的妄质分别可以用来制作非普通、珍稀、极珍惜、传说品质的魔法物品。根据 DM 的判断，可以施展五环或更高环阶的施法者可以在休整期学习如何将妄质加工成他们自己的魔法物品。
稳定妄质 Stable 
Delerium。将妄质加工成魔法物品的过程可以使这些水晶变得稳定。除非另有说明，否则角色们不会因触碰或持握稳定妄质而受到伤害或提升污染等级。伤害或摧毁稳定妄质制成的魔法物品的方式和摧毁一件该魔法物品正常版本的方式相同。
妄质尘 Delerium 
Dust。妄质尘是一种将妄质水晶互相摩擦而制成的细微粉末。除非在反魔法场中进行，否则这个制作过程非常危险。妄质尘可以用来炼金，或者作为法术的施法成分，充当酿造药水时的一种试剂，或混合进书写法术卷轴所用的墨水中去。
简易武器与弹药 Improvised 
Weapons & Ammunition。妄质晶片可以充当投石索的弹药或临时当作短棒使用。这类武器和弹药将额外造成1d6点暗蚀伤害，同时若一个未受污染的生物一旦被击中，需要通过一次 DC 10的体质豁免，否则将提升一层污染等级。妄质制成的弹药一经使用即被消耗掉。
