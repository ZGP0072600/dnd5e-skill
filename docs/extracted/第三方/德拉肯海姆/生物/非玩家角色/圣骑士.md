# 圣骑士

圣骑士 Cavalier
这些正义骑士驾着身披重铠的战马或双翼强劲的狮鹫作战。
圣骑士 Cavalier 中型类人，任意阵营 AC 19（板条甲，盾牌）HP 90（12d8+36）速度30尺力量18（+4）敏捷10（+0）体质16（+3）智力10（+0）感知12（+1）魅力15（+2）豁免：感知+4，魅力+5技能：驯兽+4，运动+7，察觉+4，游说+5感官：被动察觉14语言：任意两种挑战等级5（1,800XP）熟练加值+3勇猛 Bravery。圣骑士为抵抗恐慌 frightened 而作的豁免检定具有优势。圣光之怒 Light‘s Fury。圣骑士的武器攻击在命中时额外造成9（2d8）光耀伤害。*
老练骑手 Skilled Rider。圣骑士可以强迫针对其坐骑的攻击转而针对圣骑士自己。施法 Spellcasting。圣骑士是一位8级施法者，使用魅力作为施法属性（法术豁免 DC 13，法术攻击命中+5）。它准备了以下圣武士法术：1环（4法术位）：祝福术 bless，命令术 command，疗伤术 cure wounds2环（3法术位）：印记斩 branding smite，寻获坐骑 find steed 动作 Actions 多重攻击 Multiattack。圣骑士发动两次近战攻击骑枪 Lance。近战武器攻击：命中+7（攻击5尺内的目标时有劣势），触及10尺，单一目标。命中：10（1d12+4）穿刺伤害外加9（2d8）光耀伤害。标枪 Javelin。近战或远程武器攻击：命中+7，触及5尺或射程30/120尺，单一目标。命中：7（1d6+4）穿刺伤害外加9（2d8）光耀伤害。
