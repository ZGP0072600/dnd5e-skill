# 盗贼女王

盗贼女王 Queen 
of Thieves
中型类人，中立邪恶
AC 23（诡诈防御，法师护甲 mage armor）
HP 
120（20d8+20）
速度30尺
力量10（+0）敏捷20（+5）体质13（+1）
智力20（+5）感知16（+3）魅力19（+4）
豁免：敏捷+10，智力+9
技能：特技+10，奥秘+10，欺瞒+14，洞悉+8，察觉+8，游说+9，巧手+10，隐匿+15
状态免疫：魅惑 charmed
感官：黑暗视觉60尺，被动察觉18
语言：通用语，矮人语，精灵语，地底通用语
挑战等级15（13,000XP）熟练加值+5
特殊装备 Special 
Equipment。盗贼女王着装了一枚反侦护符 amulet of proof against detection and 
location 和间谍总管的印戒 Spymaster's Signet。她带着一件+3刺剑+3 rapier。
诡诈防御 Canny 
Defense。当盗贼女王身着轻甲或未着甲，且未持用盾牌时，她的 ac 包括其智力调整值。
天生施法 Innate 
Spellcasting。盗贼女王的天生施法属性为智力。她天生可以施展以下法术且无需任何法术成分：
随意：侦测思想 detect thoughts，易容术 disguise 
self，暗示术 suggestion
传奇抗性 Legendary 
Resistance（3/日）。盗贼女王豁免失败时，可以将其改为豁免成功。
记忆大盗 Memory 
Thief。生物不会察觉盗贼女王正在阅读其思想或是将其魅惑 charmed。此外，每当盗贼女王施展的令生物陷入魅惑 charmed 状态的法术结束时，她都可以令受影响的生物失去其记忆。目标必须成功通过一次 DC 18的智力豁免否则会遗忘至多8小时的，被盗贼女王魅惑 charmed 期间的记忆。失去的记忆可以被一道法术医疗术 Heal 或类似魔法恢复。
偷袭 Sneak 
Attack（1/回合）。盗贼女王对目标造成伤害时，如果其攻击检定具有优势，或者如果其攻击检定没有劣势且它在该目标身边5尺范围内有一个未处于失能 incapacitated 状态的盟友。满足以上条件时，盗贼女王造成额外18（5d6）点伤害。
施法 Spellcasting。盗贼女王是一位20级施法者，使用智力作为施法属性（法术豁免 DC 18，法术攻击命中+10）。她准备了以下法师法术：
戏法（随意）：颤栗之触 chill touch，法师之手 mage 
hand，传讯术 message，次级幻象 minor 
illusion，魔法伎俩 prestidigitation
1环（4法术位）：羽落术 feather fall，法师护甲 mage armor，护盾术 shield，睡眠术 sleep
2环（3法术位）：镜影术 mirror image，迷踪步 misty step，识破隐形 see invisibility
3环（3法术位）：气化形体 gaseous form，催眠图纹 hypnotic pattern，高级幻影 major image
4环（3法术位）：任意门 dimension door，高等隐形术 greater invisibility，变形术 polymorph
5环（3法术位）：支配类人 dominate person，探知术 scrying，伪装术 seeming
6环（2法术位）：摄心目光 eyebite，群体暗示术 mass suggestion
7环（2法术位）：力场监牢 forcecage，投射幻影 project image
8环（1法术位）：弱智术 feeblemind
9环（1法术位）：预见术 foresight
动作 Actions
+3刺剑+3Rapier。近战武器攻击：命中+13，触及5尺，单一目标。命中：13（1d8+8）穿刺伤害。
匕首 Dagger。远程武器攻击：命中+10，射程20/60尺，单一目标。命中：7（1d4+7）穿刺伤害。（注：括号内的数字疑似写错，实际应为1d4+5）
反应 Reactions
蓄意诱导 Misdirection。盗贼女王在即将受到伤害时立即变为隐形 invisible，并传送至60尺内一处她可见的未被占据的空间。此时，一道幻象分身出现在她先前所在的空间，持续一分钟。盗贼女王的隐形 invisible 将持续到1分钟后，或是直到她进行攻击或施放法术。
传奇动作 Legendary 
Reaction 
盗贼女王拥有3传奇动作，用于选择执行下列动作选项。每次执行传奇动作时，一次只可以选用一个选项，且只能在另一生物的回合结束时执行。盗贼女王在她的回合开始时恢复其所有已消耗的传奇动作数。
攻击 Attack。盗贼女王进行一次近战或远程攻击。
移动 Move。盗贼女王移动至多等于其速度的距离且不触发借机攻击。
