# 诈术师

诈术师 Scoundrel
“当你像我一样总是注意环境，那你说不定就能有机会学到一两种诈术。孤注一掷也许能让你赢个大的，但更可能是一无所获。”
——“黑杰克”梅尔
 
诈术师 Scoundrel 中型类人，任意阵营 AC 14（皮甲）HP 44（8d8+8）速度30尺力量10（+0）敏捷17（+3）体质12（+1）智力14（+2）感知10（+0）魅力15（+2）技能：特技+5，欺瞒+6，巧手+5，隐匿+7感官：被动察觉10语言：盗贼黑话外加任意两种语言挑战等级3（700XP）熟练加值+2灵巧动作 Cunning Action。每回合一次，诈术师可以使用附赠动作执行撤离、疾走、躲藏三个动作其中之一。偷袭 Sneak Attack（1/回合）。诈术师对目标造成伤害时，如果其攻击检定具有优势，或者如果其攻击检定没有劣势且它在该目标身边5尺范围内有一个未处于失能 incapacitated 状态的盟友。满足以上条件时，诈术师造成额外14（4d6）点伤害。施法 Spellcasting。诈术师是一位3级施法者，使用智力作为施法属性（法术豁免 DC 12，法术攻击命中+4）。它已知以下法师法术：戏法（随意）：法师之手 mage hand（生成的手是隐形的），次级幻象 minor illusion，魔法伎俩 prestidigitation1环（4法术位）：魅惑类人 charm person，易容术 disguise self，睡眠术 sleep2环（2法术位）：隐形术 invisibility，暗示术 suggestion 动作 Actions 短剑 Shortsword。近战武器攻击：命中+5，触及5尺，单一目标。命中：6（1d6+3）穿刺伤害。手弩 Hand Crossbow。远程武器攻击：命中+5，射程30/120尺，单一目标。命中：6（1d6+3）穿刺伤害。
