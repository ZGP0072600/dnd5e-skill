# 城市游侠

城市游侠 Urban Ranger
“日复一日地训练是为了在这危险城市中活下去，因为你我所背负的不仅仅是自己的生命，而只需要一次失手的攻击、一次在道路间迷失、一瞬间的疏于观察，都足以让你在危险的废墟中丧命。”
——覆影明灯的佩特拉·朗
 
阴森冷峻的城市游侠在暗巷与屋顶间蛰伏，如同其他游侠在林间小路与枝干树梢间穿行一般。它们擅长在建筑物间跃行，攀爬墙壁更是轻而易举。
城市游侠 Urban Ranger 中型类人，任意阵营 AC 15（镶钉皮甲）HP 52（8d8+16）速度30尺，攀爬30尺力量12（+1）敏捷17（+3）体质14（+2）智力11（+0）感知15（+2）魅力11（+0）技能：运动+5，察觉+6，隐匿+7，生存+6感官：被动察觉16语言：任意两种挑战等级3（700XP）熟练加值+2施法 Spellcasting。城市游侠是一位8级施法者，使用感知作为施法属性（法术豁免 DC 12，法术攻击命中+4）。它已知以下游侠法术：1环（4法术位）：疗伤术 cure wounds，云雾术 fog cloud，猎人印记 hunter's mark2环（3法术位）：行动无踪 pass without trace，荆棘丛生 spike growth 动作 Actions 多重攻击 Multiattack。城市游侠发动两次近战攻击或两次远程攻击。短剑 Shortsword。近战武器攻击：命中+5，触及5尺，单一目标。命中：6（1d6+3）穿刺伤害。长弓 Longbow。远程武器攻击：命中+5，射程150/600尺，单一目标。命中：7（1d8+3）穿刺伤害。
