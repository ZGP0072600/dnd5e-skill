# 乡野法师

乡野法师 Hedge 
Mage
”不是所有紫晶学院的毕业生都成为了学院的一员。部分毕业生离开学院，寻求自己的生活，悄悄利用魔法天赋追求自身兴趣。只要遵守律法和赦令，它们便能不受干扰地练习魔法。”
