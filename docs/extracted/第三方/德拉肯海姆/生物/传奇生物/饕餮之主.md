# 饕餮之主

饕餮之主 Lord of the Feast
这位怖人的白化加米尔犬怪身高超过了12英尺。他浑身生着黄白色的脏乱皮毛，其上沾染着无数战斗带来的斑斑血迹与一道道伤痕。他的铠甲由皮革、链条、刀片组成——其上还挂着一颗作为战利品的木乃伊头颅。如骨刺般凌厉且遍布倒钩的灰色鬃毛从他的头顶一路延伸到他那编着颅骨与骨片的白色长尾。他狰狞咆哮的面孔上长满了刀般锋利的獠牙，一对血红的眼瞳镶于其上，如同燃烧一般。饕餮之主提着一把越形的长弓，其上系着坚韧的肌腱。他背后的箭袋里满是同样巨大的箭支，有些布满妨碍行动的倒刺，有些连着便于牵扯的锁链，而另一些则使用了妄质晶块作为箭头。

大型怪兽，混乱邪恶 AC 18（天生护甲）HP 228（24d10+96）速度30尺，攀爬30尺力量18（+4）敏捷18（+4）体质18（+4）智力11（+0）感知18（+4）魅力13（+1）豁免：敏捷+9，体质+9，感知+9，魅力+6技能：运动+9，察觉+14，隐匿+14，生存+9感官：黑暗视觉120尺，被动察觉24语言：深渊语挑战等级15（13,000XP）熟练加值+5传奇抗性 Legendary Resistance（3/日）。饕餮之主豁免失败时，可以将其改为豁免成功。
再生 Regeneration。饕餮之主在其回合开始时恢复20点生命值，前提是它至少有1点生命值。
近身猎手 Close-Quarters Hunter。位于敌对生物5尺内并不会令饕餮之主在远程攻击上获得劣势。
猎兽感知 Predatory Senses。饕餮之主在感知（察觉）检定上具有优势。他可以感测120尺内所有隐形生物的位置。
