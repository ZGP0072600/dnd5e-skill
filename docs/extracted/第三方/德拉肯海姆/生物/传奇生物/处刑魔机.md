# 处刑魔机

处刑魔机 Executioner
 
“......这就是为什么我们不去戮石广场。”
————佩特拉·朗中尉
 
处刑魔机 Executioner 巨型构装，守序邪恶 AC 20（天生护甲）HP 405（30d12+210）速度40尺力量29（+9）敏捷11（+0）体质25（+7）智力10（+0）感知11（+0）魅力10（+0）豁免：力量+15，体质+13，感知+6伤害免疫：毒素，心灵；非魔法且非精金攻击的钝击、穿刺、挥砍状态免疫：魅惑，力竭，恐慌，麻痹，石化，中毒感官：黑暗视觉120尺，被动察觉10语言：理解通用语但不会说挑战等级20（25,000XP）熟练加值+6传奇抗性 Legendary Resistance（3/日）。处刑魔机豁免失败时，可以将其改为豁免成功。
不变形态 Immutable Form。处刑魔机免疫任何可以改变其形态的法术或效应。
魔法抗性 Magic Resistance。处刑魔机抵抗法术和其他魔法效应时进行的豁免检定具有优势。魔法武器 Magic Weapon。处刑魔机的武器攻击视为魔法攻击。
复生 Rejuvenation。如果处刑魔机被摧毁，它将会在一个小时内恢复其所有的生命值，除非残骸被使用了解离术 disintegrate。
动作 Actions 斩首铡刀 Guillontine Blade。近战或远程武器攻击：命中+15，触及10尺或射程30/120尺，单一目标。命中：30（6d6+9）挥砍伤害外加14（4d6）暗蚀伤害。目标必须成功通过一次 DC 21的体质豁免否则其生命值上限将承受等同该暗蚀伤害数值的减损。减损效应持续至该目标完成一次长休。如果该效应将目标的生命值上限降至0，则目标死亡。该武器会在发动一次远程攻击之后立即飞回处刑魔机手中。暗蚀毒气 Necrotic Vents（充能5-6）。处刑魔机排出一片暗蚀气体并覆盖自身半径30尺内的区域，区域内的每个生物都必须进行一次 DC 21的体质豁免，豁免失败者受到45（10d8）暗蚀伤害，豁免成功则伤害减半。失败于此豁免且差值为5或更高的生物还会获得一层污染 Contamination。传奇动作 Legendary Actions 处刑魔机拥有3传奇动作，用来选择执行下列动作项。它每次只可选择执行一个传奇动作项，且必须在另一生物回合结束时执行。处刑魔机在其回合开始时恢复所有消耗的传奇动作数。撼地前行 Lumber Forward。处刑魔机移动等于其速度一半的距离。攻击 Attack（消耗2动作）。处刑魔机使用其斩首铡刀发动一次近战或远程攻击。召唤幽影 Summon Shadows（消耗3动作）。处刑魔机指定至多六个它可见的因暴力而死的类人生物遗体。每个目标的精魄都将被唤起为一只幽影 Shadow 并出现在其尸体的空间或邻近一处未被占据的空间。这些幽影使用处刑魔机的先攻，并在其回合结束后立即行动。将持续存在直至1分钟后或是被摧毁，而处刑魔机被摧毁后幽影也会被摧毁。
