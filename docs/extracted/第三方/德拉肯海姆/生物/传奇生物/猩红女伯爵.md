# 猩红女伯爵

猩红女伯爵 Crimson Countess
猩红女伯爵是一位强大的鸟妖族长。她那猩红色的羽毛与头发很好辨认，同时她总是身着黑金相间的铠甲，手中提着她那迸射着电能的矛枪。
猩红女伯爵 Crimson Countess 中型怪兽，混乱邪恶 AC 17（天生护甲）HP 130（20d8+40）速度20尺，飞行60尺力量10（+0）敏捷21（+5）体质15（+2）智力13（+1）感知14（+2）魅力19（+4）豁免：体质+5，感知+5技能：特技+11，欺瞒+7，察觉+8伤害免疫：闪电，雷鸣感官：黑暗视觉120尺，被动察觉16（原文写错，实际计算应为18）语言：深渊语挑战等级8（3,900XP）熟练加值+3舞空者 Aerial Acrobat。猩红女伯爵在每个她的回合中都可以用附赠动作执行疾走，撤离或回避动作。敏锐视觉 Keen Sight。猩红女伯爵在基于视觉的感知（察觉）检定中具有优势。动作 Actions 多重攻击 Multiattack。猩红女伯爵使用其骇人尖啸。她随后使用她的禽爪或标枪发动两次攻击。禽爪 Talons。近战武器攻击：命中+8，触及5尺，单一目标。命中：11（1d10+5）挥砍伤害。骇人尖啸 Frightful Screech。任何由猩红女伯爵选择的位于其120尺内且并未耳聋 deafened 的生物都必须成功通过一次 DC 15的感知豁免否则陷入恐慌 frightened 状态1分钟。若一个生物通过了此效应的豁免或是其身上的此效应结束，则该生物在接下来的24小时内免疫猩红女伯爵的骇人尖啸。以此法被恐慌 rightened 的生物必须在其回合内执行疾走动作，并以此寻找最安全的路线远离猩红女伯爵，直至其无路可走为止。如果该生物在施法者视线范围外结束其回合，则其可以再进行一次感知豁免，豁免成功则该生物身上的此效应终止。标枪 Javelin。近战或远程武器攻击：命中+8，触及5尺或射程20/60尺，单一目标。命中：9（1d6+5）穿刺伤害外加14（4d6）闪电伤害。音鸣爆 Thunder Blast（充能5~6）。猩红女伯爵释放出一道雷鸣尖啸，覆盖一处15尺矩形区域。任何位于区域内的生物都必须进行通过一次 DC 15的体质豁免，豁免失败则受到14（4d6）雷鸣伤害，豁免成功则伤害减半。失败于此豁免的生物还会被推开10尺并陷入倒地 prone 状态。
