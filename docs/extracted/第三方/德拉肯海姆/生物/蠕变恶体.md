# 蠕变恶体

蠕变恶体 Protean Abomination
有时，生物的高速变异会超越其躯体的承受能力——这些可怜虫恐怖地爆开，形成一场腥风血雨，留下一团颤动的畸变肉瘤，其上生着呓语的大口与蠕动的眼球。它的肢体扭曲畸形，以无比奇诡的方式不断地膨胀、收缩着。
 
蠕变恶体 Protean Abomination 巨型异怪，无阵营 AC14（天生护甲）HP168（16d12+64）速度30尺，攀爬10尺力量21（+5）敏捷8（-1）体质18（+4）智力5（-3）感知10（+0）魅力5（-3）状态免疫：魅惑，恐慌，倒地感官：盲视60尺，被动察觉10语言：—挑战等级6（2,300XP）熟练加值+3吞噬 Absorption。当蠕变恶体杀死一个非不死也非构装的生物时，它会吞噬其遗骸并恢复20点生命值。无定形 Amorphous。蠕变恶体能够轻松穿过最窄1寸宽的狭窄空间而不用挤进去。蛛行 Spider Climb。蠕变恶体无需进行属性检定就能攀爬于困难表面，包括倒挂在天花板上。动作 Actions 多重攻击 Multiattack。蠕变恶体发动两次猛击攻击。如果两次攻击都命中一个大型或更小的生物，则该生物被擒抱 grappled（逃脱 DC 15），且蠕变恶体可以对其使用吞没。猛击 Slam。近战武器攻击：命中+8，触及5尺，单一目标。命中：19（3d8+5）钝击伤害。吞没 Engulf。蠕变恶体淹没一个被其擒抱 grappled 的大型或更小的生物。被吞没的生物陷入目盲 blinded，同时被束缚 restrained 且不能呼吸。目标生物必须在每个蠕变恶体的回合开始成功通过一次 DC 15的体质豁免，否则将受19（3d8+5）钝击伤害。蠕变恶体移动时，被吞没生物也会随之移动。蠕变恶体体内每次只能有一个被吞没生物。
