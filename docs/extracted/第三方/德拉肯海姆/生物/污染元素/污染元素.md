# 污染元素

污染元素 Contaminated Elementals
污染元素在德拉肯海姆之中随处可见，尤其是是那些骇异魔能汇聚之地。许多遭遇这些元素生物的人都将这些怪物描述为“城市本身活过来攻击我们”的象征。有报告称，废墟如同伸出双手般碾碎了冒险者，燃烧的火焰以自身的意志四下游荡狩猎，污染的水体蹒跚上地吞噬路人——就连街道上一阵清风吹来的空气也汇聚着污霭，化为变幻莫测的云雾尾随冒险者。这些魔法生物无时不刻都在提醒人们：这座城市里没有任何东西可以被信任，没有任何地方存在真正的安全。
