# 自走妄质晶簇

自走妄质晶簇 Walking Delerium Geode
大型元素，无阵营
AC 17（天生护甲）
HP 126（12d10+60）
速度30尺，掘穴30尺
力量20（+5）敏捷8（-1）体质20（+5）
智力5（-3）感知10（+0）魅力5（-3）
伤害易伤：光耀、雷鸣
伤害抗性：非魔法攻击的钝击、穿刺、挥砍
伤害免疫：暗蚀、毒素
状态免疫：力竭、麻痹、石化、中毒、昏迷
感官：黑暗视觉60尺，震颤感知60尺，被动察觉10
语言：深潜语
挑战等级5（1,800XP）熟练加值+3
自爆 Death 
Burst。如果有伤害将自走妄质晶簇的生命值降至0（光耀、力场、雷鸣伤害除外），晶簇体内的妄质晶体就会爆裂而出，形成一股污染能量的爆炸。每个身处晶簇周围30尺范围内的生物都必须通过一次 DC 15的体质豁免，豁免失败将受到42（12d6）暗蚀伤害并获得一层污染 Contamination。此外，这一爆炸还会触发一道随机的奥术异常 Arcane Anomaly （见第六章）。
遁地 Earth 
Glide。自走妄质晶簇可以掘穴穿过非魔法且未经加工的泥土及岩石中移动。以此使用掘穴速度时，自走妄质晶簇不会破坏其穿过的任何材质。
动作 Actions
碎晶猛击 Shard 
Slam。近战武器攻击：命中+8，触及10尺，单一目标。命中：14（2d8+5）钝击伤害。被此攻击命中的生物还必须通过一 DC 15的体质豁免，豁免失败将受到10（3d6）暗蚀伤害并获得一层污染 Contamination。等级。
