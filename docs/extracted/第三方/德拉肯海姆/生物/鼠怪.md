# 鼠怪

鼠怪 Ratling
“前方的黑暗隧道中，突然亮起了一双黄紫色光芒的双眼——紧接着，一双，又一双...那是一群毛茸茸的‘玩意’，它们身着裹着淤泥的服装，佩戴着油腻的皮革护具，向着你们冲锋了过来。它们用那啮齿类动物般的爪子挥舞着手中锈迹斑斑的刀刃，它们嘈杂的吱吱声中混杂着叽叽、喳喳、甚至尖叫——但最终都会齐声嚎叫着‘杀！杀！杀！’”
 
鼠怪是大号的变异老鼠，一种充满了狡猾与邪恶、由吞噬与繁殖的疯狂渴望所驱使的怪物。这些奇怪家伙的心灵饱受着懦弱的生存本能与天生的暴力冲动折磨。鼠怪们梦想着自己能吃到世间万物，它们也确实什么都吃——不过尤其钟爱吃人，特别是婴儿。虽然它们可以靠吃腐肉为生，但鼠怪也会活捉一些猎物带到巢穴作为新鲜的储备粮食。数以千计的鼠怪居住在这座城市里，它们会组成鼠群。在地下挖掘迷宫般、有着无数入口与环形通路的隧道。一个成功的鼠群的鼠口增长速度极快，因为这些家伙的幼崽只需要有充足食物就能在几周时间里快速成熟。

Tactics。若鼠怪发动攻击时，至少有一个未处于失能的盟友在攻击目标生物周围5尺内，本次攻击检定将具有优势。
