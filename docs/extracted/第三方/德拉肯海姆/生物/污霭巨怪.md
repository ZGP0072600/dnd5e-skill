# 污霭巨怪

污霭巨怪 Haze Hulk
妄质生骸仍会持续突变，持续损毁。最终，大多数生骸都只是崩解为一堆吱嘎作响的血肉团块，变成呓语之口。然而，某些个体却会成长到更大，更扭曲的形态。
污霭巨怪 Haze Hulk 大型异怪，无阵营 AC 15（天生护甲）HP 95（10d10+40）速度40尺力量21（+5）敏捷10（+0）体质18（+4）智力7（-2）感知10（+0）魅力6（-2）感官：黑暗视觉60尺，被动察觉10语言：通用语挑战等级4（1,100XP）熟练加值+2动作 Actions 多重攻击 Multiattack。污霭巨怪发动两次猛击攻击。猛击 Slam。近战武器攻击：命中+7，触及5尺，单一目标。伤害：14（2d8+5）钝击伤害。
 
变体：巨怪畸变 Hulk Mutations
一个污霭巨怪可能具有以下一套额外特性：裂体种 Gutbuster 投掷腐肉 Hurl Foetid Flesh（充能5~6）。裂体种从身上撕下一块异变血肉，掷向位于其120尺内的一处空间。任何位于该点10尺内的生物都必须进行一次 DC 14的体质豁免，豁免失败者受到28（8d6）暗蚀伤害，豁免成功则伤害减半。猎杀种 Hunter 隐身 Invisibility。猎杀种魔法性地变为隐形 invisible，直至其发动一次攻击或是其专注被打断（如同专注于一道法术）。任何其携带或着装的装备也会随之隐形。触须鞭笞 Lashing Tendrils。近战武器攻击：命中+7，触及60尺，单一目标。伤害：14（2d8+5）穿刺伤害，且若目标为一大型或更小的生物则其还会被擒抱 grappled（逃脱 DC 15）。猎杀种有两条触须，每条都可以擒抱一个生物。牵引 Reel。以一个附赠动作，猎杀种将每个被其擒抱 grappled 的生物向其拉近25尺。暴君种 Juggernaut 异变坚韧 Mutant Fortitude。当暴君种因受非光耀非重击伤害而使其生命值减至0时，它必须进行一次 DC 为5＋所受伤害值的体质豁免，豁免成功则生命值降至1。践踏冲锋 Trampling Charge。如果暴君种在猛击攻击之前直线向目标生物移动了至少20尺，则目标必须成功通过一次 DC 16的力量豁免否则陷入倒地 prone 状态。若目标倒地 prone，暴君种可以用一个附赠动作再执行一次猛击攻击。
