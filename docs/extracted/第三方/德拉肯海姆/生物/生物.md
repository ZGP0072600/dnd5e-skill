# 生物

**生物**

这里描述了一些在德拉肯海姆能发现的新怪物。本附录同样包括了很多基于核心规则的怪物，我们会在其中添加额外的特性和新的描述来让它们更符合黑暗奇幻的设定。不过，任何基于5e 的怪物都可以看作是德拉肯海姆街道上污染的产物，或者是被来自远在幽冥国度的污霭所魔法性地改变。运用以下修改来当做你将其他来源的怪物制作成你最喜欢的德拉肯海姆可憎之物的灵感。

固有特性 Mechanical 
Features

任何在德拉肯海姆中遭遇到的非类人生物都具有以下特性。这些特性不会改变怪物的挑战等级。

完全污染 Fully 
Contaminated。这些生物免疫污染 Contamination，并且在为对抗污染法术而进行豁免时具有优势。

污霭适性 Haze 
Adaptation。这些生物即使位于污霭中也仍能从长休中获得收益。它们在污霭和浓厚污霭中能看到两倍远的距离。

污霭依赖 Haze 
Dependence。这些生物每在污霭外活动24小时都会获得一层力竭 exhaustion。以此获得的力竭 exhaustion 不能在该生物位于污霭之外期间被移除。

妄质治疗 Delerium 
Healing。以一个动作，一只怪物可以触碰一块妄质晶块来恢复10（3d6）点生命值。一旦一只怪物以此法使用了一块妄质晶块，那么该晶块在接下来的24小时内无法再以此法使用。

这些好处不适用于起源于污霭之外的生物，也不适用于角色寻获或咒唤来的生物，例如魔宠或者召唤出来的怪物。

扮演特质 Roleplaying 
Traits

所有居住在德拉肯海姆的怪物都难逃异质的疯狂。许多生物因此变得渴望掠夺性且滥用暴力，但仍有少数生物维持着清醒的头脑。然而，即使是这些清醒的生物，它们的精神也并不稳定，时常在疯狂的边缘徘徊。它们不知自己凶恶的嗜好从何而来。所有的怪物都具有以下扮演特质：

个性 Personality 
Trait。我总是容易对周围的类人生物发怒或怀有敌意，残存的理智令我对此感到困惑和沮丧。

理念 Ideals。我能感觉到一股异样的冲动，渴望着污染或吞噬类人生物。

牵绊 Bonds。我痴迷于妄质晶体，希望能沐浴在它们那温暖的光芒之下。

缺陷 Flaws。我有一种或多种类型的永久疯狂，甚至于经常记不起几分钟前的自己在做什么。

一个生物对污霭中其他怪物的反应取决于它自己的本性。它可能是矛盾的、回避的、敌对的，甚至是卑躬屈膝的。

怪物调整 Monster 
Modifications

奥法邪灵 Arcane 
Wraith

许多死于德拉肯海姆的施法者都化为了无形且充斥着腐化的不死灵魄，但却仍保留着奥术的力量，只不过被扭曲成了邪恶，黑暗的形式。它们在废墟中游荡，渴望魔法的精粹和鲜活的生命能量。

·为邪灵 Wraith 添加施法 Spellcasting 特性。这令其的挑战等级提升为6。

施法 Spellcasting。奥法邪灵是8级施法者，使用魅力作为施法属性（法术豁免 DC 13，法术攻击命中+5）。奥法邪灵知晓以下术士法术：

随意：法师之手 mage hand，魔法飞弹 magic missile，次级幻象 minor 
illusion

3/日：闪电束 lightning bolt

1/日：催眠图纹 hypnotic pattern

陨石坑巨虫 Crater 
Wurm

这种潜地巨兽只会在陨石坑盆地中被目击过，它们有着花瓣状开裂的巨口，其上生着利刃般锋利的牙齿。它们如午夜之紫般色泽的厚实表皮上生着尖刺和闪烁着第八虹色之辉的粘液。那些从这巨兽的大口之下幸存下来的人讲述着这怪物在破土而出前引动的大地震颤，以及它们是如何一口将猎物吞噬殆尽。

·这些变异的紫虫 Purple 
Worms 使用以下能力替代其多重攻击和尾刺特性：

破阵 Breach。陨石坑巨虫使用其全部移动力。当它这样做时，它可以进入大型或者更小的生物所占据的空间。每当陨石坑巨虫进入一个生物所占据的空间，该生物都必须进行一次 DC 19的力量豁免。豁免成功者可以选择被推开到陨石坑巨虫现在所处空间的另一侧。如果生物选择不被推开，则将承受豁免失败的后果。

豁免失败者受到35（10d6）钝击伤害，且被推离10尺并陷入倒地 prone 状态。完成移动之后，陨石坑巨虫还可以用其附赠动作对一个倒地 prone 的生物进行一次啃咬攻击。

污化食道 Contaminated 
Gullet。每当一个被陨石坑巨虫吞下的生物在巨虫体内受到强酸伤害时，它都会获得一层污染 Contamination。

加米尔犬怪 Garmyr

加米尔犬怪曾经被认为和鬣狗人有关，这种生物是污霭笼罩中的德拉肯海姆废墟中特有的一种双足犬类生物。它们通常生着厚重的皮毛，毛色和特征从猎犬到狼一样不尽相同，但都有锋利的牙齿和爪子。它们组成了极为好战的掠夺战团，配备了从多次试图夺回城市却最终失败的军队中缴获的武器。这些野兽痴迷于它们与冒险家和覆影明灯的无数战斗。加米尔犬怪的战团将城内作为狩猎场，它们以追捕猎物时的刺激为乐，还喜欢在追捕猎物时无情地嘲笑它们。尽管战团之间经常发生内讧，但名为饕餮之主的骇人恶兽统治着所有加米尔犬怪，他是犬怪们的霸主，支配者与神灵。

·加米尔犬怪 Garmyr 使用鬣狗人 Gnoll 的数据，但将其生物类型改为怪兽。

怪诞巨妖 Grotesque 
Gargant

这些变异的人形怪物变得无比巨大。妄质碎片融入其扭曲可怖的身躯，其肌肉间涌动着奥法的能量。这些生物有着很强的领地意识，它们有时会从陨石坑中拖出巨大的妄质晶体，把它们像方尖碑一样立在废墟中。

·这种怖人之物使用石巨人 Stone 
Giant 的数据。它还有以下的额外特性，替代岩地伪装，这令其挑战等级提升至8。

再生 Regeneration。怪诞巨妖在浓厚污霭 Deep Haze 中开始回合时会恢复10点生命值。怪诞巨妖只会在以0生命值开始回合且不能再生时才会死亡。

污化射线 Contaminating 
Beam。怪诞巨妖魔法性地令其60尺内一个它可见的生物进行一次 DC 15的体质豁免。豁免失败则目标受到27（6d8）暗蚀伤害并获得一层污染 Contamination。

迷雾伪装 Misty 
Camouflage。巨妖在为躲藏进一片被迷雾笼罩的区域所进行的敏捷（隐匿）检定具有优势。

污霭行尸 Haze 
Husk

那些死在废墟中的人们无法安息，污霭迫使它们的尸体再度行走起来，在街道上游荡。它们的表情还停留在死前最后一刻看到陨星坠落的惊恐，可怖的能量从它们的眼睛，嘴巴和伤口中翻涌而出，妄质晶体已经跟它们的腐肉结合在一起。

·使用丧尸 zombie 的数据，并将其不死坚韧替换为以下的特性：

临终苦难 Death 
Burst。如果非光耀非重击伤害将污霭行尸的生命值降至0，它身体内的妄质晶体将会爆发出一阵污染能量。每一个在污霭行尸5尺内的生物必须通过一次 DC 10的体质豁免，否则受到7（2d6）暗蚀伤害。豁免结果低于 DC 5点或更多的生物还会获得一层污染 Contamination。

污霭尸妖 Haze 
Wight

污霭尸妖比污霭行尸更具有知觉和意识，但它们同样已经失去了曾经的生命和记忆。它们生前大多是死在污霭中的冒险者，军队士官或者强大武者。许多污霭尸妖仍坚信自己是伟大的英雄，且肩负着团结德拉肯海姆民众来对抗入侵者的使命。然而，在它们那已经被疯狂占据的大脑和双眼中，所有类人生物才是恐怖的怪物。

这些尸妖 wight 具有以下特性来替换生命汲取动作：

污化之触 Contaminated 
Touch。近战武器攻击：命中+4，触及5尺，单一生物。命中：10（3d6）暗蚀伤害。目标必须成功通过一次 DC 13的体质豁免，否则获得一层污染 Contamination。

创造污霭行尸 Create 
Haze Husk。尸妖指定一个10尺内由于暴力而死亡不到1分钟的类人生物。它将作为一具污霭行尸 Haze 
Husk 复活在5尺内一处未被占据的空间。污霭行尸受尸妖的控制，尸妖不能同时控制超过12只丧尸。

守塔巨龙 Tower 
Dragon

这些青年青铜龙 young 
bronze dragon 栖息在城市和德拉肯城堡的塔楼和护墙上。它们曾经是这座城市的崇高守护者，现在被污霭的意志所控制。它们具有以下额外特性，并且挑战等级提升至10：

·伤害抗性：毒素，心灵，非精金且非魔法攻击的钝击，挥砍，穿刺伤害。

·状态免疫：魅惑，力竭，恐慌，麻痹，石化，中毒。

·构装本质 Constructed 
Nature。守塔巨龙不需要呼吸，饮食和睡眠、

·拟形 False 
Appearance。守塔巨龙禁止不动时和一尊青铜雕像一模一样。

·完美追踪者 Faultless

·不变形态 Immutable

戍墙石像鬼 Wall 
Gargoyle

这些石像鬼 gargoyle 守护着德拉肯海姆的城墙，并具有以下特性：

·120尺真实视觉。

·完美追踪者 Faultless

扭曲女巫 Warp 
Witch

这些空虚的魂灵徘徊在它们死去的地方，寻找曾经的记忆。它们痛苦的精神让那些听闻其哀嚎的人精神崩溃。当它们附身在其他类人生物身体上时，会引发宿主的身体快速突变。

这些幽魂 ghost 用以下动作替代恐惧面容：

碎心哀嚎 Psychic 
Wail（充能5~6）。任何位于扭曲女巫30尺内的生物都必须成功通过一次 DC 13的智力豁免，否则受到10（3d6）心灵伤害并且陷入失能 incapacitated 状态直到扭曲女巫的下一个回合结束。

引发突变 Mutate 
Host。当扭曲女巫附身一个生物时，它可以用一个附赠动作来让其宿主的身体发生突变。被附身的生物必须成功通过一次 DC 13的魅力豁免，否则将会获得一层污染 Contamination。这个效应会直接造成一次随机突变 Random 
Mutation（见附录 C），而不需要使用掷骰来决定该突变是否发生。
