# 历史年表

历史年表 Historical Timeline

-12年

一颗金色彗星将塔尔娜引向苍牙山脉中的大天使加百列和米迦勒。塔尔娜成为第一位圣武士。

1年

圣塔尔娜殉难，她从巫王泽安德十三世的疯狂中拯救了世界。她殉难后，她的追随者分享了引导神圣之火的方式并开展了对抗巫王的战争。

约300年

巫王的统治终于崩溃。更多的人学会了使用神圣之火的光芒。这些忠实的牧师组建了神圣之火这一宗教信仰的雏形。

381年

新上任的神圣之火圣主母发誓再也不会让暴虐的巫师领主握住统治的权柄，赞同对奥术施法者进行压迫。

约500年

经过一个世纪的迫害，一群幸存的巫师 成立了紫晶学院，作为保护自己的秘密结社。

约600年

冯·德拉肯家族在德兰河沿岸建立了一个小小的王国，并在那里奠定了德拉肯城堡的基础。

687年

白银骑士团奉命对紫晶学院开展搜捕、猎杀。

约700年

紫晶学院支持的贵族家族与神圣之火信仰支持的家族之间爆发了血腥战争。

743年

炉明法令为巫师 、神职人员和贵族设置了明确的分隔，从而结束了战争。

约800年

冯·德拉肯家族灭绝，引发了长达150年的继承危机。

965年

冯·凯赛尔家族宣称并继承了德拉肯海姆的王位，建立了维斯特玛王国。

约1000年

伊利瑞亚的君主制崩溃；圣主母任命了一位摄政王。

1015年

德拉肯海姆的寰宇钟塔神秘地停止了。

1093年

卢克雷蒂亚·马西亚斯撰文预言塔尔娜彗星将提前回归，它的到来预示着神圣之火信徒新时代的黎明。她这番暗示天启的预言在神圣之火的主流神职人员中极不受欢迎，她的著作也因此被压制。

1111年

9月16日8点13分，陨星撞击了德拉肯海姆。

1112年

为收复德拉肯海姆而进行的一次尝试失败了，成千上万的士兵死在了污霭之中。少数返回的士兵则陷入了疯狂。

1113年

卢克雷蒂亚·马西亚斯在她已有成果的基础上进行了大幅扩展，出版了《天火圣约》。她早期预测的准确性为她赢得了大批追随者，她计划亲自前往德拉肯海姆朝圣。

1114年

在收复德拉肯海姆的两次军事行动都惨遭失败后，冯·凯赛尔家族幸存成员之间的继承危机爆发了，引发了维斯特玛内战。

1115年

在内战肆虐之际，几批冒险者前往德拉肯海姆废墟进行调查、侵吞掠取。意外的是，有些人活着回来了。他们说废墟里有着可怕的怪物，还带来了一种危险晶体的奇怪样本。

1116年

卢克雷蒂亚·马西亚斯完成了前往德拉肯海姆的朝圣之旅，亲自到达了陨石坑边缘。她返回伊利瑞亚传播她的教义。

1117年

紫晶学院开始雇佣冒险者收集陨星碎片进行研究。他们将这些奇怪的晶体命名为妄质。

1119年

曼弗雷德·冯·凯赛尔遇刺身亡。塞西莉亚·冯·凯赛尔神秘死亡。内战平息，没有明确的王位继承人。

1120年

卢克雷蒂亚·马西亚斯被神圣之火教会的圣主母认定为异端，被逐出教会。卢克雷蒂亚返回了德拉肯海姆。

1121年

更多的冒险者涉险进入德拉肯海姆废墟。烬木村成为寻宝者和杀手的聚集地。

1122年

关于天火的信仰迅速传播开来，越来越多的朝圣者踏上了前往德拉肯海姆的危险旅程。成功者寥寥无几。

1123年

第四非正式首都光复远征军——覆影明灯成立，并首次进入德拉肯海姆。

1124年

覆影明灯最终在旧牧羊者之门和德拉肯海姆驻地建立了行动基地，但却无法继续深入这座城市。

1125年

一小群天火信众在圣赛莉娜修道院建立了一个据点，为之后前往陨石坑的朝圣者提供帮助。

1126年

3月4日——德拉肯海姆之墟的冒险自此开始。
