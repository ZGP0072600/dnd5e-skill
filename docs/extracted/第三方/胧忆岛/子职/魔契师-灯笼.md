# 灯笼 The Lantern

与强大灵界生物缔结契约的魔契师，能够汲取流淌于灵界的魔法。他们因随身携带的一盏特殊灯笼而闻名，这些灯笼燃烧着其宗主的精粹。此器皿所散发的灵辉蕴含着灵界生物的特征，赋予魔契师一系列能力。

灵界生物与凡人结契，往往并非仅仅出于慷慨。那些强大到足以分出自身魔力的灵界生物，通常背后自有深意。或许精魂是由于某项交易或诅咒而不得不如此；也可能是它确实需要盟友——尽管它们未必愿意承认这一点。又或者，仅仅是它们在你身上看到了某种能取悦它们的东西。灵界生物的动机千变万化，与它们自身的形态一样变化多端。而真正强大的灵界生物，往往深不可测。

**启明灵光 Illuminating Aura**

1级起，你灵界生物宗主的精粹被灌注入灯笼之中，并以其微小形态显现。你可以将该灯笼作为你施展魔契师法术的法器。

你可以使用附赠动作激活你的灯笼，使其发出20尺半径的明亮光照以及额外20尺微光光照。此光照持续1分钟，或直至你松开灯笼，或选择提前结束此效应（无需动作）。在此期间，你获得以下增益：

- 每当你看到处于你灯笼明亮光照范围内的生物进行豁免检定时，你可以使用反应令其豁免检定获得等同于你魅力调整值（最低为+1）的加值。
- 每当一个位于你灯笼微光光照范围内的敌对生物豁免检定失败时，它会受到等同于你魔契师等级一半（向上取整）的光耀伤害。

一旦你使用了此特性，则直至你完成一次短休或长休前，都无法再次使用。

**揭示之光 Revealing Light**

1级起，你可以使用一个动作令你的灯笼照射出鲜艳的褐红色光芒，发出10尺半径的明亮光照以及额外10尺微光光照。处于灯笼的明亮光照范围内的隐形生物和物件会显形。灯笼以这种方式发光1小时，或直至你以附赠动作将其提前熄灭。

一旦你使用了此特性，则直至你完成一次短休或长休前，都无法再次使用。

危险器具 Dangerous 
Tool

6级起，你的启明灵光不再因你松开灯笼而提前结束。当你没有持握你的灯笼期间，你可以魔法性地控制它1分钟（无需动作），将其作为一道灵体武器 Spiritual 
Weapon 
（二环版本）来使用。在使用此特性期间，你的灯笼仍可作为你的法器，即使你已不再持握灯笼。

灵体武器 Spiritual 
Weapon 
的环阶会随着你在此职业等级的提升而提高，在10级时提升至四环，在15级时提升至六环。

一旦你以此方式控制你的灯笼，则直至你完成一次长休前，都无法再次这么做。你也可以消耗一枚魔契师法术位（无需动作）来再次控制它。

**神谕之辉 Oracle Illumination**

 
6级起，你可以使用一个动作将你的心智魔法性地连接到一个你可见或熟悉的光源，例如蜡烛或火把。该光源如同法术鹰眼术 Clairvoyance 
创造的传感器般运作，赋予你相同的增益。你可以如此使用该光源持续10分钟，或直至你再次使用此特性。

一旦你对一个光源使用此特性，则直至你完成一次长休前，你无法对此光源再次使用该特性。

**炽耀之辉 Brilliant Illumination** 

10级起，当你灯笼的启明灵光处于激活状态时，你可以使用一个动作令其光芒骤然迸发出势不可挡的耀眼闪光。除你之外，每个被你灯笼的明亮光照或微光光照（由你选择）所照亮的生物，都必须进行一次对抗你魔契师法术豁免 DC 的体质豁免检定。若豁免失败，则该生物会受到2d8+你魔契师等级的光耀伤害，并陷入目盲直至你的下一回合结束。若豁免成功，则该生物仅受一半伤害且不会陷入目盲。

一旦你使用了此特性，则直至你完成一次短休或长休前，都无法再次使用。

**愈加璀璨 Brighter Still** 

14级起，你可以随意激活你灯笼的启明灵光的能力。与启明灵光协同使用的特性，例如危险器具和炽耀之辉，仍受各自特性中描述的使用次数限制。

此外，当你持握你的灯笼期间，你可以使用一个动作令其照射出苍白蓝光，揭示一条通往一个你熟悉地点的发光奥术路径，如同施展法术寻路术 Find 
the Path 
 。

一旦你以此方式使用你的灯笼，则直至你完成一次长休前，都无法再次这么做。
