# 附灵 Spirit-Fused

即便是灵类的生命，也终将迎来终结。

成为附灵的人，鲜少是出于自愿，更多是命运的安排。要成为附灵，你必须先成为一个垂死之灵的容器——这个过程不可强求，因为至今无人真正了解这种现象为何发生，又是如何运作。

我们所知的是：当灵类的心智与躯体消散后，其精华便与战士共存，赋予他们超越常人的能力。那些能够驾驭这种力量的人，可以将潜藏的灵能注入物体之中，产生奇特的异能或强大的攻击。正是这些超凡的战斗技艺让这群独特的战士在胧忆岛上声名鹊起，引来无数关注与探询。

**奥能异象 Arcane Quirk**

作为强大灵类的宿主，你体内的魔力可能显得古怪而难以驾驭。虽然你已学会控制这些力量，但奥能仍可能自行迸发，产生种种奇特效果。

从以下奥能异象中选择一个或多个：

- 每当你使用本子职业的任何特性时，周围的初代科技设备会开始失控。
- 每当你进行武器攻击时，你随身携带的收音机或磁带播放器会播放一段音乐片段。
- 每当你受到伤害时，你会咳出泡泡。
- 精魂们会发现你的气味令它们难以忍受。
- 精魂们会发现你的气味格外吸引它们。
- 当你陷入昏迷时，你的身体会漂浮在离地1尺的高度，并可被风吹动。
- 当你对生物造成重击时，你的头部会随机变形为一个奇异的面容，比如拟人化的猪头、带着火焰发丝的赤红版本的你自己、飞头蛮的面容，或是一台 CRT 电视。这种变形持续到你再次变形或完成一次长休。

**引精注灵 Channel Essence**

从3级起，你获得了将灵之能量注入攻击的能力。你拥有四枚 d6骰，这些骰子被称为精华骰。

当你对生物进行武器攻击检定时，你可以消耗池中的一枚骰子并将其加到攻击检定上。或者，你也可以消耗一枚骰子来将其加到攻击的伤害上。你可以在进行攻击检定前或之后选择这么做，但必须在任何攻击结果生效之前决定。

当你完成一次短休或长休后，你恢复所有已消耗的精华骰。

在本职业的特定等级，你的会获得额外的 d6精华骰：7级（5d6），10级（6d6），15级（7d6），以及18级（8d6）。

**引灵御物 Object Channeling**

从3级起，你获得了将体内灵之能量注入物体的能力，能够产生各种效果。你学会一个你所选择的引灵选项（见下文"引灵选项"）。此外，你获得一件可以用来引导该选项的自选物品。

使用此特性后，你必须完成一次短休或长休才能再次使用它。

你同时获得回收技能的熟练。

**聚灵之躯 Spirit Battery**

7级起，你习得戏法电击术 Jolt，但有以下改动：

- 当你以机器为目标施展此法术时，机器的供能时间为1小时而非18秒。
- 你的法术攻击调整值等于你的体质调整值+你的熟练加值。

另外，在你的每个回合中，当你以攻击动作进行武器攻击并命中目标时，你可以用电击术的伤害替代武器伤害，每回合限一次。当你这样做时，你用电击术的伤害骰（造成闪电伤害）代替武器的伤害骰，且你不能从其他特性或法术获得任何额外的攻击伤害加成。

你的生物类型被视为灵类 spirit。

**强效引灵 Improved Object Channeling**

7级时，你学会一个额外的引灵选项，且现在可以使用等同于你熟练加值次数的引灵御物特性。当你完成一次短休或长休后，你恢复所有已消耗的使用次数。

**御灵护体 Improvised Armor**

从10级起，你学会将能量注入你所穿戴的物品中，使其在短暂时刻变得比钢铁更坚硬。当你被攻击命中时，你可以用你的反应投掷一枚精华骰，并将投掷结果加到你对抗该次攻击的 AC 上，这可能导致攻击失手。

**灵能充注 Backup Battery**

15级时，你学会汲取体内的灵之能量来获得迅速的力量潮涌。你获得一次额外的回气 Second 
Wind 特性使用次数。当你的生命值降至低于你战士等级且还有回气次数可用时，你可以选择立即使用一次回气（无需动作）。

**废品大师 Junk Master**18级起，你从引灵御物特性中获得一个额外的引导选项。

此外，当你进行先攻检定且没有剩余的引灵御物使用次数时，你恢复该特性的两次使用次数。

**引灵选项 Channeling Options** 选项按字母顺序排列。

**气囊包裹 Ballooning Bag。** 你可以使用附赠动作，使一个你正持有或随身携带的包袋（如帆布袋、背包、腰包或挎包）迅速充满空气。该包袋随后向上漂浮，将你提离地面并向正上方漂浮最多30尺。你将在该处悬停，直到你放开包袋（无需动作）或直到你下回合开始，此时包袋会缓慢飘回地面。

另外，当坠落时，你可以使用反应使包袋膨胀，获得羽落术 Feather 
Fall 的效果。

**缠绕绳索 Entangling Cord。** 
当进行攻击动作时，你可以选择用一次缠绕打击替代其中一次攻击。选择一个具有卷须状特征的物品（如卷尺、游戏手柄、延长线、绳索或锁链）。该物品可以在你身上或在你能看见的30尺范围内。

 
 当你这样做时，该物品会活化并向你能看见的一个位于其15尺内的生物伸展。使用你的体质调整值加你的熟练加值进行一次远程攻击。若命中，目标被拉向该物品最多15尺。

** 寻物感知 Find the Thing。**以一个动作，你通过导航物品（如罗盘、GPS 或地图）引灵。这让你能够施展寻物术 Locate 
Object ，但有以下改动：

- 该法术无需专注。
- 法术持续时间为1小时。
- 法术可以探测1里内的物品，而非1000尺。

**初代坐骑 First Age Steed。** 
你执行一个仪式，与一件特殊的自选载具（如自行车、助力车或类似可乘坐物品）建立联系。此载具只能承载一个生物，具有60尺的步行速度，且没有飞行速度。

以一个动作，你可以召唤该载具到你身边。被召唤时，载具处于无法破坏状态，1小时后会在一阵烟雾中消失。

** 护镜视觉 Goggled Vision。** 
当戴着眼部遮蔽物（如滑雪镜、眼镜、太阳镜或焊接面罩）时，你可以使用一个动作在60尺范围内，将微光视作明亮光照，将黑暗视作微光。在黑暗中你无法分辨颜色，只能看到灰度。

此外，你在进行智力（调查）检定判断是否为幻象时具有优势。这些效果持续1小时。

** 跃进球鞋 Leaping Sneakers。**当穿着一双鞋时，你可以使用附赠动作为你的脚步注入弹性。1小时内，你获得跳跃术 Jump 
的效果。

** 口袋声音 Pocket Voice。** 
你花费1分钟执行仪式，将你的声音绑定到一个你能看见的、看起来能发出声音的物品上（如音响、耳机、带嘴的雕像、人像画、收音机或首饰盒）。

你的声音会与该物品保持绑定1小时，或直到你以附赠动作解除效果。绑定期间，你说的每句话都能从该物品中被听到。

** 喷射推进 Water Jettison。** 
以一个附赠动作，你从随身携带的容器（如水袋、汽水罐、喷水瓶或喷雾瓶）中喷出水雾。这会使你沿直线向后移动10尺，并获得撤离动作的效果。
