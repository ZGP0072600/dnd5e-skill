# 腹酿道途 Path of the Belly Brewer

这些野蛮人擅长即兴酿造药水——而工坊就在他们的肚子里！当别人费尽心思、细心研制药水时，他们却能借助自己的饥饿，将原料中潜藏的魔力瞬间激发出来，且无需任何预先准备。 

那些天生就总是肚子打雷的人往往会发现寻常食物难以真正填饱他们，他们需要更为奇特的食物。在上一个世代，一群胃口异于常人的冒险者踏上了旅途，去寻找能够满足他们那似乎无尽的食欲的方法。正是在这段旅程中，他们发现了“腹酿”之术：将特定的食材一起吞下，不仅能暂时平息肚子的咕咕声，还能产生类似于魔药的效果——从此，这一奇术开始名声大噪。

腹中精酿 Belly 
Concoction 自3级开，你学会了一门来之不易的腹酿之术，这是一种奥术技巧：通过吞咽、混合并在体内酿造魔法药剂，从而创造出多种多样的特殊效果。 

当你进入狂暴时，你将精力集中，催动腹中精酿的魔力。选择“战斗”、“实用”或“妙想”之一，并获得对应精酿的效果（见下文）。你在每次进入狂暴都可重新选择腹中精酿的效果。

**战斗 Combat**

你的身体产生了独特的变化，它魔法性地改变了你的肉体，使其具有类似橡胶的特性。在这种弹性状态下，当你被远程攻击命中时，你可以花费你的反应用肚子将其弹开。当你这样做时，该次攻击对你造成的伤害减少1d12+你的体质调整值+你的野蛮人等级。 

此外，当你在你的回合内执行一次近战武器攻击前，你可以使用一个附赠动作拉伸手臂，使你的触及距离增加5尺，直到本回合结束。当你以此方式第一次命中一个目标时，该次攻击额外造成1d6点该武器同类型的伤害。

**实用 Utility**

你胃中正在反应冒泡的魔药被完美混合，能够溶解并吸收任何你能吞下的东西。以一个附赠动作，你可以吞下任何能被塞进你嘴里的物件。当你这样做时，作为该附赠动作的一部分，你可以消耗一枚生命骰治疗自己。掷骰，然后加入你的体质调整值，并回复等同于骰值总和（至少为1）的生命值。以这种方式被吞下的物件将被摧毁，前提是该物件不是魔法物品。你可以使用此特性次数等于你的熟练加值，并在完成一次长休后恢复所有已消耗的次数。

**妙想 Whimsy**

你专注于体内的魔法精酿，却产生了一系列稀奇古怪、难以控制的效果。在此状态下，每当你的回合开始时，你获得以下效果之一：

- 你瞬移至距离你10尺范围之内的一个无人占据的可见空间。
- 你受到1d4暗蚀伤害；在本回合结束前，你进行的下一次属性检定获得等同于掷骰点数的加值。
- 你可以立即花费一个附赠动作来执行协助动作。

此外，伴随妙想特性，你会获得一个随机的视觉效果，例如你的头发变成一团苔藓、一只幽灵鸟在你头顶盘旋，或一个跳舞男人的纹身出现在你身上并在体表四处游走。

**副作用 Side Effects** 从3级开始，由于你已经摄入了不计其数的魔法原料，你因而受到了一些魔法副作用的影响。你习得戏法魔法伎俩 Prestidigitation 和奇术 Thaumaturgy。

**快速酿造 Quick Brew**

从6级开始，你学会更好地控制腹中精酿的效果。 

选择一种你通常会饮用的普通品质魔药。你可以用一个附赠动作启动你所选魔药的效应。一旦你启动了其效应，你不能再次这么做直到你完成一次长休。如果魔药的效应要求执行豁免，豁免 DC 等于8+你的熟练加值+你的体质调整值。 

每当你在此职业提升一级，你可以将你所选的魔药更换为另一种。

**长时药效 Lingering Effects**10级时，你腹中恒久存在的秘酿效力将强大到即使在你不处于狂暴状态时也会为你带来好处。 

当你完成一次长休后，选择以下状态之一：目盲、魅惑、力竭、恐慌、麻痹、石化、中毒、倒地、束缚或震慑。在你以此特性选择一种其他状态之前，你在对抗该状态的豁免中具有优势。

**强力快速酿造 Mighty Quick Brew**

自14级开始，你在使用快速酿造特性时习得一种额外的普通魔药和一种非普通魔药。 

每当你在此职业提升一级，你可以将你所选的魔药更换为另一种。
