# 护甲 Armor

**树皮甲 Bark Armor。** 当你穿着树皮甲时，你在林地中进行隐匿检定时具有优势。

**潜水甲 Divers Armor。** 当你穿着潜水甲时，除非你的力量达到或超过18，否则你的移动速度降低10尺。此外，当你穿着潜水甲时，若有人对你施放法术水下呼吸 Water 
Breathing 
 
，该法术对你的持续时间翻倍。

**锅盖 Pot Lid。** 
当你手持一件灵巧或轻型近战武器进行攻击动作时，你可以使用附赠动作用另一只手中的锅盖进行攻击。命中时，锅盖造成1d4钝击伤害。除非调整值为负数，否则你不能将属性调整值加到附赠攻击的伤害上。
