# 魔女学徒

## 魔女学徒 Apprentice Witch

巫术之道神秘莫测，其修习者往往守口如瓶。无论是你主动寻求，还是某位魔女在你身上看到了什么特质，你都成为了一位魔女或魔女集会的学徒。在她们的指导下，你接触到了胧忆岛的民间魔法传统，学习了药剂和其他魔法制品的基础知识。虽然你曾在其门下修习一段时间，但你从未正式加入任何魔女集会，因此被视为一名无徽魔女。更多关于魔女集会的信息，请参见前文。

技能熟练项 Skill Proficiencies：奥秘、求生

工具熟练项 Tooll Proficiencies：炼金师工具

装备 Equipment：炼金工具、9份普通原料（你知晓它们的属性）、一顶帽子或一件源自学徒时期的长袍，以及9枚金花币。

特性 Feature：同门之谊 Sibling Student
。你有一位亲密可靠的朋友，他/她曾与你在同一位魔女门下学习，现已加入六大魔女集会之一。需要时，他/她会作为你与其所在集会或相关集会之间的联络人。
