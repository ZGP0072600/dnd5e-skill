# 潜水学徒

## 潜水学徒 Apprentice Diver

你现在或曾经是水手公会 Mariners’ Guild 的成员，这是一个无畏的水下探索者和航海家组织。作为公会中的潜水学徒，你对浅水区了解颇多——你知道哪些是危险的，哪些是美味的。你攀爬过岛屿周围的悬崖，探索过许多海滩，并学会了如何使用在海浪下生存所需的装备。更多关于水手公会的信息，请参见前文。

技能熟练项 Skill Proficiencies：运动、自然

**工具熟练项 Tool Proficiencies：** 潜水甲 Divers Armor

装备 Equipment：探索套组、一个箱子、爪钩、渔具、一枚标记你所受训练分会的派系硬币，以及11枚金花币。

特性 Feature：一日为会员，终身为会员 Once a Member Always a Member。当访问任一海岸潜者 Coastal Diver 分会时，你都可以免费获得食宿。此外，分会也会尽其所能协助你的事业。
