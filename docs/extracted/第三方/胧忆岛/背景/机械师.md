# 机械师

## 机械师 Mechanic

你大半生都在研究机械装置。对你而言，理解新旧技术都需要亲身实践。通过不断尝试，你学会了创造、修理和改装各种机械和电子设备，你的眼光也变得敏锐，能在遍布全岛的零散技术残片中识别出有用的部件。

技能熟练项 Skill Proficiencies：工程 Mechanics、回收 Salvage

工具熟练项 Tooll Proficiencies：获得三种工匠工具的熟练项

装备 Equipment：算盘、滚珠(1000个)、滑轮组、一瓶油、一件未完成的机械小玩意，以及15枚金花币。

特性 Feature：机械密语者 Machine Whisperer。你对机械有着独特的敏感，只需花费几分钟就能判断出机器的状态。被损毁的机械无法使用，只能拆解取件。受损的机械经过修理后可能重新运转，而功能完好的机械则能正常运作。就是这么简单！
