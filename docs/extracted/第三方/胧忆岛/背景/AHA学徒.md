# AHA学徒

## AHA 学徒 Apprentice of AHA

你曾在 AHA 组织中跟随考古学家、历史学家和档案管理员学习。这是一个由学者冒险家组成的团体，致力于探索胧忆岛的本质及其历史。在导师的指导下，你探寻着岛上谜团的答案。在学徒期间，你前往了 AHA 的一处或多处发掘现场，与组织一同工作，但从未成为正式成员。即使到现在，你在组织期间那些未解之谜依然萦绕心头。更多关于 AHA 的信息，请参见前文。

技能熟练项 SkillProficiencies：历史、调查。

工具熟练项 Tooll Proficiencies：获得一种工匠工具的熟练项

装备 Equipment：探索套组、5支蜡烛、铲子、信号哨、一本记录着你学徒时期旧研究的日志以及13枚金花币。

特性 Feature：睿智声望 Wise Reputation。你在实地考察期间接触过许多胧忆岛村落中的知识分子和富有创造力的人士，你对他们都比较熟悉。
