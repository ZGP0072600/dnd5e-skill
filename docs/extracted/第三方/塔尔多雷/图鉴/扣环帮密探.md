# 扣环帮密探

扣环帮密探 Clasp 
Operatives

夜幕降临，天色无光。都不管用，锁或石墙。像条小蛇，快快躲藏。别睁眼看，是扣环帮！ ——哄小孩的短诗

扣环帮——一个由间谍、小偷、刺客和走私者组成的秘密组织，是一群深受爱戴的民间英雄，也值得讲述睡前故事，但也是所行之事臭名昭著的一群阴险恶棍。虽然扣环帮的密探们肯定在偷偷为那些富人干一些违法的勾当，但它们也是一群在灰烬之王的暴政下拯救了无数塔尔多雷平民的浪客英雄。

他们现在正把自己的形象塑造成一个将社会团结在一起的“扣环”，毕竟它们也知道，犯罪组织的运行需要一个首先有一个正常运转的，文明的经济环境。

扣环帮割喉者 Clasp Cutthroat

当扣环帮的小偷和刺客们想要赚点外快或想“解放”某人的财产时，组织里的割喉者往往是它们的第一选项。然而，扣环帮绝不会为了让客户满意就把每个打手都称为割喉者。一个合格的割喉者必须足够耐心和认真。偷偷潜入，越过掩护，等待时机，并只在有利时刻进攻，毫不在乎荣誉感！而若是被发现，它们也会通过逃跑来误导目标——然后找到合适的新位置，在目标最意想不到的时候发动攻击。

扣环帮执行者 Clasp Enforcer

虽然扣环帮更喜欢在暗处行动，但是有些时候勒索目标需要足够的说服力。每当需要恐吓目标或展示武力时，扣环帮的执行者都给人留下了深刻的印象。执行者对承诺和收债的重视程度几乎与自己的收益一样高，因此任何违反与扣环帮的协议的人都必定会被执行者访问。“想知道执行者究竟执行了什么吗？”是一句流行的扣环帮笑话——在讲这些笑话的密探知道真相之前。执法者很难被打倒，更难被杀死。它们喜欢挥着大锤冲向目标，对那些不明智到没有还钱或逃跑的人进行“恰到好处”的恐吓。
