# 寒潮精魂

寒潮精魂 Cold Snap Spirit

许多还活着的精灵和大法师仍然记得失落的冰封岁月——塔尔多雷历史上的一个黑暗时期，大批冰封武者军团从元素位面涌出，意图将世界变成一片永冻的废土。寒潮精魂是雾凇领主埃雷文 Errevon 
the Rimelord，一个几乎把塔尔多雷据为己有的元素生物的残暴大军幸存下来的一个回响。他们徘徊在霜冻之地 Frostweald 那永冻的森林里，在崖堡那白雪覆盖的山峰上，在 Neverfields 的北极平原上。

离群精魄 Reclusive 
Spirits。当脱离元素军阀的邪恶命令时，寒潮精魂只不过是依靠本能活动的生物。它们无害地在物质位面的那些反映了它们起源位面的无尽霜冻的寒冷区域漂泊。

仇视热源 Heated 
Hatred。通常温顺的寒潮精魂会因为热量的存在而陷入可怕的狂乱之中。他们对火焰有一种本能的恐惧，即使是只是活物的温暖也足以让他们陷入凶残的狂暴之中。然而一旦热源消失，寒潮精魂就会迅速恢复到平静的状态。

不成熟的狂战士 Inchoate 
Berserkers。寒潮精魂是那些具有野心和支配意志的元素生物的完美士兵。他们对温暖气候的厌恶很快被对主人的忠诚所压倒，即使是温带地区的环境温度也足以让他们陷入持续的狂乱状态
