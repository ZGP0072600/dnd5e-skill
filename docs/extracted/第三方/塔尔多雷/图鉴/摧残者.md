# 摧残者

摧残者 Ravagers

摧残者是一支横行于分裂平原上的嗜杀邪教团，以叛神-毁灭者的名义（译注：即格乌什）制造屠戮。该邪教在被称为屠戮主宰的嗜血军阀领导，邪教的长期目标不得而知，但其成员在短期内满足于在平原上引动混乱。这种肆无忌惮的混乱屠杀令毁灭者兴奋，平原上的小村落害怕看到摧残者的身影出现在地平线上。然而，在 Westruun 的城墙内，市长利桑德拉·卡洛斯勋爵和她的议员们却因摧残者教团表现出了超越他们那嗜血军阀应有的组织能力而担心。是何人——或是何物让屠戮主宰离开阴影处？这些堕落的战士又在追求什么目的？

摧残者-屠戮主宰 Ravager Slaughter lord

分裂平原上的一些军阀渴求它们的神的奖励，于是便以血与荣誉之名转投摧残者教团。

其中少数有野心并付诸行动的人很快发展为邪教的主要派别，并被尊称为屠戮主宰。屠戮主宰渴求战斗，并尽可能多地亲自领导摧残者者的袭击。然而，他们也时常因能使自己变强的繁杂炼金仪式被迫留在自己的据点。

一支萨满和炼金师团队侍奉屠戮主宰，为他调配能增进他肌肉规模，增强他们的战术水平，并能赋予他们毁灭者神力护体的药剂，以此来人为地增幅他的力量和怒火。屠戮主宰不自然的体型是他们最明显的战力优势，甚至令他们可以双持使用哪怕最巨大的武器；而毁灭者的祝福则给予了他们仅需一个字便能引导能晃瞎甚至击败敌军的神术力量。

据推测，摧残者有四支分别由各自恐怖的屠戮主宰所指挥的主要力量，然而究竟有多少个屠戮主宰仍是未知数。

附赠动作 Bonus Actions

好斗 Aggressive。屠戮主宰向一个它可见的敌对生物移动至多等于其速度的距离。

传奇动作 Legendary Actions

屠戮主宰有3传奇动作，用来选择执行下列动作项。它每次只可选择执行一个传奇动作项，且必须在另一生物回合结束时执行。屠戮主宰在其回合开始时恢复所有消耗的传奇动作数。

攻击 Attack。屠戮主宰发动一次矛或双持巨剑攻击。移动 Move。屠戮主宰移动至多等于其速度一半的距离施法 Cast a spell（消耗3动作）。屠戮主宰施展一道天生法术。

摧残者-嗜杀刺客

探险家和雇佣兵在听到分裂平原上的游牧民族口中的可怖的故事后，通常会选择拿起武器对抗摧残者教团。他们准备与残酷的敌人进行更加残酷的战斗，也正因此对摧残者教团最可怕的武器——嗜杀刺客毫无防备。来自埃芒的新鲜面孔的战士们第一次在 Kymal 的酒馆听到这个名字时都会放声嘲笑，尽管老兵们都会闭上眼睛，匆匆地转头饮酒。

摧残者战士中的灵敏者若展现出了非凡的技能，便可能会被带到屠戮主宰面前，被迫喝下被教团称为骷髅之火的骇人酒精饮品。这种混合物燃尽了那些战士全部理性思维，把他们变成了完全服从的，渴求血肉的怪物。在战斗中，嗜杀刺客像老鼠一样成群结队，把敌人和旁观者开膛破肚，吞噬脏器。
