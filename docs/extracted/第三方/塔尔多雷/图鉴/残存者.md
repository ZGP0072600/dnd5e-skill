# 残存者

残存者 Remnants

残存者是一个肮脏的邪教，致力于帮助他们的巫妖领袖，耳语者（译注：即维克那），飞升为神。教徒认为，躯体的残缺和短暂的失败是揭开虚伪面纱、看清世界真相的必要条件——包括承认隐藏起来的真实世界已经属于耳语者。

惨痛的胜利 Bitter Victory。残存者在他们可怕的目标上既赢了又失败了，因为耳语者飞升了，但却放逐在神性之门外。他们现在将他命名为飞升者、被放逐者，或者称之为——因为所有其他神都是伪装者——“那位”，并耐心地为他的最终飞升做好准备。

残存者-受选之人 Remnant Chosen

所有耳语者的追随者都是狂热分子。他们花费数小时进行静坐，希望听到他们死去已久的主人低语出来自远方的命令。然而，只有极少数人能听到耳语者的声音蠕动着进入他们的脑海中，清空其他想法，让他们的头脑清晰，目标明确。这一目标很大程度上涉及残存者利用从阴影位面窃取的秘密来增加塔尔多雷邪教势力的计划

touch。近战法术攻击：命中+9，触及5尺，单一目标。命中：17（5d6）力场伤害外加凋零术法特性造成的额外伤害。如果该伤害将一个生物的生命值降为0，则该生物除了魔法物品以外的所有它所穿戴或携带的物品，都将被化作一堆细致的灰色粉尘。该生物只能通过完全复生术或祈愿术法术复活

残存者-教徒 Remnant Cultist

所有耳语者的信徒都要忍受迫使他们摘掉自己的一只眼睛，永远受制于领袖意志的可怖入会仪式。每一个崇拜者都能感觉到他们的邪恶神性就在物质世界之外，许多人声称在沉默之中听到 k 对他们说话。耳语者信徒仍然崇敬他们愤怒的主人，提供秘密和灵魂来得到他的青睐，并尽力学习如何隐藏自己所知的一切。教徒在自己身上培养邪恶，在他人心底唤醒恶念，鼓励他人行恶。他们希望在所有其他神的信徒心证播撒毁灭之种，直到世间仅存那些跪拜于耳语者面前之人。

施法 Spellcasting。教徒是一名11级施法者。他的施法属性是智力（法术豁免 DC 15，法术攻击命中+7）。它准备了以下法师法术：

6环（1法术位）：死亡法阵

不可知的隐秘 Unknowable Secrets。任何试图与教徒建立心灵链接，对教徒施展探知，或对教徒施展死者交谈者都会直接失败，且该生物受到21（6d6）心灵伤害。

独眼 One-eyed。教徒在对30尺外任何目标发动的攻击检定都有劣势法术窃取 Spellsteal。如果教徒成功用法术反制中和了其他生物的一道法术，直到它的下一回合结束，它都可以用自己的一个同级或更高的法术位施展这道被反制的法术且不需要任何材料成分。

动作 Actions 致伤匕首 Dagger of Wounding。近战或远程武器攻击：命中+6，触及5尺或射程20/60尺，单一目标。命中：5（1d4+3）穿刺伤害。该武器造成的伤害不能被恢复直到目标完成一次短休或长休，或直到目标被施展了再生术或类似的魔法效应。

教徒每回合有一次机会，在用该剑命中一名生物时对它造成流血效应。在目标的每回合开始时，它将因每层流血效应受2（1d4）的暗蚀伤害。之后，它可以进行一次 DC 15的体质豁免，成功则终止自己承受的所有流血效应。另外，流血的生物或是它周围5尺内的其他生物可以用一个动作进行一次 DC 15的感知（医药）检定，成功则终止该生物承受的所有流血效应。
