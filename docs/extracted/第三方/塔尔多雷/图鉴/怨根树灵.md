# 怨根树灵

怨根树灵 Wraithroot Tree

当古老或邪恶的实体需要保护具有重大影响的物品——特别是与其不朽交织在一起的物品时——它们会将这些秘密埋在怨根树灵附近，这样艾桑椎亚人便不敢践踏这些秘密。仇怨复生 Vengeful Restoration。怨根树灵是邪恶的死灵法术和异变仪式的产物。仪式开始于一只获得了智慧和机动性的启蒙树木。随后，这棵启蒙树木最初的快乐意识被惨痛地缩短——暗蚀能量在其身上流动，引来愤怒和邪恶以将其化为一株怨根树灵——其心中满是对毁灭活物的渴望。怨根分布 Wraithroot Sites。在塔尔多雷，怨根树灵在许多地方都有分布，包括白石城下的 Parchwood 林地，沿着崖堡山脉西南侧分别的 Torian 森林，克拉格海姆以北的灰色山谷以及灰守峰东南的腐化林地。阵营：中立邪恶

巨型植物

AC 16（天生护甲）

HP 270（20d12+140）速度30尺

力量23（+6）敏捷8（-1）体质24（+7）智力12（+1）感知16（+3）魅力12（+1）技能：察觉+8

伤害抗性：钝击，穿刺

伤害易伤：火焰

状态免疫：震慑，恐慌，魅惑，麻痹，中毒

感官：被动察觉18

语言：通用语，德鲁伊语，精灵语，木族语

挑战等级14（11,500XP）熟练加值+5

拟型 False Appearance。怨根树灵静止时与一颗普通的树木看起来一模一样。

怨灵步伐 Wraith Shift。怨根树灵在它的回合内时可以视同困难地形般地穿过其他生物和物件的空间。树灵在移动时不会触发借机攻击，且必须在一处未被大型或更大的物件占据的空间内结束移动。任何在树灵结束移动时位于其空间内的生物都会被推到最近的一处由该生物选择的未被占据的空间内。位于树灵所在空间内的中型或更小的物件会被会被推到最近的由树灵选择的一处未被占据的空间内。

动作 Actions

多重攻击 Multiattack。怨根树灵发动两次猛击攻击。

猛击 Slam。近战武器攻击：命中+11，触及10尺，单一目标。命中：16（3d6+6）钝击伤害。

投石 Rock。远程武器攻击：命中+11，射程60/180尺，单一目标。命中：28（4d10+6）钝击伤害。

怨灵风暴 Wraithstorm（充能5-6）。怨根树灵抽走周边区域内的生命能量，令生物短暂地脆弱。任何位于其30尺内的非构装且非不死的生物都必须成功通过一次 DC 16的体质豁免否则速度减半并获得对钝击伤害的易伤，直到树灵的下一回合结束。
