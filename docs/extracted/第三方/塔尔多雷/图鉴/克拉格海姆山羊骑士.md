# 克拉格海姆山羊骑士

克拉格海姆山羊骑士 Kraghammer Goat-Knight

崖堡山脉的峭壁和崎岖山路几乎无法以正常手段穿行。克拉格海姆的攀峰骑士——通常被称为山羊骑士——维护着山间的和平与正义。他们坚信自己的巨山羊坐骑能够征服崎岖陡峭的山坡，许多迷失在崖堡山脉中的旅行者都曾被山羊骑士救出，引导他们安全地通过可穿行的山路。这些骑士为众锤之父服务，但有时也会发现自己在维护克拉格海姆的法律和执行其神的指令之间陷入两难。

山羊骑士的坐骑 Goat-Knight Steeds

山羊骑士的坐骑使用巨山羊的数据但具有以下调整：?其智力为6（-2），且能理解矮人语。

?其生物类型是天族。

?其山羊骑士施展的射程为自身的法术同时也指定该山羊。

山羊骑士的坐骑在生命值降至0时消失不见，且不留下任何物质残余。山羊骑士也可以随时用一个动作解散该坐骑使其消失。无论何时，山羊骑士都可以施展寻获坐骑来召唤同一只坐骑。

of Protection。当山羊骑士或一个对其友好的生物在山羊骑士身边10尺内进行豁免检定时获得+2加值（已计入前文山羊骑士的豁免中）。提供该加值时山羊骑士必须保持意识清醒。

神佑 Divine 
Health。山羊骑士免疫疾病。

动作 Actions

多重攻击 Multiattack。山羊骑士发动两次战锤攻击。

战锤 Warhammer。近战武器攻击：命中+5，触及5尺，单一目标。命中：7（1d8+3）钝击伤害或是双手使用时的8（1d10+3）钝击伤害。
