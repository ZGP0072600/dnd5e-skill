# 魔像

魔像 Golems

身为不朽的无畏守护者，精工制造的魔像无误地执行主人的每一道命令。这些强大构装生物外表类似价人形，永远忠诚，且不会受制于痛苦与同情，也不需要食物与睡眠，更无视贿赂与讨还价，同样没有悔恨或犹豫。

贪婪起源 Covetous 
Origins

在奥法纪，魔法激发了许多神之子嗣们的傲慢，将奥术天赋视为具有神性的铁证。因此，许多凡人试图制造自己的“生命”。矮人最先取得了成功，他们将神性赠礼与技术知识结合在一起，活化出了自动运行的守卫，作为其先祖圣堂的守护者。矮人的知识很快被其他人窃取并扩展，创造魔像最终也不再是为了证明凡人也能像神一样塑造生命，而是为了保护生命、财富和秘密不受他人侵犯。

钴魔像 Cobalt 
Golem

做为一种经典奥术构装生物的独特形式，钴魔像是由那些认为铁是一种过于平凡的金属的奥术师制造的强大的装甲仆役。这些自动机械的起源是从天而降的陨石中小心地提取的原始钴，并通过铁磁炼金术冶炼成坚硬且富有光泽的银灰色金属。这件成品钴合金随后被模制并弯曲成装甲外壳，保护嵌着符文的肌肉般发条结构。位于中央的磁石为钴魔像供能，即为无机外壳上令其活化的魔法仪式提供能源。

Automata

锻炉守护者 Forge Guardian

正如矮人创造了最早的魔像一般，众锤之主塑造了锻造守护者，来守护设计出最初之矮人的神圣熔炉。这些罕见且巨大的构装生物由奢华的大理石制成，注入了精金，全身细节无可挑剔，是艾桑椎亚上每一个铁匠、锻造大师和工匠一生都想实现的最高艺术形式。

神性火花 Divine 
Spark。

锻炉守护者是一个力量巨大，不惧疲倦且无面孔的存在。它被一块源自众锤之父的神性火花赋予了生命，被珍重地存于环绕着奥术火焰和古老刻印的圆环之中。众锤之父的神性如同一道处于魔像体内的火山，将构装生物的血液化为熔岩。火不再能伤害它，而是治愈它。

神圣目的 Sacred 
Purpose。

隐于神圣的锻造厂中，锻炉守护者的肢体末端是锐利的钩爪和一柄刻着符文的刀刃。它可以帮助铁匠创造新的魔像，也可以践踏，粉碎，推开并击倒任何威胁这一神圣创造过程的人。

锻炉守护者也是任何希望参与众锤之父价值审判的人的对手。魔像的攻击会致盲未来的英雄，而它们必须冒着致命反击的风险夺回视野。

猎法者魔像 Mage 
Hunter Golem

众神以自由的意志诅咒凡人，但也为他们提供在原初之怒中在生存的手段——魔法的技艺。艾桑椎亚的人民用魔法抵御干旱、饥荒、火山列焰、锋利的寒冰和愤怒的大地。但是众神知道魔法的力量可能会不可避免地将艾桑椎亚撕裂，他们创造了终极魔法故障保护——反魔法。

魔法猎手 Hunter 
of Magic

对许多使用魔法的英雄来说，最可怕的噩梦便是面对一个仅靠存在便能剥夺法术和魔法赠礼之力量的庞大敌人。猎法者魔像的战术极其有效，它们用那毁灭性的反魔法场横扫战场，吞食那些对它们施展的法术，并为施法者套上项圈，让他们任由魔像摆布。
