# 余烬鹏鸟

余烬鹏鸟 Ember Roc

数百年之前，体型与最古老的古龙相当的余烬鹏鸟在原初混沌之中诞生，是消耗的化身。广阔的双翼遮蔽苍天，炽烈的光环匹敌烈日，烈焰缠身的巨鸟只会去挑战那些最致命的猎物。

火之进化 Evolutionary 
Fire.。皮拉的火之阿莎里认为，神话般的鹏鸟是维持塔尔多雷的平衡的一部分。鹏鸟会避开人们的居住点，并猎杀那些可能威胁到人们的怪物聚落。所以阿莎里们将大鹏引至元素裂隙附近，诱使它们吞噬那些企图逃到物质位面的生物。在以火元素，熔螈，火巨人等为食多年之后，鹏鸟便得到了原初的力量，火焰从其飞羽上喷涌而出，最终令其化为一只余烬鹏鸟。

绝望终局 Hopeless Ends。余烬鹏鸟在袭击前会细心观察猎物数个小时，它们猎取大象，鲸鱼甚至年轻的龙类。它会以铁爪抓起目标，不顾猎物反击地将其拖回巢穴——通常是活火山之中。而若是猎物在飞行途中造成了太多麻烦，余烬鹏鸟则会将其从高处扔下。那些逃不掉的家伙，则只能在炽热的爪子中被逐渐烤熟。

余烬鹏鸟 Ember Roc

超巨型怪兽 AC 14（天生护甲）HP 232（16d20+64）速度：20尺，飞行120尺力量26（+8） 敏捷10（+0） 体质18（+4）智力4（-3） 感知11（+0） 魅力9（-1）豁免：敏捷+5，体质+9，感知+5，魅力+4技能：察觉+5

伤害免疫：火焰感官：被动察觉15语言：理解火族语但不会说挑战等级：14（11,500XP），熟练加值+5敏锐视觉 Keen Sight。鹏鸟依赖视觉进行的感知（察觉）检定具有优势。

焦热威仪 Searing Presence。任何在余烬鹏鸟5尺内开始回合的生物都会受到7（2d6）火焰伤害。

照明 Illumination。鹏鸟发出40尺半径的明亮光照和在此范围40尺内的微光光照动作 Actions 多重攻击 Multiattack。鹏鸟发动两次攻击：一次喙啄和一次禽爪攻击。喙啄 Beak。近战武器攻击：命中+13，触及10尺，单一目标。命中：26（4d8+8）的穿刺伤害外加7（2d6）火焰伤害。禽爪 Talons。近战武器攻击：命中+13，触及5尺，单一目标。命中：22（4d6+8）的挥砍伤害外加7（2d6）火焰伤害。

狱火 Inferno（充能5-6）。鹏鸟扇动翅膀，创造涌动的焰流。任何位于其30尺内的生物都必须成功通过一次 DC 18的敏捷豁免否则受到21（6d6）火焰伤害。
