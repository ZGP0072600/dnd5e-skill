# 熔岩陆鲨

熔岩陆鲨 Magma Landshark

潜藏在火山、地下岩浆坑和火元素位面内无尽火海中的岩浆陆鲨只为进食而生存。这些生物在外形上类似于分裂平原上常见的陆鲨即鲨蜥 bulettes，但充满着翻腾的元素力量。

这种炽热的覆甲捕食者在固态岩石中挖洞和燃烧，并且能感觉到周围地面的轻微震动。当感到饥饿的时候——并且这些生物总是处于饥饿状态——熔岩陆地鲨从地下爆发而出，无视任何障碍或危险，连根拔起树木，破坏道路，引发山崩，并在其身后留下天坑。当它吞下整个猎物时，它的下巴会变宽，即使是最初在攻击中幸存下来的生物，如果无法逃脱，也必须与其酷热的灵光相抗争

Landshark

巨型元素

AC 18（天生护甲）HP 150（12d12+72）速度50尺，掘穴50尺，游泳50尺（仅岩浆）力量22（+6）敏捷11（+0）体质22（+6）智力2（-4）感知14（+2）魅力5（-3）伤害免疫：火焰

技能：察觉+7感官：黑暗视觉60尺，震颤感知120尺，被动察觉17语言：—挑战等级：14（11,500 XP）熟练加值+5焦热威仪 Searing 
Presence。任何在陆鲨10尺内开始回合的生物受到11（2d10）点火焰伤害。

立定起跳 Standing Leap。无论是否助跑，陆鲨最远可以跳40尺，最高20尺。照明 Illumination。陆鲨散发出15尺半径的明亮光照，外加15尺的微光光照。

动作 Actions 啃咬 Bite。近战武器攻击：命中+11，触及5尺，单一目标。命中：45（6d12+6）的穿刺伤害，外加14（4d6）火焰伤害。如果目标是一大型或更小体型生物，则目标必须成功通过一次 DC 18的敏捷豁免否则被陆鲨吞下。被吞下的目标陷入目盲且被束缚，其对于陆鲨身体外的攻击或效应视为处于全身掩护，并在每次该陆鲨的回合开始时受42（12d6）点火焰伤害。如果陆鲨在同一回合累计受到30点来自其体内的伤害，则其在该回合结束时必须成功通过一次 DC 23的体质豁免否则将吐出所有被其吞咽的生物，并使其摔下倒地于陆鲨周边10尺范围内。如果陆鲨死亡，则被吞下的生物将不再受束缚，且可以消耗20尺移动力逃出，然后在尸体旁现身并陷入倒地状态。致命一跃 Deadly Leap。如果陆鲨在移动中跳跃了至少15尺，则它可以用该动作四脚着陆在一处由一个或多个生物占据的空间。该空间内的每个生物必须进行一次 DC 18的力量或敏捷豁免（目标自选）。豁免失败者将应击倒地，且受到27（6d6+6）点钝击伤害外加21（6d6）点火焰伤害。豁免成功则伤害减半，且不会因此倒地，而是从陆鲨的位置被推离5尺，并选择并进入一处未被占据的位置。如果该范围内没有这样的位置，则该生物失足倒地于陆鲨所处空间。

岩浆池 Lava 
Pool（充能6）。如果陆鲨在固态地面上，它熔化周围的地面，在创造一个10尺半径的岩浆池。任何生物在一个回合内第一次进入该区域，或在该区域内开始其回合时都会受到44（8d10）点火焰伤害。
