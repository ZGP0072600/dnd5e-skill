# 平原牛

平原牛 Plainscow

高大的平原牛是一种极为坚韧的驮畜，原产于分裂平原，几个世纪一直在为塔尔多雷中部的人们所用。许多野生畜群在平原上游荡，然而驯化后的平原牛更为常见。它们通常被用作坐骑、驮畜和牲畜。平原牛对野兽有着不同寻常的同理心，并且经常与骑手终身相处。它们身高6英尺，体重约2吨，寿命长达70年。市场价值 Market Value。在 Westruun 可以花200gp 买到平原牛。一辆平原牛的载重量为1500磅。

大型野兽

AC 13（天生护甲）HP 42（5d10+15）速度40尺

力量18（+4）敏捷8（-1）体质16（+3）智力2（-4）感知10（+0）魅力6（-2）伤害抗性：非魔法攻击造成的钝击，穿刺和挥砍

技能：洞悉+2

语言：—

挑战等级2（450XP）熟练加值+2

践踏冲锋 Trampling Charge。若平原牛在蹄击攻击之前笔直朝向目标生物移动至少20尺，则目标必须进行一次 DC 14的力量豁免，豁免失败则被撞击倒地。如果目标倒地，平原牛可以用一个附赠动作再执行一次蹄击攻击。

动作 Actions

蹄击 Hooves。近战武器攻击：命中+6，触及5尺，单一目标。命中：11（2d6+4）钝击伤害。
