# 恶魔噬蛛

恶魔噬蛛 Demonfeed 
Spider

这些怪物般的蜘蛛乃是邪神蛛后的衍生，它们潜伏最幽深的黑暗中。自灾祸之战爆发之初，它们就贪婪地吞噬着当时因恶魔大量死亡而充斥着地下的恶魔脓血。

恶兽转变 Monstrosities Transformed。进食恶魔的遗骸令恶魔噬蛛化为了可怖的邪魔，但其实它们原先只是塔尔多雷上那些恐怖洞穴和森林中那些大型蜘蛛。因此，它们大多都是物质位面的原住民，故而一旦被杀，他们就会永久死亡——不像其他邪魔那般通常只是被驱逐回其的家园位面。

不休饥饿 Endless Hunger。对恶魔噬蛛而言，没有任何食物比恶魔在物质位面上被杀后留下的脓液更加美味。然而，在得不到这些遗骸的情况下，它们也能以任何生物的尸水为食。

剧毒之血 Toxic Blood。蜘蛛的骇人毒素与恶魔的邪恶脓液混合即为恶魔噬蜘蛛最有力的武器。当它的几丁质外壳被刺穿，血液流淌而出时，蜘蛛便可以将这些腐蚀性血液精确地挤出，渗入攻击者暴露出的血肉中。

大型邪魔

AC 16（天生护甲）HP 75（10d10+20）速度40尺，攀爬40尺

力量16（+3）敏捷16（+3）体质15（+2）智力6（-2）感知10（+0）魅力6（-2）

豁免：敏捷+7

技能：察觉+3，隐匿+7

伤害抗性：寒冷，火焰，闪电

伤害免疫：毒素状态免疫：中毒

感官：黑暗视觉120尺，被动察觉13

语言：—

挑战等级8（3,900XP），熟练加值+3

蛛行 Spider Climb。蜘蛛可以攀爬在难以攀爬的表面上，包括倒吊在天花板上，且不需要进行属性检定。

蛛网行者 Web Walker。蜘蛛在蛛网上移动时无视蛛网造成的移动限制。

动作 Actions 多重攻击 Multiattack。蜘蛛进行两次近战攻击：一次啃咬和一次毒刺；或者蜘蛛使用蛛网飞刺并在其后立即进行一次近战攻击。

啃咬 Bite。近战武器攻击：命中+7，触及5尺，单一目标。命中：8（1d10+3）穿刺伤害，目标必须进行一次 DC 14的体质豁免，豁免失败则受到27（6d8）点毒素伤害并陷入中毒1分钟。豁免成功则伤害减半。如果该毒素伤害将目标的生命值降至0，则目标伤势稳定但陷入中毒，该中毒状态持续1小时，此期间目标被麻痹。

毒刺 Stinger。近战武器攻击：命中+7，触及5尺，单一目标。命中：9（1d12+3）穿刺伤害，目标必须成功通过一次 DC 14的体质豁免否则被麻痹一分钟。目标能在其每个回合结束时重复豁免，成功则结束该效应。

蛛网飞刺 Web Spinner。远程武器攻击：命中+7，射程40尺，单一目标。命中：目标被向着蜘蛛拉近40尺并被擒抱（逃脱 DC 14）。蜘蛛同一时间内只能以此法擒抱一个生物。蜘蛛还可以用一个动作来把一个其擒抱的目标缠入茧中，令其被束缚并结束受擒状态。

反应 Reactions

搏血喷溅 Arterial Spray。当蜘蛛在生命值低于其上限的一半时受到伤害，它可以在15尺锥形范围内喷出毒血。任何位于该区域内的生物都必须进行一次 DC 14的体质豁免。豁免失败者受到13（3d8）毒素伤害并中毒一分钟。豁免成功则伤害减半且不会中毒。若该伤害将一个生物的生命值降为0，则目标伤势稳定但陷入中毒，该中毒状态持续1小时，此期间目标被麻痹。
