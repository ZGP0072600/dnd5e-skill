# 骷髅

骷髅 Skeletons 骷髅通常是初入死灵法师者的第一种活化产物，它们被重新组装成类似活着时的形态，充满了对生命的拙劣模仿。曾让骨架充满驱动骷髅的生命已不复存在，其唯一目的便是忠诚。骷髅遵循其创造者的指令，直至指令完成。它们顺从且不知疲倦的天性使其成为完美的一次性的战斗耗材、警卫或分散注意力的东西——而且，数量足够多的骷髅足以形成一股压倒性的骨骸、利齿与刀锋的浪潮。亡灵本质 Undead Nature。骷髅不需要空气、饮食或睡眠。阵营：中立邪恶。

“这些噩梦般的骷髅周边的奇异奥秘为其赋予了新的、暴力的目的，作为对寿命的嘲弄。但也有传言称，一种深藏于其内的黑暗智能在寻求力量，并会利用自己的骨骼，甚至是其尸体中生着的蠕虫来作为与神秘商人进行交易的货币。”

——药师威廉姆·奥姆

人马骷髅 Centaur Skeleton

Alabaster Sierras 毗邻危险的岸边悬崖和荒凉的山脉，那里挤满了饥饿的怪物。但这些山峰下面则是一片名为 Parchwood 的，茂密、美丽而又诡异的森林。在由布莱伍德家族统治的噩梦般的六年中，难民们在这里安家——但黛利拉夫人和塞拉斯爵士的实验同样如此。Parchwood 的人马骷髅根本不是人马；它们是布莱伍德家族短暂实验的残余——他们以将不同物种的遗骸重塑为独特形式来取乐。

炽焰骷髅 Flame Skeleton

许多反抗灰烬之王索达克之人的遗骨永远地承受了它的火焰吐息中所蕴涵的原初能量。在那些被红龙的力量永久伤害的地区，包括埃芒云顶区的索达克火山、Stormcrest 山脉的灰烬峡谷和崖堡山脉的巨蛇之首，都充斥着炽焰骷髅。这些亡灵对元素位面的原始能量做出反应，无法克制地想要四处传播火焰，直到万物皆被烈火吞没。

照明 Illumination。骷髅散发出10尺半径的明亮光照，外加10尺的微光光照。动作 Actions 短剑 Shortsword。近战武器攻击：命中+4，触及5尺，单一目标。命中：5（1d6+2）穿刺伤害外加3（1d6）火焰伤害。

引燃 Ignite。骷髅将自身投向一个5尺内的大型或更小的生物。目标必须成功通过一次 DC 12的敏捷豁免否则被擒抱（逃脱 DC 12）。豁免失败者还会在每个其回合开始时受到3（1d6）火焰伤害，即使擒抱结束。若目标或其5尺内的生物用一个动作来扑灭火焰或其他能导致火焰熄灭的效果影响（比如将目标浸入水中），该效应也会结束。
