# 超自然祝福

**超自然祝福：命运触及者 Fate-Touched**

大部分生命以及灵魂都很容易地能够在命运之途中找到自己的道路，渴望着成为历史的一部分，同时也不经意间地成为了历史的一部分。然而，有些灵魂却带有神秘的影响力，当他们经过时，他们命运的金色丝线便在不经意间引导着历史的进程。伟大的统治者，可怖的暴君，强大的法师，宗教的领袖，以及传奇的英雄，无论是伟大的事件还是灾难的事件皆是伴随如此的灵魂而来。他们便是命运触及者。

很少会有命运触及者意识到他们的显著本质。没有神明的亲自确认，他们中很少有人能肯定自己拥有这份祝福。而这样也是情理之中，对于拥有权利和声望的人来说，他们很容易会妒忌那些仅仅是天赋就超过自己了的人。现存的奥法纪 Age of Arcanum 史书充满了魔法统治集团奴役那些被视为命运触及者之人——或者更糟，试图以最可怕的方式从他们身上强行提取出命运的实质的传说。为天命所眷顾，可以是祝福，亦可以是诅咒。在大多数情况下拥有命运的祝福之人将被历史所铭记，而被命运诅咒之人则被历史遗忘。艾桑椎亚的大部分文明有着许多关于在有无数邪恶传闻流传着的那位*茹蒂斯，猩红之月 Ruidus, the vermilion moon*的满月之光下诞生，从而被厄运打上烙印之人的传说。

玩家角色和 NPC 们通过 DM 赠予的方式获得这份祝福，作为结果从而获得特性命运恩典。一名角色可能会以来自神明的一份礼物，或者当战役转变为聚焦于该角色的故事情节时的一个短暂而令人惊讶的奇迹的方式而收获该祝福。某位 NPC 盟友因为角色们在某个可怕威胁下保护了自己，可能会获得该祝福以出手相救。而先前和善的 NPC 因为想测试一下其命运触及者赠礼的极限，甚至可能会变成一位渴求力量的恶棍。

该书新增了下列祝福：**命运恩典 Fortune's Grace**。

命运恩典 Fortune's Grace

*祝福*

你被命运触及的本质会导致事件变得有利于你。当你进行一次攻击检定，属性检定，或者豁免检定时，你可以选择重掷该 d20。你必须在最初的骰点结果决定之前重骰该骰子。

又或者，当一个你可见的生物对你进行了一次攻击检定，或者其为对抗你的某个法术或特性而进行一次豁免检定时，你可以迫使该生物重掷该攻击检定或豁免检定。

如果触发了重骰的该检定含有优势或劣势，则两个 d20均需重骰。一旦你使用了该特性，你便无法再次使用该特性直到你完成了一次长休。
