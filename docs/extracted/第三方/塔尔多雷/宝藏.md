# 宝藏

**塔尔多雷的宝藏 Tal'Dorei Treasures**

塔尔多雷是一片充盈着魔法的土地。许多魔法组织，无论是正式的海上在暗中的都会创造出魔法物品来帮助自己的成员们执行神秘的任务。在大城市的商店，包括埃蒙的“吉尔莫的琳琅好货”（Gilmore's 
Glorious 
Goods）之中公开对任何口袋里有足够支付得起其奢侈（有的人会说这是过于高昂的）费用的金子的人出售魔法物品。DM 可以使用本章中的魔法物品——其中一些在《Critical 
Role》中曾出现，其中一些则是崭新的面孔——来在冒险中作为宝藏奖励角色们。除非另有说明，否则这些物品遵循五版核心规则中的魔法物品规则。
