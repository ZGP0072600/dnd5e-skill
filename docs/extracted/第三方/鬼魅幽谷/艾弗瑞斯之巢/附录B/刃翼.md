# 刃翼 Knifewing

刃翼是一种会滑翔的蜥蜴，体型和小狗差不多。它的滑翔翼有着硬质的边缘，当这只蜥蜴滑翔时，翼缘绷紧并变得锋利。

刃翼的皮肤因能够制作*刃翼披风 knifewing cape*倍有价值。
