# 审判领域 Inquisition Domain

审判领域反映了多元宇宙的秩序力量与邪恶魔法的对抗——至少在某些神看来是这样的。只有神的意志才是纯粹而正义的。由于神秘的魔法强大到足以挑战神，因此，审判领域的神，如 **至高大天使炽焰 Arch Seraph Empyreus** 
 
，要求使用魔法的凡人受到控制。大多数狂热分子都希望根除所有不可控的魔法使用者，即使无法消灭这些施法者，他们也会义无反顾的发起攻势。

**审判领域法术 **

| 牧师等级 | 法术 |
|---|---|
| 1st | 侦测魔法 Detect Magic，鉴定术 Identify |
| 3rd | 识破隐形 See Invisibility，沉默术 Silence |
| 5th | 解除魔法 Dispel Magic，移除诅咒 Remove Curse |
| 7th | 秘法眼 Arcane Eye，生物定位术 Locate Creature |
| 9th | 造物术 Creation，圣居 Hallow |

额外熟练 Bonus Proficiencies 

第1级起，你获得军用武器和重甲的熟练。

**猎巫打击 Witch Hunter’s Strike**

第1级起，当你用武器命中一个生物时，你可以对其额外造成1d8点力场伤害。如果该生物正处于法术专注，则变为对其造成2d8力场伤害。如果该生物因此次攻击失去专注，则你获得等同于力场伤害的临时生命。你可以使用该特性的次数等于你的感知调整值（至少为一次）。而你将在完成一次长休后重新恢复所有的使用次数。

第14级起，伤害提升至基础2d8和专注3d8。

**引导神力：术法护盾 Spell Shield**

第2级起，你可以使用引导神力特性赋予你的队友抵御奥术伤害的能力。以一个附赠动作，你展示你的圣徽，选择一名30尺内的生物（可以是你自己），该生物获得1d10+你的牧师等级的临时生命值。具有该临时生命期间，其具有对法术伤害的抗性，且其对抗法术和其他魔法效应时进行的豁免检定具有优势。该生物在1小时后失去剩余的临时生命值。

**审判官 Rebuke Invoker**

第6级起，当60尺内有生物施展一个法术，你可以用一个反应迫使该目标进行一个体质豁免，豁免失败则受到（法术环位等级)*d8+你的感知调整值的力场伤害，成功则伤害减半。戏法被视作一环法术（1d8)。

你可以使用该特性的次数等于你的感知调整值（至少为一次）。而你将在完成一次长休后重新恢复所有的使用次数。

**神圣打击 Divine Strike**

第8级起，你获得为你武器注入秩序神力的能力。每个你的回合一次，当你的武器攻击命中一名生物时，额外造成1d8力场伤害。

第14级起，伤害提升至2d8。

**神之守护 Supernal Safeguard**

第17级起，当你使用术法护盾时，你可以选择至多等于你感知调整值数量的生物（至少2名）而非一名生物。
