# 戏剧学院 College of Drama

全世界是个舞台，所有的男男女女不过是一些演员，他们都有下场的时候，也都有上场的时候；一个人的一生中扮演着好几个角色。

——威廉·莎士比亚

吟游诗人以多种多样的方式讲述故事。有的引吭高歌，有的翩然起舞。但他们总归是以艺术的形式，分享着古老的英雄史诗。而戏剧学院的诗人们，则采用了一种更为深入的方式，将这些古老的故事演绎得活灵活现。他们结合自身的技艺与魔法，不仅能改变自己的容貌与嗓音，甚至能改变整个剧团的。

通过化身为传说中的人物，这些诗人能够激励他们的观众，使其迸发出伟大的力量。他们鼓舞着盟友，去完成那些需要非凡毅力才能成就的壮举，并以此提醒同伴：昔日的英雄们曾面临更严峻的挑战，却依旧赢得了最终的胜利。

专精于此道的诗人尤其擅长渗透邪恶组织。他们的魔法能让整个队伍都扮演相应的角色，这使得团队有机会在仪式期间混入其中，或是参与大型集会而不露破绽。

**3级：悲剧英雄 Tragic Hero**

一场伟大的演出，有时也需演员付出代价。

当你失败于一次豁免检定时，你可以选择你周边60尺范围内任意数量的你可见的其他生物。每个被选中的生物在对抗触发此特性的同一效应时，其豁免检定会获得等于你魅力调整值的加值（至少+1），有可能将失败转为成功。

**3级：百面人生 One Life, Many Roles**

你运用自身的魔法与戏剧训练来扮演一个角色。

你可以使用附赠动作通过此特性施展易容术 Disguise Self。若你以此方式施展易容术 Disguise Self 并扮演一个你可见的特定生物，你便能模仿该生物的言行举止与声音。

一旦你使用了此特性，则直至你完成一次短休或长休前，都无法再次使用。

**6级：舞台指导 Stage Direction**

你获得了能在激励他人的同时进行指导的能力，让他们能最大程度地发挥你的指引。

当一个能看见你的生物通常需要掷出一枚你的诗人激励骰时，它无需掷出该骰子，而是直接取其可能的最大值。

你能使用本特性的次数等于你的魅力调整值（最少为1）。完成一次长休后，你重获所有已消耗的使用次数。

**14级：众星登场 Assemble the Troupe** 

当你使用百变人生特性施展易容术 Disguise 
Self 时，你可以指定你周边30尺范围内至多六位你可见的自愿生物为该法术的目标。由你来决定每个生物变化成的外观。在以此特性施展的易容术效应持续期间，被选中的生物在进行魅力检定时会获得等于你魅力调整值的加值（最少为+1）。

**14级：谢幕绝唱 Grand Finale**

你学会了如何使用激昂的演出来激励你的同伴做出伟大的壮举。

你可以使用魔法动作选择你周边60尺范围内至多两名能看见或听见你的其他生物。每个被选中的生物都可以在本回合内立即执行一个动作。

一旦你使用了此特性，则直至你完成一次长休前，都无法再次使用。

译注：边栏摘自《皆大欢喜》（朱生豪 译本）
