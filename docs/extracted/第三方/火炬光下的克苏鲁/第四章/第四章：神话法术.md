# 法术详述 Spell Descriptions

下文法术的设计源自克苏鲁神话及其主题。尽管这些法术已与现有法术进行了平衡，但其中许多并不允许目标进行豁免检定。其伤害在设计时便考虑到了这一点，这与魔法飞弹 Magic 
Missile 很相似。这些法术意在体现神话那可怖而又无可避免的本质。它们大多以专注为核心机制，迫使法术的目标必须伤害或击败施法者，才能终止法术效应。
