# 无形之子 Formless Spawn

这些漆黑的不定形存在能瞬间改变形态——从蟾蜍状的肿块变为拥有数百条断肢残臂的细长之物。它们能渗透狭小的缝隙，也能膨胀至阻塞整条通道，以确保无人可逃。

**骇人智慧 Alarmingly Intelligent。** 无形之子看似一团无脑的泥怪，然而实则是一个狡猾且恶毒的存在。作为克苏鲁神话力量所催生的造物，它们潜伏在被遗忘的神殿、隐秘的圣所以及废弃的遗迹之中，而这些地方曾用于举行那些供奉亵渎之神的仪式。这些无形的恐怖存在自视为守护者与看门人，它们保护着这些神圣的场所，并会迅速解决掉任何擅闯之人，无论其有意还是无意。

**无情守卫 Relentless Guardians。** 有时，尽管无形之子竭力守护，闯入者仍可能成功窃取到一件贵重遗物。通常情况下，无形之子会特地看守着某件仪式器具，例如用于献祭的匕首或记载着神话学识的古老典籍。在此情况下，无形之子会专一且无休止地追踪这名窃贼，直至夺回宝物方才休止。

## 特质 Traits

**无定形 Amorphous。** 无形之子可以移动穿过最窄1寸宽的空间而无需消耗额外的移动力。

**魔法抗性 Magic Resistance。** 无形之子对抗法术和其他魔法效应时进行的豁免检定具有优势。

## 动作 Actions

**多重攻击 Multiattack。** 无形之子使用伪足钝击或触手擒握发动至多共计四次攻击。其可以将其中任意次数的攻击替换为使用碎骨之握或投掷猎物。

**伪足钝击 Bludgeoning Pseudopod。** *近战攻击检定*：+10，触及10尺。*命中*：17（2d10+6）钝击伤害。

**碎骨之握 Crushing Grip。** *力量豁免检定*：DC 18，单一受擒于无形之子的生物。*失败*：14（4d6）钝击伤害。*成功*：半伤。

**触手擒握 Grasping Tentacle。** *近战攻击检定*：+10，触及20尺。*命中*：17（2d10+6）挥砍伤害。若目标生物不超过中型，则其陷入受擒状态（逃脱 DC 18），受擒期间，其还会陷入束缚状态。

**投掷猎物 Throw Victim。** 无形之子投掷一名受擒于其的不超过中型的生物，将其投掷至30尺内一处其可见的非空中空间。*敏捷豁免检定*：DC 18，被投掷的生物以及目标空间内的每名生物。*失败*：14（4d6）钝击伤害，且目标陷入倒地状态。*成功*：仅半伤。
