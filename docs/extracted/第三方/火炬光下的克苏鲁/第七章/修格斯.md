# 修格斯 Shoggoth

修格斯是克苏鲁神话中最为强大且令人憎恶的怪物之一。这些两栖生物形如巨大的原生质囊袋，能随心所欲地生成肢体、眼目、口器乃至更可怖的器官，它们既可借此模仿其他生命形态，又能施展超越现实极限的力量。唯有当其主人有所指令时，它们才会生成特殊器官进行交流。

大多数修格斯直接听命于强大的邪教首领或神话实体。它们被召唤来提供强大的肉体力量，既能从山峦中开凿出城市，也能以变幻莫测、反射虹光的庞大身躯碾碎整支军队。修格斯近乎永生不灭——有些修格斯在召唤它们的邪教覆灭后，仍会持续在废墟或地城中劳作；另一些则蛰伏于密封地窖或仪式密室中，静静等待好奇心重或野心勃勃的蠢货释放它们。

**危险盟友 Dangerous Allies。** 修格斯的智力会随年龄增长而提升。刚被创造或召唤出来时，它们只是单纯的野蛮生物，是绝佳的守卫、仆从和劳力。随着岁月流逝，它们通过观察世界积累知识，逐渐变得既狡黠又强横。假以时日，修格斯甚至可能推翻自己的主人，成为邪教的新领袖。若被长期孤立囚禁，修格斯的智力会逐渐退化，但一旦与智慧生物接触，便会在1d6天内重新恢复智力。

## 特质 Traits

**无定形 Amorphous。** 修格斯可以移动穿过最窄1寸宽的空间而无需消耗额外的移动力。

**水陆两栖 Amphibious。** 修格斯可以在空气和水中呼吸。

**魔法抗性 Magic Resistance。** 修格斯对抗法术和其他魔法效应时进行的豁免检定具有优势。

## 动作 Actions

**多重攻击 Multiattack。** 修格斯使用恐怖触须或抛射黏液发动共计四次攻击，其不能对同一名生物攻击超过一次。

**恐怖触须 Horrid Tendril。** *近战攻击检定*：+15，触及20尺。*命中*：（2d10+10）点钝击伤害。若目标生物体型不超过大型则陷入受擒状态（DC 20），受擒期间，目标还会陷入束缚状态。修格斯可同时擒抱最多四只不超过大型的生物。

**抛射黏液 Hurled Mucous。** *远程攻击检定*：+15，射程200尺。*命中*：21（2d10+10）强酸伤害。若目标生物不超过大型，则其陷入倒地状态。

**黏液巨浪 Slime Wave（充能5–6）。** *敏捷豁免检定*：DC 20，60尺锥状范围内的每个生物。*失败*：44（8d10）强酸伤害。*成功*：半伤。该区域在接下来的1分钟内被黏稠的酸液覆盖。当生物在回合内首次进入该区域，或是在该区域结束其回合时，其受到22（4d10）强酸伤害。每名生物每回合仅会受到一次此伤害。
