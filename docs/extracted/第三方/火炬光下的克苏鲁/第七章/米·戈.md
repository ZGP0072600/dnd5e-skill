# 米·戈MI-GO

米·戈也被称为犹格斯真菌，是一种居住在恒星之间黑暗空间中的物种。 
它们可以在真空宇宙中生存，其飞行能力还可以在太空中穿梭。米·戈通常会在世界上最高的山峰上建立殖民地，这样它们就可以安安静静地开采稀有矿物。

**邪教领袖 Cult Leaders。** 米·戈有时需要体力劳动者和其它手下来处理采矿业务。它们使用自己的魔法，并许诺给它们魔力和支持，吸引人类和其它人为它们服务。 
米·戈倾向于崇拜神话中的实体，它们招募人类加入这些邪教，以便更好地控制它们。

**奇怪科学家 Strange Scientists。** 米·戈好奇而聪慧，这使它们以科学的思维方式对待世界。它们派遣它们的爪牙收集信息或奇怪的遗物供它们研究。在它们看来，人类和同类不过是野生动物，因此在评估其野心的影响时，它们不重视人类的生命。

**奇异科技 Bizarre Technologies。** 米·戈将人类视为真正有意识的生物，这导致它们在自己的手下身上进行各种奇怪的实验。一个特别有用的人类可能会被移除大脑，并被装进一个奇怪的圆筒里。这个东西能让大脑保持活力，让米·戈在需要时与它们以前的仆人进行交流。一个大型米·戈殖民地可能有数十个大脑。让大脑存活的奇怪装置还能让米·戈或任何获得该装置的人类与大脑对话的能力。
