# 制作概要 Crafting Overview

简单来讲，玩家角色使用此规则制作的流程如下

1. **学习配方 Learn the Recipe。**玩家角色必须首先找到他们想要制作的魔法物品的配方，配方会说明物品的特性以及所需的素材。配方还会详细列出必须从哪些怪物身上采集这些素材。永久性物品（如武器和护甲）需要4到10个素材，而消耗品（如药水和卷轴）则只需要2到3个素材。
2. **采集素材 Harvest the Components。**玩家角色击败怪物后，可以花费10分钟从尸体上采集并保存素材。这个过程必须在怪物死亡后迅速完成，否则素材的蕴含的魔力会逐渐消散。DM 会描述哪些素材可用，然后每个玩家角色可以选择采集其中一种素材（无论击败了多少敌人）。这抽象地模拟了素材在战斗中可能因受损、采集方式不当等原因导致质量不足以用于制作的情况。
3. **寻找工坊 Locate a Workshop。**采集到所需的素材后，玩家角色必须将它们带到合适的工坊中制作物品。工坊的稀有度决定了可以在其中制作的魔法物品的稀有度。非普通或稀有工坊可能出现在大城市中，而极珍稀和传说工坊只存在于某些拥有奇异伟力的奇境，例如地心深处的熔岩熔炉或沿着世界之隙建造的奥术实验室。
4. **制作物品 Construct the Item。**玩家角色在工坊中花费一天的时间组合素材，新的魔法物品就制作完成了！

## 素材与工坊的稀有度 Component and Workshop Rarity

素材和工坊都有稀有度，这与用它们制作的魔法物品的稀有度相对应。例如，一件非普通魔法物品通常需要非普通素材，并且必须在非普通工坊中制作。稀有度也为玩家角色应该能够制作该物品的等级提供了一个大致的参考标准。 

采集到的素材的稀有度基于怪物的挑战等级。除非另有说明，素材的稀有度由素材稀有度表格定义。

**素材稀有度 C****omponent Rarity**

| 稀有度 | 怪物挑战等级 |
|---|---|
| 普通 | 1/4-1/2 |
| 非普通 | 1-4 |
| 珍稀 | 5-10 |
| 极珍稀 | 11-16 |
| 传奇 | 17+ |

**玩家角色制作物品的频率 How Often Do Characters Craft an Item？**

假设在整个战役过程中，玩家角色每升一级会面对大约4到8次怪物的战斗遭遇，每个玩家角色可以制作以下数量的永久性魔法物品：

- 3–4件非普通
- 2–3件珍惜
- 1–2件极珍稀
- 1件传说

这些数字是期望值，只提供了一个基本预期，你可以根据你希望玩家角色拥有的魔法物品数量进行相应调整。比如玩家角色可以制作更多的普通素材，以及各种稀有度的消耗性魔法物品（药水和卷轴）。

我们建议在使用这些规则时调整战役中发现的战利品数量。由于玩家角色会自己制作魔法物品，他们发现的魔法物品数量应约为正常情况下的一半。
