# 第二步：采集素材 Harvesting Components

玩家角色可以使用怪物采集工具从被击杀生物的尸体上采集并保存素材。

采集怪物素材或提取其精魄必须在生物死亡后的十分钟内开始。超过这个时间后，素材内的魔法能量会逐渐消散，精魄则会消散到以太之中。此后，怪物尸体上剩余的任何素材都将无法用于制作魔法物品。

成功采集怪物素材或提取精魄无需进行属性检定。只要玩家角色有足够的时间、工具并具备相应熟练度，他们总能成功地从生物尸体上采集到一个怪物素材。工具包提供了从任何生物身上提取和保存素材所需的一切。

- **怪物采集工具 Monster Harvesting Tools（50GP）。** 此工具包包含提取精魄和保存生物素材所需的全套器具：剥皮刀、剁骨刀、注射器、锤子、剪刀、手术刀、骨锯、钩子，以及一系列小瓶、容器、防水容器和包装材料。

## 怪物素材 Monster Components

当玩家角色决定采集怪物的尸体时，游戏主持人会根据情况判定可以从怪物身上采集哪些素材（如果有的话）。在本书中，每个怪物的条目都列出了该生物可提供的素材，你也可以在网上找到为《龙与地下城第五版核心规则》中的怪物列出的建议素材补充下载内容。

 

**素材稀有度 Component Rarity** 

所有素材的稀有度由怪物的挑战等级决定，具体对应关系见本章前面的表格。

多数物品配方要求使用与成品同等级稀有度的素材。但部分配方可能指定特定名称的怪物素材（如霜巨人、远古红龙或女妖）。此类情况下，若该类型怪物未提供对应稀有度的素材，则 DM 应允许使用更低稀有度的素材替代。例如：

- **幽魂视界**（极珍稀）需用到树精的骨骼与女妖的头发。此时只需获取任意树精或女妖的素材，不强制要求素材本身达到极珍稀等级。

**素材分类 Component Categories** 

每个素材都属于以下类别之一：

- **精魄 Animus。**这种特殊素材将在下文详述。
- **骨骼 Bones。**包括头骨、脊柱或植物怪物的茎。
- **粉尘 Dust。**魔物或亡灵留下的残骸。
- **流体 Fluid。**通常是怪物血液，也包括精魄物质、树液或黏液。
- **毛发 Hair。**任何皮毛或羽毛。
- **表皮 Hide。**包括怪物的皮肤、鳞片、甲壳、树皮或金属板。
- **天生武器 Natural Weapons。**几乎包括怪物卡中列出的任何天然武器，如牙、爪、角、触须等。
- **器官 Organs。**一个广泛的类别，包括内脏器官和感官器官，如心脏、大脑、腺体、眼睛或内脏。

**蕴魔精魄 Magical Animus** 

在德拉肯海姆的世界中，魔法物品是通过将一件实体物品注入形而上学理解的生物精魄来制作的。这种至关重要的能源必须从魔物中才能提取。魔物的精魄 Animus 由其体内蕴含的潜在魔法能量构成，它与生物的灵魂 soul 是不同的。异怪 Aberrations、天族 Celestials、构装 Constructs、龙类 Dragons、元素 Elementals、妖精 Fey、恶魔 Fiends、巨人 Giants、怪兽 Monstrosities、植物 Plants、泥怪 Oozes 和亡灵 Undead 都拥有蕴魔精魄。而野兽 Beasts 和类人 Humanoids 通常没有精魄。

当一个生物被杀死时，它的精魄会短暂地留存，然后消散到以太之中。不过，我们可以采集一个生物的精魄并将其封存在某种容器里，然后用其来制作魔法物品。换言之，不杀死生物就不可能提取它的精魄。

制作魔法物品需要一个或多个适当强大的生物的精魄。不过，有些魔法物品可能需要某类型的精魄，比如妖精或龙的精魄。在极少数情况下，一件魔法物品可能需要一个特定怪物的精魄。

** 精魄容器 Animus Vessels。**精魄必须被采集并储存在特制的魔法容器中。传统来说，这些容器本身是一种小型魔法物品：通常是精心制作并注入了魔力的刻有符文的宝石。然而，由于妄质的到来，德拉肯海姆的世界发生了巨变：未加工的妄质便足以成为采集精魄的容器。充满蕴魔精魄的妄质水晶会发出绚丽的第八虹光，这和妄质在接触魔法时产生能量共鸣的情形类似 
。

** 制作精魄容器 Creating Animus Vessels。**将任意三个同稀有度的素材耗时一小时进行组合，便可以制成对应稀有度的精魄容器。该容器仅能承载与之稀有度匹配的精魄。例如：这是个由三个珍稀素材制成的容器，那它仅能容纳珍稀精魄。

| 普通 | 晶屑 |
| 非普通 | 晶片 |

**采集与提取用时 Harvest and Extraction Time** 

一旦开始采集精魄或素材，就需要全身心工作十分钟。在此期间，玩家角色必须与尸体保持在5尺以内。如果这个过程被打断，采集工作必须在十分钟内重新开始，否则素材将会废弃。

一次只能有一名玩家角色从一只生物身上采集一个精魄，无论该生物体型大小如何。多名玩家角色可以合作从同一具尸体上采集多个素材。无论有多少玩家角色采集同一具尸体，他们都得各自花费十分钟来进行采集 
 。

**选择采集何种素材 Collector’s Choice** 

每次遭遇（例如战斗等情况）中，每名玩家角色可以从一只生物的尸体上采集一个素材 。

玩家角色们可以合作从一只怪物身上采集几个不同的素材，也可以分开行动，同时从几只不同的生物身上采集素材。通常情况下，一只怪物的尸体每种可用素材只提供一个，所以从同一具尸体上采集素材的两名玩家角色必须选择不同的素材，除非 DM 判定该生物独特的生理结构允许采集到多个相同的素材。例如，DM 可能会允许采集牛头人的两只牛角，但这需要两名玩家角色共同完成采集工作。某些素材的数量需要抽象处理：只需知道玩家角色们采集到了一袋龙牙或一瓶巨魔血液即可，无需担心具体的计量。

当玩家角色决定剥离尸体以获取素材时，DM 会列出该尸体上所有可能获取的素材。然后每名玩家角色可以选择一个素材进行采集。所有其他素材都会废弃且无法使用，可能是因为它们在战斗中受损或采集方式不当等原因导致质量不足以用于制作魔法物品。

## 采集示例 Example Harvesting

例如，假设玩家角色击败了一只蛇发女妖。他们决定从其尸体上采集素材。DM 描述说，这只蛇发女妖拥有以下稀有素材：器官 
眼、 骨骼 头骨、流体 怪物血液、表皮 鳞片、天生武器角、器官 毒腺以及它的精魄 
。有三名玩家角色，他们一起合作从蛇发女妖身上采集素材。塞巴斯蒂安 Sebastian 决定采集蛇发女妖的精魄，普路托 Pluto 决定拿走蛇发女妖的鳞片，维奥 Veo 决定取走蛇发女妖的怪物血液。除此以外其他材料全部。

在另一种情况下，假设四名玩家角色击败了一小群共十只鼠怪。该团队只能从其中四只身上采集素材。其他怪物的素材都将废弃 
。

**记录保存 Keeping Records**

在记录玩家角色采集到的怪物部件时，重要的是要注明素材的类别名称，以及该生物的名称、生物类型和稀有度。例如，一名玩家角色可能会记录以下素材：

- 牛头人，角（天生武器），非普通怪兽
- 吸血鬼，亡灵尘（粉尘），珍稀亡灵
- 巨魔，皮肤（表皮），非普通巨人
- 成年红龙，怪物血夜（流体），传说龙类

无需记录玩家角色从一头被击杀的红龙身上采集了多少根骨头，或者从一只死去巨魔的尸体上榨取了多少瓶血液。无论生物的实际体型大小如何，玩家角色从其尸体上采集到的素材都足以满足制作一件魔法物品所需的原料。只需记录“红龙怪物血液”或“巨魔皮毛”即可。如果玩家角色采集了多头龙或巨魔的血液或皮毛，只需记录“3份红龙血液”或“3份巨魔皮毛”。

**素材重量 Component Weight。** 
一般来说，大多数素材的重量都不超过1磅。在追踪单个素材的重量时，要考虑用其制作成品后的最终重量。大多数武器、奇物和消耗品的重量在1到5磅之间，所以大多数素材的重量也应在此范围内。DM 可以判定某些用于制作中型或重型盔甲（或其他大型物品）的素材重量要重得多。例如，为制作鳞片盔甲或板条盔甲而采集的龙鳞或蛇发女妖的鳞片可能重达50磅。

**轻轻地将你的刀沿着脊椎划过，或者尽可能靠近那些形似脊椎的部位，那根最主要的脊骨 。**

## 买卖素材 Buying and Selling Components

才华横溢的奥术师、奇械师、工匠和铁匠等人群常常不选择冒险生活，而是更愿意委托冒险者代他们采集制作魔法物品所需的稀有素材。买卖素材是玩家在冒险幕间可参与的可选日常活动。

大多数情况下，购买素材的人是在寻找一两种特定的素材，并寻求专业猎手的服务来获取这些素材。就连勤勉的紫晶学院都很少主动进行采购，而且法师们也少有愿意购买那些别人拿来的东西。

通常，找到一种怪物素材的卖家或买家需要花费的天数等于该怪物的挑战等级。
