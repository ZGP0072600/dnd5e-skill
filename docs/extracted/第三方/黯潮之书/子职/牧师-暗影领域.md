# 暗影领域 Shadow Domain

暗影领域拥抱笼罩万物的黑暗，操控着那区分光明与黑暗的灰色瞬间。暗影领域的牧师行事诡秘，他们频繁更换效忠对象且偏好秘密工作。

**暗影领域法术 Shadow Domain Spells**

| 牧师等级 | 法术 |
|---|---|
| 1级 | 灾祸术 Bane，虚假生命 False Life |
| 3级 | 目盲术/耳聋术 Blindness/Deafness，黑暗术 Darkness |
| 5级 | 闪烁术 Blink，恐惧术 Fear |
| 7级 | 艾伐黑触手 Evard's Black Tentacles，高等隐形术 Greater Invisibility |
| 9级 | 寒冰锥 Cone of Cold，托梦术 Dream |

**夜幕将临 Cover of Night**

当你在1级选择该领域时，你获得隐匿技能熟练与60尺黑暗视觉。若你已拥有黑暗视觉，其范围增加30尺。此外，当你身处微光光照或黑暗中时，你能够以附赠动作执行躲藏。

**阴影延展 Lengthen Shadow**

1级起，你可以操纵影子来延长你的触及范围。当你施展施法距离为触碰的法术时，你的影子可以如同你一般施展该法术。目标必须位于你15尺内且你必须能看见目标。即使你所在的区域没有影子，你也可以使用此特性。

**引导神力：影之擒缚 Shadow Grasp**

2级起，你能够令生物的影子转而攻击他自身。以一个动作，选择一名位于你30尺内你可见的生物。该生物必须进行一次力量豁免。豁免失败则该生物被影子束缚，直至你的下个回合结束。豁免成功则其被影子擒抱，直至你的下个回合结束。即使目标所在的区域没有他的影子，你也可以使用此特性。

**隐匿于暗 Fade to Black**

6级起，你可以将自身隐藏在阴影之中。当你身处微光光照或黑暗中时，以一个附赠动作，你可以魔法地隐形1分钟。若你发动攻击或施展法术，此效应提前结束。你能够使用此特性的次数等于你的熟练加值。当你完成长休时，你重获所有已使用的次数。

**强力施法 Potent Spellcasting**

8级起，你施展牧师戏法造成伤害时，可以再加上你的感知调整值。

**暗影军团 Army of Shadow**

17级起，你可以同时操纵多个影子。当你使用影之擒缚时，你可以影响等于你熟练加值名生物。
