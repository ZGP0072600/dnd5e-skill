# 雅灵

雅灵 Eladrin

雅灵们居住在苍翠而灿烂的妖精荒野。他们是物质位面的精灵们的亲戚。但其他精灵们能够驯服自身狂野的冲动，而雅灵则为情绪所支配——由于他们独特的魔法特性，他们的肉体也会随着心情的变化而发生改变。

雅灵在妖精荒野中度过了数个世纪，让他们中的许多都成为了妖精——比如这里列出便是成为了妖精的雅灵。其中一些仍然是类人生物，和他们的精灵亲族们类似。

雅灵体内流动的魔力会对他们的情绪状态做出反应，将他们转变成不同季节的样子，行为和能力也会随着他们的形态而改变。一些雅灵可能会在某个方面停留数年，而另一些雅灵则每周都在改变着自己的情感。

。

**多变天性 Changealbe Nature。**
每当这里列出的雅灵们结束一次长休后，只要不处于失能状态，他们便能让自己转变为另一种不同的季节。当雅灵做出改变时，他们使用新的资料卡替换原本的资料卡。在原本形态中受到的伤害、状态和其他效应依然延续至新的形态。
