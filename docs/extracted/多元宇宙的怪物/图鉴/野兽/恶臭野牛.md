# 恶臭野牛

**恶臭野牛 Stench Kow**

恶臭野牛是一种原生于下层位面的野牛，这些混杂着橙色和绿色体色的畸形野牛通过喷出一种可怕到有毒的瘴气来保护自己，一些恶毒的邪恶法师会将恶臭野牛从下层界带到主物质位面，把这些生物伪装成普通牛，让地狱般的野牛进入毫无防备的村庄。

**恶臭野牛 Stench Kow**

*大型邪魔（牛类），无阵营*

护甲等级：10（天生护甲）

生命值：15（2d10+4）

速度：30尺

力量18（+4）敏捷10（+0）体质14（+2）

智力2（-4）感知10（+0）魅力4（-3）

伤害抗性：寒冷、火焰及毒素

感官：黑暗视觉60尺，被动察觉10

语言：——

挑战等级：1/2（100 
XP） 熟练加值+2

**恶臭 Stench。** 除了同类以外的任何生物，如果其回合开始时处于恶臭野牛周围5尺则必须通过 DC 
12的体质豁免检定，如果失败则该生物到自己下个回合开始都处于中毒状态。如果豁免成功，则该生物1个小时内都免疫所有恶臭野牛的恶臭气味。

**动作 Actions**

** 角抵 Gore。**近战武器攻击：命中+6，触及5尺，单一目标。命中：7（1d6+4）穿刺伤害。如果恶臭野牛先向目标直线移动至少 
20 尺，然后在同一回合里以抵撞攻击命中该目标。此时该目标将受到额外 7（2d6）钝击伤害。
