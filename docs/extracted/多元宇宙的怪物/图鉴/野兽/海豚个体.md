# 海豚个体

**海豚 Dolphin**

海豚在许多海洋世界的住民中都是智慧和嬉戏的象征。海豚出没于海洋和水元素位面，与德鲁伊和游侠交好。在许多传说中，海豚会突然出现，保护游泳者免受鲨鱼和其他水生掠食者的伤害。

**海豚 Dolphin**

*中型野兽，无阵营*

护甲等级：12（天生护甲）

生命值：11（2d8+2）

速度：0尺，游泳60尺

力量14（+2） 
敏捷13（+1） 体质13（+1）

智力6（-2） 感知12（+1） 魅力7（-2）
