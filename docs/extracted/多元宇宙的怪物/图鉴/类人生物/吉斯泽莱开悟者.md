# 吉斯泽莱开悟者

吉斯泽莱开悟者 Githzerai 
Enlightened

吉斯泽莱人花费大量的精力进行冥想，以此超越形体的局限，领悟现实的本质。完成了进阶修行的泽锡修士会被称为开悟者。

**吉斯泽莱开悟者 Githzerai Enlightened**

*中型类人（吉斯人），任意阵营*

护甲等级：18（灵能护体）

生命值：112（15d8+45）

速度：40 尺

力量14（+2） 敏捷19（+4） 体质16（+3）

智力17（+3） 感知19（+4） 魅力13（+1）

豁免：力量+6，敏捷+8，智力+7，感知+8

技能：奥秘+7，洞悉+8，察觉+8

感官：被动察觉18

语言：吉斯语

挑战等级：10（5,900 XP） 熟练加值+4

**灵能护体 Psychic Defense。** 当开悟者未着装任何护甲也未持用任何盾牌时，它的 AC 将加上它的感知调整值。

**动作 Actions**

** 多重攻击 Multiattack。**开悟者发动三次徒手打击。

** 徒手打击 Unarmed Strike。**近战武器攻击：命中+8，触及5 尺，单一目标。命中：13（2d8+4）的钝击伤害外加18（4d8）的心灵伤害。

** 时光打击 Temporal Strike（充能6）。**近战武器攻击：命中+8，触及5 
尺，单一生物。命中：13（2d8+4）的钝击伤害外加52（8d12）的心灵伤害。该目标须进行一次 DC 
16的感知豁免，失败则其向未来穿越1轮。穿越时光的目标在效应持续时间内消失。当效应终止时，目标会重新出现在其离开时的位置，若该位置已被占据，则它出现在最近的未被占据的位置上。

** 施法 Spellcasting（灵能 Psionics）。**开悟者施展以下法术之一，使用感知作为施法属性且不需要法术成分（法术豁免 DC 
16）：

*随意：法师之手 mage hand（生成隐形的手）每项3/日：识破隐形 see invisibility 每项1/日：位面转移 plane shift，传送术 teleport*

**反应 Reactions**

** 轻身坠 Slow Fall。**当开悟者要承受坠地伤害时，其可以使用反应将伤害减少50点。
