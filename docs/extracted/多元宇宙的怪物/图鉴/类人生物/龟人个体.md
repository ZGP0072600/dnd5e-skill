# 龟人个体

龟人 
Tortle

这里的龟人资料卡通常代表了一个战士，尤其是那些进行了遥远而漫长旅行的家伙。

**龟人 Tortle** *中型类人，任意阵营*

 护甲等级：17（天生护甲）

 生命值：22（4d8+4）

速度：30尺

力量15（+2） 敏捷10（+0） 
体质12（+1）

智力11（+0） 感知13（+1） 
魅力12（+1）

技能：运动+4，求生+3

感官：被动察觉11

语言：水族语,通用语

挑战等级1/4（50xp）熟练加值+2

**屏吸 Hold Breath。** 龟人可以憋气1小时。

动作 Actions

**爪击 Claws。** 近战武器攻击：命中+4，触及5尺，单一目标。命中：4（1d4+2）的挥砍伤害。

**矛 Spear。** 近战或远程武器攻击：命中+4，触及5尺或射程20/60尺，单一目标。命中：5（1d6+2）的钝击伤害，双手持用时造成6（1d8+2）的钝击伤害。

**轻弩 Light Crossbow。** 远程武器攻击：命中+2，射程80/320尺，单一目标。命中：4（1d8）的穿刺伤害。

**龟壳防御 Shell Defense。** 龟人缩回他的壳。直至他出壳，他获得 AC+4并在力量和体质豁免上具有优势。当处在龟壳内，龟人处于倒地状态，其速度为0且不能增加，他在敏捷豁免上具有劣势，且不能使用任何反应，唯一能执行的动作是使用一个附赠动作出壳。
