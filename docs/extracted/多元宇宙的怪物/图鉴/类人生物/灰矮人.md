# 灰矮人

**灰矮人 Duergars**

那些生活在幽暗地域深处或其他无光国度的矮人被称为灰矮人。先祖被夺心魔囚禁和折磨的日子深深改变了他们的能力和个性；他们得到了强大的灵能天赋，但随之被一同灌输的是一种深沉的阴暗。在一些人那，这种悲痛的重压启迪了他们那些忧郁而又宏伟的美丽造物，而在其他人那里，它则表现为一种怒火。

就像许多生活在幽暗地域的人一样，灰矮人必须时刻警惕他们邻居的阴谋与袭击。为此，灰矮人战士们必须在战斗中起到各种各样的作用，他们常将他们的狂怒与灵能天赋结合起来战斗或训练危险的幽暗地域生物作为坐骑。

尽管常常被一些人诋毁为失去乐趣的人，事实上，灰矮人对他们所做的一切都充满了深深的热情—尽管这种热情很少表现为快乐。他们将这种热烈的情感带进了自己的生活，无论是探索临近的地道，保卫家园，与家人来往，还是制作大胆的新作品。即使跨越爆发的不可避免的挫折和绝望并不总是容易的，友谊与亲情的纽带也是如此强大。同样的是，灰矮人往往非常注重团体意识—在幽暗地域，所有人都必须团结合作才能生存下去。

对于被遗忘的国度中的灰矮人来说，创造是一个富有激情的过程。灰矮人喜欢那些宏伟而坚固，使用风格简约、朴素的几何图案所铸造的作品。他们修建了短小精悍且质朴的要塞，他们铸造了不加掩饰的暴力工具。而当其他人谴责他们的作品朴实无华且冷酷无情时，灰矮人认为他们是在尊重所使用的材料，并真诚地表达他们的想法。
