# 勇士

勇士 Champion

（译注：又译作“冠军勇士”，与战士子职“勇士 Champion”同名，但能力无关。）

勇士是强大的武者，它们在战争或角斗场中磨练他们武术。对那些士兵们与为生存而战的人来说，勇士具有贵族一般的影响力，他们的存在，对他们来说如同统治者地位的象征。

最典型的勇士往往佩戴着一枚盾徽（盾形的纹章），它们的图纹象征着勇士，并同勇士一同为世人所知。你可以为一名勇士设计一枚新的盾徽，或者投掷勇士盾徽表来决定它。

勇士 Champion

*中型类人，任意阵营*

护甲等级：18（板甲）

生命值：143（22d8+44）

速度：30尺

力量20（+5） 敏捷15（+2） 体质14（+2）

智力10（+0） 感知14（+2） 魅力12（+1）

豁免：力量+9，体质+6

技能：运动+9，威吓+5，察觉+6

感官：被动察觉16

语言：任意一门语言（通常是通用语）

挑战等级：9（5,000XP） 熟练加值+4

**不屈 Indomitable（2/日）。** 勇士重骰一次其失败的豁免检定。

动作 Actions

**多重攻击 Multiattack。** 勇士发动三次巨剑或短弓攻击。

**巨剑 Greatsword。** 近战武器攻击：命中+9，触及5尺，单一目标。命中：12（2d6+5）点挥砍伤害。如果勇士剩余一半以上的生命值，则外加7（2d6）点挥砍伤害。

**短弓 Shortbow。** 远程武器攻击：命中+6，射程80/320尺，单一目标。命中：5（1d6+2）点穿刺伤害。如果勇士剩余一半以上的生命值，则外加7（2d6）点穿刺伤害。

附赠动作 Bonus Actions

**回气 Second Wind（短休或长休后充能）。** 勇士回复20点生命值。
