# 格龙蛙人蛮荒祭司

格龙蛙人蛮荒祭司 Grung Wildling

受德鲁伊教的魔法恩赐，一名格龙蛙人蛮荒祭司往往担任着教导者、治疗者、与作物的培育者之责。

格龙蛙人蛮荒祭司 Grung Wildling

*小型类人，任意阵营*

护甲等级：16（天生护甲）

生命值：27（5d6+10）

速度：25尺，攀爬25尺

力量7（-2） 敏捷16（+3） 体质15（+2）

智力10（+0） 感知15（+2） 魅力11（+0）

豁免：敏捷+5

技能：运动+2、察觉+4、隐匿+5、求生+4

伤害免疫：毒素

状态免疫：中毒

感官：被动察觉14

语言：格龙蛙语

挑战等级：1（200 XP） 熟练加值+2

**水陆两栖 Amphibious。** 格龙蛙人可以在空气和水中呼吸

**毒性皮肤 Poisonous Skin。** 任何擒抱格龙蛙人的生物，及因其他原因尝试直接触碰到格龙蛙人皮肤的生物，都必须通过一次 DC 12的体质豁免。豁免失败将陷入 **中毒 Poisoned** 状态1分钟。因此中毒的生物不再直接接触格龙蛙人的情况下，可以在每个它的回合结束时重复该豁免，豁免成功则该毒素对其的影响提前结束。

** 站立起跳 Standing Leap。**不论是否助跑，格龙蛙人跳远时均可以跳到至多25尺远，跳高时均可以跳到至多15尺高。

动作 Actions

** 匕首 Dagger。**近战或远程武器攻击：命中+5，触及5尺，或射程20/60尺，单一目标。命中：5（1d4+3）点穿刺伤害，外加5（2d4）点毒素伤害。

** 短弓 Shortbow。**远程武器攻击：命中+5，射程80/320尺，单一目标。命中：6（1d6+3）点穿刺伤害，外加5（2d4）点毒素伤害。
