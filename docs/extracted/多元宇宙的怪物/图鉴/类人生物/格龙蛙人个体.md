# 格龙蛙人个体

格龙蛙人 Grung

格龙蛙人资料卡展示了一种典型的格龙蛙人武者或猎人的数据，你可以在任何格龙蛙人的聚落找到他们。有些格龙蛙人也会离开聚落，以雇佣兵、狩猎监督官、守卫、或强盗的身份旅行。

格龙蛙人 Grung

*小型类人，任意阵营*

护甲等级：12

生命值：11（2d6+4）

速度：25尺，攀爬25尺

力量7（-2） 敏捷14（+2） 体质15（+2）

智力10（+0） 感知11（+0） 魅力10（+0）

豁免：敏捷+4

技能：运动+2、察觉+2、隐匿+4、求生+2

伤害免疫：毒素

状态免疫：中毒

感官：被动察觉12

语言：格龙蛙语

挑战等级：1/4（50 XP） 熟练加值+2

**水陆两栖 Amphibious。** 格龙蛙人可以在空气和水中呼吸

**毒性皮肤 Poisonous Skin。** 任何擒抱格龙蛙人的生物，及因其他原因尝试直接触碰到格龙蛙人皮肤的生物，都必须通过一次 DC 12的体质豁免。豁免失败将陷入 **中毒 Poisoned** 状态1分钟。因此中毒的生物不再直接接触格龙蛙人的情况下，可以在每个它的回合结束时重复该豁免，豁免成功则该毒素对其的影响提前结束。

** 站立起跳 Standing Leap。**不论是否助跑，格龙蛙人跳远时均可以跳到至多25尺远，跳高时均可以跳到至多15尺高。

动作 Actions

** 匕首 Dagger。**近战或远程武器攻击：命中+4，触及5尺，或射程20/60尺，单一目标。命中：4（1d4+2）点穿刺伤害，外加5（2d4）点毒素伤害。
