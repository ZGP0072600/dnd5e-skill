# 龟人德鲁伊

**龟人德鲁伊 Tortle Druid**

许多龟人认为这是一个奇妙的世界。他们活着是为了有机会听到微风穿过树林，观察青蛙在睡莲上鸣叫，或者伫立在拥挤的市场上。乌龟德鲁伊比大多数人更喜欢这样的东西，引导周围天地间的自然魔法。

**龟人德鲁伊 Tortle Druid**

中型类人，任意阵营

 护甲等级：17（天生护甲）

 生命值：33（6d8+4）

速度：30尺

力量14（+2） 敏捷10（+0） 
体质12（+1）

智力11（+0） 
感知15（+2） 
魅力12（+1）

技能：驯兽+4，自然+2，求生+4

感官：被动察觉12

语言：水族语,通用语

挑战等级2（450xp）熟练加值+2

**屏吸 Hold Breath。** 龟人可以憋气1小时。

动作 Actions

**爪击 Claws。** 近战武器攻击：命中+4，触及5尺，单一目标。命中：4（1d4+2）的挥砍伤害。

**自然之怒 Nature's Wrath。** 远程法术攻击：命中+4，射程90尺，单一目标。命中：9（2d6+2）点由龟人德鲁伊选择类型的伤害：寒冷，火焰，闪电，雷鸣。

**龟壳防御 Shell Defense。** 龟人缩回他的壳。直至他出壳，他获得 AC+4并在力量和体质豁免上具有优势。当处在龟壳内，龟人处于倒地状态，其速度为0且不能增加，他在敏捷豁免上具有劣势，且不能使用任何反应，唯一能执行的动作是使用一个附赠动作出壳。

**施法 Spellcasting。** 龟人德鲁伊施展以下一道法术，使用感知作为施法属性（法术豁免 DC 12）：

随意：德鲁伊戏法，神导术
每项2/日：疗伤术，定身类人，动物交谈，雷鸣波
