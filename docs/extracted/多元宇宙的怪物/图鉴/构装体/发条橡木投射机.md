# 发条橡木投射机

发条橡木投射机 Clockwork Oaken Bolter

橡木投射机并非普通的弩炮，它是一架能够远距离打击的构装生物。它的弹药能够撕裂血肉、摧毁护甲，或将敌人拖向陷阱或近战发条机械——而在较近的射程内，它还能用爆炸性的火力轰炸敌人。

发条橡木投射机 Clockwork Oaken Bolter

*中型构装，无阵营*

 护甲等级：16（天生护甲）

 生命值：117（18d8+36）

速度 30尺

力量12（+1）敏捷18（+4）体质15（+2）

智力3（-4）感知10（+0）魅力1（-5）

伤害免疫：毒素

状态免疫：魅惑，力竭，恐慌，麻痹，石化，中毒

感官：黑暗视觉60尺，被动察觉10

语言：懂得一门制造者的语言但不会说

 
挑战等级：5（1,800XP）熟练加值：+3

**魔法抗性 Magic Resistance。** 发条机械在对抗法术和其他魔法效果的豁免检定上具有优势。

**非凡本质 Unusual Nature。** 发条机械不需要空气、食物，水或睡眠。

动作 Actions

**多重攻击 Multiattack。** 发条机械进行两次穿刺箭矢或一次穿刺箭矢和一次鱼叉攻击。

**穿刺箭矢 Lancing Bolt。** 近战或远程武器攻击：命中+7，触及5 尺或射程100/400尺，单一目标。命中：15（2d10+4）穿刺伤害。

**鱼叉 Harpoon。** 远程武器攻击：命中+7，射程50/200尺，单一目标。命中：9（1d10+4）穿刺伤害，并且目标被擒抱（逃脱 DC 12）。当处于这种受擒状态时，该生物的速度不会降低，但只能朝更靠近这台发条机械的方向移动。如果该生物从擒抱中逃脱或者做出尝试后失败了，它受到5（1d10）挥砍伤害。发条机械同时只能擒抱一个生物。

**爆炸箭矢 Explosive Bolt（充能5~6）。** 发条机械向120尺内一点射出一枚爆炸弹药。每个位于该点20尺内球形范围内的生物都必须成功头发 DC 15的敏捷豁免，失败的情况下受到17（5d6）火焰伤害，成功的情况下受到一半伤害。

附赠动作 Bonus Actions

**收线 Reel in。** 发条机械将被它擒抱的生物拉近20尺。
