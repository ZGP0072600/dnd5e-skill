# 发条机械

发条机械 Clockworks

侏儒们在魔法和机械装置上工艺制造出了许多失败的构装生物，但也同样带来了真正的进步，比如发条机械。制造这些发条机械的方法已经在侏儒社区之间流传了数代。

**个性化设计 Individual Designs**

侏儒工匠们更看重一台个性化的发条机械，而不是为了造出一台功能完美的发条机械，而盲目过照搬另一个作品的做法。发条机械可以通过在其数据中添加以下增强功能之一和一个潜在的故障来实现个性化。你可以随机选择，也可以选择一对适合其设计者风格的改造方案。

| 1d10 | 发条机械增强 |
|---|---|
| 1 | 伪装。如果发条机械还没有敏捷（隐匿）熟练项，它会获得该技能。当它静止不动时，与停止的机器没有什么区别。 |
| 2 | 传感器。发条机械的黑暗视觉范围增加60尺。此外，如果原本没有，它可以获得感知（觉察）熟练。 |
| 3 | 提高护甲。发条机械的 AC 增加2。 |
| 4 | 增加速度。发条机械的速度增加10尺。 |
| 5 | 强化结构。这种发条机械具有力场、闪电和雷鸣伤害的抗性。 |
| 6 | 自我修复。如果发条机械在其回合开始时至少有1点生命值，它将恢复5点生命值。如果它受到闪电伤害，这个能力在其下一个回合开始时不会生效。 |
| 7 | 坚固构造。发条机械的生命值上限增加等于它的生命骰数。 |
| 8 | 吸附。发条机械获得30英尺的攀爬速度。 |
| 9 | 发声谐振器。发条机械获得讲基础的通用语或侏儒语（由创造者选择）的能力。 |
| 10 | 涡轮推进。发条机械获得30尺的游泳速度。 |

| 1d8 | 发条装置故障 |
|---|---|
| 1 | 传感器失灵。在发条机械的回合开始时投掷 d6。如果骰出1，发条机械直到它回合结束为止处于目盲。 |
| 2 | 定位仪缺陷。 在发条机械的回合开始时投掷 d6。如果骰出1，发条机械的攻击骰具有劣势直到其回合结束。 |
| 3 | 接地故障。发条机械具有闪电伤害易伤。 |
| 4 | 底层循环。在发条机械的回合开始时投掷 d6。如果骰出1，发条机械会将一个它能看到的30英尺内的生物误认为其创造者。发条机械在一分钟内不会伤害该生物，直到该生物攻击它或对它造成伤害为止。 |
| 5 | 转弯卡死。发条机械必须沿直线运动。它可以在运动前和运动中点转向90度。如果其在该回合中没有移动，它可以改为自由转向。 |
| 6 | 优先自保。如果发条机械在战斗中开始其回合时只有一半或更少的生命值，投掷 d6。如果骰出1，它将从战斗中撤退。如果撤退是不可能的，它会继续战斗。 |
| 7 | 过热。在发条机械的回合开始时投掷 d6。如果骰出1，发条机械陷入失能，直到它的回合结束。 |
| 8 | 齿轮生锈。发条机械在先攻检定上有劣势，它的速度会下降10尺。 |
