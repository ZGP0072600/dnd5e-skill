# 猎魔蛛

猎魔蛛 Retriever

（译注：该名称源自“寻回犬 Retriever”一词。）

猎魔蛛是强大的、蜘蛛形的构装生物，由幽暗地域的 罗丝 Lolth 追随者们设计并制造。而他们被分配的职责只有一件事——在深渊逡巡并俘虏恶魔，供邪教徒们奴役或进行仪式。事实证明，这些自动机械高效且恐怖，因此可以胜任许多不同的其他任务。

虽然猎魔蛛被设计为供深渊作业使用，但当强大的罗丝信徒希望完好的活捉某些生物或找回某件物品时，它们也会派遣猎魔蛛。在其他生物的手中见到猎魔蛛几乎是不可能的，罗丝信徒几乎不可能出售或转手这些构装生物，因为他们不想有任何机会让他们的造物反咬自己一口。

猎魔蛛 Retriever

*大型构装，普遍守序邪恶*

护甲等级：19（天生护甲）

生命值：210（20d10+100）

速度：40尺、攀爬40尺

力量22（+6） 敏捷16（+3） 体质20（+5）

智力3（-3） 感知11（+0） 魅力4（-3）

豁免：敏捷+8、体质+10、感知+5

技能：察觉+5、隐匿+8

伤害免疫：暗蚀、毒素、心灵；非魔法且非精金攻击的钝击、穿刺、挥砍

状态免疫：魅惑、力竭、恐慌、麻痹、中毒

感官：盲视30尺，黑暗视觉60尺，被动察觉15

语言：理解深渊语、精灵语、及地底通用语，但不能说

挑战等级：14（11,500 XP） 熟练加值+5

**完美追迹者 Faultless Tracker。** 猎魔蛛的主人为猎魔蛛指定一个猎物。猎物可以是主人熟悉的特定生物或物件，也可以是猎魔蛛主人曾见过的大致种类的生物或物件。只要猎魔蛛与它的猎物身处同一存在位面，它就可以知晓自己与猎物之间的方向与距离。猎魔蛛同时只能有一个猎物。猎魔蛛同时也永远知晓其主人所在位置。

动作 Actions

**多重攻击 Multiattack。** 猎魔蛛发动两次前肢攻击，然后使用力场射线或麻痹射线（若可用）。

**前肢 Foreleg。** 近战武器攻击：命中+11，触及10尺，单一目标。命中：15（2d8+6）点挥砍伤害，

**力场射线 Force Beam。** 猎魔蛛指定周围60尺范围内的一个可见生物为目标。迫使目标通过一次 DC 16的敏捷豁免，豁免失败将受到27（5d10）点力场伤害，豁免成功则只受半数伤害。

**麻痹射线 Paralyzing Beam（充能5~6）。** 猎魔蛛指定周围60尺范围内的一个可见生物为目标。迫使目标通过一次 DC 18的体质豁免，豁免失败将陷入 麻痹 Paralyzed 状态，持续1分钟。因此被麻痹的目标可以在每个其自己回合结束时重复此豁免，豁免成功则它身上的此麻痹效应提前结束。

如果被麻痹的生物体型不超过中型，猎魔蛛可以将其捡起（作为移动的一部分），并且保持全速移动或者攀爬。
