# 火巨人无畏者

**火巨人无畏者 Fire Giant Dreadnought**

大部分火巨人不仅尊重力量，还重视锻造的技巧。火巨人的铸造厂集神殿、学校、试验场和政治中心功能于一体，是任何火巨人社群的核心。

那些以体魄为先的火巨人常被吩咐去做锻造风箱、搬运煤炭之类最底层的工作。然而，他们中最强壮的巨人可以通过担任一种独有的职位——无畏者，来胜过他人并提升等级。

强大的火巨人无畏者挥舞着两面如同犁刀般的巨盾。这些盾牌外部有尖刺，内部中空，一有危险迹象，火巨人无畏者就会将炽热的煤炭倒入其中。无畏者可以使用他的两面盾牌，对朝着任何攻击他的人竖起一堵热焰之墙。无畏者结束战斗之时，敌人往往变成了一地冒烟的残骸。

在没有战斗的时候，火巨人无畏者使用他们的盾牌将大量煤炭、石块和矿石推运向铸造厂，以维持他们的力量。火巨人无畏者偶尔也会被上级巨人要求随战争或外交代表团出访，其凶猛和威慑性的举止能增强代表团的地位。
