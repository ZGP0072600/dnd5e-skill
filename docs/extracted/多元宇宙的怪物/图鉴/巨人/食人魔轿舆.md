# 食人魔轿舆

**食人魔轿舆 Ogre Howdah**

作为最不寻常的特殊食人魔，轿舆在背后背着栅栏木堡。堡垒是大到足以作为一个容纳四个小体型类人的战斗平台。最常见的食人魔轿舆承载了配备弓箭和长矛的地精投入战斗,但是改成运输成狗头人,地底侏儒或其他类似大小的类人生物也并非不可。

**轿舆 Howdah。** 这个食人魔背着个简单的堡垒。至多四个小体型生物可以站在上面而不会挤来挤去。想要对五尺内的生物进行一次近战武器攻击，需要使用矛或者具有触及特性的武器。在堡垒里的生物对来自外面的攻击和效应具有四分之三掩护。如果食人魔死掉，那么里面的生物会出现在五尺内的空地。

**动作 Actions**

** 硬头锤 mace。**近战武器攻击：命中+6，触及 5 尺，单一 目标。命中：11（2d6+4）的钝击伤害。
