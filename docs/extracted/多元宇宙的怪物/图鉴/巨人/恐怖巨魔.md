# 恐怖巨魔

恐怖巨魔 Dire Troll

巨魔会杀死并吃掉几乎所有东西，包括在一些稀有案例中，它们会吃掉其他巨魔。这种食用同类的情况会影响巨魔导致它变的异乎寻常的大。这些恐怖巨魔非常渴望越来越多的巨魔肉体来帮助它们继续增长。

恐怖巨魔也会通过移植肉体或者器官来增加它们的大小，当一块正在活动的巨魔肉体被嫁接到恐怖恶魔的新生成的伤口时，它们会在伤口处再生并混合新的肉体进它们的肌肉中。更可怕的是，恐怖巨魔会从牺牲品上撕裂下躯体，并融合成为多手，多眼，多爪以及各种器官的怪物。

恐怖巨魔 Dire Troll

*巨型巨人，普遍混乱邪恶*

 护甲等级：15（天生护甲）

 生命值：172(15d12+75)

 速度:40尺

力量22（+6）敏捷15（+2）体质21（+5）

智力9（-1）感知11（0）魅力5（-3）

豁免 感知+5，魅力+2

技能：察觉+10

 伤害抗性：来自非魔法攻击的钝击、穿刺、挥砍

 状态免疫：恐慌，中毒

感官：黑暗视觉60尺，被动察觉20

 语言：巨人语

挑战等级：13（10000XP）熟练加值+5

**再生 Regeneration。** 巨魔在它的回合开始时回复10 点生命值。如果巨魔受到强酸或者火焰伤害，在它下个回合开始时只回复5点生命值。巨魔只会在它生命值为0，且在单次攻击中受到10点或以上强酸或火焰伤害时才会死去。

**动作 Actions**

多重攻击 Multiattack。巨魔发动五次攻击：一次啃咬和四次爪击

** 啃咬 Bite。**近战武器攻击：命中+11，触及10 尺，单一目标。命中：10 （1d8+6）的穿刺伤害，外加5（1d10）的毒素伤害。

** 爪击 Claw。**近战武器攻击：命中+11，触及10 尺，单一目标。命中：16 （3d6+6）的挥砍伤害。

** 利爪旋风 Whirlwind of Claws (充能5~6)。**在巨魔周围10尺的每一个生物必须通过一次 DC 19的敏捷豁免，豁免失败者将受到44（8d10）的挥砍伤害，豁免成功则伤害减半。
