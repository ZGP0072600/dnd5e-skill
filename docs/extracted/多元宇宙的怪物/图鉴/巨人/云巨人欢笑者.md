# 云巨人欢笑者

**云巨人欢笑者 Cloud Giant Smiling One**

欢笑者是视尊崇和效仿曼诺的诡计与欺骗高于一切的云巨人。他们是最高明的骗子，极尽花招、欺骗、误导和魔法之能事追求财富。他们的天性难以预料，还有着缺德的幽默感。欢笑者的行为僭越了礼节的所有界限，以至低俗的无赖都会认为他们的所作所为有失体面。

“欢笑者”这个名字来源于他们佩戴的古怪双面面具。双面面具的其中一半是笑脸，看起来往往更像是假笑或得意洋洋的讥笑，而非宜人的笑脸。另一半双面面具皱着眉，表明欢笑者对它们处于风暴巨人之后的第二序位位置的不满。双面面具象征着云巨人欢笑者的信仰，与此同时欢笑者面部的真实表情也隐藏在面具之下。

**云巨人欢笑者 Cloud Giant Smiling One**

*巨型巨人，普遍混乱中立*

护甲等级：15（天生护甲）

生命值：250（20d12+120）

速度：40尺，飞行40尺（悬浮）

力量26（+8） 
敏捷12（+1） 体质22（+6） 

智力15（+2） 感知16（+3） 魅力17（+3） 

豁免：体质+10，智力+6，魅力+7 

技能：欺瞒+11，洞悉+7，察觉+11，巧手+9 

感官：被动察觉21 

语言：通用语，巨人语 

挑战等级：11（7200XP） 
熟练加值+4

**操控天气（八环法术）Control Weather (8th-level Spell)。** 巨人可以施放法术*操控天气 Control Weather*，无需材料成分，使用魅力作为施法属性。

**动作 Actions**

** 多重攻击 Multiattack。**巨人进行两次猛击或两次念力打击。

** 猛击 Slam。**近战武器攻击：命中+12，触及10尺，单一目标。命中：21（3d8+8）钝击伤害加上5（1d10）心灵伤害。

** 念力打击 Telekinetic Strike。**远程武器攻击：命中+7，射程240尺，单一目标。命中：25（4d10+3）力场伤害。

** 变化形体 Change Shape。**巨人如魔法般变形成它看到的野兽或者类人生物，或者回到他的真实形态。巨人穿着或携带的任何装备都会被新形态吸收。他的数据，除了它的体形外，在每种形态下都是相同的。如果他死了，它变回其真实形态。

** 施法 Spellcasting。**巨人施展以下法术之一，无需材料成分，且以魅力为其施法属性（法术豁免 DC 15）：

*随意：侦测魔法，云雾术，光亮术，次级幻象每项3/日：隐形术，无声幻影，暗示术，羽落术，巧言术每项1/日：气化形体，高级幻影*

**附赠动作 Bonus Actions**

** 腾云步 Cloud Step（充能4~6）。**巨人和他携带、穿着的所有装备传送到至多60尺内一个他能看见且未被占据的位置。
