# 布雷祖魔

**布雷祖魔 Bulezau**

布雷祖魔有着身患疾病的外表与兽性的愤怒，体现了自然的暴力。这些恶魔潜藏在深渊各处的深邃峡谷中和高耸峭壁上，许多还在诸恶魔领主的大军中找到了自己的立足之地，作为步兵在深渊无尽的战争中效力。

布雷祖魔渴求暴力。它们渴望杀戮也乐意赴死，这让它们成为了许多恶魔领主随行军队的通常成员。当没有被更大、更强悍的恶魔管制的时候，布雷祖魔会聚集成争斗不止的大群，互相扭打战斗，直到有一个更好的目标出现，或是直到一个更强大的恶魔强迫它们服从。

损毁外表的疾病困扰着布雷祖魔：生着硬痂的眼睛，蛆虫蠕动的创口，以及无论它们去哪都如影随形的、腐烂肉体带来的恶臭。

**布雷祖魔 Bulezau**

*中型邪魔（恶魔），普遍混乱邪恶*护甲等级：14（天生护甲）

生命值：52（7d8+21）

速度：40尺

力量15（+2）敏捷14（+2）体质17（+3）

智力8（-1）感知9（-1）魅力6（-2）

伤害抗性：寒冷，火焰，闪电

伤害免疫：毒素

状态免疫：魅惑，恐慌，中毒

感官：黑暗视觉120尺，被动察觉9

语言：深渊语，心灵感应60尺

挑战等级：3（700XP） 
熟练加值 +2

**腐烂存在 Rotting Presence。** 当任何非恶魔的生物在此布雷祖魔30尺内开始其回合时，该生物必须成功通过一个 DC 13的体质豁免，否则承受3（1d6）点暗蚀伤害。

**立定起跳 Standing Leap。** 布雷祖魔可以跳远至多20尺，跳高至多10尺，无论起跳前有没有助跑。

**步履稳健 Sure-Footed。** 布雷祖魔在进行对抗使其倒地的力量豁免和敏捷豁免时具有优势。

**动作 Actions**

** 刺尾 Barbed Tail。**近战武器攻击：命中+4，触及5尺，单一目标。命中：8（1d12+2）点穿刺伤害外加4（1d8）点暗蚀伤害。若该目标是一个生物，其必须成功通过一个对抗疾病的 DC 13的体质豁免，否则中毒直至疾病结束。当因此而中毒时，该目标长出溃烂的疡肿、咳出苍蝇、皮肤腐烂剥落，且目标在之后每24小时便必须再次进行豁免。若豁免成功，则此疾病终止。若豁免失败，该目标的生命值上限减损4（1d8）点。若目标的生命值上限减至0，则目标死亡。
