# 畸魔

**畸魔 Rutterkin**

畸魔是成群游荡在无底深渊，持续不断地寻找着闯入者并将之包围吞食的扭曲恶魔。这些魔族保卫深渊不被非恶魔生物入侵。当发现任何闯入者时，它们都会集结成群，如潮水般涌上前去，在攻击前先放射出恐怖的波浪，使得其受害者因恐惧而迈不开步子。

 
 
 
 
 被畸魔咬中的生物会暴露在一种可怕的疾病中，使得他们被深渊的腐化影响所侵染。承受了这种疾病的受害者会经历巨大的痛苦，同时他们的身体会变得畸形可怖，血肉在骨骼上扭曲，直到最后被转化成无数原魔 mane（见《怪物图鉴》）中的一员，跟随在让他们退化成这样的畸魔大群后面。

**畸魔 Rutterkin**

*中型邪魔（恶魔），普遍混乱邪恶*

护甲等级：12

生命值：37（5d8+15）

速度：20尺

力量14（+2） 
敏捷15（+2） 体质17（+3）

智力5（-3） 感知12（+1） 
魅力6（-2）

伤害抗性：寒冷，火焰，闪电

伤害免疫：毒素

状态免疫：魅惑，恐慌，中毒

感官：黑暗视觉120尺，被动察觉11

语言：理解深渊语但不会说

 
挑战等级：2（450 XP） 熟练加值+2

**禁足恐惧 Immobilizing Fear。** 当一个非恶魔的生物在一个或更多畸魔30尺内开始其回合时，该生物必须进行一个 DC 11的感知豁免。若该生物的30尺内有六个或更多畸魔，则该生物在此豁免上有劣势。若豁免失败，该生物对该畸魔恐慌1分钟。当因此而恐慌时，该生物被束缚。恐慌的目标在其每个回合结束时可以重复进行此豁免，豁免成功则终止其所承受的此效应。若豁免成功，则此生物在接下来的24小时里免疫所有畸魔的 
**蹒跚恐惧 Crippling Fear**（译注原文如此，疑有误）。

** 动作 Actions**

** 啃咬 Bite。**近战武器攻击：命中+4，触及5尺，单一目标。命中：12（3d6+2）点穿刺伤害。若该目标是一个生物，则其必须成功通过一个 DC 13的对抗疾病的体质豁免，否则中毒。在每次长休结束时，因此而中毒的生物可以重复此豁免，成功则结束此效果。若该目标在因此而中毒时生命值下降为0，则其死亡并立刻被转化为一个活着的 ** 原魔 Mane**。此转变只能被一道*祈愿术 wish*法术所逆转。
