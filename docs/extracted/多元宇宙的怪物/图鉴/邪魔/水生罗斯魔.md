# 水生罗斯魔

**水生罗斯魔 Hydroloth**

正如他们所栖居的消除人记忆的冥河水一样，水生罗斯魔摄取攻击目标的记忆，偷走他们的思想，献给给他们所侍奉的主人。水生罗斯魔善于寻找丢失之物，特别是淹没在深处的东西。

两栖或者水下作战的情况中，水生罗斯魔在尤格罗斯魔中无人能比。他们有时候会袭击并击沉船只，或者突袭滨海据点。

**水生罗斯魔 Hydroloth** *中型邪魔（尤格罗斯魔），普遍中立邪恶*

护甲等级：15

生命值：135（18d8+54）

速度：20尺，游泳40尺

力量12（+1） 
敏捷21（+5） 体质16（+3）

智力19（+4） 感知10（+0） 
魅力14（+2）

技能：洞悉+4，察觉+4

伤害易伤：火焰

伤害抗性：寒冷、闪电；非魔法攻击的钝击、穿刺、挥砍伤害

伤害免疫：强酸、毒素

状态免疫：中毒

感官：盲视60尺，黑暗视觉60尺，被动察觉14

语言：深渊语，炼狱语，心灵感应60尺。

挑战：9（5,000 
XP）熟练加值：+4

**两栖 Amphibious。** 水生罗斯魔可以在水中和空气中呼吸。

**魔法抗性 Magic Resistance。** 水生罗斯魔在对法术和其他魔法效果的豁免检定上有优势。

**牢固记忆 Secure Memory。** 水生罗斯魔免疫冥河之水以及其他任何能够窃取或者修改其记忆，以及侦测或阅读其思想的效应。

**水生优势 Watery Advantage。** 浸没在水中时，水生罗斯魔的攻击检定具有优势。

**动作 Actions**

** 多重攻击 Multiattack。**水生罗斯魔进行两次啃咬或爪抓攻击。其中一次攻击可以替换为施展一个法术。

** 啃咬 Bite。**近战武器攻击：+9命中，触及5尺，单一目标。命中：16(2d10+5)点穿刺伤害，外加9（2d10）点心灵伤害。

** 爪击 Claws。**近战武器攻击：+9命中，触及5尺，单一目标。命中：14(2d8+5)挥砍伤害，外加9（2d10）点心灵伤害。

** 施法 Innate Spellcasting 。**水生罗斯魔施展如下法术之一，且不需要材料成分，其施法属性为智力（法术豁免 DC 16)：

*随意：黑暗术 darkness，侦测魔法 detect magic，解除魔法 dispel magic，隐形术 invisibility (仅自身)每项3/日：操控水体 control water，疯狂冠冕 crown of madness，恐惧术 fear，暗示术 suggestion***记忆窃取 Steal Memory (1/日)。** 水生罗斯魔以60尺内可以看到的一个生物为目标。目标受到14（4d6）点心灵伤害，并且需要进行一次 DC 16的智力豁免。豁免成功则该生物24小时内免疫该水生罗斯魔的记忆窃取。豁免失败则目标失去所有熟练项；不能施法；不能理解语言；并且如果他的智力和魅力属性值大于5，则变为5。每当目标完成一次长休便可以重新进行豁免，豁免成功时效应终止。一个施展在目标身上的*高等复原术 greater restoration*或者*移除诅咒 remove curse*可以提前结束效应。

**传送 Teleport。** 水生罗斯魔可以携带任何穿着或携带的物品传送到60尺内能看见的未被占据的空间里。
