# 卓格洛斯魔

**卓格洛斯魔 Draegloth**

卓格洛斯魔是由罗丝的一位精灵祭司在一个渎神的、危险的仪式中创造的恶魔，在这个仪式中，它被注入了制造者的妖精本质和迷诱魔的邪魔本质。这种仪式很少成功，但罗丝的忠实信徒认为它值得冒险，因为所得到的生物被赋予了天生的魔法和身体力量。卓格洛斯魔通常服务于它的制造者，将它对毁灭的渴望赋予制造者战胜对手的计划。

卓格洛斯魔是一种食人魔大小的四臂两足动物，紫色皮肤，头发苍白。它的两只手臂肌肉发达，末端有锋利的爪子；另外两个是精灵手臂的大小和形状，能够做出精致的动作。尽管这种生物肌肉发达，但却像精灵一样优雅。它野兽般的脸上长着发光的红眼睛，狗一样的鼻子，还有一张满是锋利牙齿的嘴。

在被遗忘的国度，魔索布莱城的卓尔贵族世家中，一位高级女祭司成功地创造出一只卓格洛斯魔被视为罗丝对其世家青睐的标志，也是恶魔领主漠视家族的没有这一天赋的对手的标志。这一造物促使世家的领导人开始制定新的计划，在卓格洛斯魔完全长大后打击对手。这些计划以卓格洛斯魔为重要角色，因为它的能力可以在与没有卓格洛斯魔的世家的战斗中扭转局势。

虽然卓格洛斯魔在罗丝的邪教的计划中扮演着重要角色，但卓格洛斯魔不能超越在该邪教中祭司的受欢迎仆人的地位。在卓格洛斯魔被赋予任何职责之前，它会接受指令，接受为其设置的角色，而不是挑战权威。大多数卓格洛斯魔都非常讨厌别人的命令，但由于他们的训练，他们通常会把挫折感发泄在制造者的敌人身上，而不是发泄在制造者身上。一个无法抑制自己野心的卓格洛斯魔可能会抛弃它的制造者自立谋生。目前尚不清楚这些反叛的卓格洛斯魔是否是罗丝播种混乱计划的一部分。

**卓格洛斯魔 Draegloth**

*大型邪魔（恶魔），普遍混乱邪恶*

护甲等级：15（天生护甲）

生命值：123（13d10+52）

速度：30尺

力量20（+5）敏捷15（+2）体质18（+4）

智力13（+1）感知11（+0）魅力11（+0）

技能：察觉+3，隐匿+5

伤害抗性：寒冷，火焰，闪电

伤害免疫：毒素

状态免疫：中毒

感官：黑暗视觉120尺，被动察觉13

语言：深渊语，精灵语，地底通用语

挑战等级：7（2900xp） 
熟练加值 +3

**妖精血统 Fey Ancestry。** 卓格洛斯魔对抗魅惑时进行的豁免检定具有优势，并且不会因魔法而陷入睡眠。

动作 Actions

**多重攻击 Mutiattack。** 卓格洛斯魔发动一次啃咬和两次爪击。

**啃咬 Bite。** 近战武器攻击：命中+8，触及5尺，单一目标。命中：16（2d10+5）穿刺伤害。

**爪击 Claw。** 近战武器攻击：命中+8，触及10尺，单一目标。命中：16（2d10+5）挥砍伤害。

**施法 Spellcasting。** 卓格洛斯魔施展以下法术中的一个，且不需要材料成分，以魅力为施法关键属性（法术豁免 DC 11）：

*随意：舞光术 dancing lights，黑暗术 darkness 每项1/日：困惑术 confusion，妖火 faerie fire*
