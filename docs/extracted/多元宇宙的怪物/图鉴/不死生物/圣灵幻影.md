# 圣灵幻影

圣灵幻影 Eidolon

（译注：在万智牌中被译作“幻灵”。）

众神往往会放置圣灵幻影来守护那些被称之为圣地的地方。圣灵幻影，一种被束缚在圣地的鬼魂，由坚定不移的信仰者之魂铸成。它们潜伏在神殿与宝库中，防止此地不会被敌人玷污、破坏、掠夺。如果有敌人胆敢踏入圣灵幻影所守护的圣地，圣灵幻影将钻入专门为其灵魂量身打造的雕像之中，将其活化、驾驶，并以此驱赶入侵者。

圣灵幻影 Eidolon

*中型亡灵，任意阵营*

护甲等级：9

生命值：63（18d8-18）

速度 0尺，飞行40尺（悬浮）

力量7（-2） 敏捷8（-1） 体质9（-1）

智力14（+2） 感知19（+4） 魅力16（+3）

豁免：感知+8

技能：察觉+8

伤害抗性：强酸、火焰、闪电、雷鸣；非魔法攻击的钝击、穿刺、挥砍

伤害免疫：寒冷、暗蚀、毒素

状态免疫：魅惑、力竭、恐慌、擒抱、麻痹、石化、中毒、倒地、束缚

感官：黑暗视觉60尺，被动察觉18

语言：生前所知晓的语言

挑战等级：12（8,400 XP） 熟练加值+4

**虚体移动 Incorporeal Movement。** 圣灵幻影可以如同穿过困难地形一般穿过其他生物和物件。若回合结束时，其仍处在圣灵塑像以外的物体内部，则受到 5（1d10）点力场伤害。

**圣灵活化 Sacred Animation（充能5~6）。** 当圣灵幻影进入圣灵塑像所占据的空间时，它可以传入雕像，并让圣灵塑像转变为一个由它控制的生物。此时，圣灵幻影改为使用 圣灵塑像 Sacred Statue 的数据。

**驱散抗性 Turn Resistance。** 圣灵幻影为对抗任意驱散亡灵的效应所做的豁免具有优势。

动作 Actions

**神圣敬畏 Divine Dread。** 圣灵幻影迫使其周围60尺范围内的所有生物进行一次 DC 15的感知豁免，豁免失败的生物将陷入 **恐慌 Frightened** 状态1分钟。因此恐慌期间，恐慌生物必须在其每个回合开始时执行 疾走 Dash 动作并尽可能向着远离圣灵幻影的方向，沿着安全的路径移动。若其已无处可逃，则其陷入 ** 震慑 Stunned** ，直至其再次能够逃离。一个因此恐慌的生物可以在其回合结束时重骰此豁免，豁免成功则它身上的此恐慌效应将提前结束结束。不论是生物豁免成功，亦或是生物身上的此恐慌效应结束，接下来24小时内，该生物免疫于所有圣灵幻影的 神圣敬畏 Divine Dread 动作项。

圣灵塑像 Sacred Statue

*大型构装，阵营同内部的圣灵幻影*

护甲等级：19（天生护甲）

生命值：95（10d10+40）

速度：25尺

力量19（+4） 敏捷8（-1） 体质19（+4）

智力14（+2） 感知19（+4） 魅力16（+3）

豁免：感知+8

伤害抗性：强酸、火焰、闪电、雷鸣；非魔法攻击的钝击、穿刺、挥砍

伤害免疫：寒冷、暗蚀、毒素

状态免疫：魅惑、力竭、恐慌、擒抱、麻痹、石化、中毒、倒地、束缚

感官：黑暗视觉60尺，被动察觉14

语言：内部的圣灵幻影生前所知晓的语言

挑战等级：——

**拟形 False Appearance。** 当圣灵塑像在战斗开始时保持静止，它进行的先攻检定具有优势。不仅如此，如果一个生物未曾见过这尊圣灵塑像移动或行动，生物必须得通过一次 DC 18的智力（调查）检定才能发觉这是一尊特制的雕像，而不是一般物件。

**灵之居所 Ghostly Inhabitant。** 圣灵幻影传入圣灵塑像后，它将持续保持在雕像内部的状态，持续直至圣灵塑像的生命值降至0、圣灵幻影用一个附赠动作主动传出塑像、圣灵幻影被驱散，或圣灵幻影被诸如法术 驱逐善恶 Dispel Evil and Good 之类的效应强迫逐出为止。不论圣灵幻影如何离开圣灵塑像，它都出现在雕像周围5尺范围内的一处未被占据的空间内。

**惰性 Inert。** 若没有任何圣灵幻影进入，圣灵塑像是一个物件。

**非凡本质 Unusual Nature。** 塑像无需呼吸、吃、喝、以及睡觉。（译注：你的意思是圣灵幻影需要？）
动作 Actions

**多重攻击 Multiattack。** 塑像发动两次猛击或投石攻击。

**猛击 Slam。** 近战武器攻击：命中+8，触及10尺，单一目标。命中：43（6d12+4）点钝击伤害。

**投石 Rock。** 远程武器攻击：命中+8，射程60尺/240尺，单一目标。命中：37（6d10+4）点钝击伤害。
