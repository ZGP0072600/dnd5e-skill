# 剑下鬼卒

剑下鬼卒 Sword Wraith Warrior

你也许能在那些古老的斗争之地找到剑下鬼。那里，乃古代士兵们被无情围剿、被无情屠戮之地。

剑下鬼卒 Sword Wraith Warrior

*中型亡灵，普遍守序邪恶*

护甲等级：16（链甲衫，盾牌）

生命值：45（6d8+18）

速度：30尺

力量18（+4） 敏捷12（+1） 体质17（+3）

智力6（-3） 感知9（-1） 魅力10（+0）

伤害抗性：暗蚀；非魔法攻击的钝击、穿刺、挥砍

伤害免疫：毒素

状态免疫：力竭、恐慌、中毒、昏迷

感官：黑暗视觉60尺，被动察觉9

语言：它生前知晓的语言

挑战等级：3（700XP） 熟练加值+2

**非凡本质 Unusual Nature。** 鬼卒无需呼吸、吃、喝、以及睡眠。

动作 Actions

**战斧 Battleaxe。** 近战武器攻击：命中+6，触及5尺，单一目标。命中：8（1d8+4）点挥砍伤害（双手持斧时9（1d10+4）点挥砍伤害）。

**长弓 Longbow。** 远程武器攻击：命中+3，射程150/600尺，单一目标。命中：5（1d8+1）点穿刺伤害。

附赠动作 Bonus Actions

**战争狂怒 Martial Fury。** 鬼卒发动一次战斧攻击或是长弓攻击。但是，直到其下个回合开始前，任何对此鬼卒发动的攻击检定均具有优势。
