# 水元素侍卫

水元素侍从 Water Elemental Myrmidon

*中型元素生物，普遍绝对中立*

护甲等级：18（板甲）

生命值：127（17d8+51）

速度：40尺，游泳40尺

力量18（+4） 敏捷14（+2） 体质16（+3）

智力8（-1） 感知10（+0） 魅力10（+0）

伤害抗性：强酸；来自非魔法攻击的钝击、穿刺、挥砍

伤害免疫：毒素

状态免疫：麻痹，石化，中毒，倒地

感官：黑暗视觉60尺，被动察觉10

语言：水族语，外加一门其创造者所选的语言

挑战等级：7（2900XP） 熟练加值+3

动作 Actions

**多重攻击 Multiattack。** 元素侍从发动三次三叉戟攻击。

**三叉戟 Trident。** 近战武器攻击：命中+7，触及5尺，单一目标。命中：7（1d6+4）点力场伤害（双手持戟时8（1d8+4）点力场伤害）。

**冻寒打击 Freezing Strikes（充能6）。** 元素侍从使用它的多重攻击，且每一次攻击击中都会额外造成5（1d10）点寒冷伤害，并使目标的速度降低10尺，持续直至元素侍从的下个回合结束。
