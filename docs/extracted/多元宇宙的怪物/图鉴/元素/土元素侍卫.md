# 土元素侍卫

土元素侍从 Earth Elemental Myrmidon

*中型元素生物，普遍绝对中立*

护甲等级：18（板甲）

生命值：127（17d8+51）

速度：30尺

力量18（+4） 敏捷10（+0） 体质17（+3）

智力8（-1） 感知10（+0） 魅力10（+0）

伤害抗性：来自非魔法攻击的钝击、穿刺、挥砍

伤害免疫：毒素

状态免疫：麻痹，石化，中毒，倒地

感官：黑暗视觉60尺，被动察觉10

语言：土族语，外加一门其创造者所选的语言

挑战等级：7（2900XP） 熟练加值+3

动作 Actions

**多重攻击 Multiattack。** 元素侍从发动两次巨锤攻击。

**巨锤 Maul。** 近战武器攻击：命中+7，触及5尺，单一目标。命中：11（2d6+4）点力场伤害。

**震颤打击 Thunderous Strike（充能6）。** 元素侍从发动一次巨锤攻击。若该次攻击命中将造成额外16（3d10）点雷鸣伤害，并且目标必须通过一次 DC 14的力量豁免，豁免失败将应击 倒地 Prone 。
