# 菲涅克斯

菲涅克斯 Phoenix

（译注：又译作“凤凰”或“不死鸟”。DND 的这只是两者的嵌合体并非两者之中的任何一位，因此各位用自己喜欢的称呼便是。）

一只从内层位面释放而出的菲涅克斯，将会创造出一片沿天空中蔓延的火焰。在火焰的中心有一只庞大的火鸟——一只沉溺于将一切焚烬殆尽的 上古元素 Elder 
Elemental 。菲涅克斯企图将世界化为火狱，因此它很少长久驻留在某一地点。

菲涅克斯 Phoenix

*超巨型元素，普遍绝对中立*

护甲等级：18

生命值：175（10d20+70）

速度：20尺，飞行120尺

力量19（+4） 敏捷26（+8） 体质25（+7）

智力2（-4） 感知21（+5） 魅力18（+4）

豁免：感知+10，魅力+9

伤害抗性：非魔法攻击的钝击、穿刺、挥砍

伤害免疫：火焰、毒素

状态免疫：力竭、擒抱、麻痹、石化、中毒、倒地、束缚、震慑

感官：黑暗视觉60尺，被动察觉15

语言：——

挑战等级：16（15,000 XP） 熟练加值+5

**浴火重生 Fiery Death and Rebirth。** 当菲涅克斯死亡时，它将炸裂。以其为中心，60半径尺球状区域内的每个生物都必须通过一次 DC 20的敏捷豁免。豁免失败将受到22（4d10）点火焰伤害，豁免成功则只受到半数伤害。迸发的火焰将点燃区域内所有未被着装或携带的可燃物件。

爆炸会摧毁菲涅克斯的身体并在原地留下一颗重五磅的蛋型余烬。任何尝试触碰这枚余烬蛋的生物都将受到21（6d6）点火焰伤害（每轮仅一次）。这颗余烬蛋免疫所有伤害，而在1d6天后，它将孵化出一只新的菲涅克斯。

**火焰形态 Fire Form。** 菲涅克斯可以穿过至小1寸宽的狭缝而无需跻身。

任何生物触碰菲涅克斯、或在菲涅克斯周围5尺范围内以近战攻击命中菲涅克斯时，皆会受到5（1d10）点火焰伤害。此外，菲涅克斯可以进入并停留在敌对生物所占据的空间。当菲涅克斯在一回合内首次进入一个生物的空间时，该生物将受到5（1d10）点火焰伤害。

菲涅克斯也能通过触碰来点燃区域内所有未被着装或携带的可燃物件（不需要动作）。

**飞掠 Flyby。** 菲涅克斯飞行离开敌人的触及范围时不会引发对方的借机攻击。

**照明 Illumination。** 菲涅克斯散发出60尺半径的明亮光照，与之外30尺的微光光照。

**传奇抗性 Legendary Resistance（3/日）。** 菲涅克斯在豁免失败时改为豁免成功。

**攻城怪物 Siege Monster。** 菲涅克斯对物件和建筑物造成双倍伤害。

动作 Actions

**多重攻击 Multiattack。** 菲涅克斯进行两次攻击：一次喙啄和一次焚爪。

**喙啄 Beak。** 近战武器攻击：命中+13，触及15尺，单一目标。命中：15（2d6+8）点火焰伤害。

**焚爪 Fiery Talons。** 近战武器攻击：命中+13，触及15尺，单一目标。伤害：17（2d8+8）点火焰伤害。

传奇动作 Legendary Actions

菲涅克斯拥有3传奇动作，用来选择执行下列动作项。祂每次只可选择执行一个传奇动作项，且必须在另一生物回合结束时执行。菲涅克斯在其回合开始时恢复所有消耗的传奇动作数。

**喙击 Peck。** 菲涅克斯发动一次喙啄攻击。

**移动 Move。** 菲涅克斯移动等同至多等于其速度的距离。

**凤落 Swoop（消耗2动作）。** 菲涅克斯移动至多等同于其速度的距离，并发动一次焚爪攻击。
