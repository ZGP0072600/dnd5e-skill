# 连枷蜗牛

链枷蜗牛 Flail Snail

链枷蜗牛是一种由元素之土构成的生物，以它那多彩而又正规的外壳而著称。它看似无害，实则它们会在足够大的威胁生物靠近时，闪耀出刺眼的光芒，并用锤子一般的触手发起攻击。

在没有干扰的情况下，链枷蜗牛会沿着地面缓缓移动着。它会食用着表面上的一切东西，包括岩石、沙子、与土壤，也会停下来仔细品味水晶簇和其他大型矿产。它会走过的地方残留下一道闪闪发光的残留物。残留物会迅速凝固成一层薄薄的，近乎透明的，对蜗牛来说不可食用的玻璃状物质。这种残留物可以被收集，切割成各样的窗玻璃，也可以被加热并纺成其他玻璃制品——因此，有些居民会尝试尾随链枷蜗牛，收集残留物谋生。

使用链枷蜗牛壳 Using the Shell of a Flail Snail

一枚链枷蜗牛的壳重250磅，且有许多用处。一块完整的壳可以被以5000gp 的价格卖出。

许多猎人为了蜗壳的反魔法能力而寻找它。一个专业的护甲师可以用一枚蜗壳做三面盾牌，每面盾牌都可以基于给持用者链枷蜗牛的 **反魔法蜗壳 Antimagic Shell** 特性。盾牌的魔力可以持续一个月，魔力消失后，留下的独特盾牌也是用于制作 抗法盾 Spellguard Shield 的完美素材。

链枷蜗牛壳也可以用于制作 虹光法袍 Robe of Scintillating。在用于制作法袍时，需要将一枚蜗壳碾成粉末，一部分添加到染料中，一部分作为给法袍附魔的仪式的材料成分。

链枷蜗牛 Flail Snail

*大型元素生物，无阵营*

护甲等级：16（天生护甲）

生命值：52（5d10+25）

速度：10尺

力量17（+3） 敏捷5（-3） 体质20（+5）

智力3（-4） 感知10（+0） 魅力5（-3）

伤害免疫：火焰，毒素

状态免疫：中毒

感官：黑暗视觉60尺，震颤感知60尺，被动察觉10

语言：——

挑战等级：3（700 XP） 熟练加值+2

**反魔法蜗壳 Antimagic Shell。** 蜗牛在对抗法术时所进行的豁免具有优势；任何生物对蜗牛发动法术攻击时，其攻击检定将具有劣势。

当蜗牛对抗法术的豁免成功时，或当一次来自法术的攻击以蜗牛为目标但是没有命中它时，如果那次法术的环阶为1环或更高，则蜗牛的外壳将部分来袭的法术能量转换为一阵破坏性的力量。此时，所有处于蜗牛周围30尺范围内的每个生物都必须通过一次 DC 15的体质豁免，豁免失败将受到该法术环阶每环3（1d6）点力场伤害，豁免成功则只受半数伤害。

动作 Actions

**多重攻击 Multiattack。** 链枷蜗牛发动五次链枷触手攻击。

**链枷触手 Flail Tentacle。** 近战武器攻击：命中+5，触及10尺，单一目标。命中：5（1d4+3）点钝击伤害。

**闪耀之壳 Scintillating Shell（短休或长休后充能）。** 蜗牛的外壳发出耀眼的彩光。外壳将散发出半径30尺的明亮光照，与之外的30尺的微光光照。任何处于此明亮光照范围内，且能看到蜗牛的生物，必须通过一次 DC 15的感知豁免，豁免失败将陷入 **震慑 Stunned** ，持续直到光照消失。光照将一直持续到蜗牛的下一回合结束为止。此外，光照存在期间，任何可以看见蜗牛的生物对蜗牛发动的攻击检定具有劣势。

** 壳体防御 Shell Defense。**链枷蜗牛缩回它的壳中。直到它出壳为止，其 AC 获得+4加值并陷入 ** 束缚 Restrained** 状态。它能在它的回合中以一个附赠动作重新出壳。
