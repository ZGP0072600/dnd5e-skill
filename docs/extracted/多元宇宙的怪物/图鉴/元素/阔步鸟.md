# 阔步鸟

阔步鸟 Giant Strider

这些凶猛而雄壮的怪物同时具备禽类与爬行类生物的特征，但它既不是禽类也不是爬行类生物。阔步鸟对火焰有着超自然的亲和力，且能向着远处的敌人吐出烈焰。他们常常活动在热带、活火山地带、或是有着极端高温与水源的地带。

熔螈们十分中意这种生物，它们会尽可能的寻找并驯养阔步鸟。它们在自己的巢穴中为阔步鸟建起厩房，而作为回报，阔步鸟会十分乐意去服侍这些 熔螈骑士 Firenewt Warrior（记载于本书中）。

阔步鸟 Giant Strider

*大型元素生物，无阵营*

护甲等级：14（天生护甲）

生命值：22（3d10+6）

速度：50尺

力量18（+4） 敏捷13（+1） 体质14（+2）

智力4（-3） 感知12（+1） 魅力6（-2）

伤害免疫：火焰

感官：被动察觉11

语言：——

挑战等级 1（200XP）

**火焰吸收 Fire Absorption。** 每当阔步鸟接收到火焰伤害时，它将不会受到任何伤害，且会恢复对阔步鸟造成火焰伤害数额一半的生命值。

动作 Actions

**啃咬 Bite。** 近战武器攻击：命中+6，触及5尺，单一目标。命中：8（1d8+4）点穿刺伤害。

**火焰爆发 Fire Burst（充能5~6）。** 阔步鸟向它60尺内可见的一点吐出一团烈焰。以该点为中心，10尺半径的球状区域内的每个生物都必须通过一次 DC 12的敏捷豁免，豁免失败将受到14（4d6）点火焰伤害，豁免成功则只受到半数伤害。火焰会扩散并绕过转角，且它会点燃范围内没有被穿戴或携带的可燃物。
