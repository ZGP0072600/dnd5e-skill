# 四臂猿怪

四臂猿怪 Girallon （注：原译作四臂猩猩）

四臂猿怪看起来像一只体型超大的四臂猿，皮肤呈灰褐色，毛发呈白色。它凶狠的獠牙和利爪显示出它是一种可怕的掠食者。

四臂猿怪最常见于温带或温暖的森林环境中。它们与猿类一样善于攀爬，不过很少有树木能承受它们半吨重的体重。城市废墟，尤其是深山老林和丛林中的城市废墟，似乎很吸引四臂猿怪。在它们看来，城市的建筑就像一种高级森林，其最顶端的 
"树枝 "可以安全地支撑它们。这种生物可以轻易地攀爬城墙和城垛，它们栖息在塔顶和其他高处，以监视周围的情况。

在无法攀爬的情况下，四臂猿怪会在森林地面上潜行，潜伏在狭窄的峡谷或浅洞中，或者躲在废墟中等待猎物靠近。考虑到其体型和缺乏伪装，四臂猿怪的隐蔽性令人惊讶。

四臂猿怪由几个个体和它们的后代组成松散的群体，通常由一位占主导地位的成年四臂猿怪领导，这位成年四臂猿怪往往也是群体中最年长的成员。在远离巢穴狩猎时，四臂猿怪使用吼叫和肢体语言进行远距离交流。每个个体通常单独狩猎，并与其他个体分开，以确保每个人都能获得充足的饲料。不过，首领可能会组织成员一起合作，进行一次大的捕杀。如果他们成功了，群体中的每个人都会分享战利品，最好的部分会给那些照顾幼崽的人。

四臂猿怪奇特的外表和对废墟的吸引力让贤者们相信，它们是通过魔法创造出来的，是某个失落帝国的守护者。当那个帝国在很久以前灭亡后，四臂猿怪变成了蛮兽，并散步到世界各地。

无数生物试图驯服、征服这些怪物，或与它们合作。例如，一些居住在森林中的民族捕获四臂猿怪，并将它们训练成哨兵。其他民族认识到四臂猿怪在同类中是和平的，因此他们学会了如何接近四臂猿怪群体的首领，提供食物和其他礼物，希望与四臂猿怪建立联盟关系。

受到善待的四臂猿怪可能愿意充当守卫，不过它们缺乏智慧，无法承担比攻击进入其领地的陌生人外更复杂的任务。年轻时被捕获并经过精心训练的四臂猿怪可能会去一个看似不太可能的地方，比如守卫城市盗贼公会的入口。不过，饲养四臂猿怪的人必须时刻保持警惕，因为这种生物随时都可能恢复捕食本性。

**爪击 Claw。** 近战武器攻击：命中+6,触及10尺,单一目标。命中：7（1d6+4）的挥砍伤害

**附赠动作 Bonus Actions**

** 好斗 Aggressive。**四臂猿怪向一个它能看见的敌对生物移动至多等同于其速度的距离。
