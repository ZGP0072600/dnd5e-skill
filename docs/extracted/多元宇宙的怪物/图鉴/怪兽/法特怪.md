# 法特怪

**法特怪 Xvarts**

法特怪是一种懦弱、贪婪的生物，由一位叛逆的半神 **莱兹爱渥 Raxivort** 
所产生。他们有着蓝色的皮肤，橙色的眼睛，后退的发际线，反映了他们的创造者的外表。它们大约有3英尺高。

莱兹爱渥花了几个世纪的时间来看守 **格拉兹特 Graz'zt** 的宝库，最终，莱兹爱渥洗劫了他领主的金库。他偷走的宝物之一是无限之轴，这是一块水晶碎片，甚至可以把像莱兹爱渥这样卑微的生物变成半神。在他被神化之后，莱兹爱渥建造了黑色下水道，这是一个混乱的领域，他在里面放满了他心爱的动物，老鼠和蝙蝠，直到今天，他们都是他的朋友。在格拉兹特发动复仇之前，他的统治只是短暂的。恶魔王子号召世界各地的恶棍去为自己去追寻无限之轴，并摧毁莱兹爱渥。
