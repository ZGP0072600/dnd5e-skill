# 骑蛛

**骑蛛 Steeders**

巨型的狩猎蛛—骑蛛会在黑暗的深处徘徊。雌性骑蛛比雄性骑蛛长得更大更壮，交配后，雌性骑蛛常常会吃掉雄性骑蛛。许多幽暗地域的居民驯养了骑蛛，尤其是灰矮人（也在此书中）。一般来说，雌性在战斗中充当坐骑，而雄性则被用作驮运。

骑蛛会试图撕裂感知到的所有威胁，甚至把其他骑蛛视为敌人。当它们被投入工作时，它们的饲养员必须把它们分开，并给它们戴上眼罩，以防止它们互相残杀。

骑蛛足够聪明，可以学习简单的手势和口头指令，但即使是驯化的骑蛛也可能会反抗它的主人。训练骑蛛需要骑手与骑蛛建立良好的关系，这个过程在骑蛛孵化后不久就得开始。随着它的成长，它的骑手为指导它的捕食本能，需与它一同进行工作。

骑蛛不会结网，而是从腿上分泌一种粘性物质。这种粘液使它们能够沿着墙壁和天花板爬行，并捕获猎物。
