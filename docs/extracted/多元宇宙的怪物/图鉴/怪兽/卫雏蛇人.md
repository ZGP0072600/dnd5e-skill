# 卫雏蛇人

**卫雏蛇人 Yuan-Ti Broodguard**

卫雏蛇人曾是类人生物——现在却已经被蛇人们转化成了的头脑简单、通体复鳞的怪物，并按照其蛇形主人的命令行事。这一转变过程不仅改造了这些受害者的肉体，还异化了它们的思维，使其本能地服从任何蛇人，并会对非爬行类生物产生汹涌的狂怒。

虽然卫雏蛇人智力低下，但他们能够在领地中执行简单却必要的任务，如看守蛇卵或巡逻以防范入侵者。其他蛇人们称其为“histachii”，意为“蛇卵守卫”。

大多数卫雏蛇人皆是由人类囚犯转化而来——它们会被迫饮下一种魔法药剂，这会剥夺它们的反抗能力，并让受害者在无助中永久转化为怪物。变成卫雏蛇人的人类不仅失去了曾经的外表，就连过去曾为人类的痕迹也几乎无法辨认。卫雏蛇人通体无毛、瘦弱、体表复鳞。它们有着分叉的舌头，还带有一股腐肉的味道。卫雏蛇人能够说话，但很少这样做——它们更喜欢用蛇般的嘶声和喉音交流。

**创造卫雏蛇人 Making a Broodguard**

蛇人会使用捕获来的类人生物制造卫雏蛇人。每个受害者都会被喂食一种特殊的药剂，使其立即陷入失能状态，并在接下来的第1d6+6天后变成一名卫雏蛇人 Yuan-Ti Broodguard。一个非自愿摄入药剂的生物可以进行一次 DC 15的体质豁免；豁免成功则只受到14（4d6）毒素伤害且不受其他效果的影响。

一道法术次等复原术或移除诅咒可以在转化过程完成前中止它。而在过程完成后，便只有法术祈愿术才能解除转变。

**卫雏蛇人 Yuan-Ti Broodguard**

*中型怪兽，通常中立邪恶*

 护甲等级：14（天生护甲）

 生命值：45（7d8+14）

速度：30尺

力量15（+2）敏捷14（+2）体质14（+2）

智力6（-2）感知11（+0）魅力4（-3）

豁免：力量+4，敏捷+4，感知+2

技能：察觉+2

伤害免疫：毒素

状态免疫：魅惑，麻痹，中毒

感官：黑暗视觉60尺，被动察觉12

语言：深渊语，通用语，龙语

挑战等级2（450XP）熟练加值+2

**鲁莽 Reckless。** 在它的回合开始时，卫雏蛇人可以令其在本回合内的所有近战武器攻击中获得优势；但在下一回合开始之前，对它的攻击也会有优势。

动作 Actions

**多重攻击 Multiattack。** 卫雏蛇人发动一次啃咬攻击和两次爪击攻击。

**啃咬 Bite。** 近战武器攻击：命中+4，触及5尺，单一目标。命中：6（1D8+2）穿刺伤害。

**爪击 Claw。** 近战武器攻击：命中+4，触及5尺，单一目标。命中：5（1D6+2）挥砍伤害。
