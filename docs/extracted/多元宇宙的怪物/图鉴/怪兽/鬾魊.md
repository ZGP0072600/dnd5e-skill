# 鬾魊

**鬾魊 Meazel**（注：曾音译作“米泽尔”，现采用博德之门3的译名）

鬾魊是一群恶毒的隐士，他们逃到堕影冥界以规避他们的肉体凡胎，思考他们的苦难。在那里，暗影改变了他们，他们的痛苦使他们扭曲和残忍。现在仇恨在他们的心中燃烧，他们憎恨任何侵入他们的痛苦，伏击那些敢于太过接近他们巢穴的旅行者。

鬾魊的邪恶腐化赋予了它们魔法般的力量，使它们可以轻松地穿过阴影。它们可以从一个黑暗之池进入另一个黑暗之池，并利用这个天赋伏击猎物。有时它们会用绳索缠住受害者的喉咙然后走开；其他时候，它们把受害者送到孤立无援的地方，然后把这些不幸的灵魂留给潜伏在那里的恐怖存在。

任何从阴影中穿过的生物都会被鬾魊的邪恶魔法诅咒。这种诅咒就像灯塔；**哀恸魔 Sorrowsworn**（出现在本书中）、亡灵、和其他恐怖的存在会意识到他们的位置，并撕裂那些深陷其中的受害者们。
