# 青年地刺虫

**青年地刺虫 Young Kruthik**

地刺虫是由雌性成虫产下的卵经过一个月的孵化期孵化而来的生物。它们的每颗卵大小与成年人的头相仿。这些小小的地刺虫幼崽并不能造成威胁，它们也几乎不会主动远离巢穴，其食物主要以动物内脏和彼此为食。 
再经过一个月之后，这些幸存下来的幼崽将成长为青年地刺虫，有着足够大的体型，得以捕猎并保护自己。

**集群战术 Pack Tactics。** 如果地刺虫攻击的目标周围5尺范围内有至少一个地刺虫的未失能盟友，则地刺虫对该生物发起的攻击具有优势。

**掘道者 Tunneler。** 地刺虫能以一半的掘穴速度在坚硬的岩石中挖掘，并留下一条直径2.5尺的隧道。

**动作 Actions**

** 猛刺 Stab。**近战武器攻击：命中+5，触及5尺，单一目标。命中：5（1d4+3）点穿刺伤害。
