# 蛇人深坑之主

蛇人深坑之主 Yuan-Ti Pit Master（注：原译作原体·坑主）

生着蛇形臂膀的深坑之主是与蛇神麦尔绍克 Merrshaulk 缔结契约的恶咒蛇人祭司。它们向神祇献祭类人生物，以求能将祂唤醒。它们在态度上是蛇人中最传统的，并且认为自己是最可能实现族群大业的人选。

深坑之主们深入参与了蛇人们介入类人生物政府的长期计划，并致力于保护其城市不被敌人发现或攻击。它们反对鲁莽地行事，主张在所有事情上都采取缓慢、谨慎的态度。

**蛇人深坑之主 Yuan-Ti Pit Master**

*中型怪兽（魔契师），通常中立邪恶*

 护甲等级：14（天生护甲）

 生命值：88（16d8+16）

速度：30尺

力量16（+3）敏捷14（+2）体质13（+1）

智力14（+2）感知12（+1）魅力16（+3）

豁免：感知+4，魅力+6

技能：欺瞒+6，隐匿+5

伤害免疫：毒素

状态免疫：中毒

感官：黑暗视觉120尺，被动察觉11

语言：深渊语，通用语，龙语

挑战等级5（1,800XP）熟练加值+3

**魔鬼视界 Devil's Sight。** 魔法黑暗不会阻碍蛇人的黑暗视觉。

**魔法抗性 Magic Resistance。** 蛇人为抵抗法术和其他魔法效应而作的豁免检定具有优势。

动作 Actions

**多重攻击 Multiattack。** 蛇人发动三次啃咬或两次灵体毒牙攻击。

**啃咬 Bite。** 近战武器攻击：命中+6，触及5尺，单一目标。命中：5（1d4+3）穿刺伤害外加7（2d6）毒素伤害。

**灵体毒牙 Spectral Fangs。** 远程法术攻击：命中+6，射程120尺，单一目标。命中：16（3d8+3）毒素伤害。

**麦尔绍克之沉眠 Merrshaulk's Slumber（1/日）。** 蛇人指定至多5个位于其60尺内的它可见的生物。目标必须成功通过一次 DC 13体质豁免检定否则将进入魔法睡眠并昏迷10分钟。如果一个沉睡的目标受到伤害，或是有人用一个动作来摇晃或拍打它，目标就会醒来。这种魔法睡眠对免疫魅惑的生物没有影响。

**施法 Spellcasting（仅蛇人形态）。** 蛇人施展以下一道法术，无需任何材料成分并使用魅力作为施法属性（法术豁免 DC 14）：

随意：化兽为友（仅蛇类），神导术，法师之手，传讯术
3次/日：暗示术
每项2/日：定身类人，隐形术

附赠动作 Bonus Actions

**变形 Change Shape。** 蛇人变为一条中型蛇，或是变回其真实形态。它的数据在每种形态都是相同的。任何它所携带的装备都不会随之变形。
