# 悲惨之哀恸魔

**悲惨之哀恸魔 Wretched Sorrowsworn**

悲惨，这种骇人的小怪兽，成群结队在堕影冥界中狩猎。此类可怜的存在依靠生命力而生，所以当它们找到生物时，便会涌上前去将其尖牙塞进受害者体内，啜饮其生命能量和恐惧。

**惨不忍睹的集群战术 Wretched Pack Tactics。** 如果至少有一个悲惨的盟友在目标生物的5尺以内，并且该盟友没有失能，则悲惨在攻击该目标生物时具有优势。否则悲惨的攻击具有劣势。

**动作 Actions**

** 啃咬 Bite。**近战武器攻击：命中+3，触及5尺，单一目标。命中：6（1d10+1）穿刺伤害，并依附于目标。依附于目标时，它无法攻击，在悲惨的每回合开始时，被依附的目标受到6（1d10+1）暗蚀伤害。

当被依附的目标移动时，悲惨随之移动并且无需为此花费移动力。在悲惨的回合内，它可以花费自身5尺的移动力来使自身脱离依附的目标。目标生物可以使用一个动作来摆脱悲惨的依附。
