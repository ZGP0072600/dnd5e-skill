# 沼喉怪

沼喉怪 
Froghemoth

沼喉怪是一种像大象一样大的两栖捕食者。它栖息在沼泽中，有四条触手，厚实的胶质皮肤，带着一条能抓物的舌头的充满尖牙的口器，及一根可伸展的肉茎上朝向不同方向运动的三个眼球。

 沼喉怪是不属于这个世界的生物。一本据说是很久以前由法师 卢姆 Lum 写的日记里，描述了埋藏在地下的奇怪柱形金属房间。沼喉怪从那里出现，但是没有关于这些地方位置的可靠报告。

每隔几年，一头沼喉怪就可以在不交配的情况下产下一颗受精卵。沼喉怪不关心它的卵，也可能把它吃掉。一头年幼的沼喉怪的生存通常是建立在它的父母在冷漠中离弃了它。一头新生的沼喉怪通过不加选择的捕食它沼泽领地里的其他生物在一个月时间里成长为完全的大小。他学会了把它巨大的躯体隐藏在浑浊的水塘里，只把它的眼睛露在水面上以观察路过的生物。当食物触手可及时，沼喉怪就会从它的水塘里射出触须和舌鞭。它可以同时攫取多个目标，使它们陷入绝境，在它用舌头卷起另一个目标并把它拉过来吞下时。

如果一个啵灵蛙部落遇到了一头沼喉怪，啵灵蛙会把沼喉怪当做神明看待，竭尽所能把这怪物哄诱进他们的巢穴。啵灵蛙通过提供食物来（勉强）驯服这些沼喉怪，且可以与它进行基本的沟通。所以绝大部分的这种生物往往很少捕食这些啵灵蛙。啵灵蛙收集食物作为给它的贡品，为它提供一个舒适的巢穴，狂热的保护它免受伤害，并努力确保任何年幼的沼喉怪都能发育成熟。

**水陆两栖 Amphibious。** 沼喉怪可以在空气和水中呼吸。

**震电敏感 Shock Susceptibility。** 如果沼喉怪受到闪电伤害，直到它的下一个回合结束它遭受以下两种减益：它的速度减半, 且在敏捷豁免上具有劣势。

动作 Actions

**多重攻击 Multiattack。** 沼喉怪发动一次啃咬攻击和两次触须攻击。它还使用舌卷。

**啃咬 Bite。** 近战武器攻击：命中+10，触及5尺，单一目标。命中：22（3d10+6）点穿刺伤害，如果目标是一个中型或更小的生物则被吞下。一个被吞下的目标陷入目盲且被束缚，其对于沼喉怪身体外的攻击或效应视为处于全掩护，并在每次该沼喉怪的回合开始时受到10（3d6）的强酸伤害。沼喉怪的食道一次可以容纳两个生物。

如果沼喉怪在同一回合受到20点或更多来自其体内的伤害，则该沼喉怪在该回合结束时必须成功通过一次 DC 20的体质豁免否则将吐出所有被吞下的生物，每个都摔下倒地于沼喉怪周边10尺范围内。如果该沼喉怪死亡，则被吞下的生物将不再受束缚，且可以消耗10尺移动力逃出，然后在尸体旁现身并陷入倒地状态。

**触须 Tentacle。** 近战武器攻击：命中+10，触及20尺，单一目标。命中：19（3d8+6）点钝击伤害,如果目标是巨型或更小的生物则被擒抱（逃脱 DC 16）。直到擒抱结束，沼喉怪不能对另一个目标使用这条触手。沼喉怪有四条触手。

**舌卷 Tongue。** 沼喉怪选择一个20尺内它能看见的中型或更小的生物。目标必须成功通过一次 DC 18的力量豁免。豁免失败则目标被拉至沼喉怪5尺内一处未被占据的空间。
