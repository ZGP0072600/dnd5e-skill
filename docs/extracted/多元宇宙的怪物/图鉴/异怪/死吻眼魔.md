# 死吻眼魔

**死吻眼魔 Death Kiss**

死吻眼魔是从眼魔关于失血的清晰噩梦中产生的低阶眼魔。其外观颜色与梦见它的本体眼魔相似，只是颜色更为暗淡。它没有眼波射线，取而代之的是十根末端长着尖牙大嘴的长触须。这些触须上的大嘴还可以开口讲话，声调如鼻音般低沉。

死吻眼魔惧怕真眼魔，因为真眼魔可以轻易将其杀死或压制。死吻眼魔通常都会服从创造它的眼魔或是所遭遇眼魔的支配，然后尽可能的在其主人分神时逃跑。

死吻眼魔靠吸食血液维持生存，其所吸收的血液会在身体里转化成电能供其利用。由于偏执般的惧怕饿死，死吻眼魔饥不择食，连老鼠一样的小生物都会被着魔般的吸尽血液。留下一具具枯干的尸体。死吻眼魔在地底时会使用触须探查周围的环境，而在地面时则通常将触须缩回身体，然后在敌人疏忽时瞬间将触须完全伸长以捕获对方。

死吻眼魔没有眼魔那样的战斗技巧和高智商。大多数情况下，它只会尽量将触须贴上猎物，然后直接吸血至猎物倒下。当自身占据优势而对手又不构成威胁时，它就会肆意玩弄这些到口的食粮，慢慢的将该生物的生命挤干吸净。而若是遇上了同类，根据健康状况和自尊程度，死吻眼魔便可能交战、遁走或是干脆结伴。

**死吻眼魔 Death Kiss**

*大型异怪（眼魔），普遍中立邪恶*

 护甲等级：15（天生护甲）

 生命值：142（12d10+60）

速度：0尺，飞行30尺（悬浮）

力量18（+4）敏捷14（+2）体质18（+4）

智力10（+0）感知12（+1）魅力10（+0）

豁免：体质+8，感知+5

技能：察觉+5

伤害免疫：闪电

状态免疫：倒地

感官：黑暗视觉120尺，被动察觉15

语言：深潜语，地底通用语

挑战：10（5900XP）熟练加值+4

**闪电之血 Lightning Blood。** 身处死吻眼魔身边5尺范围内的生物，使用近战攻击对其造成穿刺或挥砍伤害时，该生物将受到5（1d10）的闪电伤害。

动作 Actions

**多重攻击 Multiattack。** 死吻眼魔进行三次触须攻击。其中至多三次攻击可以替换为吸血攻击，所替换的每次攻击必须对应有一根正擒抱某生物的触须。

**触须 Tentacle。** 近战武器攻击：命中+8，触及20尺，单一目标 。命中：11（2d6+4）穿刺伤害，若目标为巨型或更小的生物，则其还会被擒抱（逃脱 DC 14）并陷入束缚状态直至擒抱结束， 此时死吻眼魔无法使用同一根触须擒抱另一个生物。死吻眼魔共拥有十根触须。

**吸血 Blood Drain。** 一个正受死吻眼魔触须擒抱的生物必须进行一次 DC 16的体质豁免。豁免失败则受22（4d10）的闪电伤害，同时死吻眼魔恢复该伤害值一半的 HP。
