# 孢子人个体

孢子人 Vegepygmy

孢子人起源于被赤霉杀死类人生物或巨人留下的遗骸。一天后的遗骸之上，一个或多个孢子人将诞生于此。

孢子人 Vegepygmy

*小型植物，普遍绝对中立*

护甲等级：13 （天生护甲）

生命值：13 （3d6+3）

速度：30尺

力量7（-2） 敏捷14（+2） 体质13（+1）

智力6（-2） 感知11（+0） 魅力7（-2）

技能：察觉+2、隐匿+4

伤害抗性：闪电、穿刺

感官：黑暗视觉60尺，被动察觉12

语言：孢子人语

挑战等级：1/4（50XP） 熟练加值+2

**植物伪装 Plant Camouflage。** 若孢子人身处有充足遮挡植物的地形上，所进行的敏捷（隐匿）检定具有优势。

**再生 Regeneration。** 孢子人在自己回合开始时恢复3生命值。如果它受到了寒冷、火焰、或暗蚀伤害，这以特性在孢子人下一回合开始时无效。只有当孢子人再生能力失效且以0生命值开始其回合时，它才会死亡。

动作 Actions

**爪击 Claws。** 近战武器攻击，+4 命中，触及5尺，单一目标。命中：5（1d6+2）挥砍伤害。

**投石索 Sling。** 远程武器攻击：命中+4，射程 30/120尺，单一目标.。命中：4（1d4+2）钝击伤害。
