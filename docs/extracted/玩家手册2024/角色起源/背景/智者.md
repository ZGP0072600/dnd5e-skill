# 智者

## 智者 Sage

属性值：体质、智力、感知

专长：魔法学徒（法师）（见第五章）

技能熟练：奥秘和历史

工具熟练：书法工具

装备：选择 A 或 B：(A)长棍、书法工具、书籍（历史）、羊皮纸（8张）、长袍、8GP；或(B)50GP

你长大的岁月便在宅邸和修行院之间的奔波中过去，做着各式各样的零活和服务，以此访问他们的图书馆藏。你花了那么多漫长的夜晚学习书籍和卷册，阅览多元宇宙的见闻，甚至涉猎魔法的本质——而你的心灵正渴望着了解更多。
