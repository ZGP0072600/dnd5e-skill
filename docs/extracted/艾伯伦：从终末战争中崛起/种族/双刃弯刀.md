# 双刃弯刀

**双刃弯刀 Double-Bladed Scimitar**

双刃弯刀是维伦那精灵的标志性武器。一根细木棒支撑著两端弯曲的长刃。经过几个世纪的技术打磨，这些刀刃坚固、锋利、非常轻。每一把弯刀都是一件杰作，因此双刃弯刀是一件昂贵的武器（100gp）——尽管很少有人有机会买一把。非精灵手中的维伦那之刃通常被认为是从倒下的敌人那里偷来或掠夺来的，而维伦那之刃的持有者可能会觉得自己有权要求归还或挑战持刃者，以证明自己有能力使用它。

双刃弯刀是一种军用武器，重 6 磅，攻击命中造成 2d4 挥砍伤害。 具有双手属性与以下特殊属性:

- 如果你在你的回合的攻击动作中使用了双刃弯刀发动攻击，你可以紧接着使用一个附赠动作来再发动一次近战攻击。此攻击命中时造成1d4挥砍伤害，而非2d4。

**专长：英魂之刃 Revenant Blade** 

*先决条件: 精灵*

你是一个双刃弯刀大师的后代，而这种能力的一部分已经传给了你。你获得下列增益：

· 你的力量或敏捷增加1，上限为20。

· 当你双手持握双刃弯刀时，你的 AC 得到+1加值。

· 在使用双刃弯刀时视同具有灵巧属性。
