# 种族

龙裔 Dragonborn

数万年前，阿贡尼森巨龙在现在的夸巴拉建立了一个龙裔驻军。 这些战士被派去保护这个地区不受砂主 Lords

这个龙裔国家在一个古老邪魔霸主

到目前为止，这些龙裔基本上忽略了夸巴拉的人类，而且那些碰巧接触到龙裔的人相信他们是某种外来的蜥蜴人。如果你是龙裔的，是什麽让你从夸巴拉中出现?你是在寻求帮助你的人民还是在对抗砂主?你是被旅行的欲望还是好奇心驱使?在终末战争中你当过雇佣兵吗?你的旅行如何与你的人民的命运联系在一起?

矮人 Dwarves
金子是山脉的赠礼，但铁是血和骨头化成的。
-Mror Proverb

矮人的起源一直是个谜。他们的一些传说讲述了一场伟大的移民，导致古代矮人从 “一个永无止尽的冰天雪地” 来到了科瓦雷。许多人认为这是指北极地区的霜落之地 Frostfell，而另一些人则认为第一批矮人一定来自于霜冻的瑞西亚 Risia 位面。 无论他们的根在哪里，这些移民矮人在科瓦雷的表面下建立了一个强大的国家。

大多数矮人传说都不关心起源的问题，只专注于远古居住在深处的矮人制造的强大的神器和无价宝藏，以及他们对达坎地精帝国的血腥战争。今天的矮人珍视这个古老民族的记忆，因为在科瓦雷的所有矮人都是被赶出地下国度的流亡者后裔 -- 而这个王国后来被异变魔摧毁了。特别是在摩洛领中，许多矮人都有回到地下世界的想法，重建他们古老的国家，恢复他们几乎已经遗忘的伟业。
