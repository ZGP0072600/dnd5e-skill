# 魔法物品

**魔法物品 Magic Items**

艾伯伦世界中轰鸣着魔法之力。奇械师 Artificer、魔匠师 magewrights（详见艾伯伦怪物手册部分，他们是使用法术完成日常工作的厨师、园丁等）、法师以及其他施法者们生产了海量的魔法物品，尤其是那些珍稀度为普通的。

**魔法物品详述 Magic Item Descriptions**

本节按字母顺序列出了一部分魔法物品。有关使用魔法物品的规则，请参阅《城主指南》。

本章中的一些魔法物品是由异变魔（daelkyr）创造的共生体（symbiotic 
object）。虽然没有完整的意识，但这些共生体一定程度上算是某种器官。和共生体同调意味着你在物理上与它连接在一起。虽然这些东西并不邪恶，但有些人看到你穿着活的盔甲或是使用触手当作鞭子很可能会感到不安！
