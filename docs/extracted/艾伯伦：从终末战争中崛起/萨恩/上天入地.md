# 上天入地

第九节 上天入地萨恩的五大主要城区构成了这座城市的主体。但是还有一些区域位于这座城市的天上或地下，这也包括了大多数人都不愿意去的奇妙或危险之地。 天城区萨恩上层层区有许多悬浮塔楼，但是与天城区相比就显得相形见绌了。天城区是悬浮在中央高地和孟西斯高地之间的一片区域。建立在固化的云岛上，天城区拥有科瓦雷最富有的人的豪宅，他们的脚甚至不用触及地面。这是一个奇妙的领域，充满了所有钱可以买到的最好的商品、奢侈品还有魔法造物。云池公园是由固化的云雕塑而成，通过窗户上的冰块可以俯瞰地上的城市。天城区是豪宅区。在终末战争之前，五国贵族都在这里拥有不少豪宅。尽管其中一些仍然在袁主任手中，但其他的已经在终末战争期间被布鲁兰王室没收，转售给了富裕的布鲁兰公民。居民的涌入也给这个地区带来了更多的特色，龙纹贵胄，乃至于富有的罪犯都和贵族们生活在一起。天城区最宏伟的庄园是泰恩庄园，也是泰恩家族每月举办泰恩酒会的地方（详见本章前文中的“节日”部分）。这里提供整座城市最昂贵的服务。龙窖是一个伽兰达家族经营的旅游胜地，在那里旅行者可以享受最奢华的服务，只要他们负担得起最昂贵的价格。天景和云龙是顶级的餐厅。蔚蓝天门是一家新的酒店和餐厅，取代了伽兰达家族的老店。店主的身份没有透露，这种神秘感激起了不少人对这里的兴趣。如果你有兴趣和名人贵胄打交道，或者你想抢劫萨恩最富有的人，那天城区当然是个好选择。没有任何桥梁或电梯连接天城区和城市的其他部分，所以通常来说，前往天城区的方法是乘坐空中客车或是租用飞行坐骑。 崖边区崖边区包含了萨恩的码头和海滨部分，沿着匕首河的边缘一直延伸到悬崖上。巨大的升降机将货物运送到摇摆区的空港。这个地区贫穷混乱，到处都是水手和码头工人。许多喧闹的人喜欢崖边区的生活，就像其当地俗话一样“崖边区的一大杯麦芽酒，胜过中央高地的葡萄酒”。在这里你可以找到许多合法的服务——船舶所需的补给和维护，货物的储存，船员的雇佣，就算是海员们也需要找一张干燥的床好好睡一觉——但还有其他服务，满足各种不同的需求：妓院、酒馆、赌场，等等为水手们提供的娱乐场所。尽管萨恩看守们象征性的存在于这里，但崖边区仍然是城市里犯罪活动最猖獗的地区之一。鲨华鱼人的存在加剧了这种情况，他们以向导服务换取报酬，帮助前往希恩德瑞克的船只。事实上，鲨华鱼人很少恶意攻击他人，但是他们的好斗的天性导致和醉酒水手之间经常发生冲突。 齿轮区齿轮区位于萨恩地下深处，还在下水道和萨恩古城（Old 
Sharn，本章后文会有详述）之下。费尼亚的熔岩河流经此地，几个世纪以来，卡尼斯家族帮助布鲁兰在这里建立了大型铸造厂，用来开发这种魔法资源。灰黑区和黑骨区是工业区，主要居住着战俑和其他种族的可怜工人。据说卡尼斯家族有一个据点，弗格霍德，位于灰黑区，这里是马利克斯·D=卡尼斯男爵进行实验的地方。萨恩看守和卡尼斯家族的监工驻扎在这里，负责维持秩序。在这些工业区之外，深邃的隧道连接着凯博尔之门区，一个廉价的住宅区。黑骨区有许多不同寻常的地方。比如昂纳塔之泪池，被认为是天命锻造之神的神迹。现任的看守者是一名名叫史密斯（Smith，铁匠）的战俑，他相信昂纳塔已经为战俑指明了前路，战俑要与血肉生物共同生存。萨恩的绝大部分使命从来没有去过齿轮区。这个地区确实是这个城市中大部分战俑的居住地，如果你要找某个特定的战俑，最好从这里找起。金融间谍和劳工斗争可以为巨大铸造厂中的冒险增加戏剧性，更不用说在齿轮区中隐藏的秘密工坊。 表3-16

简介蔚蓝天门
天城区这家豪华的酒店最近取代了伽兰达家族经营的一家老店。其菜单是“伽利法融合”风格的，融合了五国的菜系。 天景
天城区这家传奇餐厅的老板是天城区的城市议员，艾威克斯·伊尔=玛拉莎女士。地板由水晶板嵌成，食客们可以随意俯瞰下面的城市。食物并不是很好吃，但其景色让这里宾客如云。 云龙
天城区这家店供应传统的布雷什菜肴，而且做得特别好。这家店不像天城区的其他场馆那样爆满，但当地人知道它的价值。 龙窖
天城区这个度假村由伽兰达家族经营，为那些能负担的起昂贵价格人提供一切奢侈享受。 泰恩庄园
天城区萨恩的社交网络围绕着伊尔=泰恩家族展开。在这里每月一次的泰恩晚会是上流社会的入场券。 泥浆洞穴
崖边区这个棚户区坐落在崖边区的边缘，几代穷苦的人们在崖壁上建造了自己的家园。这里的生活悲惨，但大多数居民没得选择。这里也是逃犯的绝佳藏身之处。 萨恩的欢迎
崖边区也被称为崖边区码头，这个地方又脏又乱。他的酒馆经常爆发争吵，而旅馆也容易产生各种各样的欺骗和抢劫。 船塔
崖边区沿着崖边区码头漂流的一系列木筏、船只和游艇，他们绑在一起形成了一个漂浮的社区。 凯尔多兰铸造厂
灰黑区虽然齿轮区中的大部分铸造厂都致力于大规模生产，但矮人安托斯·凯尔多兰经营着一家小型铸造厂，并生产定制的盔甲和武器。 昂纳塔之池
黑骨区在黑骨区的一个小型神殿里有一个水池，据说收到昂纳塔的祝福。科瓦雷各地的铁匠都来到这里朝圣，把他们的工具浸入水池。 红锤
黑骨区红锤是萨恩唯一由战俑经营的旅馆。它为那些在齿轮区工作的战俑提供了一个社交场所，同时也提供了修理和养护服务。
