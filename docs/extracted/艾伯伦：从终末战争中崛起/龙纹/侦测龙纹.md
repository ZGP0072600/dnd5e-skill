# 侦测龙纹 The Mark of Detection

> “如果你想要有一道肌肉墙为你挡住刀锋，那就去找邓奈斯家族。如果你想要有人预见到威胁，确保你和那个拿刀的人不会同时在房间里，那就来找我们。”
>
> ——崔利普·D·梅丹尼男爵

侦测龙纹是调查员梦寐以求的东西。它会提升观察能力和直觉，让持有者找出别人可能会忽略的潜在联系和线索。通过积极利用它的力量，持有者可以侦测毒素和研究魔法的能量。

# 梅丹尼家族 House Medani

领袖：

总部：

梅丹尼家族的预警行会经营保镖和调查业务。梅丹尼顾问专门研究风险的评估和管理，能够保护顾客免受人身和社会性伤害。虽然梅丹尼家族的业务和萨拉释克调查员及邓奈斯保镖有所重叠，但比起前两者，预警行会专攻于对付潜藏的威胁和复杂的谜团。崔利普男爵负责在沃奥特的裁判塔里管理行会事务，在那里，家族还会为布鲁兰的勃兰奈尔国王审问囚犯。

梅丹尼家族的成员是半精灵，在布鲁兰有着深厚的根基。梅丹尼对其他龙纹家族之间时有发生的争权夺利没什么兴趣。许多梅丹尼后裔比起利润更关注于帮助自己所在的社区，梅丹尼后裔经常会和当场执法部门进行合作或帮助那些无力支付服务费的人。

# 半精灵变体：侦测龙纹 Variant Half-Elf: Mark of Detection

如果你的半精灵角色持有侦测龙纹，下列特性将替代《玩家手册》中列出的半精灵属性值增加、多才多艺。

属性值增加。你的感知属性增加2。再选择其他一项属性增加1。

推理直觉。当你进行智力（调查）或感知（洞悉）检定时，你能够掷一个 d4，将结果加到你的属性检定结果中。

魔法侦测。你能够以此特性施展侦测魔法 Detect Magic 和侦测毒性和疾病 Detect Poison and Disease 法术，从3级起，你还能通过这一特性施展识破隐形 See Invisibility。一旦你通过此特性施展了前述法术，在完成一次长休前，你无法通过此特性再次施展那个法术。对你而言，这些法术的施法属性是感知，且你无需为这些法术提供材料成分。

龙纹法术。如果你拥有施法或契约魔法职业能力，侦测龙纹表中的法术将加入你施法职业的法术列表中。

| 法术环阶 | 法术 |
|---|---|
| 1 | 侦测善恶 Detect Evil and Good、侦测毒性和疾病 Detect Poison and Disease |
| 2 | 侦测思想 Detect Thoughts、寻找陷阱 Find Traps |
| 3 | 鹰眼术 Clairvoyance 、回避侦测 Nondetection |
| 4 | 秘法眼 Arcane Eye、预言术 Divination |
| 5 | 通晓传奇 Legend Lore |
