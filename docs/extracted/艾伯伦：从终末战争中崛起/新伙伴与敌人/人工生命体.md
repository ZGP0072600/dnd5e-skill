# 人工生命体

**人工生命体 Homunculi**

人工生命体是为某些任务而创建的一个构装生物仆人。现存的人工生命体大多是由奇械师和法师创造的。

每一种人工生命体都有一个由不同材料构成的身体，包括粘土、铁、毛发和羽毛。在创造人工生命体 homunculus 的过程中，这些材料与造物主的血液混合在一起，并通过一个漫长的魔法仪式来激活。

**构装本质 Constructed Nature。** 人工生命体不需要空气、食物、饮水或睡眠。

（注：文中使用的是 Homunculus，标题的 Homunculi 为复数形式。此外 Homunculus 在《怪物图鉴》中有出现过。）
