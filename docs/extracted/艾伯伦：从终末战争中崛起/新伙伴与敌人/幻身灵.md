# 幻身灵 Changeling

幻身灵是一群人形的变形者，他们用假面掩盖自己的真实身份。他们的模仿天赋使他们能够以任何类人文化的成员的身份出现，今天扮演侏儒，明天扮演龙族。尽管换领者可以采取任何形式的伪装，但大多数都依赖于少数已经确立的角色，每个角色都有自己的发展历史，都有自己的朋友和熟人网络。
