# 螯爪梦灵

**螯爪梦灵 Tsucora Quori**

（3R 译文为：螯爪灵）

螯爪梦灵是一种可怕的生物。它们的无头躯体上布满了眼睛和抽搐的四肢，包括两条粗壮的胳膊，末端是有力的钳子，还有一条蛇形的尾巴，末端是一根恶毒的毒刺。他们残忍而精于算计，享受着自己对他人施加的暴力，他们精心策划阴谋，以提升自己的地位，败坏竞争对手的名声。

无情的阴谋家们 Relentless 
Schemers。每一个螯爪梦灵都希望能转世投生后成为黑暗之梦中更强大的仆人。因此，他们的计划往往集中在摧毁竞争对手，而在 il-Lashtavar 计划的推进上也是如此。当他们不在噩梦国度的大城市中工作时，他们就会猎捕睡梦中凡人的精神能量。

恐惧催眠。他们有能力控制凡人的恐惧，螯爪梦灵常常通过梦启者 Inspired 进入艾伯伦。没有什么比恐惧更能使类人的奴隶保持井然有序的了，螯爪梦灵是操纵暴徒和迷惑高官的大师。螯爪梦灵寄宿的梦启者的有效性深受百目梦灵 Kalaraq 
Quori 的青睐。

14)。它可以天生施展以下法术，不需要任何相应的法术成分：

*随意:魅惑类人 charm person 每项1 /天:恐惧术 fear*

**动作多重攻击 Multiattack。** 梦灵会进行三次攻击:一次钳击攻击，一次爪击攻击，一次钉刺攻击。

**钳击 Pincer。** 近战武器攻击:命中+6，触及5尺，单一目标。命中:8 
(1d10 + 3)钝击伤害。如果目标是生物，且体型不超过大型，目标会被擒抱(逃脱 DC 
14)。梦灵有两个钳子，每个钳子可以抓住一个目标。

**爪击 Claws。** 近战武器攻击:命中+6，触及5尺，单一目标。命中:13 
(4d4 + 3)挥砍伤害。

**钉刺 Stinger。** 近战武器攻击:命中+6，触及10尺，单一目标。命中:8 (1d10 + 
3)穿刺伤害+ 10 (3d6)心灵伤害，目标必须通过一次 DC 
14感知豁免检定，否则被梦灵恐慌一分钟。目标可以在每次回合结束时重复豁免，成功豁免便结束影响。

**附身 Possession(充能6)。** 梦灵5尺内的一个它可见的类人生物必须进行一次 DC 14魅力豁免检定，失败则被梦灵附身；然后梦灵消失，目标陷入 
**失能 incapacitated** 并失去对其身体的控制。梦灵会接管目标的身体控制权，但不会剥夺目标的意识。梦灵不能被任何攻击、法术或其他效果指定为目标，梦灵保留了自己的阵营、智力、感知、魅力和免疫魅惑和恐慌。其他资料则与被附身的目标相同，但不能获得该目标的知识、职业特性和熟练值。

该身体生命值降到0时，受到驱散、驱逐善恶的法术效果逼出来时，附身终止。梦灵也可以主动用一个附赠动作终止附身。当附身结束时，梦灵会重新出现在离目标5尺以内的未被占据的空间位置。目标在豁免成功后或在附身结束后24小时内对该梦灵的附身免疫。
