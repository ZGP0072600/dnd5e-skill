# 魔匠师

**魔匠师 Magewright**

在科瓦雷，魔法是日常生活的一部分。厨师可以用魔术来加热和调味食物，而铁匠用修复术 mending 来进行小的修理和引导以启发他们的工作。那些在工作中使用小魔法的人被称为魔匠师。

与一般的施法者相比，魔匠师的魔法能力要有限得多，他致力于学习一些法术，魔匠师可以把不是戏法的法术当作仪式施法——即使是那些通常不能这么施法的法术。大多数魔匠师的仪式施法只需要10分钟，但某些复杂的仪式可能需要1个小时。不管这个仪式需要多长时间，它都需要额外的施法材料，通常是龙晶 dragonshard.

> **边栏：创建一个魔匠师 ** 魔匠师特性提供了魔匠师的基础信息。你可以从魔匠师特性表中选择一个职业来添加到基础模板中，或者骰出一个随机特长。这个职业决定了魔匠师已知的其他法术，包括那些只能仪式施法的法术。该职业也会给魔匠师带来更多的熟练项。
