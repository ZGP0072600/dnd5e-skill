# 离梦人 Kalashtar

离梦人已经与善良的梦灵精神结合，他们通过梦想和异象与他们的主人沟通。作为那些将自己的身体提供给逃避达·库尔 Dal 
Quor 的梦灵当作避难所的僧侣后裔，离梦人现在的战斗预示着光明与平衡的新时代的来临。
