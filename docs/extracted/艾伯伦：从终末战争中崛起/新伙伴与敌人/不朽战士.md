# 不朽战士 Undying Soldier

不朽战士组成了艾伦诺的死者之城的军队。他们守卫着不朽王庭的神庙和艾伦诺贵族的府邸，而艾伦诺大使经常有不朽者的保镖陪伴。不朽战士身穿传家宝盔甲，挥舞华丽的武器。
