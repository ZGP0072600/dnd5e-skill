# 泰坦战俑 Warforged Titan

泰坦战俑这种无心智的战争机械是于终末战争中制造的第一批战俑的大型发展版。泰坦战俑和其 
他战俑不同，不是活体构装生物；它们几乎没有自我意识，它的智能仅仅能够服从发自于激战区的冲锋命令。

巨人的遗产。在巨人时代，巨人奇械师建造盲目的战争魔像来帮助他们对抗梦族。几千年后，一些寻找那个时代秘密的冒险家发现了这些魔像，然后在终末战争中交给了为龙纹家族工作的奇械师们。卡尼斯家族研究了这些设计，在研究设计制造魔像的巨人工匠的秘密的过程中，卡尼斯家族创造了第一批泰坦战俑。
