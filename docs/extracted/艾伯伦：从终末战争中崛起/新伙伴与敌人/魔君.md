# 魔君

**魔君 Overlords**

在世界诞生之初，凯博尔的孩子们从黑暗中降生，统治着艾伯伦。他们之中最为强大者便是魔君，他们统治下的世界满是恐惧、战争和死亡，直到艾伯伦和西伯瑞斯的孩子们奋起反抗。巨龙的大军和凯博尔的邪魔血战。尽管霸主们无法被摧毁，但羽蛇们牺牲了自己的生命，制造出一座天界光芒铸就的监狱：银焰，将魔君们再次束缚在凯博尔之中。这些束缚已经存在了无数世代，但魔君们仍然渴望挣脱束缚，夺回地上的世界。

当魔君们被银焰所缚时，他们便无法在世界上显现。但每一位魔君都体现了邪恶一个特定方面，当他们的仆从——被称为砂主的邪魔们——谋划着释放他们古老的主人时，邪恶的力量便会增长。当凡人们接受邪魔为他们铺就的道路，魔君便会获得力量。随着他们愈发强大，他们也会获得更多的影响力。

大约有三十位魔君被束缚在凯博尔中，这里详述了其中两位：拉克·图尔克什和苏·卡特什。他们都处于监禁之中，无法以实体现身。这里的资料卡反映的是如果他们被释放到这个世界上，他们将会拥有的力量。
