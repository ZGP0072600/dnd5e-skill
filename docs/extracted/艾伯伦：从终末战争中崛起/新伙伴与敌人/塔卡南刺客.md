# 塔卡南刺客 Tarkanan Assassin

塔卡南刺客是精英杀手、间谍和盗贼，他们为专门从事盗窃和暗杀的犯罪组织塔卡南家族工作。除了他们的致命技能，塔卡南刺客还拥有一个异种龙纹——一种扭曲的印记为他们提供了魔法力量。塔卡南家族积极寻找和招募有异种龙纹的人。
