# 扎吉亚罗刹 Zakya Rakshasa

在类人种族崛起之前的数百万年，艾伯伦的罗刹统治着跨越科瓦雷的文明。作为战斗的主人，艾伯伦的罗刹挥舞着他们的武器，带着恶魔般的狂怒，冲进战场，对那些想封印他们那极恶如魔的主人进行复仇。

罗刹在怪物手册中有描述。这里呈现的是一种战斗分支——扎吉亚罗刹。

扎吉亚罗刹是罗刹部落嗜血的步兵。他们迫切想释放他们极恶如魔的主人，对龙和崇拜羽蛇的信徒*（注1）进行报复。与其他穿着华丽长袍和服饰的罗刹不同，扎吉亚罗刹几乎总是穿着战斗装备:破旧的鳞甲，锋利的长剑，以及带有他们极恶如魔的主人标志的厚重盾牌。
