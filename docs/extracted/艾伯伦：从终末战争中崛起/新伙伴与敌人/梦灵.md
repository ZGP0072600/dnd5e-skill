# 梦灵

**梦灵 Quori**

（3R 译作：梦族）

达·库尔 Dal 
Quor 是梦境的位面，由支配其名为 il-Lashtavar 支配着黑暗力量，或者称其为黑暗梦境 Dreaming 
Dark。（*注：黑暗梦境是一个梦灵的间谍组织）

 Il-Lashtavar 有许多化身，这些化身是梦和噩梦的体现。 
由于任何物理位面的东西都很难在达·库尔 Dal Quor 进出，因此在艾伯伦的梦灵需要寄宿在别人身上。 梦启者 Inspired（3R 译作：半魇灵）是梦灵的最常见的自愿寄宿形式，本章前面已对此进行了介绍。
