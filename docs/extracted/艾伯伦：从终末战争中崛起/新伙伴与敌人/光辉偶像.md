# 光辉偶像

**光辉偶像 Radiant Idol**

光辉偶像是自苍蓝天庭西拉尼亚遭到驱逐而坠入主物质位面的天使。引诱他们堕落罪只有一个：渴望被凡人崇拜。如今在凡人的国度，大多数光辉偶像都着手建立了教团，致力于吸纳崇拜者。

**堕落天使：** 想要被崇拜的渴望无穷无尽，从生理和心理上改变了堕落的天使。在其真实的形态下，光辉偶像的样貌是一个被扭曲了的天使。他的翅膀可能被血迹斑斑的骨茬所取代，又或者被一条由他的自傲所化成的锁链束缚着，垂向地面。光辉偶像会使用易容术掩盖他的腐化，粉饰成依旧享有天界荣耀的图景。

**腐化之重：** 在光辉偶像通过花言巧语和展示力量来赢得一定规模的崇拜者之后，他就会撕下虚伪的面具，而事态也就开始急转直下。随着光辉偶像在主物质界权能的泥潭中越陷越深，他开始扭曲自己的崇拜者，引导他们迈向不详的仪式、享乐主义的蒙昧和极端主义所带来的毁灭。

**不朽本质：** 光辉偶像不需要吃、喝和睡眠。

**伪神光环 Aura of False Divinity：** 
只要光辉偶像尚未失去意识，在其身周30尺范围内开始其回合的生物就必须通过一次 DC 17的感知豁免。在检定中失败的生物将被光辉偶像魅惑。以此法被魅惑的角色可以在结束自己的回合时重试豁免检定，成功则本效果对其自身的影响终止。生物一旦在豁免检定中成功，就可以免疫伪神光环24小时。
