# 不朽议员 Undying Councilor

不朽议员指导和保艾伦诺的人民。 他们是不朽王庭中最强大的成员，仍然保持身体形态。 
虽然许多不朽议员花费自己的时间去指导生者，但也有些不朽者花费数年时间将自己的意识投射到星界之上，寻求新知识与人们分享。 
不过，并非所有人都能承受永生，不朽议员有时候会屈服于压力并发疯。
