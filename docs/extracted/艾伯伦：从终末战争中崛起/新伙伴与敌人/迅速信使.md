# 迅速信使 Expeditious Messenger

迅速信使是一种快速飞行的信使，旨在迅速为其创建者传递消息。 
他们的迅速和高效的态度使迅速的信使变得相当健谈，并且他们在移动时就迅速行动。这些信使有多种形式，通常看起来像机械鸟 mechanical 
birds 或小仙灵 sprites。
