# 化兽者

**化兽者 Shifter**

化兽者与原始的灵魂联系在一起，他们中的大多数人将其称为“内心的兽性”。它们姿态轻盈，具有兽性的特征:大眼睛、扁鼻子、尖耳朵，身体大部分部位覆盖着浅色皮毛。当一个化兽者通过“化形”完全释放兽性，这些特征会变得更明显。
