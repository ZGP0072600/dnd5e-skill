# 战俑士兵

**战俑士兵 Warforged Soldier**

战俑士兵是由木头和钢铁制成的类人生物，然后神奇地充满了生命和知觉。战俑是为了在终末战争中战斗而制造的，在那场战争的余波中，他们努力理解自己在世界上的地位。

大多数经历过战争的士兵仍然为他们的前主人服务，经常守卫有龙印记的房屋或商队。许多战俑发现很难将自己与士兵区分开来，他们寻找保镖、卫兵和护卫的工作。另一些人把他们不知疲倦的力量作为劳动者，致力于重建他们在战争中防卫或摧毁的城镇。

**战俑特性 Warforged Resilience。** 战俑在中毒 poisoned 状态豁免具有优势，并且免疫疾病。魔法无法让它睡眠。

**动作多重攻击 Multiattack。** 战俑士兵可以进行了两次武器攻击。

**臂刃 Armblade。** 近战武器攻击:命中+5，触及5尺，单一目标。伤害:6 
(1d6 + 
3)挥砍伤害。

**标枪 Javelin.。** 近战或远程武器攻击:命中+5，触及5尺或30/120尺射程，单一目标。伤害:6 
(1d6 + 3)穿刺伤害。

**反应保护 Protection。** 当战俑士兵看到敌人对自己5尺范围内的生物进行攻击检定时，战俑士兵可以对使敌人攻击检定具有劣势。
