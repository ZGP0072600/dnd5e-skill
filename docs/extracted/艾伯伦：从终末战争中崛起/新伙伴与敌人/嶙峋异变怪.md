# 嶙峋异变怪 Dolgaunt

嶙峋异变怪是一种带有疾病色彩的瘦弱的妖怪。他们的眼窝张开着，嘴巴上方是一张像虫一样的舌头。

扭动的纤毛覆盖着它们的身体，它们的头周围有更长的卷须，裸露的肩膀上伸出两根结实的触须。嶙峋异变怪是盲人，但可以通过覆盖在皮肤上的敏感纤毛感知周围环境。它也可以通过它的触须吸收生命，让它吸干它接触的任何生物的活力。

**扭曲的混乱。** 当异变魔 daelkyr 从瑟瑞奥特 Xoriat 出来征服科瓦雷 Khorvaire 时，他们捕获并改造了当地的土著生物创造一支由丑陋的战士组成的军队。腐化者-迪恩 Dyrrn 
the 
Corruptor 把大地精 hobgoblin 变成了嶙峋异变怪，把他们变成了聪明、冷酷、高效的杀手。

当异变魔 daelkyr 被击败后，嶙峋异变怪和他们的主人一起藏在凯博尔 Khyber 的深处。在那里，他们在洞穴般的寺庙里学习，把自己的身体锻造成活的武器，奉献给那些主人赋予他们的使命。嶙峋异变怪经常被发现指挥狰狞异变怪 dolgrims 的小队，也可以被发现与黄泉巨龙会 Cults 
of the Dragon Below 一起工作——特别是那些献身于腐化者-迪恩 Dyrrn the Corruptor 的。
