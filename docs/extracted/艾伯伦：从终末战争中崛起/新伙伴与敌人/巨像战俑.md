# 巨像战俑 Warforged Colossus

在终末战争的最后几天，卡尼斯家族释放了最强大的作品：巨像战俑。这些巨大的机械战士在赛尔上轰鸣，摧毁了他们前进道路上的一切，并在他们身后留下了废墟。这只巨兽打算彻底结束战争，将数百名士兵打碎在脚下，并用从嘴里射出的炽热光束焚烧整个军团。在哀伤之日，这种横冲直撞的行为被打断了。吞噬了赛尔的奥术灾难摧毁了大部分巨像战俑，使它们变成了一堆废铁。

**幸存的巨像。** 一些战火巨人或多或少地在哀悼日幸存下来，被认为只是处于待机状态，等待被唤醒。卡尼斯家族的奇械师渴望找到他们，并确定他们是否可以被唤醒，或者至少可以挽救他们的魔咒和其他魔法物品。以下是传闻中作战作战巨像居住的地点：

**空心结构。** 虽然它可以在没有机师的情况下运作，但巨像战俑是用来运送法师、奇械师、魔匠师 magewrights 和精锐部队的。由于这个原因，这些建筑内部大部分是中空的，带有隧道、梯子、储藏区和观景台。

巨像除了里面死去的人的尸体外，还可能充满了怪物和秘密。在第四章中，地图4.8描述了一个不活跃的巨像战俑，并解释了巨像作为冒险地点的作用。
