# 科瓦雷的德鲁伊

·
吾等灰烬誓盟之德鲁伊，身为自然世界之战士，抵御一切威胁自然之物。

·
吾等凛冬之子之德鲁伊，身为生死自然循环之捍卫者。疾病与衰败将清除弱者，让生命愈加强大。吾等剿灭亡灵，以确保生死循环得以维系。

·
吾等护门者之德鲁伊，保护自然世界不受异界之力的影响，更要抵御那些自黑暗深渊之中潜伏而来之物。

·
吾等绿吟者之德鲁伊，欢欣于自然世界中的魔力及其体现之物，妖精。身为妖精与凡人之间的使者，吾等予两者同等之保护。

·
吾等森林守望者之德鲁伊，保护着所有的艾伯伦之子，无论是野兽，还是市民。我们维护着自然与文明之间的平衡，帮助它们了解彼此。

无论哪个派系的德鲁伊都在关照自然世界，但他们采取的行动有所不同。五条确立已久的信仰之路囊括了科瓦雷大部分的德鲁伊。在创建德鲁伊角色时，考虑你是否与这些教派之一有关联，你又因何离开了你的教团。你在执行任务吗？你在探索世界吗？如果你被放逐了，是出于正当理由，还是其他原因？

灰烬誓盟
灰烬誓盟保护自然世界免受任何威胁。一些灰烬誓约教派的德鲁伊认为文明本身就是一种威胁和袭击，因为聚居点侵蚀了荒野。其他人则把愤怒倾泻在了龙纹家族身上，又或者试图解放被束缚的元素。

凛冬之子
凛冬之子*相信死亡与衰败是生死自然循环里的重要一环。他们相信如果循环的平衡被打破，世界就会生出一场大灾变，以重置这一平衡。他们一方面与亡灵战斗，另一方面采取行动清除弱者。众所周知的是，这一教派的极端分子会在城市里散布瘟疫。凛冬之子们对解开大哀恸 **之谜特别感兴趣，因为他们中有些人认为这是一个征兆，预示着他们一直恐惧的大灾变即将来临。

护门者
护门者是最古老的德鲁伊教派之一，根据地在阴影湿地，时常穿梭于矿砂之间。护门者的任务是保护艾伯伦不受异怪和其他非自然生物的侵害，并试图防止异位面渗透和袭击。护门者保卫着古老的封印，将被遗忘已久的邪恶拒之门外。

绿吟者
绿吟者献身于妖精，充当妖精和凡人之间的调解人。绿吟者的成员包括吟游诗人以及与至高妖精签订契约的魔契师；某位德鲁伊或游侠也可能为特定的至高妖精服务。

森林守望者
森林守望者相信文明在世界上有一席之地。作为一名森林守望者，你帮助他人了解自然，确保他们不会造成无意地伤害或是陷入危机。森林守望者们在埃鲁登原野充当民兵和调解员。这个教派是几个德鲁伊教派中规模最大，也是最受认可的一个。

萨恩探秘者报
森林有奇事：让抱树人告诉你一切！

安黛尔西部的农民反叛军联盟已经宣布与齐云森林的大德鲁伊欧里安结盟，声称位于森林边界的农田现在是埃鲁登原野的一部分。大德鲁伊是一个传奇人物，一个强大而神秘的伟人，更是遍布这一地区的各色德鲁伊的精神领袖。想知道关于欧里安的另一个有趣的事实吗？他是棵树！有人说他是艾伯伦亲生之子，也有人说他是一个古代德鲁伊英灵，只不过被困在大树形态之中。不管真相是什么，伟大的德鲁伊欧里安看起来就像一棵巨大的松树，呆在齐云森林深处一片被称为绿之心的树林里。
