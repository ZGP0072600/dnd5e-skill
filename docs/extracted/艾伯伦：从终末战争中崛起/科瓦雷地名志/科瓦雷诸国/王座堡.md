# 王座堡

首都： 王座港
Throneport
特点： 艺人，国际阴谋，间谍

在建立了一个联合王国之后，伽利法一世很早就意识到，国王不可能从一个现存的王国开始统治。 这样做会使五国中的一个凌驾于其他国家之上，并导致分裂，最终摧毁王国。 因此，他在传承海峡 Scions Sound 的一个岛上建立了自己的权力中心。 他在那里建造了宏伟的王座堡。

在嘉奥特 Jarot 死后，对导致终末战争的继承权的拒绝，岛上和城堡基本上被遗弃了。 一个邓奈斯家族的特别卫队，即王座卫士 Throne
Wardens ，保留下来保护城堡， 但随着伽利法王国的崩溃，政府的所有职能都停止了。

城堡阴影下的王座港小镇成了异见人士、罪犯、间谍和雇佣兵的聚集地，这个曾经安全的港口也变成了一个混乱的小镇，没有任何一个国家效忠。

王座堡的有趣之处
这五国的代表(和间谍)聚集在王座港，在那里是不可能分辨朋友和敌人，联盟每天都在变化。
虽然赛尔在岛上不再有维和部队，但​​许多赛尔人在这里找到了为其他国家做间谍和特工的工作。
王座堡角色
当你从王座堡创建一个角色或 NPC 时，考虑以下几点:

冷静的战士. 王座堡是外交官和间谍的角斗场，是无聊贵族的危险游乐场，也是奸诈的双重间谍的天堂。 具有诈欺师、间谍、贵族或犯罪背景的角色都可能在这里测试他们的技能。

游戏人间. 如果说外交官和间谍有一个共同处的话，那就是他们的弱点都是容易分心。 当他们不玩欺骗游戏的时候， 他们会寻找岛上最好的娱乐和陪伴。 有艺人背景的角色在这里会交到很多靠不住的朋友。

终末战争的后果
再 975 YK ，王座港已经发展成为一个小城市，所有的国家和龙纹家族至少都有一个小小的存在， 把这里变成了国际阴谋的温床。 王座堡条约进一步巩固了王座港作为一个由来自安黛尔 、布鲁兰、坎纳斯和瑟雷恩的小型维和部队控制下的多国首都的地位， 并配备了邓奈斯的王座卫士以确保条约得到遵守。 今天，这个小城市是中立的地方，但城堡和它的场所仍然是禁区， 受到看守的严密保护。
