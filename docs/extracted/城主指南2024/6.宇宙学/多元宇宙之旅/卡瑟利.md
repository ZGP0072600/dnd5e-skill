# 卡瑟利 Carceri

卡瑟利，这片隔离与绝望之位面，是世间一切监狱的惨烈灵感之源。六个位层如同囚徒噩梦的延伸：广袤沼泽、腐败密林、荒凉沙漠、嶙峋山脉、寒冷汪洋，以及黑冰冻原。这里是背叛者与阴谋家的最终归宿，凄凉无际，绝望无边。

**卡瑟利诸位层**

| 位层 | 描述 |
|---|---|
| 沙沼狱奥瑟瑞斯 | 冥河蜿蜒流经无边的沼泽与流沙。 |
| 朽瘴狱卡瑟瑞斯 | 腐朽气息弥漫在猩红平原与腐臭密林中。 |
| 岚漠狱米奈瑟斯 | 无尽沙暴咆哮肆虐，滚滚飞沙将古老流沙城的废墟隐藏其下。 |
| 刃渊狱科洛瑟斯 | 深渊与山峦交错纵横，使人寸步难行。 |
| 蚀瀛狱波法提斯 | 终日飘洒的黑雪落入冰冷酸洋之中。 |
| 黯冰狱艾嘉西斯 | 透出血红痕迹的漆黑坚冰覆盖着这一冰寒界域。 |

## 监狱位面 Prison Plane

没人能轻易地离开卡瑟利。用除祈愿术 Wish 外任何法术离开此地的尝试都会失败。连接此地的传送门和通道仅能单向使用。虽然说存在离开卡瑟利的秘密逃生通路的确存在，但它们被诡异的机关和致命的怪物严密守卫。冥河虽通向与卡瑟利相邻的位面，但在此段河流航行极度危险，而离开此地的渡船稀少且昂贵。

## 卡瑟利冒险 Carceri Adventures

卡瑟利无烈日之光，只余绝望蔓延。这里是激情与毒物、背叛与痛苦的聚合地，仇恨如深流，缓缓蚕食一切，无法预测下一刻将毁灭何人何物。据说，囚徒想要逃离此地，唯有变得比囚禁他们的力量更为强大。但在这片滋生绝望与背叛的领域，众志难成，更遑论联合抗争。

冒险者可能需要直面禁锢自身的种种力量——不仅是实体的桎梏，还有心理、灵魂的束缚，甚至恶魔的压迫。冒险任务或许包括解救一名囚徒，无论是一个蒙冤的灵魂，还是被流放至此的远古神祇。然而，这样的救赎是否会引来更深重的代价，则无人知晓。
