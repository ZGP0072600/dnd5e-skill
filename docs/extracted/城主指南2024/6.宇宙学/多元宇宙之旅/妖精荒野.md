# 妖精荒野 Feywild

妖精荒野，也称仙灵位面，是一个充满柔光与奇迹之地，一片交织着音乐与魔法的梦幻国度。在这里，情感的波动能够引发世界的回响：花朵会随着激烈的争吵转动颤抖，草地会在恶意者的脚下枯萎，而鸟雀会为欢乐者欢唱，为阴郁者愤鸣。

在妖精荒野，时间与距离如同幻影一般变幻无常，这片土地的地形同样随心而变。道路稀少，且那些存在的路径如同周围的风景般瞬息万变。妖精荒野的居民早已习惯了这片领域的无定形态，但对初到此地的旅行者而言，这种变幻却极易令人迷失。

妖精荒野与物质位面平行而存，乃与之占据相同宇宙空间的另一次元。穿越物质位面进入妖精荒野时，旅行者通常会发现自己仍置身于类似的地点，但那里的景象却更加绚丽与神秘，更显生机与色彩。例如，攀登物质位面火山的冒险者也许在眨眼间便出现妖精荒野的水晶尖峰上，山峰直抵天穹，而期内闪烁着火光；而在物质位面中一条泥泞宽阔的河流，则可能在妖精荒野中化作一条闪耀如钻石的蜿蜒小溪；阴郁的沼泽中心，或许隐藏着通往鬼影迷离、灯火诡异的广袤泥泽的门户。而从物质位面的废墟进入妖精荒野，可能将旅行者带至某位强大至高妖精的城堡门前。

## 欢欣领域 Domains of Delight

妖精荒野中的许多区域都由被称为至高妖精的强大妖精所统治。他们的领地被称为欢欣领域，反映出其统治者的性情与欲望。有些领域阳光普照、五彩缤纷，沐浴在永恒的白昼中；而另一些领域则灰暗单调，笼罩在无尽的黄昏之下。大多数领域会随着领主的情绪而变化，宛如其内心的镜像。

以下是几个著名欢欣领域的描述：

- **寓言丘 Fablerise。**寓言丘由一位名叫纺语者的蜘蛛至高妖精统治，这位至高妖精很擅长讲故事。这是一片由盘根错节的树根、带刺藤蔓和蜿蜒蔓延的植物交织而成的茂密丛林。植被形成了长长的隧道、宽广的大厅和巨大的穹顶。纺语者钟爱为领域内的动物朗诵故事。
- **薄暮王庭 Gloaming Court。**薄暮王庭由空暗女王统御，这是一片由暮光、萤火虫、蛛网与秋叶构成的领域，伴随着猫头鹰的鸣叫与青蛙的呱噪。薄暮王庭的妖精排斥仲夏王庭（见下）的规范礼仪与繁琐仪式，而更崇尚直觉与本能。
- **棱境 Prismeer。**棱境是至高妖精泽碧尔娜的广大领地，包括三大区域：广阔的沼泽，此境；古老的森林，彼境；以及风暴肆虐的山地，他境。泽碧尔娜居住在心愿宫，这座宫殿坐落在三地交汇之处，传说中是任何寻求心愿之人最终的目的地。在某些世界中，泽碧尔娜被视为一位仙灵教母，愿意为那些迷失、被遗弃或背叛者实现愿望。然而，她的愿望有时会带来幸福，有时则带来绝望。（棱境在《巫光之外的荒野》中详述。）
- **仲夏王庭 Summer Court。**仲夏王庭由至高妖精泰坦妮娅女王统治，是妖精荒野中最为安定与田园牧歌式的领域。这里总是沉浸在永恒的夏日阳光下，蝴蝶飞舞，鲜花如海，处处洋溢着繁盛与生机。仲夏王庭的居民身着优雅华丽的服饰，重视精致的仪式和形式化的礼节，那些无视此处繁复规则的访客往往会被迅速赶走。

## 妖精通道 Fey Crossings

妖精通道是物质位面与妖精荒野之间神秘而美丽的交界地，它们与妖精荒野中某处完美地映照对接，在位面相接之地形成一道传送门。旅行者可以通过进入一片林间空地、走入一汪水池、踏入蘑菇圈或钻过树干下方，悄然穿过妖精通道。在旅行者看来，自己只是一步踏入了妖精荒野中。而其他观察者则只会看到该旅行者在那一刻消失无踪。

正如其他跨位面的传送门，大部分妖精通道只有在特定条件下才会开启。比如在满月之夜、在某个特定日子的黎明，或某人持有某种特定物品才会开启。一旦妖精通道两端的地貌发生剧烈变化——比如物质位面的空地上建造了城堡——通道便可能永远关闭。

## 妖精荒野魔法 Feywild Magic

传说中，孩童会被妖精掳走，带往那神秘的妖精荒野。多年之后，他们奇迹般地回到父母身边，面容依旧如初，未曾老去一日，却对掳走他们的生物及那片奇异领域毫无记忆。同样，许多冒险者从妖精荒野归来时，惊愕地发现，妖精荒野的时光与他们熟悉的现实大相径庭，而他们的记忆也变得模糊而遥远。下述可选规则可以用于反映弥漫这片异境的奇妙魔法。

- **记忆丧失 Memory Loss。**当一名生物离开妖精荒野时，需进行一次 DC 10的感知豁免。妖精以及拥有妖精血统特质的种族（如精灵）自动通过此豁免。若豁免失败，该生物完全遗忘自己在妖精荒野中的经历；若豁免成功，则记忆虽保存完整，但却仿佛隔着迷雾，变得模糊不清。任何能够结束诅咒的法术都能恢复这些失去的记忆。
- **时间扭曲 Time Warp。**尽管在妖精荒野中时间似乎流逝如常，但角色或许只在那里逗留一天，离开后却发现整个多元宇宙的时间流逝截然不同。

当一名或一群生物在妖精荒野待满至少一天后离开，DM 可以选择适合剧情的时间变更效果，或在妖精荒野时间扭曲表上掷骰决定。一道祈愿术 Wish 可以移除最多十名生物的时间影响。有些强大的妖精能够施展这样的祈愿术，但通常会要求受益者接受一道指使术 Geas 并在祈愿术施法完成后完成某个任务。

**妖精荒野时间扭曲**

| 1D20骰点 | 结果 |
|---|---|
| 1-2 | 日变为分钟 |
| 3-6 | 日变为小时 |
| 7-13 | 无变化 |
| 14-17 | 日变为周 |
| 18-19 | 日变为月 |
| 20 | 日变为年 |

## 妖精荒野冒险 Feywild Adventures

妖精荒野是强烈情感的物质化表述，这一位面同样精于隐喻。在这片领域中，角色可能会失去一段珍贵的记忆，或一个深藏的遗憾，而失落的记忆可能会以小雕像或挂坠的形式具现。一个调皮的小仙灵或许会趁某位角色放声大笑时，悄悄夺走他们的笑声，从此令角色再也无法发笑，直至找到小妖精并取回那笑声——也许它已化作一束芬芳馥郁的花束。
