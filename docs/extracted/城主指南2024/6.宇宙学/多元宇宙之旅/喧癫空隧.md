# 喧癫空隧 Pandemonium

喧癫空隧，一个充满压倒性混乱的位面，是一块充满咆哮狂风雕就隧道的巨石。这里寒冷、喧闹且黑暗，没有一丝自然光。狂风会迅速熄灭如火把和营火等非魔法的明火，也使得交谈只能通过大声喊叫进行，且声音至多只能传到十尺远。关于喧癫狂风，参见第三章“环境效应”。

此位面的大多数居民是被放逐至此、绝无逃离希望的生物。无休止的狂风迫使他们躲藏在那些能稍微隔绝风声的地方，即便如此，啸号风声仍犹如远处传来的痛苦哀嚎。

**喧癫空隧诸位层**

| 位层 | 描述 |
|---|---|
| 风啸隧层潘迪斯摩斯 | 呼啸狂风、通向冥河的黑暗溪流与飞雪席卷着广阔而荒凉的洞窟。 |
| 哀鸣隧层考库托斯 | 更狭窄的隧道增强了风力和哀嚎声，此层也因此被称为“哀鸣隧层”。 |
| 虚暗隧层弗勒格松 | 隧道墙壁吸收光线，而水流则创造出复杂的岩石结构。 |
| 秘隐隧层阿伽西翁 | 封闭的隧道几乎无法从外界进入，因而成为存放古老秘密的理想之地。 |

## 喧癫空隧冒险 Pandemonium Adventures

喧癫空隧是绝望的终点。这里狂风的无休哀嚎最终会把所有身处其间的人推到崩溃的边缘——无论是因愤怒而失控，因绝望而垮塌，还是彻底陷入迷乱。仅仅在此位面维持存在都令人筋疲力尽，而尝试完成哪怕最简单的对话也会令人不堪重负。

在喧癫空隧的冒险，是一种探寻角色最糟糕日子的方式。当一切都在崩溃，连风的咆哮都拒绝停歇时，角色会如何应对？然而，关键在于将角色在此经历的挫败感表达出来，而不将这种挫败感转嫁给玩家。

在哀啸隧层某处，有一处名为风嚎尖岩的崎岖尖峰，据传拥有一种独特的魔法属性：在尖岩顶端向某人喊的一切话语，都能通过凛冽尖啸的寒风找到它的目标——无论那人在多元宇宙的何处。
