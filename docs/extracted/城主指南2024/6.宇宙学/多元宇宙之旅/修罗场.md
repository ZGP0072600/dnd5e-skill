# 修罗场 Acheron

修罗场中遍布巨大的铁块，其表面回响着无尽军队的行军步伐。这些铁块漂浮在虚空中，有时彼此猛烈碰撞，发出可怕的巨响，粉碎其间的一切，并让整个位面为之震颤。

修罗场共有四层，越往上铁块越大。一些学者提出理论，认为顶层的巨块在互相撞击的过程中最终会分解成较小的物质碎块并沉入底层。然而，事实恰恰相反：在其最底层，寒渊墟奥坎瑟斯极深处的冰冷泥沼会崩裂出细小的碎片，这些碎片在邻近的机械境影响下，逐渐组装成战魂墟阿瓦拉斯的完美立方铁块。

修罗场的本质会激发访客的嗜血欲望（见第三章“环境效应”）。

**修罗场诸位层**

| 位层 | 描述 |
|---|---|
| 战魂墟阿瓦拉斯 | 灵体士兵在遍布残骸的战场上进行无尽的战争。 |
| 遗械墟图尔达宁 | 坑洼和空心的立方体内充斥着废弃的战争机器。 |
| 冥忆墟丁提布鲁斯 | 锯齿状的巨块在黑暗中翻滚，封存着被冥河剥离的冰冷记忆，最终结晶为短暂一瞬。 |
| 寒渊墟奥坎瑟斯 | 锋利碎片在黑冰沼泽上空的旋涡中盘旋。 |

## 修罗场冒险 Acheron Adventures

修罗场是强制秩序之位面，在这里，严苛军令会压碎魂灵，粉碎希望。这里的精魂完全无法想象有人会拒绝履行指挥官的命令。他们是忠诚的士兵，却不知自己为何而战。

前往修罗场的旅途是一场与无尽冲突中冷酷虚无的对抗，是面对威权统治的严酷现实，以及毫不留情的社会同化压力的体验。这也是角色们质疑自己为何而战的契机——尤其是在那些早已忘记了提问为何物的军队之中。

说得更实际一点，修罗场的冒险可能包括阻止某个反派从遗械墟中回收战场残骸，以制造新的恐怖战争武器。又或许，这趟冒险需要从冥忆墟的石块中取回被囚禁的思维或记忆，以揭开其中隐藏的秘密。
