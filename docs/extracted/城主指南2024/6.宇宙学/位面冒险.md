# 位面冒险

**位面冒险 Planar Adventuring**

在现实世界的神话、传说和文学作品中，冒险进入其他存在位面绝不仅是造访奇异生物居住的非常环境，而是一场由符号与隐喻层叠描绘而成的意义之旅。D&D 宇宙的位面冒险也有机会创造出相似的充实而深厚的旅途。

位面冒险应当是非同寻常的。当冒险者在你的战役中抵达中高等级的时候，他们会有更充分的理由去探索家园世界界限之外的领域。如果你已经准备好让小队去完成真正的神话任务，比如寻找失落的神器或神秘的异界实体，那么广袤的多元宇宙将任你探索。这可不只是考验冒险者的物理与魔法能力，他们可能还会面对内心情感、价值观和精神上的挑战。

## 血战 The Blood War

纵观历史，数量庞大的深渊恶魔一直在与军纪严明的九狱军团争夺宇宙的霸权。在物质位面的世界中，那些了解这场冲突的人将其称为“血战”。这是一场持续数万载，波及所有下层位面的冲突。

这场战争主要发生在九狱和深渊，但物质位面、巨轮上九狱和深渊之间的下层位面、以及任何恶魔或魔鬼可能聚集的地方亦有不少冲突。尽管冲突的激烈程度有升有降，战争的前线也时常变化，但在多元宇宙的这些地方，恶魔与魔鬼的厮杀永不停歇。

不少恶魔和魔鬼都痴迷于为己方在血战中谋取优势，而强大的凡人冒险者有时也会被卷入那错综复杂的阴谋中。此外，邪魔间的公开冲突在大部分下层位面都是常态，即使这与冒险者的主要任务毫无关系。比如，虽然任务只是寻找一件藏于哈迪斯中的宝藏，但如果这宝藏位于战场中心，这任务可就没那么简单了！

上层位面的原住民也与血战有着千丝万缕的联系。虽然其中的大多数乐见邪恶势力黑吃黑，但他们仍会采取行动使血战对多元宇宙的其他部分产生的威胁最小化。每当血战波及到除深渊和九狱之外的地方时，天使和其他神仆（包括具有非凡勇气的凡人）便会随时准备干涉。

## 位面冒险情景 Planar Adventure Situations

你可以使用位面冒险情景表而非第四章中的“依等级排列的冒险情景”部分作为那些将冒险者引入其他存在位面冒险的灵感。这些冒险灵感最适用于等级11+的角色。

**位面冒险情景**

| d10骰点 | 情景 |
|---|---|
| 1 | 当魔法无法复活死者时，唯一的解决方案便是前往外层位面，找到那人的灵魂，要么将其从某种监狱释放出来，要么说服那人起死回生。 |
| 2 | 进入森林中的堡的人们意外闯进了妖精荒野或堕影冥界。他们也许永远不会回来了，也可能回来了却没意识到过去了多久时间，又或者回来了却发生了非常大的变化。 |
| 3 | 一位死亡许久的先知是唯一知道如何避免一则可怕预言成真的人，但预言已经开始灾难性地实现。 |
| 4 | 一位神停止回应其祈祷者，也拒绝回应任何通神术。 |
| 5 | 一位恶魔诱使一位天使参与血战，而天使正在为此寻找凡人的援助。 |
| 6 | 其中一位角色的祖先必须先被说服以祝佑该角色，否则其血脉便无法觉醒。 |
| 7 | 一位愚勇的骑士携带着神圣武器在九层地狱执行着注定失败的任务，而天界山的大能希望角色能取回这件武器及骑士的遗体。 |
| 8 | 一位泰坦被囚禁在外层位面。角色可能要阻止那些试图释放它的人，或者释放它以从更大的威胁手中保护世界。 |
| 9 | 为了证明自己配得上更伟大的任务，角色被派去猎杀可怕的怪物、赢得某个强大位面生物的青睐、调停两个敌对的位面派系，或是取回某个在异界失落已久的物品。 |
| 10 | 印记城、黄铜之城或其他某个位面大都市的拍卖会上正在出售一件传奇物品。 |
