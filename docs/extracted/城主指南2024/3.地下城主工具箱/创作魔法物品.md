# 创作魔法物品 Creating a Magic Item

角色们在冒险中可能发现无数的魔法宝藏，而第七章中的魔法物品不过是其中很小的一部分。你可以参考以下指导来修改现有的物品或是进行全新的设计。

玩家制作魔法物品的相关规则见第七章。

## 修改魔法物品 Modifying an Magic Item

你可以通过调整与整合现有物品来创作一件新物品。以下是相关建议：

修改效果 Altered Capabilities

将其中一项效果替换为相近的另一效果。例如将攀爬药水改为游泳药水。

修改类别 Altered Form
 你可以在保留原有词条的情况下修改魔法物品的类别。例如你可以将一件公羊戒指改为一根魔杖，或是将一件防护斗篷改为其他穿戴物（比如一件头环），这样做无需改动物品的词条。

修改伤害类型 Altered Damage Types

你可以将一件魔法物品造成的伤害类型改为另一种，例如将一把焰舌剑从造成火焰伤害改为造成闪电伤害。

整合物品 Combining Items
 你可以将两件相同稀有度的魔法物品的词条组合成一件物品，只要这二者不都需要同调。例如，你可以将通晓语言头盔与心灵感应头盔的功能整合为同一顶头盔的功能。这样做会使这件物品变得更加强大（有可能增加其稀有度），但又不至于破坏整个游戏。

特殊特性与智能 Special Feature and 
Sentienc

第七章中提供了关于赋予魔法物品有趣的历史、小词条、小特点还有智能的规则。

## 创作新物品 Create a New Item

如果修改一件物品不足以满足你的需求，你还可以从头创建一个全新的物品。一件魔法物品要么可以让角色做他以前不能做的事，要么能够提高角色的能力去做一些他本身已经能做的事情。例如，*跳跃戒指 ring of jumping*可以让它的着装者跳得更远，从而提升角色已有的能力。另一种情况是，一枚*公羊戒指 ring of the ram* 
 
可以赋予角色制造力场伤害的能力。

你所用的方法越简单，角色使用这些物品时就越容易。你可以给一件物品加上充能，尤其是当它有几种不同的能力时；不过，你也可以直接决定一件物品是始终可用或是每天仅能使用特定的次数，这会更加方便使用。

威力水平 Power Level
 如果你创作的物品可以让角色用它直接杀死其命中的任何目标，则该物品可能会使你的游戏失去平衡。另一方面，很少能发挥作用的物品也不能算作是奖励。

使用稀有度对应魔法物品威力表作为指引，基于物品的稀有度来确定一件物品应该具有的威力。

**稀有度对应魔法物品威力 Magic Item Power by Rarity**

| 稀有度 | 最大魔法环阶 | 最大加值 |
|---|---|---|
| 普通 | 1环 | —— |
| 非普通 | 3环 | +1 |
| 稀有 | 5环 | +2 |
| 极珍惜 | 8环 | +3 |
| 传说 | 9环 | +4 |

- **最大法术环阶 Maximum Spell Level。**表格这一列表示物品应该授予的最高环阶法术效应，其形式为每日一次或类似的带有限制的词条。例如，一件普通等级的魔法物品可以每日一次（如果是消耗品那就仅一次）地赋予一项1环法术的效应。一件珍稀、极珍稀或传说级的物品可以让其持有者更频繁的施展一项低环阶的法术。
- **最大加值 Maximum Bonus** 。如果一件物品提供永久的 AC、攻击检定、豁免检定或属性检定加成，则可参考该列给出的加值建议（基于物品稀有度）。

**同调 Attunement**

最后，确认这件物品是否需要一名玩家角色与之同调才能使用其词条。以下是一些参考建议：

- **共享限制 Limit Shairing** 。如果队伍中的每个角色可以通过互传该物品来从其长效增益中获益，则该物品可能破坏平衡因而需要同调。
- **堆叠限制 Limit Stacking** 。如果该物品提供的增益其他物品也同样具备，则要求进行同调会是一个好点子，这样其使用者就不会试图收集过多的同类物品。
