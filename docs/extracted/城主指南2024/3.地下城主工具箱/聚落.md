# 聚落

**聚落 Settlements**

战役所发生的世界通常存在许多聚落。角色们可以抵达这些地方，甚至还可能将某个聚落作为基地，并在达到足够等级之后在其中或附近建设他们的据点（见第八章）。

基于规模的聚落表给出了村庄、小镇还有城市的建议人口范围，以及每种规模的聚落的交易限额。这些数据皆可根据当地情况而酌情调整。举例而言，一瓶治疗药水（售价50GP）超过了大多数村庄的交易限额，所以通常无法在村庄中购入；不过若是村子里住着一位炼金术士、草药学家或药水酿造师，那在这里就说不定能买到一两瓶治疗药水。

**基于规模的聚落 Settlements by Sizes**

| 聚落 | 人口规模 | 交易限额 |
|---|---|---|
| 村庄 | 至多500 | 20GP |
| 小镇 | 501-5000 | 2000GP |
| 城市 | 5001或更多 | 200000GP |

| 边栏：我需要画地图吗？Do I Need a Settlement Map？ 并不是每个聚落都需要一份地图。通常来说，只需要简单和玩家描述一下聚落的情况便足够了。但若是想让玩家记住特定的建筑或某些其他地点在聚落中的位置，那地图便派上用场了。附录 B 的十字路口村庄便是一份范例聚落地图。 |
