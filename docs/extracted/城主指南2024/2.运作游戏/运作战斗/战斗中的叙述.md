# 战斗中的叙述

战斗中的叙述 Narration in 
Combat

虽然要让玩家理解规则很重要，但如果每个人都只使用“游戏术语”，游戏就会变得乏味：“命中扔了18”，“你命中了，现在扔伤害”，“11点”，“好的，现在到先攻13的角色”。取而代之，你应该利用规则和对场景的了解来帮助你去叙述战斗。如果18只是堪堪命中，但11点伤害对敌人来说却是严重的创伤，你可以说：“你疯狂地舞起剑来，那骑士虽举起盾牌但为时已晚。剑刃刺中了他的下巴，划出一道深深的裂口。他被震得后退几步，血流不止！”

当角色与怪物战斗时，如下文所描述那样，你可以揭示一些信息来帮助玩家做出正确的选择。

**失去生命值 Loss of Hit Points**

你可以用描述性的语言来形容一个生物的伤势，从而让玩家了解他们与那个生物的对抗情况。例如，如果生物已经浴血，你可以说它身上有明显的伤口，看起来被打得惨兮兮的。这样的信息可以让玩家了解目前的进展，并可能会促使他们继续进攻。另一方面，如果角色们对那生物没造成什么伤害，那就让玩家知道那生物看上去毫发无损。这也可能会鼓励玩家改变他们的计划。

**能力，强项与弱点 Abilities, Strengthes and Weaknesses**

当与角色战斗的生物的特性在战斗中逐渐显露时，请与玩家分享这些信息。例如，如果法师对火元素 Fire 
Elemental（一种免疫火焰伤害的生物）施放了火焰箭，让玩家知道这个法术看上去完全不对这个生物构成威胁。玩家可能会正确地猜想，由火组成的生物可能不会被火焰箭伤害，而你大可以证实他们的猜想。

**战斗中的动作 Actions in Combat**

当怪物在战斗中执行动作时，玩家需要同时了解游戏中和规则上发生的情况。这意味着，当一名手持着弩的敌人执行预备动作来掩护门前的区域时，你应当让玩家清楚认识到，如果他们的角色走到那扇门前，敌人就会向他们射击。怪物图鉴中关于怪物的描述通常会解释当他们使用特殊动作时这个世界会发生什么变化。描述动作表中的描述可用于解释生物使用它们可用的常见动作时会发生什么。

**描述动作表 Describing Actions**

| 动作 | 描述 |
|---|---|
| 疾走 | 你的敌人没有发动攻击，而是急匆匆地跑向对面。 |
| 撤离 | 敌人防备地和你拉开距离。 |
| 回避 | 你的敌人紧盯着你，试图格挡住你的攻击。 |
| 协助 | 当它的盟友朝你攻击时，敌人四处乱窜，分散了你的注意力。 |
| 魔法 | 你的敌人故作姿态，念出一段咒文。 |
| 预备 | 你的敌人似乎正伺机待发。 |

你可以将叙述性描述与游戏规则相结合：“敌人没有发动攻击，而是执行疾走动作急匆匆地跑向对面。”

**施法中的怪物 Monsters Casting Spells**

玩家能判断出对手何时在施法这点非常重要，这让角色们有机会施展法术反制或干扰其施法。

当怪物施放法术时，检查法术所需成分并适当描述其活动。如果法术包含言语成分，怪物可能会吟唱、大声疾呼或嘶鸣出法术的神秘音节。姿势成分包括怪物以优雅的规律移动双手（或类似的附肢），摆出不规则的手势，或向前猛挥。最后，怪物可能会持握着施法法器或其他材料成分。

有些怪物具有特殊能力来无视咒语的某些或全部常规成分，这可能会导致角色无法认出怪物正在做什么。同样，当怪物使用不涉及施法的魔法性能力时，请确保玩家能够清楚认识到怪物正在使用其独特的魔法性能力，而不是正在施展角色们可以反制的法术。
