# 投掷先攻

投掷先攻 Rolling 
Initiative

战斗唯有在你宣布开始时才会展开。有些角色可以在先攻掷骰时触发能力，但是否投掷先攻和何时投掷先攻由你决定，而非玩家。因此，一位高等级的野蛮人不能直接给他的圣武士朋友来上一拳，然后通过投掷先攻来恢复已消耗的狂暴次数。

在任何情况下，如果是因角色的动作而引发了战斗，你可以给予进行该动作的角色一次先攻掷骰的优势。例如，因为术士确信眼前这位 NPC 是个变形怪所以决定中断与他的对话，并使用繁彩球法术指定了它，那么所有人都要投掷先攻，且术士的掷骰具有优势。如果变形怪掷骰的结果不错，它可能仍然会在术士施法之前行动，这反映了怪物对法术的预判能力。

**使用先攻定值 Using Initiative Score**

你可以使用先攻定值来代替掷骰，从而更快地进入战斗状态。您可以只让角色使用；只让怪物使用；也可以两边都使用。

- **角色的先攻定值 Initiative Scores for Characters**。角色的先攻定值通常是10加角色先攻掷骰的所有调整值（包括敏捷调整值和任何特殊调整值）。如果你希望你的玩家使用先攻定值，让他们在角色表上记录这些数值，并自己列一份先攻值的清单。
- **怪物的先攻定值 Initiative Scores for Monsters**。怪物图鉴中的怪物数据卡会包括先攻定值，列于其先攻加值之后。
- **优势和劣势 Advantage and Disadvantage**。如果生物在先攻掷骰时具有优势，则先攻定值增加5。如果生物在先攻掷骰时具有劣势，则先攻值减少5。
