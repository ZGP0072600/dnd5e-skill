# 察觉

察觉 Perception

作为 DM，你是玩家和游戏世界之间的接口。你得告诉他们角色所觉察到了什么，所以至关重要的是你要确保告诉了玩家他们角色周遭的重要信息。察觉技能和感知检定是你的关键工具。本节提供了帮助你使用《玩家手册》中察觉规则的指南。

**何时要求进行检定 When to Call For a Check**

当其他生物正在采取躲藏动作时，就是一个要求进行感知（察觉）检定的重要时间点。注意到躲藏的生物并不容易，但也绝非不可能，所以角色们总是可以尝试进行感知（察觉）检定来这么做。

- **使用被动察觉 Using Passive Perception**。有时，让玩家为角色进行感知（察觉）检定，会提醒玩家，让他们意识到应该去寻找什么东西，从给出一条你不希望他们知道的线索。这种情况下可以使用角色的被动察觉值来代替。
- **使用调查技能 Using the Investigation Skill**。调查技能适用于角色通过推理与演绎来对正在查验的事物推出结论的情况。角色试图弄清一件事物的运行原理时也适用调查技能--如何开启一扇机关门，如何进入秘密隔间等等。

不要使用调查技能来确定角色是否注意到某些东西——这是察觉技能的范畴。例如，成功通过感知（察觉）检定后，角色可以找到一扇密门或使其暴露的线索，例如门边细缝。如果密门被锁住了，成功通过智力（调查）检定后，角色可以找出开启它的方法——例如转动附近的雕像让其朝向门的方向。

**冒险中的隐藏事物 Hidden Things in Adventures**

密门、暗格、隐蔽陷阱和被藏匿的宝藏是冒险中的常见元素。如果有被隐藏起来的东西，使用这些元素时，请留出角色们可能发现不了它的余地。把额外宝藏或够意思的惊喜藏起来确实不错，但请不要将对角色们的成功必不可少的元素藏在它们可能找不到的地方。

即使隐藏物品对于冒险成功并非不可或缺，但也可以通过提示来让玩家了解到，可能有一些隐藏的东西等着他们去发现。这些提示可能很微妙（角色打开桌子抽屉时听到了奇怪的响声，暗示着抽屉后面或底部可能存在暗格），也可能很明显（清晰的脚印穿过房间，指向一堵空白的墙，那实际上是一扇暗门）。这些提示让玩家们能发现有趣的秘密，而无需花费大量时间在每个房间和走廊里地毯式搜索。

**察觉与遭遇 Perception and Encounters**

如果角色遇到另一群生物，且双方都没有隐匿行踪，那么当它们进入彼此的视力或听力范围时，就会自动发现对方。** 声音传播距离表 **可以帮助你确定听力范围，而后文将确定能见度。如果一群生物试图躲避另一方，则使用《玩家手册》中的规则。

**声音传播距离表 Audible Distance**

| 噪音 | 距离 |
|---|---|
| 尽量保持安静 | 2d6×5 尺 |
| 正常噪音等级 | 2d6×10 尺 |
| 非常响 | 2d6×50 尺 |

- **户外能见度 Visibility Outdoors**。在户外旅行时，除了有障碍物遮挡视线的地方，大多数角色在晴朗的日子里都能向任何方向看到大约2英里。如果身处山丘或高岗之上，或者能够从高处俯瞰下方区域，这个范围将增加至40英里。轻度遮蔽状态会降低能见度：下雨时最大能见度降低到1英里，而雾气则使其降低到100-300尺之间。 户外地形决定了角色遭遇其他生物的距离。旅行地形表（见下文“旅行”）给出了不同类型地形的建议遭遇距离。
- **海上能见度 Visibility at Sea**。在晴朗的天空和相对平静的海面上，站在船上的瞭望台上可以看见10英里外的事物。阴沉的天空会让能见度降低一半。如陆地上一般，轻度遮蔽状态会降低能见度。
- **水下能见度 Visibility Underwater**。水下能见度取决于水的清澈度和可用光照。使用水下遭遇距离表来确定水下遭遇的距离。

**水下遭遇距离表 Underwater Encounter Distance**

| 能见度 | 遭遇距离 |
|---|---|
| 清澈水体，明亮光照 | 60 尺 |
| 清澈水体，微光光照 | 30 尺 |
| 浑浊水体或黑暗 | 10 尺 |
