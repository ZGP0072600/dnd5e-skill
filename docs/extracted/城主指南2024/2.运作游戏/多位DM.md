# 多位DM

**多位 DM Multiple DMs**

许多游戏团队都会时不时地切换 DM。下节介绍了可供多位 DM 的情况，以及让多位 DM 为团队增添乐趣的方法。

**偶尔放假 Occasional Breaks**

如果你需要重振创作激情，为你的下一场战役构思剧情弧光，或刚完成你正着手的冒险，你可以暂时放下 DM 的身份，给自己放个假。当你休息时，也创造了让另一玩家担任一两场游戏回中的 DM 角色的机会。

如果不是所有人都能参加预定的游戏回，这也是个让不同的 DM 运作一场短冒险的契机。

**多样系列 Variety Series**

一些团队不想要剧情线跌宕起伏的长战役；它们更喜欢短程而不关联的冒险。此种风格的游戏中，不同玩家会在一至三次游戏回中轮流担任 DM，每次冒险都是一个独立的故事。

**同期战役 Concurrent Campaigns**

你和你团队里的其他 DM 在几周或几个月之间轮流运行冒险，且你的战役会在另一 DM 运作其他战役时暂停。一些团队每周都有多次游戏，由不同的 DM 在不同的日子里运行各自的战役。

**共享世界 Shared World**

一些团队会使用一个大型而完善的战役设定并将其按地理位置进行划分，以便不同的 DM 们能够在相同的设定下分开运作战役。理论上，角色们可以从其中一位 DM 的世界区域移动至另一位的，创造出战役的连续性，即便角色们可能会牵扯多条剧情线中。

比起以地理位置划分战役，你与团队中的其他 DM 们还可以用情节进行划分。使用本书第五章的设定作为例子，每个 DM 都可以在它们的战役中专注于该设定中的三个全局冲突之一。

**联合 DM Joint DMs**

两个或更多个 DM 可以共同创建单次战役，合作维持游戏回之间的连续性，并确保每个 DM 的冒险都能够推进世界和角色的宏观故事。当玩家同时是 DM 时，它们不应该让其对战役故事的了解去影响它们角色的行动。如果轮到某名玩家成为 DM，其角色亦会在此期间暂时退出游戏。

联合 DM 们也可以组队运作战役中的单次游戏回，其中每位 DM 都专注于它们最享受的游戏方面，或 DM 们的专注焦点也可以随着每次游戏回而变化。一位 DM 可能运作战斗中的描述来保持战斗运行，而另一位专注于微缩模型和音乐。两位 DM 可以在交涉遭遇中扮演两位不同的 NPC。在游戏回之间，他们可以合作或分工建设世界、创建遭遇和其他任务。
