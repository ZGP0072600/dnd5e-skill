# 即兴伤害

即兴伤害 Improvising Damage

**即兴伤害 ** 表为决定临时出现的伤害给出方针。

 

**即兴伤害 Improvising Damage**

| 伤害 | 例子 |
|---|---|
| 1d10 | 被炭烫伤，被倒下的书柜砸伤，被毒针刺伤 |
| 2d10 | 被闪电劈中，跌进火坑中 |
| 4d10 | 在坍塌的隧道里被落下的碎石砸中，滚入装满强酸的桶里 |
| 10d10 | 被紧实的墙壁碾压，被飞旋的钢铁利刃击中，淌过熔岩 |
| 18d10 | 淹没进熔岩里，被坠毁的飞行堡垒砸中 |
| 24d10 | 被卷入火元素位面的火焰涡流，被类神生物或和月亮一般大的怪物一口咬碎 |

**伤害危害度与等级 ** 表是一份关于不同等级的角色而言伤害总数有多致命的指南。交叉比对角色等级与即将收到的伤害来衡量伤害的危害度。

**伤害危害度与等级 Damage Severity And Level**

| 角色等级 | 妨害 | 致命 |
|---|---|---|
| 1-4 | 5(1d10) | 11(2d10) |
| 5-10 | 11(2d10) | 22(4d10) |
| 11-16 | 22(4d10) | 55(10d10) |
| 17-20 | 55(10d10) | 99(18d10) |

妨害型伤害很少在角色处于所示等级时将他们置于死地，但已被严重削弱的角色可能会被这次伤害所击溃。

致死型伤害对处于所示等级的角色构成了显著的威胁，且有可能杀死已经失去生命值的角色。
