# 即兴答复

即兴答复 Improvising Answers

只要稍加准备和大量的灵活运用，你就能应对玩家们朝你抛来的任何话题。

即兴剧的基石之一被称作“是，然后...”。这基于一种理念，即演员需接受与他对手戏的演员所演的任何东西，并以此为基准进行创作。当你为你的玩家们运行游戏回时，也可以应用类似的原则。尽可能多的将玩家们给予你的内容编入故事里。

一个同等重要的原则是“不，但是...”。有时角色无法完成玩家想做的事，但你可以提供另一选项来让游戏继续进行。

举个例子，想象一下角色们正在寻找某位巫妖的巢穴。一位玩家询问你附近城市里是否开张着魔法师工会，希望能找到任何提及这位巫妖的记录。这并非你已经预料到的一种可能，且你对此没有做任何准备。一个选项是说“可以”并用可使用的工具来创造一个合适的魔法师工会。通过这样做，你为玩家的奇思妙想给出奖励。同样的，该工会也可以作为冒险引子的绝佳来源。

另一选项是说“不行”，但镇中一个独居的魔法师可能手握所需的信息。这种方式在奖励了有创造力的玩家的同时也减少了你的工作量。

**即兴答复小帮手 Aids To Improvisation** 

当你需要临场编造出某些东西-比如，你先前从未计划过的，镇子中的魔法师工会-你可以利用这些丰富的资源：

- NPC 姓名列表（见第三章中“非玩家角色”）
- 随机表格（例如第三章中“聚落”一节中的表格）
- 战役日志（在第五章中所述）
- 地图（见附录 B）
