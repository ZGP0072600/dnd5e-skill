# 豁免检定

豁免检定 Saving Throws

与属性检定相反，豁免检定是对效应的即时反应，且几乎无法自愿进行。一次豁免检定在某些坏事威胁到角色，而角色有机会去避免或抵抗时进行则最为合理。

大多数时间下，豁免检定只会在某种效应(例如法术、怪物能力或陷阱）要求你为其进行豁免时才需要进行，且该效应亦会告知你需进行哪种豁免检定，并提供其 DC。

而在其他要求进行豁免检定的情况下，则由你来决定该豁免与哪个属性值相关。属性、属性检定和豁免检定表为你提供了建议。
