# 态度

态度 Attitude

DM 控制的每个生物有以下三种对冒险者的态度：友好 Friendly、冷漠 Indifferent 或敌对 Hostile。第四章的“
**怪物行为 **”部分提供帮助你确定生物的初始态度的指南。

 角色可以通过言行改变生物的态度。例如，为一群态度为冷漠的矿工买饮料可能会让他们态度变为友好。当态度发生变化时，请向玩家描述。例如，矿工可能会通过提供一些有用的信息、许诺日后报答善意、或邀请角色加入一场饮酒友谊赛来体现他们刚刚建立的友好关系。

**交涉中的属性检定 Abilitiy Checks in Social Interaction**

你可以决定属性检定会有多大程度上影响交涉的结果。一次简单的交涉可能只涉及简短的对话和一次魅力检定，而一次更复杂的遭遇可能需要多次属性检定来帮助引导对话的走向。

**使用协助动作 Using the Help Action**

当一个角色使用协助动作来协助另一个角色影响一个 NPC 或怪物时，鼓励进行协助角色的玩家参与对话，或至少描述一下他们的角色为了协助另一个角色取得成功正在做什么或说什么。
