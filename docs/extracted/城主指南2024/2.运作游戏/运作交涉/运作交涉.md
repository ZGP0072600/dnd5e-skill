# 运作交涉

**运作交涉 Running social interaction**

在交涉中，冒险者们通常都具有一个目的，比如想获取信息、获得援助、赢得他人信任、逃避惩罚、避免战斗、洽谈条约或实现其他目标。成功完成遭遇则意味着达成目的。

有些 DM 将交涉视为一种形式自由的角色扮演机会，而骰子很少在游戏中发挥作用；另外一些 DM 也会通过让角色进行一些魅力检定来解决问题。大多数游戏则介于两者之间，于角色扮演和偶尔的属性检定之间找到平衡。
