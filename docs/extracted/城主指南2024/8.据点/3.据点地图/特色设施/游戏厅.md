# 游戏厅

**游戏厅 Gaming Hall** *9级据点设施*

**先决条件 **：无

** 空间 **：庞大

** 雇员 **：4

** 指令 **：贸易

游戏厅提供娱乐活动，例如象棋、飞镖、卡牌或骰子。

**贸易：赌场 Gambling Hall** ：当你对此设施下达贸易指令时，设施雇员将游戏厅变为一个赌场，持续 7 天时间。在第七天结束时，掷 d100 
并根据下列表格来决定你作为庄家赚了多少钱。

| d100骰点 | 赢得的钱 |
|---|---|
| 01-50 | 1d6*10GP |
| 50-85 | 2d6*10GP |
| 86-95 | 4d6*10GP |
| 96-100 | 10d6*10GP |
