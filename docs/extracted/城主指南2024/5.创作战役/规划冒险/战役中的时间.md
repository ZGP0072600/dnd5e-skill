# 战役中的时间

**战役中的时间 Time in the Campaign**

D&D 战役中的大多数冲突需要数周或数月的世界时间来解决。一个典型的战役会在世界中的时间的一年内结束，除非你允许角色们在冒险之间享受更长时间的安静休息。

如果你不想跟进天数、周数和月份的流逝，则你可以改为跟进季节和季节性节日，来以此体现的流逝；诸如“这次冒险发生在什么时候？”这样的问题用“在冬天”或“在秋收节期间”来轻易回答。

## 定时事件 Timed Events

与一年中的某些特定时间相关联的超凡事件是绝佳的冒险机会。也许每到冬至，某座山丘上就会出现一座幽灵城堡；或者，每到第十三次满月时，升起的月亮都是血红色的，并令狼人的嗜血欲望大幅增强。一道彗星划过天空，可能预示着各种重大事件的发生。诸神的节日亦可以作为冒险的起点，尤其是在诸神本身参与到了故事之中的那些时候。
