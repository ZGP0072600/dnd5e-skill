# 冒险赞助者

**冒险赞助者 Adventure Patrons**

许多冒险都开始于一位赞助者请求角色们去接手一个请求或任务，并为他们的付出提供报酬。

花点时间去充实一个赞助者身份的 Npc 角色。偶尔，让角色们的赞助者背叛他们也很有意思。但是，如果你在一场战役中三番五次地滥用这种手法，也很可能导致玩家不愿再相信任何赞助者，并对你摆在他们面前的任何冒险引子都产生怀疑。

**赞助者引子表 ** 将提供一些可以让赞助者引导玩家进入一个冒险情景的建议。第五章的“开始战役”一节也提供了更多关于赞助者的建议。

**赞助者引子表 Patron Hooks**

| d6 | 引子 |
|---|---|
| 1 | 一个城镇布告员宣布有人正希望雇佣冒险者。 |
| 2 | 角色们想要示好或需要寻求帮助的对象要求他们处理一个冒险情景。 |
| 3 | 当角色们来到一个新城市时，他们找到一个招聘栏，上面发布着征集冒险者的消息。 |
| 4 | 一个了解到冒险者们的功绩的富有赞助者给他们写了一份信，愿意为他们的才能买单。 |
| 5 | 一个需要帮助的市民在了解到冒险者们的功绩和善举后，不远千里来找到他们，寻求他们的帮助。 |
| 6 | 冒险者们被捕（罪名可以成立或捏造），并有机会通过完成一项任务来逃脱责罚。 |
