# 宝藏主题

**宝藏主题 Treasure Themes**

正如《怪物图鉴》中所述，怪物们有着属于自己的宝藏偏好。这些偏好将会表现为“主题”，如宝藏主题表中所总结的那样，这可以帮助你决定角色们在怪物的宝库中找到了怎样的宝藏。

有关如何在冒险中安放宝藏的建议，见第四章“**冒险奖励 **”。

如果要随机决定作为宝藏发现的魔法物品，请使用本章末尾的表格。

 

** 宝藏主题表**

| 主题 | 对应宝藏 |
|---|---|
| 奥秘 | 宝石以及超自然或充盈魔能的魔法物品 |
| 武备 | 钱币或贸易金条以及在战中很有用的魔法物品 |
| 器具 | 钱币、贸易金条或商业货物以及侧重于实用性的魔法物品 |
| 圣物 | 艺术品以及具有宗教起源或归宿的魔法物品 |
