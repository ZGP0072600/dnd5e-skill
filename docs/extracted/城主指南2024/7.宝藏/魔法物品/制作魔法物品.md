# 制作魔法物品

**制作魔法物品 Crafting Magic Items**

玩家手册内含有关于酿造治疗药水与抄写法术卷轴的规则。而要制作其他魔法物品，则需遵循下述规则。在这些规则中，“你”指的是正在制作魔法物品的角色。

## 奥秘熟练项 Arcana Proficiency

若要制作魔法物品，则你与你的每位助手都必须具有奥秘技能的熟练。

## 工具 Tools

魔法物品工具表列出了制作每个类别的魔法物品所需的工具。你必须使用所需工具来制作物品且必须具有该工具的熟练。你的每位助手也必须具有该工具的熟练。有关工具的更多信息详见玩家手册。

**魔法物品工具 Magic Item Tools**

| 物品种类 | 所需工具 |
|---|---|
| 护甲 | 皮匠工具，铁匠工具，或者织布工具（根据工具描述中所注明的护甲种类进行选择） |
| 药剂 | 炼金工具或者草药工具 |
| 戒指 | 珠宝匠工具 |
| 权杖 | 木雕工具 |
| 卷轴 | 书法工具 |
| 法杖 | 木雕工具 |
| 魔杖 | 木雕工具 |
| 武器 | 皮匠工具，铁匠工具，或者木雕工具（根据工具描述中所注明的武器种类进行选择） |
| 奇物 | 修补工具或者作为魔法物品之基础的非魔法物品所需求的制作工具 |

## 法术 Spells

若某件魔法物品允许其使用者从该物品中施展任意法术，则你必须在制作该魔法物品的每一天都准备着所有的这些法术。

## 时间与成本 Time and Cost

制作魔法物品所花费的时间与成本均基于该物品的稀有度，如 **魔法物品制作时间与成本 ** 表所示。

**每日的工作 Work per Day** 
。你必须每日工作8小时以进行制作。如果一件魔法物品需要多日完成，则这些日期不必是连续的。

** 助手 Assistants**。角色们可以齐心协力来缩短制作工期。将制作某件魔法物品所需的时间除以参与制作该物品的角色数量。一般来说，只有一名角色可以为你提供帮助，但 DM 可能也会允许更多的助手参与进来。

** 原材料 Raw Materials** 
。该表内所述成本即代表制作一件魔法物品所需的原材料。由 DM 决定合适的原材料是否可以获取。在城市中，原材料将有75%的概率可以获取，而在任何其他聚居区中，该概率为25%。若材料无法获取，则你必须等待至少7天，然后才能再次勘察该材料的可获取性。

若某件魔法物品内含一件拥有购买价格的物品（比如一把武器或一套护甲），则你还必须支付该物品的完整费用或者使用玩家手册中所述的规则以制作魔法物品。例如，要制作一套 **+1护甲 **（板甲），则你必须花费3500gp，或者花费2000gp 并在一件板甲的基础上进行制作。

**魔法物品制作时间与成本 Magic Item Crafting Time and Cost**

| 稀有度 | 时间* | 成本* |
|---|---|---|
| 普通 | 5日 | 50gp |
| 非普通 | 10日 | 200gp |
| 珍稀 | 50日 | 2000gp |
| 极珍稀 | 125日 | 20000gp |
| 传奇 | 250日 | 100000gp |

scroll
