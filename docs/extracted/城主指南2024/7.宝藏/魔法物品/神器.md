# 神器

**神器 Artifacts**

神器是具有其独特起源与历史的，独一无二而世所罕见之强大的魔法物品。它可能被创造于某场威胁一个王国、一个世界甚至整个多元宇宙的灾难之中，并且在历史上的关键时刻曾发挥了相当大的作用。

某些神器会在人们最需要它时出现。不过对另一些神器来说，情况则正好相反，当这些神器被发掘出来之时，就连天地也因其现世而战栗。但无论是哪种情况，要将一件神器引入战役之前都必须三思。这件物品可能会是对立双方均希望占有的宝物，或者它将会是角色们克服最终挑战的必需品。

角色们通常不会在冒险的平常旅途中找到神器。事实上，神器只有在你想让它出现的时候才会出现，因为它们比起魔法物品更像是推动剧情的道具。追寻并找回某件神器往往是一场冒险的主要目的。角色们必须追索流言与传说，经历必要的试炼，闯入将要被遗忘的危险地域以找到他们苦苦寻求的神器。另一种情况下，某位著名的恶人已经拥有了该神器。取得神器并将其摧毁可能将是确保其力量不被邪恶利用的唯一方式。

## 神器的词条 Artifact Properties

除了神器描述中所述词条之外，一件神器还可能将拥有其他或减益或增益的词条。你可以在该小节所列出的表中选取这样的词条或者随机决定。你也可以发明一些新的增益或减益词条。每当某件神器重新出现于世间，其所具有的这些词条通常也会发生改变。

一件神器最多拥有4项次要增益词条与2项主要增益词条。该神器也最多拥有4项次要减益词条与2项主要减益词条。

| 31-40 | 在与该神器同调期间，你具有对魅惑状态与恐慌状态的免疫。 |
| 41-50 | 在与该神器同调期间，你具有对由 DM 所选择的一项伤害类型的抗性。 |
| 51-60 | 在与该神器同调期间，你可以从该神器中施展一道戏法（戏法由 DM 选择）。 |

| 91-00 | 在与该神器同调期间，你的护甲等级获得+1加值。 |
