# 传说

##### 召唤泰坦卷轴 Scroll of Titan Summoning

*卷轴，传说*

当你以魔法动作阅读此卷轴时，卷轴指定的特定泰坦会显现在你1里内的地面或水里的未占据空间中。由 DM 选择合适的泰坦或根据下表随机决定（生物的数据见怪物图鉴）。

泰坦对其他所有生物态度敌对，并将在生命值降至0时消失。若泰坦被召唤到不足以容纳它的空间中，则召唤失败且卷轴被浪费。

| d100 | 泰坦 |
|---|---|
| 01-15 | 兽主 Animal Lord （译注：在怪物图鉴2025中，兽主实际上并非泰坦） |
| 16-30 | 吞世黏浆 Blob of Annihilation |
| 31-45 | 巨神兵 Colossus |
| 46-60 | 元初天劫 Elemental Cataclysm |
| 61-75 | 至高天 Empyrean |
| 76-90 | 克拉肯 Kraken（召唤一只克拉肯需要足以容纳它躯体的水体，否则召唤失败且卷轴被浪费） |
| 91-00 | 泰拉斯奎 Tarrasque |
