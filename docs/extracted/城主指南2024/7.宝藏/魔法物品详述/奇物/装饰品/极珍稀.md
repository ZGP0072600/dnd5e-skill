# 极珍稀

##### 位面护符 Amulet of the planes

*奇物，极珍稀（需同调）*

佩戴此护符期间，你能够以一个 **魔法 ** 动作指名一个位于其他存在位面的你所熟悉的地点。那之后你进行一次 DC 15的智力（奥秘）检定。如检定通过，你施展位面转移 Plane Shift。如检定失败，你和位于你15内的每个生物与物件将被传送至一个由投掷 d100并根据下表决定的随机目的地。

| d100 | 目的地 |
|---|---|
| 01-60 | 与你指名地点同一位面的随机地点。 |
| 61-70 | 投掷1d6决定某内层位面中的随机地点：1气之位面；2土之位面；3火之位面；4水之位面；5妖精荒野；6堕影冥界。 |
| 71-80 | 投掷1d8决定某外层位面中的随机地点：1奔放之野；2世外桃源；3万兽乡；4双生天堂；5极乐境；6机械境；7天堂山；8伊斯加德。 |
| 81-90 | 投掷1d8决定某外层位面中的随机地点：1无底深渊；2修罗场；3卡瑟利；4焦炎火狱；5哈迪斯；6混沌海；7九层地狱；8喧癫空隧。 |
| 91-00 | 星界位面中的随机地点。 |
