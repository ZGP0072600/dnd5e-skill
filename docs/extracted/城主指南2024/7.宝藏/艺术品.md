# 艺术品

**艺术品 Art Objects**

纯金铸造的偶像，珠光宝气的项链，古代国王的肖像，镶满金玉的餐盘——艺术品包含这些所有，乃至更多。

若一份宝藏中包含艺术品，你可以根据下表基于艺术品的价值随机决定发现的艺术品类型。根据宝藏库中艺术品的数量，多次依照表格进行掷骰。同一件艺术品可能会重复出现。
