# 使用本书 Using This Book

本书是一扇通往多元宇宙冒险的大门，主要关注两道通往位面的门扉：印记城（被称为众门之城）以及外域（被视作通往外层位面的枢纽）。更多关于位面和其组织形式的内容，请查阅《**城主指南 **》。 

DM 可以自行决定和玩家分享多少本书的内容。物质位面的土著角色们对于本书提到的细节一无所知，而经验丰富的位面探索者则可能对本书所有内容了如指掌。 

这个引言为想要探索位面者提供了一份重要的概览。 

第一章为玩家提供创造异度风景战役角色的详细信息，将介绍门镇守卫和位面哲学家背景、位面相关专长以及适用于位面旅行者的法术和魔法物品。 

第二章介绍了令人费解的印记城、神秘的痛苦女士以及其他城市居民。 

第三章介绍了位于巨轮中心的中立位面——外域，以及通往外层位面的门扉——门镇，还有其他令人惊叹的界域的详情。
