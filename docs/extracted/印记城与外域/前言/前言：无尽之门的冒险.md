# 前言：无尽之门的冒险 Infinite Doors to Adventure

多元宇宙是一切已知与未知的事物的总和。世界、位面、生命和死亡……多元宇宙那无限的无限之中充满了无限的奇迹、恐惧和秘密——当然，其中最重要的是无限的可能性。每个 DND 冒险都发生于多元宇宙之中。在物质位面孤立的世界之外，还有无数其他的现实，以及连接着无尽时空的通路和传送门。那些寻求位面奇观的人，已经迈出了进入异度风景战役无尽可能性的第一步。
