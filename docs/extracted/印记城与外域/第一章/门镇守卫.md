# 门镇守卫 Gate Warden

*先决条件：异度风景战役*

你曾在受位面力量影响之地或是通往其他存在位面的传送门周围生活了许久，例如外域中的某个门镇。你早已习惯那些令人闻风丧胆的异界恐怖与引人入胜的异域风光，且对你来说，与那些天族们与邪魔们打交道，就如同跟门镇里的小贩打交道一样稀疏平常（虽然有可能那个小贩就是天族/邪魔）。

**技能：** 游说，求生

**语言：** 自选两门语言（推荐选择深渊语、天界语或炼狱语）

**装备：** 一轮不知道对应什么锁的钥匙串，一本空白的书籍，一根墨水笔或羽毛笔，一瓶黑墨水，一套旅行服装，以及一个装有10gp 的小包。

## 特性：位面注能 Planar Infusion

居住在门镇或类似门镇的地方使你长期浸泡在位面能量之下。你获得专长外层位面后裔（见本章后文）。此外，你熟知你成长的这座社区里哪里能找到免费的、俭朴的饮食与住宿。

## 建议特征 Suggested Characteristics

外层位面的影响塑造了你的看法，门镇守卫个性特点表向你建议了一些你可以为你的角色选用的个性特点。

**门镇守卫个性特点表 **

| d6 | 个性特点 |
|---|---|
| 1 | 我对那些稀奇古怪的事件与异界生物见怪不怪。 |
| 2 | 我觉得这才叫交易：一物换一物，空手换空手。 |
| 3 | 我以一种罕见的腔调说话。 |
| 4 | 我说话时总是带点其他位面的语言的单词或脏话。 |
| 5 | 我已经见过太多次了，以至于我明白靠脸看人靠不住，所以我会仔细端详每一个人。 |
| 6 | 我因我成长的门镇而获得了一项奇特的习性，像是在紧张的时候我会去摸摸铁器，或是非要把某些东西排顺不然就不舒服之类的。 |

## 创建门镇守卫角色 Building a Gate Warden Character

这些长期生活在通往另一个位面的永久传送门附近的人，会吸取到源自传送门彼端界域的精华。受到同一个位面影响的人，在行为举止与身体外表上总有着某些共同之处。 

**门镇守卫饰品 Gate Warden Trinkets。** 当你创建你的角色时，你可以在门镇守卫饰品表上进行投掷，而非在《玩家手册》中的饰品表上进行投掷。 

**门镇守卫饰品表 **

| d6 | 饰品 |
|---|---|
| 1 | 一个装着一滴蜂蜜的小瓶子吊坠，蜂蜜会发出微光。 |
| 2 | 一块小铅锭，上面压着一个奇怪的拇指印，紧握时会发出低语。 |
| 3 | 两个磁石球，当它们互相吸引时会发出鸣声。 |
| 4 | 一块闷燃的煤块，虽然它总是很烫，但它不会灼伤皮肤、毛皮、鳞片或衣物。 |
| 5 | 一根散发着5尺半径微光光照的羽毛。 |
| 6 | 一个用链环制成的戒指，一旦戴上后，不用拽到生疼的力气是摘不下来的。 |
