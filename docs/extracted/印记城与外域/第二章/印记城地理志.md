# 印记城地理志 Sigil Gazetteer

众门之城被划分为数个城区，其风貌如其居民般多样。从官僚体系森严的职员区高塔林立的楼宇，到底层印记城霉气冲天的无政府街巷。每个城区至少拥有一个派系的总部——那是印记城哲学家们聚集、划分城市职责的大型建筑。这些派系总部周边的场所，往往会呈现出该派系的性格。比如说，市民欢乐堂周围的商店就满足感觉团的享乐主义倾向——你会看到酒馆、音乐厅，以及贩售一次性新奇玩意的小贩摊位。

下方的印记城城区表概览了每个城区以及总部在对应城区的掌权派系，该表也标注了六个次要派系，这些次要派系被星号（*）标记。

**印记城城区 **

| 城区 Ward | 主要派系 Prominent Factions |
|---|---|
| 职员区 | 定命盟，咒法会*，感觉团 |
| 巢穴区 | 无望党，祸乱之手，灰烬先驱 |
| 女士区 | 末日守卫，秩序厅，和谐营，屠悯会 |
| 下层区 | 神明非神会，送戒人*，心灵之眼 |
| 市场区 | 自由联合会*，超然社 |
| 底层印记城 | 蛋糕社*，革命联盟*，无分会*（详见本章节后文“底层印记城”部分） |

接下来的章节将逐一介绍每个区域的细节，遵循以下格式：

**区域遭遇 Ward Encounters。** 
介绍冒险者在该区域内可能遭遇的事件或互动。

** 派系 Factions。**说明设于该区的派系与其职能运作。

** 地点 Locations。**列出该区著名地点，包括派系总部在内。所提及的地点在印记城海报地图中皆有标示。

** 巨型建筑 Megastructures。** 
多数区域会附上一座印记城巨型建筑的地图。这些巨型建筑常是派系总部，也能容纳无数冒险。地图仅为建筑内重要区域之概览，不代表其全貌。为配合冒险需要，这些地点的细节与比例皆可依需求进行调整。
