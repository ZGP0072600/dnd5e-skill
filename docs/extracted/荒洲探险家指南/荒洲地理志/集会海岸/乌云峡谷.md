# 乌云峡谷 Wuyun Gorge

这片干旱、岩石遍布的山谷将西里奥斯山脉和灰烬守护者山脉之间的南部通道隔开，被称为乌云峡谷，是乌云门的所在地，乌云门是敦达里安帝国和集会海岸之间的过渡点。穿过峡谷的道路通行频繁，全年都能看到往来不断的车队和商队。偶尔，游荡的生物、凶暴的巨人族和饥饿的怪物会从山上下来捕食过路的旅行者。因此，有钱人通常会在乌云峡谷两侧最近的定居点——尼科德拉纳斯或特罗斯滕瓦尔德雇用雇佣兵以确保安全。
