# 塞佩斯卡遗址 Ruins of Sepesca

在灾祸之战后的几个世纪里，基瑙人建立了塞佩斯卡，意为“自然的尽头“，这里是他们最神圣的墓葬神殿。神殿建在一片茂密的树林里，据说在人神诀别之前，荒野之母的血曾在这里洒落。每位逝去的基瑙祭司和国王都被安葬在塞佩斯卡宏伟的石质结构中。

卢什古特森林的地精和基瑙人之间的宿怨在几个世纪前达到了顶峰，当时地精们袭击了这个地区，亵渎了荒野之母的神殿。基瑙的守护者被迫逃离，无法抵挡格乌什信徒的侵袭，而废弃的圣地逐渐沦为废墟。据说复仇的幽灵和扭曲的怪物游荡在沉没的大厅里，守卫着毁灭者的忠实信徒留下的黑暗种子。

在集会海岸， 
仍然有少数基瑙人的后裔在寻找方法来净化这片圣地，并最终为被那遗忘的王族带来安息。
