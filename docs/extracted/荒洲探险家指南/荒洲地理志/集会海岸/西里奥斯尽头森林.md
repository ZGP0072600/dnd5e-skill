# 西里奥斯尽头森林 Cyrios End Forest

在西里奥斯山脉的南部边缘，是一片覆盖着针叶林和丛林植被的茂密林地。西里奥斯尽头森林提供用于制作优质纸张的木材，但采伐者通常只在外围活动。在错综复杂的小径和布满蕨类植物的地面深处，致命的生物在阴影中狩猎，而围绕着一座倒塌神庙的废弃的小屋和木屋则已崩塌，那座神庙如今大部分已被森林重新吞噬。冒险家和历史学家曾带领探险队进入这个基瑙村庄和神庙遗迹，但他们经常受伤或染病归来，甚至有些人根本没有回来。
