# 赫雷斯矿井 Herethis Mines

铂金既稀有又美丽，所以当在迪斯托克 Deastok 以东的朱乐斯自治领 Julous 
Dominion 境内发现了一种非常丰富的贵金属矿脉时，对这种经济来源的贪婪欲望最终促使帝国打破了他们的协议，征服了自治州。在王室的控制下，一个巨大的矿山已经扩展。以提炼这种极具价值的金属。然而，矿井靠近寄生在长青林的顽强的 Ankheg 种群仍然是一个挑战。此外，在过去的五十年里，万象会已经慢慢渗透到了矿山的运营中，定期从铂金矿中提取利，获取资金。
