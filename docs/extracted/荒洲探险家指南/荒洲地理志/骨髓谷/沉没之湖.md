# 沉没之湖 Ounterloch

Onterloch 布满苔藓的绿色水域在拉班达沼泽 Labenda 
Swamp 的中心，带着咸味和恶臭。这个湖是沼泽野兽和邪恶势力的家园，被埋在藻类和石头下的神秘秘密所吸引，这个湖是整个骨髓谷的民间神话的源泉。许多人谈到这座曾经坐落在北岸的寺庙，在灾祸之战所引发的地震之前，它沉入了黑暗的水中。另一些讲述了沼泽塞壬 Marsh 
Siren 的故事，她在梦中呼唤人们，并用她的歌将他们吸引到沼泽中。这些不幸的灵魂要么迷失在沼泽中，要么被她淹没在水底的怀抱里。 
然而，大多数讨论沉没之湖的人只是说它是多么的悲惨。
