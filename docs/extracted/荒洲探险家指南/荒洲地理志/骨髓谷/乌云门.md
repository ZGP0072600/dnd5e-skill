# 乌云门 Wuyun Gate

乌云门是 Dwendalian 
Empire 最南端的边界，坐落在乌云峡谷 Wuyun Gorge 中，在那里灰烬守护者峰 Ashkeeper 
Peaks 和西里奥斯 Cyrios 山脉相遇。这座雄伟的大门大约有20英尺高，15英尺宽，当旅客想要进出帝国时，大门就会打开。一堵25英尺高的石墙从大门的两侧继续延伸，一直延伸到附近的群山。城墙上有巡逻的王冠卫队 Crownsguard，他们的武器时刻准备着。它最初是作为军事防御工事建造的，以防止克洛维斯联邦(Clovis 
Concord)对帝国的攻击，现在是贸易路线上的一个航路点，当危险的生物开始远离附近的山脉时，它偶尔也会作为正义烙印 Righteous Brand 的前哨站。

乌云门的冒险 Wuyun Gate Adventures

角色可以在怪物攻击时帮助保卫乌云门前哨，并守住大门直到援军到来，或者他们可以在帝国沦陷后为帝国夺回大门。

收回大门（中级）Take Back The Gate(Mid 
Level)当角色穿越该地区时，他们需要通过大门。当他们到达哨所时，他们得知山里的怪物（鬣狗人 Gnolls, 大地精 Hobgoblins, 
兽人 Ores，或巨魔）谋杀了王冠守卫 Crownsguard，并声称大门是他们的。从这个安全的位置，他们向骨髓谷发动了更多的袭击。必须有人阻止他们，把大门夺回来。
