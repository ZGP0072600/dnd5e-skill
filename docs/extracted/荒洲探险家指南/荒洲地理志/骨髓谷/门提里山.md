# 门提里山 Mount Mentiri

门提里山是一片刺穿天空的深红色巨石，是特鲁斯坎山谷 Truscan 
Vale 中永远存在的视觉地标。缕缕细细的蒸汽从散布在这座大山各处的众多喷口中升起。附近的布罗姆基恩山 Bromkiln Hills 和普莱德集会 Pride' S 
Call 附近的北部土地已经被门提里下面的岩石放射出的元素力量所改变。

这些地下热量的来源对帝国人民来说是个谜。学者们提出了许多理论，认为门提里可能是一座年轻的火山——或者它位于火元素位面的裂缝之上。

他们的理论都不正确。事实是，暮光凤凰，阿斯蒙蒂斯的坐骑，被囚禁在山下的洞穴深处，她的监狱被火焰巨城乌索斯 Vuthos 包围着。自从卡摩达城建立以来，迪斯亚特 Desirat 的存在就一直影响着卡摩达的人民，她的讯息在梦中召唤着那些意志薄弱的人，敦促他们深入山底，把她当作神明来崇拜。
