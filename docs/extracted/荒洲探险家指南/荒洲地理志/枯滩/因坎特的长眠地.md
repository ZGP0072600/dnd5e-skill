# 因坎特的长眠地

因坎特的长眠地 Incanter's Rest

这个灾祸之战年代的古战场现在是一片巨大的墓场，不死的幽魂与不可名状的恐怖在这里以残忍和无休止的复仇肆虐。以把这个地方变成目前环境的无名法师来命名，因坎特之息除了三具巨大的泰坦骨骸以外，几乎没有其他特殊的地貌特征。巨大的骨架犹如塔楼伫立在草原上，呼啸的风在吹过这里的时候夹杂着幽灵的气息。四处散落着生锈的武器和铠甲，标记着堕落者们的坟墓。

一些历史学家认为因坎特之息其实是多个冲突的聚集点，而其他人声称这里是荒洲大陆上规模最大的战场。不管什么原因，因坎特之息到处是许多法师梦寐以求的强大死灵法术种子。
