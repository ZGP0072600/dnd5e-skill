# 灰败荒原 Greying Wildlands

在维南迪尔的东北海岸，是一片冰雪沙漠，锯齿状的敦洛克山脉和 Quannah 
Breach 将其与敦达里安帝国分隔开来。这片由浓密、干燥的森林、起伏的山脉和冰冷的苔原组成的土地被称为“灰败荒原”，它是凛寒北境 Biting 
North 的南部，包括这一地区和埃塞克罗斯 Eiselcross 岛。帝国一直在努力向该地区扩张，让其居民专注于自己在这片严酷的领土上的生存，游荡的外来者和野蛮的猎人在这片未受控制的荒野中潜行。乌索顿 Uthodurn 的矮人城堡隐藏在帝国的边界之外，穿过密密的、冰冻的树叶带来的阴影，这里成为了许多在莫莱斯迈 Molaesmyr 毁灭后幸存下来的精灵的家园。
