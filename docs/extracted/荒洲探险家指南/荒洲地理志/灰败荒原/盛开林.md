# 盛开林

盛开林 Blooming Grove

Shadycreek 
Run 西北方向近10英里处，在萨瓦里之森深处，有一片有几百年历史的柳树林和一座小寺庙，阻止了笼罩这片森林的魔咒。克雷 Clay 家族向荒野之母亲 Wild 
Mother 和至高之心 Arch 
Heart 宣告，他们建造了这个神圣的地方作为新生的殡葬林，旨在让堕落者的身体回归自然，埋藏在沼泽池塘和生机勃勃的树叶中的每一根血统都孕育着一朵完全独特的花朵，使该地区获得了骨果园 Bone 
Orchard 的可怕绰号。

然而，时间并不是静止的。森林的诅咒已经侵蚀了树林的神圣边界。深色的藤蔓和荆棘在栅栏和墓碑上一层一层地生长起来，把曾经广阔的小树林变成了寺庙底部一小块目中无人的绿色。只有一个看守人留下来照看他们的果园，现在这个名叫卡杜修斯 Caduceus 的牧师离开了，他希望能找到一个方法，以梅洛拉 Melora 的名义收回果园和它所保护的祖先。
