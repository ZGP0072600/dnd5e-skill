# 地之湖

地之湖 Erdeloch

地之湖 Erdeloch 是维南迪尔西部 Western Wynandir 最大的湖泊，经常被用作前往首都的旅行者的地标。 
其冰冷的水域是帝国最丰富的淡水捕鱼场所，埃尔德洛克南侧每天都能看到相当多的大大小小的渔船进出奥德斯洛 Odessloe。与该湖泊相交的众多河流使其成为帝国内河流运输的枢纽，尤其是对伐木者而言。北岸坐落在珍珠弓荒野 Pearlbow 
Wilderness 之中，以充满当地野生动物和土著怪物而闻名，因此很少有人敢在夜间在树线附近登陆。
