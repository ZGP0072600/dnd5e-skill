# 泽姆尼原野 Zemni Fields

沿着北方平原上霜冻斑驳的田野，Dwendalian 帝国的中心是由一大片泥泞的道路、茂密的森林和严酷的城镇组成的，这些城镇击退了荒野中危险的生物。这个地区被希尔伯奎尔山脊(Silberquel 
Ridge)的山脉与南部的骨髓谷(Marrow 
Valley)分开，以灰色的天空、阵阵的冷雨和偶尔的降雪而闻名，这让居民们的脚一直是泥泞的，他们的庄稼也很健康。褐岩山脉 Dunrock 
Mountains 保护着人民免受国王统治边界之外的威胁，东部的岩卫要塞 Rockguard Garrison 坚定地抵御着克灵王朝的致命军队和 Ashkeeper 
Peaks 山峰之外酝酿的危险。这里是强大的帝都 Rexxentrum 的所在地，是帝国权力和商业的中心。外交官们经常发现自己往返于 Zemni 
Fields，而来自世界各地的商人和商人则要穿越漫长的道路，希望在首都赚取财富。尽管如此，这个地区的野兽和生物越来越凶猛，与正在恢复的犯罪分子的窃窃私语和与克灵人不断升级的战争交织在一起，创造了一种永恒的紧张和焦虑气氛。
