# 诺格弗罗特

诺格弗罗特 Nogvurot

人口:15270(70%人类，17%矮人，8%精灵，5%其他种族)

政府:Starosta Brumae 
Luddie 在执法官的王冠卫队 Crownsguard 的支持下处理城市事务。斯达洛斯塔 Starosta 属下的贵族掌管地方行政。

防御:许多王冠卫队 Crownsguard 与一个被称为为“最后神盾”（Last 
Aegis）的正义烙印士兵团 Righteous Brand 共同承担保护职责。

商业:库存充足的贸易站和旅馆使大多数物资都很容易获得，但很难找到稀有或异国情调的商品。

组织:有帝国认可的神明的寺庙，大多数企业都是在没有公会监督的情况下独立经营的。

诺格伍罗 Nogvurot 是从首都向东的荣耀大道 Glory Run 
Road 上的一个主要站点，接待着穿越帝国的旅行者，他们中的许多人需要补给，并喜欢购买当地制造的小饰品作为旅途的纪念品。在城市北部的褐岩山 Dunrock 
Mountains 山脚下的一个小矿场里有一些生意可做，但大多数诺格伍罗人靠手工艺、耕种或加入当地军队来维持生计。

健康的防御 A Healthy Defense

这个定居点最初是作为 Quannah Breach 
附近的哨所而建立的，以监视从萨瓦里之森游荡的致命生物，该定居点后来发展成为正义烙印的一个相当大的军事储备。驻扎在城内的正义烙印团将成为抵御任何前往首都的攻击的防御部队，并授予他们“最后神盾”（Last 
Aegis）的称号。该市近一半的人口是后备军，在任何特定时间都有超过六千名士兵在场。无聊或焦躁不安的士兵为了消遣和娱乐，将神盾区 Aegis 
Block 的兵营发展成为一个体育和竞技游戏的场所，通常会在结果上投入大量的硬币。然而，在关于克灵人占领 Ashguard 
Garrison 的报道和关于克灵人在骨髓谷 Marrow Valley 内部发生无法解释的袭击的谣言之间，神盾对此非常紧张。

政府 Government

Starosta Brumae 
Luddie，一位正义烙印 Righteous 
Brand 的退役将军，已经守护了这座城市十五年。布鲁梅被强烈的责任感和对帝国的自豪感驱使，从不质疑国王的话。由于布鲁梅被大多数人视为公平公正的 
Starosta，一些人担心他的高龄将很快结束他的任期，并让不那么谨慎的人物争夺他的职位。

犯罪 Crime

诺格伍罗 Nogvurot 
内强大的军事力量将城市内的轻微犯罪降至最低。不幸的是，王冠卫队 Crownsguard 经常挺起胸膛，试图与正义烙印 Righteous 
Brand 的士兵单打独斗——一些当地的犯罪分子利用这种小规模的竞争来分散士兵的注意力，并继续他们的犯罪行为不受阻碍。腐败也成为一个问题，因为最后神盾 Last 
Aegis 的士兵利用他们在城市中的影响力，通过对进出 Shadycreek Run 和灰败荒地 Greying 
Wildlands 其他犯罪热点的走私视而不见来轻松赚钱。

地理 Geography

这座长方形的城市分为东部的神盾区 Aegis 
Block 和西部地区，东部地区是大部分驻扎士兵，西部地区被分为两个部分：北部的分散区 Scatters 和南部的小贩街 Peddler's 
Row。分散区 Scatters 是更大的区域，拥有小酒馆、简陋的寺庙和大多数平民，他们住在许多摇摇欲坠的住所社区。相比之下，小贩街 Peddler's Row 
是当地手工艺行会的繁华之家，也是荣耀大道 Glory Run Road 穿过城市的一个色彩缤纷但泥泞不堪的市场。

诺格伍罗的冒险 Nogvurot Adventures

角色可以去诺格伍罗寻找城里的某个人或加入正义烙印 Righteous 
Brand。对于希望在与训练有素的士兵的竞技游戏中测试自己勇气的冒险者来说，这座城市也是一个好地方。

谁是叛徒（低级）Traitors In The Midst (Low Level)。 特尔瓦 Telva 
Cornwrath（绝对中立，女性，人类间谍）接近角色们，并他声称为奥根信托 Augen Trust 工作。 她雇佣这些角色渗透到诺格伍罗 Nogvurot 
的正义烙印 Righteous Brand 的行列，并发现哪些士兵正在接受 暗影帮 Shadycreek Run 的贿赂。 
她想知道是哪个犯罪家族贿赂了每个腐败的官员。她可能是她声称的那个人，或者她可能是万象会 Myriad 的间谍，或者是暗流帮 Shadycreek Run 
中寻求消除竞争的帮派之一。 在后一种情况下，她可能不希望角色留在那里无所事事。
