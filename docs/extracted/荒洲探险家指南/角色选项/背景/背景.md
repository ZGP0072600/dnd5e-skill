# 背景 Background

本节介绍了两个新的背景——笑面人 Grinner 和沃什塔克特工 Volstrucker 
agent——以及对《玩家手册》中的一些背景进行调整的建议，以便将它们与荒洲的派系更紧密地联系在一起。

荒洲是一个充满秘密的地方，所以如果你的角色的背景对其他玩家保密的话，效果可能会更好——特别是如果你选择笑面人 Grinner 或者伏击者特工 Volstrucker 
Agent 背景时。与你的 DM 一起确定第二个假背景，可以帮助隐藏你的真实背景；或者一个伪装背景，如果你的真实背景涉及一个秘密身份，可以使用这个背景作为掩护。
