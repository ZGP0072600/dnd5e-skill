# 齿轮守护者Gearkeeper construct

这些由金属板和弯曲的刀片组成的旋转的、致命的旋风发条机械，最初是由修补匠和法师工程师设计的，目的是在 Age of Arcanum 充当地下金库最深处的哨兵和守卫。虽然这些宏伟而致命的古物大部分已经锈迹斑斑、破败不堪，连同它们长期被摧毁和掩埋的社会一起被人们遗忘，但仍有一些古物继续在被遗忘的坟墓和地牢的隧道和通道上巡逻，遵循它们最初的行动和指令来保护其领地内的物品。
一个齿轮守护者的外形类似于一个滚动的像盾牌一样的板球，可以快速通过走廊来寻找入侵者。当发现一个未被识别的生物时，护盾会展开，露出许多金属腿的缝隙，以便精确移动。带刺的附体在建筑周围嗖嗖作响，威胁着那些离得太近的倒霉的挖洞者，而沉重的钢矛可以从装置的核心部位射出，刺穿不受欢迎的客人。如果不堪重负，这种结构可以喷出一股加热的、锯齿状的金属碎片，以制服成群的入侵者。

从回收的外壳和挖出的齿轮保护装置的碎片逆向工程，许多现代修补者开发了这些哨兵的更新变种。这种设计直到最近才在一些特定的监狱中使用，或者被那些想要保护自己藏品安全的富有的收藏者使用。

构装本质：守护者无需空气、饮食和睡眠。
