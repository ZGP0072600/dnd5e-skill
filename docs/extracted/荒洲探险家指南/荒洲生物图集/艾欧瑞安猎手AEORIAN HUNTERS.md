# 艾欧瑞安猎手AEORIAN HUNTERS

在奥法纪期间，飞行之城艾欧尔的法师们创造出来艾欧瑞安猎手，用于对付天使、恶魔、魔鬼、妖精以及神灵本身。当艾欧尔于艾泽库罗斯（Eiselcross）坠毁之时，许多这些突变的怪物幸存下来，而如今它们在艾欧尔的废墟和群岛上的冰封废土上游荡。
自实验而生 Created by Experimentation.艾欧瑞安猎手们通过对野兽和类人生物——俘虏与志愿者皆有，进行的实验创造。这些实验导致怪物们具有色泽鲜艳的血肉和抵抗并毁灭艾欧尔之敌的力量。理论上讲，艾欧尔法师们必然有某种能控制他们的怪兽士兵的方法，但在艾泽尔库罗斯的荒原中并没有发现过这样的装置。龙语是这些法师们训练和约束艾欧瑞安猎手们的语言，即使猎手们口不能言，它们依然能理解这一语言。

不朽的疯狂 Madness through Immortality.艾欧瑞安猎手们不会衰老。因此，它们的创造者灌输给它们的暴力欲望在数个世纪没有战争之后变得贪婪。大多数猎手会径直攻击任何它们看到的生物。

永不相残 Never Harm Another Hunter.虽然它们渴求暴力，但艾欧瑞安猎手们从未被看到有攻击彼此的行为。考虑到创造每个猎手所需的黄金和魔法成本，大多数学者相信一定是艾欧尔的法师们将这些行为加诸于它们。艾泽库罗斯的一些冒险者吹嘘说，他们通过伪装和幻术模仿了猎手的外形，以从艾泽库罗斯的猎手身边溜走，但这些都很可能只是荒诞不经的故事。

永恒记忆 Ageless Memories.每一个艾欧瑞安猎手都能完美回忆起自其诞生以来的每一次经历。它们有限的智力和暴力欲望阻止了他们有效的交流，但是每一个猎手都拥有丰富的知识，通常包括在其头脑中珍稀的艾欧瑞安遗迹的位置。

完美士兵 Perfect Soldiers.艾欧瑞安猎手们不需要进食和饮水。

艾欧瑞安汲能兽 AEORIAN ABSORBER
从远处看起来，汲能兽类似于犬科或猫科的四足动物，当它靠近时，其怪物本质则显而易见。汲能兽以能将魔法能量吸收到身体中而得名，它能够通过尾巴释放出毁灭性的能量，而尾端的眼睛则时刻盯紧猎物。

伤害：22（4d10）心灵伤害。

艾欧瑞安灭法者 AEORIAN
NULLIFIIER
灭法者是所有魔法使用者的克星，因为其肉体形态和内在力量都是为了抵抗并消除魔法效应而塑造。在对付其他敌人时，灭法者同样可怕，其浑身覆盖着长满锋利尖牙的怪异大嘴，从中发出刺耳的声音可以让其他生物趋于疯狂。

1-4：该生物直到回合结束被震慑。
5-6：该生物直到回合结束陷入恐慌并使用其移动力尽可能地远离灭法者。
7-8：该生物无法移动，并使用其动作对一个随机生物（除了自己之外）进行一次近战攻击，若其触及范围内没有目标，则它什么都不做。

法术反制 counterspell. 灭法者试图打断一个生物正在施展的一个法术。如果该生物正在施展3环或更低环阶法术，则其法术失败且不产生任何效应。如果该生物正在施展4环或更高环阶法术，则灭法者进行一次魅力检定，其检定 DC 等于10＋目标法术的环阶。检定成功则目标法术失败且不产生任何效应。
