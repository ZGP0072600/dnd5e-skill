# 石化海蛇Swavain  Basilisk

采珠者和海底拾荒者有时会讲一些关于神秘的海底花园的故事，里面充斥着海洋生物和水手的雕像——那是致命的石化海蛇的水下洞穴。尽管以最常见的 Swavain Islands 群岛命名，但这些危险的捕猎者经常出没于内陆的水道，甚至有人在地下下水道系统中发现过它们。

死亡的黏滑之拥 Death 's Slimy Embrace。 石化海蛇在外形与陆地上的表亲不同，是一种细长的海蛇，通常长到25英尺。 它弯曲的身体让它在水中快速移动，并同样轻松地捕食猎物。 与生活在陆地上的同类的致命凝视不同，海蛇分泌出一种稠密的油，可以把其他生物变成石头。 被石化的猎物沉到海底，海蛇在那里悠闲地进食，一旦被吞噬，该部分就会恢复成血肉。 如果在经验丰富的炼金术士的监督下进行正确的加工，海蛇分泌的油可用于逆转其引起的石化作用。

海难拾遗 Shipwreck Steward。石化海蛇偶尔会在夜幕的掩护下在岸上捕猎，但更多时候，它们会被吸引到海难地点附近狩猎和筑巢。 当打捞者和寻宝者为了利润而寻找沉船的残骸时，海蛇会为它的收藏而增加更多沉溺的石化者。

水陆两栖 Amphibious。海蛇可以在空气和水中呼吸。

石化粘液 Petrifying Secretions。如果生物在距离海蛇5尺范围内，使用武器进行攻击命中海蛇，或者在其回合开始时被海蛇擒抱，那么该生物必须进行 DC 13体质豁免。除非豁免成功，否则该生物将开始魔法般的变成石头，并受到束缚，并且该生物必须在自己的下一回合结束时重复豁免。若豁免成功，则效果结束。若豁免失败失败，该生物就会被石化。
