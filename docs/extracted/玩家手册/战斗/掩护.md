# 掩护 Cover

墙壁、柱子、大树等掩体可以在战斗中提供掩护，使其后方的目标更难以受伤害。只有当攻击或效应源自掩体对面时目标才能够从掩护中得益。掩护有三个级别。如果目标获得多个掩体的掩护，则只按其中级别最高的掩护作计算，且掩护的级别不能累加。例如，一名同时在某生物和一棵树干后面的目标从生物处获得半身掩护，也从树干获得四分之三掩护，则此目标算作处于四分之三掩护。处于半身掩护 half 
cover 下的目标其 AC 和敏捷豁免视为具有 +2加值。如果掩体挡住了目标身体的一半或更多，即目标处于半身掩护。掩体可能是矮墙、大件家具、小树干、其他生物（不论敌友）。处于四分之三掩护 three-quarters 
cover 下的目标其 AC 和敏捷豁免视为具有 +5加值。当掩体挡住目标大约四分之三的身体，即目标处于四分之三掩护。掩体可能是栏栅，箭台开孔，或粗树干。处于全身掩护 total 
cover 下的目标无法被直接作为攻击或法术的目标，但很多范围性法术可以影响到他们。如果掩体完全的挡住了目标，即目标处于全身掩护。
