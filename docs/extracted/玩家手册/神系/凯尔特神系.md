# 凯尔特神系

凯尔特神系 The 
Celtic Pantheon

据说每个灵魂深处都潜藏着野性，包括夜里}人的雁叫，还有穿过松林的飒飒风声，以及莫名出现在一颗橡树上的红色槲寄生，而这一切正是凯尔特诸神的居所。祂们从溪流中喷涌而出，因橡树的力量而变得强大，因林地与阔野而变得美丽。在林地居民们为树干上呈现的面容，或是溪流发出的耳语呼唤出其圣命名时，正是这些神使自身在现世显身的一刻。 德鲁伊常常如同牧师一样侍奉凯尔特诸神，祂们与德鲁伊崇拜的自然力量密切相关。
