# 非人类神祇

非人类神 Nohuman 
Deities

一些与非人类种族息息相关的神在许多不同的世界被崇拜着，即使不总是通过同一种方式。而被遗忘国度和灰鹰的非人类种族则信仰相同的神。 非人类种族往往有自己完整的神系。例如矮人神系包括摩拉丁的妻子，蓓伦妮·真银 Berronat 
Truesilver，以及其他一些被视为其子嗣的神：如阿巴索 Abbathor，克兰贾汀·银须 Clangeddin 
Silverbeard，杜马松 Dumathoin，杜格马睿·明罩 Dugmaren 
Brightmantle，贡姆·古辛 Gorm 
Gulthyn，海·明斧 Haela 
Brightaxe，马萨摩·杜因 Marthammot 
Duin，萨德·哈尔 Thard 
Harr 和维加丁 Vergadain。具体的矮人氏族和王国可能会崇拜其中一些，全部，或者都不崇拜。他们也会崇拜外族一些未知的神灵，或某些已知却使用了其他名字的神灵。
