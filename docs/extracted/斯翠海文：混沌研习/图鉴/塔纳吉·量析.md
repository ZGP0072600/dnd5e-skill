# 塔纳吉·量析

**塔纳吉·量析 Tanazir Quandrix**

巨龙塔纳吉量析是统治物理现实和理论抽象的最强大的魔法大师之一。通过这些知识，她可以改变存在的物理属性，获得对基本法则的理解，并操纵思想的流动。

塔纳吉创建了量析学院，以培养那些追求知识的人的好奇心。其目标是训练那些以描述和控制现实本质的数学原理为指导，追求知识本身的法师。

塔纳吉的法术、传奇动作和吐息武器展现了几何形光的明亮图案。它们有各种形态，比如围绕着法术目标的连锁牢笼，或者在她的吐息区域内无限复制的分形漩涡。
