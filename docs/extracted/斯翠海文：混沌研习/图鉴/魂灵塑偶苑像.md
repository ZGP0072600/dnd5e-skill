# 魂灵塑偶苑像

**魂灵塑偶苑像 Spirit Statue Mascot**

为了帮助他们对过去的研究，衡鉴的学生经常召唤死亡生物的灵魂。从古代灵魂那里寻求长期帮助的魔法师已经学会了召唤魂灵塑偶来安置它们，给它们一个更长久的躯壳来帮助生者。

其中一些魂灵塑偶只与一个特定的法师有合作，但大多数为各种不同的个人服务，从调察时的同事到作为军事战术家。无论如何，它们的出现在衡鉴校园里非常普遍，以至于它们已经成为了学院的苑像。
