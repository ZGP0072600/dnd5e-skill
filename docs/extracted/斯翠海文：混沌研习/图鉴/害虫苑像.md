# 害虫苑像

**害虫苑像 Pest Mascot**

脾气暴躁的、带有尖刺的、通常令人讨厌的害虫苑像栖息在潮湿的衰草原河口。然而，这些雪貂大小的沼泽生物的生命本质却是极其深邃的生命源泉，使它们成为靡华法师魔法的方便的燃料来源。因此，许多靡华的学生把河口的害虫作为宠物，而这些的多刺的害虫，荣耀地成为了学院的苑像。
