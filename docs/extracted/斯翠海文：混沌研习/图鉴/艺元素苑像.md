# 艺元素苑像

**艺元素苑像 Art Elemental Mascot**

通常被称为一种“有生命的艺术作品”，艺元素是创造力和情感的一种物质表达——这一特点使它成为粹丽学院的完美苑像。

要创造一个艺元素苑像，法师需从混合的自然元素中汲取力量，提取它们充满活力的颜色，制造一个旋转的外壳，然后向其核心注入一个平面的精神。作为精确的元素研究和叛逆的情感的共同产物，艺元素既能破坏元素，也能在情感上吸引周围的生物。
