# 驮宝惰兽

**驮宝惰兽 Relic Sloth**

在长时间的探险中，衡鉴法师经常使用驮宝惰兽来运输冒险设备和挖掘出来的文物——只要他们的研究不是特别匆忙。驮宝惰兽以极其缓慢的速度前进，仔细考虑每一步，只有在必要时才会移动。但这种谨慎使这些笨重的生物非常适合穿越危险和摇摇欲坠的废墟，而其他擅长负重的野兽面对这些危境则会动摇。驮宝惰兽前肢上独特的镰刀状爪子主要用于攀爬，但是它们在必要时也可以抵御饥饿的捕食者。
