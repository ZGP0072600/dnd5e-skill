# 社团活动增益

**社团活动增益 Extracurricular Benefits**

当学生参加课外社团时，他们会获得一个学生骰和一个关系点，如下所述。

**学生骰 **

一个角色每参加一门课外社团，都会获得一枚 d4 
学生骰。玩家可以在该课外社团中列出的属性检定中，添加学生骰的掷骰结果。玩家可以在骰完属性检定的 d20 之后再使用学生骰，但必须在 DM 
判定检定结果之前使用。

每次检定至多只能使用一个学生骰。在使用一个学生骰之后，该玩家不能再使用学生骰直到角色完成一次长休。

**关系点数 **

当一名学生参加课外社团时，该学生与一名同时也参加该课外社团的学生 NPC 
获得关系点数，该关系点是正或负由玩家自行选择。（参见人际关系规则）。
