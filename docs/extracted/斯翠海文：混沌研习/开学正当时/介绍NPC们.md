# 介绍NPC们

**介绍 NPC 们 Introducing the NPCs**

你可以使用下文“同学”章节的内容填充斯翠海文——无论是冒险之中还是在两场冒险之间。在角色进行工作或者课外社团时强调他们同事/团员 
NPC 的存在，也可以在课堂场景中加入任意的你想扮演的 NPC。

在进行社交遭遇时，你可以和玩家预先商量，他们是否对某个特定的 NPC 
更感兴趣，并可以从而安排那个 NPC 在遭遇中和玩家进行互动。或者当角色进行工作和社团活动时，就可以和在干同样事情的 NPC 
进行遭遇。

你也可以允许玩家阅览“同学”章节来更多的了解那些 NPC。
