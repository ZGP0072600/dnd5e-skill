# 斯翠海文背景

**斯翠海文背景 Strixhaven Backgrounds**

本节提供 5 
个背景选项，选择这些背景，用以反映你的角色为进入斯翠海文特定学院所做的学习准备。在斯翠海文战役中你能选择的背景不受限于这 5 
个；这些背景只是你能选择的一种方式，表示你的角色深入了解其所属的学院。

如果你选择了一项其中的背景，你可以认为到现在这个节点之前，你的人生都在学习，为能入学斯翠海文做准备，你可能对除了预备功课之外的世界都不太了解。你做好胜任学生这个身份的准备了，作为回报，这也会让你驰骋毕业之后踏上的任何道途。
