# 巨人与魔法

巨人与魔法 Giants and Magic

巨人与魔法的关系有点暧昧。表面上最接近魔法的巨人种族是云巨人，风暴巨人紧随其后。这两种巨人天生可以使用某些形式的魔法如涉及空气、天气和重力的魔法之类。此外，还有极少数的巨人会以与人类、矮人和精灵等相似的方式学习魔法。奥术学识本身不被序位所认可，它既不 maug 也不 maat。但花时间掌握魔法的奥秘某种程度上需要巨人牺牲提升序位的其他机会，因此很少有个体选择这条道路。而符文魔法却是个例外。巨人着迷于永恒固定的魔法符文。石巨人是优秀的符文雕刻家，其原因既来自其艺术性追求，还因为他们所在的环境非常适合使用这些符文。任何种类的巨人里都能找到少量的 skiltgravr（“符文雕刻者”），哪怕迟钝的山丘巨人中也有个别个体可以在山腰上砸出印记，或者在自己身体上刻下符文。创作这种魔法非常费神而缓慢。你可以将其比作一个法师制作一张魔法卷轴，还要将便捷的羊皮纸和油墨换成石头和凿子 ，冰川和斧头，或者钢铁和熔炉。将一个魔法符文刻到一个物品上可以为其灌注力量。跟其他魔法物品一样，它可以用来激活一种或更多的魔法效应。魔法符文还可以书写于某种表面以创造出魔法效应，其效果类似于法术守护结界 Glyph of Warding 或者徽记术 symbol。符文本身决定了物品或表面所蕴含的魔法。例如，雕刻在一颗石头上的一枚风暴符文可以让石头的所有者控制天气。同样的符文雕刻在门或者箱子上可以在任何人开启它们时造成雷鸣伤害。
