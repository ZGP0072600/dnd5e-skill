# 巨人的行囊

| 巨人的行囊 A Giant's Bag |
|---|

奔波中的巨人经常在肩上挂着一个麻包。这些巨人行囊的主要作用就是携带粮食。由于要填饱一个巨大的胃囊（特别是在山丘巨人的状况），旅行中的巨人不随身带着点补给品实在不够明智 。巨人还会在行囊中带上一些岩石：有些用来战斗，有些用来狩猎 ，还会有一两块样子独特的用来作游戏。此外，提袋还可能装着其他东西 ：工具、纪念品、交易品或者只是该巨人随身携带的珍玩。一些可能包含在内的东西如下：·一只活猪·三张熊皮·包在染血斗篷里的长剑（用作小刀）·小桶麦酒·关在笼子里的半身人（用作消遣）·装满窗玻璃碎片的箱子·装满硬币的人类背包·一只枭熊的头骨·藤蔓绑起来的一大捆干木柴·一条曾经精美的挂毯，被用作手帕而变脏了·四个大致完整的斗车轮子·一块墓碑（用来渡河）
