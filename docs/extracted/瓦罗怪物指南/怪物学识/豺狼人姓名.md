# 豺狼人姓名

鬣狗人姓名 Gnoll Names

对于那些语言不过只是哀嚎，咆哮和尖叫的生物来说，大多数鬣狗人都没有名字，姓名也没什么用处。强大的鬣狗人，通常是毒牙，头领和弗林德，这些直接从耶诺古获得的名字。同样地情况也适用于耶诺古在人类、兽人和其他种族中眷顾的追随者。鬣狗人姓名
