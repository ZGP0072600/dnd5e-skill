# 扮演豺狼人和耶诺古的信徒

扮演鬣狗人和耶诺古信徒

鬣狗人们在性格和观点上几乎没有变化。他们都是被恶魔领主驱使着传播死亡和毁灭的强大武力。

与鬣狗人互动的唯一真正机会是那些有时跟随着一支鬣狗人战团的邪教徒。这些类人暴徒可能拥有人物所需的信息，他们甚至可能是被对耶诺古的崇拜所腐化的曾经的友人。如果要描绘一个比正常大多数更聪明或更世俗化的鬣狗人，你可以赋予他类似于耶诺古信徒的特征。

鬣狗人/邪教徒信念

| D6 | 信念 |
|---|---|
| 1 | 力量 Strength。 我必须保持强大才能生存。（任意） |
| 2 | 屠杀 Slaughter。 如果我消灭了弱者，我就能让耶诺古满意。（邪恶） |
| 3 | 毁灭 Destruction。 当这里只剩下那些值得他愤怒的事物时，耶诺古就会重回大地。（邪恶） |
| 4 | 猜疑 Paranoia。 其他人在计划杀了并吃掉我。我必须找到方法先把他们杀了吃掉。（混乱） |
| 5 | 傲慢 Self-Sufficiency。 当时机一到，我的盟友也会死在我的手里。（邪恶） |
| 6 | 领导 Leadership。 我不是这些家伙中的一员。我，高于他们。（混乱） |

| 4 | 耶诺古的预示指导着我的每一个选择。 |
