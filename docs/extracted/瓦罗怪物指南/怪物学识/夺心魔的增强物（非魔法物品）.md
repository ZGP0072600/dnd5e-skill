# 夺心魔的增强物（非魔法物品）

夺心魔的增强物 Mind Flayer Augmentation

一些夺心魔社群使用非魔法装备来增强它们的奴仆，让他们更好地担当巢穴的守卫和卫兵。这里给出了两个夺心魔增强物的例子。

剥皮爪 Flensing Claws

灵吸怪并不总是给他们的奴隶提供普通的武器，比如剑和斧头。有时它们通过赋予奴仆新的身体结构来提高奴仆的自然能力。剥皮爪呈关节状，延展成长长的金属刀片。这些爪子被固定在生物手臂的血肉和骨头里，如果不进行手术切除，就无法取出。

每一套剥皮爪都是为特定的生物设计的，其他人不能使用。一个装备有剥皮爪的生物可以消耗动作用爪子进行一次近战武器攻击。爪子根据生物体型的大小造成挥砍伤害:小型，1d8；中型，1d10；大型，1d12；或者巨型，2d8。该生物可以将其熟练加值和力量调整值添加到任何用爪子发动的攻击检定上，并将其力量调整值加到用爪子命中目标时造成的伤害骰上。微型和超巨型的生物不能装上剥皮爪。

生存覆甲 Survival Mantle

这种甲壳状的增强物包裹着佩戴者的肩膀、脖子和胸部。一件生存覆甲相当于一套非魔法的半身板甲，并且需要同样的时间才能穿上或脱下。它不能和任何其他护甲同时佩戴。

穿着生存覆甲的生物可以在任何环境下正常呼吸(包括真空)，并且在进行对抗有害气体(例如由死云术、臭云术、吸入性毒药和某些龙的喷吐武器所产生的气体)的豁免时具有优势。
