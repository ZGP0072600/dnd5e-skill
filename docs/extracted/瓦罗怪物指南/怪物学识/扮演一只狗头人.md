# 扮演一只狗头人

扮演一只狗头人 Role-playing a Kobold

狗头人清楚自己在充满敌意的世界中能力低弱的现实。它明白自己地位卑微，会遭到更大生物的剥削。它很可能在年轻时就死去，而其一生大部分时间都将用于艰辛的工作。尽管狗头人的前景似乎一片暗淡，但依然有着能够令其获得满足的事情，比如自己的工作，或是自己部族的幸存，又或是当分享到先祖强大的龙族遗留下的知识时。狗头人不算聪明，但又不像兽人那么愚蠢。某人可以用好话或者小聪明愚弄一只狗头人，但当这只狗头人意识到自己被戏弄后，就会一直记得被羞辱的事实。而一旦它找到机会就必定会对戏弄者施以报复，而哪怕最后只是以某种可怜又小气的方式实现它也可以接受。狗头人不喜欢被逼到角落或者孤单一人。它会需要一个自认安全的逃生通道，或者至少需要有个盟友在旁以增加其逃走的几率。当这两个条件都无法满足时，狗头人就会变得焦虑不安，其行为也会不断的在沉默与歇斯底里间频繁转换。
