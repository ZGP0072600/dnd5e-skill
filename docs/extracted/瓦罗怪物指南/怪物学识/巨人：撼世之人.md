# 巨人：撼世之人

巨人：撼世之人 Giant：World Shakers

| 根据我的经验，任何崛起的王朝最终都将走向没落，最后则只剩下断壁残垣。——瓦罗 |
|---|

巨人族的英雄传说早在世界的太古时期就已在传颂。彼时，精灵们还正优雅的踱步走出妖精领域，龙族也还没意识到日后自己将拥有的力量与荣誉，而巨人脚步的惊雷却早已撼动世界的骨脊。随着人类族群踏上历史舞台，那个古老时代的记忆早已消散在传奇的迷雾中。现今就连巨人们也并不清楚本族起始时期的全部真相。所有种类的巨人及其近亲都明确的认识到互相间近亲种族的关系。类人生物如精灵、人类和矮人彼此的身形尺寸，比不同种类的巨人彼此的身形尺寸更加相近 ，但这些种族并没有共同的传承。然而不管哪个种类的真巨人都可以追溯其血统至众神之父安南 Annam the All-Father 。大多数巨人还坚信，安南选择了包括其妻子欧缇雅 Othea 在内的数名配偶，以诞下不同相貌与能力的巨人诸族。巨人及其近亲跻身于世界最可怕生物的行列，恰如其名的将年轻种族们踩在脚下。时至现代，大多巨人都已隐居到世界上不为人知的角落里，而其祖先从前恢弘的团结力量已经不复存在。
