# 眼魔：噩梦成真

眼魔：噩梦成真 
Beholders：Bad Dreams Come True

| 你们费心编排过阴险狡猾的诡计吗？又或者尝试安排那种备好后续计划和撤退地点，精心策划的阴谋？而眼魔们手中所掌握的同类算计都是在无数的取舍与反复中不断改良和精进的成果。与之相比，人类的这些精心设计在最普通的眼魔看来都还只是儿戏。——伊尔明斯特 |
|---|

很多人为打败眼魔或其他目的对其进行研究，而与之相关的所有信息却都诡异莫测。这些奇异的生物拥有异常高的智商以及超越人类的感知能力，而且甚至能以其意志直接改变现实——有时其存在本事就足以做到这点。凭着绝伦的智力和强横的眼波射线，一只身处其地下巢穴合适范围内的眼魔几乎无懈可击。眼魔表现出的某些行为动机跟人类和其他智慧生物相仿，只是程度有所不同。比如，某个孤高自傲的人类可能会受极大的威胁，而自大如一只眼魔则只笃信一条定理：即其自身是所有生物的顶点，甚至凌驾于其他眼魔之上。人类棋手因能看穿对手未来的棋步而成为一名大师——而这远不及一只眼魔凭借其超高智慧和警觉所能达到的程度。眼魔的意志无比强大，而其敏锐性足以预想到几乎所有可能发生的状况从而为之做好万全准备。因此，没有入侵者能够把握抓住它的机会。眼魔的思维方式好比一种偏执——一种妄想的最极端形式。类比如人类暴君察觉某些隐藏威胁的蛛丝马迹，眼魔的偏执甚至涉及其察觉到的所有东西，也即是其天性使然的无休止的警觉性。眼魔是可以直接影响身边存在事实的少数生物之一。眼魔们在休息时并不会真正的陷入沉睡，而是保持一种意识半朦胧的状态。然后，在非常罕见的情况下，当一只眼魔梦见另一只眼魔时，其梦境将使现实扭曲并塑造一个现实的存在然后演变成另一个真实的眼魔。这并不是简单的复制，而大多数情况下新旧两只眼魔还会进行一场殊死搏斗——这对于全世界而言还真是件值得感恩的事。
