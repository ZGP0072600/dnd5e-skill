# 语言

语言 Langue

鬣狗人的语言，十分简陋，由夹杂着手势和表情的呜呜声、咯咯声和嚎叫声构成。鬣狗人用它们来传达一些基本的概念，比如警示已经靠近猎物或者号召他们的盟友加入战斗。当鬣狗人们互相争斗的时候，他们很少在互相攻击前用威胁或言语来烦扰。

当鬣狗人头目必须彼此交流一些复杂的概念时，他们使用的是耶诺古赐予他们的一种残缺形式的深渊语。尽管鬣狗人中的精英可以利用他们有限的深渊知识来留言，但鬣狗人语言缺乏手写或书面的形式。当然，在大多数情况下，一个鬣狗人战争团伙很少使用手写的笔记或符号。鬣狗人只是游荡、攻击、杀戮和进食。任何更复杂的事情都是鬣狗人团伙所不关心的。
