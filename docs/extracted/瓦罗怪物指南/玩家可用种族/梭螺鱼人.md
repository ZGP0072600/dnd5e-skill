# 梭螺鱼人

梭螺鱼人 Triton

啊，梭螺鱼人。想像一下，如果精灵在海中度过了几个世纪，在那里他们的傲慢与自负能够不受干扰地增长。无论如何，梭螺鱼人耗费了时间与鲨华鱼人以及更糟的东西战斗，所以你知道你能在战斗中依靠他们。——布里戈·石心，船长。

梭螺鱼人守护着海洋深处，在海沟，连通元素位面的传送门，以及远离陆地住民视野的其他危险之处附近建造小型的聚居点。深海海底的长久守护者们，近些年来高尚的梭螺鱼人们在世界上日益活跃。

水中十字军
数个世纪以前，为了应对邪恶元素日益增长的威胁，梭螺鱼人来到了世界中。梭螺鱼人在水元素位面对他们的敌人发动了多次战争，驱赶它们进入了黑暗的海渊，在那里它们逃到了压倒性的压力和彻底的漆黑中。
一段时间以后，梭螺鱼人发现他们古老的元素仇敌变得安静了。潜入深处的远征表明，克拉肯，鲨华鱼人，和更麻烦的敌人从水元素位面逃到了主物质位面。
梭螺鱼人，在责任感的驱使下，不会让他们的敌人如此轻易地逃脱。一个伟大的梭螺鱼人秘会选择熟练使用武器和魔法的志愿者作为远征军的一部分进入主物质位面搜寻他们的敌人。
这些梭螺鱼人遍布在世界的海洋中，建立领地用以监视看守海沟，传送门，海底洞穴以及他们的敌人可能潜藏的其他地方。他们在找到它们的时候击败敌人，并把残兵败将驱赶到更深处。
随着他们的敌人被驱逐至海洋的最深处，梭螺鱼人则定居在那里监视着它们杀回的任何迹象。随着时间的推移，梭螺鱼人在海底从他们最初的定居点和建造的前哨基地扩大管辖范围，以建立与其他种族的贸易。尽管这样扩张，还是很少有人知道他们。他们的居所是那么的遥远，即使是人鱼和海精灵都很少遇到他们。

傲慢的贵族
由于他们的与世隔绝和对主物质位面的理解有限，梭螺鱼人可能会显得傲慢与自大。他们视自己为海之守护者，即便不是完全的顺从，他们也要求其他生物给予他们深深的敬意。
这种态度可能会激怒他人，然而它来自事实依据。几乎没有人知道梭螺鱼人对可怕的海底威胁的伟大胜利。梭螺鱼人对这等无知几乎没有宽容，并且乐于讲述他人对他们的巨大亏欠。
在其他人欢迎他们成为受人尊敬的盟友和导师的假设下，梭螺鱼人也有一种脱离他们的孤立状态的倾向。除此之外，会与这种态度保持足够的距离。梭螺鱼人对世界的有限理解使得他们对陆上世界的国家，战争，以及其他斗争一无所知。梭螺鱼人乐意关注这样的小事，以作为世界的真正守护者的梭螺鱼人地位的一个侧面。

坚定的捍卫者
尽管他们做法惹人生厌，梭螺鱼人是心存仁善的生物，坚信其他文明的种族应该得到他们的保护。他们的态度可能令人烦躁，但是每当海盗舰队掀起恶浪或者克拉肯自沉睡中苏醒时，他们是第一个拿起武器保护他人的人。
为了大义，梭螺鱼人会欣然牺牲自己。他们将为人类，人鱼，还有其他生物无畏地奋战至死。他们自恋的天性使得他们忽视了其他生物的历史，但他们也因为让水元素位面的邪恶之物进入主物质位面，并威胁到了其住民而承受着负罪感。梭螺鱼人确信他们对世界负有荣誉的债务，他们将以战斗和死亡来支付。
有时，他们的热情和对世界的无知会使他们误入歧途。第一次遇到其他生物的梭螺鱼人可能会低估他们，使梭螺鱼人容易受到欺骗。凭借他们强悍的好战传统，梭螺鱼人有时会急于进入战斗。

外地人上岸
鉴于他们的与世隔绝，大多数梭螺鱼人从未去过陆上世界。他们的斗争观念使他们不能轻易地远离海域，四季的变化也在困扰着他们。
梭螺鱼人还发现各种社会制度，国家，和其他规矩令人眼花缭乱。对于陆上世界所有得意的文化，他们仍然是一无所知的。典型的梭螺鱼人国团结紧密，有条不紊，并统一围绕着共同的事业。一个在陆地上的梭螺鱼人容易因为繁杂的联盟，敌对，以及一己私利阻止着陆上住民真正的统一而显得困惑不解。
最糟糕的是，梭螺鱼人的傲慢混合了梭螺鱼人不了解陆上世界的性格。对梭螺鱼人而言，很容易把那些梭螺鱼人难以理解的社会惯例归咎于陆上住民的野蛮，缺点，或者软弱。

梭螺鱼人的个性

梭螺鱼人的姓名
大多数梭螺鱼人的姓名有两到三个音节。男性名通常以元音和字母 s 结尾，女性名传统上用 n 结尾。梭螺鱼人使用他们的本国为姓，通过在国名末尾添加一个元音，后面加上“th”构成姓名。
女性梭螺鱼人的名字：Aryn，Belthyn，Duthyn，Feloren，Otanyn，Shalryn，Vlaryn，Wolyn
男性梭螺鱼人的名字：Corus，Delnis，Jhimas，Keros，Molos，Nalos，Vodos，Zunis
梭螺鱼人的姓氏：Ahlorsath，Pumanath，Vuuvaxath

梭螺鱼人特性
你的梭螺鱼人角色拥有以下种族特质。
属性值提升。你的力量，体质与魅力值都增加1点。
年龄。梭螺鱼人在15岁左右成年，能活到200岁。
阵营。梭螺鱼人倾向于守序善良。作为最黑暗海域的守护者，他们的文化把他们推向秩序和仁慈。
体型。梭螺鱼人比人类稍矮，平均约5尺高。你的体型为中型。
速度。你的基础步行速度是30尺，你拥有30尺的游泳速度。
两栖。你能呼吸空气和水。
控风操水。作为一位大海之子，你能呼唤气元素和水元素的魔力。你可以用这个特性施展云雾术。从3级开始，你能用它施展造风术，从5级开始，你也能用它施展水墙术（见边栏中的法术）。一旦你以此施展了某个法术，直到你完成一次长休前都不能再次以此施展法术。你的这些法术的施法能力值是魅力。黑暗视觉。在昏暗环境下，你可以将你周围60尺内的区域视为明亮光照；在黑暗环境下，你可以将光照视为微光光照。你在黑暗中无法分清颜色，只能看到灰色的影子。
海之使者。水生野兽与你的种族有着非凡的亲和力。你可以与能在水中呼吸的野兽简单沟通交流。它们能够理解你的话的意思，虽然你没有特殊的能力来明白它们的回应。
海渊守护者。你的种族适应了最极端的深海环境，你获得寒冷伤害的抗性。语言。你可以说，读，写通用语和原初语。
