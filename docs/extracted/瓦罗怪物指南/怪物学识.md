# 怪物学识

怪物学识 Monster Lore

经验丰富的瓦罗遭遇过各种怪物，而我们将在此介绍其中某几个或暴躁或恼人的种类。本章内容主要是针对几种特选的 D&D 标志性怪物，详述其相关的起源、分布、习性和巢穴等信息，并以此为《怪物图鉴》的信息作补充。如此详解一种怪物的资料所需要的工作量非常庞大，因此我们并没有为所有的怪物作这样的总结而只从中选择了九种 D&D 战役里常用到的生物：

| 眼魔 Beholders | 类地精 Goblinoids | 夺心魔 Mind flayers |
|---|---|---|
| 巨人 Giants | 鬼婆 Hags | 兽人 Orcs |
| 鬣狗人 Gnolls | 狗头人 Kobolds | 蛇人 Yuan-ti |

你可以从本书中搜刮创意或地图素材作为你下次创作冒险或设计反派时的参考，而这也正是这份材料的价值所在。我们希望书中的怪物资料能够借鉴到你自创的 D&D 战役中，你或许还能从书中获得灵感并以此为你的 PC 设计新的挑战新的玩法。愿文字蕴含的闪光为你点燃想象的花火！你或许会说：为什么必须是这几种怪物而不是其他？龙类 dragon 和吉斯扬基人 githyanki 呢？怎么没有邪魔 fiend 和亡灵 undead？我们有计划在往后的产品中继续处理这些本书中未及详述的怪物们，而在这之前就请大家多注意阶梯下藏身的狗头人，并当心带着古怪礼物的鬼婆吧。
