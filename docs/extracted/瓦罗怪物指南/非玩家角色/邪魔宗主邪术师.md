# 邪魔宗主邪术师

**邪魔宗主魔契师 Warlock of the Fiend**

邪魔宗主魔契师可以通过与下层恶魔的魔法契约获得他们的力量。这些魔契师经常把小魔鬼或小恶魔当作同伴，他们的行为趋于极端：与崇拜恶魔的邪教信徒交往，或者投身于摧毁恶魔邪教。
