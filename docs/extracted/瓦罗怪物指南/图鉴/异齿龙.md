# 异齿龙

**异齿龙 Dimetrodon**

这些背鳍呈帆状的爬行动物通常能在恐龙生活的区域里被找到。它们往往在岸边和浅水中狩猎，在环境中扮演着与鳄鱼类似的角色。

**异齿龙 Dimetrodon**

*中型野兽，无阵营*

护甲等级：12（天生护甲）

生命值：19（3d8+6）

速度：30尺，游泳20尺

力量14（+2） 敏捷10（+0） 体质15（+2）

智力2（-4） 
感知10（+0） 魅力5（-3）

技能：察觉+2

感官：被动察觉12

语言：——

挑战等级：1/4（50XP）

**动作 Actions**

** 啃咬 Bite。**近战武器攻击：命中+4，触及5尺，单一目标。命中：9（2d6+2）穿刺伤害。
