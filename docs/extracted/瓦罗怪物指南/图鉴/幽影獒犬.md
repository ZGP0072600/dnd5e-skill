# 幽影獒犬 Shadow mastiff

这些幽影界的黑色猎狗无影无迹地穿梭于阴影之中，总是处于狩猎状态。在那些迷雾笼罩幽影界与主物质位面最薄弱的黑暗地带，它们可以穿越到世界的黑暗领域。

**贪婪的潜伏者：** 在幽影界的阴影獒犬总是成群结队地狩猎，因此当它们其中的一只进入两个世界的裂隙时，其他的獒犬也一定会跟随。每个阴影獒犬群落都由一只阿尔法獒犬（雄性或是雌性）领导，这些阿尔法是族群里最坚韧最具智慧的首领。每只阿尔法獒犬必须保持冲刺来使整个族群保持一致，以免它被杀死或是被取代。

当一个阴影獒犬族群处于饥饿状态而且感知到猎物在附近，首领阿尔法獒犬发出一声悠长的嚎叫用以震慑在附近听到其嚎叫的野兽或类人生物，同时，这种嚎叫也传递一种信号，使战群其余的獒犬投入到杀戮之中。幽暗为每只阴影獒犬提供了超自然的保护屏障，赋予它们在弱光和黑暗环境下抵御非魔法武器的能力。虽然阴影獒犬能忍耐短暂的强光，但通常它们都有意识的躲避阳光。

**召唤服务：** 有些神明的信仰就是致力于幽影和黑暗，就如同费伦世界的莎尔女神，她的邪恶信徒在执行不洁仪式时通常从幽影界召唤阴影獒犬并把它们用做为神殿守卫，贴身保镖以及对于教义的无信者、叛徒或是异教徒的惩戒者。然而将这些阴影獒犬带领到主物质世界的方法也被一些具有强大意识或邪恶思想的个人所掌握，他们将这些獒犬作为大本营的守卫。

**以太之视：** 除了其他的能力，每只阴影獒犬还能观测到在以太位面生物或者物体，这种额外的知觉使得阴影獒犬成为一种特别有技巧的守护者，尤其是当魔法或精神入侵很可能发生的时候。
