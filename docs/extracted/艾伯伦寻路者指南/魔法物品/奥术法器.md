# 奥术法器

**奥术法器**

去萨恩的集市或者任何一个卡尼斯家族的飞地，你会发现很多可供选择的奥术法器。对于一名施法者来说，选择一个奥术法器的重要程度和一名决斗者选择用刺剑还是巨锤一样。你是会用费尼亚白蜡木魔杖作为施展火焰箭的法器，还是会选择使用瑞西亚法球来驾驭防御能量？

奥术法器是一种工具，并且只在你积极使用它时才能享受它带来的好处。一个放在背包里的防护法球不会为你提供防护；你必须要把它拿在手里。

在科瓦雷，奥术法器被认为是武器：如果守卫正在出于安保需要对剑和弓施加固定，那么他们也会要求你交出你的杖。

**蕴能木杖 Imbued Wood**

*奇物，普通（需同调）*

强力的显能区域能够将位面能量注入当地的树木中。一名有才能的奇械师能够利用这种树木来制造一根魔杖、一根权杖或一根法杖，它在引导一种特定类型的能量方面特别有效。

这种奥术法器能够在你施展造成特定伤害类型的法术时，让伤害掷骰的结果+1。具体见下

·费尼亚白蜡木：火焰伤害。

·伊瑞安红木：光耀伤害。

·奇斯瑞毒番石榴木：酸液或毒素伤害。

·拉玛尼亚橡木：闪电或雷鸣伤害。

·玛伯尔乌木：暗蚀伤害。

·梦族山毛榉木或瑟瑞奥特崖豆木：心灵伤害。

·瑞西亚松木：寒冷伤害。

·撒法拉斯桦木：力场伤害。

**防护法球 Orb of Shielding**

*奇物，普通（需同调）*

防护法球是由来源于特定位面的水晶或石头制成的。当你握着法球的时候，它会从某一种特定类型的能量下保护你。任何情况下，当你受到的对应类型伤害减少1d4。材料与对应的伤害类型见下。

·费尼亚玄武岩：火焰伤害。

·伊瑞安石英：光耀伤害。

·奇斯瑞矽卡岩：酸液或毒素伤害。

·拉玛尼亚燧石：闪电或雷鸣伤害。

·玛伯尔黑曜石：暗蚀伤害。

·梦族天青石或瑟瑞奥特大理石：心灵伤害。

·瑞西亚页岩：寒冷伤害。

·撒法拉斯黑硅石：力场伤害。

> **可选规则：双手奥术法器 **
>
> 在艾伯伦，奥术法器是一种武器。正因为如此，关注一些与普通武器相同的权衡取舍是很有趣的：你是否为了获得更大的力量而牺牲一只手的自由？
>
> 如果你使用这个可选规则，当施法者使用双手奥术法器来施展攻击性戏法（即需要豁免或掷攻击骰的戏法）时，该戏法的射程增加50%。
>
> 在使用此规则时，法杖总是占据双手，而权杖既能够单手使用，又能双手使用。使用双手奥术法器能够满足法术的姿势成分。
