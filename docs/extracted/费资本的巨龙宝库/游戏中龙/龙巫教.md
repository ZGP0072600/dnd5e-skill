# 龙巫教

龙巫教 Cult of the Dragon

创立于费伦的龙巫教是基于一个翻译错了的预言："死亡巨龙将统治整个世界"。在其大部分历史中，该教派一直专注于创造龙巫妖以实现这一预言。其成员创造的龙巫妖比物质世界的任何其他势力都要多。由于每当龙拥抱不死之身时，腐化就会在世界之间蔓延（见本章前面的
"不死龙类 Undead"），而该邪教的影响是巨大的。

近年来，提亚马特借以龙巫教的一名年轻成员之手改变了这个教派的信仰。龙后引导这位名叫赛维林的先知修正了其邪教基本预言的翻译错误。"......除了破碎的王座外，什么都不会留下，没有统治者，除了死亡。巨龙将统治整个世界......" 
在这个新愿景的指导下，赛维林不知疲倦地工作，以实现向龙的升华，甚至试图将提亚马特的化身带到这个世界上进行破坏（他几乎成功了）。

教派的分歧 Dissent in the Rank。自从赛维林将提亚马特带入世界的尝试失败后，龙巫教就出现了剧烈的分裂。一些成员追求赛维林对邪教的新愿景，将注意力放在他对预言的重新解释上。其他人仍然支持预言的旧解释，继续努力创造龙巫妖。

冒险引子 Adenture Hooks。巨龙僭政冒险详述了龙巫教为将提亚马特的化身带入世界所做的努力。除了这个模组外，龙巫教冒险引子表为冒险者如何遇到这个邪恶的教派提供了建议。
