# 龙的仆从

龙的仆从 Dragons‘s Minions

弥散在龙和龙巢中的魔法有时也会延伸到龙的仆从身上。你可以在仆从的数据中添加一个或多个如下特性以反映其与它的主龙间的关系。

自爆仆从 Explosive 
Minion。
 仆从死亡时会释放出一道能量爆发，覆盖一处半径5尺的球形区域。任何位于区域内的生物都会受到1d8的伤害，伤害类型与其主龙的吐息武器造成的伤害相同。仆从心智 Minion’s Mind。
仆从不能被强迫以违背其主人指示的方式行事。仆从献身 Sacrificial 
Minion。 仆从死亡时，只要主龙在它的60尺范围内，其主人就会恢复等于其挑战等级四倍的生命值。无私护卫 Selfless 
Bodyguard。
当一次攻击命中其主人，而仆从其他主人的5尺范围内时，仆从就可以用自己的反应来使攻击改为命中自己。仆从感应 Telepathic 
Minion。 只要都在同一个位面上中，仆从和它的主人就可以通过心灵感应互相交流。
