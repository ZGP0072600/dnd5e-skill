# 作为策划者的龙

作为策划者的龙 Dragon as Schemer

本书中的大多数冒险引子都涉及到龙在漫长的一生中可能实施的计划，可能利用追随者作为世界上的代理人。在实践中，许多策划龙的目标可能与上方那些过于危险的龙的目标相同：获取食物、确保巢穴、增加宝藏。但其他的阴谋可能围绕着获取爪牙、推进龙族派系的事业、建立龙的威望、为整个龙族的福祉而努力，或者改变世界的性质和魔法。

当龙的计划与人物的生活发生交集时，它们就会成为冒险的基础。最直接的是，人物可能被要求帮助龙的计划或反对它们。交锋理由表提供了灵感，帮助你把与龙有关的冒险引子变成影响游戏中人物的冒险。

交锋理由

| d6 | 龙的计划...... |
|---|---|
| 1 | ...直接针对一个或多个角色。当龙或龙的爪牙与角色对峙时，冒险开始。 |
| 2 | ...威胁到一个角色的牵绊或对一个或多个角色很重要的 NPC。 |
| 3 | ...挑战一个角色的理想或利用一个缺陷。角色可能会听到关于龙的活动的传闻，刺激他们采取行动。 |
| 4 | ....把人物困在冲突的势力之间。 |
| 5 | ...为人物追求个人目标创造机会。 |
| 6 | ...造成威胁人物的事故或灾难。 |

一个以龙的阴谋为主题的冒险通常最适合作为基于事件的冒险 Event-Based 
Adventures，如城主指南中所述。无论龙是反派、赞助者还是感兴趣的旁观者，都可以利用本章和第五章的表格和建议来充实龙的个性、目标和其他特征。龙的巢穴可能是冒险的关键地点（可以参考或使用第五章的地图），但龙的计划也可以把人物带到任何地方——甚至是其他世界和星球。
