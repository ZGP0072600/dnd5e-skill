# 跨越世界的回响

跨越世界的回响 Echos Across the World

不需要费什么力气就能成就无限。如同蜘蛛依靠众多眼睛视物，龙就是存在于世，无论是一还是众。

——费资本

龙与物质位面的魔法和第一世界历史的独特联系，使其与物质位面无数世界的其他龙有着神秘的联结。理论上说，每一条曾经存在过的色彩龙、金属龙和宝石龙在物质位面的每一个世界都有一个独特的回响。其中有些回响彼此非常相似，但有些回响则有更大的差异。

你可以使用龙之回响的特征表来帮助你决定龙的回声在哪些方面与你所知道的龙相似，哪些方面不同。然后你就可以利用本章中的其他表格（包括龙类外表、龙类举止、龙类牵绊、龙类缺陷和秘密表），以及第五章中的个性和理念，来帮助你决定一条龙和其回响间的差异。

龙之回响的特性

| d8 | 回声特征 |
|---|---|
| 1 | 相同的龙种和年龄，只在外形上有差异 |
| 2 | 相同的龙种和年龄，只是在行为举止上有差异 |
| 3 | 相同的龙种和年龄，但在性格、理想或阵营方面有明显的差异 |
| 4 | 相同的龙种，大体上相似，但在年龄上要更大或更小。 |
| 5 | 相同的年龄，同一龙种中的不同种类（色彩、宝石或金属），并且在外观和性格上相似 |
| 6 | 相同的年龄，不同龙种中的不同种类（有相似的吐息武器或栖息地），有类似的外观，但有非常不同的理想或羁绊 |
| 7 | 在几乎所有可以想象的方面都有根本性的不同，但仍然被跨越世界的命运所连结 |
| 8 | 重骰。此外，回响是龙类亡灵（龙巫妖、空骸龙、幽魂龙或龙灵碎晶）或幽影龙。 |
