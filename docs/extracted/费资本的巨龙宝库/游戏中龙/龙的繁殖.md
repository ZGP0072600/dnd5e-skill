# 龙的繁殖

龙的繁殖 Reproduction

龙就像蜥蜴。它们存在于人类存在之前，并将存在于人类消亡之后，它们可以重塑现实的结构，他们超越了自然的法则！哦等等……不是像蜥蜴。对，对！是像魔法一样。

——费资本

在诸多物质位面的世界中，龙的繁殖也因世界不同而具有诸多差异，并且甚至会因所涉及的龙的家族、种类或个体性质而产生进一步的不同。长期以来，研究龙的人类学者一直认为它们的生殖方式类似于普通的爬行动物——即，两条不同性别的龙交配并产生一窝蛋。许多世界中的龙都大多如此，然而龙的繁殖涵盖了其他更加广泛的可能性。你可以从使用龙蛋起源表中选择一种方式用于你的世界中的龙的繁殖，或是作为特定的某窝龙蛋的起源，亦或是介于二者之间。

龙蛋起源

| D10 | 起源 |
|---|---|
| 1 | 自发转世 Spontaneous Reincarnation。当一条成年或更老的龙死亡时，龙逐渐解体的尸骸中会形成一个或更多的蛋。这些蛋可能会如真菌般在尸体腐烂时生长而出，也可能出现在消逝于烈火中的尸体所留下的余烬中，亦或是需要从已经变成坚石或金属的龙骸中开采出来。 |
| 2 | 初世之苗 Seedlings of the First World。在世界的中心（地下，或远离文明的原始旷野）矗立着一棵树，龙蛋像果实一样在上面生长。每条龙一生中都会有一次压倒性的冲动，去寻找这棵树。在找到它之后，龙必须说服圣树守护者赐予它一个或更多的蛋。 |
| 3 | 吞噬宝物 Consuming Treasure。龙吃掉其宝库相当重要的一部分，以此令自己产下一窝蛋。 |
| 4 | 秘法杰作 Magical Crafting。与巴哈姆特和提亚马特创造龙相似，龙通过复杂而精巧的仪式制出龙蛋，并为其注入生命。 |
| 5 | 龙脉转化 Draconic Transformation。蒙受启示的非龙类（大多是类人生物）会在迎接死亡、完全醒悟，或是经历也许会涉及龙血沐浴或被龙吞噬的仪式之后，就会被转化为龙蛋。类人生物和龙类都将这种转变视为向更高级的存在形式的过渡。 |
| 6 | 神造起源 Divine Origin。只有巴哈姆特和提亚马特能创造龙蛋——每颗龙蛋都是神圣干预的结果。 |
| 7 | 自发形成 Spontaneous Forming。龙蛋会在某些地区自然生成，对于金属龙和宝石龙来说，通常是在金属矿或宝石矿的地下矿脉中，对于色彩龙来说则是在极端环境（如火山、冻原、沼泽等）中。龙蛋也许会自行孵化，或者必须先由其他生物挖掘出来，无论是由成年或更老的龙还是某个倒霉的矿工。 |
| 8 | 孤雌生殖 Parthenogenesis。只要觉得已经做好了饲养雏龙的准备，一个成年或更老的龙就可以自行产下一窝蛋——或是把抚养的责任交给其他龙。 |
| 9 | 龙类誓约 Dragon Conclave。五条不同种类的成年或更老的龙聚集在一起创造一窝龙蛋，参与仪式的每条龙都将献出一些生命能量。这窝龙蛋将孵化出五条不同种类的龙，这取决于组成誓约的龙。 |
| 10 | 龙类复生 Rejuvenation。远古龙自愿深入出神直至死亡，并留下一颗龙蛋。 |

龙蛋诞生之后，必须在适宜的环境下才能孵化成雏龙。通常情况下，龙蛋只能在适合对应种族龙类的环境中成长——红龙或金龙在熔岩中，白龙或银龙在冰中，黑龙或绿龙在腐殖质中，黄铜龙或蓝龙在暴晒中的沙里，青铜龙或赤铜龙则在海蚀洞穴或荒凉的荒地上。宝石龙蛋通常需要在纳入岩石或小型洞穴中孵化。无论是哪种龙蛋，其孵化期可能从六个月到数十年不等，也可能更长。
