# 龙类特征

龙类特征 Dragon Characters

《城主指南》一书中含有一系列旨在帮助你制作 NPC 的表格和指南。而本节介绍了这些表格的替代方案，帮助你将一条龙从一张资料卡和简述转变为冒险中的一个有血有肉的角色。

在本书的第5章中，你可以找到为各种龙定制的性格特征与理想的附加表格。

龙类的外貌

| D20 | 外貌 |
|---|---|
| 1 | 刻着在位置显著的鳞片上的龙语单词或符号 |
| 2 | 巨大的疤痕 |
| 3 | 一只眼睛缺失或呈乳白色 |
| 4 | 弯曲的牙齿 |
| 5 | 明显的超重或瘦削 |
| 6 | 修长的、蜿蜒的身体 |
| 7 | 短小、粗壮的身体 |
| 8 | 嵌入鳞片的钱币或宝石 |
| 9 | 雕刻过的角或爪子 |
| 10 | 鳍或翅膀边缘穿透着金属环 |
| 11 | 披着某种形式的衣服，从披肩到长袍。 |
| 12 | 不寻常的颜色（例如橙色、棕色或紫色的红龙）。 |
| 13 | 低伏下身，像猫一样的猎食姿态 |
| 14 | 直立的、几乎两足步行的姿态 |
| 15 | 鳞片上有仆从、崇拜者或雏龙的爪印图案 |
| 16 | 用敌人的骨头、角、爪子或牙齿做成的项链 |
| 17 | 额外的角或刺 |
| 18 | 长长的、像獠牙一样的牙齿，或额外的一排牙齿 |
| 19 | 尖锐的或锯齿状的鳞片 |
| 20 | 与龙的吐息武器相对应的元素能量从鳞片之间渗出 |

龙类习惯

| D20 | 习惯 |
|---|---|
| 1 | 我在倾听或说话时，头部会不断左右晃动 |
| 2 | 我的尾巴像蛇一样不断蠕动 |
| 3 | 我缓慢而持续地扇动翅膀 |
| 4 | 我在说话时惯于俯下身子与小个子种族平视——除非我生气了。 |
| 5 | 我即使在说其他语言时，也会夹杂一些龙语的词汇和短语 |
| 6 | 我用兽嘶作为讲话的标点——咆哮、吼叫、呼噜声、鸣叫或口哨声 |
| 7 | 我常常摆弄尾巴尖 |
| 8 | 我心不在焉地咀嚼剑、矛、长枪或用它们剔牙 |
| 9 | 我总会迅速忘记其他人名字，并随意的编造绰号。 |
| 10 | 我在谈话中经常引经据典，尽管很多时候完全不相干。 |
| 11 | 我不断地晃动尾巴——偶尔会扑向它 |
| 12 | 我说一种古老的通用语（相当于文言文），不理解当代的俚语和习语。 |
| 13 | 我常常发出可怕的、阴阳怪气的声音，这实际上是笑声 |
| 14 | 我喜欢模仿类人生物的声音 |
| 15 | 我将现代文化视为历史奇观，并认为它们很快就会崩溃。 |
| 16 | 我难以辨别像类人生物那样小个子生物的细节 |
| 17 | 我常在身边的石头表面上磨利爪子或角 |
| 18 | 我会表现出吐息武器的小迹象，如吐出黑烟，齿间电光闪烁，或吹出酸性飞沫 |
| 19 | 我不停地叹气，给人以极度无聊的印象。 |
| 20 | 我疑心极重，把所有遇到的类人生物都视为敌对龙的潜在代理人 |

龙类牵绊

| D10 | 牵绊 |
|---|---|
| 1 | 我的宝库中珍藏着一件特别的事物——那是我所爱的人送给我的礼物，但对方早已去世。 |
| 2 | 我深爱着我的兄弟姐妹，他们的巢穴就在附近。我会不惜一切代价保护它们，或者为它们报仇。 |
| 3 | 附近的某个人用诱人的谜题和奇异的想法吸引着我。 |
| 4 | 我收集关于物质位面中诸多世界的信息，并希望有一天能访问另一个世界。 |
| 5 | 我献身于巴哈姆特或提亚马特，把祂们的利益放在我自己的利益之上。 |
| 6 | 我决心毁灭那些杀害我父母的冒险者——以及所有与它们有关的人。 |
| 7 | 我痴迷于吸引另一条龙的注意。 |
| 8 | 我觉得有义务保护居住在我的领地上的生物（除了好吃的那些）。 |
| 9 | 我正在努力收集一套极其罕见的无价之宝。 |
| 10 | 在我找回从宝库中被偷走的物品前，我不会休息。 |

龙类缺陷和秘密

| D10 | 缺陷和秘密 |
|---|---|
| 1 | 我觉得冒险者很可怕，我感觉有一天我会在他们手中遭遇灭顶之灾。 |
| 2 | 我宝库中一件东西的价值相当于我其他所有宝藏之和，而担心它失窃的恐惧无时无刻不在侵扰我。 |
| 3 | 只要有机会，我就会一口气吃到撑，然后陷入漫长的沉眠。 |
| 4 | 比起和入侵者战斗，我宁愿让我可怕的名声吓跑他们。 |
| 5 | 如果其他龙知道我多么喜欢那些小个子仆从，他们肯定会嘲笑我。但它们真的太可爱了! |
| 6 | 我很害怕来自外层位面的生物，尤其是魔冢 modrons。 |
| 7 | 另一条龙已经发誓要找到并宰了我。 |
| 8 | 人类还没有准备好去了解我收藏书籍中可怕的宇宙真理。 |
| 9 | 再活几个世纪的前景让我筋疲力尽。 |
| 10 | 我确信，在繁多主物质位面中的另一个世界里，另一个我正谋划摧毁我并夺走我的宝藏。 |
