# 半龙

半龙 Half Dragons

正如龙的繁殖不一定是一个简单的生物学问题一样，半龙也可能是通过各种方式产生的。下面的半龙起源表为你提供了一些例子。

龙出于各种原因创造了半龙后代。有些龙创造半龙是为了获得可靠的手下。其他龙则在他们的巢穴周围创造半龙子嗣，以使该地区对入侵者更加危险。更多的时候则是，半龙只是自发出现，或者是龙之魔力及影响的额外结果。

在某些世界中，这些现象也可以用来解释龙裔、狗头人以及第二章中描述的龙族赠礼的来源。

半龙起源

| D6 | 起源 |
|---|---|
| 1 | 区域效应 Regional。龙之巢穴的魔法影响令半龙自发地出现在该地区的其他生物中。 |
| 2 | 龙之禁果 Forbidden Fruit。龙可以使一棵树结出神奇的果实。当被能够生育的生物吃掉果实时，这个生物就将诞下一只半龙。 |
| 3 | 真爱之礼 True Love‘s Gift。爱情偶尔会在龙和其他种类的生物之间萌发，这种生命的连结会产生半龙的子嗣。大多数因爱而生的半龙都是以这种方式来到这个世界的。 |
| 4 | 龙血转化 Blood Transformation。沐浴龙血中或饮用龙血的生物有时会变成半龙。一条龙可能会自愿提供一些血液来满足这一条件，而当一个屠龙者被溅上龙血时，这种转变同样可能产生。 |
| 5 | 龙类赐福 Cradle Favor。龙可以将龙的力量赐予新生儿或未出生的孩子，作为祝福或诅咒。 |
| 6 | 宝库窃贼 Hoard Thieves。一条老龙的宝藏充满了龙的力量。而当这种力量渗入染指古龙宝库的窃贼身上时，这个小偷可能就会变成半龙。 |
