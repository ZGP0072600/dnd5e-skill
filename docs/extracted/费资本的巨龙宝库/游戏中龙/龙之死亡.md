# 龙之死亡

龙之死亡 Death of a Dragon

当我走的时候，我梦想着把我的精神传递到与我有过最深刻经历的地方。首先想到的是福克斯鸭酒馆的焦糖糖浆。嗯！很美味!

——费资本

龙与支撑物质世界的魔法结构紧密相连。因此，任何龙的死亡都是一个具有巨大意义的事件——而那些为龙带了死亡的人往往会被永远地改变。

一些贤者将龙描述为类似于魔网中的节点——以具有物理存在的力量焦点。当龙死亡时，这个节点将会崩毁，这有时会导致魔法能量的激增或爆发。你可以使用龙陨终局"表来帮助你决定龙死后会发生什么。当龙最终败退时，杀死龙的角色往往也已经奄奄一息了，所以如果某个效果可能会杀死一个角色，请再掷一次，或者选择一个危险性较低的结果。

龙陨终局

| d6 | 效果 |
|---|---|
| 1 | 龙立即使用它的一个巢穴动作，并将效应集中在自己身上。例如，一条败亡的黑龙可能会将自己和它周围的任何角色拉进附近的水塘，而一条赤铜龙的身体可能突然被泥浆吞没。 |
| 2 | 龙最后一次使用它的吐息武器。 |
| 3 | 龙的遗骸会转化为磐石、金属、熔岩、寒冰或雾气。或者身体消散，只留下一颗异化的心脏或其他器官。 |
| 4 | 在《玩家手册》中术士部分的 "狂野魔法浪涌 "表上掷骰，并重骰任何你认为不合适的结果。 |
| 5 | 一股生命能量从龙身上爆发出来，使身体周围120英尺内的每个生物都获得了完成一次长休的所有好处。 |
| 6 | 尸体会被吸引到另一个世界回响的巢穴中，这将增加该回响的力量。 |

区域反转 Regional Reversal 正如《怪物图鉴》中所描述的和第4章中详细讨论的，龙的存在会改变龙巢周围的区域。当龙死亡时，这些影响通常会立即结束或在数天内逐渐消失。但是创伤性的死亡有时会将负面效应延伸到整个地区。这些影响可能包括瞬时效应，也可能是持续数天的挥之不去的魔法，如下面的例子：

污化水体 Fouled Water。巢穴周围6英里内的水会变色并被污染1d10天。

噩梦缠身 Reseless Sleep。任何在巢穴6英里范围内睡觉的生物都会在接下来的1d10天内被噩梦所困扰。地动山摇 Trembling Earth。巢穴周围6英里范围内的大地会发生巨大的震颤。

兽群异变 Wildlife Changes。先前被龙的存在所吸引的生物会发生剧烈的行为变化，可能会攻击它们看到的任何生物或是开始大规模迁徙，远离巢穴。

转变之风 Winds of Change。巢穴周围6英里范围内的天气戏剧性地改变。

凋零迸发 Withering Burst。巢穴周围1英里内的普通植被会黑化枯萎。

衣钵传承 Passing the Mantle

当龙死亡时，龙身上的力量并不只是消逝了。虽然随着时间的流逝，这种力量终将消散，但在龙死前的那段时间里，这种力量可以传递给其他人，或者被人夺取。有时（如上文
"龙的繁殖"一节中所述），龙的死亡会产生一个蛋，将自身的力量直接传给新的一代。在其他情况下，垂死的龙会将将力量注入另一个生物体内——通常是龙，但有时也会是君王、贤者或冒险者。此外，那些杀死龙的人或许也能夺取龙的力量。

这种力量的转移所能带来的好处由你决定。它可能包括本书第二章中列出的龙族赠礼，或者《城主指南》中的护咒或祝福。你也可以使用《塞洛斯神话奥德赛》中的超自然赠礼或《范·里希滕的鸦阁指南》中的黑暗赠礼作为奖励（如果是黑暗赠礼的话，同样会有明确的负面效果）。

临终遗赠 Investiture。垂死的龙可能会以不同的方式传递力量。最常见的过程是让接受者吸入龙的临终吐息——这通常至少要受术者受到龙经过削弱后的吐息。宝石龙可能会在受术者的身体内灵能性地植入一颗携带龙的力量的宝石。当然，龙也可能会在受术者身上放置一只爪子，或者遗留一件宝物作为转移力量的体现。

龙之骨血 Body and Blood。当一条龙死亡时，龙的力量可能会在尸体上残留一段很短的时间。饮用或沐浴龙血；在龙的尸体里睡眠；或食用龙的心脏、大脑、眼睛、基血管或肌肉，这些都是夺取巨龙魔力的有效方法。然而，以这种方式获取力量本身就可能是一种考验，因为即使在龙死后，龙体内残留的能量也可能严重伤害那些贸然接近的生物

自动转移 Automatic Transfer。龙的力量也有可能立即自动转移到参与龙的死亡或龙死亡时在场的生物身上。在龙死亡的那一刻，一股力量可能会从龙身上散发出来，影响到身体周围300英尺内的每个生物。或者，其力量的残余可能进入每个在龙的生命过程中对其造成伤害的生物。
