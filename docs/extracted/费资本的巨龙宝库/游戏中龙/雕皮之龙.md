# 雕皮之龙

雕皮之龙 Hidecarved 
Dragons

在物质位面中，龙经常会形成一种由三到五条龙组成的小型结社，这被称为密约 lauths。这些龙分享的领地和宝藏，在严格的纪律与守则下生活，并为一个共同的目标奉献自己。它们最显著的特点是在鳞片上刻下具有保护效果的秘法印记。

共同目标 Common Purpose。当龙确定了一个共同的目标，而这个目标的完成需要一群不同的龙的持续努力时，它们经常就会形成一个密约集会。密约目标表提供了一些例子。

密约目标

| d6 | 目标 |
|---|---|
| 1 | 支持建立一个强大的帝国，以维护密约领土的秩序。 |
| 2 | 使一个强大帝国覆灭 |
| 3 | 结束源自幽暗地域中的异怪的影响 |
| 4 | 寻找并摧毁由某个古老法师制造的数百件被诅咒的魔法物品 |
| 5 | 摧毁一个大恶魔、一个恶魔王子、一个强大的天族或一个至高妖精，他们的影响改变了密约成员们的领地。 |
| 6 | 实现（或阻止）几百年前由一条远古龙做出的预言。 |

多样的龙 Diverse Dragons。雕皮之龙的密约团体有着共同的目标，但往往没有什么其他共同点。设计上，每个成员通常是不同类型的龙，为团体的目标带来独特的能力和观点。同样地，这样一个团体往往涵盖了不同的年龄段的龙，以确保该团体的传统比其最年长的成员更长久。

镌刻护咒 Engraved Wards。雕刻在龙的鳞片上的秘法印记能为它的主人提供一定的保护，它们使龙在执行密约任务时更能抵御可能面临的威胁。龙加入密约的时间越长，它们得到的保护就越多。大多数作雕皮之龙都有一个这样的护咒，每比雏龙高出一个年龄层，即可再加上一个额外的条目。你可以在密约护咒表上掷骰或者按照你的意愿决定具体效果。

密约护咒

| d6 | 符文效果 |
|---|---|
| 1 | 不变形态 Immutability。龙免疫任何会改变其形态的法术或魔法效应。 |
| 2 | 精神壁垒 Inscrutability。龙免疫任何能感知其情感或读取它的思想的效应。为确定其意图或辨别言辞真伪的智力（洞悉）检定具有劣势。 |
| 3 | 魔法抗性 Magic Resistance。龙在面对法术和其他魔法效果时，有优势。 |
| 4 | 伤害抗性 Damage Resistance。龙对来自非魔法攻击的钝击、挥砍、穿刺的伤害具有抗性。 |
| 5 | 额外抗性 Additional Resistance。龙对一种类型的伤害有抗性。掷一个 D8，如果你得到一个龙已经具有抗性或免疫的伤害类型，则重新掷。1，强酸；2，寒冷；3，火焰；4，闪电；5，暗蚀：6，心灵；7，光耀；8，雷鸣。 |
| 6 | 防死结界 Death Ward。龙可以对自己施放一次防死结界 death ward 并在完成一次长休后恢复施放的能力。 |

神话徽记 Mystic Markings。如同龙巫妖一样，镌刻在雕皮之龙鳞片上的符号会对位于其他物质位面中的回响产生影响。当一条龙加入了一个密约团体，其他世界上的回响就会从龙的符文徽记中获得一定程度的魔法保护。一些回响接受了这一莫名其妙的恩惠，而没有质疑或是对它产生好奇。而有些则尝试刨根问底，最终可能会唤醒它们的龙视，发掘影响自身的龙的本质——也许最终会与其他处于类似情况的龙一起建立自己的密约。
