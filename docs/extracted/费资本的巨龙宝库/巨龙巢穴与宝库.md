# 巨龙巢穴与宝库

**巢穴和宝库 Lairs and Hoards**

龙与它们居住的世界之间有着深刻的联系。物质位面的魔力在他们身上流动。它赋予了龙的吐息武器和巨大的力量，聚集在它们的巢穴中，纠缠在它们的宝藏中。这一节为地下城主准备，探讨了龙与它们所居住的世界之间的联系，以及这种联系如何在每一条龙的巢穴和宝库中体现出来。

**“龙的巢穴”** 探讨了龙身上的魔法是如何扩散到被这些生物称之为家的地方附近的世界的。本节包括关于龙喜欢在哪些地方建造巢穴的建议，以及可能在巢穴周围产生的其他区域效应，还有其他巢穴动作。

**“龙之宝库”** 探讨了龙的宝库的大小与龙的魔法力量之间的关系。本节讨论了龙用将其宝库分割为多份的方式来扩大其魔法范围的做法，以及可以将这些宝库联系在一起的物品。这一节也探讨了包括闹鬼和诅咒在内的，可以产生大量的魔法力量并集中于一处的魔法异能。

**“宝库里有啥？”** 提供了用来确定龙宝藏内容的表，并在《城主指南》中的宝藏表上展开。
