# 大海蛇

大海蛇 Sea Serpent

> 当一条大海蛇穿过并擦蹭船体时，它只是试图加速蜕皮过程。当然，这艘船可能会沉没，但这绝不是蛇的错。费资本

海蛇和龙龟并列为深海中最可怕的捕食者，它们激发了许多水手最可怕的噩梦。它们强壮的鳍推动它们在海洋中高速前进。

青年海蛇是敏捷的猎人。他们利用暗礁和天然伪装在袭击前隐藏起来，一个接一个地猎捕粗心船员。而远古海蛇长得足以大到摧毁整艘船，然后悠闲地以水中无助的水手为食。在任何年龄段，海蛇都会以可怕的撕咬、鞭状的尾巴、强有力的缠绕和冰冷的吐息武器进行攻击，这种吐息会让水手们的尸体冻结在船的残骸中。

海蛇和其他龙一样具有凶猛的领土性，它们不会善待未经许可便随意通过其领土的船只。有些人可以通过提供财宝来安抚，但大多数大海蛇还要求船员牺牲自己的一个或多个来满足它。

海蛇生活在沉船的残骸中或深海洞穴中，它们收集沉船上的财物，大量囤积贸易货物、水手的小饰品和海盗的战利品。

远古大海蛇 ANCIENT SEA SERPENT

超巨型龙类，通常为中立

AC 17(天生护甲)

生命值 170（11d20+55）

速度 20尺，游泳速度60尺

力量24（+7）
 敏捷15（+2） 体质20（+5） 智力13（+1） 感知16（+3） 魅力12（+1）

豁免：力量+12，体质+10

技能：察觉+8，隐匿+7

伤害免疫：寒冷

感官：黑暗视觉120尺，被动察觉18

语言：通用语、龙语

挑战等级：14 (11,500 XP) 熟练加值+5

两栖生物：大海蛇可以在空气和水中呼吸

传奇抗性（2/天）：若远古大海蛇在一次豁免中失败，它可以选择直接成功。

攻城怪物：大海蛇对坚实物体和建筑物造成双倍伤害

动作

多重攻击：远古大海蛇进行一次撕咬攻击和一次绞缠或尾击

撕咬 Bite:近战武器攻击：命中+12，触及15尺，单一目标。命中：20（2d12+7）的穿刺伤害，外加6（1d12）的寒冷伤害。

绞缠 Constrict：近战武器攻击：命中+12，触及20尺，单一生物。命中：29（4d10+7）的钝击伤害，且目标被擒抱（逃脱 DC 20）。直到擒抱解除前目标被束缚，且该海蛇不能绞缠另一目标。

尾击 Tail。近战武器攻击：命中+12，触及20尺，单一目标。命中：13（1d12+7）的钝击伤害。如果目标是生物，则该生物必须成功通过一次 DC 20的力量豁免，否则被推离距大海蛇至多30尺并应击倒地.

寒霜吐息 Rime Breath（充能5~6）。远古大海蛇呼出冰霜并覆盖一处60尺的锥状区域，该区域内的每个生物都必须进行一次 DC 18的体质豁免，豁免失败者将受到49（9d10）的寒冷伤害，豁免成功则伤害减半。

传奇动作：

远古大海蛇拥有3传奇动作，用来选择执行下列动作项。它每次只可选择执行一个传奇动作项，且必须在另一生物回合结束时执行。远古大海蛇在其回合开始时恢复所有消耗的传奇动作数。

尾击 Tail 。远古大海蛇发动一次尾击攻击。

撕咬 Bite（消耗2动作）。远古大海蛇发动一次撕咬攻击。
