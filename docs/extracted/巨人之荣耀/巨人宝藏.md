# 巨人宝藏

**第五章：巨人宝藏 Giant Treasures**

巨人在他们漫长的一生中会收集无数各种各样的宝藏与饰品。本章向您本意是为 DM 所准备的展示了许多宝藏，也许无畏的冒险者们会在与《怪物图鉴》中的巨人六族的遭遇战下获得这些宝物。

本章包含以下小节：

**“巨人的行囊”** 提供了一系列可用于决定巨人行囊内带有何种物品的表格当然，包括贵物与贱物。

**“巨人国度之遗”** 详细展示了一系列古代巨人文明留下的，价值不菲的宝藏。

**“魔法物品”** 向您呈现了与巨人、巨人的传说、巨人的魔法符文相关的魔法物品。
