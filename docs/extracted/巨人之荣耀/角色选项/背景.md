# 背景

**背景 Backgrounds**

这一章节为你呈现了两个与巨人们的古代魔法有关联的背景选项。巨人养子背景代表着角色在超巨型生物与他们的超自然力量的夹缝之间的生活。符文雕刻者背景则象征着角色曾为探寻巨人符文工艺的古代传统进行过长时间的钻研。

**奖励专长 Bonus Feats**

如果 DM 决定允许使用巨人养子与符文雕刻者背景，战役中的所有角色都将可以获得选取奖励专长的权利。如果你选择上述的这些背景，你将获得背景中记述的对应奖励专长。如果你选择的背景并没有给予你专长，你则可以从下述列表中选择并获取一个奖励专长。（这些是出现在《玩家手册》中的专长）

- 熟习 Skilled
- 健壮 Tough
