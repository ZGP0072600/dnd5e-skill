# 石巨人遭遇

**石巨人遭遇 Stone Giant Encounters**

石巨人遭遇表展示了石巨人和可能出现在它们所居住区域的生物，这些和室内外可能是由山丘巨人们挑选出来的，也可能只是地理原因而出现在这的。一些智慧生物也许会被石巨人的艺术品和石雕所吸引，另一些智慧生物可能会被巨人从他们的梦中寻求真理和洞察力的激情所感染。

这些生物可以出现在地面世界，而不是在典型的巨大岩石隧道中。也许它们是在跟随巨人寻找表面的“梦境世界”，或者他们正在巨人出现时与他们进行贸易和交流。
