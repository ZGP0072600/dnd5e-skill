# 巨人类遭遇

**巨人类遭遇 Giantkind Encounters**

这部分包括怪物手册中六种巨人的遭遇表，以及 **死亡巨人 **（可以在第六章中发现）与 ** 废陋巨人 **的遭遇表。每张表格都包括巨人以及可能在巨人家里或周围发现的生物，包括那些通常被巨人当作宠物的生物（比如云巨人的 ** 斑巨狮 **，山巨人的 ** 恐狼 **）。

> **边栏:挑个巨人，哪个都行 Pick a Giant, Any Giant** 如果你想随机确定在某个地区占统治地位的巨人的种类，请投 **“巨人遭遇”** 表。您还可以投两次，并使用生成的两个表在指定的巨人之间创建合作或竞争关系。
