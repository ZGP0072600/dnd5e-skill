# 巨人坟冢遭遇

**巨人坟冢遭遇 Giant Necropolis Encounters**

使用巨人坟冢遭遇表来适配那些巨人曾经兴盛的地方，这些地方已经被摧毁、衰落、或屈服于死灵魔法。死亡巨人的干预或堕影冥界的影响都可能导致一个巨人飞地迅速堕落成一座坟冢，像是这张表格上的这些生物们将在那里蓬勃发展。

即使这次冒险不是行尸走肉的主题，你也可以用这些遭遇来增添一种绝望的味道，或用以暗示不死巨人们对某一地点感兴趣的神秘原因。
