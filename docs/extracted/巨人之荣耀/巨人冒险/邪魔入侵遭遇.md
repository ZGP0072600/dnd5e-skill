# 邪魔入侵遭遇

**邪魔入侵遭遇 Fiendish Incursion Encounters**

在以下情况下可以使用邪魔入侵遭遇表：巨人不再向序位之神服务，转向为恶魔服务（如第二章所述）；失落巨人王国的古老魔法打开了通往下层位面的传送门（也许是无意的也许是有意的）；魔法符文出现可怕的错误并将恶魔腐化释放到了世界；亦或者恶魔在凡人中寻求强大盟友以加速或完成他们的邪恶计划。
