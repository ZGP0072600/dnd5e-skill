# 霜巨人行囊

**霜巨人行囊 Frost Giant Bags**

霜巨人的行囊涂有防水漆，内衬是厚厚的毛皮，以保证包里的东西在冰霜巨人北极领地的恶劣天气中保持干燥。除了武器和旅行用品，霜巨人还喜欢随身携带战利品来炫耀。

**霜巨人行囊内容：**
