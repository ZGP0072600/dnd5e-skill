# 云巨人行囊

**云巨人行囊 Cloud Giant Bags**

大多数云巨人行囊都是由多彩染布缝制而成，并镶有金边。带着这类行囊的巨人乐于展示他们的全部财富，并且他们会携带奢华玩意来进一步彰显他们的富足。

还有一些云巨人行囊具有欺骗性：它们看似小巧扁平，但却暗袋密布，可以存放各种物品。背着这种行囊的巨人在他们的守护神曼诺诡计多端的一面找到了认同感，并以自己的聪明才智为荣。
