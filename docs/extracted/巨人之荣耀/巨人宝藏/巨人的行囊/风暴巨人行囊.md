# 风暴巨人行囊

**风暴巨人行囊 Storm Giant Bags**

由于喜好卜筮，许多风暴巨人都有迷信的倾向，时刻在世界上寻找预兆。他们经常随身携带许多护身符和符咒，这些饰品反映了风暴巨人的来处。生活在海中的风暴巨人可能会使用特殊的海草和贝壳来占卜。经常呆在天空中的风暴巨人可能会把闪电视为好运的预兆，并收集被闪电击中的物体或类似闪电的物品，将它们制成护身符。
