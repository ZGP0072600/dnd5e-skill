# 火巨人行囊

**火巨人行囊 Fire Giant Bags**

火巨人中不乏技艺精湛的铁匠，其中一些烧制粘土，吹制玻璃以创造华丽的作品。

火巨人通常都很务实，只携带必要的工具和最优秀的作品。他们的行囊是用厚皮革等弹性材料缝制的，可以承受锻造的高温和火巨人自身的灼热。
