# 宝藏种类

**宝藏种类 Kinds of Treasure**

请将以下的这些指南，与《城主指南》中的文本结合使用，以此决定出冒险者们在巨人建造的古代遗迹中能够发掘的那些宝藏。

**钱币 Coins** 巨人文明的钱币乃铜、银、银金、金、与铂金所铸。这些为巨人设计的钱币的尺寸自然也与冒险者们使用的不同它们大得多（大部分在巨人行囊中发现的钱币都是近代文明制造的普通钱币）。

不同文明的巨人的钱币，其外形、设计各色各样，但通常它们直径在4寸左右，厚度至少有1/4寸。由于钱币的造价明显比小钱币更大，这些钱币的价值自然也是小钱币的十倍。举例来说，一枚巨铜币价值10cp（或1sp），一枚巨金币价值10gp（或1pp）。

这些巨人铸造的钱币同样也比小钱币重得多，单单5枚任何类型的巨钱币就能重达1磅。

****

**宝石 Gemstones** 对于绝大部分巨人而言，这些小不点们切割制成的珠宝或其他珍贵石头都太小了，巨人们根本难以注意到这些石头，更别提发现这些宝石的价值了。但远古巨人们也许发现了采集并切割与他们体型相仿的宝石的技术，而他们也可以从那些 
巨蚌 giant clams 身上获取巨大的珍珠。

你可以用《城主指南》中记述的“宝石”章节来随机生成那些能在巨人宝库中找到的珠宝。只不过，一颗巨型的切割宝石是其正常价值的10倍。举例来说，一颗巨型切割 
石榴石 Garnet ，其在《城主指南》中处于100gp 宝石表中，但它的巨型版本价值1000gp。

**艺术品 Art Objects**

在古代巨人以及中发掘到的艺术品往往反映着曾经居住于此的巨人社会与那些巨人的类型。比如，古代石巨人的艺术品也许是雕刻的精致无比的石制雕塑，且刻印着一些靠触摸来阅读的文本一种让那些不适应无光深渊的客人们也能读懂上面的文本的系统（盲文）。而风暴巨人的艺术品可能会反映那些巨人们居住的环境，在他们的宝库中，冒险者可能发现珍珠与贝壳，也可能发现闪电电镀的金属制品与充满狂风暴雨之力的珠宝。

你可以结合《城主指南》中的表格，加之下述的表格来为巨人的特定遗迹或宝库生成与众不同的巨型艺术品。
