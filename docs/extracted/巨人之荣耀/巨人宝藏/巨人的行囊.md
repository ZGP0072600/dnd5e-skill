# 巨人的行囊

**巨人的行囊 Giant Bags**

奔波中的巨人扛着巨大的麻包，里面装着多年收集来的杂物。巨人的行囊是个奇特的储藏室，包含不寻常的纪念品，超巨形工具以及供巨人食用的粮食与饮品。本节为确定每类巨人包中到底有什么提供了一些思路。

本节中的每张表都包括钱币（取代《城主指南》中的 
"个体宝藏 
"表）、用以投掷的巨石以及其他一些通过在另一张表上掷骰子来确定的非魔法物品。这些物品包括贵重物品、食物、工具和奇珍异宝。任何巨型袋子里还可能装有一个行囊果冻 bag 
jelly（见第 6 章）。
