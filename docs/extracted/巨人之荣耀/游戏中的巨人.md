# 游戏中的巨人

**第二章：游戏中的巨人 Giants in Play**

本章是专为地下城主准备，讲述了巨人日常和社会的关键方面。当你准备在《龙与地下城》中使用巨人时，本片包含的灵感和表格或许能给你带来些许启发。本材料分为以下五个主要部分：

**“扮演巨人”** 讨论了如何让你游戏里的巨人活灵活现。

**“序位”** 探索了不同巨人种类之间和内部之间的关系，重点是序位作为冒险推动力时的作用。

**“诸神与宗教”** 则描述了全父安南与其孩子们所组成的神系。它还包括了林林总总的“外神”是怎么在安南缺位时介入巨人崇拜的。

**“社会结构”** 部分则探讨了巨人社会的各种模式，以及巨人与其他生物之间的关系。

**“组织”** 详细介绍了几个组织，这些组织将各个领域的巨人们团结了起来，用共同的信念和目标使他们走到一起。

与本书其余部分一样，本章主要关注《怪物图鉴》中所描述过的巨人主族，不过其中的一些建议也会适用于其他巨人类型的生物。
