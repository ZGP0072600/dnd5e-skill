# 迪安卡斯特拉萨迦

> *历经百战，疲惫不堪，在旅程中披上伤疤，迪安卡斯特拉走进了全父的厅堂。她冷静而又自信，索要她应得的事物，生而有之的权利，来自她父神圣的礼物*。
>
> *“智慧者，全父安南，古老的世界塑造者啊，在时间的黎明，你铸下伟大的功业。你在第一世界的破碎之后播种；你用梦想创造了宇宙，世界丰饶，奇迹显著。”*
>
> *“六位神子，女神降生，手握权柄，把那诸界分统。但们行动笨拙，举止粗鲁，让那任务失就。我虽不过凡女所生，却更堪配神座。”*
>
> *全智者安南面带愠怒，向女儿问出*：
>
> *“我的儿子们有着强大力量，你同父异母的姐姐，西亚缇，也显现一身英胆，将多头孽物屠戮。你又做出了什么成就？”*
>
> *“我解开了斯芬克斯的谜语，赢得了她的宝物我探究了大海的秘密。冒险进入受尊敬的龙虾之神迷藏宝库，偷走她爱惜的珍珠也包括你这秘密的圣所。”*
>
> *“这不过是恶作剧和叛逆！证明你的价值所在，而不是这种如同小孩子的任性。我要试炼你的心性；接受一项挑战：带给我纯粹的完美，还得是我也不能超越的。”*
>
> *迪安卡斯特拉毫不畏惧，离开了圣所。她航向了彼岸的水域。航过被遗忘的海洋，进入混沌的风暴，宣示了自己所找到的一粒物质，无形，无相，变化无穷，全父嘲笑于她，然后使之变化，变作一片肥沃的良田。很快，摇起了头，并不满意，又再次变化，变作风暴肆虐的汪洋大海，变作炽热不息的火炉山峰。最终，迪安卡斯特拉笑着说*：
>
> *“放弃你的诡诈吧！混沌是用作创作的画布，无拘无束，完美之物没有止境，无法形塑。便是你也不能完美地发挥出未显现的潜力。”*
>
> *安南的大厅被笑声撼动，震耳欲聋，摇晃着地面与天空。全父不言，承认了失败，答应了迪安卡斯特拉的要求，赐予她神性的火花，将之点亮*。
>
> *“我裁定儿子们的行为时太过武断，只是忙着将们谴责，现在我也将再次观察们，看看们是否拥有，如你这样的智慧，看看是否有未显现的完美。”*
>
> *将混沌打造成了一顶冠冕，满怀感激地授予给她。神圣迪安卡斯特拉离开了的圣所，怀抱着受苦手足们的希望，全父已然预到*。
