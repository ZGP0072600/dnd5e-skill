# 巨魔聚合体

**巨魔聚合体 Troll Amalgam**

巨魔聚合体是多只巨魔在再生期间融合在一起形成的。有时，当多只巨魔在一场战斗中同时被击杀，巨魔的身体部位被丢入藏尸处或大坟坑，或是通过邪恶施法者的扭曲仪式时，这一物种就会诞生。有时，瓦普拉克的巨魔追随者们会听到一个号召，他们将聚集在幽暗地域中进行一场骇人的仪式，他们将在那里融合为巨魔聚合体，其被视为是他们那恐怖之神的化身。

巨魔聚合体是一团由橡胶血肉、爪子和脸庞构成的无形肉体。当有生物尝试在其触及范围内攻击它或是从一条小通道中逃跑时，巨魔聚合体会从其身体上撕下一部分投向该生物。这些身体部件能够最终再生为一只完整的巨魔，且有时他们还会重新聚集形成一只全新的巨魔聚合体。

** 魔法抗性 Magic Resistance.** 巨魔聚合体在进行抵抗法术和其他魔法效应的豁免检定时具有优势。
