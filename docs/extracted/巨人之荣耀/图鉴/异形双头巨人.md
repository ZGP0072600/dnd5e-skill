# 异形双头巨人

**双头巨人心变者 Ettin Ceremorph**

《怪物图鉴》中的夺心魔由灵吸怪蝌蚪植入类人生物脑中后经蜕变而来。夺心魔曾尝试在巨人的身体上实验这一过程以培养出更加强大的夺心魔，但这些尝试都以失败告终：巨人的庞大身体是一个小蝌蚪难以掌控的。

然而，双头巨人被证实是最完美的实验品。双头巨人的两颗头颅为两只蝌蚪提供了充足的食物，使其能够完整转化双头巨人的身体。在转化过程中，双头巨人的其中一颗头颅会沉入身体之内，那颗头用以控制身体。而另一颗头则专注于认知事物和施展灵能力量，尽管其灵能威力并不如夺心魔一样强大。

双头巨人心变者侍奉于夺心魔群体，保护着掌管社群的主脑以及守卫他们的财宝。他们已经失去了曾经作为双头巨人的任何记忆。
