# 风暴先声

**风暴先声 Storm Herald**

在海洋深处有一群古老的异质存在，或是于此生息，或是于此沉睡，待到某种宇宙环境的改变发出信号之刻，便是他们升起之时。某些风暴巨人在从世界中撤退的时候，可能会为了寻求某种预兆，愿景或者预言的解析而误入这些生物的轨道。克拉肯，底栖魔鱼还有旧日支配者们都乐于将风暴巨人们腐化成自己的奴仆，将他们变成风暴先声。

　　风暴先声会长出鱼鳍和触手，给予他们怪物般的外表。他们的思想已经完全处于主人的控制之下，先声还获得了可怕的灵能力量用以推进其主人的计划。当先声死后，他们会变回原本的形貌。

　　风暴先声可能会尝试说服其他风暴巨人来为他的主人服务，而其他的则会让次等生物们聚集成邪教来服务深海中的怪物。
