# 巨猞猁

**巨猞猁 Giant Lynx**

尽管它们常常被误认为是普通猫儿，但巨猞猁是一种源自妖精荒野白雪皑皑的林地的魔法生物。它们的毛皮外套在冬天颜色更浅，能够让它们哪怕是大地银装素裹的时候也能在自己最喜欢的栖息地中进行伪装。

巨猞猁是狡猾的猎人，它们以小团队的形式合作捕杀猎物，偶尔还能用魔法的天赋来侦查可能的狩猎地点。它们同样足够聪明，能够智胜许多试图诱捕它们猎取皮毛的猎手，还喜欢破坏在自己领地上发现一切捕猎者的圈套。

霜巨人和云巨人有时会和巨猞猁交朋友，并把它们当做心爱的宠物。只要猞猁的智慧和独立自主得到尊重，它们就会乐意和巨人伙伴分享他们的预言魔法。

**猞猁之眼 Lynx’s Sight（1/每日）。** 猞猁能够施展*鹰眼术 clairvoyance*，无需任何法术成分，使用感知作为其施法关键属性。

**林地伪装 Woodland Camouflage。** 猞猁在灌木或雪地地下中进行的隐匿检定具有优势。

 **动作 Actions**

** 爪击 Claw。**近战武器攻击：命中+6，触及5尺，单一目标。命中：8（1d8+4）挥砍伤害。如果巨猞猁在攻击前向目标直线移动至少20尺然后命中目标，则目标必须进行一次 DC 
12的力量豁免，豁免失败者将应击倒地。如果目标处于倒地状态，则该巨猞猁可以用一个附赠动作对目标再发动一次爪击。
