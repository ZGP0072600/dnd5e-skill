# 邪气之云巨人

**邪气之云巨人 Cloud Giant of Evil Air**

考虑到他们和气元素的内在联系，背弃序位诸神的云巨人常常被邪恶气元素亲王延西冰所吸引。这些巨人狡猾而迷人，纯粹的肉体力量和魔法能力会让他们极为适合担任邪恶元素教派的领导职位。一个以云巨人为头领的邪教可能成为一股强大的力量，在世界上散播腐化和毁灭。而仿佛这些还不足以证明自己的能耐一般，云巨人们常常会带来巨额的财富，富足了邪教的金库，为更深远的运营提供资金。

一个忠于邪恶大气的云巨人身穿一件魔法背心，其上装着由大鹏羽毛制成的翅膀，上面附着的气元素魔法让巨人能够飞行。这件背心只对穿着它的巨人生效。

**邪气之云巨人 Cloud Giant of Evil Air**

*巨型巨人，通常中立邪恶*

AC 
14（天生护甲）

HP 
200（16d12+96）

速度 
40尺，飞行40尺

力量27（+8）敏捷10（+0）体质22（+6）

智力12（+1）感知16（+3）魅力19（+4）

豁免：体质+10，感知+7，魅力+8

技能：洞悉+7，察觉+7，隐匿+4

感官：被动察觉17

语言：气族语，通用语，巨人语

挑战等级12（8,400XP）熟练加值+4

**飞掠 Flyby。** 巨人飞出敌人的触及范围时不会引起借机攻击。

 **动作 Actions**

** 多重攻击 Multiattack。**巨人进行两次弯刀攻击或两次风暴回旋镖攻击。

** 弯刀 Scimitar。**近战武器攻击：命中+12，触及10尺，单一目标。命中：18（3d6+8）挥砍伤害。

** 风暴回旋镖 Storm Boomrang。**远程法术攻击：命中+12，射程60/120尺，单一目标。命中：15（3d4+8）钝击伤害，外加7（2d6）雷鸣伤害，并且目标必须成功通过 DC 16的体质豁免否则直到自己的下个回合结束陷入震慑状态。无论命中与否，回旋镖都会在攻击后因魔法自动飞回巨人手中。

** 施法 Spellcasting。**巨人施展以下一个法术，不需要任何材料成分且使用魅力作为施法关键属性（法术豁免 DC 16）。

随意：*侦测魔法 detect magic，云雾术 fog cloud，光亮术 light*

2/日：*心灵遥控 telekinesis*

1/日：*气化形体 gaseous form*
