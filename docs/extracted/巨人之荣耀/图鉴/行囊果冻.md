# 行囊果冻

**行囊果冻 Bag Jelly**

行囊果冻是呈橄榄绿色的泥怪，其外形与那些在《怪物图鉴》中的记述的地城清道夫们差不多。如它们名字所示，对，这些生物最常见于那些巨人们带着的行囊之中，将发现的任何有机材料都当做它们的食粮。行囊果冻有着十分强大的抗挤压能力，这也是他们赖以在巨人手下生存的秘诀——当然了，包括但不限于在行囊被巨人扔来扔去、被巨人一屁股坐下去、被巨人随手拿起来当钝器用的情况下生存。

少部分巨人会想办法清理自己的行囊来根绝行囊果冻。但更多的巨人利用这些小清道夫来阻碍窃贼。大部分时候，那些想趁着巨人睡觉偷偷翻找行囊的窃贼都会发现自己不知不觉中就被行囊果冻黏着上了，而窃贼的挣扎声会吵醒那些刚才还在睡眠的巨人，让窃贼错失逃跑的机会。
