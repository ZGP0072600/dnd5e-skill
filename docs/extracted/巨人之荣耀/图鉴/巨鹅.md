# 巨鹅

**巨鹅 Giant Goose**

巨人饲养鹅作为产蛋的家畜，因为鹅是优秀的哨兵，能用它们敏锐的感官和响亮的叫声保卫巨人的财产。拥有妖精血统的巨鹅不仅比普通的鹅大得多，它们还具有两项使它们变得更为珍贵的魔法性特征：它们的嘎嘎叫声是抵御入侵者的强大手段，而且它们能产下拥有纯金蛋壳的蛋。

当它们的饲养员下令时，一只巨鹅可以产下一枚金蛋一个长1尺、重1磅、价值100gp 的空心黄金壳。有时候，蛋里会莫名其妙地含有某种小饰品或小魔法物品。一旦巨鹅产下了一枚金蛋，它就不能在一年零一天内再次这样做了。

你可以使用鹅蛋饰品表来确定一枚金蛋的内容物。一枚巨鹅的蛋可以是一种给予角色们一个对冒险故事或你的战役来说至关重要物品的好方法。

感官：黑暗视觉60尺，被动察觉16

语言：懂巨人语和木族语但不会说

熟练+2

**动作 Actions**

** 多重攻击 Multiattack。**鹅发动一次喙啄攻击和两次振翅攻击。

** 喙啄 Beak。**近战武器攻击：命中+4，触及10尺，单一目标。命中：9（2d6+2）点钝击伤害。

** 振翅 Wing。**近战武器攻击：命中+4，触及5尺，单一目标。命中：7（1d10+2）点钝击伤害，且目标必须成功通过一次 DC 12的力量豁免，否则应击倒地 prone。

** 雷鸣嘎嘎 Thunderous Honk（充能5~6）。**鹅发出震耳欲聋的嘎嘎叫声。每个在鹅30尺内的其他生物都必须进行一次 DC 12的体质豁免，失败则受到16（3d10）点雷鸣伤害且直至鹅的下一回合开始前陷入耳聋 deafened 状态，成功则只受到一半伤害。这叫声在300尺内都能被听见。
