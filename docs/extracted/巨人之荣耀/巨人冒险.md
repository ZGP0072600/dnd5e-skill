# 巨人冒险

**第三章:巨人冒险 Chapter3: Giant Aventures**

这一章是为地下城主设计的，里面有大量的表格和灵感，您可以用这些表格和灵感来创建遭遇、冒险、整个世界和战役，让巨人扮演合适的巨人角色。本章共分为三个部分：

**“遭遇”** 提供了工具来帮助您建立涉及巨人和经常伴随他们的生物的遭遇战。您可以使用本章节中的表格来填充第四章或任何其他遭遇中的巨人主题相关部分。

**“冒险”** 包括冒险引子和其他工具，您可以用它们来制作一个涉及巨人和他们的世界的冒险。

**“战役”** 可以帮助您在游戏世界中建立巨人的角色，并将这些过去或现在的强大民族置于战役的最前沿。
