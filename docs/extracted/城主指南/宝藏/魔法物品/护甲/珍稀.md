# 珍稀

**抗性护甲 Armor of Resistance**

*护甲（轻甲、中甲、或重甲），珍稀（需同调）*

着装这套护甲期间，你获得一种特定伤害类型的抗性。具体伤害类型由 DM 选择，或由 DM 从以下选项中随机选定。

| d10 | 伤害类型 |  | d10 | 伤害类型 |
|---|---|---|---|---|
| 1 | 强酸 |  | 6 | 暗蚀 |
| 2 | 寒冷 |  | 7 | 毒素 |
| 3 | 火焰 |  | 8 | 心灵 |
| 4 | 力场 |  | 9 | 光耀 |
| 5 | 闪电 |  | 10 | 雷鸣 |

**易伤护甲 Armor of Vulnerability**

*护甲（板甲），珍稀（需同调）*

着装此护甲期间，你具有以下三种伤害类型之一的抗性：钝击、穿刺、或挥砍。具体伤害类型由 DM 选择，或由 DM 随机选定。

> **诅咒 Curse。** 这是一件被诅咒的护甲，这一点只有在你同调此护甲或对它使用 *鉴定术 identify* 时才会揭晓。同调此护甲将会使你被诅咒。卸除该护甲不会终止诅咒，只有对你施展法术 
>  *移除诅咒 remove curse* 
>  或以类似的魔法才能将其移除。此诅咒使你具有上述三种伤害类型中另外两种的易伤（除护甲提供抗性的类型之外）。

**精灵链甲 Elven Chain**

*护甲（链甲衫），珍稀*

着装该护甲期间，你的 AC 获得+1加值。即使没有中甲的熟练项，你依然视为拥有该护甲的熟练项。

**幻惑镶钉皮甲 Glamoured Studded Leather**

*护甲（镶钉皮甲），珍稀*

着装该护甲期间，你的 AC 获得+1加值。此外，你可以用一个附赠动作说出该护甲的命令语，使护甲的外观幻化为任何普通服饰或其他种类的护甲。幻化后的颜色、样式、与配件都由你自己决定，但它的体积和重量不会改变。幻化外观的效应持续到你脱掉此护甲后或再次改变护甲外观时为止。

**吸矢盾 Arrow-Catching Shield** *护甲（盾牌），珍稀（需同调）*

持用这面盾牌期间，你对抗远程攻击的 AC 具有+2加值。该加值将加在盾牌通常提供的 AC 加值上。此外，当又攻击者向你周围5尺范围内的目标发动远程攻击时，你可以使用反应以替代该目标成为此次攻击的目标。

**引弹盾 Shield of Missile Attraction**

*护甲（盾牌），珍稀（需同调）*

持握此盾牌期间，远程武器攻击对你造成伤害时，你具有该伤害的抗性。

> **诅咒 Curse。** 与这面受诅咒的盾牌同调的生物将受到其诅咒影响。摘下盾牌不会终止诅咒，只有对你施展 
>  *移除诅咒 remove curse*或类似的魔法才能移除。因诅咒的影响，任何向你周围10尺内的目标发动的远程武器攻击都将转为攻击你。
