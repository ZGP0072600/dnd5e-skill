# 极珍稀

**恶魔护甲 Demon Armor**

*护甲（板甲），极珍稀（需同调）*

着装该护甲期间，你的 AC 获得+1加值，且可以理解并说出深渊语。另外，护甲如利爪般的护腕会令你的徒手打击（仅限手）转化为造成挥砍伤害的魔法武器，其伤害骰为1d8，且你以此发动的攻击检定与其伤害掷骰具有+1加值。

> **诅咒 Curse。** 着装这件被诅咒的护甲后，除非你被施予法术 
>  *移除诅咒 remove curse* 
>  或类似的魔法效应，否则你无法将其卸下。着装该护甲期间，你对恶魔进行的攻击检定具有劣势，且你对抗恶魔的法术与特殊能力进行的豁免检定同样具有劣势。

**龙鳞甲 Dragon Scale Mail**

*护甲（鳞甲），极珍稀（需同调）*

龙鳞甲是以某一种龙的鳞片制成的护甲。有时候是龙送给类人生物作礼物的褪下鳞片，有时候是猎人们小心的剥下的死去龙类的鳞与皮。无论如何，龙鳞甲都是非常珍贵的物品。

 
着装该护甲期间，你的 AC 获得+1加值，且为对抗龙类的骇人威仪特性与吐息武器时所作豁免具有优势。此外，根据龙鳞甲使用的龙鳞种类，你还会获得对一种伤害类型的抗性（见下表）。

另外，你还可以用一个动作集中精神来感知距你最近，且与龙鳞甲的鳞片提供者类型相同的龙类，并得知其相对于你的方向与距离。这条龙必须在你周边30里以内且与你处于同一位面。这一特殊动作一经使用，直至次日黎明前都无法被再度使用。

| 龙种 | 抗性 |  | 龙种 | 抗性 |

| 黑龙 | 强酸 |  | 金龙 | 火焰 |

**矮人板甲 Dwarven Plate**

*护甲（板甲），极珍稀*

着装该护甲期间，你的 AC 获得+2加值。此外，当某种效应将要在违背你意愿的情况下使你沿地面移动时，你可以用反应来减少你被移动的距离，至多减少10尺。

**活化盾 Animated Shield**

*护甲（盾牌），极珍稀（需同调）*

持握此盾牌期间，你可以用附赠动作说出其命令语使其活化。盾牌将飞到空中并悬浮在你所占据的空间中保护你，如同你正持用着它一般，且同时解放你的双手。

盾牌保持活化1分钟，如果你陷入 **失能 Incapacitated**、死亡、或用附赠动作终止其效应时，活化会提前结束。如果活化结束时你有空闲的手手空着，盾牌将落回你手里。否则它会掉落在地上。

**抗法盾 Spellguard Shield**

*护甲（盾牌），极珍稀（需同调）*

持握此盾牌期间，你对抗法术与其他魔法效应所作的豁免检定具有优势，而对你发动的法术攻击检定具有劣势。
