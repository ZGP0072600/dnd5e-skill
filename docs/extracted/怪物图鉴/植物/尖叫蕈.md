# 尖叫蕈

尖叫菌 Shrieker

尖叫菌是像人一样大的蘑菇，会在被打扰时发出刺耳的尖叫。有些生物会利用这种真菌发出入侵的预警信号，而很多幽暗地域的智慧种族会在城市的郊区成片种植尖叫菌，用以把不速之客拒之城外。

尖叫菌 Shrieker

中型植物，无阵营

护甲等级：5

生命值：13（3d8）

速度：0尺

力量1（-5）
敏捷1（-5）
体质10（+0）智力1（-5）
感知3（-4）
魅力1（-5）

状态免疫：目盲，耳聋，恐慌

感官：盲视30尺（此范围外视为目盲），被动察觉6

语言：——

挑战等级：0（10 
XP）

拟形 False 
Appearance。静止不动的尖叫菌与普通的真菌一模一样。

反应

尖叫 Shriek。尖叫菌在其身边30尺内出现生物或光亮时发出尖叫，其周围300尺范围内都能听到声音，尖叫声持续到干扰离开警戒范围，或是持续到尖叫菌发声1d4回合。
