# 耶诺古毒牙（邪魔）

鬣狗人·耶诺古毒牙 Gnoll 
Fang of Yeenoghu

鬣狗人主持恶魔仪式来庆祝胜利，并为耶诺古献祭鲜血。恶魔领主有时也会奖励这些崇拜者，以让其中一名被恶魔的精魄附身。这名被标记为耶诺古宠儿的鬣狗人将幸运的被选为耶诺古毒牙，鬣狗人领主的选民。耶诺古毒牙所杀死的敌人喂食给普通鬣狗时，被喂食者便会如同耶诺古所制造的第一批鬣狗人一样，在经历一场可怕转变之后彻底成长为一名成年鬣狗人。根据军团中鬣狗的多寡，耶诺古毒牙可能会使其鬣狗人同胞的数量急剧增长。而设法找到并杀死这些毒牙是控制鬣狗人人口的唯一方法。

横行 Rampage。鬣狗人在其自己回合以一个近战攻击使某个生物生命值降至0时，该鬣狗人可以执行一个附赠动作以移动至多等同于其速度一半的距离并发动一次啃咬攻击。

动作

多重攻击 Multiattack。鬣狗人发动三次攻击：一次啃咬和两次爪击。

啃咬 Bite。近战武器攻击：命中+5，触及5尺，单一生物。命中：6（1d6+3）点穿刺伤害，目标必须进行一次 DC 
12的体质豁免，豁免失败则受到7（2d6）点毒素伤害。

爪击 Claw。近战武器攻击：命中+5，触及5尺，单一目标。命中：7（1d8+3）点挥砍伤害。
