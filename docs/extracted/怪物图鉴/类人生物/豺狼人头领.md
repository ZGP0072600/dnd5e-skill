# 豺狼人头领

鬣狗人头领 Gnoll 
Pack Lord

鬣狗人集群中的阿尔法即鬣狗人头领，靠着自身的力量与狡猾统领狼群。头领会独占整个鬣狗人群战利品中最好的赃物、食物、有用的小饰品和魔法物品。它会用一些野蛮的穿刺行为和奇形怪状的战利品装饰自己的身体，将恶魔般的印记染在自己的毛皮上，以此祈求耶诺古让自己变得所向披靡。

鬣狗人头领 Gnoll Pack Lord

中型类人（鬣狗人），混乱邪恶

护甲等级：15（链甲衫）

生命值：49（9d8+9）

速度：30尺

力量16（+3） 敏捷14（+2） 体质13（+1）智力8（-1） 感知11（+0） 魅力9（-1）

感官：黑暗视觉60尺，被动察觉10

语言：鬣狗人语

挑战等级：2（450 
XP）

横行 Rampage。鬣狗人在其自己回合以一个近战攻击使某个生物生命值降至0时，该鬣狗人可以执行一个附赠动作以移动至多等同于其速度一半的距离并发动一次啃咬攻击。

动作

多重攻击 Multiattack。鬣狗人发动两次长柄刀攻击，或发动两次长弓攻击，如果条件允许则还可以再使用它的煽动横行

啃咬 Bite。近战武器攻击：命中+5，触及5尺，单一生物。命中：5（1d4+3）点穿刺伤害。

长柄刀 Glaive。近战武器攻击：命中+5，触及10尺，单一目标。命中：8（1d10+3）点挥砍伤害。

长弓 Longbow。远程武器攻击：命中+4，射程150/600尺，单一目标。命中：6（1d8+2）点穿刺伤害。

煽动横行 Incite 
Rampage（充能5~6）。指定鬣狗人周边30尺内它能看见的一个生物，如果该生物拥有横行 Rampage 特质且能够听到该鬣狗人的声音，则该生物可以用它的反应发动一次近战攻击。
