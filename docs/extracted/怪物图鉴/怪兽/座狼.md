# 座狼

座狼 Worg

座狼是一种喜欢捕猎和噬食比自己弱小生物的邪恶捕食者。狡猾而恶毒的座狼游荡在偏远的野外，或由地精和大地精驯养。这些生物会这些生物会将座狼用作坐骑，只是座狼稍感被亏待或是需要营养补给时，便会转而对付其骑乘者。座狼会说自族的语言和地精语，其中某些甚至还会说通用语。

动作

啃咬 Bite。近战武器攻击：命中+5，触及5尺，单一目标。命中：10（2d6+3）点穿刺伤害。如果目标是生物，则目标必须进行一次 DC 
13的力量豁免，豁免失败则应击倒地。
