# 德鲁伊

**德鲁伊 Druids**

德鲁伊居住在深林里或其他隐蔽的野外地点中。他们保护自然世界使之免受怪物破坏，并远离文明世界的侵蚀。某些德鲁伊会作为 **部族萨满 tribal shamans** 
向动物之灵祈祷，并且为人治愈疾病或担任其他人的心灵导师。

**德鲁伊 Druid**

*中型类人（任意种族），任意阵营*

护甲等级：11（带树肤术16）

生命值：27（5d8+5）

速度：30尺

力量10（+0） 
敏捷12（+1） 体质13（+1）

智力12（+1）

魅力11（+0）

技能：医药+4，自然+3，察觉+4

12，法术攻击命中+4）。德鲁伊准备了以下德鲁伊法术：

戏法（随意）：*德鲁伊伎俩 druidcraft，燃火术 produce flame，橡棍术 shillelagh*1环（4法术位）：*纠缠术 entangle，大步奔行 longstrider，动物交谈 speak with animals，雷鸣波 thunderwave*

2环（3法术位）：*动物信使 animal messenger，树肤术 barkskin***动作 **
