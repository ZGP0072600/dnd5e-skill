# 邪教徒

**邪教徒 Cultists**

元素亲王、恶魔领主或是大魔鬼们，各方黑暗力量各自存在着对其宣誓效忠的邪教徒 cultists。这些邪教徒大多隐藏着自身的信仰，以免因此遭受他人排挤、监禁甚至处死。与邪恶的侍僧不同，邪教徒们的行为和信仰通常都更为疯狂。

**邪教徒 Cultist**

*中型类人（任意种族），任意非善良阵营*

护甲等级：12（皮甲）

生命值：9（2d8）

速度：30尺

力量11（+0） 
敏捷12（+1） 体质10（+0）

智力10（+0） 
感知11（+0） 
魅力10（+0）

技能：欺瞒+2，宗教+2

感官：被动察觉10

语言：任意一门语言（通常是通用语）

挑战等级：1/8（25 
XP）

**黑暗忠诚 Dark Devotion。** 邪教徒为抵抗魅惑和恐慌而进行的豁免检定具有优势。

**动作 **

** 弯刀 Scimitar。**近战武器攻击：命中+3，触及5尺，单一生物目标。命中：4（1d6+1）点挥砍伤害。
