# 强盗

**强盗 Bandits**

四处游荡的强盗经常由 **暴徒 thug**，** 老兵 veteran**以及施法者所带领。并不是所有强盗都邪恶。苛政、旱灾、疾病以及饥荒都会逼迫殷实的普通民众无奈的过上强盗生活。

** 海盗 pirates**则是公海里的强盗。他们可能是只对宝藏和杀戮感兴趣的掠夺者，也可能是由王权默许的私掠船队，用以专门掠劫敌对国家的船只。
