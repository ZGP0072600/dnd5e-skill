# 渡鸦群

渡鸦集群 Swarm of Ravens

微型野兽的中型集群，无阵营

护甲等级：12生命值：24（7d8-7）速度：10尺，飞行50尺

力量6（-2）
敏捷14（+2）
体质8（-1）智力3（-4）
感知12（+1）
魅力6（-2）

技能：察觉+5伤害抗性：钝击、穿刺、挥砍状态免疫：魅惑，恐慌，麻痹，石化，倒地，束缚，震慑，擒抱感官：被动察觉15语言：—挑战等级：1/4（50 
XP）

集群 Swarm。集群可以进驻另一生物身处的空间位置，反之亦然。而且集群可以通过任何足够一只微型渡鸦通过的通道。集群不能恢复生命值也不能获得临时生命值。

动作

喙啄 Beak。近战武器攻击：命中+4，触及5尺，单一目标。命中：7（2d6）点穿刺伤害，或者集群生命值等于或低于一半时3（1d6）点穿刺伤害。
