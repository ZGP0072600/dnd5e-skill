# 蝙蝠群

蝙蝠集群 Swarm of Bats

微型野兽的中型集群，无阵营

护甲等级：12生命值：22（5d8）速度：0尺，飞行30尺

力量5（-3）
敏捷15（+2）
体质10（+0）智力2（-4）
感知12（+1）
魅力4（-3）

伤害抗性：钝击、穿刺、挥砍

状态免疫：魅惑，恐慌，麻痹，石化，倒地，束缚，震慑，擒抱

感官：盲视60尺，被动察觉11

语言：—

挑战等级：1/4（50 
XP）

集群 Swarm。集群可以进驻另一生物身处的空间位置，反之亦然。而且集群可以通过任何足够一只微型蝙蝠通过的通道。集群不能恢复生命值也不能获得临时生命值。

回声定位 Echolocation。蝙蝠陷入耳聋时无法使用盲视。

敏锐听觉 Keen 
Hearing。蝙蝠依赖听觉进行的感知（察觉）检定具有优势。

动作

啃咬 Bites。近战武器攻击：命中＋4，触及0尺，单一身处集群内的目标。命中：5（2d4）点穿刺伤害，或者集群生命值等于或低于一半时2（1d4）点穿刺伤害。
