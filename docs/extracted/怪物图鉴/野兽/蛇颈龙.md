# 蛇颈龙

蛇颈龙 Plesiosaurus

蛇颈龙是一种长着强壮脚蹼，紧凑身躯的海生恐龙。富有掠夺性和侵略性的蛇颈龙会袭击所遇到的任何生物。蛇颈龙灵活的脖子可以长至三倍体长，这使得蛇颈龙可以向任何方向进行强有力的啃咬。

蛇颈龙 Plesiosaurus

大型野兽，无阵营

护甲等级：13（天生护甲）

生命值：68（8d10+24）

速度：20尺，游泳40尺

力量18（+4）
敏捷15（+2）
体质16（+3）智力2（-4）
感知12（+1）
魅力5（-3）

技能：察觉+3，隐匿+4

XP）
