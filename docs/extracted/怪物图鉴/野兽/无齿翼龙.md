# 无齿翼龙

无齿翼龙 Pteranodon

这些飞行蜥蜴拥有15尺到20尺的翼展并能以极小的角度向海面的猎物发动俯冲。而无齿翼龙善于把握机会，且乐于袭击任何看起来能下肚的生物。无齿翼龙如其名般没有牙齿，但会用尖长的啄碎猎物，然后整块吞下。

无齿翼龙 Pteranodon

中型野兽，无阵营

护甲等级：13（天生护甲）

生命值：13（3d8）

速度：10尺，飞行60尺

力量12（+1）
敏捷15（+2）
体质10（+0）智力2（-4）
感知9（-1）
魅力5（-3）

技能：察觉+1

感官：被动察觉11

语言：——

挑战等级：1/4（50 
XP）

飞掠 Flyby。无齿翼龙以飞行离开敌人的触及范围时不会引发对方的借机攻击。

动作

啃咬 Bite。近战武器攻击：命中+3，触及5尺，单一目标。命中：6（2d4+1）点穿刺伤害。
