# 索尔石怪

石鼎兽 Xorn

“口袋里有宝石时，饥饿的石鼎兽就是有用的石鼎兽。”

X 地牢求生奥妙第6条

奇异的石鼎兽是土元素位面的原生生物，他们从土层和岩层中嗅探出宝石和贵金属，并掘道穿越前去索取这些宝藏作自己的食粮。只是在主物质位面时，由于这些稀缺材料极其匮乏，石鼎兽必须穿越幽暗地域更深更远处觅食，并时常因此与矿工和寻宝者发生冲突。 石鼎兽非自然的血统起源从外表就能一眼认出。它的身躯沉重无比，头顶长一张强劲的大嘴。它有三条长着锋利爪子的长臂，以及可以同时看向各个方位的三颗石睑大眼睛。

元素旅者 Elemental Travelers。拥有土元素之力的石鼎兽，可以像水中的鱼一样遨游在土石之间。它穿越土层和岩层时，岩土会从它身边流过再重新合并，因而并不会在它通过处留下任何坑道洞穴或任何可见的踪迹。 石鼎兽更愿意待在自己的原生位面，那里可以随意就找到宝石和贵金属果腹。然而石鼎兽还是会因好奇或意外落入物质位面，并从此一边不停觅食一边寻路回家。

乞丐与窃贼 Beggars and Thieves。石鼎兽穿越地下寻找稀有金属和石头时，对其他无法吸收的“有机”生物毫无兴趣。然而，石鼎兽嗅探金属和石头的能力往往会使它们注意到冒险者们携带的金币和宝石。石鼎兽并不邪恶，因此它们一般情况下会试图恳求或说服对方，用自己提供的情报换取对方的财宝。如果石鼎兽的要求被无视，恼羞成怒的它会试图威胁或恐吓。而当它饥饿或愤怒时，甚至会诉诸武力。

遁地 Earth 
Glide。石鼎兽可以掘穴穿过非魔法且未经加工的泥土及岩石中移动。以此使用掘穴速度时，石鼎兽不会破坏其穿过的任何材质。

岩地伪装 Stone 
Camouflage。石鼎兽在岩地地形中为进行躲藏而作的敏捷（隐匿）检定具有优势。

财宝感知 Treasure 
Sense。石鼎兽可以通过气味确定在其 60 
英尺内贵金属和宝石的位置，比如硬币和宝石。

动作

多重攻击 Multiattack。石鼎兽发动三次爪击和一次啃咬。
