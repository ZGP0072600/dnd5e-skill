# 人工生命体

人工生命体 Homunculus

人们用一种粘土、灰烬、曼德拉根和血液的混合物塑型，并引导一种罕见的仪式魔法来制造一个松鼠般大小的忠实伙伴。 人工生命体是一种如其制作者身体延伸般活动的构装生物，它与其制作者以一种神秘的链接来分享思想、感官和语言。每个主人只能同时拥有一只人工生命体（试图创造另一个时总会失败），当它的主人死亡时人工生命体也会一同死去。

共享意识 Shared Mind。人工生命体知晓其制作者的一切，包括其能说会读的所有语言。而只要同处一个位面，即使两者相隔遥远，其主人都可以得知该构装生物感官所及的一切。人工生命体被用作密探、斥候、使者或信使，是那些困身于秘密实验或冒险中的施法者非常宝贵的仆从。

人工生命体 Homunculus

微型构装，绝对中立

护甲等级：13（天生护甲）

生命值：5（2d4）

速度：20尺，飞行40尺

力量4（-3）
敏捷15（+2）
体质11（+0）智力10（+0）
感知10（+0）
魅力7（-2）

伤害免疫：毒素

状态免疫：魅惑，中毒

感官：黑暗视觉60尺，被动察觉10

语言：懂其制作者的语言但不能说

挑战等级：0（10 
XP）

心灵连结 Telepathic 
Bond。当人工生命体与其主人处于同一位面时，它可以用魔法将自己的感官传递给主人，且两者可以通过心灵感应进行交流。

动作

啃咬 Bite。近战武器攻击：命中+4，触及5尺，单一生物。命中：1点穿刺伤害，且目标必须进行一次 DC 
10的体质豁免，豁免失败则陷入1分钟的中毒状态。如果豁免结果为5或更低，则该目标改为陷入5（1d10）分钟的中毒状态，且在该中毒状态持续的同时陷入昏迷。
