# 三元冢

三元冢 Tridrone

三元冢的形状像一个倒金字塔。它们在战斗中统领着更低阶的魔冢。

三元冢 Tridrone

中型构装，守序中立

AC15（天生护甲）HP16（3d8+3）速度：30尺。

力量12（+1）
敏捷13（+1）
体质12（+1）智力9（-1）
感知10（+0）
魅力9（-1）

感官：真实视觉120尺，被动察觉10语言：魔冢语挑战等级：1/2（100 
XP）

公理之念 Axiomatic 
Mind。三元冢不会被迫去做违反其天性或其原有指令的行为。

解体 Disintegration。三元冢死亡时，它的身体会瓦解为尘埃，留下它的武器和其他携带的东西。

动作

多重攻击 Multiattack。三元冢发动三次拳击攻击或三次标枪攻击。

拳击 Fist。近战武器攻击：命中+3，触及5尺，单一目标。命中：3（1d4+1）点钝击伤害。

标枪 Jacelin。近战或远程武器攻击：命中+3，触及5尺或射程30/120尺，单一目标。命中：4（1d6+1）点穿刺伤害。
