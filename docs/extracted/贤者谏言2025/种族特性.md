# 种族特性

半身人 Halfling 的 天生擅匿 Naturally 
Stealthy 特质能否让其在被他人看见的情况下进行躲藏？通常情况下，要执行躲藏动作，你需要以下环境因素：

 你必须身处重度遮蔽、位于四分之三掩护、或位于全身掩护下。

 你必须处于所有敌人的视线之外。
 半身人的天生擅匿特性可以让你在有体型比你大至少一级的生物遮蔽你的情况下，无视上述这些环境因素。不过，你依旧需要如同正常的执行躲藏动作那般，正常进行敏捷（隐匿）检定，所以你的躲藏并不一定能确保成功。
