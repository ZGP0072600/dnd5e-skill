# 法师

防护师 Abjurer 的 
奥术守御 Arcane Ward 只有在其生命值为0时才能恢复生命值吗？每当防护师消耗法术位去施展防护学派法术，都会为“奥术守御”恢复生命值，无论它当前剩余多少生命值。和普通的治疗一样，奥术守御恢复生命值也无法超过其生命值上限：法师等级的2倍+法师的智力调整值。

仪式施展 警报术 Alarm 是否能恢复 
奥术守御 Arcane Ward 的生命值？不能。以仪式施展某道法术不会用到法术位，而奥术守御仅仅会在防护师消耗法术位施展防护学派法术时，才能恢复生命值。

防护师 Abjurer 的 
奥术守御 Arcane Ward 与临时生命 Temporary Hit Points 之间的处理顺序是什么样的？当一个有着奥术守御的防护师承受伤害时，将准许以下的顺序处理：

1. **抗性与易伤。**如果法师本人有相应抗性或易伤，先对伤害整体处理这些内容。
2. **奥术守御。**奥术守御首先承受这一伤害。
3. **伤害免疫。**若奥术守御生命值被降至0，剩余部分的伤害将受到法师本人的免疫影响。
4. **临时生命值。**法师本人将承受剩下的这些伤害，如果他有临时生命值，先用临时生命值去承担伤害。
5. **真实生命值。**在临时生命值后，剩下的所有伤害由法师本人的生命值承担。

幻术师 Illusionist 
的14级能力是能让你每轮把一个幻象化为真实，还是只能生效一次？亦真亦幻能力的本意是让你在一道幻术系法术中，把一个幻象物件化为真实。

位于 
以太位面 Ethreal Plane 的 预言师 Diviner 
可以对他看到的、位于物质位面的生物使用 预兆 Portent 
吗?

是的。预兆仅仅需要你能看见该生物，它没有距离限制。

**对于 法师 Wizard 来说，要施展一道其法术书中的仪式法术，他们是需要阅读这本书，还是把法术书作为仪式的一部分?**

由于法师有其仪式学家特性，他们可以无需准备地，仪式施展法术书中有仪式标签的法术。不过这一特性是对仪式法术的基本规则的衍生。如果一名法师想要仪式施展一道未准备的法术，他就必须阅读他的法术书（如仪式学家的文本所示）。不过，这名法师若是想仪式施展一道已准备的法术，基于基本规则，他将不需要用到他的法术书（除非他打算用法术书作为此法术的施法法器使用）。
