# 冒险

**若我处于躲藏下，有 盲视 Blindsight 或 真实视觉 Truesight 的生物看见了我，我还算处于躲藏下吗？** 不。处于躲藏下是一个特定的游戏状况，其会给予你隐形状态。如果有生物找到了你，你也将不再算作躲藏，也会失去那个状态。此事在躲藏动作中有所解释（StartFragment 见《玩家手册》的附录 C）。

**震慑 Stunned 状态现在不再给予移动阻碍。这一修改是你们的本意吗？** 是的。我们的本意就是要增加震慑状态与麻痹状态之间的区别，所以震慑状态下你可以正常移动了。

**是不是所有魔法 黑暗 Darkness 都会阻挡 黑暗视觉 Darkvision ？** 魔法黑暗只有在“规则文本写明了这一黑暗实例确实会...”的情况下阻挡黑暗视觉。举例来说，法术 
黑暗术 Darkness 写明了黑暗视觉无法看穿它。这种视觉阻碍效果是该法术的一个特性，并不是所有魔法黑暗的通用规则。

**在一天中角色可以进行多少次 短休 Short Rest，这一点有严格的限制吗，还是这完全取决于 DM？**

有关你可以进行短休的次数，唯一的严格限制是你在一天之中可以利用多少时间。实际而言，你也会承受限制来自故事性上的时间压力以及敌人的中断。

**一个超维度的空间（例如一处半位面或是由 魔绳术 Rope Trick 创造出来的空间），是否被视为不同的存在位面？**

一个超维度空间（也被称为超位面空间）存在于其他位面之外。因此，如果你身处物质位面而你的敌人在超维度空间，那么你们二者确实不处于同一个存在位面。

我该如何判断游戏中的某些事物是否为魔法性？

游戏文本会明确注明哪些是属于魔法性的效应（简称魔法效应）。由法术与魔法物品创造的效应始终是魔法性的。详见《玩家手册》附录 C 中的“魔法效应”一节。

**举例来说，有什么效果可以降低 力竭 Exhaustion 呢？**

法术 高等复原术 Greater Restoration 
可以降低力竭。

在20级之后会发生什么？角色的等级会有上限吗？

角色可以抵达的最高等级是20级。有关20级以后角色的提升方式，《玩家手册》中第二章的“20级后的额外专长”边栏为 DM 准备了一些使用专长作为后续提升的思路。

**恐慌状态的说明提及“当它的恐惧源位于视线内”。这是否意味着即使来源不可见，但只要你与它的空间之间畅通无阻，你在攻击检定和属性检定上也有劣势?**

并不。如果某些东西是你看不见的，它就不算在你的视线内。说到“视线 line of 
sight”这个词，游戏使用了该术语的英文含义，在游戏规则中并没有特殊含义。
