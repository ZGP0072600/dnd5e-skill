# 牧师-奥秘领域

奥秘领域 Arcana 
Domain

魔法在制造毁灭的同时也带来新生。作为一种能量，它弥散在多元宇宙的每一个角落。而掌管奥秘领域的神知晓魔法的秘密和潜能。对于一些神而言，魔法知识是一种重要的理解现实本质的途径；而另一些神单单只把魔法视为纯粹的能量，并拿它用来做他们认为适当做的事情。

这一领域的神往往和知识领域有联系，毕竟学习和奥术力量的关系往往亲密无间。在这个范围中，掌管这个领域的神包括阿祖斯和密斯特拉，还有精灵之神科瑞隆。在其他世界，这个领域包括赫卡忒，麦斯麦索威，还有伊西斯；克莱恩的三月神索林那瑞，努林塔瑞和努塔瑞；灰鹰的博卡布，维克那和维婕斯。

奥术传承 Arcana 
Initiate 从1级起，你得到奥秘技能的熟练。你从法师法术列表习得两个你自选的戏法，对于你而言，这些戏法是牧师戏法。引导神力：奥秘阻绝 Channel Divinity: Arcane 
Abjuration 从2级起，你可以使用你的引导神力来抵御异界的生物。 
以一个动作，你可以展示你的圣徽，同时你选择一个30尺内能看见或者听见你声音的天族、元素、妖精或邪魔，它必须成功通过一次感知豁免。否则将被驱散，持续1分钟或在他受到任何伤害时提前结束。 
一个被驱散的生物必须使用他的回合尽力远离你，并且无法主动靠近你30尺以内。同时他们无法使用反应。在他们行动中，只能使用疾走或者从任何妨碍他们移动的效果中挣脱出来。如果没有任何地方移动，该生物使用闪避动作。 
在你达到5级后，如果一个挑战等级小于等于奥秘放逐表内数值且不是这个位面原住民的生物在对抗你的奥秘阻绝能力时豁免失败，他将在1分钟之内被放逐（如同法术 
放逐术 banishment 的效应，但不需要专注）。

破法者 Spell 
Breaker 从6级起，每次你使用1环或者更高环数的法术恢复了一名盟友的生命值，你同时可以选择作用于此生物之上的一个法术终止。被终止的法术环阶必须在你用来施放恢复法术所占用的法术位的环阶以下。

强力施法 Potent Spellcasting 从8级起，你在你的任何牧师戏法造成的任何伤害中获得你的感知属性调整值加值。

奥术精通 Arcane 
Mastery 从17级起，你从法师的施法列表中选择一些法术从以下环阶中各选择一个法术：6环、7环、8环、9环，你将选择的法术加入到自己的领域法术列表。如同其他领域法术一样，你总是视为准备了它们，且对你而言它们被视为牧师法术。
