# 战狂道途 Path of the Battlerager

**战狂道途通常被那些选择追随战争之神的矮人选择，在矮人语中他们被称作为“库扎尔夫”（意为“斧头笨蛋”）。他们常穿着厚重的、带刺的盔甲冲入战场，用他们自己的身体在战斗中挥洒怒火。**

**限制：限定矮人 ** 只有矮人才可以选择战狂道途。在矮人的社会和文化中战狂占据了一个特殊的位置。你的 DM 可以调整这一规则使之更符合战役。

**战狂装甲 Battlerager Armor**

从3级起，你得到将钉刺甲（见边栏“** 钉刺甲 **”）作为武器使用的能力。若你穿着钉刺甲，在狂暴期间，你便可以以附赠动作，用甲上钉刺对5尺内的目标进行一次近战武器攻击。若攻击命中，钉刺将造成1d4穿刺伤害。这次攻击检定与伤害掷骰使用你的力量调整值。此外，当你使用攻击动作去擒抱一个生物时，擒抱成功时，那个生物将受到3穿刺伤害。

**凶蛮无羁 Reckless Abandon**

从6级起，当你在狂暴期间使用鲁莽攻击时，你还可以得到等于你的体质调整值（至少为1） 
的临时生命。这一临时生命值在你狂暴结束后还剩余一部分时，剩余部分会消失。

**战狂冲锋 Battlerager Charge**

从10级起，你在狂暴期间可以以附赠动作执行疾走动作。

**钉刺反冲 Spiked Retribution**

从14级起，狂暴期间，若你未处于失能状态、且穿着钉刺甲，当你周围5尺范围内有生物用近战攻击命中你时，攻击者将受到3穿刺伤害。

**钉刺甲 Spiked Armor**

钉刺甲是一种罕见的中甲，它们出自矮人之手。它的造型看上去像是覆盖了金属尖刺的皮革外套和护腿。
