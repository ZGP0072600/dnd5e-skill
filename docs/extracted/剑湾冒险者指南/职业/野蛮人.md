# 野蛮人

**野蛮人 Barbarians**

> *最靠近黑暗精灵们的潘特 Pwent 低头瞄准卓尔护甲的接合处，轻松而粗野的用头盔上的长刺贯穿了一个精灵的胸膛。另一名卓尔用其双剑将头盔的长刺拨开，打算偏转战狂的下一次冲锋。只是一发武装的重拳带着指关节的尖刺袭来，直击卓尔下巴并撕开其喉咙。为争取喘口气的卓尔阴险的向对手的后背发出两记偷袭，但在怒目疾驰的矮人面前其并不凑效*。
>
> > > > > > > > ——R.A. 
> > > > > > > >  萨尔瓦多，《暗军突袭》

剑湾和北地许多地区都非常荒蛮。在这些为日常生存而奋斗的土地上孕育着诸多顽强的部族和凶猛的武士，比如北地的瑞格 Reghed 和乌斯伽 Uthgardt 的野蛮人，以及月影群岛 Moonshae 
Isles 和剑湾北海域的远海北陆人 Northlanders。

这些地区的野蛮人通常是人类或半兽人。野蛮的人类部族偶尔也会与北地或西央土的精灵结合而出现半精灵血统的野蛮人。此外还有在部族中与邪魔结合而产生的提夫林。自傲的矮人在收复秘银厅和冈特格瑞姆时，其部队中野蛮人也是声明卓著而让人闻风丧胆。其他种族的野蛮人则大多自于温暖的南方而非荒蛮北地 Savage 
North，不过一些南方的弃儿会被北地的部族收养并在此长大成人。

**原初道途 Primal Paths**

被遗忘国度的野蛮人除追随《玩家手册》中提及的原初道途外，还可以有其他一些额外的选择。瑞格人和北陆人的野蛮人倾向于追随“狂战士道途”，而乌斯伽的野蛮人则几乎总是追随“图腾武者道途”。
