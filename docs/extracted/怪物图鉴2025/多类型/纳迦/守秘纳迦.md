# 守秘纳迦 Guardian Naga

*亘古长存的护识蛇灵*

**栖息地**：荒漠，森林，异界（上层位面）　　**宝藏**：圣物

守秘纳迦是不朽的蛇形学者，享有着完美的记忆力。它们收集着来自与其一同生活的族群的历史与秘闻，并看护着不同文化的故事再将其准确无误地传递给新生代的后裔们。一些比共生文明更为长存的守秘纳迦仍在残垣断壁之间徘徊，铭记着已逝文明的故事，让其亡佚的子民在传承中延续下去。

掷一次守秘纳迦秘闻表或从中选择一个结果，以此启发该纳迦的所见所闻。

## 守秘纳迦秘闻 

| 1d8 | 守秘纳迦追忆着…… |
|---|---|
| 1 | 一段古代贤者或领袖人物的遗言。 |
| 2 | 一座隐秘城市或一片大陆的位置。 |
| 3 | 一句咒语，密码，或谜语的答案。 |
| 4 | 每位曾向它讲述故事之人的姓名。 |
| 5 | 一项本该被遗忘的仪式或法术。 |
| 6 | 多份使用了本土原料的配方。 |
| 7 | 诸多个讲述被遗忘的神祇与土著精魂的故事。 |
| 8 | 一只传奇怪物的致命弱点。 |

## 动作 Actions

**多重攻击 Multiattack。** 该纳迦发动两次啃咬攻击。其可以将其中一次攻击替换为使用剧毒蛇涎。

**啃咬 Bite。** *近战攻击检定*：+8，触及10尺。*命中*：17（2d12+4）穿刺伤害加22（4d10）毒素伤害。

**剧毒蛇涎 Poisonous Spittle。** *体质豁免检定*：DC 16，单一60尺内纳迦可见的生物。*失败*：31（7d8）毒素伤害，且目标陷入目盲状态直至纳迦的下个回合开始。*成功*：仅半伤。

**施法 Spellcasting。** 该纳迦施展以下一道法术，无需姿势或材料成分并使感知作为施法属性（法术豁免 DC 16）：

**随意：** 奇术 Thaumaturgy

**每项1/日：** 鹰眼术 Clairvoyance，疗伤术 Cure Wounds（六环版本）， 焰击术 Flame Strike（六环版本）， 指使术 Geas，真知术 True Seeing
