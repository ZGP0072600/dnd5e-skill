# 绿鬼婆 Green Hag

*邪恶荒野的恶巫婆*

**栖息地**：森林，丘陵，沼泽　　**宝藏**：奥秘

绿鬼婆编织着邪恶的魔法，毒害一切善良和纯洁的事物。无论独身居住还是和其他鬼婆结为集会，这些古老的坏巫婆们都能唤来恶毒的力量，传播腐化，并对那些惹恼它们的倒霉蛋降下灭顶之灾。绿鬼婆是精明的骗子，会使用幻象把自己藏在普通的外表中，以期能将无辜者诱入险途。这些鬼婆常常会把受害者带回自己怪诞的巢穴，它们会在那里囚禁俘虏，或者把俘虏做成可怕的美餐。

绿鬼婆常常会知晓一些古怪的魔法或被遗忘的秘辛，比如恶人的弱点、失落宝藏的埋藏地或打破诅咒的办法。它们可能会用这知识来交换罕有的魔法或一些具有某种象征意义的宝藏。掷一次绿鬼婆交易表或从中选择一个结果，以此启发想要换取绿鬼婆的秘密需要付出什么代价。

## 特质 Traits

**水陆两栖 Amphibious。** 鬼婆可以在空气和水中呼吸。

**集会魔法 Coven Magic。** 在30尺内有至少两个鬼婆盟友的情况下，鬼婆可以施展以下一道法术，无需材料成分，使用法术的原本施法时间，并使用智力作为施法属性（法术豁免 DC 11）：卜筮术 Augury，寻获魔宠 Find 
Familiar，鉴定术 Identify，物件定位术 Locate 
Object，探知术 Scrying 或隐形仆役 Unseen 
Servant。 鬼婆必须完成一次长休后才能再次用此特质施展其所选的法术。

**拟声 Mimicry。** 鬼婆可以模仿动物的声音和类人生物的语音。听到该声音的生物可以成功通过一次 DC 14的感知（洞悉 
）检定来发现其是模仿。
