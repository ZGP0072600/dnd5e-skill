# 前言：如何使用怪物 How to Use a Monster

每个怪物都是冒险的源泉。这本龙与地下城的怪物图鉴涵盖了各式各样的怪物，无论是怪异的还是异想天开的、庄严的还是怖人。请从中选出您最喜爱的那些怪物，并将它们融入您的 D&D 游戏中。

《**怪物图鉴 **》（2025）与《** 玩家手册 **》（2024）以及《** 城主指南 ** 
》（2024）一样，是 D&D 的核心基础规则之一，需配合上述书籍使用。本书专为地下城主（DM）设计，提供了由 DM 控制的各类生物——无论敌友——的游戏数据，这些数据以数据卡的形式呈现。本书以首字母顺序*向你呈现这些怪物的章节，以及收录有各类动物的附录 A。

 
官方 D&D 
冒险均引用了本书中的怪物。您可结合本书怪物以及《城主指南》中的自定义怪物规则和遭遇构建规则，打造自己的冒险。请参阅附录 B（译注：不全书未录入），了解有助于您构建冒险的怪物列表。

译注：为方便使用，不全书以生物类型为大类，生物子类为小类进行排版，由于其中部分同类怪物包含不同生物类型的子怪（如吸血鬼中包含类人的血仆和亡灵的吸血鬼），因此在两种生物类型种均有出现，并非排版错误。

2025版本中有何新内容？本书为5版《怪物图鉴》的2025版本。若您读过2014版本，本书的大部分内容会让您感到熟悉，因为基础规则和怪物种类是相同的。书中大部分内容包含新增或重新设计的元素，且本书中的怪物取代了旧版书中的版本。以下是一些重点：

**更多怪物 More Monsters。** 怪物数量已增加至超过500张数据卡。 

**全新怪物 New Monsters。** 数十种新怪物，包括更多高等级威胁，为各个等级的角色们提供挑战。 

**优化规则 Refined Rules。** 
所有数据卡均已更新，以提升易用性与游戏平衡性。 

** 改进数据卡 Enhanced Stat Blocks。** 新的数据卡设计和语言优先呈现关键的游戏细节。 

** 通用分组 Versatile Groups。** 
非玩家角色不再单独置于附录，而是与其他怪物一同出现，并能用来代表任何类人种族中的个体。 

** 改进编排 Improved Organization。** 
怪物现按名称首字母顺序统一排列，同一怪物的变体集中归类。

** 扩展列表 Expanded Lists。** 
附录 B 整合并扩展了2014版《城主指南》中出现的怪物列表。 
2014版《怪物图鉴》中的所有怪物均已收录于本书，或有相同 CR 的对应替代方案（在附录 B 中有作详细说明）。
