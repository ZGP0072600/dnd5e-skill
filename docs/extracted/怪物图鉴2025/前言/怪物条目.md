# 怪物条目 Monster Entries

本书主体由怪物条目构成，每个条目标题下方包含以下部分。

## 栖息地 Habitat

怪物的栖息地条目会标注其典型的栖息区域。按栖息地分类的怪物列表见附录 B。 

译注：怪物的栖息地包括寒带、海岸、荒漠、森林、草原、丘陵、山地、沼泽、幽暗地域、水下、城市以及异界。

## 宝藏 Treasure

怪物的宝藏条目表明该生物是否囤积宝物，以及其偏好的宝物类型，具体定义如下：

**任意 Any。** 该怪物的宝库可包含金钱类宝藏以及任意种类的魔法物品。

**个体 Individual。** 该怪物不囤积宝藏，但可能携带金钱类宝藏。

**宝藏主题 Treasure Theme（奥秘 Arcana，武备 Armaments，器具 Implements，圣物****Relics）。** 该怪物的宝库以标注主题的魔法物品为主。《城主指南 》详述了各主题对应的宝库配置规则。

**无 None。** 该怪物不关心宝藏。其携带的宝藏都是偶然所得。怪物拥有的任何宝藏都是其数据卡装备条目中列出的装备之外的额外物品。《城主指南》提供了更多关于怪物宝藏的信息。

## 叙事描述 Narrative Description

怪物基础信息之后，是适用于该怪物在多元宇宙中任何地方的详细信息。您可以在使用怪物时自定义这些细节，以令其适配您的冒险。

## 特殊巢穴 Special Lairs

一些怪物会改变巢穴周边的环境。此类生物的 **区域效应 regional effects** 将在巢穴一节详细描述。怪物在其巢穴内还可能变得更加强大，具体的增强方式还在其数据卡中给出。
