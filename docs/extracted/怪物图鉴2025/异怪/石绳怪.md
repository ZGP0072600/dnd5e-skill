# 石绳怪 Roper

*地下的触须设陷者*

**栖息地**：幽暗地域　　**宝藏**：任意

伪装成岩石的石绳怪是一种异怪伏击者，它们会潜伏并等待更小体型的生物出现。这些诡谲的地底猎手会延展它们橡胶般的触须来探索和接触周围的环境，它们的探知范围往往超出它们的视觉所及。当它们遭遇猎物时，这些触须就会诱捕受害者并将他们拽向石绳怪长满尖牙的口器。如果这些触须被切断，石绳怪将快速地再生出替代组织。

石绳怪可以移动，尽管移速缓慢。通过覆盖于它们身体底部的粘性纤毛，石绳怪可以在墙面上攀爬并垂悬于天花板之上。这些猎手经常蛰伏于意想不到或险要的位置，利用周身的环境来削弱它们的猎物。掷一次石绳怪险境表或从中选择一个结果，以此启发石绳怪在设伏时利用了何种险境。

## 特质 Traits

**蛛行 Spider Climb。**
石绳怪可以在难以攀爬的表面上攀爬，包括沿着天花板移动，且无需为此进行属性检定。

## 动作 Actions

**多重攻击 Multiattack。** 石绳怪发动两次触须攻击，使用拖拽，并发动两次啃咬攻击。

**啃咬 Bite。** *近战攻击检定*：+7，触及5尺。
*命中*：17（3d8+4）穿刺伤害。

**触须 Tentacle。** *近战攻击检定*：+7，触及60尺。*命中*：目标被六条触须之一擒抱，陷入受擒状态（逃脱 DC 14），且目标陷入中毒状态直至擒抱结束。

触须可被伤害（AC20，HP10，免疫毒素和心灵伤害），当触须被摧毁时，受擒于该触须的生物被释放。伤害触须不会对石绳怪造成伤害，且被摧毁的触须将在石绳怪的下个回合开始时重新长出来。

**拖拽 Reel。** 石绳怪将受擒于其的每名生物直线拉近至多30尺。
