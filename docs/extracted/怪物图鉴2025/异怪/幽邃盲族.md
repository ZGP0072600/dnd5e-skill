# 幽邃盲族 Grimlock

*夺心魔威胁所创造的提线木偶*

**栖息地**：幽暗地域　　**宝藏**：任意

幽邃盲族是夺心魔们的生物实验的受害者。灵吸怪会猎捕类人生物，将他们暴露于某种怪异的幽暗地域辐射之中，然后再向他们脑内植入指令，从而创造出幽邃盲族。创造幽邃盲族的过程会撕裂生物的意识，令该个体完全丧失曾经的个性。

幽邃盲族原先是眼睛位置现在只有一道浅浅的凹痕，但幽邃盲族能使用第六感来感知它们周围的环境。在转变为幽邃盲族时产生的灵能力量会萦绕于其体内，而且他们会在攻击时运用上这些可怕的力量。

掷一次幽邃盲族任务表或从中选择一个结果，以此启发幽邃盲族如何侍奉灵吸怪。

| 3 | 假装乐于助人，令旅人陷入虚假的安全感之中。 |
