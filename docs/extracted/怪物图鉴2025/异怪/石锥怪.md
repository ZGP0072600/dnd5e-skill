# 石锥怪 Piercer

*异种拟态钟乳石*

**栖息地**：幽暗地域　　**宝藏**：个体

石锥怪形似钟乳石，但具有带齿的口器和一只独眼。它们常栖身在幽暗地域居民频繁活动的路径上方，悬挂于洞顶处。石锥怪有时会潜伏数个月，等待任何体型相仿的生物从它们下方经过。当一个潜在的猎物移动到它们下方时，石锥怪会松脱并坠落，试图一击刺穿猎物。如果成功造成击杀，石锥怪会吞食猎物并缓慢爬到一个新的伏击位置。如果未能命中或是没能击杀目标，它们就会蠕动着逃跑，但它们很容易被觉察到它们存在的生物干掉。

石锥怪是石绳妖的幼年体。石锥怪幼体会尽可能地远离其他石绳妖以躲避它们因饥饿而进行的无差别吞食行为。许多石锥怪甚至会长途迁移穿过幽暗地域，来到近地表的洞穴或被掩埋的废墟中。

## 动作 Actions

**啃咬 Bite。** *近战攻击检定*：+3，触及5尺。*命中*：5（1d8+1）穿刺伤害。

**坠落 Drop。** 石锥怪落下。*敏捷豁免检定*：DC 11，单一位于石锥怪正下方的生物。*失败*：10（3d6）穿刺伤害。*失败或成功*：石锥怪因坠落受到的伤害减少20。
