# 甲伏怪 Chuul

*原初力量的甲壳仆从*

**栖息地**：海岸，沼泽，幽暗地域　　**宝藏**：圣物

甲伏怪起源于某个失落纪元，那时，底栖魔鱼和更加诡异的存在仍统治着水下的异星帝国。底栖魔鱼将无数的深海掠食者改造为仆役，令其可以远渡重洋以收集更多可利用的魔法和生物。甲伏怪即是这些异种仆役中存续至今的典型。

许多甲伏怪都侍奉着底栖魔鱼领主，它们在无光之海和原始沼泽中执行着主人的怪异理想。其他的甲伏怪则会跟随新的异怪主人，例如眼魔、触须怪或夺心魔。还有些甲伏怪会遵从自己的渴望，无止境地收集远古的魔法秘宝或以扭曲的形式继续执行着古老的使命。不论其动机如何，甲伏怪都会以巨钳擒住其他生物，然后用其触须麻痹生物，令其丧失行动能力。

甲伏怪不会衰老且可以在隐匿处休眠数千年，直到某个威胁，古老的使命，或异常的状态将其唤醒。

## 特质 Traits

**水陆两栖 Amphibious。** 甲伏怪可以在空气和水中呼吸。

**感知魔法 Sense Magic。** 甲伏怪可以探知120尺内的魔法。此特质如同法术侦测魔法 detect 
magic ，但本身不是魔法。

## 动作 Actions

**多重攻击 Multiattack。** 甲伏怪发动两次钳击攻击并使用麻痹触须。

**钳击 Pincer。** *近战攻击检定*：+6，触及10尺。*命中*：9（1d10+4）钝击伤害。若目标生物体型不超过大型，则其被两个巨钳之一擒抱，陷入受擒状态。

**麻痹触须 Paralyzing Tentacles。** *体质豁免检定*：DC 13，单一受擒于甲伏怪的生物。*失败*：目标陷入中毒状态并在其回合结束时重复豁免，成功则终止其身上的该效应。1分钟后，其豁免自动成功。 中毒期间，目标陷入麻痹状态。
