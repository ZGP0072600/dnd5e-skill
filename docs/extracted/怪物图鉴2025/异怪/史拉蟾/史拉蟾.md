# 史拉蟾 Slaadi

*孽生于混乱的混沌海族裔*

**栖息地**：异界（混沌海）　　**宝藏**：任意

不可捉摸的史拉蟾在混沌海那瞬息万变的混乱中吞噬和繁衍。这些如蟾蜍般的外域众生体现了它们的原生存在位面的无限潜能。尽管史拉蟾并无邪恶本质，但它们有着狂野的欲望且往往造成破坏。许多史拉蟾都在这种欲望的驱使下通过超自然的方式繁衍子嗣。不幸的是，这些方式对于其他生物通常来说是致命的。

史拉蟾不具有正式的社会架构。相反，强壮的史拉蟾支配着弱小的个体。史拉蓝蟾与红蟾在史拉绿蟾的指引下，于混沌海中横冲直撞并蔓延至其他世界。更加强大的史拉蟾则联系着产卵石，那是混沌魔法的源头，第一只史拉蟾便起源于此。产卵石藏匿在混沌海的深处，传说它可以溯源至魔冢的至高领主奇点，或是已陨落的史拉蟾领主，例如“金色流变之恐惧”森达姆和“飞翼之骸”伊戈尔。这些史拉蟾领主与其他存在密谋着将史拉蟾散播至多元宇宙的每个角落。
