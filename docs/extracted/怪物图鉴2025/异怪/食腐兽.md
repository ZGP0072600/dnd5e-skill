# 食腐兽 Otyugh

*垃圾堆中的老饕*

**栖息地**：幽暗地域　　**宝藏**：任意

食腐兽为食而生——越恶心的食物越受其亲睐。它们将所有触手可及且非其族类的生物视为其无尽饕餮生涯中的一餐。它们盘踞于垃圾场、下水道、受污染的废墟，和诸如此类的污浊深渊之中，贪婪地吞食垃圾、腐尸及任何可以被触须塞入其滔天巨口中的东西。有些人会通过投喂垃圾来招募食腐兽作为他们的守卫，尽管它们令人作呕，但足够警觉。

食腐兽经常将自己埋在垃圾堆下，通过长有眼球的茎须观察周围的情况。它们会利用亮闪闪的垃圾和心灵感应来引诱生物靠近，随即用其带刺的触须和污秽巨口发起攻击。掷一次食腐兽诱饵表或从中选择一个结果，以此启发食腐兽如何引诱猎物靠近。

## 食腐兽诱饵 

| 1d4 | 为了吸引潜在的大餐，食腐兽…… |
|---|---|
| 1 | 将触须伪装成垃圾。 |
| 2 | 用食腐兽语唱着诱人的歌谣。 |
| 3 | 用心灵感应发送信息，例如“这儿有好东西！”或“救命啊！我太美味了？” |
| 4 | 用心灵感应发送一副巨大宝石，扭曲武器，或受潮糕点的图像。 |

## 动作 Actions

**多重攻击 Multiattack。** 食腐兽发动一次啃咬攻击和两次触须攻击。

**啃咬 Bite。** *近战攻击检定*：+6，触及5尺。*命中*：12（2d8+3）穿刺伤害，且目标陷入中毒状态。每当中毒生物完成长休时，其承受以下效应。*体质豁免检定*：DC 15。*失败*：目标的生命值上限减少5（1d10），并且其生命值上限无法恢复直至中毒状态结束。

*成功*：中毒状态结束。

**触须 Tentacle。** *近战攻击检定*：+6，触及10尺。*命中*：12（2d8+3）穿刺伤害。若目标生物体型不超过中型，其被两条触须之一擒抱，陷入受擒状态（逃脱 DC 13）。

**触须猛击 Tentacle Slam。** *体质豁免检定*：DC 14，受擒于食腐兽的每名生物。*失败*：16（3d8+3）钝击伤害，且目标陷入震慑状态直至食腐兽的下个回合开始。*成功*：仅半伤。
