# 远古穴居攫怪 Grick Ancient

穴居攫怪可以存活数个世纪，最古老的穴居攫怪可以生长到令人恐怖的体型。比起埋伏等待，远古穴居攫怪会更积极地进行狩猎，而且它们会毫无顾虑地吞食较小的穴居攫怪。

## 动作 Actions

**多重攻击 Multiattack。** 穴居攫怪发动一次喙啄攻击，一次猛击攻击和一次触须攻击。

**喙啄 Beak。**
*近战攻击检定*：+7，触及10尺。
*命中*：22（4d8+4）穿刺伤害。

**猛击 Slam。**
*近战攻击检定*：+7，触及10尺。
*命中*：7（1d6+4）钝击伤害。若目标生物的体型不超过中型，其陷入倒地状态。

**触须 Tentacles。**
*近战攻击检定*：+7，触及10尺。
*命中*：15（2d10+4）挥砍伤害。若目标生物的体型不超过中型，则其被全部四条触须擒抱，陷入受擒状态（逃脱 DC 14）。
