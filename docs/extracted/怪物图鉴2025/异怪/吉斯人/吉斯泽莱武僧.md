# 吉斯泽莱武僧 Githzerai Monk

吉斯泽莱武僧通过磨练肉体与灵能天赋来追求对心智的控制。他们可能会离开吉斯泽莱之圣地，以期在多元宇宙的危险极端环境中考验自己。

## 动作 Actions

**多重攻击 Multiattack。** 吉斯泽莱人发动两次灵能打击攻击。

**灵能打击 Psi Strike。** *近战攻击检定*：+4，触及5尺。*命中*：6（1d8+2）钝击伤害外加9（2d8）心灵伤害。

**施法 Spellcasting。** 吉斯泽莱人施展以下一道法术，无需法术成分并使用感知作为施法属性：

**随意：** 法师之手 Mage Hand（手是隐形的）

**1/日：** 识破隐形 See Invisibility

## 附赠动作 Bonus Actions

**灵能跃迁 Psi-Powered Leap（2/日）。** 吉斯泽莱人施展跳跃术 Jump，无需法术成分并使用与施法动作相同的施法属性。
