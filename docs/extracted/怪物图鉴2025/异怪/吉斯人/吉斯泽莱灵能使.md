# 吉斯泽莱灵能使 Githzerai Psion

这些强而有力的灵能使监管吉斯泽莱圣地、研究晦涩的位面事件，并领导 **“拉克玛 Rrakkma”**——即针对夺心魔殖民地的突袭行动，以歼灭灵吸怪并揭露远古吉斯民族被改造的秘密。

## 动作 Actions

**多重攻击 Multiattack。** 吉斯泽莱人发动三次心灵扭曲攻击。

**心灵扭曲 Psychic Warp。** *近战或远程攻击检定*：+8，触及5尺或射程120尺。*命中*：26（4d10+4）心灵伤害，目标陷入某种由吉斯泽莱人选择的状态：(A)目标陷入魅惑状态直至吉斯泽莱人的下个回合开始；(B)若目标生物体型不超过大型，其陷入倒地状态。

**施法 Spellcasting。** 吉斯泽莱人施展以下一道法术，无需法术成分并使用智力作为施法属性（法术豁免 DC 16）：

**随意：** 法师之手 Mage Hand（手是隐形的）

** 每项1/日：** 位面转移 Plane Shift、识破隐形 See Invisibility
