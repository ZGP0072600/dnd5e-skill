# 吉斯泽莱泽锡修士 Githzerai Zerth

吉斯泽莱泽锡修士奉行由首位领袖泽锡蒙所提出的戒律。他们的灵能控制力协助其保护族人，并使得他们能在存在位面间穿行而无惧吉斯洋基人或夺心魔的追猎。

## 动作 Actions

**多重攻击 Multiattack。** 吉斯泽莱人发动两次灵能打击攻击。

**灵能打击 Psi Strike。** *近战攻击检定*：+7，触及5尺。*命中*：11（2d6+4）钝击伤害外加13（3d8）心灵伤害。

**施法 Spellcasting。** 吉斯泽莱人施展以下一道法术，无需法术成分并使用感知作为施法属性（法术豁免 DC 14）：

**随意：** 法师之手 Mage Hand（手是隐形的）

**每项1/日：** 幻影杀手 Phantasmal Killer（六环版本）， 位面转移 Plane Shift， 识破隐形 See Invisibility

## 附赠动作 Bonus Actions

**灵能跃迁 Psi-Powered Leap（2/日）。** 吉斯泽莱人施展跳跃术 Jump，无需法术成分并使用与施法动作相同的施法属性。
