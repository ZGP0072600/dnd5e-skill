# 吉斯泽莱人 Githzerai

*求极现实的探险家*

**栖息地**：异界（混沌海）　　**宝藏**：奥秘，个体

吉斯泽莱人是枯瘦的人形生物，他们的生理构造与吉斯洋基人完全一致。与此同时，他们与吉斯洋基人共享一段同样的历史——皆遭遇过夺心魔在生理与心灵上的改造（详见“吉斯洋基人”）。吉斯泽莱人深知其民族在肉体与心智上均被前灵吸怪奴役者所操控。他们从未屈从于这种程序式的改造，而是遵循首位领袖泽锡蒙的教导，重塑身心以寻求平静。

吉斯泽莱人通过灵能在多元宇宙的混乱区域中创造宁静隐秘的庇护所。此类堡垒多数漂流于混沌海位面，但吉斯泽莱聚落亦存在于无底深渊、元素混沌与妖精荒野。吉斯泽莱人建立这些封闭的庇护所旨在磨砺其灵能、参悟多元宇宙真谛，并躲避吉斯洋基人与夺心魔。

角色们可能会以多种方式卷入涉及吉斯泽莱人与吉斯洋基人的冲突。掷一次吉斯人冲突表，或从中选择一个结果，以此启发这些古老宿敌的相关冒险。

**吉斯冲突 **

| 1d8 | 冒险者们将…… |
|---|---|
| 1 | 被要求向巫妖女王维拉基斯传递信息或神秘包裹，或从其处接收此类物品。 |
| 2 | 受一只伪装的噬脑怪怂恿，追捕一名行踪不定，难以捕获的吉斯领袖。 |
| 3 | 被恳求协助遭吉斯洋基人摧毁圣地的吉斯泽莱逃难者。 |
| 4 | 受托延续或破坏吉斯洋基人与红龙的同盟关系。 |
| 5 | 受邀与吉斯洋基人会猎灵吸怪 |
| 6 | 被要求揭发某位面国家或法驱舰船上的吉斯人间谍。 |
| 7 | 受派执行任务，探明英雄吉斯或泽锡蒙最后已知的踪迹。 |
| 8 | 负责将阵亡吉斯洋基骑士的佩剑归还其族人。 |
