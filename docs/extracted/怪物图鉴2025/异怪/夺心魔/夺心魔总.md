# 夺心魔 Mind Flayers

*食脑的幽暗地域暴君*

**栖息地**：幽暗地域　　**宝藏**：奥秘

作为曾摧毁过世界的多元宇宙帝国的遗民，夺心魔潜伏于各物质位面世界的隐秘集会地。在夺心魔——亦被称作灵吸怪——的异界圣所中，这些邪恶的策划者重塑着幽暗地域及其住民，利用他们来实现自己莫测高深的理想。夺心魔因其灵能力量而使人闻之色变，它们会利用这股力量来震慑或控制其他生物，并用其以一种恐怖的方式进食：使用它们四条黏滑的触须来直接抽出受害者的大脑。

夺心魔是恶名昭彰的阴谋家和操控者，它们编制的计划常超出地下的领土范围。除了利用它们的灵能力量来操控生物，夺心魔也经常运用它们的生命机理来进行实验，通过向其他生物体内植入灵吸怪幼体来创造超自然仆从。诸如幽邃盲族和噬脑怪之类的生物都是夺心魔生物实验和干预的产物，而其他居住于幽暗地域的怪物——包括寇涛、幽邃熊怪和穴蜥人——也常常是灵吸怪首领的仆从。吉斯洋基人和吉斯泽莱人则因其曾遭受灵吸怪的奴役，与它们之间有着跨越时间的血仇。

夺心魔会单独行动，也有夺心魔会组成隐秘的阴谋组织，还有些夺心魔会参与到横跨世界的灵吸怪阴谋集团之中。当集体行动时，夺心魔会在主脑的组织下——一个具有恐怖灵能力量的巨型脑状生物——执行诡异的计划。如果缺少这样一个领导者，夺心魔群体将会陷入自我毁灭倾向的争执中。掷一次夺心魔阴谋表或从中选择一个结果，以此启发灵吸怪集团的阴谋。
