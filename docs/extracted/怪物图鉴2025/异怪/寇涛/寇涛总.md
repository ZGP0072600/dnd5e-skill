# 寇涛 Kuo-toa

*深渊的鱼人狂徒*

**栖息地**：海岸，幽暗地域　　**宝藏**：圣物

寇涛有着粘滑的类人身躯，头上有着如深海鱼类一般的头颅与凸眼。它们自称曾经统治过整个世界，在它们的鱼类神明们的护佑下，其帝国疆域横跨陆地与海洋。寇涛也说不上来究竟是怎样的灾难终结了它们的辉煌文明，但精灵，人类，夺心魔以及寇涛的神明成了它们责备的对象。寇涛从陆地及浅层海洋最终退回了孔道遍布的深沟以及幽暗地域之海当中。在这些隐蔽的地带，寇涛仍对它们曾经失去和遗忘的一切无法释怀，为实际可能从未发生过的轻蔑与怠慢而密谋着复仇。

由于相信自己是由来已久的歧视与经久不息的阴谋的受害者，寇涛仇视水表与幽暗地域的一切文明。寇涛执行着精心的谋划以推进自身的统治地位，它们经常把人绑来学习寇涛自己的秘辛，或者把人拿去给奇怪的神明进行可疑的献祭。为了推进这些计划，寇涛会尝试使用捕网或其他奇怪的武器活捉生物。住在幽暗地域的卓尔、矮人和侏儒，以及靠近地下通道的地表社区，常常成为寇涛袭击的目标，或是他们执行其他计划的目标。

寇涛常常侍奉着如底栖魔鱼与克拉肯这样恶毒的幕后主脑。这些寇涛相信这样强大的生物是寇涛神祇的使者，抑或这些生物本身就是神明。寇涛也可能会与其他邪恶生物结成暂时盟友，但是这样的同盟会随着寇涛领导者解读来自它们那日新月异的神祇之预兆而名存实亡。

寇涛一直试图寻找能够安抚它们那神秘莫测的神明们的办法。然而，很少有寇涛能够确定它们神明的身份，不同寇涛社区之间亦很少取得一致。唯有形似人类的躯体却长着龙虾一般的头颅和螯爪的“海之母”**布里博杜普 Blibdoolpoolp** 享受着普遍的信仰。

由于有关寇涛的其他神明样貌资料的缺乏，所以寇涛祭司会为这些神明编造新的躯体，他们用手边随便什么物件来创造神圣的偶像。但无论这些寇涛祭司从信仰，从幻象，从天资非凡之人当中，亦或是从某个诡异的超自然源泉当中吸取力量，总会有某些力量回应它们的祈求。掷两次寇涛神明特征表或从中选择一个结果，以此启发寇涛如何描绘神明。
