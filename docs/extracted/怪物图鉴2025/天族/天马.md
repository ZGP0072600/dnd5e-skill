# 天马 Pegasus

*行踪难测的有翼神骑*

**栖息地**：森林，草原，丘陵，异界（上层位面）　　**宝藏**：无

天马是具有智慧且高贵的有翼骏马，它们威严神秘，而又行踪难测。它们大多会避开其他生物，更喜欢栖身于诗情画意的牧场或浮空岛屿之上，或者干脆栖身于其他存在位面之中。部分天马会效忠于妖精荒野和上层位面的神祇，协助有需要的英雄。在极少数情况下，天马可能会与品行高尚之人结下友谊，成为他们的伙伴与坐骑。

由于邪恶势力的爪牙会猎捕天马，因此这些有翼骏马会在见到陌生人的第一时间就逃跑。掷一次天马献礼表格或从中选择一个结果，以此启发表露善意的方式。

## 天马献礼 

| 1d4 | 若某人……天马才不会拔腿就跑 |
|---|---|
| 1 | 带着天马曾帮助过的某位英雄的装备。 |
| 2 | 献上魔法水果或圣泉之水。 |
| 3 | 使用天界语、德鲁伊语或木族语吟唱一首歌。 |
| 4 | 穿着古代英勇教团的服饰。 |
