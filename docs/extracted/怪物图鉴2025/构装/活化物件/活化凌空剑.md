# 活化凌空剑 Animated Flying Sword

活化凌空剑就像由看不见的武者挥舞着一样。无论活化凌空剑是崭新锻造的，还是残缺不全的，其行为都一样。《玩家手册》中的其他武器也可以被活化，并使用与此处类似的游戏数据。
