# 活化盔甲 Animated Armor

活化盔甲的移动可能稳重审慎，也可能脚步笨拙。它们通常由板甲制成，很容易被误认为士兵或恐怖铠甲。其他类型的护甲，甚至金属雕像也可能成为活化盔甲。
