# 活化扫帚 Animated Broom

活化扫帚能保持周围环境整洁，并在必要时保卫周围环境。这些扫帚在空中四处飞舞，从意想不到的角度攻击敌人。其他飞行物件，如活化工具或法杖，也可以使用这个数据卡。
