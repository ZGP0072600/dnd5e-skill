# 铁魔像 Iron Golem

*永恒之物的守护者*

**栖息地**：任意　　**宝藏**：任意

铁魔像以坚不可摧的装甲保护其魔法核心，守卫重要场所与物件。它们被铸造成双足形态，细节由其创造者决定，多似披甲卫士或传奇英雄。铁魔像以压倒性的物理力量与魔法核心的爆发出的力量迎击敌人，其魔法轰击表现为炽焰箭矢与喷涌毒气。

铁魔像世代守护其职责。掷一次铁魔像指令表或从中选择一个结果，以此启发其遵循的指令为何。

## 铁魔像指令 

| 1d4 | 铁魔像遵循指令以…… |
|---|---|
| 1 | 封锁一扇从未开启的门扉，直至某条预言应验才会离开。 |
| 2 | 持续喷吐毒气，仅当有人说出密语时暂停。 |
| 3 | 静立如雕像，除非社群处于危难之中。 |
| 4 | 镇守强大魔法物品的存放之地。 |

## 特质 Traits

**火焰吸收 Fire Absorption。** 每当魔像将受火焰伤害时，其恢复等于火焰伤害数值的生命值。

**不变形态 Immutable Form。** 魔像无法变形 。

**魔法抗性 Magic Resistance。** 魔像对抗法术和其他魔法效应时进行的豁免检定具有优势。

## 动作 Actions

**多重攻击 Multiattack。** 魔像使用利刃铁臂或火炎箭发动共计两次攻击。

**利刃铁臂 Bladed Arm。** *近战攻击检定*：+12，触及10尺。
*命中*：20（3d8+7）挥砍伤害外加10（3d6）火焰伤害。

**火炎箭 Fiery Bolt。** *远程攻击检定*：+10，射程120尺。
*命中*：36（8d8）火焰伤害。

**毒素吐息 Poison Breath（充能6）。** *体质豁免检定*：DC 18，60尺锥状区域内的每名生物。
*失败*：55（10d10）毒素伤害。
*成功*：半伤。
