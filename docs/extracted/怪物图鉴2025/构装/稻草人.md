# 稻草人 Scarecrow

*自迷信中诞生的使仆*

**栖息地**：草原　　**宝藏**：无

复仇精魂束缚于粗糙简陋的支架之上，以此诞生了稻草人。或许是某些民俗魔法创造了他们，或许是平民的祈祷唤来了他们，也或许是那些有着暴力工作还未尽了的精魂附于其上从而产生了他们。稻草人也许会服务于那些创造他们的人，或者守护一处地方、一个家庭、或者一片社区免受威胁——不论是物理意义上的还是生活方式上的守护。

稻草人的名字取自乡村田野里搭建的那些人形驱鸟架子，不过他们可能有着各种各样的拼凑外观。掷一次稻草人支架表或从中选择一个结果，以此启发设计稻草人外观。
