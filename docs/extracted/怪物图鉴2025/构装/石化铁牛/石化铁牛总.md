# 石化铁牛 Gorgon

*呼出石化气体的牛形卫士*

**栖息地**：森林，草原，丘陵　　**宝藏**：任意

石化铁牛形似身披铁铠的强壮公牛，是如活物一般的构装生物。它们会摧毁所有踏入其领地的生物。除了用致命的牛角顶撞，用铁蹄践踏之外，石化铁牛还能呼出剧毒的气体

石化铁牛是施法者创造的卫士。制造石化铁牛的过程劳神费力且危险，其中一种办法需要将公牛的骨架、美杜莎的血和鸡蛇的大脑组装到施了魔法的钢铁框架之中。如果这一过程失败，石化气体会从材料中溢出，产生一种魔法威胁，会充满建筑物内部并经年不散。

当施法者创造石化铁牛时，他们通常会用魔法让其能忽视那些在它们面前提供了特别的秘钥（通常是密码或特别的信号）的人。一旦石化铁牛被设置为守卫一个区域，它就会进攻进入的一切闯入者，直到敌人逃跑或被毁灭。而闯入者如果提供了秘钥，只要仍在它的视线范围内，石化铁牛便会无视其存在。但如果闯入者胆敢离开视线再返回却没有出示秘钥，石化铁牛便会发起进攻。身处石化铁牛领地的时候必须时刻小心，掌握这怪物的具体位置，否则便有可能被原以为已经不再构成威胁的石化铁牛攻击。

创造石化铁牛的人会让秘钥难以发觉。在石化铁牛创造者的记录或石化铁牛保护的区域中，可能会找到关于秘钥的提示——可能是闯入者生命的最后潦草写下的。

掷一次石化铁牛秘钥表或从中选择一个结果，以此启发什么样的口令或者信号才能暂时解除石化铁牛的威胁。

**石化铁牛秘钥 **

| 1d6 | 要让石化铁牛不攻击，你得…… |
|---|---|
| 1 | 当着石化铁牛的面施展一个特定的法术。 |
| 2 | 背对石化铁牛，或表现得像没有看到它一样。 |
| 3 | 给它一杯血、水或酒。 |
| 4 | 背诵一首特定的押韵诗或唱一首特定的歌。 |
| 5 | 把创造者的名字倒过来念。 |
| 6 | 带上一个面具，也许是公牛的形状，也许是某个对其创造者有特别意义的动物。 |
