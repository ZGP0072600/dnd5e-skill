# 魔冢·二元冢 Modron Duodrone

二元冢可以同时执行两项任务并通常负责监管单元冢的工作小组。比起单元冢，它们对陌生人的怀疑程度略高一些。

## 特质 Traits

**解体 Disintergration。** 若魔冢死亡，其解体为尘埃，留下其着装或携带的任何东西。

## 动作 Actions

**多重攻击 Multiattack。** 魔冢发动两次时械剑刃攻击。

**时械剑刃 Clockwork Blade。** *近战或远程攻击检定*：+3，触及5尺或射程30尺。*命中*：4（1d6+1）力场伤害。*命中或失手*：剑刃会在被用于一次远程攻击后立即魔法性地回到魔冢手中。
