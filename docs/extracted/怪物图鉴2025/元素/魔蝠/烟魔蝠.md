# 烟魔蝠 Smoke Mephit

烟魔蝠是一种由炽热余烬和灼热空气组成的形影难辨的存在。

它们喜欢误导生物，然后嘲笑它们的轻信之举。

## 特质 Traits

**自爆 Death Burst。** 魔蝠在死亡时爆炸。*体质豁免检定*：DC 11，源自魔蝠的5尺光环区域内的每名生物。

*失败*：目标陷入中毒状态，持续至魔蝠的下个回合结束。

*命中*：4（1d4+2）挥砍伤害。

**燃烬吐息 Cinder Breath（充能6）。** *敏捷豁免检定*：DC 11，单一15尺内魔蝠可见的生物。
*失败*：目标陷入目盲状态，持续至魔蝠的下个回合结束。
