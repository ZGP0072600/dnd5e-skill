# 魔蝠 Mephits

*恶意肆虐的元素顽徒*

**栖息地**：异界（元素位面）　　**宝藏**：无

魔蝠栖息于元素位面，是性情顽劣的恶作剧者。最常见的六种魔蝠形似半身人大小的石像鬼，拥有翅膀、夸张的五官、以及由两种元素交织而成的身躯。它们大多自私自利，沉溺于扭曲的幽默感或膨胀的自负，在自己的家园位面恣意妄为。有些魔蝠则充当巨灵或魔法使用者的信使与间谍。

魔蝠极不情愿离开它们极端元素之力满盈的家园。如若被放逐至物质位面或其他世界，它们便会以恶毒的恶作剧发泄不满，甚至折磨弱小生物取乐。当魔蝠被摧毁时，它们会在一阵元素魔法爆裂中化为乌有。
