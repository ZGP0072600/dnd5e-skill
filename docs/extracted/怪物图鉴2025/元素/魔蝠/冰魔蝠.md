# 冰魔蝠 Ice Mephit

冰魔蝠的身体由寒冷空气和冻结的水组成。它们喜欢冻结物体，并将冰块丢进人们的衣服中。

## 特质 Traits

**自爆 Death Burst。** 魔蝠在死亡时爆炸。*体质豁免检定*：DC 10，源自魔蝠的5尺光环区域内的每名生物。*失败*：5（2d4）寒冷伤害。*成功*：半伤。

## 动作 Actions

**爪击 Claw。** *近战攻击检定*：+3，触及5尺。*命中*：3（1d4+1）挥砍伤害外加2（1d4）寒冷伤害。

**云雾术 Fog Cloud（1/日）。** 魔蝠施展法术云雾术 Fog Cloud ，无需法术成分并使用魅力作为施法属性。
