# 鸟羽人 Aarakocra

*翱翔苍穹的卫戍之翼*

**栖息地**：山地，异界（气元素位面）　　**宝藏**：器具，个体

形似人形鸟类的鸟羽人翱翔于浩瀚诸界与气元素位面的无尽苍穹之间。它们的造型常与其栖息地附近的鸟类种群相似；有些形似鹰与秃鹰，另一些则更像蜂鸟和始祖鸟。许多土地之上的鸟羽人都传唱着它们的古老传说——关于远古时期的鸟羽人与艾夸的风之公爵们并肩作战，英勇地对抗邪恶的混沌女王的故事。
