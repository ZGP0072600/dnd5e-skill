# 人鱼 Merfolk

*巡游大洋的探险卫士*

**栖息地**：海岸，水下　　**宝藏**：个体

人鱼居住于海面之下，这是一种融合了人类与海洋生物之特征的神秘生物。一些人鱼对陆上人们很是好奇，而另一些则对海面之外的人保持着警惕。
