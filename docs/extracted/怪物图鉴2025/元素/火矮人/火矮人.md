# 火矮人 Azer

*锻火铸炎的金属生命*

**栖息地**：山地，异界（火元素位面）　　**宝藏**：任意

火矮人是活着的人形青铜生命体，他们利用原初的创造之元素，在多元宇宙中最为险要的恶火横行之地打造武器与魔法奇观。
