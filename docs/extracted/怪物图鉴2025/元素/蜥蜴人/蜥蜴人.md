# 蜥蜴人 Lizardfolk

*守护大地的爬行族裔*

**栖息地**：森林，沼泽　　**宝藏**：个体

蜥蜴人栖居于萦绕着原初魔法的荒野之中。尽管许多蜥蜴人都是掌握了各种技艺的类人生物，但是其中一些蜥蜴人与土元素位面缔结了强大的联系，将自身与成长和再生的魔法循环紧密相连。
