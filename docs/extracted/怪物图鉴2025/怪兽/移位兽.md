# 移位兽 Displacer Beast

*狡诈的猫科潜行者*

**栖息地**：森林　　**宝藏**：无

移位兽形似一只瘦削的六足黑豹，双肩生有带刺的触手。这种掠食者通过天生魔法扭曲光线，使他人眼中的它与自己的实际位置偏移数尺。

移位兽猎杀不仅为了进食，更因其嗜血成性。一旦开始追踪猎物，它们不死不休。虽然移位兽通常栖息于森林之中，但也会长途追猎旅行者，甚至闯入城市或地牢。这些掠食者能设伏或潜伏数日击倒猎物，狡诈程度远超普通野兽。

移位兽偶尔会穿过位面传送门追猎，故多元宇宙中皆有其踪迹，尤其是物质位面、堕影冥界与妖精荒野。它们打破所到之处的生态平衡，导致其他物种灭绝，因此许多德鲁伊结社与妖精都视其为死敌。

## 特质 Traits

**神避 Avoidance。** 当移位兽受到一个允许其进行豁免来只承受一半伤害的效应影响时，其豁免成功时不受伤害，豁免失败时只承受一半伤害。其无法在失能期间使用此特质。

**移影换位 Displacement。** 移位兽投射的魔法幻象使自己看上去位于真实位置附近的另一个位置，导致所有对其进行的攻击检定具有劣势。移位兽失能期间该特质失效。

## 动作 Actions

**多重攻击 Multiattack。** 移位兽发动一次撕裂攻击和一次触手攻击。

**撕裂 Rend。** *近战攻击检定*：+6，触及5尺。
*命中*：9（1d10+4）挥砍伤害，若目标生物体型不超过大型，则其陷入倒地状态。

**触手 Tentacle。** *近战攻击检定*：+6，触及10尺。
*命中*：11（2d6+4）穿刺伤害。
