# 鲨蜥 Bulettes

*贪婪的地底陆行鲨*

**栖息地**：草原，丘陵，山地　　**宝藏**：无

鲨蜥也叫“陆行鲨”，是一种执着的掠食者。它们会掘地潜行、飞跃障碍、冲锋突破来追捕猎物。它们在地底快速钻行，感知到动静就破土而出，以其巨嘴捕食猎物。
