# 美杜莎 Medusa

*视之石化的蛇发隐士*

**栖息地**：荒漠　　**宝藏**：任意

美杜莎是居住在早已不复荣光之地的傲慢生物，以其蛇发和臭名昭著的石化凝视而闻名。它们通常栖息在文明的边缘，或是披上伪装四处旅行，留下其石化受害者的痕迹。有些美杜莎以石化诅咒为要挟，支配其他的怪物或恶徒；其他那些则招聘免疫石化的仆从，如石像鬼和石化牛。

美杜莎往往诞生或源于某种超自然因素。掷一次美杜莎命运表或从中选择一个结果，以此启发该美杜莎的起源。

## 动作 Actions

**多重攻击 Multiattack。** 美杜莎发动两次爪击和一次蛇发攻击，或是发动三次毒素射线攻击。

**爪击 Claw。** *近战攻击检定*：+6，触及5尺。*命中*：10（2d6+3）挥砍伤害。

**蛇发 Snake Hair。** *近战攻击检定*：+6，触及5尺。*命中*：5（1d4+3）穿刺伤害外加14（4d6）毒素伤害。

**毒素射线 Poison Ray。** *远程攻击检定*：+5，射程150尺。*命中*：11（2d8+2）毒素伤害。

## 附赠动作 Bonus Actions

**石化凝视 Petrifying Gaze（充能5~6）。** *体质豁免检定*：DC 13，30尺锥状区域内的每名生物。若美杜莎在锥形区域内看到自己的倒影，其也必须进行此豁免。*首次失败*：目标陷入束缚状态，目标在下个回合结束时仍处于束缚则重复豁免，成功则终止其身上的该效应。*再次失败*：目标陷入石化状态替代其束缚状态。
