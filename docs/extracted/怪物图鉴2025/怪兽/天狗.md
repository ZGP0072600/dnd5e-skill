# 天狗 Kenku

*无法飞翔的学舌之禽*

**栖息地**：森林，异界（堕影冥界），城市　　**宝藏**：器具，个体

天狗作为类鸟种族也曾翱翔于天际，咏唱迷人歌曲，但诅咒夺去了它们的双翼，改变了它们的声音。如今天狗们只能在堕影冥界或城市的阴影中摸爬滚打，试图挽回它们曾经失去的事物。对某些天狗来说，这意味着寻求终结诅咒之法；而另一些天狗则试图借助魔法或者发明装置以令它们能够再次飞翔，再次歌唱。

诅咒影响着天狗，使其只能通过模仿自己曾听过的声音以口头交流。天狗可以超自然般地再现各种各样的噪音，从婴儿的呼喊再到流水的声音，以及用他人的声线说出一些短句。狡猾的天狗使用其拟声技巧以欺骗敌人，吸引生物进入埋伏，以及为盟友发出示意。

## 特质 Traits

**拟声 Mimicry。** 天狗可以模仿其听过的任何声音，包括声线。听到该声音的生物可以成功通过一次 DC 14的感知（洞悉）检定来发现其是模仿。

## 动作 Actions

**影之刃 Shadow Blade。** *近战或远程攻击检定*：+5，触及5尺或射程60尺。*命中*：6（1d6+3）暗蚀伤害。*命中或失手*：影之刃会在被用于一次远程攻击后立即魔法性地回到天狗手中。
