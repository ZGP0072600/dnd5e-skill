# 亡命恶犬 Death Dog

*双头的疫病传播者*

**栖息地**：荒漠　　**宝藏**：无

亡命恶犬是它们所居住的土地上的恶疫。这些凶猛的双头犬会伏击它们认为比自己弱小的生物，尤为偏好受伤和病弱的猎物。它们会肆无忌惮地发起攻击，尽可能用染疫的牙口感染尽可能多的生物。即使被赶走，亡命恶犬也会在受害者附近地带徘徊，让感染削弱猎物后再度发起进攻。

传说常将亡命恶犬与恶毒的死亡之神、冥界和被诅咒的君王联系在一起。这些故事都是基于亡命恶犬所传播的疫病。

掷一次亡命恶犬疫病症状表或从中选择一个结果，以此启发亡命恶犬咬伤传播的疫病症状。这些症状只是表面的，不会改变亡命恶犬啃咬动作的实际效果。当生物不再陷入亡命恶犬咬伤带来的中毒状态时，这些症状也会消失。

## 亡命恶犬疫病症状 

| 1d6 | 亡命恶犬的疫病会造成…… |
|---|---|
| 1 | 受害者身上出现了许多犬科动物的牙印，仿佛还被恶犬紧紧咬住。 |
| 2 | 受害者的身体逐渐干枯，仿佛正暴露在沙漠的高温之下。 |
| 3 | 受害者会被只有他自己能听到的声音扰乱心神，这些声音像是来自远处的嗥叫或是各种模糊耳语。 |
| 4 | 受害者的血肉像尸体一样开始腐烂。 |
| 5 | 受害者浑身发痒，仿佛他的皮肤下面有跳蚤或沙子。 |
| 6 | 邪恶的符号逐渐浮现在受害者的身体上，并不断蔓延。 |

## 动作 Actions

**多重攻击 Multiattack。** 亡命恶犬发动两次啃咬攻击。

**啃咬 Bite。** *近战攻击检定*：+4，触及5尺。*命中*：4（1d4+2）穿刺伤害。若目标生物体型不超过大型，其承受以下效应。*体质豁免检定*：DC 12。*首次失败*：目标陷入中毒状态。中毒期间，目标的生命值上限在完成长休时无法恢复。此后每24小时目标重复豁免，成功则终止其身上的该效应。*后续失败*：中毒目标的生命值上限减少5（1d10）。
