# 梅洛人鱼 Merrow

*如食人魔般的海底掠夺者*

**栖息地**：海岸，水下　　**宝藏**：任意

梅洛是凶狠的水域猎手，兼具食人魔与原始掠食性鱼类的特征。梅洛潜伏在沿海水域，坑害毫无戒备的猎物，它们猛然从水中跃起，抓住目标，或用致命的鱼叉将猎物刺穿。随后，被捕获的陆地住民会被拖入阴暗的海底巢穴。梅洛常常将囚徒关押在储藏室中，以作为未来的美餐。

它们还会袭击沿海定居点和人鱼社区，窃取武器和宝藏。这引发了人鱼与梅洛之间的冲突，也导致陆地住民误解，错误地将梅洛的袭击归咎于人鱼。

## 动作 Actions

**多重攻击 Multiattack。** 梅洛人鱼使用啃咬、爪击或鱼叉发动共计两次攻击。

**啃咬 Bite。** *近战攻击检定*：+6，触及5尺。*命中*：6（1d4+4）穿刺伤害，并且目标陷入中毒状态，持续至梅洛人鱼的下个回合结束。

**爪击 Claws。** *近战攻击检定*：+6，触及5尺。*命中*：9（2d4+4）挥砍伤害。

**鱼叉 Harpoon。** *近战或远程攻击检定*：+6，触及5尺或射程20/60尺。*命中*：11（2d6+4）穿刺伤害。若目标生物体型不超过大型，则梅洛人鱼将目标直线拉近至多15尺。
