# 蚊蝠 Stirge

单只蚊蝠通常只是个小麻烦，但如果它们吸附的速度快于受害者挣脱的速度，数只蚊蝠便可能致命。

## 动作 Actions

**长喙 Proboscis。** *近战攻击检定*：+5，触及5尺。*命中*：6（1d6+3）穿刺伤害，且蚊蝠会吸附在目标身上。蚊蝠吸附期间，其无法发动长喙攻击，且目标在蚊蝠回合开始时受到5（2d4）暗蚀伤害。

蚊蝠可以消耗5尺移动力解除依附。目标或其5尺内的生物可以用动作来解除蚊蝠的依附。
