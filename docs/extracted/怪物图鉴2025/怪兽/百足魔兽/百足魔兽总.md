# 百足魔兽 Remorhazes

*极地极热节肢动物*

**栖息地**：寒带　　**宝藏**：无

百足魔兽是蜈蚣样的恐怖生物，它们在冰雪中穿梭，捕食所有闯入其冰冻领地的比它们小的生物。
