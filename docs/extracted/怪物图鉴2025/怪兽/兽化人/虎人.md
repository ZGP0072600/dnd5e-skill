# 虎人 Weretiger

*化猛虎之威*

**栖息地**：荒漠，森林，草原　　**宝藏**：武备

虎人可化身为老虎或半虎半人形态。他们虽然可以随意或在其魔法本质驱使下变形，但多数虎人是夜行生物，入夜便化为兽形。有一些虎人的变形与弦月、季节或重大事件相关。虎人多视这种能力为天赐恩典或家族荣光，常以此守护历史重地。掷一次虎人守护表或从中选择一个结果，以此启发虎人守护之物为何。

## 动作 Actions

**多重攻击 Multiattack。** 虎人使用抓挠或长弓发动共计两次攻击。其可以将其中一次攻击替换为一次啃咬攻击。

**啃咬 Bite（仅虎与混合形态）。** *近战攻击检定*：+5，触及5尺。*命中*：12（2d8+3）穿刺伤害，若目标是类人生物，则其受到下述效应。

体质豁免检定：DC 13。*失败*：目标被诅咒。若被诅咒目标的生命值降至0，作为代替，目标变成受 DM 控制的一只具有10生命值的 **虎人 Weretiger**。*成功*：目标在24小时内免疫此虎人的诅咒。

**抓挠 Scratch。** *近战攻击检定*：+5，触及5尺。*命中*：10（2d6+3）挥砍伤害。

**长弓 Longbow（仅类人与混合形态）。** *远程攻击检定*：+4，射程150/600尺。*命中*：11（2d8+2）穿刺伤害。

## 附赠动作 Bonus Actions

**潜步 Prowl。** 虎人移动至多等于其速度一半的距离且不会引发借机攻击。虎人可以在此次移动结束时执行躲藏动作。

**变形 Shape-Shift。** 虎人变形为类人与虎的混合形态（大型），或虎形态（大型），或变回其真实的类人形态。除体型以外，其各形态下游戏数据均相同。虎人着装或携带的任何装备都不会随之变化。
