# 鼠人 Wererat

*化硕鼠之狡*

**栖息地**：森林，城市　　**宝藏**：个体

鼠人可化身为巨鼠或半鼠半人形态。鼠人可自愿转变形态，但也有一些鼠人会在完全黑暗中或新月之夜期间被魔法性地强迫变形。鼠人的本质大部分情况下都源自神圣诅咒——对其欺瞒本性或其背信弃义之家族的惩罚。鼠人常结党为黑帮或盗贼公会。

## 动作 Actions

**多重攻击 Multiattack。** 鼠人使用抓挠或手弩发动共计两次攻击。其可以将其中一次攻击替换为一次啃咬攻击。

**啃咬 Bite（仅鼠与混合形态）。** *近战攻击检定*：+5，触及5尺。*命中*：8（2d4+3）穿刺伤害。若目标是类人生物，则其承受以下效应。

*体质豁免检定*：DC 11。*失败*：目标被诅咒。若被诅咒目标的生命值降至0，作为代替，目标变成受 DM 控制的一只具有10生命值的 **鼠人 Wererat**。*成功*：目标在24小时内免疫此鼠人的诅咒。

**抓挠 Scratch。** *近战攻击检定*：+5，触及5尺。*命中*：6（1d6+3）挥砍伤害。

**手弩 Hand Crossbow（仅限类人与混合形态）。** *远程攻击检定*：+5，射程30/120尺。*命中*：6（1d6+3）穿刺伤害。

## 附赠动作 Bonus Actions

**变形 Shape-Shift。** 鼠人变形为类人与鼠的混合形态（中型），或鼠形态（小型），或变回其真实的类人形态。除体型以外，其各形态下游戏数据均相同。鼠人着装或携带的任何装备都不会随之变化。
