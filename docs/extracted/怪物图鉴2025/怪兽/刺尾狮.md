# 刺尾狮 Manticore

*有翼的食人狮*

**栖息地**：寒带，海岸，草原，丘陵，山地　　**宝藏**：任意

刺尾狮有着狮子的利爪、带皮膜的翅膀和满口的獠牙，它们自天空俯冲突袭，吞噬旅行者。刺尾狮痴迷人类的滋味，但若没有这道最爱的珍馐，它们也会贪婪的吞噬其他人畜。刺尾狮的尾部密布可脱落的钉刺。这些怪物会向猎物发射尾刺，刺穿地面的猎物或是击落空中生物。

尽管刺尾狮贪婪无度，但它们乐于与将被吞噬的猎物交谈。有时它们还会与猎物达成交易，掷一次刺尾狮协商表或从中选择一个结果，以此启发刺尾狮可以提供什么来换取更诱人美食。
