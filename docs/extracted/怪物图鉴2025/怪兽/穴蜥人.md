# 穴蜥人 Troglodyte

*臭气熏天的地下猎手*

**栖息地**：幽暗地域　　**宝藏**：武备

穴蜥人的长得很像苍白的洞穴蜥蜴，它们潜伏在幽暗地域之中，永无休止地进行猎食。穴蜥人几乎什么都吃：骨头、巨虫、还有其他地底土著。它们在地下和靠近幽暗地域入口的那些社群之中捕猎，偷鸡摸狗，甚至绑架居民。

穴蜥人喜欢伏击猎物，它们还可以改变自身鳞片的颜色，和环境融为一体。它们经常会沿着洞穴的岩壁爬行，从深邃的裂缝之间钻出来，让猎物猝不及防。不过，虽然穴蜥人行动无踪，但是它们身上会散发出一股掩盖不住的熏天恶臭。有关穴蜥人的味道的描述简直罄竹难书。尽管这种恶臭令人作呕，但它也是穴蜥人发起突袭之前的预警。

## 特质 Traits

**恶臭 Stench。** *体质豁免检定*：DC 12，在源自穴蜥人的5尺光环区域内开始其回合的每名非穴蜥人生物。*失败*：目标陷入中毒状态，直至其下个回合开始。*成功*：目标在1小时内免疫所有穴蜥人的恶臭。

**日照敏感 Sunlight Sensitivity。** 若穴蜥人身处阳光下，其进行的属性检定和攻击检定具有劣势。
