# 雪怪 Yeti

*冻原上的冷血猎手*

**栖息地**：寒带　　**宝藏**：任意

雪怪栖息在高山之巅和蛮荒冻土上，猎杀那些闯入其领地的猎物。它们生性孤僻、冷酷无情，就像长着苍白毛发和山羊般犄角的巨猿。雪怪能够轻易将自己的身影混入雪地与冰山悬崖，在用冰冷的爪子发起攻击前，它们会发出令人毛骨悚然的嚎叫。除了它们强悍的肉体，雪怪还能用视线来冻结生物，让敌人冻在原地。它们还能咒唤出冰块并用它投向敌人。

由于雪怪难得一见，关于它们的民间故事往往比目击事件更多。很少有人能确定远处的呼啸究竟是雪怪的呼号还是仅仅是一阵狂风。无论真相如何，许多山区聚居地都会为此燃起篝火，利用这些恶兽对火焰的厌恶来抵御雪怪。
