# 石化蜥蜴 Basilisk

*以凝视石化之守宫*

**栖息地**：山地，幽暗地域　　**宝藏**：任意

石化蜥蜴是笨重的捕食者，它有着八只爪子，水晶似的脊背和巨大的下颚。石化蜥蜴并不追逐猎物，而是用超自然的石化凝视将猎物变成石头再恣意吞噬它们。虽然石化蜥蜴更喜欢待在地下的巢穴中，但总会有些无耻的家伙抓捕石化蜥蜴并把它们关起来，用以看管自己的宝藏。

石化生物的残骸散落在石化蜥蜴的猎场中。这些生物可能只是平凡生物，也可能是遭遇恐怖的石化蜥蜴的非凡生灵：掷一次石化蜥蜴受害者表或从中选择一个结果，以此启发出现在石化蜥蜴猎场中的雕像。这些雕像有50%的可能缺少肢体或破成碎片。
