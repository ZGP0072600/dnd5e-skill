# 幽邃熊怪 Quaggoths

*神出鬼没的地下追踪狂*

**栖息地**：幽暗地域　　**宝藏**：无

幽邃熊怪是的野兽般的猎手，深居地下，在幽暗地域的洞窟中艰难求生。尽管仅靠苦涩的地衣和有毒的菌类也能维生，但是它们也会残忍地攻击任何能让它们大快朵颐的东西，无论是巨蜘蛛还是探险者。幽邃熊怪有时候也会充当幽暗地域中那些恶棍的帮凶打手。

幽邃熊怪经常会组成一个个小团体，由其中最可怖的成员充当领导者。这样的小团体不仅兴风作浪，而且睚眦必报。幽邃熊怪团体仇恨着一切伤害过团中幽邃熊怪——或仅仅是被团体怀疑有这种意图——的家伙，不论原因也不论过错。这种宿怨有时会超出个体的界限，蔓延在整个幽邃熊怪族群。幽邃熊怪可能会因为某个聚落数十年前的怠慢而世代对其发起报复。只有幽邃熊怪团体的领导者才能让一场宿怨终止。
