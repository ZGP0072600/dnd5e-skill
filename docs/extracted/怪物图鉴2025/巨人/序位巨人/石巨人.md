# 石巨人 Stone Giant

*磐石大地之巨人*

**栖息地**：山地，幽暗地域　　**宝藏**：武备

译注：附录 B 中石巨人的栖息地也包括丘陵

幽深的洞穴和山间的峡谷中，石巨人凝视着那充满力量的，坚韧的大地。石巨人拥有粗犷的外貌，皮肤上的纹路和色调与它们家园附近的岩石相似。这使得体型庞大的它们能够巧妙地融入石质环境中。

石巨人很少干涉其他生物——无论是体型较小的邻近族群，还是其他巨人族类。它们大多行动迟缓，宁可忍受艰难困苦，也要静待危机过去。但当它们被激怒时——尤其是当古老的奇迹之地或它们的家园遭受威胁时，石巨人能释放出山峦般的伟力，以山崩之势碾碎敌人。

石巨人常常沉迷于自然奇观之谜，如山峰的尖顶、水晶的构造或神秘的岩画。它们中的一些人对隐藏于大地之中的魔法和秘密了如指掌。那些被困于幽暗地域的石巨人，往往将地表世界及其居民视为由沉睡中的原始生物、奇异神灵或其他实体在梦中构想出来的幻象。

## 动作 Actions

**多重攻击 Multiattack。** 石巨人使用石棒或投石发动共计两次攻击。

**石棒 Stone Club。** *近战攻击检定*：+9，触及15尺。*命中*：22 （3d10 + 6）钝击伤害。

**投石 Boulder。** *远程攻击检定*：+9，射程：60/240 尺。*命中*：15 （2d8 + 6）钝击伤害，若目标生物体型不超过大型，则其陷入倒地状态。

## 反应 Reactions

**偏转飞弹 Deflect Missile（充能5–6） 。** *触发*：石巨人因远程攻击检定被命中，并因此受到钝击、穿刺或挥砍伤害时。*响应*：石巨人受到的此次攻击的伤害减少11（1d10 
+ 6）。若该伤害因此降至0，石巨人可以重定向此次攻击的部分力量。*敏捷豁免检定*： DC 17，单一60尺内石巨人可见的生物。*失败*：11（1d10 + 6）力场伤害。
