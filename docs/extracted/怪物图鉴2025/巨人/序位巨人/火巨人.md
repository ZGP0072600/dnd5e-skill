# 火巨人 Fire Giant

*灼热火渊之巨人*

**栖息地**：山地，幽暗地域　　**宝藏**：武备

火巨人栖居于山脉深处的熔渊空洞与炽热岩浆之间。他们利用地底深处丰富的热能与矿藏打造出各种奇物，比如巨大的战争兵器和精妙的艺术造物。

火巨人拥有魁梧的身躯，皮肤呈现出多种岩石的色调，发丝如同跃动的火焰。

大多数火巨人居住在活跃的火山山脉或幽深的洞穴之中，那里矗立着他们熔炉要塞。邪恶的火巨人往往崇尚武力，他们锻造强大的兵器征服邻族并攫取珍贵资源。较为温和的火巨人则会以工艺制品换取所需物资，甚至可能与其他匠人分享巨人古老的锻造技法。无论哪一种火巨人，他们都热衷于实施宏大的建造计划，且绝不容忍旁人打扰他们巨大工坊中的神圣创作。

## 动作 Actions

**多重攻击 Multiattack。** 火巨人使用火焰之剑或投掷炎锤发动共计两次攻击。

**火焰之剑 Flame Sword。** *近战攻击检定*：+11，触及10尺。*命中*：21（4d6+7）挥砍伤害外加10（3d6）火焰伤害。

**投掷炎锤 Hammer Throw。** *远程攻击检定*：+11，射程60/240尺。*命中*：23（3d10+7）钝击伤害外加4（1d8）火焰伤害，目标被火巨人直线推离至多15尺。目标下个回合结束前进行的下次攻击检定具有劣势。
