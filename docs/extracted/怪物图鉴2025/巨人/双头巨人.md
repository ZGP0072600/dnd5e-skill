# 双头巨人 Ettin

*长了两个脑袋的好斗巨人*

**栖息地**：丘陵，山地，幽暗地域　　**宝藏**：个体

双头巨人是身强体壮的巨人，还长着两个脑袋。虽然很多双头巨人具有和山丘巨人类似的特征，但还有一些双头巨人具有更兽性或更非同寻常的特征，如长着獠牙、短角或每个脑袋只长了一只眼睛。

双头巨人常常和重视他们力量的巨人或团体结盟，如山丘巨人、匪帮或食人魔。一些双头巨人与他们居住的土地有着某种联系，他们可能会知晓或守护着德鲁伊或妖精们的秘密。

双头巨人的每个脑袋都有独立的人格。尽管这让一些双头巨人既和自己，也和他人吵架，但大多数双头巨人的两个脑袋还是团结一致。一个双头巨人可能两个脑袋都有自己的名字，或者两个脑袋将自己当成一体——他们共用一个名字，或者两个名字组合在一起。

掷一次双头巨人互动表或从中选择一个结果，以此作为启发遇到一个双头巨人的时候，它的俩脑袋如何互动。
