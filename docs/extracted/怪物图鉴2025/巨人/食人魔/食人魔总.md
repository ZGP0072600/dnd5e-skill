# 食人魔 Ogres

*横行霸道的囤积狂巨怪*

**栖息地**：寒带，荒漠，森林，草原，丘陵，山地，沼泽，幽暗地域　　**宝藏**：武备

食人魔是自私的劫掠者，充满恨意的超自然力量造就的恶食巨怪。它们分布广泛，从荒凉的废墟到贫瘠的内陆，到处都有食人魔在劫掠脆弱的社群，埋伏过路的旅人。食人魔只钟爱食物和财宝，它们会肆意毁坏艺术品、书籍、发条器件，还有其他所有精美、可爱的玩意。它们偶尔会把受害者绑回去以供日后食用，或者把中意的艺人抓回去，虽然后者的情况更为罕见。

食人魔的起源可上溯至诸位代表愤怒的神明，如 **厄瑞斯努 Erythnul**，** 塔克西丝 Takhisis**和 ** 瓦普拉克 Vaprak**。它们在那些被邪神、邪恶魔法或上古诅咒所腐化的土地上魔法般地突然出现。有些食人魔会带着自己出生地的痕迹，比如明显的石头状的老茧、满身的苔藓、或结冰的伤疤。
