# 还魂鬼 Revenants

*坟中掘起的复仇亡魂*

**栖息地**：森林，沼泽，城市 任意　　**宝藏**：任意

还魂鬼是一心只想复仇的愤怒恶灵。它们附身于尸体，或是其他载体之上，驾驭这些载体去伸张自己的正义，或是将滔天的怒火向着令其蒙冤之人肆意倾泻。还魂鬼拒绝接受死亡，直到其想要惩戒之人不复存在。即使其载体被摧毁，还魂鬼们也不会停歇——它们会寄宿于新的载体，继续那永无止境的恶灵追猎，直至为仇敌送上命定的审判。
