# 还魂巨构 Haunting Revenant

还魂巨构是附身了废墟或是其他被遗忘之地的还魂鬼，这些被附身之地通常与还魂鬼的死因有关。典型的还魂巨构身地包括被遗弃的废楼、失事的船只、甚至是一整座垃圾场。这些还魂鬼们就这样潜伏在光天化日之下，等待敌人靠近——随后将其一口“吃”下，困在其内部。进入还魂巨构之内者通常还不得不要跟活化家具或是其他更加令人不安的恐怖显化作战。还魂巨构所在之地通常很快就会变得恶名远扬。

## 特质 Traits

**缠魂之地 Haunted Zone。** *体质豁免检定*：DC 17，在还魂巨构的空间中施展法术的每名生物。*失败*：该法术失败并被浪费。

**亡灵复苏 Undead Restoraton。** 若还魂巨构死亡，除非对其遗留物施展驱逐善恶 Dispel Evil or 
Good ，否则其在24小时后复生。还魂巨构复生后会活化同一存在位面中的另一个超巨型物件或建筑；这会改变其外形，但不会改变其游戏数据，且归来时还魂巨构具有全部生命值。

## 动作 Actions

**多重攻击 Multiattack。** 还魂巨构发动两次物件猛袭攻击并使用请君入瓮。

**物件猛袭 Object Slam。** *近战或远程攻击检定*：+9（若目标位于还魂巨构所处的空间内则具有优势），触及10尺或射程30/90尺。*命中*：27（5d8+5）钝击伤害。

**请君入瓮 Invitation。** *魅力豁免检定*：DC 17，60尺锥状区域内的每名生物。*失败*：目标被传送至还魂巨构所处的空间内并被吞下。被吞下生物对还魂巨构体外的攻击或其他效应而言处于全身掩护。

只要还魂巨构仍具有生命值，被吞下的生物就不能离开还魂巨构所处的空间，除非其使用能够在位面之间旅行的魔法，例如法术位面转移 Plane 
Shift
 。
