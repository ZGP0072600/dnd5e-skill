# 蠕行之爪 Crawling Claws

*恶意附属，藕断丝连*

**栖息地**：任意　　**宝藏**：无

蠕行之爪是被切下来的手掌，会遵从自身的杀戮欲望进行活动。这些不死附肢是以各种杀手或恶人的断手复活而来，某些邪恶施法者会把这些秽恶之物作为随从。蠕行之爪的形态多种多样，从腐烂的人手以外，到各种野兽或怪物的新鲜爪子无所不包。
