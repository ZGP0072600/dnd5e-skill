# 食尸鬼 Ghoul

*已死飱徒*

**栖息地**：沼泽，幽暗地域，城市　　**宝藏**：任意

食尸鬼以群体行动，出没在世界各处腐败的角落，如饥似渴地寻找着尸体，以及将要成为尸体的猎物。这些腐尸骨瘦如柴，但是身形矫捷，有着非常长的舌头。它们栖息在各种地下墓穴与废墟之中，喜欢吞吃坟墓里的东西，能用有毒的爪子麻痹敌人。
