# 食尸鬼 Ghoul

食尸鬼生前一般是食人族和各种恶人，伴随着无休止的饥饿复生。它们成群结队寻找食物以满足各自的饥渴。

## 动作 Actions

**多重攻击 Multiattack。** 食尸鬼发动两次啃咬攻击。

**啃咬 Bite。** *近战攻击检定*：+4，触及5尺。*命中*：5（1d6+2）穿刺伤害，外加3（1d6）暗蚀伤害。

**爪击 Claw。** *近战攻击检定*：+4，触及5尺。*命中*：4（1d4+2）挥砍伤害。若目标非亡灵且非精灵，其承受以下效应。体质豁免：DC 10。*失败*：目标陷入麻痹状态，直至其下个回合结束。
