# 食人魔丧尸 Ogre Zombie

食人魔丧尸是不知疲倦的劳工，永无休止的战争武器。这类庞大的丧尸拥有巨大的体型和力量，某些屏障能拦住小型丧尸，但会被它直接突破。

## 特质 Traits

**不死坚韧 Undead Fortitude。** 当丧尸因非光耀非重击伤害生命值降至0时，其进行一次体质豁免（DC 为5+所受伤害）。豁免成功则改为将生命值降至1。
