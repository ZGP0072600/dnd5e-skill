# 丧尸 Zombies

*再活化之无情死尸*

**栖息地**：异界（堕影冥界），幽暗地域，城市　　**宝藏**：无

丧尸是一种没有思想的活化尸体，通常带有各种因腐烂或者受伤导致的可怕创口。它们会为赋予其生命力的任何超自然力量服务——通常是某些邪恶死灵师或者邪祟。丧尸无休无情，而且极其坚韧，哪怕尸体遭受严重的创伤，也依旧有继续活动的可能。虽然它们能接受并服从某些简单命令，但这并非是出于思想，而是某种本能。为了完成命令，它们会不知疲倦地工作和杀敌，但很容易就会被某些屏障或意外情况拦下来。

虽然丧尸一般是由某种类人生物尸体创造的，但其他生物的遗骸也可以变成丧尸。这些丧尸可能拥有生前具备的力量或者某些超自然能力，但现在也最多是胡乱使用罢了。
