# 丧尸 Zombie

类人生物的丧尸一般会被用作邪恶施法者的守卫、仆人或士兵。在极少数情况下，邪恶魔法也可能导致死者大规模复活，放出成群的丧尸威胁生者。

## 特质 Traits

**不死坚韧 Undead Fortitude。** 当丧尸因非光耀非重击伤害生命值降至0时，其进行一次体质豁免（DC 为5+所受伤害）。豁免成功则改为将生命值降至1。
