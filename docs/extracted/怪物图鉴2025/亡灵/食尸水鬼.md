# 食尸水鬼 Lacedon Ghoul

食尸水鬼皮肤松弛，是一种水生的食尸鬼。它们通常源自贪婪的海盗、在荒岛上饿死者、或者在水中溺亡的恶人。

## 动作 Actions

**多重攻击 Multiattack。** 食尸鬼发动两次阴寒啃咬攻击。

**阴寒啃咬 Icy Bite。** *近战攻击检定*：+4，触及5尺。

*命中*：9（2d6+2）寒冷伤害，且目标的速度降低5尺，直至食尸鬼的下个回合开始。

**爪击 Claw。** *近战攻击检定*：+4，触及5尺。*命中*：4（1d4+2）挥砍伤害。如果目标为非亡灵生物或非精灵，其承受下述效应。*体质豁免检定*：DC 10。*失败*：目标陷入麻痹状态，直至其下个回合的结束。

## 附赠动作 Bonus Actions

**蹿腾水中 Watery Rush。** 若身处水下，食尸鬼移动至多等于其游泳速度一半的距离，且不会引发借机攻击。
