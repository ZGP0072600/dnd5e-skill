# 骷髅牛头人 Minotaur Skeleton

骷髅牛头人是复生的牛头人骸骨，或者是用多种生物骸骨拼合成了类似牛头人的形状。这些骷髅巨怪有着更快的速度和更大的力量。其角和巨斧都是对活物的威胁。

## 动作 Actions

**顶撞 Gore。** *近战攻击检定*：+6，触及5尺。*命中*：11（2d6+4）穿刺伤害。如果骷髅在此次攻击前立即向着目标直线移动了20+尺，且目标生物体型不超过大型，则目标额外受到9（2d8）穿刺伤害陷入倒地状态。
