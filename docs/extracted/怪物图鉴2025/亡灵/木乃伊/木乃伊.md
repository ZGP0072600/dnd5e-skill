# 木乃伊 Mummy

普通的木乃伊一般由牧师、贵族或卫道士的遗骸以特殊的死后仪式处理而来。有些是通过亚麻布或者黏土裹身，也有通过泥炭、冰、魔法或者其他方式保存下来的。

掷一次木乃伊复苏表或从中选择一个结果，以此启发木乃伊复活的原因。

## 木乃伊复苏 Mummy Resurrections

| 1d8 | 木乃伊复活是为了…… |
|---|---|
| 1 | 捍卫制作时就预定保护的圣地。 |
| 2 | 服从木乃伊领主的召唤。 |
| 3 | 对抗某个死而复生的仇敌。 |
| 4 | 保护后代免受古老的威胁。 |
| 5 | 惩罚诅咒己身者的后代。 |
| 6 | 夺回坟墓中保存的宝藏。 |
| 7 | 侍奉在其墓前祈祷之人。 |
| 8 | 杀掉一切所见活物。 |

## 动作 Actions

**多重攻击 Multiattack。** 木乃伊发动两次腐朽拳击攻击，并使用恐怖怒视。

**腐朽拳击 Rotting Fist。** *近战攻击检定*：+5，触及5尺。*命中*：8（1d10+3）钝击伤害，外加10（3d6）暗蚀伤害。若目标为生物，则其被诅咒。受诅咒期间，目标无法恢复生命值，也无法在完成长休时恢复生命值上限，并且诅咒每持续24小时，其生命值上限降低10（3d6）。若生物因此攻击生命值降至0，则其立即死亡并化为尘土。

**恐怖怒视 Dreadful Glare。** 感知豁免：DC 11，单一60尺内木乃伊可见的生物。*失败*：目标陷入恐慌状态，直至木乃伊的下个回合结束。*成功*：目标在24小时内免疫此木乃伊的恐怖怒视。
