# 红龙雏龙 Red Dragon Wyrmling

红龙雏龙是天生的破坏者，自幼时起便以焚物灼人为乐。它们有时也会收起性子，不去灼烧生物，而是威逼他者听令献礼。

**多重攻击 Multiattack。** 红龙发动两次撕裂攻击。

**撕裂 Rend。** *近战攻击检定*：+6，触及5尺。*命中*：9（1d10+4）挥砍伤害外加3（1d6）火焰伤害。
