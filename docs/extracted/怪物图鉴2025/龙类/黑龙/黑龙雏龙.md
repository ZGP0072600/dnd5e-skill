# 黑龙雏龙 Black Dragon Wyrmling

黑龙雏龙蛰伏于沼泽地以及染污的水道中捕猎，并寻找可供欺凌的弱小生物。尽管较为年长的黑龙雏龙通常会寻找属于自己的领土，但刚孵化不久的那些雏龙亦有可能袭击自己的同类，以屠戮其无法征服的同胞确立统治。

## 动作 Actions

**多重攻击 Multiattack。** 黑龙发动两次撕裂攻击。

**撕裂 Rend。** *近战攻击检定*：+4，触及5尺。*命中*：5（1d6+2）挥砍伤害外加2（1d4）强酸伤害。

**强酸吐息 Acid Breath（充能5~6）。** *敏捷豁免检定*：DC 11，15尺长、5尺宽的线状区域内的每名生物。*失败*：22（5d8）强酸伤害。*成功*：半伤。
