# 伪龙 Pseudodragon

*善变玲龙*

**栖息地**：海岸，荒漠，森林，丘陵，山地，城市　　**宝藏**：奥秘

伪龙栖居于风光旖旎的荒野，尤爱猎物易捕、生活闲适之地。其如巨龙般悠然自得，于苍木峭壁间筑起小巢，缀以莹石彩贝与无主珍宝，并对其宝库严加看护。

伪龙能长到大型家猫的大小，一般通体覆有赤褐鳞甲。也有一些伪龙拥有异色鳞片或是斑纹，但均与其巨龙远亲迥异。

诸多魔法使都想讨好伪龙，希望其成为他们的魔宠。伪龙的灵慧心智与魔法抗性堪称绝佳伙伴，在某些施法者圈子中，豢养伪龙更被视为一种身份象征。

伪龙多耽于逸乐。它们可能倾向帮助那些以其嗜好之物讨好它们的人。反之，怠慢伪龙魔宠者，恐遭其弃如敝履。掷一次伪龙嗜好表或从中选择一个结果，以此启发伪龙喜欢何种礼物。

## 动作 Actions

**多重攻击 Multiattack。** 伪龙发动两次啃咬攻击。

**啃咬 Bite。** *近战攻击检定*：+4，触及5尺。*命中*：4（1d4+2）穿刺伤害。

**蛰刺 Sting。** *体质豁免检定*：DC 12，单一5尺内伪龙可见的生物。*失败*：5（2d4）毒素伤害，且目标陷入中毒状态，持续1小时。失败差值5或更多：目标在中毒期间还会陷入昏迷状态，昏迷在其受到伤害或被目标5尺内的一名生物用一个动作摇醒时提前结束。
