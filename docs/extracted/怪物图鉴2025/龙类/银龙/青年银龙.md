# 青年银龙 Young Silver Dragon

青年银龙通常与更年长的金属龙或是英雄楷模关联密切，并帮助那些志向远大者实现其愿景。青年银龙亦可作为此等善行者的信使或代表出面。

## 动作 Actions

**多重攻击 Multiattack。** 银龙发动三次撕裂攻击。其可以将其中一次攻击替换为使用其麻痹吐息。

**撕裂 Rend。** *近战攻击检定*：+10，触及10尺。*命中*：15（2d8+6）挥砍伤害。

**寒冷吐息 Cold Breath（充能5~6）。** *体质豁免检定*：DC 17，30尺锥状区域内的每名生物。*失败*：49（11d8）寒冷伤害。*成功*：半伤。

**麻痹吐息 Paralyzing Breath。** *体质豁免检定*：DC 17，30尺锥状区域内的每名生物。*首次失败*：目标陷入失能状态直至其下个回合结束，此时目标重复豁免。*再次失败*：目标陷入麻痹状态，并在其回合结束时重复豁免，成功则终止其身上的该效应。1分钟后，其豁免自动成功。
