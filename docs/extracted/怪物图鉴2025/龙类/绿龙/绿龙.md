# 绿龙 Green Dragons

*欺诈与嘲弄之龙*

**栖息地**：森林　　**宝藏**：奥秘

禁林深处，绿龙向世界播撒着邪恶，用低语操控着凡人。它们是色彩龙族中阴险、狡诈、又自负的一支，亦会颇具耐心地蚕食短生种的恐惧，令其腐化堕落、孤身无援。绿龙能在迷宫般的荒野之中潜伏数百年而不暴露自身的存在；即使是它们手下最忠实的追随者，往往也只能见到林间之音或梦中低语。

尽管绿龙的身体素质很强大，但它们大多鄙夷使用暴力，视战斗为其仆人们的职责——绿龙本身则更倾向于将敌人诈骗进危险或持续性的剥削之中。绿龙会收集那些体现其操纵之网的“玩物”充当装饰以及勒索工具，例如有失体面的文件、传家的珍宝、以及寄托情思的宝物。

绿龙栖息于远古森林之中。它们经常利用交织枝条、中空树干、粗壮根系留下的洞穴编织巢穴。它们亦可能生活在森林中的废墟里，其前身很可能是被其征服的族群们的家园。

成年绿龙及远古绿龙的巢穴所在区域将因其存在而异变，产生以下效应：

兽之探查 Beasts Spies。身处巢穴1里内期间，微型野兽魔法性地获得理解龙语且能以心灵感应和绿龙交流的能力。

毒瘴密林 Poisonous Thicket。巢穴1里内生长的普通植物会毒害周围的空气。每当一名非绿龙亦非其盟友的生物在该区域内完成长休时，该生物必须成功通过一次 DC 15的体质豁免，否则陷入中毒状态，持续1小时。

若绿龙死亡或将其巢穴搬至别处，这些效应立即结束。
