# 青铜龙雏龙 Bronze Dragon Wyrmling

青铜龙雏龙自信能够解决任何难题，甚至会试图亲自证明这点。它们会寻找地区性的挑战，例如旱灾或是猖獗的匪帮。虽然说这样的热情很好，但这些小龙也可能会因此陷入困境，需要他人援手并指出其错误之处。

**水陆两栖 Amphibious。** 青铜龙可以在空气和水中呼吸。

## 动作 Actions

**多重攻击 Multiattack。** 青铜龙发动两次撕裂攻击。

**撕裂 Rend。** *近战攻击检定*：+5，触及5尺。*命中*：8（1d10+3）挥砍伤害。

**闪电吐息 Lightning Breath（充能5~6）。** *敏捷豁免检定*：DC 12，40尺长、5尺宽的线状区域内的每名生物。*失败*：16（3d10）闪电伤害。*成功*：半伤。

**斥力吐息 Repulsion Breath。** *力量豁免检定*：DC 12，30尺锥状区域内的每名生物。*失败*：目标被青铜龙直线推离至多30尺并陷入倒地状态。
