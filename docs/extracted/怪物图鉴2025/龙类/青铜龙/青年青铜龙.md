# 青年青铜龙 Young Bronze Dragon

**水陆两栖 Amphibious。** 青铜龙可以在空气和水中呼吸。

## 动作 Actions

**多重攻击 Multiattack。** 青铜龙发动三次撕裂攻击。其可以将其中一次攻击替换为使用斥力吐息。

**撕裂 Rend。** *近战攻击检定*：+8，触及10尺。*命中*：16（2d10+5）挥砍伤害。

**闪电吐息 Lightning Breath（充能5~6）。** *敏捷豁免检定*：DC 15，60尺长、5尺宽的线状区域内的每个生物。*失败*：49（9d10）闪电伤害。*成功*：半伤。

**斥力吐息 Repulsion Breath。** *力量豁免检定*：DC 15，30尺锥状区域内的每个生物。*失败*：目标被青铜龙直线推离至多40尺并陷入倒地状态。
