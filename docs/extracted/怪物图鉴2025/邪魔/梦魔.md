# 梦魔 Incubus

*吸干生命的幻梦魔影*

**栖息地**：异界（下层位面），城市　　**宝藏**：任意

梦魔掠食凡人的梦境。它们悄无声息地潜入熟睡者的家中，将其幻梦当作食粮吞噬殆尽，只余下一片可怕的噩梦。梦魔每晚都会造访受害者，直到其被吸干死去。在那之后，梦魔会寻找新的受害者——尤其是上一位受害者的所爱之人。

梦魔可以转化为魅魔，反之亦然，取决于这些邪魔是想在玩弄受害者的梦境，还是玩弄受害者的肉体。

被梦魔光顾之人会反复陷入噩梦之中。掷一次梦魔噩梦表或从中选择一个结果，以此启发这些受害者会梦到什么样的恐怖。

## 梦魔噩梦 

| 1d8 | 梦魔的受害者梦到了…… |
|---|---|
| 1 | 一位愤怒的家人或是上级。 |
| 2 | 自己在野外被人追杀。 |
| 3 | 被野兽或其他怪物吞食。 |
| 4 | 不断地坠落，挣扎着溺亡，或无助地窒息。 |
| 5 | 足以令自己社会性死亡的尴尬现场。 |
| 6 | 幽影般的入侵者或是剪影怪物。 |
| 7 | 过去的某次让他烙下心灵创伤的事件。 |
| 8 | 一位访客，为他带来了一段或古怪或神秘的消息。 |

## 特质 Traits

**化身魅魔 Succubus Form。** 梦魔在完成一次长休时可以变形为 **魅魔 Succubus**，并使用魅魔的游戏数据替代其原来的数据。梦魔着装或携带的任何装备都不会随之变化。

## 动作 Actions

**多重攻击 Multiattack。** 梦魔发动两次毁梦之触攻击。

**毁梦之触 Restless Touch。** 近战攻击：+7，触及5尺。*命中*：15（3d6+5）心灵伤害，且目标被诅咒，持续24小时或在梦魔死亡时提前结束。受诅咒期间，目标无法从完成短休中获益。。

**施法 Spellcasting。** 梦魔施展以下一道法术，无需材料成分并使用魅力作为施法属性（法术豁免 DC 15）：

**随意：** 易容术 Disguise Self，以太化 Etherealness

**每项1/日：** 托梦术 Dream，催眠图纹 Hypnotic

## 附赠动作 Bonus Actions

**噩梦 Nightmare（充能6）。** *感知豁免检定*：DC 15，单一60尺内梦魔可见的生物。*失败*：若目标的生命值为20或更低，则其陷入昏迷状态，持续1小时并在其受到伤害或被目标5尺内的另一生物用一个动作摇醒时提前结束；否则，目标受到18（4d8）心灵伤害。
