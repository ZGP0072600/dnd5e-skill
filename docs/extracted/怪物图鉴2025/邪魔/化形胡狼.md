# 化形胡狼 Jackalwere

*荒野中的变形诈术师*

**栖息地**：荒漠，草原　　**宝藏**：器具

化形胡狼和胡狼在原始形态上没有两样，但化形胡狼会通过变形来欺骗他人。这些变形生物拥有三种形态：胡狼，人类，或两者的怪异混合形态。化形胡狼很容易被误认为是狼人 werewolves，但化形胡狼并未受到超自然的折磨——它们的胡狼外形就是它们原始的形态。化形胡狼还拥有能让敌人陷入沉睡的魔法凝视，这让化形胡狼得以随意地玩弄计谋，或是在威胁面前占据上风。

化形胡狼栖息于恶劣的荒野中，并以自己的聪明才智为傲。当有人旅行经过它们的领地而没有留下任何财宝或新鲜食物，它们就会对此感到愤怒。掷一次化形胡狼计谋表或从中选择一个结果，以此启发化形胡狼如何对此种无礼加以报复。

## 特质 Traits

**集群战术 Pack Tactics。** 若化形胡狼的攻击目标生物5尺内存在有至少一名化形胡狼未失能的盟友，则化形胡狼对该生物进行的攻击检定具有优势。

## 动作 Actions

**多重攻击 Multiattack。** 化形胡狼发动两次撕裂攻击或猛击攻击。

**撕裂 Rend（仅胡狼或混合形态）。** *近战攻击检定*：+4，触及5尺。*命中*：5（1d6+2）穿刺伤害。

**猛击 Slam（仅人类或混合形态）。** *近战攻击检定*：+4，触及5尺。*命中*：4（1d4+2）钝击伤害。

**昏睡凝视 Sleep Gaze（充能5~6）。** *感知豁免检定*：DC 10，单一30尺内化形胡狼可见的生物（此豁免构装和亡灵自动成功）。*失败*：目标陷入昏迷状态，持续10分钟，目标身上的此效应在其受到伤害或被目标5尺内的另一生物用一个动作摇醒时提前结束。 *成功*：目标在24小时内免疫此化形胡狼的昏睡凝视。

## 附赠动作 Bonus Actions

**变形 Shape-Shift。** 化形胡狼变形为中型的人类或类人与胡狼的混合形态（中型），或变回其真实形态（即小型胡狼）。除体型以外，其各形态下游戏数据均相同化形胡狼着装或携带的任何装备都不会随之变化。
