# 梦魇 Nightmare

*下层位面的恐怖战马*

**栖息地**：异界（下层位面）　　**宝藏**：任意

梦魇形似拥有烈焰鬃毛、燃火马蹄和阴燃双目的马匹。它们恐吓弱小的生物，并常与下层位面的住民结盟以实施恶行。这些超自然的马匹天生能够在以太位面和物质位面之间穿行，且许多个体都知晓前往下层位面、堕影冥界和其他险恶领域的传送门位置。

梦魇的速度、耐久力以及在存在位面间驰骋的能力令其成为了恶徒们垂涎的坐骑。掷一次梦魇骑手表或从中选择一个结果，以此来为什么存在可能雇佣一只梦魇坐骑提供灵感。

## 梦魇骑手

|---|---|
| 1 | 某个邪恶神明的勇士或信使。 |

| 3 | 一个被困于狂野邪魔之身的无辜灵魂。 |
| 4 | 一个追寻知识的魔法师 mage，邪教徒 cultist 或巫妖 lich。 |
| 5 | 一个在位面间放牧幼虫魔 larvae 的夜鬼婆 night hag 。 |
| 6 | 一位邪恶的骑士，例如一个死亡骑士 death knight，一个罪魔 erinyes，一个梦魔 incubus 或一个吸血鬼 vampire 。 |

## 特质 Traits

**赋予火焰抗性 Confer Fire Resistance。** 骑乘梦魇期间，梦魇可以赋予一名骑手对火焰伤害的抗性。

**照明 Illumination。** 梦魇散发出半径10尺的明亮光照以及额外10尺的微光光照。

## 动作 Actions

**蹄击 Hooves。** *近战攻击检定*：+6，触及5尺。*命中*：13（2d8+4）钝击伤害外加10（3d6）火焰伤害。

**以太折跃 Ethereal Stride。** 梦魇和其5尺内至多三名自愿的生物一起从物质位面传送至以太位面，或反之。
