# 原魔 Manes

*疯狂与恐慌并存的恶魔*

**栖息地**：异界（无底深渊）　　**宝藏**：无

当最令人唾弃的灵魂被判入无底深渊时，原魔便诞生了，它是恶魔的最低等形态。这些畸形恶魔有着扭曲的体态与肢干，上面爬满了深渊中的寄生虫。恶魔的冲动与持续的恐惧驱动着原魔，使它们只知道如何造成惊吓与爆发疯狂。
