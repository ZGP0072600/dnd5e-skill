# 怯魔 Dretches

*疯狂与粗鄙并存的恶魔*

**栖息地**：异界（无底深渊）　　**宝藏**：无

怯魔是更强大的恶魔的仆从与牺牲品，有着粗鄙的身形、混乱的冲动，以及暴力的倾向。在无底深渊的深处有着不计其数的怯魔，它们散发着恶臭，组成了庞大的恶魔大军。
