# 蜡融妖 Yochlol

*堕落意志的恶魔*

**栖息地**：异界（无底深渊）　　**宝藏**：无

蜡融妖是无底深渊邪恶意志与认知污染的化身。这些有毒的欺诈师在鲜有出现的真实形态下看起来像是不断移动的、带着不断滴液的触手的一团剧毒血肉，顶部带有一只阴险的眼睛。但更常见的情况下，蜡融妖们以蜘蛛或狂信徒的形态出场。它们使用操弄人心的魔法与危险的言论传播恶魔崇拜，腐化正义，并进一步推进它们的邪魔领主的阴谋。它们喜欢强迫不知情者参与到恶魔阴谋之中，让凡人自相残杀。

大多数蜡融妖为 **罗斯 Lolth** 服务。恶魔蜘蛛女王宣称所有的蜡融妖都为仆从，并命令消灭忤逆的蜡融妖。稀少情况下，蜡融妖可能会为其他恶魔领主服务，尤其是像 **格拉兹特 Graz’zt**，** 朱庇莱克斯 Juiblex**和 ** 祖格莫伊 Zuggtmoy**这样惑控人性，嬗变的恶魔领主。

尽管蜡融妖们为恶魔领主服务，它们自己也有邪恶的念头与雄心。它们可能会声称自己代表着它们的恶魔领主，以进一步实现自己的野心，或是试图揭露其竞争对手的自私目标，以获得其恶魔主人的青睐。

## 特质 Traits

**恶魔复苏 Demonic Restoration。** 若蜡融妖于无底深渊之外死去，其身躯会化为灰烬，并立即在无底深渊某处获得一具新的身体，以满生命值复活。****
** 魔法抗性 Magic Resistance。**蜡融妖对抗法术和其他魔法效应时进行的豁免检定具有优势。****
** 蛛行 Spider Climb。**蜡融妖可以在难以攀爬的表面上攀爬，包括沿着天花板移动，且无需为此进行属性检定。****
** 蛛网行者 Web Walker。**蜡融妖在蛛网上移动时无视蛛网造成的移动限制。

## 动作 Actions

**多重攻击 Multiattack。** 蜡融妖发动两次苛性鞭笞攻击，并使用施法施展蛛网术 Web 或支配类人 Dominate 
Person（若条件允许）。

**苛性鞭笞 Caustic Lash。** *近战或远程攻击检定*：+8，触及10尺或射程120尺。*命中*：25（6d6+4）强酸伤害。

**施法 Spellcasting。** 蜡融妖施展以下一道法术，无需材料成分并使用魅力作为施法属性（法术豁免 DC 15）：

**随意：** 侦测思想 Detect Thoughts， 气化形体 Gaseous Form（仅自身）， 蛛网术 Web

**1/日：** 支配类人 Dominate Person

## 附赠动作 Bonus Actions

**变形 Shape-Shift。** 蜡融妖变形为中型类人生物或中型蜘蛛，或变回其真实形态。其各形态下游戏数据均相同，且其着装或携带的任何装备都不会随之变化。

## 反应 Reactions

**毒性逃脱 Toxi Escape。** *触发*：蜡融妖被一次攻击检定命中。*响应*：蜡融妖令此次攻击对其造成的伤害减半（向下取整），并传送至30尺内一处其可见的未占据空间。体质豁免：DC 15，蜡融妖目标空间5尺内的每名生物。*失败*：目标陷入中毒状态，持续至其下个回合结束。中毒期间，目标陷入失能状态。
