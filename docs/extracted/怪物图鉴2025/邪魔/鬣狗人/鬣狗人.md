# 鬣狗人 Gnolls

*拥有野性肉身的邪魔*

**栖息地**：荒漠，森林，草原，丘陵　　**宝藏**：武备，个体

第一批鬣狗人诞生于摄食了被深渊侵染的肉体的鬣狗。它们的腐化与暴力取悦了恶魔领主耶诺古，其增加了它们的数量，并在多元宇宙散播它们。从那之后，鬣狗人们成为了耶诺古咯咯叫的仆从，存在即为造成毁灭，享用残羹剩宴。
