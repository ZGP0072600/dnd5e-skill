# 小魔鬼 Imp

*小气又多疑的魔鬼*

**栖息地**：任意　　**宝藏**：无

小魔鬼因其胆小怯懦与擅长阿谀奉承而闻名，它们服侍着其他魔鬼和邪恶的魔法使用者。小魔鬼可以变成其他形态或者完全隐去身形，这使得它们能担任优秀的间谍，也保证了它们能从危险中逃开。小魔鬼会被它们的主人派去监视其他生物并向主人汇报，但小魔鬼时常会遗漏重要信息，或是以最坏的方式描述发生的事情。它们是故意的——这样就能误导自己的主人，让主人盲目相信自己手下的小魔鬼“谋士”们。

没有主人的小魔鬼喜爱操纵他人以及自吹自擂。它们有可能聚集起一群比自己还弱的怪物，成为这群怪物的头头；又或者，它们会装出善良而乐于助人的样子，蒙骗有影响力的人去追求邪恶目的。

**魔法抗性 Magic Resistance。** 小魔鬼对抗法术和其他魔法效应时进行的豁免检定具有优势。

## 动作 Actions

**钉刺 Sting。** *近战攻击检定*：+5，触及5尺。*命中*：6（1d6+3）穿刺伤害外加7（2d6）毒素伤害。

**隐形术 Invisibility。** 小魔鬼对自身施展隐形术 Invisibility，无需法术成分并使用魅力作为施法属性。

**变形 Shape-Shift。** 小魔鬼变形为老鼠（速度20尺）、渡鸦（速度20尺，飞行速度60尺）、蜘蛛（速度20尺，攀爬速度20尺），或变回其真实形态。除速度外，其各形态下游戏数据均相同。小魔鬼着装或携带的任何装备都不会随之变化。
