# 劣魔 Lemures

*痛苦而绝望的魔鬼*

**栖息地**：异界（九层地狱）　　**宝藏**：无

劣魔从邪恶的灵魂中唤起，而它们凡人时期的记忆则早已丧失。混成一团难以分辨出的模糊肢体外加痛苦扭曲的融化面容，附着在这团纯粹的地狱原质之上，这就是这种最低级的魔鬼的组成部分了。
