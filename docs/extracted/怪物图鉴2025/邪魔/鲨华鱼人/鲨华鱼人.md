# 鲨华鱼人 Sahuagin

*波涛之下的劫掠者*

**栖息地**：海岸，水下　　**宝藏**：任意

鲨华鱼人是捕猎水上与水底生物的邪魔般的恐怖存在。鲨华鱼人是无情的劫掠者，被沿海社区的居民们称为“海魔鬼”。它们会洗劫船只、渔村与海底社区，满足它们的嗜血欲望，夺取财宝，并向它们的凶残神祇—鲨鱼之神 **瑟寇拉 Sekolah** 献祭。

鲨华鱼人不断地对任何居住在其领地附近的族群发动战争。人鱼与其他水生族裔在这些攻击中首当其冲，但鲨华鱼人也会猎杀那些在海魔鬼宣称的水域中航行或游泳的空气呼吸者。鲨华鱼人通常与鲨鱼一起发动攻击，它们可以通过心灵感应命令鲨鱼。
