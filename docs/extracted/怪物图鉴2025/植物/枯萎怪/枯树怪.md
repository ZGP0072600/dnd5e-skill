# 枯树怪 Tree Blight

枯树怪长得就像一颗古老的、有着扭曲枝干和裂开空洞的朽木。它们以活物为食，具有强烈的嗜血欲望。它们过去进食的残留往往混杂在它们的树枝和根部之中。枯树怪会与其他枯萎怪合作，但当它们遇到其他树状生物，比如启蒙树木和树人时，它们也会发动攻击。

## 动作 Actions

**多重攻击 Multiattack。** 枯树怪发动两次枯枝撞击攻击并使用一次根系缠绕。

**枯枝撞击 Branch。**
*近战攻击检定*：+9，触及15尺 *命中*：16（3d6+6）钝击伤害

**根系缠绕 Grasping Root。**
*力量豁免检定*：DC 17，单一15尺内枯树怪可见的不超过大型的生物。
*失败*：目标被枯树怪拉近至多10尺，并被其六条树根之一缠绕，陷入受擒状态（逃脱 DC 16）。目标在其回合开始时受到13（2d6+6）钝击伤害，直至擒抱结束。

## 附赠动作 Bonus Actions

**朽躯啮咬 Gnash。**
*敏捷豁免检定*：DC 17，单一受擒于枯树怪的生物。*失败*：19（3d8+6）穿刺伤害。*成功*：半伤。
