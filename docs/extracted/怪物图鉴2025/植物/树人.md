# 树人 Treant

*智慧而强大的生命之树*

**栖息地**：森林　　**宝藏**：无

作为森林的古老住民，树人是布满皱纹的巨型活化树木。这些古老生灵大多存活了数百年，它们通晓自然界的秘密。树人通常避免卷入短命种族的纷争，但会坚持守护自己的森林家园。若被激怒，树人能活化树木作为援军。树人守护着森林的秘密，同时也被这些秘密所影响。掷一次树人秘密表或从中选择一个结果，以此启发树人守护的秘密之事。

## 树人秘密 

| 1d6 | 这个树人…… |
|---|---|
| 1 | 曾被神明祝福，能结出魔法果实。 |
| 2 | 生长在地城入口或通往妖精荒野的传送门之上。 |
| 3 | 是皮克精 Pixie 或小仙灵 Sprite 族群的家园。 |
| 4 | 是失落的德鲁伊知识最后的守护者。 |
| 5 | 扎根于一位英雄的墓冢，能够活化树木，使其形似追寻使命的骑士。 |
| 6 | 曾被火焰灼伤，它的树洞中藏着纵火者的遗骸。 |

## 动作 Actions

**多重攻击 Multiattack。** 树人发动两次猛击攻击。

**猛击 Slam。**

*近战攻击检定*：命中+10，触及5尺。*命中*：16 （3d6 + 6）钝击伤害。

**碎木飞刺 Hail of Bark。**
*远程攻击检定*：+10，射程180尺。
*命中*： 28 
（4d10 + 6）穿刺伤害。

**活化树木 Animate Trees（1/日）。** 树人用魔法活化60尺内其可见的至多两棵树木。每颗被活化的树木均使用 **树人 Treant** 的数据卡，但有以下区别：树木的智力和魅力值为1且无法说话，并缺少活化树木动作。树木服从树人的命令，使用与树人一样的先攻并在其回合结束后立即执行回合。树木的活化持续1日，或在以下情况提前结束：其死亡时、该树人死亡时或其与该树人之间的距离超过120尺时。解除活化后，若条件允许，树木重新扎根。
