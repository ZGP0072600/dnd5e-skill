# 尖叫菌 Shrieker Fungus

## 反应 Reactions

**尖叫 Shriek。**
*触发*：生物或明亮光照的光源移动至尖叫菌的30尺内。
*响应*：尖叫菌发出300尺内都能听见的尖叫，持续1分钟或在尖叫菌死亡时提前结束。
