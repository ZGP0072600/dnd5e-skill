# 启蒙植物 Awakened Plants

*被魔法注入生命的植物*

**栖息地**：森林　　**宝藏**：无

魔法能够赋予移动能力，智慧，乃至声音。诸如启蒙术、或者其他位面存在的影响都可以给平凡的植物带来生命。当然，一些具有非凡特性的植物可能天生就有这些特质。
