# 人马 Centaurs

*妖精荒野的捍卫者*

**栖息地**：森林，草原，异界（妖精荒野）　　**宝藏**：武备，个体

人马是森林、平原和充满原初伟力之地的捍卫者。人马的上半身就像人类，而下半身则是骏马，他们会冲锋突入战场，挺身对抗那些伤害他们盟友的敌人。
