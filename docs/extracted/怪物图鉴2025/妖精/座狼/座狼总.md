# 座狼 Worgs

*狡诈恶毒的类狼破坏者*

**栖息地**：森林、草原、丘陵、异界（妖精荒野）　　**宝藏**：无

座狼是凶残的猎手，有时会被误认为巨狼。这些拥有智慧的捕食者会说话，并会经常嘲弄它们的猎物，享受猎物恐惧的滋味。
