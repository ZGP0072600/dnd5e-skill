# 半羊人 Satyrs

*羊角羊蹄的狂欢者*

**栖息地**：森林、异界（妖精荒野）　　**宝藏**：器具

半羊人象征着荒野之上那无拘无束的快乐。狂欢之热令他们陶醉——他们宴饮，他们歌舞，他们比武，他们嬉戏。
