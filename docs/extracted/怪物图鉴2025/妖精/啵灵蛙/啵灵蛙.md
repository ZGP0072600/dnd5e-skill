# 啵灵蛙 Bullywug

*沼泽与滩涂中的两栖蛙灵*

**栖息地**：沼泽　　**宝藏**：器具，个体

啵灵蛙是沼泽地的妖精化身。它们守护着朦胧的野外沼地——并认为自己担任的这一角色广受欢迎。啵灵蛙大小与人类相仿，但却长着蛙类和蟾蜍的形貌；这些蛙人和沼泽地中栖息的生物们同样关系密切。
