# 皮克精 Pixies

*浩瀚森林的要好朋友*

**栖息地**：森林、异界（妖精荒野）　　**宝藏**：奥秘

皮克精们不到一尺高，外表像一群小号的，背后长有精巧轻盈翅膀的精灵。他们悄无声息地偷窥那些进入他们林地之家附近的家伙，遇到友好的家伙才会现身。至于那些不友好的家伙嘛，等着被皮克精们恶作剧吧。
