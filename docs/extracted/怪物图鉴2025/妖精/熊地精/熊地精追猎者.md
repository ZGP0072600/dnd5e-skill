# 熊地精追猎者 Bugbear Stalker

熊地精追猎者经常将受害者劫为人质，并乐于借此机会囚禁和恐吓其他生物。

## 特质 Traits

**劫持 Abduct。** 熊地精移动一名正被其擒抱的生物时无需额外消耗移动力。

## 动作 Actions

**多重攻击 Multiattack。** 熊地精使用标枪或钉头锤发动共计两次攻击。

**标枪 Javelin。** *近战或远程攻击检定*：+5，触及10尺或射程30/120尺。*命中*：13（3d6+3）穿刺伤害。

**钉头锤 Morningstar。** *近战攻击检定*：+5（若目标因该熊地精而受擒则具有优势），触及10尺。*命中*：12（2d8+3）穿刺伤害。

## 附赠动作 Bonus Actions

**快速擒抱 Quick Grapple。** *敏捷豁免检定*：DC 13，单一10尺内体型不超过中型且熊地精可见的生物。*失败*：目标陷入受擒状态（逃脱 DC 13）。
