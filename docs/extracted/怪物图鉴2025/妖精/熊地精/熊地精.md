# 熊地精 Bugbears

*潜隐伏击的类地精蛮兵*

**栖息地**：森林，草原，异界（妖精荒野），幽暗地域　　**宝藏**：武备，个体

熊地精代表着对荒野的恐惧和自然地域的威胁。它们以行踪诡秘著称，冒险进入它们领地的敌人通常会消失得无影无踪。
