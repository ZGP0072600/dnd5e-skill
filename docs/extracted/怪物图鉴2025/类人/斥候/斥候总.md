# 斥候 Scouts

*漫游荒野，守望文明*

**栖息地**：任意　　**宝藏**：个体

斥候是活跃于野外的武者，修习狩猎与追迹之道。它们可能是探险家，亦可能是陷阱师，还可能有着更具军事素养的身份，例如弓箭手、赏金猎人、游骑兵。
