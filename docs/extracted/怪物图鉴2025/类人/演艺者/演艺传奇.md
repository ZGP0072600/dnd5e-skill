# 演艺传奇 Performer Legend

演艺传奇名扬海内外，驰名于整片大陆，甚至整个世界。它们的声望可以助其踏入社会顶流，或是反过来利用表演来操弄观众。演艺传奇通常都有独属于自己的某种特征或是天赋，例如独特的声音、非凡的运动能力、极具特色的穿衣风格。演艺传奇也经常和其他演艺者、警卫、或者贵族金主一并旅行。

## 动作 Actions

**多重攻击 Multiattack。** 演艺传奇发动三次镶玉指挥杖攻击。

**镶玉指挥杖 Bejeweled Baton。** *近战攻击检定*：+9，触及5尺。*命中*：10（2d4+5）钝击伤害外加10（3d6）心灵伤害。

**庄严之歌 Majestic Song。** *感知豁免检定*：DC 17，以120尺内一点为中心半径20尺的球状区域内的每个生物。*失败*：22（4d8+4）心灵伤害，且目标陷入魅惑或恐慌状态（由演艺传奇选择），直至演艺传奇的下个回合结束。*成功*：仅半伤。

**施法 Spellcasting。** 演艺传奇施展以下一道法术，无需材料成分并使用魅力作为施法属性（法术豁免 DC 17）：

**随意：** 法师之手 Mage Hand，次级幻象 Minor Illusion，魔法伎俩 Prestidigitation

**每项1/日：** 高级幻影 Major Image，投影术 Project Image

## 反应 Reactions

**魅力之佑 Warding Charm。** *触发*：一名生物对演艺者进行攻击检定并命中。*响应-感知豁免检定*：DC 17，触发生物。*失败*：此次攻击检定改为失手，且目标陷入魅惑状态直至演艺传奇的下个回合结束。
