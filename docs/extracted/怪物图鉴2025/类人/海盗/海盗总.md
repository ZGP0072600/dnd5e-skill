# 海盗 Pirates

*劫掠者，觅宝人*

**栖息地**：任意　　**宝藏**：个体，器具

“海盗”一词泛指各类海员，包括于海上漫游的恶匪，寻猎船只的私掠船，被诅咒的宝藏猎人，以及其他意图在海上寻得财富与名誉之人。

海盗在冒险中可以是盟友，可以是敌人，可以是乱入者，甚至可以亦敌亦友。它们是商船和沿海社群的灾祸，却也熟知海底的秘密，并知晓如何躲避水中的危险。某些海盗甚至将目光从海浪之上移开，转而使用飞艇、魔法船、跨位面载具、甚至是更加诡异的载具，在不可思议的领域中探索，劫掠。

为了震惧对手，散布威名，海盗们会挂上独特的旗帜。掷两次海盗旗表或从中选择两个结果，以此启发一伙海盗们所使用的旗子是什么样子。
