# 海盗船长 Pirate Captain

海盗船长号令着手下的众多船员，为其指明目的与方向。海盗船长们会为自己营造戏剧般的恐怖形象，将自己描绘成比凡人更可怕的存在，以此吓退敌人，威慑船员们保持忠诚，并吸引更多追随者加入到他们的旗帜之下。许多海盗船长都有专属的绰号，其灵感来自于自己身上标志性的特征和惯用的手段。

## 动作 Actions

**多重攻击 Multiattack。** 海盗船长使用刺剑或手铳发动共计三次攻击。

**刺剑 Rapier。** 近战攻击：+7，触及5尺。*命中*：13（2d8+4）穿刺伤害，且海盗船长在本回合结束前进行的下次攻击检定具有优势。

**手铳 Pistol。** 远程攻击：+7，射程30/90尺。*命中*：15（2d10+4）穿刺伤害。

## 附赠动作 Bonus Actions

**领袖魅力 Captain’s Charm。** 感知豁免：DC 14，单一30尺内海盗船长可见的生物。*失败*：目标陷入魅惑状态直至海盗船长的下个回合开始。

## 反应 Reactions

**弹反 Riposte。** *触发*：海盗船长在持握武器期间因近战攻击检定被命中。*响应*：海盗船长令其对抗那次攻击的 AC+3，可能令那次攻击改为失手。若那次攻击失手，且触发生物位于船长的触及范围内，则船长对该生物发动一次刺剑攻击。
