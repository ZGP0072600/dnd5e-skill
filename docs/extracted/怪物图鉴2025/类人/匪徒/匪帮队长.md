# 匪帮队长 Bandit Captain

匪帮队长指挥着恶徒们结成的帮派，它们亦可能充当更高级的罪犯手下的警卫与打手。

## 动作 Actions

**多重攻击 Multiattack。** 匪帮队长使用弯刀或手铳发动共计两次攻击。

**弯刀 Scimitar。** *近战攻击检定*：+5，触及5尺。*命中*：6（1d6+3）挥砍伤害。

**手铳 Pistol。** *远程攻击检定*：+5，射程30/90尺。*命中*：8（1d10+3）穿刺伤害。

## 反应 Reactions

**格挡 Parry。** *触发*：匪帮队长在持握武器期间因近战攻击检定被命中。*响应*：匪帮队长令其对抗那次攻击的 AC+2，可能令那次攻击改为失手。
