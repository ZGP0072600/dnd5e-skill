# 打手 Toughs

*街头打手，横行暴徒*

**栖息地**：任意　　**宝藏**：武备

“打手”涵盖了保镖、斗殴者、甚至是力工。这些人使用强健的身体威吓敌人，其身份可能是好斗的罪犯、吵闹的酒客、甚至经验丰富的工人，以及一切使用肌肉来实现目的之人。
