# 祭司 Priests

*裁量圣旨，审断凡尘*

**栖息地**：任意　　**宝藏**：个体，圣物

牧师驾驭信仰之力，于尘世中行施奇迹。这些宗教信徒的类型和其所信仰的对象那般多样。一些祭司服从于其所信仰的神祇及其仆从，而另一些祭司则践行着古老的信条。信仰引领着祭司们的行动，为它们的魔法之力指引方向，让祭司们依照自己所属的意识形态塑造这个世界。

掷一次祭司身份表或从中选择一个结果，以此启发这名牧师属于什么样的类型。
