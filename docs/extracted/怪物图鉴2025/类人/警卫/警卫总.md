# 警卫 Guards

*警惕街巷，抵御众敌*

**栖息地**：任意　　**宝藏**：武备，个体

警卫们在金钱或是责任心的驱使下守卫着民众、守卫着地点、甚至守卫着特定物品。它们在履行职责时可能十分谨慎，亦可能粗心大意。一些警卫只要发现危机的痕迹就会在第一时间发出警报，并拼命守卫自己应守之地。另一些警卫则可能考虑自己的薪资与面对的危机是否匹配——若不匹配，他们就会直接逃跑。
