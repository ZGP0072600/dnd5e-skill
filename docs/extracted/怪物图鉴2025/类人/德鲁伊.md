# 德鲁伊 Druid

*卫戍自然，维护自然*

**栖息地**：任意　　**宝藏**：圣物，个体

德鲁伊们驾驭原初魔法，遵循古老教义，与动物和其他玄秘存在紧密联结，捍卫着广袤自然，净化着世界的疾病。这些魔法使可能是隐居于某地的隐士，亦可能从属于某一神秘组织。掷一次德鲁伊使命表或从中选择一个结果，以此启发这名德鲁伊如何使用魔法践行其使命。

## 德鲁伊使命 

| 1d6 | 这位德鲁伊是一位…… |
|---|---|
| 1 | 与伤害自然的破坏型文明抗争的复仇战士。 |
| 2 | 帮助旅者们安全穿过栖息着妖精、野兽与植物之国度的向导。 |
| 3 | 独自守护着作为其家园的大地、海洋、甚至是天空的隐士。 |
| 4 | 修复自然界的创伤的旅者，治愈天灾与人祸给世界留下的伤痕。 |
| 5 | 松散组织的成员，坚守着古老的仪式，守卫着自然的秘密。 |
| 6 | 监视着现实世界的基石、警惕着来自异界之威胁的卫士。 |

## 动作 Actions

**多重攻击 Multiattack。** 德鲁伊使用荆藤法杖或苍翠之力发动共计两次攻击。

**荆藤法杖 Vine Staff。** *近战攻击检定*：+5，触及5尺。*命中*：7（1d8+3）钝击伤害外加2（1d4）毒素伤害。

**苍翠之力 Verdant Wisp。** *远程攻击检定*：+5，射程90尺。*命中*：10（3d6）光耀伤害。
