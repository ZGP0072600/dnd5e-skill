# 魔法师 Mages

*魔法学士，驾驭法咒*

**栖息地**：任意　　**宝藏**：奥秘，个体

魔法师们是行使魔法，创造奇迹之人，从拥有施法能力的霸主到隐世研学的女巫，都可以被称为魔法师。它们研究玄奥的隐秘，研习怪物、传说、预兆、以及其他学识。魔法师们经常召集盟友或雇佣助手来助其进行研究，或是获得强大的魔法力量。
