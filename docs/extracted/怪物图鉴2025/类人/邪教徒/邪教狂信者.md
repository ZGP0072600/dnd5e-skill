# 邪教狂信者 Cultist Fanatic

邪教团通常围绕着一位魅力十足或独断专政的领导人建立了一系列等级制度。虽然邪教团的成员可能会自发地行动，但它们通常都听命于掌握了更强大非凡力量的上级。任何邪教团都可以包括以下几种成员：

邪教狂信者领导着小规模的邪教，它们亦可能是更大的教团中的一份子。狂信者们比低阶的信徒们更加了解教团的隐秘，并能从其宗主处获得魔法力量。
