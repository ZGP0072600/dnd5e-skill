# 邪教徒 Cultists

*宣扬末日，不移狂信*

**栖息地**：任意　　**宝藏**：个体，圣物

邪教徒使用魔法以及其他极端手段来传播其激进的信仰。一些邪教团是追求玄奥隐秘的密教团体，而另一些则是活跃于暗面的阴谋集团，意图实现某种怖人的大业。邪教徒们通常遵循着晦涩的密教习俗行事，或是醉心于解读古老的预言。它们还可能将其信仰献上给超自然的宗主——神灵、异界的生物、擅于操弄凡人的异质心灵、甚至是其他更加诡异的力量。掷一次邪教大业表或从中选择一个结果，以此启发一伙邪教徒们会意欲达成什么样的目标。

**邪教大业 **

| 1d6 | 这一邪教团致力于…… |
|---|---|
| 1 | 终结一个政权、一个时代、甚至是整个世界。 |
| 2 | 燃尽世间那些欺骗着凡人的美好，揭露其下的残酷现实。 |
| 3 | 通过洗脑和压迫来扩张其的信仰。 |
| 4 | 改变整个世界，例如令大地悉数沉没、令万千火山喷发。 |
| 5 | 大规模地重新塑造生命的形态，改变其他生物的肉体或精神。 |
| 6 | 将其“神灵”召唤入现世，至少也要召唤其神灵的令使、武器、甚至是“神国”。 |

邪教徒通常依靠圣印来分辨教团的成员。这些圣印通常印在邪教的标志性物品上，亦可能印在衣物上、甚至可能直接刻在邪教徒的身体上。圣印的意义可能很容易理解，亦可能只有教团成员才能分辨。掷一次邪教圣印表或从中选择一个结果，以此启发邪教团会使用什么作为自己的标志。

**邪教圣印 **

| 1d10 | 圣印是…… | 被描绘为…… |
|---|---|---|
| 1 | 一枚炼金学符印 | 一份万年历或地图 |
| 2 | 一种动物 | 一份羽饰或某种纹章 |
| 3 | 一颗天体 | 一份复杂的图样 |
| 4 | 一尊神灵的徽记 | 一张暗示性的图画 |
| 5 | 一种元素 | 一个神秘存在 |
| 6 | 一颗眼睛 | 某份公式的一部分 |
| 7 | 一个几何图形 | 一张重复的图案 |
| 8 | 一个字母或是一个数字 | 一系列抓痕 |
| 9 | 怪物的一部分 | 一个简单的象形字画 |
| 10 | 一颗颅骨 | 一把武器或工具 |
