# 元素邪教徒 Elemental Cultist

邪教团可以围绕任何神秘传统建立组织，但大部分邪教团侍奉某个超自然存在。而这些邪教团的成员则拥有与其崇拜的力量相关的能力。

元素邪教徒驾驭着毁灭性的原初伟力，用破坏去净化文明在世界上留下的痕迹。它们的目标亦可能是证明某一元素应当凌驾于构成世界的其他元素之上。这些邪教徒与元素怪物以及诸如邪恶元素亲王或上古元素之眼的不朽邪恶结盟。
