# 吉斯洋基人 Githyanki

从左至右：吉斯洋基人星辰先知、吉斯洋基人异构师、吉斯洋基人海盗

吉斯洋基人和吉斯泽莱人曾经都是一个古代民族的后裔。这些高大、枯瘦的类人生物拥有强大的灵能，并且绝大多数都居住在星光位面里。其中最著名的吉斯洋基人是巫妖女王维拉基斯的追随者。它们为星界带来恐惧，闯进了荒宇并掠劫整个多元宇宙的魔法和财富。

《怪物手册》和《多元宇宙怪物书》中包含了各种各样的吉斯洋基人供你使用。本节还提供了其他选项。

## 吉斯洋基海盗 Githyanki Buccaneer

吉斯洋基海盗们致力于在星界中寻找财富，然后将搜刮来的财富带回星界深渊的隐藏堡垒中。它们中的许多人是不再愿意为巫妖女王维拉基斯服务的战士；它们更喜欢按照自己的准则生活，或者陶醉于自己那肆无忌惮的自由。

## 吉斯洋基星辰先知 Githyanki Star Seer

吉斯洋基星辰先知相信星星是多元宇宙的眼睛。它们利用魔法联系埃克马（Acamar）、凯丰（Caiphon）、哈达（Hadar）等远古星际实体，希望了解它们的秘密，然后将这些秘密记录在日记中。它们也在荒宇中寻找新的实体，希望成为第一个联系它们的人。

先知们所学到的一些秘密是如此神秘，以至于需要多年的研究才能破译，但对于主要居住在星界的星辰先知来说，时间并不那么重要。

## 吉斯洋基异构师 Githyanki Xenomancer

吉斯洋基异构师前往最远的荒野和星海，甚至不时访问主物质位面的世界，研究和编录它从未遇到过的生物。与智能生物的友好接触可以将异构师的外交技能发挥到极致，而敌对接触则成为对异构师生存技能的考验。

有时，异构师的研究需要捕获并监禁样本（以研究其行为）或杀死并解剖（以研究或收集其内部构造）。许多异构师更喜欢在它们位于星光位面的实验室里做这项工作。
