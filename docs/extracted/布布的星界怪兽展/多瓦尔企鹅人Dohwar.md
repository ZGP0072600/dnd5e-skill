# 多瓦尔企鹅人 Dohwar

多瓦尔企鹅人，是一种矮小，粗胖，不会飞的鸟类生物，外形和动作都非常像企鹅。它们打从骨子里就是商人，永远在寻找着下一个商机。多瓦尔企鹅人的交易从来都是秘密进行，哪怕它们买卖的东西既不违法又不危险，它们也喜欢在各种黑街暗巷和偏僻角落里面做买卖。它们的服装一般都花里胡哨，但也有的喜欢只披件连帽斗篷。多瓦尔企鹅人的集团会被称为“垄断联盟（Cartel）”

多瓦尔企鹅人平均身高3尺，羽色鲜艳。它们没有翅膀，但是有胳膊和小手。

多瓦尔企鹅人喜欢用一种它们称之为“交融（Merging）”的心灵感应方式交流，即两个多瓦尔企鹅人保持精神联系，同时都与第三个生物进行心灵感应。

多瓦尔企鹅人有着自己的神系，全都是与商业和财富有关之神，包括渥金 Waukeen，阿巴索 Abbathor 和兹尔图斯 Zilchus 等。多瓦尔人希望通过在专门崇拜这些神的同时不疏远或冒犯其中任何一位来成为全多元宇宙最厉害的商人。
