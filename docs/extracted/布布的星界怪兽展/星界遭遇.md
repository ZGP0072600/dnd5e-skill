# 星界遭遇

你可以通过掷骰并对照荒宇遭遇表 Wildspace Encounters table 或星界海遭遇表 Astral Sea Encounters table 来随机生成一个星界遭遇或者按照自己的心意直接选择。如果该遭遇有涉及到魔法船，你可以通过掷骰并对照舰船遭遇表 Ship Encounters table 来决定船的类型以及船员，或者你也可以自创一个属于你自己的舰船遭遇（参见《星界冒险者指南 Astral Adventurer’s Guide》来获得舰船的详述）。

标有星号(*)的生物均出现在本书中；其余的则在《怪物图鉴》中都有描述。任何带有匕首()标记的生物都可以充当法驱者，因为它是一位施法者。

如果一名类人生物不具有特定种族，它可以是你所选择的任何种族。

初始态度 Initial 
Attitude

根据遭遇表上的标记来掷态度骰，然后在初始态度表上进行对照以随机决定遭遇生物的初始态度。

初始态度 Initial 
Attitude

| 态度骰总值 | 初始态度 |
|---|---|
| 4或更低 | 敌对 |
| 5-8 | 冷漠 |
| 9或更高 | 友好 |
