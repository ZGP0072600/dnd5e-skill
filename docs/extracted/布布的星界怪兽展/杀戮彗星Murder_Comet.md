# 杀戮彗星 Murder Comet

通过融合土元素和火元素的精华，邪恶的施法者创造出了杀戮彗星。一颗杀戮彗星看起来像是一颗尖啸的岩石脑袋，周身环绕着烈焰。当它拖着燃烧的尾迹在荒宇中飞行时无论是单独还是与其他杀戮彗星一起杀戮彗星都会寻找船只、船员或其他毫无防备的目标，然后将它们统统摧毁殆尽。

杀戮彗星的创造者可以将自己的灵魂绑定于彗星身上。在这种形态下，彗星的创造者会停止衰老，变得不朽，脱离空气、睡眠和物质给养的桎梏，自由地在荒宇中飞驰。杀戮彗星的面孔会呈现出其创造者的容貌，而其数据发生以下改变：

将彗星的智力、感知和魅力替换为其创造者的数值。

将彗星的阵营和语言替换为其创造者的阵营和语言。

### 杀戮彗星 Murder Comet

中型元素，普遍中立邪恶

护甲等级 19（天生护甲）

生命值 66（7d8+35）

速度 0尺，飞行120尺

力量15（+2）敏捷15（+2）体质20（+5）

智力6（-2）感知10（+0）魅力6（-2）

伤害免疫 火焰，毒素

状态免疫 力竭，麻痹，石化，中毒，倒地，昏迷

感官 黑暗视觉240尺，被动察觉10

语言 火族语，土族语

挑战等级 5（1，800XP）熟练加值+3

爆炸 Explode。当彗星生命值降低为0时，它会爆炸并波及以自己为源点20尺半径的球形区域。区域内的每个生物都必须进行一次 DC 16的敏捷豁免，失败则受到28（8d6）点火焰伤害，成功则仅受到一半伤害。

飞掠 Flyby。飞出敌人的触及范围时，彗星不会引发借机攻击。

照明 Illumination。彗星散发出30尺半径的明亮光照，以及在此之外30尺的微光光照。

攻城怪物 Siege Monster。彗星对物件和建筑物造成双倍伤害。

非凡本质 Unusual Nature。彗星不需要空气、食物、水、与睡眠。

动作

多重攻击 Multiattack。彗星进行一次猛击和一次喷火攻击。

猛击 Slam。近战武器攻击：命中+5，触及5尺，单一目标。命中：6（1d8+2）点钝击伤害，外加7（2d6）点火焰伤害。

喷火 Spit Fire。远程武器攻击：命中+5，射程60尺，单一目标。命中：13（2d10+2）点火焰伤害。
