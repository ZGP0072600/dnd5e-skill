# 灵蠕妖Psurlons

## 灵蠕妖 Psurlons

灵蠕妖是一种邪恶的、生活在星界的蠕虫状生物。一个完全长大的样本有7英尺长。它们的腿部末端是蹄子，手臂的末端则是三根长长的手指。灵蠕妖没有眼睛的头部类似于蚯蚓，上面有一张长满牙齿的大嘴。灵蠕妖以色彩丰富的长袍为装饰，很少穿戴盔甲或携带武器。

灵蠕妖的寿命长达数千年，因为它们大部分时间都居住在星界深渊*。每隔一百年左右，灵蠕妖就会离开它们的星界据点，入侵荒宇系，并沉浸在一个被称为“凡世盛宴”的、长达7年的仪式中，在此期间，它们会尽可能多地吞噬知性生命，然后返回星界。灵蠕妖更喜欢人类和半身人的肉，但也不介意在其它种族身上大快朵颐。它们会利用自己的法术能力，来潜入它们的目标受害者的定居点。

在物质位面中，已知灵蠕妖会与夺心魔合作。它们会一起收集可以食用的受害者；夺心魔吞噬受害者的大脑，而灵蠕妖则享用猎物的其余部分。由于这个联盟，吉斯洋基人也憎恶灵蠕妖，并攻击一切它们能够在星界深渊找到的灵蠕妖堡垒。

当灵蠕妖死亡时，其它灵蠕妖会将其尸体存放在一个安全的地方。当尸体腐烂后，灵蠕妖就在里面产下一个或多个卵。这些卵在24小时后孵化，每一个都会诞生出一条微型幼虫。在接下来的七天里，这些幼虫以尸体为食，并相互吞食，直到只剩下一只。这条幼虫从其死亡宿主的腐烂遗体中爬出来，成为一个成年灵蠕妖。

## 灵蠕妖假替者 Psurlon Ringer

可以使用魔法来模仿一个特定中型类人生物的一种灵蠕妖。首先，灵蠕妖必须吃下它想模仿的生物。然后它进入持续8小时的心灵出神状态，结束时它就会变成所吃生物的样子。灵蠕妖会获得该生物的记忆和语言，但不会获得其职业特性或其它能力。这种转变是永久性的，只有通过祈愿术才能解除。无论外表如何，灵蠕妖仍然是一个异怪，其它灵蠕妖也能认出它的真身。
