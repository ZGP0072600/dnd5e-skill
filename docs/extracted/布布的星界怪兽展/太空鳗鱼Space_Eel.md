# 太空鳗鱼 Space Eel

除非饿得发慌，否则太空鳗鱼一般会避免与其他比自己巨大的生物的正面接触。这些食腐动物长约5尺，可能会尾随一艘魔法船，并以它们船体上脱落的藤壶为食。荒宇的猎人们会试图捕杀这些鳗鱼来猎获它们的肉这工作说起来容易做起来难。

### 太空鳗鱼 Space Eel

小型野兽，无阵营

AC：14

HP：7（2d6）

速度：10尺，飞行30尺

力量12（+1） 敏捷18（+4） 体质11（+0）

智力 1（-5） 感知10（+0） 魅力 1（-5）

感官：黑暗视觉60尺，被动察觉10

语言：-

挑战等级：1/2（100 
XP）熟练加值：+2

非凡本质 Unusual Nature。太空鳗鱼不需要空气。

动作

多重攻击 Multiattack。如果其未附着于一个生物，鳗鱼进行一次啃咬攻击和一次尾棘攻击。

啃咬 Bite。近战武器攻击：命中+3，触及5尺，单一生物。命中：4（1d6+1）点穿刺伤害，且鳗鱼附着于目标身上。附着时，鳗鱼不能进行啃咬攻击。取而代之的是目标会在鳗鱼的每个回合开始时受到4（1d6+1）点穿刺伤害。鳗鱼可以用一个附赠动作解除自己的附着。包括目标在内的一个生物可以用一个动作让鳗鱼解除附着。

尾棘 Tail Spine。近战武器攻击：命中+3，触及5尺，单一生物。命中：3（1d4+1）点穿刺伤害，且目标必须成功通过一次 DC 10的体质豁免，否则陷入中毒1分钟。目标在中毒期间还将陷入麻痹。目标可以在其每个回合结束时再次豁免，豁免成功则结束该效应。
