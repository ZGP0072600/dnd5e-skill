# 布布的星界怪兽展

简介

太空小丑之船

[振奋起来吧，冒险家们，你们已经获得了布布的青睐！ -明斯克]

[*吱！* 
-布布]

布布是一只仓鼠。准确来说，它是一只迷你巨型太空仓鼠而且实际上是一只相当有名的仓鼠。布布的凶残程度，以及与游侠明斯克的冒险经历都堪称传奇。但布布将会是第一个告诉你它只是来自被称为星界位面的星宇国度中无数令人惊叹的生物之一。

显然，巨型太空仓鼠不像它们更小的这位表亲那样聪明或传奇。明斯克在旅途之中用它那响亮的声音毫不保留地分享了这些来自布布的信息以及许多其他的知识。而这些知识中有很大一部分都会在本书中呈现但请不要以为那只著名的仓鼠会亲自来给你说明。因为它正忙于拯救多元宇宙正如明斯克所说“为了良善而狂踢屁股。”
