# 星海商族 Mercane

星海商族是一群主要交易魔法物品与先进技术的商人，其中甚至包括神器与法驱魔舵。这些瘦长的蓝色巨人身高12尺，穿着优雅的长袍，头和手指也是同样细长。它们驾驶着魔法船航行于星界海，并在各个荒宇系中的世界之间来回穿梭，开展主要业务。只有很罕见的情况下才能在同一时间目击复数个星海商族，不过它们通常也有下属与保镖陪同。

星海商族会以公平诚信的可靠态度与任何人做买卖，只要对方并没有在过去伤害或欺骗过另一位星海商族。星海商族有着一种特殊的心灵感应能力，使它们能够在多元宇宙之间彼此进行交流。星海商族会用这个能力警告其他星海商族危险或不可靠的对象。当某个人冒犯了星海商族之后，想要再赢回它们的好感是几近不可能的。

### 星海商族 Mercane

大型巨人，普遍守序中立

AC：13（法师护甲）

HP：75（10d10+20）

速度：30尺

力量16（+3） 敏捷10（+0） 体质15（+2）

智力18（+4） 感知16（+3） 魅力15（+2）

豁免：智力+7，感知+6，魅力+5

技能：洞悉+9，察觉+6，游说+5

感官：被动察觉16

语言：通用语，巨人语，心灵感应60尺（另见“星海商族心灵感应”）

挑战等级：5（1，800 XP）熟练加值：+3

星海商族心灵感应 Mercane Telepathy。星海商族可以用心灵感应的方式与所有它认识的星海商族进行交流，无论它们之间的距离有多远。

动作

多重攻击 Multiattack。星海商族进行三次灵能注刃攻击。

灵能注刃 Psi-Imbued Blade。近战武器攻击：命中+6，触及5 尺，单一目标。命中：10（2d6+3）点挥砍伤害，如果目标是生物，则该生物必须成功通过一次 DC 15的感知豁免，否则陷入对星海商族的恐慌，直到它的下个回合结束。

施法（灵能）Spellcasting(Psionics)。星海商族可以施展下列的一道法术，而不需要任何法术成分，其施法属性为智力（法术豁免 DC 15）：

随意：侦测魔法，光亮术

每项1/日：任意门，隐形术，法师护甲（仅自身）
