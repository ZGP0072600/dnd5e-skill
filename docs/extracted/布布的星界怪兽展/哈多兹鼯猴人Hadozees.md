# 鼯猴人 Hadozees

鼯猴人是苗条的、适应性强的类人生物，具有人猿的特征和自己的语言。它们的脚和手一样灵巧，甚至可以把脚趾当作对生拇指来使用。翼膜宽松地挂在它们的手臂和大腿之间。通过伸展四肢来将这些翼膜绷紧，鼯猴人就可以进行滑翔。

在它们的家乡世界亚齐尔，鼯猴人住在树上的高处，以躲避在陆地上行走的大型捕食者。今天，它们中的大多数已将荒宇当成了家园。鼯猴人经常被其它宇航种族雇为佣兵。它们天生的攀爬能力很有用，它们的滑翔能力也是。习惯于太空旅行的鼯猴人知道如何在战斗中利用船只的重力平面从船只的一侧跳下，在龙骨下滑翔而过，然后落在船只的另一侧。

## 鼯猴人探索者 Hadozee Explorer

鼯猴人探索者在荒宇系中寻找财富和冒险。它们经常在魔法船上担任领航员。对它们来说，星界是一个值得大步探索、遍布神秘的广阔空间。

反应

安全降落 Safe 
Descent。当鼯猴人即将受到坠落伤害时，只要没有穿着重甲，它就能通过展开翼膜来将坠落伤害减少至0。

Wingbats）”和“夜嚎者（Night 
Howlers）”*。
