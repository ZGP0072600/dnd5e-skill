# 毕罗格四臂巨人 B'rohg

毕罗格四臂巨人是一种高达15尺，肤色焦黄、身材魁梧的四臂巨人。它们鼻平而耳尖，一头长发从秃顶的边缘垂落下来。它们穿着简朴，喜欢投掷沉重的岩石。毕罗格四臂巨人的四条手臂均可持用武器，但它们更喜欢赤手空拳地猛击敌人以及将比自己小的生物撕成碎片。

毕罗格四臂巨人通过咕噜声和基础的手势交流；它们没有语言或文字。它们喜欢独处，因此通常不会对其他更小的生物构成威胁。但由于它们拥有的体型和力量，角斗场往往用它们来吸引观众，令其被迫为食物而战。

毕罗格四臂巨人已习惯于在自己的自然栖息地中勉强生存，很少与其他种类的生物联合。从陌生人的善意行为中受益的毕罗格四臂巨人首先会对这种努力感到疑惑，然后是警惕。如果这种善意足以赢得其信任的话，毕罗格四臂巨人就会努力回报陌生人——也许是为他们携带装备或帮助他们穿越危险的地形。一个被冒险者队伍结识的独行毕罗格四臂巨人可能会跟随他们一段时间，但它很快就会厌倦他们的陪伴，再次寻找其他同类。

### 毕罗格四臂巨人 B'rohg

巨型巨人，任意阵营

AC 14（天生护甲）

HP 92（8d12+40）

速度40尺

力量21（+5）敏捷14（+2）体质21（+5）

智力 5（-3）感知10（+0）魅力 7（-2）

技能 运动+8，求生+6

感官 被动察觉10

语言 —

挑战等级6（2，300XP）熟练加值+3

动作

多重攻击 Multiattack。毕罗格四臂巨人发动四次拳击或两次投石攻击。

拳击 Fist。近战武器攻击：命中+8，触及10尺，单一目标。命中：12（2d6+5）点钝击伤害。

投石 Rock。远程武器攻击：命中+5，射程60/240尺，单一目标。命中：23（4d8+5）点钝击伤害。

骇人撕裂 Hideous Rend。毕罗格四臂巨人用它的全部四只手抓取自身10尺范围内的一个至多为大型的可视生物。目标必须进行一次 DC 16的敏捷豁免否则将被擒抱（逃脱 DC 16）。直到擒抱结束，毕罗格四臂巨人都不能使用拳击或投石攻击，且目标在其每个回合开始时受到49（8d10+5）点钝击伤害。一个被该伤害降至0生命值的生物会被撕成四块。
