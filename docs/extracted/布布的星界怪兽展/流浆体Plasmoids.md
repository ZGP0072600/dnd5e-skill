# 流浆体Plasmoids

## 流浆体 Plasmoids

流浆体是一种可以随意改变自身形状的知性泥怪。它们无定形的身体具有柔软、可塑的质地，流浆体可以挤出多达五个伪足，每个伪足长达5英尺。它们利用它们的伪足来行动，也用来操纵物体。

每个流浆体都有由大量的神经组成的一颗大脑。流浆体没有人类所知的内部器官，但它们有用于吸收氧气的气孔。流浆体在其一生中最多有四次机会，可以通过与同族的另一个成员结合来进行繁殖。结合后，两个流浆体分离，其中一个流浆体（随机确定）会分裂成两半。当一个流浆体一分为二时，产生的其中一个生物是原来的流浆体，而另一个是其物理上的复制体，缺乏父母的经验和知识。

流浆体通过强迫空气通过其身体中的管状空腔来说话，使其声音听起来如同口哨。

## 流浆体头目 Plasmoid Boss

有时，由于某种生体异常，一个流浆体在成年（20岁左右）后会产生体积膨胀。流浆体利用其增大的体型和力量对更弱小的生物发号施令，从而可以轻易地篡取权力，聚拢起一帮忠诚的下属、佞臣和潜在的篡夺者。流浆体头目可能会是一个公会的主人，一个犯罪企业的领导者，一个流浆体社区的大王，或是一艘魔法船的船长。无论它们为自己争取到什么样的角色，除了已有的力量和体型之外，流浆体头目还需要不可小觑的魅力和智慧，才能长久地保有权力。
