# 美学舰兽 Esthetic

每一只美学舰兽，都是一位雷加尔人（见本书后面的“雷加尔人”部分）创造出来的生化共生体。其本质上是一艘有且只有最基本意识的有机舰船。如果没有雷加尔人的引导，美学舰兽就会像一个没有思想的实体一样，只具备自我保护的本能。

雷加尔人使用它们创造出来的美学舰兽作为法驱战舰。当乘坐着美学舰兽的雷加尔人遇上了敌舰，它会使用一种魔法性的振动波让对方载具的法驱魔舵失效。接着雷加尔人就会命令美学舰兽抓住对面的船员，让它的触须分泌酸液将它们的血肉溶解。

每一只美学舰兽的外观都是独一无二的。可能一只是轴对称的（左右两边完全相同，就像是沿着中轴对称的类人生物身体一样），另一只则是放射性对称的（像海星），又或者干脆是不定型的。从远处看，美学舰兽很容易会被误认为是一只生活在太空中的巨大水母或者头足纲动物。其外壳是由一种会发出生物光的树脂制成的。

美学舰兽内部的空间足以舒适地容纳它的雷加尔主人以及额外最多六位中型乘客。船只的舱口只能通过雷加尔人（除此以外谁都不行）的触摸来打开或关闭。

美学舰兽可以在星光位面永远存活，前提是它的创造者也同样在世。如果美学舰兽的创造者死亡，美学舰兽也会在1至12天内生病，然后逝去。

### 美学舰兽 Esthetic

超巨型异怪，无阵营

AC：14（天生护甲）

HP：217（14d20+70）

速度：0尺，飞行60尺（悬浮）

力量24（+7） 敏捷8（-1） 体质20（+5）

智力1（-5） 感知10（+0） 魅力1（-5）

伤害免疫：强酸

状态免疫：目盲，魅惑，耳聋，力竭，恐慌，石化，倒地

感官：盲视300尺（在此范围外目盲），被动察觉10

语言：-

挑战等级：12（8，400 XP）熟练加值：+4

生物光照 Bioluminescence。只要其至少拥有1生命值，美学舰兽就会向周边30尺内发出明亮光照，以及再向外延伸30尺的微光光照，且它的内部空间是微光光照。

法驱 Spelljamming。美学舰兽拥有法驱魔舵的特性（见《星界冒险者指南》），但只有创造它的雷加尔人才能与之同调。

非凡本质 Unusual Nature。美学舰兽不需要空气、食物和水。

动作

多重攻击 Multiattack。美学舰兽进行两次触手攻击。

触须 Tentacle。近战武器攻击：命中+11，触及30尺，单一目标。命中：17（3d6+7）点力场伤害，如果目标是生物，则会被其擒抱（逃脱 DC 17）。擒抱期间，该生物在其回合开始时受到18（4d8）点强酸伤害，同时美学舰兽不能使用此触须攻击其他目标。美学舰兽拥有1d4x2根触须，每根触须都可以擒抱一个目标。

附赠动作

扰驱啸 Jammerscream。（充能6）美学舰兽以其300尺范围内的一艘魔法船为目标，将其法驱魔舵魔法地压制，持续2d10日。如果船上有多个魔舵，则随机确定是哪个魔舵受影响。与魔舵同调的生物可以进行一次 DC 17的魅力豁免。豁免失败时，该生物受到42（12d6）点心灵伤害，但魔舵只被压制2d10小时，而非2d10日。豁免成功时，该生物收到的伤害减半，魔舵只被压制2d10分钟，而非2d10日。
