# 邪魔灵魄 Fiendish Spirit

*六环 咒法（魔契师、法师）**Legacy*

施法时间：1 动作

施法距离：90 尺

法术成分：V, S, 
M（装有类人生物血液的红宝石瓶，价值至少 600gp）

持续时间：专注，至多 1 
小时

你唤来一个邪魔灵魄。其在施法距离内你可见的一个未占据位置化为实体。这一实体使用下面的 **邪魔灵魄 Fiendish Spirit** 
的数据。当你施展法术时，从恶魔 Demon，魔鬼 Devil 或尤格罗斯魔 Yugoloth 中选择一种。此生物外形如同你所选择类型的邪魔，其资料表中的部分特性也受此影响。当其生命值降至 
0 
或法术结束时，生物消失。

该生物在法术持续时间内对你和你的同伴们友好。在战斗中，该生物使用你的先攻，但在你回合结束后立即行动。它遵从于你的口头指令（无需任何动作）。如果你不发令，它将执行回避动作并远离危险。

升环施法。使用七环或更高法术位施展该法术时，该生物数据中使用更高的法术环数。
