# 天界灵魄 Celestial Spirit

*五环 咒法（牧师、圣武士）*
*Legacy* 

施法时间：1 动作

施法距离：90 尺

法术成分：V, S, M（一个的金制圣物匣，价值至少 
500gp）

持续时间：专注，至多 1 小时

你唤来一个天族灵魄。其在施法距离内你可见的一个未占据位置化为实体。这一实体使用下面的 
**天界灵魄 Celestial Spirit** 
 
的数据。当你施展法术时，从复仇圣灵 Avenger 或守护圣灵 Defender 中选择一个。此生物资料表中的攻击方式由你的选择决定。当其生命值降至 0 
或法术结束时，生物消失。

该生物在法术持续时间内对你和你的同伴们友好。在战斗中，该生物使用你的先攻，但在你回合结束后立即行动。它遵从于你的口头指令（无需任何动作）。如果你不发令，它将执行回避动作并远离危险。

升环施法。使用六环或更高环阶的法术位施展该法术时，该生物数据中使用更高的法术环数。
