# 教团

教团 Religious Order

> 嗯，为教团服务，然后很快你就会忙于处理成百上千件大祭司的洗衣杂活，因为——这可以方便地说——那是神的意愿。
>
> ——塔莎

**教团的类型 Types of Religious Orders**

教团类型 Religious Order Type

| d6 | 教团类型 |
|---|---|
| 1 | 亡灵猎手 Undead Hunters。 这个由学者和怪物猎手组成的组织不辞辛劳地调查着躁动的亡灵，追踪到他们的老巢，并让他们得到永恒的安息。 |
| 2 | 虔诚学者 Devout Scholars。 这一联合组织珍视和他们神灵有关的知识和文本，他们收集神圣的书籍，并记录带来神迹的先知们生平的故事。 |
| 3 | 圣物收集者 Relic Collectors。 这个考古学家侍僧团体试图以故事中的圣物装满他们博物馆般的庙宇殿堂。 |
| 4 | 慈善传教士 Charitable Missionaries。 这一教团相信着宗教将会赋予文明力量，他们不远万里去帮助被压迫者，以自己的道德榜样吸引新的信众。 |
| 5 | 激进派审判官 Militant Inquisitors。 一位武断而顽固的主教希望能排除一切信仰的威胁。 |
| 6 | 末日传道者 Doomsaying Evangelists。 这个教团认为世界将要迎来终结，他们认为说服他人相信这个事实可能会让自己在将至的厄运前幸免于难。 |

| 5 | 神眷治疗师 Beloved Healer。 一位著名的治疗师引导你到最需要你之处，尽管在你到达之前，他们这么做的缘由还不甚明了。 |
| 6 | 神之声 Divinity's Voice。 来自异界的讯息指引你前去执行神圣的任务。 |
