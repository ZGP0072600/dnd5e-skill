# 君主

君主 Sovereign

> 我对统治从来没有什么兴趣，有部分原因是这些头衔听起来都太无聊了。不过感觉“巫术女王”还挺顺耳的，你不觉得吗？
>
> ——塔莎

**君主类型 Types of Sovereign**

译者注：对于第二句，我的看法是意译为“王后、酋长、诸侯和宗亲”。不然费伦和灰鹰里难道有穆斯林和印度人吗？

再注：看了后文，我并不奇怪了。这都是什么用词，好多现实世界词汇，国会山都来了。

君主类型 Sovereign Type

| d6 | 君主类型 |
|---|---|
| 1 | 村中长老 Village Elder。一名衰老的长者照看着社区的居民，并指引他们的品行。 |
| 2 | 年轻贵族 Young Noble。一位高贵的野心家迫切地想要改革社会，以符合自己的个人愿景。 |
| 3 | 落难领袖 Shipwrecked Governor。一个绝望的领导者努力在陷入荒野时带领人们艰难求生。 |
| 4 | 往日之王 Ruler Returned。备受尊敬的部落领袖从死后世界归来，想要重获昔日的荣光。 |
| 5 | 隐秘势力 Hidden Power。一个神秘的人物暗中控制着国家的傀儡君王，以掌控政府真正的法令。 |
| 6 | 本应摄政 True Regent。合法的王位继承人试图从篡位者手中夺回权力。 |
