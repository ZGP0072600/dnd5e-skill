# 权贵

权贵 Aristocrat

> 强盗之国有句话，大意是 "贪财之人在收藏家来时最为亲切。"我不记得具体措辞了，但它用在这很应景。

**权贵的类型 Types of Aristocrats**

权贵类型 Aristocrat Types

| d6 | 权贵类型 |
|---|---|
| 1 | 当地领主 Local Lord。这位小贵族坚信权力与威望就在前方，他会抓住任何机会提升地位。 |
| 2 | 商业领袖 Merchant Mastermind。对于一个生意遍及世界的商业家族头领来说，名誉、财富与权力本就是三位一体的。 |
| 3 | 云游的王子 Nomadic Princeling。没有什么比财宝更加吸引这个小王子了。他庞大的商队尾随着他的轿子，蔓延到视线之外。 |
| 4 | 背叛者 Double Dealer。这位贵族家族的领袖为了个人利益或是其他原因，已经背叛了他的君主和他的国家。 |
| 5 | 野心勃勃的创业者 Ambitious Entrepreneur。 作为一笔庞大财富的唯一继承人，这位企业家寻求着能达成国际间、全球乃至位面间商业合作的盟友，公共增长他们的财富。 |
| 6 | 未来的统治者 Future Ruler。这位年轻的贵族注定要权御一方，但目前他心血来潮的想法是诡变而危险的。 |

权贵联络人 Aristocrat Contact

| d6 | 联络人 |
|---|---|
| 1 | 寻常联络人 Common Contact。一个决心获得官衔的仆从担当你赞助人的中介。 |
| 2 | 专家 Professional。一个头脑清醒的顾问或是赞助人生意的管理者指挥着你，以确保你那莽撞的雇主的安全。 |
| 3 | 家族调解者 Family Peacemaker。你的赞助人指派了一位天真的权贵，他希望你协助他维系家族成员间的岌岌可危的关系，而不是赞助人的首要任务。 |
| 4 | 亲密关系者 Intimate Connection。 你的联系人是赞助人出身平平的知己或爱人，他指导你以最大化赞助人的家族利益为目的进行工作。 |
| 5 | 出局的家人 Insider Outside。一位被逐出家族的贵族被你的赞助人信赖并指派为联系人，尽管已被流放，他还是把家族的利益放在首位。 |
| 6 | 入局的外人 Outsider Inside。 一个神秘的存在掌控着一个贵族家族的命运。他通过你的赞助人雇佣你，利用你操控这个被选中的家族数世纪的命运轨迹。 |
