# 远古存在

远古存在 Ancient Being

> 每当一个深不可测的强大存在突然出现在你们面前并用天神般的报酬换取一些小恩小惠时，那从来都是骗人的。但我除外，我永远不会欺骗你，我最亲爱的读者。
>
> ——塔莎

**远古存在类型 Types of Ancient Beings**

远古存在类型 Academy Type

| d6 | 远古存在类型 |
|---|---|
| 1 | 古龙 Elder Dragon。一条远古巨龙寻求着知识或力量。它希望获得更伟大的财富来充盈其宝山，其野心随着其年龄一同增长。 |
| 2 | 巫妖 Lich。一个拥有强大力量的不死施法者雇佣了你的团队。它的利益所在多种多样，并且看起来正当。也许他不像传统冒险智慧所述的一般邪恶？ |
| 3 | 受缚邪魔 Bound Fiend。无论是以其真实形态还是以附身的灵魂形态，此邪魔被束缚于某个地点。无论是受困于不可破坏的束缚法阵印记里，还是灵魂被封印在一处巨大雕像中，邪魔的影响力都在驱动你的团队。 |
| 4 | 天界守护者 Guardian Celestial。一位天使或其他强大的天族对物质位面的特定区域起了兴趣。它培养了一张凡人情报网和许多代理人们为其事业服务。 |
| 5 | 不朽者 The Endless。这个个体经历了许多次生命，因为他们不会死——至少不会永远死去。无论什么杀死了他们，他们总是回归。表面上看，他不过是活生生的凡人，但他们掌握着一个不朽者才能积累的资源。 |
| 6 | 原初显现 Primal Manifestation。仅仅只是简单地存在着，它们的存在就违背了凡人的理解。它们可能是一种被唤醒了自我意识的自然界的原初之力，栖息于山水之中；或是通过代言、预兆或偶像低语着的异域智能。 |

远古存在联络人 Ancient Being Contact

远古存在联络人 Ancient Being Contact

| d6 | 联络人 |
|---|---|
| 1 | 雇主 Employer。一位当地社会的成员成为了你与赞助者之间的接头人，并且可以为你们提供合法的工作作为掩护。他们也许是酒保、店家、当地官员或贵族。 |
| 2 | 后台交易者 Back-Room Dealers。一处位于公共场合，但需要口令或者令牌才能进入的秘密区域。你们能在那里和赞助者的神秘代理人联系。 |
| 3 | 投放魔法信息 Magical Message Drop。来自你们联络人或赞助者的魔法信息记录出现在偏僻的地方。你们知道该检查哪处预先确定的位置，如古代巨岩或特定坟墓的缝隙之中，来获得指示。 |
| 4 | 幻象 Visions。你们的赞助者不使用中间人，而是通过梦境、预兆或幻觉来与你们对话。当你们入眠的时候，该存在就会出现在你们的脑海里，控制梦境，传递难以忽视的指令。 |
| 5 | 霎时回响 Ephemeral Echo。你们的赞助者从不以物理形态展现在你们面前。也许他是一位死者的幽魂，一个位于时间流之外的实体，又或者是一个永远不会离开秘密圣所的存在投射的幻象。 |
| 6 | 代言人 The Mouthpiece。。这位远古存在的声音通过另一个人的嘴唇低语而出。你们的赞助者也许会借助一位陌生人的身体或者团队的一员来和你们交流。 |

**远古存在任务 Ancient Being Quests**

| d6 | 任务 |
|---|---|
| 1 | 救援 Rescue。一个自作主张的代理人在收集信息或材料时失踪了。你们必须查出他们下落，并找回他们以及他们的发现。 |
| 2 | 破坏 Sabotage。你们必须破坏敌对组织的一部分，要么暗杀一个关键目标，要么摧毁一个重要物件。 |
| 3 | 巧计 Artifice。你们的特殊技能有助于为一场强大魔法仪式或一件物件收集组件。 |
| 4 | 策反 Treachery。另一位强大人物的高调手下有可能为了地位背叛到你的赞助者这边。你们必须说服他们投奔到你们的组织，或者干脆把他们从现在的敌对区域中带出来。 |
| 5 | 清理 Culling。你们赞助者的一位受尊敬的代理人（可能是一个盟友，或者你们团队的导师）已经妥协了。也许他们正在准备投靠一位对手，或试图夺取远古存在的力量。不管原因为何，你们必须抓住他们并结束其威胁。 |
| 6 | 星劫 Astral Heist。你们赞助者的一位强大对手将秘密储存到了一座位于星界的精神保险库中。这代表他们不能因被欺骗或强迫透露任何信息，也不会被读取思想。你们必须找到保险库，穿越对手的致命记忆，找到你们赞助者所要的知识。 |
