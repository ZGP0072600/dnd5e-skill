# 定制血统

**边栏：定制血统 Custom Lineage**

1级时，你可以选择不使用游戏中已有的种族，而是使用以下特性定制角色的血统，让你可以完全自定角色的出身。

**生物类型 Creature Type。** 类人生物。由你自己决定的外貌，以及你是否与同族长相类似。

**体型 Size。** 从小型或中型中选择一个。

**速度 Speed。** 你的基础步行速度为30尺。

**专长 Feat。** 你获得一个你自选的满足条件的专长。

**属性值提升 Ability Score Increase。** 选择一个属性+2。

**可选特性 Variable Trait。** 从以下两种特性中选择一个：

> （a）60尺黑暗视觉
>
>  或
>
> （b）自选一项技能的熟练项。

**语言 Language。** 你可以说、读、写通用语和另一门你和你的 DM 认为合适的语言。

你的种族现在在任何要求特定种族——例如精灵或矮人——的游戏特性中，被视为一名“定制血统”的角色。
