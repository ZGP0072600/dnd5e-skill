# 角色选项

**角色选项 Character Options**

当你制作你的 D&D 角色时，你在《玩家手册》中会有一系列的选项可供你打造你希望的那种角色。这一章添加了这些选项，使角色概念具体化成为可能。如果你将这些选项与在《珊娜萨的万事指南》中的选项混合使用，你创造角色将会获得巨大的可能性。

这是本章的所记述的可选项：

- 一种依靠修改种族特质来自定义你角色出身的方式。
- 关于更换你的子职业的事宜
- 奇械师，一个魔法造物的专家
- 每个《玩家手册》中的职业都有的新职业特性与子职业
- 为使用专长可选规则的人使用的新专长
