# 战地匠师

**战地匠师 Battle Smith**

战地匠师在卡尼斯家族制造战斗构装生物和最初的战俑时扮演了关键的角色。在终末战争过后，这些奇械师致力于救助在恐怖的战争中受伤的人。许多运用着近似于创造生命力量的战地匠师将他们的聪明才智用于摸索如何制造技术上无比卓越的小狗小猫。看来我小看了他们。——塔莎

工具精通 Tool Proficiency 第3级战地匠师特性

**战斗准备 Battle Ready**

*第3级战地匠师特性*

你的战斗训练和魔法实验为你带来了两种好处：

· 你获得军用武器的熟练。· 当你以魔法武器攻击时，你可以在攻击检定与伤害掷骰中使用智力调整值，以代替力量或敏捷。钢铁守卫 
Steel Defender 第3级战地匠师特性你的工匠技艺为你诞生出了一名可靠的同伴，一个钢铁守卫。它对你和你的同伴友好，并且服从你的命令。钢铁守卫的游戏数据见下，它的数据中某几项会适用你的熟练加值。该生物的外观，两足步行还是四足步行皆由你决定；你的决定对它的游戏数据没有影响。

在战斗中，钢铁守卫使用你的先攻，并且在你的回合后立即行动。它能自由移动和执行反应，但除非你以一个附赠动作命令它进行动作，否则它在回合中只能进行闪避动作。他可以执行下方写出的动作选项，或是其他的动作选项。如果你陷入了失能，钢铁守卫可以依靠它自己的选择进行任何动作，而不仅仅是闪避。

如果一个 修复术 Mending 
被施展到了钢铁守卫的身上，它将恢复2d6生命值。如果它在最近一小时内死亡，你能够以一个动作使用铁匠工具花费一个至少1环的法术位来在距离你5尺的空格内复活它。钢铁守卫会在1分钟后复活，在复活时会回满生命值。

当你在手头有铁匠工具的情况下完成一次长休时，你可以创造一个新的钢铁守卫。如果你已经通过此特性拥有一个钢铁守卫，则上一个会立即消失。如果你死亡，钢铁守卫也会随之损毁。

**额外攻击 Extra Attack**

*第5级战地匠师特性*

你在自己回合内执行攻击动作时可以发动两次攻击而非一次。

**奥能震荡 Arcane Jolt**

*第9级战地匠师特性*你学会了新的方法来引导奥术能量进行伤害或治疗。当你使用魔法武器攻击或钢铁守卫命中目标时，你可以将奥术能量引导至攻击中，以获得以下一种效应：

- 目标受到额外的2d6点力场伤害。
- 选择距离目标30尺以内的一个可见生物或物体。治愈能量涌进选择的对象，为其恢复2d6生命值。

你可以引导奥术能量的次数等同于你的智力调整值（至少1次），但每回合只能这么做一次。在你完成长休时，恢复所有已被消耗的次数。改良守卫 
Improved Defender 第15级战地匠师特性 到达15级后，你的奥能震荡和钢铁守卫变得更强了：

- 奥能震荡特性的额外伤害和生命恢复变为4d6。
- 你的钢铁守卫 AC 获得+2加值。
- 当你的钢铁守卫使用偏转攻击时，攻击者受到1d4+你的智力调整值的力场伤害。
