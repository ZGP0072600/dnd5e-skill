# 炼金师

**炼金师 Alchemist**

炼金师与女巫的魔法都是源自一个适用于整个多元宇宙的真理凡人无法抗拒带泡泡的东西。——塔莎

工具精通 Tool 
Proficiency

第3级炼金师特性

你获得炼金工具的熟练。如果你已经拥有此熟练，作为代替，你能获得另一种工匠工具的熟练。

**炼金师法术 Alchemist Spells**

*第3级炼金师特性*

只要你达到相应的职业等级，炼金师法术表格中的法术总是被视为已准备。这些法术对你来说是奇械师法术，且不会计入你可以准备法术的总数。

炼金师法术 Alchemist Spells

| 奇械师等级 | 法术 |
|---|---|
| 3rd | 治愈真言 healing word，致病射线 ray of sicknbess |
| 5th | 炽焰法球 flaming sphere，马友夫强酸箭 Melf's acid arrow |
| 9th | 气化形体 gaseous form，群体治愈真言 mass healing word |
| 13th | 枯萎术 blight，防死结界 deathword |
| 17th | 死云术 cloudkill，死者复活 raise dead |

**实验性灵药 Experimental Elixir**

*第3级炼金师特性*

每当你完成一次长休，你能在一个你接触的空的 扁瓶 flask 
中魔法性的制造出一瓶实验性灵药通过实验性灵药表来掷骰决定这瓶灵药在被某人饮用时的效果。

生物能以一个动作饮用这瓶灵药、或是将灵药灌给一个失能生物饮用。

你能通过消耗一枚1环或更高环的法术位来制造额外的实验性灵药。

此时，你以一个动作接触一个空的 
扁瓶 flask ，在其中充入灵药，然后从实验性灵药表中选择这瓶灵药的效果。

你必须要拥有炼金工具才能制造实验性灵药，你通过此特性创造的任何灵药会一直存在，直到被饮用、或是你的下一次长休结束时。

当你在本职业达到某些特定等级时，你将能够在长休结束时制造更多灵药：6级时2瓶，15级时3瓶。

每一瓶灵药都需要掷骰决定效果，且都需要用独立的 
扁瓶 flask 盛放。

实验性灵药

| d6 | 效果 |
|---|---|
| 1 | 治疗。饮用者恢复等同于 2d4+你智力调整值 的生命值。 |
| 2 | 疾速。饮用者的速度增加10尺，持续1小时。 |
| 3 | 韧性。饮用者的 AC 获得+1加值，持续10分钟。 |
| 4 | 气魄。饮用者在接下来的一分钟内，能够在每个攻击检定与豁免检定中投掷一枚 d4并将结果加入到检定中。 |
| 5 | 飞行。饮用者获得10尺飞行速度，持续10分钟。 |
| 6 | 变形。饮用者的身体产生如同受到法术变身术 Alter Self 一样的变化。法术的变化具体怎样由饮用者决定，该效果会持续10分钟。 |

**炼金术掌握 Alchemical Savant**

*第5级炼金师特性*

你对魔法化学物质的掌握变得非常熟练，这使得你通过炼金物质造成的治疗和伤害更加有效。

当你使用炼金工具作为法器施展任何法术时，你在法术的一次掷骰中获得等同于你的智力调整值（最低+1）的加值。该掷骰必须是恢复的生命值量，或是造成强酸、火焰、暗蚀、或毒素伤害。

**复原药剂 Restorative Reagents**

*第9级炼金师特性*

你可以在你的一些作品中加入复原药剂：

- 每当一个生物饮用一瓶你制造的实验性灵药时，生物都将获得 2d6+你智力调整值 的临时生命值（至少1点临时生命点）。
- 只要你使用炼金工具作为你的法器，你就可以在不花费法术位、无需准备的情况下施展 *次等复原术 Lesser Restoration* 。你能够这么做的次数等于你的智力调整值（最少1次），当你完成一次长休后，你恢复所有已被消耗的次数。

**化学专家 Chemical Mastery**

*第15级炼金师特性*

你已经接触了太多的化学物质，并解开了它们的秘密，它们对你的危害很小，你可以使用它们来迅速治愈某些疾病：

- 你获得了对强酸伤害和毒素伤害的抗性，并且你免疫于中毒状态。
- 只要你使用炼金工具作为你的法器，就能够在不消费法术位、不需要准备、不需要材料成分的情况下施展 *高级复原术 Greater Restoration* 与 *医疗术 heal*。一旦你通过此特性施展这些法术中的一个，你就不能再以这种方式施展那个法术，直到你完成一次长休。
