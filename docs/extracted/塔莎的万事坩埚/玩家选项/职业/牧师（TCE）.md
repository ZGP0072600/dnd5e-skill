# 牧师（TCE）

牧师 Cleric 牧师职业在本节中获得了新的职业特性与子职业选项。

可选职业特性 Optional Class Features
 
你可以在达到特定等级时获得玩家手册中描述的职业特性。这一部分提供了一些你作为牧师的额外特性。与玩家手册中的特性不同的是，你不会自动获得这些特性。在你达到该部分描述的职业特性的需求等级且决定获得这些特性时，与你的
DM 先商量商量。这些特性能够被一个一个单独选走，这意味着你可以使用其中一些、全部或不使用它们。 
若你将一个特性替换为另一个特性，那么你将不会受到被替换的特性所带来的好处，你也不会满足在游戏中任何需求被替换的特性的东西。

*第8级牧师特性，替代 神圣打击 Divine Strike 特性或 强力施法 Potent Spellcasting 特性*

 
在战斗中，你受到神力的祝福。当你通过戏法或武器攻击来伤害一个生物时，你能够对该生物造成额外1d8光耀伤害。当你造成一次此伤害后，你直到自己的下一个回合开始前无法再次使用此特性。
