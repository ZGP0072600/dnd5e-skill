# 术士（TCE）

术士 
Sorcerer 术士职业在本节中获得了新的特性与子职业选项。

**可选职业特性 Optional Class Features** 

 你可以在达到特定等级时获得玩家手册中描述的职业特性。这一部分提供了一些你作为术士的额外特性。与玩家手册中的特性不同的是，你不会自动获得这些特性。在你达到该部分描述的职业特性的需求等级且决定获得这些特性时，与你的 DM 先商量商量。这些特性能够被一个一个单独选走，这意味着你可以使用其中一些、全部或不使用它们。

超魔法选项 Metamagic Options 第3级术士特性 
当你选择超魔法选项时，你能够选择以下超魔法选项。

> 追踪法术 Seeking Spell
> 当你为法术进行一次攻击检定但未命中时，你可以使用2术法点重投那枚 d20，你必须使用重投后的结果。追踪法术超魔可以与其他超魔法选项合用。
>
> 转化法术 Transmuted Spell 当你施展一道造成以下列表中某一种类型伤害的法术时，你可以使使用1术法点将法术的该种伤害类型转换为列表中的另一种伤害类型：强酸、寒冷、火焰、闪电、毒素、雷鸣。

Versatility
第4级术士特性 
每当你通过提升术士等级获得属性值提升时，你可以进行以下一种行为，代表着你内在的法力流向了新的途径： 用另一个你能够选择的超魔法选项替换你当前的一个超魔法选项。 用另一个你能够从术士法术列表中选择的戏法替换你当前已知的一个术士戏法。魔法指引 Magical Guidance 第5级术士特性

你能够挖掘内在的魔力源泉，从失败中召唤成功。当你进行一次属性检定并失败时，你可以使用1术法点重投检定的 d20。你必须使用重投后的结果。此举可能导致该次检定转为成功。
