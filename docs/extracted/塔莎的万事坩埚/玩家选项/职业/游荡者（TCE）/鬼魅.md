# 鬼魅 Phantom

游荡者的生活常常是日复一日的游走在生死边缘，一边冒着生命危险，另一边取走别人的性命。在生死之间冒险的时候，一些游荡者与死亡本身结下了奇妙的不解之缘。他们浸淫于与负能量之中，从死者那里获取知识，直到他们自身都变得如同幽灵一般。盗贼工会对鬼魅们高度重视，因为他们是极其高效的情报员和间谍。

堕影冥界的诸多影灵专精于这些骇人的技艺，有些甚至愿意传授。在塞尔 Thay（被遗忘的国度）与坎纳斯 Karrnath（艾伯伦）有不少潜心修炼的死灵法师，一名鬼魅可能成为一位法师的密友抑或是左右手。在死神的神殿之中，鬼魅则会是追踪逃避死亡之人并获取墓穴中失落的知识的特工。

你是如何发现的这一黑暗力量？你是否曾长眠于墓场之中并在醒来时意识到了自己的特异能力？抑或是你在侍奉于一名死神的神殿或盗贼工会中获取了这份力量？

**亡者余声 Whispers of the Dead**

*第3级鬼魅特性*

死者的回响依附着你，一个幽灵般的存在会向你分享它的知识。每当你完成一次短休或长休时，你可以选择一项你缺乏熟练的技能或工具，并获得它的熟练项。当你用这个能力来获得一项不同的熟练时，你会失去原来选择的那项熟练项。

**墓穴惊嚎 Wails from the Grave**

*第3级鬼魅特性*

当你将某人推向墓穴时，你也可以沟通死亡之力，使得亡者的嚎哭短暂环绕着新的目标。当你在你的回合对于一个生物造成了偷袭攻击的伤害之后，你可以立即使该生物30尺内第二个可见生物受到一次暗蚀伤害。掷你一半数量的偷袭骰（向上取整），你对第二目标造成等同于投掷结果之和数值的暗蚀伤害。

你可以使用此特性的次数等同于你的熟练加值次，当你完成一次长休后你重获全部已消耗的使用次数。

（译者注：3级1d6，5级2d6，9级3d6，13级4d6，17级5d6）

**逝者遗物 Tokens of the Departed**

*第9级鬼魅特性*当生命终结于你的眼前，你可从离去的灵魂中抓取出一小片其生命精华的具现化象征：当你30尺内一个可见生物死亡时，你可以以一个反应张开一只空闲的手，使得一个微型的饰品出现在你的掌中，那是一个魂器。DM 决定魂器的外观，或者让你骰《玩家手册》中的饰品表来决定它的外观。

你最多可以同时持有你的熟练加值个的魂器，当你所持魂器数目达到上限时你不再能制造新的魂器。

你能以下列方式使用魂器：

- 当你随身携带着魂器时，由于你的生命力受到了魂器内蕴藏的生命精华的强化，你在死亡豁免和体质豁免上具有优势。
- 当你在你的回合里造成偷袭攻击伤害时，你可以销毁你的一个魂器，在不消耗能力使用次数的情况下，立刻使用墓穴惊嚎特性。
- 以一个动作，你可以销毁一个你的魂器（无论它在哪里）。当你如此做时，你可以向与魂器相关的灵魂一个问题。灵魂将出现在你面前并以它生前所知的一种语言回答。它并无义务对你诚实，并由于其对自由的渴望将回答地尽可能的简略。这个灵魂应当只会知道他生前所知的事实，不过一切以你的 DM 为准。

**幽灵漫步 Ghost walk** *第13级鬼魅特性*

你可以部分的进入死亡的领域，变得如同一个幽灵。作为一个附赠动作，你化为一种灵体的形态。处于这种形态时，你获得10尺的飞行速度、能悬浮、且以你为目标的攻击检定都将具有劣势。你也可以如同是困难地形那样穿过生物或者物品，但是如果回合结束时你仍然在一个生物或物品的位置上，你将受到1d10点力场伤害。

该形态持续10分钟。你也可以用一个附赠动作将其提前解除。若你想要再次使用特性，要么你得完成一次长休，要么你得销毁一个你的魂器（作为激活幽灵漫步的附赠动作的一部分）。

**死亡之友 Death's Friend**

*第17级鬼魅特性*你与死亡的联系变得如此紧密，以至于你获得了如下增益：

- 当你使用墓穴惊嚎的时候，你可以同时对你的攻击目标和第二目标造成那份暗蚀伤害，
- 在一次长休的结束时，若你没有任何魂器，一些死者的灵魂会自动被你吸引，在你手中形成一个魂器。
