# 巨灵 The Genie

我知道在灰鹰找房子很难，但如果巨灵或者魔契师要便宜租房子给你，快跑。

——塔莎

你与巨灵中的稀有群体，巨灵贵族缔结了契约。巨灵贵族乃是广阔的元素位面的统治者，并对其他次等巨灵与次等元素生物具有莫大的影响力。巨灵贵族们的动机各不相同，但都傲慢自大且有着匹敌次等神明的强大伟力。他们喜好让热衷于缚誓巨灵的凡人们反过来服侍他们，并对于缔结能扩张他们在多元宇宙中势力的契约有着巨大的热情。

你可以选择你宗主是哪种巨灵，也可以通过巨灵种类表让骰子决定。

**巨灵器皿 Genie's Vessel**

*第1级巨灵宗主特性*

你的宗主赐予你一个包含巨灵之力的魔法器皿。该器皿是一个微型物件，你可以将其作为施展你魔契师法术的法器。你可以自行决定物件的类别，或者你也可以通过巨灵器皿表来决定它。

在你触碰你器皿的情况下，你可以用以下的方式使用它：

> **瓶中小憩 Bottled Respit。** 以一个动作，你可以用魔法消失并进到你的器皿内，器皿将留在你所在的原地。器皿内是与其本身相似的20尺半径，20尺高的圆柱形异次元空间。器皿内部温度适宜，并安放有舒适的垫子和茶几。在器皿之中时，你可以如同身在器皿所在之处一般听到外界的声音。你在器皿内可停留的时间等同于你熟练加值双倍数目的小时。如果你死亡，或是器皿被摧毁，抑或是使用一个附赠动作离开，你将提前从器皿中出来。当你离开器皿时，你出现在一处距离它最近的未被占据的空间中。在器皿内的一切物件将被留在其中直到被取出，而倘若器皿被摧毁，其中的物品则会完好无损地出现在距离它最近的不被占据的空间中。一旦你进入了器皿，直到你完成一次长休前你都不能再度进入它。
>
> **巨灵之怒 Genie's Wrath。** 在每个你的回合中，你可以在你攻击检定命中时，对攻击目标造成等同于你熟练加值的额外伤害。额外伤害的类型取决于你的宗主类型：钝击（土巨灵）、雷鸣（气巨灵）、火焰（火巨灵）、寒冷（水巨灵）。这一额外伤害每回合只能产生一次。

器皿具有与你的法术豁免 DC 相同数值的 AC，它的生命值等于你的魔契师等级加熟练加值，且免疫毒素和心灵伤害。

倘若器皿被摧毁或是丢失，你可以进行一场持续1小时的仪式来从你的宗主处获得一个替代品。这一仪式可于短休或长休时进行，并会摧毁之前仍存在的器皿。当你死亡时，器皿将在一阵元素能量的闪耀中消失。

**元素赐福 Elemental Gift**

*第6级巨灵宗主特性*

你逐渐显露出你的巨灵宗主的特质。你对一种特定的伤害类型具有抗性，具体由你的宗主类型决定：钝击（土巨灵）、雷鸣（气巨灵）、火焰（火巨灵）、寒冷（水巨灵）。

 
此外，以一个附赠动作，你可以获得30尺飞行速度，并且可以悬浮，持续10分钟。你可以使用此附赠动作的次数等同于你的熟练加值次，在完成一次长休后你重获全部已消耗的使用次数。

**器皿庇护所 Sanctuary Vessel**

*第10级巨灵宗主特性*

当你使用 瓶中小憩 Bottled Respite 
特性进入你的巨灵器皿时，你可以选择30尺内最多5个可见自愿生物与你一同进入器皿。

以一个附赠动作，你可以将任意数目的生物从你的器皿中逐出。如果你离开器皿、或是器皿被摧毁，所有的生物都将被一并逐出。

同时，在器皿中停留至少10分钟的任何生物（包括你自己）可视为完成了一次短休，且在短休时使用生命骰决定生命值恢复时，其恢复量中可以加入你的熟练加值。

**有限祈愿术 Limi****ted Wish**

*第14级巨灵宗主特性*

你祈求你的宗主，并实现你的小小愿望。以一个动作，你可以向你的巨灵器皿说出你的愿望，并请求实现一个6环或更低环阶的，且施法时间为1动作的法术效应。这一法术可以来自任何职业的法术列表，且你不需要满足法术的任何需求，包括那些有价值的材料成分；该法术仅仅是作为此动作的一部分产生自己的效应。

你使用此特性后，直到你完成1d4次长休前，你都不能再次使用它。
