# 这都是可选的

**这都是可选的 It's All Optional**

本书内提及的任何东西都是可选的。任何团队都应该由 DM 来决定本次战役中是否使用、使用哪些选项。你可以使用一部分、全部、或完全不使用本书的内容。我们鼓励您去选择哪些最适合战役故事与小组游戏风格的内容。

无论使用哪种哪些选项，本书的内容均基于《玩家手册》、《怪物图鉴》与《城主指南》中的规则，且它可以与《珊娜萨的万事指南》或其他 D&D 书籍中的选项搭配使用。
