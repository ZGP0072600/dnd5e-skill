# 使用本书

**使用本书 Using This Book**

《塔莎的万事坩埚》提供了一系列的 Dungeons & Dragons 
新选项，我们的旅途将跟随法师塔莎的笔记来探索这些选项。塔莎，即著名法术 *塔莎狂笑术 Tasha's hideous laughter* 的作者，她的一生在 D&D 多元宇宙中被记载为了无数的故事。她被 芭芭雅嘎 Baba Yaga，巫师之母 
本人亲手养大，并在 灰鹰 Greyhawk 世界各地与其他著名冒险者们结交朋友...外加有时结交死敌——比如魔邓肯。时过境迁，她成为了巫后，自更名为 
伊格薇尔伏 Iggwilv ——一个人们耳熟能详、敬畏和崇拜的传奇人物。

本书专为玩家与地下城主所写，提供了在任何 D&D 世界中可用于角色与战役上的选项，不论你是在灰鹰、在其他官方世设、还是在自创世界观下都可以使用的选项。
