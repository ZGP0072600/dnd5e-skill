# 检视怪物

**检视怪物 Monster Research**

冒险者可以检视怪物，来确定怪物更喜欢或偏好什么事物。怪物检视表提供了某些生物类型该使用哪些技能来了解其偏好的建议。使用相关技能进行属性检定的 DC 为 
10 + 目标生物的挑战等级。
