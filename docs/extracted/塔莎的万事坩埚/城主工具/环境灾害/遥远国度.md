# 遥远国度

**遥远国度 Far Realm**

当灵魂在死后离开物质位面时，他们会作为精魂停留在星界或前往外层位面之一继续自己的旅行。但一些存在找到了前往外层位面之外的方式，并定居于遥远国度之中。在那里，他们经历亘古的岁月化作可憎异形和上古邪物，于自身法则构成的现实中蠢蠢欲动。所有在遥远国度停留之人最终都会被那里的魔力所扭曲，化作异界的形状。遥远国度的有害影响通常细微难辨，它们通过现实的薄弱之处或导致生命沿着异界的路径发展的侵入性思想渗入物质位面。当为遥远国度所接触的区域中发生以下情况时，可以考虑根据遥远国度效应表掷骰决定发生的效应。·一位旧日支配者异界宗主的魔契师在一次属性检定、攻击检定或豁免检定中使用 d20投出1或20。·角色们在区域中进行一次短休或长休之后。·一个生物花费一个小时以上阅读一本由亲眼见过或以其他方式接触过遥远国度之人所写的可怖卷册。遥远国度效应 Far Realm

| 86-90 | 这一区域内，圆形的东西（比如纽扣，水晶球，太阳等）看上去离奇失常。一个在区域内开始回合的随机生物必须成功通过 DC14智力豁免，否则会在其回合中大声地试图摧毁这些事物。 |
| 90-95 | 圆睁的泪眼出现在整个区域中的无生命物体上。这些眼睛会监视角色们。只要眼睛存在，区域内的生物就不会被角色所突袭。如果受到伤害，一只眼睛会闭上并消失。如果你已经投出过这一效应，则重新掷骰。 |
