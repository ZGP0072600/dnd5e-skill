# 心灵共振

**心灵共振 Psychic Resonance**

边栏：启智万物 Awaken 
Anything 传说中有不少关于人格化的自然力量或异变的物件引导或对抗英雄们的故事：歌唱的雕像，低语着秘密的动物，说出他们想法的云彩。尽管如元素、格莱石人和树人这样的怪物能够用来代表这些具有生命的自然伟力，但并非每一块会说话的石头或智慧的溪流都需要一张资料卡。也许你的冒险会将队伍带到一个植物们能自由交谈的国度，或是一片被诅咒的恶土，那些被石化的可怜灵魂在祈求帮助。如果你想给故事注入额外的活力，不要被《怪物图鉴》中的选项或活化物件 animate 
object 法术限制所限制。让女巫的城磐中堆满喃喃自语的遗物，妖精的森林中长满会窃听的植物，天空之神的庙宇中则有鸟儿组成的唱诗班。诸如活化物件 animate 
object、启蒙术 awaken、动物交谈 speak with animals、植物交谈 speak with plants 这样的法术都为如何让普通的生物和物件更具活力提供了指导。
