# 镜像区

**镜像区 Mirror Zone**

镜像区出现在位面和魔法力量交汇之处，会创造出一片反射区域区域。生物，物件和能量在这里会产生反射、折射、复制或传送到其他地方。这样的区域诞生的原因通常是理论存在的镜像位面入侵了物质位面，或者涉及变化、防护或预言的魔法产生了意想不到的效应。当区域中出现以下情况时，可以按照镜像区效应表格掷骰决定出现的效应。·一个生物打碎了一面镜子。·一个生物使用了传送魔法。·一个幻象出现。·一个生物冒充另一个生物。镜像区效应 Mirror Zone 
Effect

| d100 | 效应 |
|---|---|
| 01-06 | 区域中的生物开始表现出并非自己的特点。在接下来24小时中，被影响的生物在伪装自身的时候进行的魅力（欺瞒）检定具有优势。 |
| 07-12 | 海市蜃楼 hallucinatory terrain 法术（豁免 DC15）影响了区域中的自然地貌，将它们变成了不同的地形（由 DM 选择）。 |
| 13-18 | 一个区域中的随机生物在一分钟内获得闪现术 blink 的法术增益，其身上闪烁着层层叠叠的破碎反光。 |
| 19-24 | 区域内的生物不会形成倒影。对这些生物进行的感知（洞悉）检定具有劣势，并且该生物对注意到这一点的对象进行的魅力（游说）检定具有劣势。当他们离开该区域时，他们会重新获得自己的倒影，该效应将结束。 |
| 25-34 | 区域中1d4个生物的倒影走出镜子并发起攻击。倒影是二维的，其形象和映出他们的生物无二。使用幽影 shadow 作为其数据，但倒影是妖精而非亡灵，它们对钝击伤害而非光耀伤害易伤。 |
| 35-40 | 一位区域内的角色获得镜影术 mirrow image 的法术效应。创造出的镜影有时可能会自行动作或发表自己的看法。 |
| 41-46 | 在接下来24小时内，区域内的伤口会引来玻璃的碎片，造成额外的伤害。任何非不死和构装的生物在被一次造成穿刺或挥砍伤害的攻击命中之后都会血流不止，每回合损失1d4生命值。如果流血的生物被另一次这样的攻击命中，则流血失去的生命值增加1d4。任何生物都能以一个动作通过 DC10感知（医药）检定来包扎伤口。流血效应也会在生物受到魔法治疗后停止。 |
| 47-52 | 区域内的镜子和其他反光面能作为魔法通道使用。一个触碰未佩戴的物品中自己影像的生物能够施展迷踪步 misty step 法术，而无须任何法术成分。 |
| 53-58 | 一位角色在接下来24小时内能够施展一次探知术 scrying，他能够使用镜子或其他光滑的表面而无须其他任何法术成分。 |
| 59-64 | 在接下来24小时内，一个随机生物的皮肤变为银色并闪闪发光。在此期间，该生物对法术进行的豁免检定具有优势，对该生物进行的法术攻击检定具有劣势。 |
| 65-70 | 一把剑刃由镜子碎片做成的长剑或短剑出现在一个随机生物的60尺内。该武器是一把血光剑 sword of wounding（见《城主指南》）。如果使用该武器在一次攻击中掷出了1或20，则这把武器会碎裂并在本次攻击之后破坏。 |
| 71-76 | 在接下来24小时内，当区域内的任何人以一次攻击动作命中一个生物并投掷伤害时，攻击者必须成功通过 DC13的魅力豁免否则受到自己造成伤害的一半伤害。 |
| 77-82 | 两个反光的直立圆盘出现在区域内未被占据的空间并持续存在一分钟。每个圆盘直径6尺，浮在地面以上1尺高度。其中一个出现在队伍30尺范围内未被占据空间内。任何通过一个圆盘的生物会立即出现另一个圆盘旁未被占据的5尺内。 |
| 83-88 | 下一次队伍中的某个角色照镜子时，他们的倒影会获得生命并开口说话。如同施展预言术 divination 倒影会提供一个问题的答案 |
| 89-94 | 悬浮的镜子碎片在区域中旋转，映照出并不实际存在的生物和地点的倒影。在先攻轮到20时（同先攻值最后），碎片会对区域中一个随机生物进行一次远程武器攻击（命中+6）。命中后，该目标受到10（3d6）挥砍伤害。 |
| 95-00 | 一个随机生物的镜像出现在其30尺内未被占据的空间内。镜像的外表，游戏数据和装备都与该生物相同，并希望杀死自己的原型。如果镜像死去，它和它所有的装备都会破碎成镜子碎片。如果镜像在1小时内未能杀死该生物，它便会消失。 |
