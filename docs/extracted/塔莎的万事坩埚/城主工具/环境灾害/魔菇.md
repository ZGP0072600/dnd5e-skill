# 魔菇

**魔菇 Magic Mushrooms**

蘑菇可以是致命的，是美味的，或两者都是。有些蘑菇带有魔法效果，特别是那些在充满神秘能量的地点，比如幽暗地域或妖精荒野生长的蘑菇。拥有医药、自然或生存熟练项的生物可能熟悉真菌这一学问，特别是有关魔菇的学问，因为其益处可以救命或赋予人们奇异的能力。不过，只有专家才能在遇见未知种类的真菌时检定它并确定它的效益。为了判断食用这些蘑菇会产生的效果，在神奇蘑菇效果表上骰点。
