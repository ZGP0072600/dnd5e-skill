# 闹鬼

**闹鬼 Hauted**

闹鬼的环境包括背负黑暗过往的房屋、大屠杀的旧址，以及有人曾因为经历强烈的恐惧、背上和怨恨而死去的地点。闹鬼之处有着像幽灵一样过去的回音，不断侵扰着访客，不让他们在此地经年的创伤中有得片刻喘息。很少有地方会无缘无故闹鬼，而你可以轻易对表格中的结果进行定制，以与可怕的故事相衬。当区域中发生以下情况时，可以考虑根据闹鬼效应表掷骰决定发生的效应。·一个生物获得恐慌状态时。·复数生物无法视物时。·一个生物落单时。·午夜时分或另外某个不详的时点到来时。·一个幽灵或其他生物与此处的可怕历史相关的生物对队伍构成威胁时。闹鬼效应 Hauted

|---|---|
| 01-05 | 一场猛烈的风暴开始以该区域为中心形成，直到队伍离开该区域前，它都不会结束。 |
| 06-10 | 一栋区域内的随机建筑在接下来持续24小时内获得铜墙铁壁 guards and wards 的法术效应（豁免 DC13）。 |
| 11-15 | 一位随机角色周围物品中的寻常几件——也许是树干或动物头颅活化持续1分钟并低语着警告或威胁要泄露这位角色的秘密。 |

| 21-25 | 在接下来1d6小时内，区域内的气温每小时降低10华氏度。温度将在之后回到正常。如果足够寒冷，冰晶将会结成不祥的图案。 |
| 26-30 | 一个随机生物的影子在24小时内开始自主活动。影子的动作与主人不同步，可能会戏剧性地作窒息哽咽状或试图谋杀另一个影子。 |
| 31-35 | 在下一次黄昏之后，太阳在36小时内不会再度升起。在此期间，区域上空会高悬一轮猩红的月亮，被翻滚的雾气所覆盖，或是闪烁着异界的星辰。 |
| 36-40 | 在下一个夜晚中，一个随机睡着的生物会消失并出现在睡下之处的地下1尺——通常是被毫无知觉地埋在泥土或者地板下层中。该生物能够通过成功的 DC13力量（运动）检定解救自己。 |
| 41-45 | 区域内一个随机的生物被浮空术 levitate（豁免 DC15）影响1分钟。 |
| 46-50 | 一个不暴力但令人不安的幽魂 ghost——也许是一只宠物，一个容易出事的孩子，或一只被肢解的拇指出现在一位随机角色身边并在24小时后消失前紧随他们。幽魂会在生命值降低到0时消失。 |
| 51-55 | 一位玩家角色的外貌在24小时中变得能反映出这一区域的闹鬼历史。比如说，他们可能会会生出一条与死于本地的著名暴君有关的疤痕。 |
| 56-60 | 在接下来24小时中，一切在这一区域被杀死的类人生物肉体会立即腐烂分解并在死去10分钟后复生为一具骷髅 skeleton。 |
| 61-65 | 在接下来24小时中，当任何生物受到伤害时，它的血液（或者类似的体液）会流淌成一则短讯或一幅可怕的图案。 |
| 66-70 | 一个灵体寄宿于一位角色的一把简易或军用武器之中，直到角色离开该区域前，该武器会被化作一把智能魔法物品。通过《城主指南》“智能魔法物品”一节随机生成决定该物品的属性。 |
| 71-75 | 一股神秘力量对一位角色显现，该角色能问出一个问题，并如同卜筮术 augury 般收到回答。该灵力可能表现为自主动作的说话板，起雾玻璃上书写的文字，或者成群昆虫形成的讯息。 |
| 76-80 | 在下一个夜晚间，一位在区域中睡着的角色会如同被施展了托梦术 dream 一般收到一则幻象。梦境短暂且令人不安，它展示出一些历史环境中的要素，并将角色置身于横遭惨剧之人的视角。 |
| 81-85 | 一具棺材或其他区域内密闭的狭小空间——可能是古董箱子，石头洞窟或者用石头堵住的树桩中散发出显而易见的恶意。当一个生物第一次打开它时，进行一次掷骰。若结果为偶数，则该生物会收到一个可怕的幻象，并在接下来24小时内陷入对一切生物的恐慌状态。如果结果为奇数，则一个如同万象无常牌 deck of many things 中头骨牌召唤的死亡化身从中出现并发起攻击。 |
| 86-90 | 接下来24小时内，区域内一个生物通过法术恢复生命值时，这道法术会留下伤痕。这可能同时伴随着黑色胆汁变得纯净，或是一股灵体力量从生物体内挣脱出来。这些伤疤只能通过高等复原术 greater restoration 或祈愿术 wish 去除。 |
| 91-95 | 接下来24小时内，一缕明亮的蒸汽会在该区域内的尸体或坟茔上空飘荡。如果蒸汽被装入一个容器中，持有该容器的生物能够施展一次无须任何成分的复生术 resurrection 这同时也会导致蒸汽消失。任何以这种方式复活的生物都会经历古怪的梦境。 |
| 96-00 | 一股神秘的雾气自阴影中升起。浓雾会对以区域内一个随机生物为中心、50尺半径球状区域内的一切构成重度遮蔽。任何在雾中开始回合的生物必须成功通过 DC10的体质豁免否则获得一级力竭等级。在雾气中时，该力竭无法被去除。此外，生物会在雾中看到令人不安的景象，比如不祥的废墟或无声的人影在逃离追捕。雾气无法被任何风所吹散，但会在1分钟后自行消失。 |
