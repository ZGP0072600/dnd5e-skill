# 自定义内容库（Homebrew）

玩家 / DM 自创或本地重建、官方书未收录的内容，格式与 `docs/extracted/` 一致（`.md` + frontmatter）。

**查询优先级**（车卡 / 跑团 / 枚举时按序命中，先命中先用）：
1. `campaigns/<当前战役名>/homebrew/`（战役专属，最高优先级）
2. `homebrew/`（全局，跨战役复用）
3. `docs/extracted/`（官方书，兜底）

子目录按类型分：`races/ classes/ subclasses/ feats/ backgrounds/ equipment/ magic-items/ monsters/ spells/`。
